# 網站文章資料_元智大學研發「下肢機械服」　協助長者、傷者行走_210

## 一、文章基本資料

- **文章ID**: 210
- **文章標題**: 元智大學研發「下肢機械服」　協助長者、傷者行走
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【NOWnews/記者鍾佩芸／台北報導】

元智大學電機系副教授邱智煇指導學生余東浩、吳其原、馮震昊、林杰民、吳佳霖研發「下肢機械服」，使用者穿戴上後可透過腦波或搖桿控制行走，不但可協助脊損傷友、不良於行的長者復健，也可協助搬運或災難救援等工作。

下肢機械服是下肢外骨骼的輔具，研發團隊在各關節上加裝電動缸，能夠增強下半身出力與保護下肢關節，甚至可以作為人類下肢使用，邱智煇說明，他們實驗室原本就投入相當大的心力與經費製作研究專用的大型人形機械，為了跳脫研究層次，結合民生科技，於是指導學生投入製作下肢機械服，其實這類輔具在國際上有不少研發成果，美國國防部甚至拿來作為戰場搬運工具。

邱智煇表示目前下肢機械服僅能帶動使用者「向前走」或「停止」，仍缺乏病友願意穿戴測試，近期的目標是簡化管線及感測器，未來也可能將實心鋁塊改為碳纖維，減輕重量，希望未來有機會量產，造福廣大長者、病友。
