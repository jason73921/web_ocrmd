# 網站文章資料_人工智慧能取代人腦嗎 元智大學教卓談_1669

## 一、文章基本資料

- **文章ID**: 1669
- **文章標題**: 人工智慧能取代人腦嗎 元智大學教卓談
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【大紀元記者徐乃義臺灣桃園報導】

當Google人工智慧AlphaGo，擊敗了韓國職業九段棋士李世乭，成為各方熱議的發燒話題，而人工智慧也成了當代討論度之高的顯學。元智大學副校長林志民教授，以近年討論度之高的“人工智慧”為主題，分享其近年來豐碩的研究成果。林志民的研究領域，主要為人工智慧方法之開發及應用，歷年來共發表期刊論文172篇及會議論文168篇，研究領域包含人工智慧、智慧型控制、機器人、醫學應用、訊號處理、飛行導引控制、目標偵測與追蹤。

林志民表示，人工智慧並非近年才出現，早在十幾年前，就有許多專家學者投入研究，像是美國、中國政府就投注了很多資金於此。但究竟何謂“人工智慧”？林志民說，人工智慧其實是一種類神經網路，透過精密科學方式，模仿、製造並且操作其運作系統，使之擁有與人同等的智慧與反應，並且能夠廣泛應用於各領域。

林志民表示，人工智慧就像是真實人類的智慧，可以透過反復練習、無數經驗而趨近成熟完美。人工智慧就跟人類一樣，你只要訓練它，它就會越來越好。就像老醫師帶著實習醫生臨床看診，久而久之，透過長時間訓練與實戰經驗，實習醫生也會變正式醫生，人工智慧亦是同樣道理。基於此概念，只要在各個領域上培養不同人工智慧，人工智慧就能在各領域上發揮所長。

這幾年的研究成果，便將人工智慧應用於乳癌腫瘤電腦輔助診斷系統，透過人工智慧的精密檢測，輔以其他醫療儀器，乳癌腫瘤就能夠準確被察覺。或是將人工智慧應用於汽車自動跟車防撞系統、防鎖死刹車系統設計，讓汽車安全設備更加健全有保障。

林志民說，智慧型飛彈導引系統設計，也能透過人工智慧之指引，讓飛彈導引系統運作更為縝密，還有飛機自動降落系統，透過人工智慧的幫忙，亦能保障飛安。人工智慧系統應用領域極為廣泛，而林志民在醫學、國防及工程各個領域都有豐碩的研究成果。

人工智慧會不會取代人腦？林志民樂觀表示，人工智慧仍無法取代人腦，但以資料資料為主的職業，就會逐漸被取代，像是會計師等。隨著時代的進步，找出自己的利基，關注時勢的發展，找出自己的定位，如此就不怕被取代啦。
