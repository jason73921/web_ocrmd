# 網站文章資料_使用設定_251

## 一、文章基本資料

- **文章ID**: 251
- **文章標題**: 使用設定
- **文章網址**: https://www.yzu.edu.tw/admin/is/index.php/tw/2015-09-21-05-36-11
- **文章簡介**: 
- **文章全文**: 如果你的無線網路卡驅動程式已經設定好，且可正常使用，網路相關設定值如下，只要設定無誤即可使用網路了。

TCP/IP:設成自動取得IP。
 SSID:連線至"YZUWLAN"。
 如果你不太懂得如何設定網路，請詳細閱讀，會讓你得到你要的資訊。因各家網卡安裝程式皆不相同，故以本校所提供的網路卡做範本。

●無線網路卡設定
 首先要確定無線網卡是否有連結到無線存取集線器(Access Point ; AP)，以下簡稱AP。在工作列右邊可發現無線網路連線圖示。
 在該圖示上按一下左鍵會顯示所搜尋到的AP，請點選"YZUWLAN"後按連結。
 如果都沒有找到任何的AP或無法取得正確的IP(正確的IP為140.138開頭)，可以試著按「重新整理」鈕以重新尋找AP或重新連結。
 不建議使用自行安裝的連線軟體，因為許多連線軟體會紀錄之前所連結過的AP，造成無法連線。

 ●TCP/IP網路設定
 請在工作列中點選「開始」→「設定」→「網路與撥號連結」→「區域連線」(這選項請視個人電腦硬體來做選擇)，會出現一個視窗，請在此視窗中選擇「Internet Protocol(TCP/IP)」，然後按「內容」，在所顯示視窗中就可設定IP位址。
 請將IP設成「自動取得位址」以及DNS設成「自動取得DNS伺服器位址」 。然後按「確定」即可得到IP位址了。

 ●瀏覽器設定
 使用自動組態設定方式，相關設定請參照瀏覽器如何設定proxy。

 ●使用網路
 當以上設定完成後，就可以連結網路了。
 在使用網路前，還必須要通過身份認證方可使用。
 當打開瀏覽器時，安全機制會先導到認證網頁(如下圖)，輸入個人電子郵件帳號與密碼後，如確認無誤即可使用網路了。
