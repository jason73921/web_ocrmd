# 網站文章資料_國立高雄科技大學:敬邀貴校踴躍參觀「台北國際航太暨國防工業展覽會」_7869

## 一、文章基本資料

- **文章ID**: 7869
- **文章標題**: 國立高雄科技大學:敬邀貴校踴躍參觀「台北國際航太暨國防工業展覽會」
- **文章網址**: https://www.yzu.edu.tw/admin/rd/index.php/tw/?option=com_content&view=article&id=7869
- **文章簡介**: 主旨：敬邀貴校踴躍參觀「台北國際航太暨國防工業展覽會」，詳如說明，請查照。

說明：

一、為提升國內大專校院與高中職校對航太與國防工業之知識與產業認識，台灣區航太工業同業公會於展覽會設置產業專區，與教育部促進產學連結合作育才平臺-國立高雄科技大學執行辦公室長期合作推動產業人才培育，邀請全國學校相關科系師生參觀。

二、展覽資訊：

(一)展覽時間：2025年9月18日至9月20日9:00至17:00。

(二)展覽地點：台北南港展覽館1館1樓(台北市南港區經貿二路1號)。

(三)團體參觀時間：9月18日12:00至17:00、9月19日至9月20日9:00至17:00。

(四)若有意申請團體進場參觀請於9月3日中午12:00前將申請 文件寄至karlhsiao@nkust.edu.tw進行報名統籌。

(五)團體參觀申請文件(學生團體申請表、學生團體資料表)請至網站下載：https://www.tadte.com.tw/zh-tw/menu/9BC8A4C92D674714D0636733C6861689/info.html。
