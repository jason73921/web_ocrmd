# 網站文章資料_元智研發「可視資訊隱藏」 上網更安全_273

## 一、文章基本資料

- **文章ID**: 273
- **文章標題**: 元智研發「可視資訊隱藏」 上網更安全
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【聯合報/記者馮靖惠/即時報導】

2015年台北國際發明暨技術交易展上周落幕，元智大學資傳系教授林珮瑜的作品，結合現代人社群分享、網路資安需求，透過一鍵軟體設定，馬上對拍攝的照片與影片，進行即時所有權保護與機密分享，讓上網更安全，獲得銅牌獎，該研發已申請專利中。

本次發明競賽中，元智大學共獲得3面金牌、1面銀牌以及2面銅牌。其中，資傳系教授林珮瑜以作品「可視資訊隱藏方法」獲得銅牌獎殊榮。林珮瑜表示，智慧型手機、平板電腦等行動裝置設備已成為最廣泛使用的拍照設備，媒體在社群分享與網路傳輸是必然的應用趨勢。但上傳媒體時，保護媒體所有權及共享機密訊息，是目前行動裝置在拍照與社群共享時重要的保護需求，然而現階段的行動裝置，卻相當缺乏有效的即時媒體保護應用技術。

林珮瑜表示，此研發可廣泛應用到行動裝置拍攝的媒體，相容於任何有拍攝功能的行動裝置與數位攝錄影機，無需擴充周邊設備並節省能源消耗。透過一鍵軟體設定，馬上對拍攝的照片與影片，進行即時所有權保護與機密分享。此研發讓資訊安全不再是無趣的技術，結合行動裝置運算與社群應用，人人都能保護自己的媒體。

林珮瑜說，使用介面易於操作，以活潑的色系，做為Secret Photo的主軸，充滿生氣又具年輕化，就像Secret Photo將資訊安全與密碼學變成有趣的事情。繽紛的色彩也能捉住熱愛使用照片編修軟體的年輕族群、社群分享族群以及藝術創作者。

此研發在2015年台北國際發明暨技術交易展中，脫穎而出獲得銅牌獎項，林珮瑜說，本研發有效結合現行手持裝置內建圖形處理，運算量小、複雜度低，可大量普及應用至智慧型手機、平板電腦、PDA、數位攝錄影機等手持裝置以及電腦設備，是現代智慧手機應用最佳夥伴。
