# 網站文章資料_YC資源網 青年創業最佳幫手_954

## 一、文章基本資料

- **文章ID**: 954
- **文章標題**: YC資源網 青年創業最佳幫手
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【大紀元記者徐乃義台灣桃園報導】

網路的時代，資訊更新的速度十分快速，台灣青年必須面對來自全球的競爭挑戰，別人無法代替青年規劃未來，但當青年需要時，提供最好的支援。因為，同 溫層現象限制青年取得資訊的廣度，青年創新創業的方向五花八門，應該要建立一個公開且廣納各項資源的平台，讓青年適才適所、朝自己的夢想邁進，「TYC資 源網」將會是青年就業、創業的最佳幫手。

桃園青年事務局長陳家濬指出，與會廿三歲的中大動力機械工程系碩一生張東琳，是航見科技執行長，透過TYC資源網了解創業空間相關資訊，找到適合的空間─青創指揮部，已進駐，與12個工作團隊一起完成創業夢想。

元智大學工業工程管理系沈同學表示，透過TYC資源網，找到許多有興趣的課程及活動，即使有時因時間衝突無法參加的講座或是課程，TYC資源網提供了線上課程，讓他可以隨時隨地上網充實自己。

陳家濬指出：「TYC資源網」，分為「職涯發展」及「創業」兩大專區，囊括在地鏈結、學習資源及實務機會的資訊，整合各類創業與職涯相關活動訊息， 並提供桃園創業社群、活動空間的登錄服務，以及創業者最直接的創業達人輔導服務申請，集所有青年需要的創業、職涯發展資訊於一地，讓青年輕鬆找到需要的資 源！

TYC資源網以架構開放、資源整合的概念運行，期望成為未來桃園青年在職發展與就業創業的選擇上最好的資源參考網，對創業、職涯發展有想法的青年搶先體驗，一起享用桃園創新創業生態圈的資源。
