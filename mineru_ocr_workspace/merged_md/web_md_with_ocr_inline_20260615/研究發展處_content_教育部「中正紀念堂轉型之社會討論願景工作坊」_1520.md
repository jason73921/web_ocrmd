# 網站文章資料_教育部「中正紀念堂轉型之社會討論願景工作坊」_1520

## 一、文章基本資料

- **文章ID**: 1520
- **文章標題**: 教育部「中正紀念堂轉型之社會討論願景工作坊」
- **文章網址**: https://www.yzu.edu.tw/admin/rd/index.php/tw/?option=com_content&view=article&id=1520
- **文章簡介**: 主旨：轉知文化部推動中正紀念堂轉型之社會討論，將在107年2月及3月間，增辦3場願景工作坊，邀請民眾報名參加，請協助公告，請查照。

說明：

一、依據文化部107年1月25日文藝字第10730026031號函辦理。

二、文化部推動中正紀念堂轉型之社會討論計畫，繼去年12月已舉辦4場願景工作坊之後，為擴大民眾參與將增辦3場次，於今年2月3日(星期六)、4日(星期日)及3月10日(星期六)在文化部南海工作坊(臺北市南海路20號)各辦理1場，即日起開放報名。

三、活動採審議民主機制，希望透過多元形式的社會討論，鼓勵民眾參與中正紀念堂轉型之議題思考及意見分享，從相互理解、深度對話，聚焦未來願景。有興趣者可上網填寫表單(連結：https://goo.gl/8U9UPZ)報名，或下載報名表(如附件，連結：https://goo.gl/hEXcJS)填寫後郵寄至台北市懷寧街92-1號7樓，沃草有限公司收。

四、每場參與人數以20人為原則，本案執行單位將從報名者中依性別、年齡、地區等原則抽樣選出後通知參加者。每場次均提供網路現場直播，讓未能參與的民眾線上觀看。

五、有關中正紀念堂轉型之社會討論，已辦理完成的願景工作坊情形及報名資訊，詳見「中正紀念堂轉型給問嗎」臉書粉絲專頁，及計畫官網「中正紀念堂轉型給問嗎-轉型正義暨中正紀念堂資訊平台」（http://___cks.moc.gov.tw/event），也可洽願景工作坊執行單位聯絡人：王先生，電話：02-2361-7588，電子郵件：wangseal@watchout.tw。

六、檢附文化部函文及報名表各1份。
