# 網站文章資料_【函轉】「跟著鏡頭看臺灣-2017紀錄片行動列車」活動_726

## 一、文章基本資料

- **文章ID**: 726
- **文章標題**: 【函轉】「跟著鏡頭看臺灣-2017紀錄片行動列車」活動
- **文章網址**: https://www.yzu.edu.tw/library/index.php/tw/2017071901
- **文章簡介**: 一、為拓展紀錄片映演空間及發表園地，增加全國觀眾觀賞紀錄片之機會，本局委託財團法人公共電視文化事業基金會辦理「跟著鏡頭看臺灣─2017紀錄片行動列車」活動，自本(106)年7月22日至10月22日期間於全臺15個據點巡迴映演。
二、本年度精選25部本國優秀紀錄片，分為三大單元「生命的禮讚」、「人文的關懷」、「時代的悸動」放映；並舉辦25場映後座談，邀請導演或影人與觀眾面對面交流，詳情可參閱活動官網（http://www.docsbamid.org）。
三、敬請貴單位協助轉知旨揭活動訊息，鼓勵所屬人員踴躍參與。本活動配合「公務人員終身學習入口網站 (https://lifelonglearn.dgpa.gov.tw)」與「全國教師在職進修資訊網(http://www2.inservice.edu.tw/index2-3.aspx)」，凡映後座談場次全程參與者，可獲登錄學習時數2小時，相關資訊亦可至前揭網站查詢。
