# 網站文章資料_砸3萬「學睡覺」疲憊經濟超夯！企業還發睡眠獎金_3500

## 一、文章基本資料

- **文章ID**: 3500
- **文章標題**: 砸3萬「學睡覺」疲憊經濟超夯！企業還發睡眠獎金
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20190116
- **文章簡介**: 【東森財經新聞】

對於許多學生和上班族來說，「睡覺」絕對是很可貴的休息時光，不過卻有不少人常常有睡不飽或是睡不好的狀況，導致平常容易感覺到疲累，休假補眠甚至還越睡越累。英國一名健身教練就對此推出一門「睡眠指導課程」，讓客戶能感受到真正的休息放鬆，然而為了睡一頓好覺，收費可不便宜。

根據《金融時報》(Financial Times)報導，英國倫敦知名的健身教練Robert McCabe最近推出3個月的睡眠指導課程，他指出，有一位客戶覺得自身睡眠狀況沒問題，不過卻總覺得累，實際追蹤後才發現，該客戶每晚睡眠時間其實低於他所說的7小時。

之後這名客戶花了750英鎊（約台幣3萬元），向Robert McCabe買了為期12周的睡眠課程，經過調整之後，他每晚都能多睡2個小時，不但精神變好，且體力也提升，以前單手舉啞鈴只能舉16公斤，現在可以舉到30公斤。

Robert McCabe的睡眠指導課程屬於一種「疲憊經濟」，因為有許多人都因睡眠問題，進而影響到生活上的表現。對此，睡眠學校（The Sleep School）的主任Guy Meadows表示，幾年前多數企業還不太重視睡眠品質問題，但這幾年有愈來愈多公司都對睡眠課程產生興趣，可見企業文化已有所轉變。根據調查有70％的受訪主管認為企業應該重視睡眠管理；日本就有公司提供「睡眠獎金」，鼓勵員工要睡飽有精神。

在這個繁忙的世代，人人都處於龐大的壓力之下，因此如何排解壓力便是相當重要的議題。日前，元智大學完成一份有關壓力的問卷調查，其中回收了562份的有效問卷，對於壓力的處理方式，其中有4成的受訪者選擇睡覺，而找心理醫生的比例不到1%。（待續）
