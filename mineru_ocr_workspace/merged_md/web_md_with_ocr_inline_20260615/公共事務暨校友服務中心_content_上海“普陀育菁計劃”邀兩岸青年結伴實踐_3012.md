# 網站文章資料_上海“普陀育菁計劃”邀兩岸青年結伴實踐_3012

## 一、文章基本資料

- **文章ID**: 3012
- **文章標題**: 上海“普陀育菁計劃”邀兩岸青年結伴實踐
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20180717-6
- **文章簡介**: 【新華社/上海7月13日電/記者潘清】

“我是一個喜歡挑戰和改變的人。”台灣交通大學電子工程專業四年級學生王意麟說，為了挑戰和改變，他報名參加上海“普陀育菁計劃”兩岸青年企業實踐訓練營，並順利成為50名營員之一。

未來的一個月內，王意麟將和來複旦大學的女孩牛小雪結成“學伴”，一起進入知名企業麗嬰房實習。雖然時間不長，王意麟仍希望從這次實踐活動中學習到與自己專業“不一樣的東西”。

本屆訓練營吸引了兩岸及海外3000餘名大學生報名。經過層層遴選，50名優秀學生最終幸運入選。其中，25名大陸營員分別來自複旦大學、交通大學、華東師範大學、北京科技大學、浙江大學等高校，25名台籍營員則來自元智大學、輔仁大學、台灣中山大學、台北科技大學等十餘所台灣及海外大學。
據介紹，為期一個月的訓練營以“互聯共享、助力成才”為主題，並採用“學伴+企業導師+實習+特色活動”的模式。主辦方精心選擇了一批創新型優質企業作為實習基地，並聘請優秀青年企業家和資深職業經理人擔任專業導師，指導營員們完成實訓任務。(待續)
