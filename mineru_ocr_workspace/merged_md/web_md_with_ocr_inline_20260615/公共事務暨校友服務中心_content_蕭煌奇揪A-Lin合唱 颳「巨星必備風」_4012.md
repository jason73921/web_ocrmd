# 網站文章資料_蕭煌奇揪A-Lin合唱 颳「巨星必備風」_4012

## 一、文章基本資料

- **文章ID**: 4012
- **文章標題**: 蕭煌奇揪A-Lin合唱 颳「巨星必備風」
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20191005
- **文章簡介**: 【中國時報】

蕭煌奇18日將推出全新國語專輯《候鳥》，第二波主打單曲〈迷路在雲端〉邀天生歌姬A-Lin合唱，兩人赴桃園元智大學拍攝MV時，校園颳起少見的強風，一度讓劇組被迫暫停，等待期間A-Lin幽默說：「這風吹來的效果有如『巨星必備風』，拍起來效果氣勢十足，完全不用準備電風扇。」〈迷路在雲端〉單曲MV在4日已數位上架。

蕭煌奇說，一開始聽到〈迷路在雲端〉時，完全沒考慮第二人選，認為只有A-Lin能詮釋，A-Lin也義氣相挺秒答應。其實兩人在20年就認識，當時都在西門町音樂餐廳駐唱，常被安排在同一天前後上下場的駐唱時段，如今再度同框演出，兩人笑說「淵源很深，太有緣」。（待續）
