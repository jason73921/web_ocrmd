- **文章全文**: 【經濟日報 劉美恩】 


2016「第21屆全國大專校院資訊應用服務創新競賽」日前圓滿落幕，桃園市政府首次以「桃園開放資料創新應用組」設題，吸引多所大專校院資訊相關科系學生組隊參賽，作品主題包括疲勞駕駛偵測、視障服務、停車系統及防災等面向，獲評審及企業肯定。桃園市政府研究發展考核委員會主任閻俊如表示，桃園市政府從去年底推動Open Data以來，積極開放與民眾生活相關的資料，參與本次競賽的目的是希望能迅速讓全台師生知道桃園已經有開放資料，並鼓勵青年學子多多運用桃園開放資料，進行創新應用。

最後，由元智大學設計的疲勞駕駛偵測作品「TiRing for Taoyuan」，拿下該組別的第一名。作品是應用影像辨識技術，觀察汽車駕駛人眨眼的頻率，判斷並警示駕駛人是否處於疲勞駕駛的狀態。透過桃園的開放資料，導引駕駛人就近至便利商店、咖啡廳、連鎖餐廳、休息站或旅館休息，藉此降低交通事故發生。 


第二名的作品是國立交通大學的「視障者行動輔助系統」，是從「桃園開放資料」中找出可能的路況問題，例如：道路施工或是車禍肇事熱門路段，將地址轉為GPS提供提醒報讀，指引視障者安全地通過潛在危險路段，並輔佐專員視訊及設 定專屬手勢進行行動支付提供多元服務。此作品還同時拿到本屆競賽的「台達特別獎」，別具意義。第三名是世新大學的「Besafe 安全顧問」，而佳作則由國立臺中科技大學的「停車資訊系統」及東海大學的「以Google fusion table為基礎之火災發生統計與消防資源分布開放資料混搭服務」拿下。

除學生競賽外，今年也於10/27舉辦「桃園Open Data x智慧城市研討會」邀請Open Data Tw張維志計畫負責人、天氣風險管理開發(股)公司彭啟明總經理以及景翊科技(股)公司陳奕廷總經理暢談「開放資料企業案例」及「與智慧城市之鏈結」，交流座談中也針對桃園在地特色，就公共服務與商業應用議題提供寶貴意見；9/24舉辦「Maker X Hacker桃園開放資料黑客松」，桃園市副市長王明德特別到場給予現場所有團隊加油打氣，並表示：「青年代表創意與希望，期待透過黑客松活動加速年輕人多加使用桃園開放資料產生更多創新服務，未來讓社會及各行各業都能享受到開放資料所帶來的美好」。本次首獎得獎團隊「防盜風水師」，其作品主要是讓民眾可對自己的住家進行防竊檢測及預防，保障住家財產及人身安全。獲得評審一致認同的原因是本作品使用的政府開放資料多樣性高、綜合「跨領域」的資料以不同的觀點來解決問題，並將開放資料進一步整合及分析，達到「加乘」的效果，以「視覺化」的方式呈現，讓使用者覺得有感、好用！ 


今年三場活動成果均獲各界不錯的回響，歡迎有興趣的民眾可上「桃園開放資料. 揪Open活動網（https://od.tycg.gov.tw）」查詢活動內容及作品介紹。未來將持續透過多元形式鼓勵開發者善用桃園開放資料，開發更多觸動人心的應用服務，讓桃園成為幸福城市。
