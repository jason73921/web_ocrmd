掠奪性期刊

嫌疑黑名單



n   查詢出版商或期刊是否被列入Beall"s List of Predatory Journals and Publishers

(Beall"s List；本清單於2017年停止更新)






開放取用期刊白名單



n   利用Open Access Scholarly Publishers Association(OASPA；開放取用學術出版者協會)查詢投稿出版社是否為謹守學術倫理的發行機構

n   利用Directory of Open Access Journals(DOAJ；公開取用期刊指南) 查詢投稿期刊是否為具公信力的OA期刊

自我查核表



n   透過Think. Check. Submit.協助判斷期刊是否來自合法的組織，當研究者的回答皆為「是（Yes）」時便能考慮投稿。

n   中文版：http://thinkchecksubmit.org/translations/chinese-traditional/

n   想一想、檢查檢查、再投稿（臺灣師範大學圖書資訊學研究所柯皓仁教授兼圖書館館長翻譯；收錄於百世經綸個人部落格）
