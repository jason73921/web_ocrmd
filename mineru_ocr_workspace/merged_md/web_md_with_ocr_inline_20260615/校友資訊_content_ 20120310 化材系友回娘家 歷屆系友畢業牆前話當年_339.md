# 12012年化材系友回娘家歷屆系友畢業牆前話當年

化學工程與材料科學學系配合元智大學23週年校慶活動，當天（3/10）於二館2207會議室舉辦「系友回娘家」活動，活動開始由張幼珍主任介紹母系現況及目前企業對校友評價狀況，接著工程學院院長莊瑞鑫教授及系友會理事長林祈明學長分別致詞，隨後由李忠先生向系友介紹「校友資料庫」使用說明，希

![](images/928ad3cc16e88546a63abe1e05ebf5a96bab38d58f9848f80f73eee147713d8a.jpg)

望系友經常進入此资料庫更新近況，另外告知母系成立【facebook】平台，藉由此平台可更快速獲知系上發展近況及最新活動及就業訊息。

隨後安排師長與校友餐敘，與會張幼珍主任、林秀麗老師、孫一明老師、洪信國老師及陳永信老師與學長姐言歡敘舊，系友在母系「歷屆系友畢業照」牆面中見到當年畢業照，紛紛感動回憶當年在校時光，表示能有機會回到母校和師長及學弟妹們相聚、和久未見面同學一邊餐敘，一邊互聊近況，實在是難得的機會，也期許明年大家相約再回到母校一同歡慶學校生日！
