# 網站文章資料_一位台灣7年級生的重慶咖啡故事！_1750

## 一、文章基本資料

- **文章ID**: 1750
- **文章標題**: 一位台灣7年級生的重慶咖啡故事！
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【Etoday】 


一個人需要多久才會愛上一個陌生的城市？對32歲的臺灣桃園人黃星瑋而言，只用了短短半年時間。重慶的南濱路，北臨長江，背依南山，這裡不僅有著如台北忠孝東路般的繁華，也像台北的西門町一樣，播下了許多年輕人最炙熱的夢想。黃星瑋的「玻璃屋花園咖啡」夢就是其中之一。

「2010年，我從台灣元智大學交流至重慶大學進行半年的學習。那時，重慶的奶茶、咖啡店數量非常有限，身邊的同學好多都是喝即溶咖啡。」黃星瑋告訴記者，在臺灣，咖啡是一種普及率很高的飲品，台灣人平均每天都要喝上2-3杯咖啡。「幾乎沒有人會去喝即溶咖啡，尤其是年輕人。」

「當時，我內心就埋下一個種子，想著以後有機會可以來重慶開家咖啡館，在我喜愛的這座城市裡，讓更多的人體會到咖啡的美好。」他說。

回臺灣之後，這樣的念頭也時常在黃星瑋的腦海裡盤旋。

「我大學畢業之後在臺灣有一份穩定的工作，在聯邦快遞從事報關員的工作，薪資很高，親戚朋友都很滿意。但我自己卻總覺得少了點什麼，我還這麼年輕，為什麼不勇敢地去為自己的夢想拼搏一把呢？」黃星瑋說。

就這樣，黃星瑋辭去了在台灣的工作，隻身一人來到大陸，開始了自己的逐夢之旅。

2014年8月，黃星瑋來到重慶，在四川外國語大學附近開了自己的第一家咖啡店——莎士比亞咖啡。

談起自己的「處女作」，黃星瑋至今仍感慨良多。

「當時，重慶市場對現磨咖啡的認知度並不高，咖啡豆的進貨管道也不是很順暢，一度讓我很受挫。」他說，「但很幸運的是，我當時碰到了一個好心的房東，她看我一個臺灣小青年來重慶創業，就給我提供了很低廉的租金，大大節約了我的成本，她算是我當時的第一個支持者。」

頂著外人「還不如開麻將館賺錢」的調侃，黃星瑋的咖啡之路卻越走越堅定。

「至今為止，我覺得超自豪的是，很多重慶大學生的第一杯現磨咖啡是在我這裡喝的。」他笑言道，自己也因此結交了許多大陸的朋友。「我會給他們介紹台灣的美食和旅遊，他們會帶我去重慶一些好玩的地方，我很喜歡和他們交流。」

2016年，「不安分」的黃星瑋將莎士比亞咖啡轉給了一位重慶的朋友，又經過多番對比，和兩個合夥人在南濱路上開了這家「玻璃屋花園咖啡酒吧」。

「我這個咖啡館有三個股東，一個重慶的姐姐，還有一個澳籍華裔的哥哥，這樣的股東構成也是挺有趣的吧。」他說。

在記者採訪的間隙，咖啡館裡來了一對頭髮花白的夫妻，黃星瑋笑著對我們說：「看到他們沒？幾年前，我根本不敢想像像這樣的中老年人會來喝咖啡，現在連他們都來了，我覺得自己太厲害了。」

隨著大陸「大眾創業、萬眾創新」的氛圍愈發濃厚，越來越多的的臺灣年輕人也希望融入到這股「雙創」熱潮之中。

「我身邊有好多臺灣朋友其實很想來大陸發展，但是卻缺乏對大陸的瞭解和創業的勇氣。」黃星瑋頗為無奈地告訴記者一件「趣」事：「我之前在臉書上曬過咖啡館的照片，好多臺灣朋友居然問我說這是不是重慶前三名的咖啡館，我當時哭笑不得，台灣年輕人實在太不瞭解大陸了。」

「大陸的市場遠遠超出台灣青年的想像範疇，只要擺正心態，足夠努力，足夠勇敢，試著用當地人的思維去思考，在大陸一定能拼出來！」黃星瑋說。（新聞來源：新華社港台部提供）
