# 網站文章資料_免口試！大學個人申請 23校試辦「逕行錄取」_1688

## 一、文章基本資料

- **文章ID**: 1688
- **文章標題**: 免口試！大學個人申請 23校試辦「逕行錄取」
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【聯合新聞網／記者洪欣慈／台北報導】

大學個人申請3月啟動，大學招聯會106學年首度試辦個人申請直接錄取免口試，有23校參與，被錄取的考生不用再舟車勞頓、奔波甄試，名額上限2成，未來將視成效調整。升學輔導專家建議，若有不擅長面試或口試的學生，可以好好把握此機會。

升學輔導專家張財銘表示，今年共23所學校參與「逕行錄取」試辦，若是不擅長面試或口試的考生，可以好好把握，用心準備書審資料，就有機會免經二階段直接錄取。

大學招生委員會聯合會去年宣布，個人申請表現優異考生，各學系可自行決定直接錄取不用面（口）試，106學年先試辦1年，由各大學自行決定是否試辦及辦理方式，總招生名額以個人申請管道2成為上限。 

大學招聯會統計，106學年共有23校參與試辦方案，包括清華大學、政治大學、中央大學、中山大學、元智大學、東吳大學等校。

大學招聯會發言人、靜宜大學校長唐傳義指出，有些科系如資工等，已經累積很好的書審經驗，較不需要透過面試來觀察學生特質，就可以找出適合的學生。

清大教務長戴念華表示，個人申請第一階段多是3倍率篩選，若該科系要錄取50人，就有150名考生得參加第二階段甄試，過去不少學生、家長都反映，參加甄試的住宿、交通等費用太高，如果表現特別優秀者能直接錄取免口試，考生也不用舟車勞頓。

戴念華指出，106學年仍是試辦性質，由各校自行決定，清大也開放各系自行參加試辦，若考生被試辦科系通知直接錄取免口試，但是考生仍願意到系上參加團體或個別面試，學校也不會拒絕。
