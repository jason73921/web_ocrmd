全國文獻傳遞服務(NDDS)

申請步驟：

1. 申請NDDS讀者帳號讀者帳號（首次使用者），請先進入NDDS系統，申請讀者帳號。待圖書館以E-mail寄發啟用通知。（註：無法以 Firefox 或 Chrome 連結NDDS系統者，請暫時改用 IE 瀏覽器）。
2. 查詢聯合目錄或國內其他圖書館館藏目錄。 
3. 線上填寫NDDS申請單（儘可能填寫3個申請對象）。 
4. 申請件進度查詢：請至NDDS「讀者專區」選項中查詢。
5. E-mail通知申請者取件。 
6. 到館付款取件：請至圖書館館合櫃台辦理。 

◎各館複印件收費一覽表 

◎各館圖書借閱收費一覽表

◎相關諮詢：電洽03-4638800#2294。

◎注意事項：

1. 請尊重智慧財產權，申請複印資料以部份為限，並僅供學術研究之用。
2. 請遵守各合作館文件複印及借書的規範。資料逾期未還或未取，須自行負擔該筆申請件費用(含申請費用、郵資)，並停止館合服務申請權及本校借閱權至繳清費用止。
3. 送出表單後，請隨時檢查個人的電子郵件信箱，注意是否有申請館際互借或複印的資料到館通知。

圖書服務計畫-化學中心總圖書室New

申請步驟：


進入GOOGLE表單提出申請。

接獲E-Mail取件通知後，請至圖書館櫃檯取件。

處理時間：上班日早上9:00-下午5:00，於受理申請後四小時內回覆，當日下午4:00後申請單，會於隔上班日回覆。
