一、   職別：專任教師。

二、   名額：2名。

三、   起聘日期：自112年2月1日起聘。

四、   資格：具備國內外資訊、電機，電子等相關領域博士學位，研究上有優秀表現，具高度教學與服務熱忱、團隊精神、能與人和諧相處者，並具英語授課之能力者。

五、   專長領域：

(一)     Electronic Design Automation, Computer Architecture, IC Design, Embedded System, Internet of Things, Quantum Computing  

(二)     Network and Computer Security, Virtual Reality, Cyber Physical System, Big Data Analytics, Cloud Computing and Service

(三)     Operating System, Database System, Compiler

六、   意者請備妥下列文件，寄至：32003桃園縣中壢市遠東路135號資訊工程學系收。請於信封袋上註明「應徵專任教師」，應徵資料恕不退還，資料請以書面郵寄(請同時寄送相關資料之電子檔)，恕不接受傳真。隨到隨審，聘足為止。

(一)     個人履歷、自傳（含大學以上學經歷、專長）。

(二)     最高學歷證件影本【凡持國外學歷者，請繳交經駐外單位驗證過之畢業證書，尚未取得畢業證書者得以臨時學位證明書代替】。

(三)     最高學歷成績單影本。

(四)     具教育部教師證書者，請繳交證書影本。

(五)     代表著作全文。（1.須與個人學歷及學術專長相符 2.非以學位論文為代表著作者，須為近5年內公開出版之學術論文或專書。）

(六)     近七年研究著作目錄(需註明出處)、曾經從事過之研究與重要成果。

(七)     推薦信函（1.至少兩封 2. 請由推薦者彌封。）

(八)    未來研究方向論述與可開授課程之教學計畫書。(可參閱相關課程網址https://portal.yzu.edu.tw/cosSelect/index.aspx?D=G )

(九)     元智大學教師資格審查履歷表。下載網址：https://reurl.cc/5gY5A7

(十)    應徵資料登錄表。下載網址：https://u.yzu.edu.tw/RXcN92JQA 請同步將word格式電子檔寄至：csdept@saturn.yzu.edu.tw

(十一)       其他有助審查之文件 (如得獎或專業領域之專利等) 。
