# 網站文章資料_111年指考刪3主科 商管系擬加採公民_5059

## 一、文章基本資料

- **文章ID**: 5059
- **文章標題**: 111年指考刪3主科 商管系擬加採公民
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20210604-1
- **文章簡介**: 【聯合報/記者潘乃欣／台北報導指考】

大學招聯會昨宣布，一一一學年各校系循考試分發管道招生，至少要採計一科分科測驗成績，只有音樂術科校系暫緩一年實施。許多商管學群科系目前只採計國文、英文、數乙三科，但分科測驗刪除這三科，家長憂這些科系將無這些主科可採，衝擊學生權益。不少相關科系只好考慮加採公民科。

按招聯會公布一一一學年考招方案，各校系在考試分發入學管道至少採計一科分科測驗（考數甲、歷史、地理、公民、物理、化學、生物等七科）成績，以及至多四科學測（考國文、寫作、英文、數Ａ、數Ｂ、社會、自然）成績，並加總二者。

元智大學教務長、管理學院教授曾芳美曾說，管院的財金、資管系尤重數學與邏輯，往年都採國文、英文、數乙三科，指考改制後失去合適的對應科目。可能的因應方案是採計公民，該科包含經濟學，較歷史、地理等貼近商管科系所學。（待續）
