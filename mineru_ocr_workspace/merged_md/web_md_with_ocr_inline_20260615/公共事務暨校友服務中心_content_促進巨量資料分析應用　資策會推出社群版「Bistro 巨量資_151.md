# 網站文章資料_促進巨量資料分析應用　資策會推出社群版「Bistro 巨量資料管理平台」_151

## 一、文章基本資料

- **文章ID**: 151
- **文章標題**: 促進巨量資料分析應用　資策會推出社群版「Bistro 巨量資料管理平台」
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【大成報記者羅蔚舟／台北報導】

在法人科專支持下，資策會數據科技與應用研究所(大數據所)今(20)日舉辦「Bistro 巨量資料管理平台記者發表會暨論壇」，正式公開發表「Bistro 巨量資料管理平台」社群版Bistro Community產品，資策會大數據所所長林蔚君指出，該所研發的Bistro Community產品，以及簡化巨量資料分析運算所需基礎環境的部署與管理，提供彈性的軟體套件安裝與簡單易用的管理界面等等，都讓資料分析師可以專注在領域巨量資料分析的工作，降低巨量資料開源軟體的技術門檻與投資門檻，促進國內在各領域巨量資料分析應用。

資策會MIC研究報告指出，巨量資料分析市場在雲端運算、行動應用、社群商務逐步發展成熟，以及企業採用意願提升等因素驅動下，預估2015年的服務市場將呈現穩定的成長趨勢，台灣巨量資料的市場規模預計將達新台幣81億元，較去年成長21.4%，2017年市場規模更上看新台幣111億元。巨量資料儼然成為下一波產業創新與企業競爭的主要動力。

目前巨量運算平台皆普遍建置於開源軟體(是一種原始碼可以任意取用的電腦軟體，這種軟體的版權持有人在軟體協定的規定之下保留一部分權利並允許使用者學習、修改、增進提高這款軟體的品質)之上，對於一般公司而言，分析巨量資料的門檻頗高－例如：開源軟體眾多，套件版本管理複雜、非結構化資訊的蒐存取用處理困難、計算資源利用欠缺彈性以及計算效能差等相關問題。

「Bistro 巨量資料管理平台」社群版發表會活動，資策會大數據所特別邀請松崗國際、探網科技及新鼎系統共同舉辦，鼓吹網路開發社群能共同參與改進巨量資料開源軟體並探討相關應用機會。在Bistro 巨量資料管理平台論壇安排了合作夥伴單位「探網科技」孫貴章董事長、「新鼎系統」陳振欽副總經理、「松崗國際」呂蘊桐總經理及王冠先先生等，分別以「大數據資料探勘-網路新媒體之非結構化資料分析與應用」、「大數據分析於製程產業之應用」與「大數據到大知識及巨量數值分析」為題，發表專題演說。未來不論在巨量的文字探勘(text mining)分析與巨量的日誌數據(log data)分析，都可運用「Bistro 巨量資料管理平台(社群版)」所提供的開源軟體套件，發展各式巨量資料分析應用。

資策會副執行長王可言指出，在法人科專支持下，由資策會大數據所研發之「Bistro 巨量資料管理平台」，係為是一套兼顧實用、效能升級特色的簡易巨量資料平台，目前就業界或學界試用與教育用途，提供線上軟體下載、線上論壇及線上諮詢服務，對於想入門或從事巨量資料分析的單位來說，正是一大福音。

資策會大數據所所長林蔚君進一步補充表示，資策會大數據所研發的Bistro Community產品，係整合Hadoop(Apache Hadoop是一款支持數據密集型分布式應用並以Apache 2.0許可協議發布的開源軟體框架。它支持在商品硬體構建的大型集群上運行的應用程序)、Spark(Apache Spark是一個開源叢集運算框架，由加州大學柏克萊分校AMPLab所開發。Spark使用了記憶體內運算技術，能在資料尚未寫入硬碟時即在記憶體內分析運算。運算速度能做到比Hadoop MapReduce的運算速度快數倍)等兩大主流巨量資料開源軟體。

透過資料在記憶體的管理優化計算效能的Berkeley BDAS(由 U.C. Berkeley AMP Lab. 發展的大數據分析平台與軟件，它與 Apache Hadoop 是互補的，同時，對比原有組合，BDAS的性能要快得多)等開源軟體套件，以及簡化巨量資料分析運算所需基礎環境的部署與管理，提供彈性的軟體套件安裝與簡單易用的管理界面等等，都讓資料分析師可以專注在領域巨量資料分析的工作。

值得一提的是，資策會大數據所預計於今年(104)年底，將繼續推出功能強化的企業版，升級提供常用非結構化資料轉結構化資料之「資訊管理」與多叢集(可以將一組硬體整合成高度緊密地協作完成計算工作。叢集系統中的單個電腦通常稱為節點，通常通過區域網路連線，用來改進單個電腦的計算速度或可靠性)部署之「系統管理」等功能，企盼能夠協助產業界運用「Bistro 巨量資料管理平台」、掌握國內外龐大市場商機。



(圖由資策會提供／資策會大數據所今(20)日舉辦「Bistro 巨量資料管理平台記者發表會暨論壇」並發表「Bistro 巨量資料管理平台」社群版－Bistro Community產品，特別邀請資策會王可言副執行長(中) 、資策會大數據所林蔚君所長(右三)與合作夥伴緯創資通吳治平副總監(左一)、探網科技孫貴章董事長(左二)、元智大學資訊學院賴國華院長(左三)、新鼎系統陳振欽副總經理(右一)、松崗國際陳友士總經理(右二)等產業界貴賓蒞臨共襄盛舉。)
