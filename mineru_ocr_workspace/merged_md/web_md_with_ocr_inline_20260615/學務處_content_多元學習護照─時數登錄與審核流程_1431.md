# 網站文章資料_多元學習護照─時數登錄與審核流程_1431

## 一、文章基本資料

- **文章ID**: 1431
- **文章標題**: 多元學習護照─時數登錄與審核流程
- **文章網址**: 
- **文章簡介**: 多元學習護照認列活動一覽表   請查詢Portal或Outlook信箱公告可認列多元時數之活動。

依學期別請往下查閱相關規定：

(一)111-2學期領取紙本護照者

112年度基本資料登錄 (←領取護照後請點選並線上填寫) 領取護照後請同學務必線上填寫完成。並依本校多元學習護照實施細則第五條規定辦理首領金認證。

 

111-2學期 多元學習護照時數登錄表單   (←可點選後進行線上填寫)請同學依照各月公告日之規範時程內，線上填寫繳交，每次登錄前請同學務必線上先填寫時數與當月心得。請先寫email與您的輔導老師約定見面時間地點，再依約前往請輔導老師協助您在本次活動登錄最後一筆的頁面上簽名。最後再攜帶您的學生證及多元學習護照在公告時數登錄期間到R8303辦理登錄及認證。

 

112年度多元學習護照-學習記錄暨輔導表 (←可點選後進行線上填寫)一學期僅須填交一次，請同學依照公告日之規範時程內，線上填寫繳交。另請先寫email與您的輔導老師約定時間地點，再依約前往請輔導老師在護照學習紀錄暨輔導表簽名。



   (二)112-1學期 符合多元學習護照系統資格者                                                                         

112學年入學之新生：

* 完成UCAN職場共通職能診斷線上測驗

* 至Portal->多元學習護照線上系統->諮就組期初認證，輸入基本資料

* 點選【諮就組期初認證】，輸入基本資料(←請點選完成線上表單)

* 參與UCAN解測專案說明會線上場次（9/27、10/5請擇一參加）

* 至Portal->多元學習護照線上系統->設定輔導老師，經輔導老師確認核可即完成首領金認證

* 每月最後一個工作天前於Portal->多元學習護照線上系統->時數登錄，登錄線上多元時數與學習心得

* 輔導老師確認當月登錄內容，系統會寄送確認信給學生

 

111學年(含)已經領取紙本護照之舊生，當次登錄僅需登錄112/09/11(含)之後的多元時數，請辦理portal線上登錄：

* 至Portal->多元學習護照線上系統->設定輔導老師，經輔導老師確認核可

* 每月最後一個工作天前於Portal->多元學習護照線上系統->時數登錄，登錄線上多元時數與學習心得

* 輔導老師確認當月登錄內容，系統會寄送確認信給學生

 

111學年(含)已經領取紙本護照之舊生，當次登錄僅需登錄112/09/10(含)之前的多元時數，請辦理實體登錄：

*111學年(含)已經領取紙本護照之舊生，九月份辦理紙本登錄時，請將112/09/10(含)之前所有獲得認證的多元時數都辦理實體登錄

* 至Portal->多元學習護照線上系統->設定輔導老師，經輔導老師確認核可

* 事先與輔導老師約定時間和地點，請輔導老師於護照指定處簽名

*  每月公告日前完成線上表單填寫https://forms.gle/24LomnpbPS7SAYMm7，公告期間內至R8303，登錄紙本多元時數與相關資料

* 本次登錄不受每次最低10小時最多25小時限制，但審查仍只能報支10-25小時限制，超過25小時將儲存在portal，作為下次登錄的前次剩餘時數，須優先登錄

 

111學年(含) 已經領取紙本護照之舊生且需登錄紙本時數及portal線上時數者：

* 紙本與線上可合併計算，每月登錄總時數須達10小時以上，至多25小時

* 至Portal->多元學習護照線上系統->設定輔導老師，經輔導老師確認核可

* 112/09/10(含)之前的多元時數接受紙本認證記錄，請事先與輔導老師約定時間和地點，請輔導老師於護照指定處簽名；每月公告日前依據紙本認證紀錄完成google表單填寫https://forms.gle/24LomnpbPS7SAYMm7，公告期間內至R8303，登錄紙本多元時數與相關資料

* 112/09/11(含)之後的多元時數僅接受portal線上登錄

* portal線上登錄請於每月最後一個工作天前於Portal->多元學習護照線上系統->時數登錄線上多元時數與學習心得

* 輔導老師確認當月登錄內容，系統會並寄送確認信給學生

 

注意事項：

* 112學年度入學之新生一律採線上登錄，並不發放多元紙本護照。

* 112學年度入學之新生須完成首領金認證，方可做後續時數登錄。

* 112-1學期，紙本與線上可合併計算，每月登錄總時數須達10小時以上，至多25小時。

* 若活動參與時間重複，兩筆活動的多元時數將不給予認證。

* 多元護照線上系統優先採用先前的「剩餘時數」，若剩餘時數不足10至25小時之登錄規範，可在系統下方勾選需認證的活動，以滿足10至25小時。

* 多元學習護照助學金用罄時，優先補助經濟條件較為不利者。
