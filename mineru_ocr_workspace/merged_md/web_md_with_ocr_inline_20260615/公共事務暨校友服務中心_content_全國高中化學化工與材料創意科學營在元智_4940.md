# 網站文章資料_全國高中化學化工與材料創意科學營在元智_4940

## 一、文章基本資料

- **文章ID**: 4940
- **文章標題**: 全國高中化學化工與材料創意科學營在元智
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20210318
- **文章簡介**: 元智大學化學工程與材料科學系3/6及3/13舉辦2021年全國高中化學化工與材料創意科學營，吸引來自全台各地區20所高中，共88名學生參與，藉由科學的奧妙及創意實作，體驗化學與材料的神奇, 進而對化學、物理與工程知識產生興趣。

化工及材料產業在台灣經濟發展過程一直扮演舉足輕重的角色，創意科學營除了以知識課程與創意實作活動讓學生更深入了解化學工程與材料科學對於文明社會的關聯與貢獻，並且將化工與材料之科學教育推廣至高中學生。

元智大學化材系主任藍祺偉表示，感謝桃園市青年事務局的贊助，期望透過此活動強化學員們對化工材料領域與台灣化工產業之初步認識，相信有助於高中學員們對科學與工程產生興趣與自我興趣探索。

營隊課程規畫包括，知識課程由化材系藍祺偉主任介紹新世代化工與材料科技及漫談化材系與大學特色介紹，讓學員對化工與材料相關領域有進一步認識；創意實作由化材系洪信國教授、姚少凌教授、黃駿教授、張幼珍教及王清海教授帶領學生團隊，輔導學員進行高分子聚合日常應用、生物材料與自動化材料加工等創意實作，包括隱形眼鏡製作、微生物晶球創意挑戰、纖維生質材料之電漿改質技術、3D列印自動化系統元件組合等。

藍祺偉說，科學與工程是國家科技與經濟發展的基石，透過科學營隊的舉辦讓學生接觸相關的知識，可助於產業科技人才培育，相信學員們在此次活動的學習，更能深刻感受元智大學以創新教學及重點研究為特色，擁有創新、多元、智慧及人文的永續發展校園環境。
