# 網站文章資料_【轉知】教育部112年度大專校院ODF-CNS15251競賽簡章_464

## 一、文章基本資料

- **文章ID**: 464
- **文章標題**: 【轉知】教育部112年度大專校院ODF-CNS15251競賽簡章
- **文章網址**: https://www.yzu.edu.tw/admin/is/index.php/tw/112070901
- **文章簡介**: 一、依據行政院推動ODF-CNS15251為政府文件標準格式計畫，為推廣及鼓勵大專校院教師投入ODF文件格式推動事宜，辦理大專校院ODF-CNS15251競賽。
二、競賽內容重點摘要如下：
(一)報名資格：大專校院專、兼任教師。
(二)報名及佐證資料上傳期間：即日起至112年7月9日(星期日)止。
- **文章全文**: (三)評審方式：分初賽及決賽，初賽以參賽教師繳交資料進行評審；決賽以參賽教師現場簡報進行評審。參與競賽之每位教師，經評審委員檢視ODF成果佐證資料，如符合資格且具相關成效者，由承辦單位提供參賽證明。
(四)獎項：
１、第一名(1名)：獎金新臺幣(以下同)50,000元及教育部獎狀乙紙。
２、第二名(2名)：獎金30,000元及教育部獎狀乙紙。
３、第三名(4名)：獎金10,000元及教育部獎狀乙紙。
４、佳作(5名)：教育部獎狀乙紙。
５、影音特別獎(1名)：獎金5,000元及教育部獎狀乙紙。
６、範本特別獎(1名)：獎金3,000元及教育部獎狀乙紙。
三、本競賽各獎項名額得依報名情形及評審結果增減之。
四、如有競賽辦理相關問題，請逕洽本案聯絡窗口：社團法人中華民國大專校院資訊服務協會林小姐(電話：(02)2577-4249#839、Email：isac_liz@mail.isac.org.tw)。
