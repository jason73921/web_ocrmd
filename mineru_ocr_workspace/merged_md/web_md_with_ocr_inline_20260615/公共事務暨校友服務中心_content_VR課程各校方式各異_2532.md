# 網站文章資料_VR課程各校方式各異_2532

## 一、文章基本資料

- **文章ID**: 2532
- **文章標題**: VR課程各校方式各異
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20180112
- **文章簡介**: 【銘報】

目前在傳播相關科系開設虛擬實境(VR)課程的學校有政治大學、銘傳大學及文化大學，但各自開設的課程方式都不同，有產學合作、開設專門實驗室和微學分課程。

文化大學資傳系系主任柯舜智表示，開設學分課程需要先向教育部提出申請，再經過審核才能開課，後續的採購和師資也需要時間籌畫，這樣大致需要花上5年的時間，所以文化大學資訊傳播系採用微學分課程的方式，請目前在業界的老師授課，讓學生可以在10小時裡學到VR基礎的技術。

而元智大學資傳系助理教授黃怡錚則認為，VR的技術其實對學生來說是容易學習的，所以元智大學資傳系選擇開設AR相關課程，但仍舊鼓勵學生主動去學習更多新的技術。(待續)
