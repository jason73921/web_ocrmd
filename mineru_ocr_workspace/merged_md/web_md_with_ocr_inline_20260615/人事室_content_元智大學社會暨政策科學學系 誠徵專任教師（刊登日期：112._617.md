# 網站文章資料_元智大學社會暨政策科學學系 誠徵專任教師（刊登日期：112.3.28)_617

## 一、文章基本資料

- **文章ID**: 617
- **文章標題**: 元智大學社會暨政策科學學系 誠徵專任教師（刊登日期：112.3.28)
- **文章網址**: https://www.yzu.edu.tw/admin/ho/index.php/tw/kan-deng-ri-qi-112-3-28
- **文章簡介**: 一、需求職別：助理教授以上之專任師資

二、徵聘名額：壹名

三、起聘日期：自2024年02月01日起聘

四、新資福利：依本校專任教師規定辦理

五、資格與專長：

1. 獲臺灣教育部認可之社會科學或政策科學相關領域博士學位之臺灣或外籍學者。

2.   優先考慮可教授社會學、社會工作、社會政策，與公共政策相關課程者，具英語授課能力者尤佳。

3. 具高度教學與服務熱忱、團隊精神，與跨領域研究潛力者。

4. 具發表國際學術論文能力者。

六、繳件截止日期：即日起至2023年07月31日止，郵戳為憑。

七、檢具文件：

1. 履歷（含(1) 大學與研究所就讀之校系或學程、(2) 參與與/或執行之研究計畫、(3) 相關教學經歷）

2. 自傳

3. 最高學歷證件影本（凡持國外學歷者，請繳交經駐外單位驗證過之畢業證書，尚未取得畢業證書者得以臨時學位證明書代替並註明將畢業日期）

4. 最高學歷成績單影本（凡持國外學歷者，請繳交經駐外單位驗證過之成績單）

5. 教師資格證書影本（未取得者教師證書者得免附）

6. 五年內出版之代表著作、參考學術著作及著作目錄各一份

7. 可開授課程之教學計畫書（詳參本系課程於http://sc.hs.yzu.edu.tw/content.aspx?id=7與http://sc.hs.yzu.edu.tw/content.aspx?id=8）

8.   簡述未來研究方向

9. 推薦信貳封

八、應徵資料寄送注意事項：

1.   應徵資料應於截止日前郵寄至：「32003桃園市中壢區遠東路135 號   元智大學社會暨政策科學學系」，並加註「應徵專任教師」字眼。

2. 恕不接受傳真或電子郵件方式傳遞應徵資料。

3.   應徵資料恕不退件，如需索回，請註記並附上回郵，俾便寄回。

備註：

1. 初選入選者，本系將另行通知前來面試（含英文教學錄影），參與複選，惟旅費自理。

2. 進入複審者需再另行提供「教師資格審查資格履歷表」（請至教育部學審會網站填寫並同時繳交書面及.xml電子檔），國外學歷者另加「國外學歷送審教師資格修業情形一覽表」及「出入境記錄（請洽移民署辦理）」。請參考本校人事室網址： https://www.yzu.edu.tw/admin/ho/index.php/tw/2016-03-17-06-14-27
