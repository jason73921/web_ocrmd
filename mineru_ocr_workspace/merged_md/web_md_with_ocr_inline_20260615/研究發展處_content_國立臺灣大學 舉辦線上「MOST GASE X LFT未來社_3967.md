# 網站文章資料_國立臺灣大學 舉辦線上「MOST GASE X LFT未來社會與領導力培訓營」_3967

## 一、文章基本資料

- **文章ID**: 3967
- **文章標題**: 國立臺灣大學 舉辦線上「MOST GASE X LFT未來社會與領導力培訓營」
- **文章網址**: https://www.yzu.edu.tw/admin/rd/index.php/tw/?option=com_content&view=article&id=3967
- **文章簡介**: 主旨：科技部全球事務與科學發展中心（MOST GASE）謹訂於本 （110）年8月9日（星期一）至13日（星期五）舉辦線上 「MOST GASE X LFT未來社會與領導力培訓營」，敬請惠予協助公告，請查照。

說明：

一、本培訓營由科技部全球事務與科學發展中心（MOST GASE） 與社團法人臺灣領導未來協會（Lead For Taiwan，LFT） 共同合作規劃，挑選「自駕車」、「環境永續」、「區塊 鏈」、「數位科技與學習」等四大主題，透過問題導向學 習法讓青年學子認識尖端科技背後之原理。四大主題同時 涵蓋六大核心戰略產業中「資訊及數位產業」、「資安卓越產業」與「綠能及再生能源產業」等三大面向，在探索主題的過程中，鼓勵青年學子瞭解臺灣科技的卓越發展， 並以此出發解析全球，成為國際科技人才。本培訓營共包含議題探究、工作坊、專題演講及真人圖書館等多項活動。

二、《MOST GASE X LFT未來社會與領導力培訓營》資訊如 下：

(一)時間：本（110）年8月9日（星期ㄧ）至8月13日（星期 五）。

(二)線上會議平台：Cisco Webex（會議網址將於報名錄取後 以電子郵件告知）。

(三)參與對象：全國大學校院學生。

(四)報名方式：即日起至本（110）年7月16日受理報名，報名網址：https://www.surveycake.com/s/kxyWW。

三、如需進一步資訊，請參考活動網站《MOST GASE X LFT未來社會與領導力培訓營（網址：https://gase.most.ntu. edu.tw/events/70）。
