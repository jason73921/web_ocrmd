# 網站文章資料_【亮點新聞】對話機器人黑客松競賽 元智資管系學生勝出得四獎項_684

## 一、文章基本資料

- **文章ID**: 684
- **文章標題**: 【亮點新聞】對話機器人黑客松競賽 元智資管系學生勝出得四獎項
- **文章網址**: https://www.yzu.edu.tw/alumni/index.php/tw/20210930-3
- **文章簡介**: 叡揚資訊偕同台灣微軟日前舉辦暑期Chatbot開發黑客松，邀請大專院校學生參與。此活動透過工作坊實際操作和競賽演練，使參賽者與專家零距離互動，深入瞭解Chatbot原理、機制和多面向的應用，培養專業技能，成為未來職場方向，或求職創業的加分利器。

元智大學資管系大四學生汪文豪、陳宜湞、陳侑辰與鄭淂元，參加「叡揚資訊暑期Chatbot開發黑客松」，脫穎而出獲得「第一名」、「評審青睞獎」、「人氣獎」及「實用獎」共四個獎項，是參賽團隊中唯一獲得四獎項的團隊，成績斐然；另外一組團隊資管系大四呂芷婷、杜旻庭、陳詩婷及大三的呂欣澄、曾謙文以作品「做伙食」獲得亞軍殊榮。

獲得冠軍團隊的鄭淂元表示，團隊開發的聊天機器人「Davis帶位仕」是透過叡揚的iota C.ai對話服務平台與微軟提供的Azure服務為主而開發的聊天機器人，團隊以MVC開發的餐廳管理介面伺服器架設於Azure上，並使用Azure SQL Database進行資料的儲存及處理。鄭淂元說，「Davis帶位仕」將帶位、點餐、結帳等服務一次到位；此機器人發布於Line上，提供內用、預約與外帶服務，透過自動化的餐廳服務取代大部分人力，除了可大幅度的減低餐廳的人事成本，且在大數據分析下，可以更容易蒐集分析顧客點餐狀況與滿意程度。

鄭淂元說，從摸索聊天機器人的基本開發方式，到機器人發布上線並實際操作，在有限的時間壓力下完成目標，成就感非常大。作品雖只包含了三大功能，希望未來還可以增加交談式AI客服問答等功能，將「帶位仕」做得更加完整，達到使用者可以透過Line機器人完成到餐廳用餐的各種面向需求。



(陳宜湞（左）、陳侑辰（右）等人參加黑客松競賽，脫穎而出獲得四獎項/照片由本報提供)

獲得亞軍團隊的資管系呂芷婷表示，作品「做伙食」以食譜為主題，提供使用者方便的操作介面，讓機器人推薦符合各種需求的食譜。呂芷婷說，這個APP的使用方式與一般的對話問答不同，「做伙食」讓使用者可自行輸入冰箱食材，透過AI的斷詞系統，分析使用者的意圖，同時利用同義詞管理，讓使用者能快速的找到適合做的料理。呂芷婷表示，這app針對上班族主婦或是初次做菜者而言，真是一大福音，很多主婦常常冰箱打開，看到一堆食材不知如何料理，此APP的分析，可以一目了然，輕易了解冰箱食材及料理方式，不但省時間也可幫助做菜者彙整食材。呂芷婷說，團隊當初會設計這個對話機器人，其主要目的就是想讓使用者擁有一個最貼近日常生活快速的料理功能，讓對話機器人的運用，走進你我生活當中，相信這個改變可以為我們帶來更便利的生活。



(資管系大四呂芷婷、杜旻庭、陳詩婷及大三的呂欣澄、曾謙文以作品「做伙食」獲得亞軍殊榮/照片由本報提供)
