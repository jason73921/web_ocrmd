# 網站文章資料_培育全球流通人才 創新、卓越 元智大學_781

## 一、文章基本資料

- **文章ID**: 781
- **文章標題**: 培育全球流通人才 創新、卓越 元智大學
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【聯合線上】

元智大學由遠東集團興辦，2014-2015蟬連英國泰晤士報高等教育專刊評等金磚五國及新興經濟體國家前一百名最佳大學，2015 並獲QS世界大學排名中心評等為亞洲前三百名最佳大學。連續榮登國際性大學排行榜，顯示元智之教學研究已臻國際頂尖水準。

甫於日前公佈的英國泰晤士報高等教育特刊2015-2016年全球前800名最佳大學排行榜，台灣有24所大學入列，元智排名第14名，不只在全國私立大學中僅次於4所醫學類大學，更一舉超越多所國立大學，表現再創高峰。104人力銀行針對全台公私立大學30歲以下畢業校友高薪比率，元智名列全國第11名，獲得社會各界之肯定，並再次奪下非醫學類私立大學之冠。

橋接遠東集團 入學即搭上就業直通車 

遠東集團擁有十大事業體，以此背景優勢，元智與集團企業合作「遠東集團產學實習暨人才培育計畫」，學生於在學期間到集團企業實習，並享有獎學金補助，畢業後可直接進入集團企業任職，取得就業保障。

參與此項計畫的集團企業有遠東新世紀、遠東國際大飯店、亞洲水泥、遠東百貨、遠傳電信、遠東商銀等共計40間公司，提供給學生多元及適性的選擇，及早認識並適應職場生態，為生涯發展做出最佳之準備。

全方位提昇學生競爭力　頂尖研究立足國際 
元智大學以「Global Mobility」為定位，積極培養學生的「全球流通」能力。新生入學前即需參加挑戰英語極限營，入學後更有1/4以上專業課程以英語授課，畢業前更舉辦「益勵講堂」與「GMAT」講堂全面提昇國際就業力與卓越競爭力。近年來，元智大學擴大招收外籍生，並持續辦理移地教學，鼓勵學生參與國際交流，積極推展國際志工，培養學生國際視野。

元智「大數據與數位匯流創新」研究中心獲得教育部邁向頂尖大學計畫補助，研究觸角擴及零售通路、交通運輸、能源、電信、傳播、光電及半導體等產業，並與國際名校緊密合作，預期能帶領臺灣成為亞洲大數據研究的重鎮；元智燃料電池中心為國家重點能源發展之重鎮，開創台灣能源產業佈局，並持續協助相關產業進行技術擴散與移轉；而管理學院以「主修」替代「系所」運作，更於2011年獲得AACSB認證，躋身全球不到5%的菁英商學院之列。

師生團隊表現優異 國內外獲獎連連 
元智學生表現優異，於國內外迭獲大獎肯定：電機系團隊於甫落幕的「第二屆半導體大數據分析競賽」榮獲全國亞軍；通訊系團隊於「2015 iWEM R&S Student Competition」榮獲全國亞軍；工管系團隊於「2015阿里百川夢想創業大賽」以「揪飯」點餐App榮獲阿里台灣優勝獎；機械系團隊於「2015海峽兩岸青年大學生實作型創新創業大賽」榮獲一等獎；管院團隊於「2015千里馬尋桃花源創業競賽」榮獲社會企業組冠軍、於2015年「三星電子innoBomb社會創新創業競賽」榮獲社會創新銀獎；管院經管碩士班「CON-NEX」團隊於2015年第十二屆「ATCC全國大專院校商業個案大賽」近200支隊伍中脫穎而出，榮獲亞軍！

資傳系張世明老師帶領之團隊則於2015年「美國國際電影暨電視節」中榮獲評審團大獎、金獎、銀獎等殊榮；2015年「德國紅點設計獎」中獲得傳達設計獎；並兩度獲得法國坎城三大獎項之一的「坎城企業媒體暨電視獎」，在在顯示元智培育學生的研發、創新、創業能力，均以培育未來各行各業的新領袖人才為目標。
