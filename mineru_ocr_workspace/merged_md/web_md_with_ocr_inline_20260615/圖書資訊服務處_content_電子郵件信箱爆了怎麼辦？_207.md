# 網站文章資料_電子郵件信箱爆了怎麼辦？_207

## 一、文章基本資料

- **文章ID**: 207
- **文章標題**: 電子郵件信箱爆了怎麼辦？
- **文章網址**: https://www.yzu.edu.tw/admin/is/index.php/tw/2014-09-12-02-17-36
- **文章簡介**: 
- **文章全文**: 如果你的E-Mail信箱收到了由系統管理者寄來的警告信，警告你的信箱大小已經超過限制的話，那表示你的信箱已經爆了，將無法收信及寄信，這時候可以用以下的方式來解決：

一、利用Outlook 等收信軟體將信收到您自己的電腦上

      Outlook 2003:

      ->工具

      ->電子郵件帳號

     ->檢視或變更現有的電子郵件帳號

     ->下一步

     ->變更

     ->其他設定

     ->進階

     ->不要勾選“在伺服器上保留一份郵件複本”

     ->按下“確定”按鈕-->再按“下一步”按鈕->再按“完成”按鈕

     ->再按下“傳送與接收”按鈕，你的信件就會收下來了。

 

     Outlook 2007:

    ->檔案

    ->帳戶設定

    ->變更

    ->其他設定

    ->進階

    ->不要勾選“在伺服器上保留一份郵件複本”

    ->按下“確定”按鈕-->再按“下一步”按鈕->再按“完成”按鈕

    ->再按下“傳送與接收”按鈕，你的信件就會收下來了。

二、利用瀏覽器至WebMail刪除郵件到安全值底下自動恢復正常


請同學們平常要注意“刪除的郵件”及“寄件備份”及“自行新增子資料夾”裡面的郵件，

通常同學們都會忘記清理這些資料夾的內容，以致於信箱空間就這樣被吃光。

若信箱已經爆了，剩餘信件大小在安全值底下自動恢復正常。
