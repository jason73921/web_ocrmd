- **文章簡介**: 🗓 申請及收件日期

申請期間：114年9月8日(一) 至 10月1日(三) 下午4:30 補助名單公告：115年1月 注意事項：逾期申請者依規定不予受理，一學年申請一次

🎓 申請條件


申請資格：中華民國國籍，且就讀國內大專校院（不含延修生及研究所在職專班），修業年限內的學生。

家庭經濟條件：家庭年所得 ≤ 70萬元（學士班 ≤ 90萬元）、家庭不動產 ≤ 650萬元、家庭利息所得 ≤ 2萬元。

學業要求：前一學期平均成績 ≥ 60分（新生及轉學生除外）。 備註：未婚學生未成年與父母/監護人合計；已成年與父母或未成年時監護人合計；已婚學生與配偶合計；離婚或配偶死亡者以本人所得計算。

補助範圍：學費、雜費、學分費等（不包含延長修業年、重修、補修等費用）。

同類助學措施不得重複申請：已申請教育部及其他政府助學措施者，不得再申請本計畫。


📄 應繳資料

請於 114年10月1日前 送交至 活動中心2樓 生輔組8203室


申請表：學校Portal下載 → 詳填 → 列印簽名 申請連結

3個月內戶籍謄本正本 或 新式戶口名簿影本（含父母或監護人及學生本人）

前一學期成績單（113-2）（新生與轉學生免附）
