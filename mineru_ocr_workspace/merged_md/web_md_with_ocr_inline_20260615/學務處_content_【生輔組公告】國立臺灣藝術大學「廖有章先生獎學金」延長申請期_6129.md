# 網站文章資料_【生輔組公告】國立臺灣藝術大學「廖有章先生獎學金」延長申請期限至114年9月20日_6129

## 一、文章基本資料

- **文章ID**: 6129
- **文章標題**: 【生輔組公告】國立臺灣藝術大學「廖有章先生獎學金」延長申請期限至114年9月20日
- **文章網址**: 
- **文章簡介**: 一、申請項目及資格：

(一)急難救助（60萬元/年）：申請對象為各大專院校在學學生；為經濟狀況困難的學生提供資助，減輕其學費及生活費壓力，需提交經濟狀況說明或相關證明。

(二)藝術表現優異學生（40萬元/年）：申請對象為各大專院校在學學生；鼓勵藝術表現卓越的學生，獎勵其優異的藝術成績成就；繳交老師推薦信及其他證明文件。

(三)國際與兩岸交流（40萬元/年）：申請對象為各大專院校在學學生；促進國際或兩岸大學生的交流與合作，提供赴國際或兩岸學習、研究的機會；提交交流計畫書、交流邀請函或其他相關文件。

二、申請期程：

(一)延長申請期限至114年9月20日。

(二)急難救助之申請得隨時為之，不受上述申請期限之限制。

三、申請方式：採紙本申請，請依格式填具資料後擲寄該校，並註明：申請「廖有章先生獎學金」。
