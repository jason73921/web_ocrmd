# 網站文章資料_全產總戴國榮：年輕人紛出走台灣快變老人島_3254

## 一、文章基本資料

- **文章ID**: 3254
- **文章標題**: 全產總戴國榮：年輕人紛出走台灣快變老人島
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20181022-2
- **文章簡介**: 【中國評論新聞網/記者楊騰凱】

全國產業總工會秘書長戴國榮接受中評社專訪時表示，大陸惠台31條、福建惠台66條等對台灣人才產生的吸引力道，超乎我們的想像。台灣的教授去大陸教書，薪資都高於台灣的2倍到3倍，當年輕人、社會菁英紛紛離開台灣到大陸，台灣還剩下什麼？恐怕會被掏空變成“老人島”。

戴國榮為元智大學管理研究所碩士，出身公營事業，曾任台灣石油工會秘書長，現為全產總秘書長。全產總由30個工會組成，包括台灣電力工會、台灣石油工會、台灣自來水公司企業工會、中華電信工會、中華郵政工會、中華航空企業工會、台灣銀行企業工會等，旗下工會會員數有30萬人。

戴國榮向中評社表示，現在台灣面對兩大問題，首先是內部整體的勞動市場、勞動環境，還沒辦法有效的改善，不僅吸引不了國外人才，連本地人才都開始流失，另外就是投資環境還沒有改善，未創造更好的就業機會。（待續）
