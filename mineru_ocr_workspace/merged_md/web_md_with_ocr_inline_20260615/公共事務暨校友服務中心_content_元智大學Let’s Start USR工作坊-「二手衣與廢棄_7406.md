元智大學管理學院112年獲得教育部大學社會責任萌芽型計畫，計畫核心 團隊由計畫主持人黃瓊億教授、共同主持人陳志萍教授、曾詠青教授及協同主持人黃敏萍教授共同合作，計畫聚焦「二手衣與廢棄衣物」後端的綠色循環經濟可能性，並與遠東新世紀研發中心簽訂長期合作備忘錄，遠東新世紀研發中心提供循環技術與知識業師，整合管院商業模式立基，共同將現今衣服垃圾議題帶入課程，並從校園及內壢里作為實踐場域，形成教師、業師、里民與學生共學共解的培育系統。
 
2023年原由元智大學校內USR-Hub支持在校內舉辦第一次高中生「Let’s Start USR工作坊：創意x創新 x 創業」USR兩天營隊工作坊，獲得初步好評，2024年本計畫支持舉辦第二次營隊，開始將「二手衣」議題導入課程，今年有來自11所高中學生參加，主要九成來自桃園在地高中，共30位學生為期兩天的小型黑客松模式，進行團隊解題合作。（待續）
