# 網站文章資料_【原資中心公告】桃園市原住民族學生獎助—清寒獎助金及優秀獎學金（3/1-3/31）_4558

## 一、文章基本資料

- **文章ID**: 4558
- **文章標題**: 【原資中心公告】桃園市原住民族學生獎助—清寒獎助金及優秀獎學金（3/1-3/31）
- **文章網址**: 
- **文章簡介**: 一、申請資格

獎助對象為設籍四個月以上，就讀教育部核准立案之國內公私立國民小學至大專校院原住民族學生。但不含各級學校一年級第一學期學生、各級學校進修部、推廣進修班、建教班、在職專班、進修補習學校、遠距教學、學分班、在職進修班、延長修業年限及就讀第三年起之研究所學生。

二、申請期限及事項

（一）申請期限：113年3月1日至3月31日止。

（二）申請方式：學生填寫申請表單，並於4月1日前將紙本送至原資中心（活動中心二樓R8203）

（三）申請資訊：可至「桃園市政府原住民族行政局＞業務資訊＞教育＞桃園市原住民族學生獎助學金」查詢。

三、獎助學金不得與「原住民族委員會獎助大專校院原住民學生實施要點」及同性質補助要點或計劃重複申請。

四、相關詳細資訊及申請表單電子檔請至「桃園市政府原住民族行政局＞訊息公告＞最新消息」下載。
