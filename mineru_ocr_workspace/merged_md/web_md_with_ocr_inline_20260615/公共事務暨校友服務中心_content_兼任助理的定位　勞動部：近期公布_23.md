# 網站文章資料_兼任助理的定位　勞動部：近期公布_23

## 一、文章基本資料

- **文章ID**: 23
- **文章標題**: 兼任助理的定位　勞動部：近期公布
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【蘋果日報/記者許敏溶/台北報導】 


大學生兼任助理和大學之間到底是學習或僱傭關係，至今爭議不休，在今早一場座談會上，勞動部指出，近期將公布明確的定義，但各大學及與會團體代表也擔心，若無簡單明確的規定，未來爭議仍會持續發生。 


師大學務處今舉辦「從校園文化談學習與雇傭關係」座談會，邀請教育部、勞動部、台灣高等教育產業工會與師生代表進行對談，勞動部專委黃琦雅指出，大學生兼任助理，到底是僱傭或學習關係，紛擾至今超過2年，過去的確沒有法令保障學生兼任助理這一群人，勞動部最近將公布明確的定義，並會舉行說明會，讓各大學了解。 


黃琦雅也提醒大學，若新規定實施後被認定為雇傭關係的研究或教學助理，未來因為校方經費不足而想縮減學生薪水，此將會違法，只能在未來聘用的人員薪資減少。 


對於勞動部將公布的新規定，大學方擔心沒有明確定義，爭議恐怕不會減少，元智大學生活輔導組長王越指出，希望新規定不要讓學校有模糊空間，一旦解釋不夠清楚，學校沒人敢碰，建議有勞僱關係的檢查表，否則校方擔心觸法，恐將會縮減相關費用，例如原補助弱勢生的獎助學金，不少人是請學生到某處去看書，並沒有什麼勞動，一旦認定是僱傭關係，影響的恐怕是學生。 


台大學務長陳聰富也認為，沒有一個公務員願意違法，希望勞動部要有明確的定義，但勞動部又說要提出明確的定義又很困難，最好作法是學校要跟學生多多溝通，而對於給弱勢生的工讀金，他坦承為了不要讓學生認為拿錢的是窮人，很多部分是請學生去看書，「這是事實」，一旦認定這是勞雇關係，恐怕往後這種幫助弱勢的工讀金會取消。 


高教工會代表則表示，一旦新規定導致大學的負擔增加，希望政府能夠編列預算幫助大學，但教育部高教司科長王淑娟坦言，這是不可能。
