# 網站文章資料_111年指考刪3主科 藝術系擬加採歷史、商管採公民_4516

## 一、文章基本資料

- **文章ID**: 4516
- **文章標題**: 111年指考刪3主科 藝術系擬加採歷史、商管採公民
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20200817-2
- **文章簡介**: 【聯合報/記者潘乃欣/台北即時報導】

111學年大學指考改為分科測驗，刪除國文、英文、數乙，但全國近400系目前只採這三科，以藝術和商管學群為多。家長憂音樂、美術等藝術學系無考科可採，若因此停收指考生，將衝擊科班生升學權益。大學主管表示，改制後確實面臨無合適考科可採的情形，藝術學群擬加採歷史，商管科系擬加採公民來因應。

111學年分科測驗科目只剩數甲、物理、化學、生物、歷史、地理、公民七科，各系若要收指考生，可採計學測成績至多四科，及分科測驗成績至少一科。按大考中心規畫，分科測驗不考國文、英文，是因學測範圍已是高中所教內容，無須再考一次。刪除數乙則是為減少考科來減輕學生升學負擔。

邱莉倩呼籲111學年指考仍考國文、英文，或要設立音樂系考生的配套措施。他表示音樂班是資優教育的一環，應考量資優生的性向和優勢能力，以特殊教育方案來辦理。

商管學系未來也將面臨無考科可採的困境，元智大學教務長、管理學院教授曾芳美說，管院的財金、資管系尤其重視數學與邏輯，往年都採國文、英文、數乙三科，指考改制後失去合適的對應科目。據了解，多數商管學系擬加採公民，但未到提交資料時間，各系仍在討論採計科目。（待續）
