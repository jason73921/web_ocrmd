- **文章簡介**: 智慧生產與管理創新研究中心

工業4.0是全球先進製造國家下一波的競爭舞台，也是台灣製造業發展的新契機。元智大學成立智慧生產與管理創新研究中心，針對遠東集團與台灣地區的重要產業，進行開發與合作。目前規劃先以PCB、紡織、物流、IOT物聯網、智慧管理與製造、綠色能源與生醫等產業為先行產業，藉此以帶動遠東集團與台灣地區的重點產業升級。

1. 兩大服務平台

        元智大學各教授、學生為本中心主要研發人力並與校外AIOT領域之學者、專家和研究人員組成製程串流服務平台、大數據應用服務平台等之研發團隊，負責對短、中、長程研發與計畫之構想形成、申請作業和計畫執行。其中，製程串流服務為協助廠商智能感測器佈建，進行各種製程及設備狀況採集，然後透過網路傳遞資訊達到人、機器和系統三者之間的智能化、互動式無縫連接。大數據應用服務為建置雲端伺服器，可存放各種採集之數據，協助廠商經由各種智慧技術分析，整合企業各種資源，使產線自我調整達較優之狀況，改善生產與管理效率。

2. 中心核心技術

        本中心核心技術包含1)製造現場iOT數據蒐集與後製處理服務(與泓格科技合作)，2) AI cloud server架設與運行(與Cisco與Levono合作)，以及3) AI智慧預測模型開發等三大項。 中心目前與泓格科技合作，建構物聯網數據蒐集系統，從執行智能產業應用計畫中，累積了第一手的物聯網數據蒐集與後製處理服務的核心技術，針對桃園在地各種不同產業 (製造業、醫療、農業、物流等)，提供物聯網感測器與數據蒐集硬體架構之架設，運作與維護的相關經驗，以及數據檔案的後製，儲存，管理等軟體系統操作的協助。中心的第2項核心技術是「智慧雲端系統」，中心目前與Cisco合作建構桃園地區第一個智慧雲端系統(AI cloud server)，未來可提供學界與產業界AI cloud server 的架設，運作與維護相關經驗與諮詢。 中心的第3項核心技術是「智能化數據分析技術」，包含各類型的智慧預測人工智慧模型(CNN, RNN…等)的架構與參數調整，利用IOT物聯網蒐集的數據，進行各種智能化預測，如製造設備維修預測，高齡跌倒風險預測，農作物收成預測等。

3. 智慧應用

3.1 大數據應用服務平台技術結合IOT數據蒐集技術建構智慧製造環境

在智慧製造領域，中心將以大數據應用服務平台技術結合IOT硬體設備及智慧感測器布建，蒐集製造現場的產線及廠房環境監控，庫房監控，生產管理監控等數據，上傳至雲端伺服器經大數據應用服務平台技術智慧分析建立1. 可自主調整廠區與產線之產能配置。2.可自主調整上下游供應配送。3.可自主優化生產環境之資源與能源配置4.可輔助人員正確完成各種操作與組裝測試。5.可即時逆向追蹤生產進程與履歷之智慧製造環境。

3.2 智慧健康照護

    在智慧健康照護方面，中心目前以各種感測器蒐集之數據來讓健康照護人員即時瞭解高齡身心理健康狀況。針對此一新興的科技趨勢，本中心整合現有的資訊、人因工程、以及智慧數據分析等技術，開發動作數據蒐集平台，包含微機電系統慣性偵測器(MEMS inertial sensor)，數據蒐集APP，以及雲端數據儲存平台，並建立數據蒐集的流程，例如感測器的擺放位置，方向，影響數據量測的各種因素等，以分析從感測器蒐集的大量數據,發現高齡者趨勢與差異,針對不同使用經驗的高齡者進行不同關懷，溝通與調整。
