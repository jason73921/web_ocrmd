n   活動名稱：110學年度邁向研究卓越研習營(Ⅵ)學術寫作：寫好論文不需抄襲線上課程。

n   活動時間：111.05.25(星期三)下午13:00-14:30。

n   講者：國立陽明交通大學教育研究所終身講座教授周倩博士

n   課程簡介：

n   寫報告好麻煩，上網存取資料和圖片，複製貼上很方便?

n   反正文獻回顧、研究方法內容大同小異，直接把別人學位論文整理過的資料拿來用就好了?

n   畢業在即但論文還沒寫完，乾脆拆解不同文章的段落再拼拼湊湊比較省事?

「誠實」是學術誠信的基本價值，在寫作的過程中，無論是課堂作業、專題研究或畢業論文，如有參考他人的論述都必須和說明資料的來源，並善用寫作的技巧，使人清楚區分自己與他人創作的差異。貪圖方便複製貼上的行為不但構成抄襲，個人的研究能力與成果都會受到質疑，得不償失!

為持續營造校園良善的研究風氣，研發處特別邀請國立陽明交通大學教育研究終身講座教授周倩博士蒞臨本校演講。周教授長年致力於資訊素養與倫理、學術研究倫理數位課程之設計、製作與發展，並為教育部「校園學術倫理教育與機制發展計畫」推動案主持人，開發「臺灣學術倫理教育資源中心」線上學習平台，針對不同背景之學習者（大學生、碩博生、研究人員）設計適合之訓練課程。相信藉由周教授豐富的教學與研究經驗，可協助研究者建立正確的學術寫作觀念，歡迎報名參加!!
