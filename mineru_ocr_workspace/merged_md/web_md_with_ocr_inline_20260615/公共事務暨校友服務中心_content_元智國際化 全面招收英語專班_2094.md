# 網站文章資料_元智國際化 全面招收英語專班_2094

## 一、文章基本資料

- **文章ID**: 2094
- **文章標題**: 元智國際化 全面招收英語專班
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20170727
- **文章簡介**: 【 Yahoo奇摩新聞】

元智大學積極朝雙語大學邁進，今年起全面招收英語專班（學士英語學程），每學院招收30至40名，將結合企業產學合作學程與實習，全面提升國際移動力，培育產業優秀人才。

元智大學有來自50個國家的學生，姊妹校更達2百多所。該校近年致力於雙聯學位、海外移地教學、出國交換及海外企業實習等計畫，國際化程度躍升，曾獲雜誌評比該校跨國學位合作校數居全國第1，整體國際化程度躍居全國前6名。

校長吳志揚指出，元智為全國首創雙語大學，雖然校齡年輕，學生數也不多，但國際化程度卻是在1百多所大學中居前，也是入榜校齡最年輕的學校。
吳志揚認為，英語不代表國際化，但國際化少不了英語，尤其政府推動新南向政策，東南亞各國語言非常多元，星 、馬、菲、泰、印等國家，在品牌、商標或宣傳、標示，還是以英文為主，落實英語學習才是最基本。(待續)
