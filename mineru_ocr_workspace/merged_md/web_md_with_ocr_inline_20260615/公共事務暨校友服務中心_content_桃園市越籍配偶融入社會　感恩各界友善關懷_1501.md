# 網站文章資料_桃園市越籍配偶融入社會　感恩各界友善關懷_1501

## 一、文章基本資料

- **文章ID**: 1501
- **文章標題**: 桃園市越籍配偶融入社會　感恩各界友善關懷
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【ET today東森新聞】

桃園市的外配人數約5萬多人，其中，越籍配偶佔外配人數4分之一，約有萬餘人，3日於中壢區中正公園，舉辦「105年優質新臺灣媳婦暨學習楷模」表揚活動，18位受表揚者中，即有賴玉荷、阮蓓詩、林桂金、陳玉白、羅氏成等5位越籍配偶，分別入選優質媳婦與學習楷模，接受各界表揚，深受注目。

5位越籍配偶共同認為，桃園市政府和民間對新移民很友善，讓她們實實在在的感受到被關懷和幫助，內心充滿無限感激。

入選優質媳婦的賴玉荷與阮蓓詩，居住桃園已多年，平日除了照顧家庭與小孩，讓先生專心為工作打拚，兩人閒暇之餘也積極參與一些新移民姐妹的各項活動，同時從未放棄學習，通過努力，考得了美甲的證照。其中，阮蓓詩目前在家中開設一家美容美甲的小店面，提供給左鄰右舍、新移民姐妹及鄉親們一些美美的服務。 


林桂金、陳玉白、羅氏成三人，獲得新移民女性學習楷模。現年40歲的林桂金，87年因結婚來台灣後，感受到台灣的教育水準很高，在先生的同意與鼓勵下，先後至「華僑大學先修班」修讀課程，結業後又到「元智大學」學習，於98年畢業於「元智大學」企業管理學系。目前，林桂金積極參與一些新移民的教育培訓課程，持續不斷的充實自我。

陳玉白於83年起，在群賀公司工作，至今已經12年了，與同事相處融洽。羅氏成因結婚來台12年，平時照顧現今已95高齡的婆婆，目前從事為外勞公司當翻譯，來台10多年中，從來沒有間斷過學習，於學校得過無數的成績優異奬，也積極參加公益活動，曾獲得怡仁愛心基金會頒發的「不一樣的媳婦、一樣的愛」活動感謝狀。
