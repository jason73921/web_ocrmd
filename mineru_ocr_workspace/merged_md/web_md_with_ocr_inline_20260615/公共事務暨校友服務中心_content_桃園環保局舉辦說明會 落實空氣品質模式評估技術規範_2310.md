# 網站文章資料_桃園環保局舉辦說明會 落實空氣品質模式評估技術規範_2310

## 一、文章基本資料

- **文章ID**: 2310
- **文章標題**: 桃園環保局舉辦說明會 落實空氣品質模式評估技術規範
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20171027
- **文章簡介**: 【HINET新聞/記者黃駿騏/桃園報導】

為了使桃園市的環評相關開發案，於辦理環境影響評估作業時，能確實依循空氣品質模式的評估技術規範，以降低對空氣的負面影響，桃園市環保局26日於桃園市勞工育樂中心辦理「空氣品質模式評估技術規範說明會」，邀請轄內相關列管事業單位和顧問公司參加，以增進業者和開發單位對空品模式評估的瞭解。

此次說明會由元智大學機械系江右君教授擔任講座，分享技術規範的內容與案例，並針對業者常見的闕漏提出具體建議。環保局指出，本技術規範是依據開發行為環境影響評估作業準則第49條，及行政院環保署公告的「空氣品質模式評估技術規範」來辦理。在進行環境影響評估作業之空氣品質模式之使用時，應考量模擬區域的氣象及地形特性、開發行為的特性、模式的限制條件等因素；且開發地區同時包括二種地形以上者，經審查認定應進行第二階段環境影響評估時，應至少提出二種模式加以比對、分析，並依範疇界定會議討論確認辦理。(待續)
