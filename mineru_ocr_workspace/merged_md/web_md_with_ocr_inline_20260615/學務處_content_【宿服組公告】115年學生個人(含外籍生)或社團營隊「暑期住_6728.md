# 網站文章資料_【宿服組公告】115年學生個人(含外籍生)或社團營隊「暑期住宿申請」作業事項_6728

## 一、文章基本資料

- **文章ID**: 6728
- **文章標題**: 【宿服組公告】115年學生個人(含外籍生)或社團營隊「暑期住宿申請」作業事項
- **文章網址**: https://www.yzu.edu.tw/admin/st/index.php/tw/?option=com_content&view=article&id=6728
- **文章簡介**: 說明： 

一、申請條件：

（一）個人申請：需為暑期上課、研究、實習、實驗或校內工讀等事由方可提出，且只接受全期住宿。〈境外生有短期特殊需求者除外〉

注意：碩博生住宿費已含寒暑假費用，如下學期無住宿者，本學期住宿截止日為7月31日。

（二）團隊申請：校隊、社團或校內各單位短期之活動需暑宿者，可由主辦單位負責同仁或同學統一至宿服組辦理申請。

二、申請時間：115年5月11日(一)起至5月14日(四)16:00 止。

三、暑假開放住宿期間：

（一）申請住宿期間：115/07/01～115/09/04。
（二）入住及遷出時間：6月30日17：00(入住)及8月27日(遷到115學年度住宿房間)需配合調整房間，如無法配合請勿申請暑期住宿。

注：不受理境外生不連續住宿之申請。

四、申請流程：

➤ 個人申請流程 >>

（一）進個人portal點選「住宿申請」選項，並依指示完成申請程序。

（二）網路申請完成後將申請表單印出簽名後，送至宿服組審核。

（三）持申請表至總務處或財管組繳費。

（四）5月14日16:00前將蓋妥繳費章之申請表送回宿服組，由宿服組統一安排床位，繳費收據請妥為保存，以備查詢。

➤ 團體申請流程 >>

（一）為配合退宿期程及新生活動，不接受7月1日(三)前與8/14(五)以後之住宿申請。

（二）主辦單位要指定一位元智同學進個人portal點選「住宿申請」申請寒假住宿，以便輸入繳款費用。

（三）請逕自下載「元智大學學生宿舍寒暑假社團營隊住宿申請表」並填寫完成後檢附住宿名單（連結）送交宿服組。

（四）將蓋妥繳費章之申請表送回宿服組，由宿服組統一安排床位。

五、開放宿舍：第一、二宿舍（如暑期宿舍有重大工程施作，暑期住宿區將配合調整）。

六、暑期住宿費用：〈中途退宿概不退費，請審慎考慮後再提出申請〉

➤ 住宿費：第一、二宿舍 150元／天。

➤ 清潔費：個人申請75元／人（境外生以住宿原床位為原則除外）；團隊申請300元／間。

註：熱水供應／人數少以電熱水器為主（24hr.），反之熱水以鍋爐供應為主（19:00~22:00）。
