# 網站文章資料_台鐵票價政治化凍漲　專家：安全是需要成本的_3266

## 一、文章基本資料

- **文章ID**: 3266
- **文章標題**: 台鐵票價政治化凍漲　專家：安全是需要成本的
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20181023-9
- **文章簡介**: 【TVBS/編輯吳秉嵩報導】

台鐵普悠瑪號出軌意外造成18人死亡，行政院長賴清德勘災時甚至表示「不排除停駛」，並將成立跨部會小組調查。元智大學工業工程管理學教授，同時也是鐵道暨國土規劃學會副理事長任恆毅呼籲，應該設立常態性的「運輸安全委員會」，並檢討台鐵票價凍漲機制，因為「安全是有價且需要成本的」。

對於飛機失事意外，國內有行政院轄下的三級獨立機關「飛航安全調查委員會」負責，任恆毅認為，應該仿照此制度，設立常態性的「運輸安全委員會」，處理各種陸上運輸的事件；若交由台鐵局內部負責，會有球員兼裁判的疑慮。

對於賴揆提到的跨部會專案小組，他則持保留態度。任恆毅表示贊同政府積極處理的心態，但對於公務員來說，跨部會組織就等於是兼職，會有心態上的問題，為了要顧及原本正職，不容易全心全意付出。他認為若要讓「降災」、「減災」的運輸安全持續，必須有常態性部門來運行。(待續）
