# 網站文章資料_大學推「新雙語」 程式語言成顯學_4436

## 一、文章基本資料

- **文章ID**: 4436
- **文章標題**: 大學推「新雙語」 程式語言成顯學
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/2020062905
- **文章簡介**: 【聯合報/記者章凱閎/台北報導】

 

資訊時代來臨，各大學近年廣開資訊及程式設計相關課程，例如元智大學「新雙語政策」，把基礎程式設計課程列為大學部必修；輔仁大學和師培大學如台北教育大學，也把程式設計融入現有專業課程中。教育部統計，各大學修過程式課的學生比率，兩年來從三成一提高至六成。

且根據教育部一○八學年下學期的最新統計資料，非資訊領域學生中，也有五成三修過程式設計相關課程，人數首次突破四十萬人，顯示運算思維或程式設計已成為不分文理組、不分領域的共同素養。

為提升我國「資訊國力」，教育部從二○一八年委託政大等十三所學校成立「推動大學程式設計教學計畫」，協助各大專校院達到半數以上學生修讀過程式設計相關課程的目標。計畫執行至今已超過二年，成果比預期還好。

例如元智大學實施「新雙語政策」，意即把「程式語言」當作第二語言，基礎程式設計課程也列為大學部必修課。東海大學也發展ＰＢＬ議題導向的課程，開發輔助學習程式設計的流程圖工具，輔仁大學現有的專業課程，也融入程式設計概念，達到跨域學習目的。（待續）
