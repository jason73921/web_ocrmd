# 網站文章資料_校園創意提案競賽　展現了同學們的創意及活力_2894

## 一、文章基本資料

- **文章ID**: 2894
- **文章標題**: 校園創意提案競賽　展現了同學們的創意及活力
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20180525-1
- **文章簡介**: 【銘報即時新聞/記者/王書馨】

由TAA主辦的校園創意提案競賽19日在集思北科大會議中心舉行，由「YACHHRISOO」拿下媒體創意特別獎及第一名。在頒獎前，TIPRA國際公共關係協會總經理唐淑珍代表評審委員發言，她表示，品牌力不等於商品力，商品力不等於購買力，那要如何將三者串聯起來，就要找到這其中的創意點，並且要在有限的預算中做無限的發揮。

她表示，以這個時代的傳播力和影響力，能夠被消費者分享，甚至產生行動是很艱難的，但是每組同學都很努力去做到，同時展現自己的創意。
TAA理事長白友烺表示，校園創意提案競賽已經邁入第16年，每年的競爭都很激烈，經過評審委員的篩選，在將近200組報名隊伍中選出10組來進行決賽。在評審過程中也發現了很多同學的活力以及創意，也為廣告主提供了不錯的創意點。

當天能夠站在台上提報的隊伍是由評審委員層層篩選，對創意、策略、執行、目標等進行評分，他們分別來自世新大學，輔仁大學，元智大學，政治大學以及中正大學。決賽的提案形式不限，限時10分鐘，若超時，評審可斟酌扣分，簡報後，每組有5分鐘被評審提問時間。最後由四位評審對創意、策略、團隊精神及提案技巧進行評分，選出媒體創意特別獎、佳作、優選及前三名，第二名為「我們組」、第三名：「踢被被-TBB」。
