# 網站文章資料_花蓮客運老董憶求學苦　二度捐贈聖約翰科大新校車_2790

## 一、文章基本資料

- **文章ID**: 2790
- **文章標題**: 花蓮客運老董憶求學苦　二度捐贈聖約翰科大新校車
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/2018-04-27-02-05-26
- **文章簡介**: (中央社訊息服務20180423 14:31:50)

「有機會回饋母校，這是我的榮幸」，畢業於新埔工專電子科的花蓮客運公司董事長張譽興因為感念母校的栽培，在橄欖球隊沈文賜校友的引薦下，102年間捐贈一部豪華遊覽車給母校--聖約翰科技大學，做為教職員工的大校車。今年張譽興再度捐贈第2部遊覽車給學校，從事交通業的他認為「安全第一」是首要之事，希望這部大校車能夠讓教職員工的乘車安全更有保障。贈車儀式於107年4月20日舉行，張譽興專程從花蓮北上參加，並參訪母校降臨堂、長齡生命關懷中心及數位藝文中心，更重返風雨球場與學校橄欖球隊打球，重溫求學時在泥濘中打滾、拚戰的青春歲月。花蓮客運公司董事長張譽興表示，民國59年至64年他在聖約翰科大前身新埔工專唸書，那時候坐落於淡水灰窯子的新埔工專還是一片鳥不生蛋的荒野之地，到處都是塵土飛揚的黃土，宿舍還是教室改裝而成，36個同學擠在一間大教室裡。那時候蘇花公路還沒有鋪設柏油，住在花蓮的他每次往返至少得花8-10小時，一年只能回家兩次，因此假日都待在學校裡打橄欖球。那時候求學非常辛苦，雖然環境惡劣，生活單調，但他很感恩曾經走過那段青春歲月，因為他從母校學習到敢衝、敢撞、不怕苦、不怕難的橄欖球精神，讓他一生受用無窮。擔任過海鷗橄欖球隊第三任隊長的張譽興回憶說，當年操場坑坑洞洞、凹凸不平，一群學生就在這樣惡劣刻苦的環境裡打橄欖球、讀書學習，這樣的環境培養出學生耐摔、不怕痛、勇往直前的鬥牛精神，更凝聚深厚的同窗情誼，他認為這就是所謂的「新埔精神」。而且昔日新埔橄欖球隊戰無不勝、攻無不克，甚至曾在風雨球場上打贏身形180公分以上的美國橄欖球隊，堪稱所向無敵。張譽興說，「凡流淚撒種的，必歡呼收割」、「有機會回饋母校，這是我的榮幸」，期盼聖約翰科大校務發展更加欣欣向榮，持續往國際化的綜合大學邁進。艾和昌校長說，張譽興不僅捐贈母校2部校車，也捐贈「海鷗獎學金」及橄欖球隊發展基金，實質獎勵在社團貢獻及體育表現上具有傑出優異成績的學生，並扶植母校橄欖球隊購置設備及訓練費用。他說，新埔風、新埔雨，造就不平凡的新埔人，新埔精神就是不服輸的精神。校友向心、學生認同是成就一所好的大學的保證，感謝過去新埔學長的努力，為現在學弟妹撐起未來的根基。訊息來源：聖約翰科技大學本文含多媒體檔 (Multimedia files included)：http://www.cna.com.tw/postwrite/Detail/232317.aspx
