# 網站文章資料_【實習招募】國家發展委員會檔案管理局實習招募_8132

## 一、文章基本資料

- **文章ID**: 8132
- **文章標題**: 【實習招募】國家發展委員會檔案管理局實習招募
- **文章網址**: 
- **文章簡介**: 為促進交流合作，多方培育檔案管理相關領域人才，增進大學院校學生瞭解檔案管理實務作業，國家發展委員會檔案管理局提供寒假實習計6 個名額。

實習單元與名額：


國家檔案保存與複製作業/正取2名、備取2名

展場服務/正取2名、備取2名

圖書管理/正取2名、備取2名


實習時間：115年1月12日至115年2月13日

實習時數：70小時

※實習期間本局不支付任何報酬(薪資、交通費、膳食及意外保險等補助)，亦不供宿。

申請期限：自即日起至114年12月12日止免備文逕送本局(以郵戳、傳真或電子郵件寄達日期為憑)，逾期不受理。

申請資料：申請表及歷年在校成績單影本(A4格式)
