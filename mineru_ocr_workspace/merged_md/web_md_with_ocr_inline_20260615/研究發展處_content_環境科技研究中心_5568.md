- **文章簡介**: 本中心創立1990年，主要任務是結合企業與學校機構一同為環境保護及企業發展而努力。設有健全之「水質水量檢驗分析室」、「固體廢棄物毒性分析室」、「土壤污染分析室」、「無菌操作室」及「精密儀器分析室」等，為研究型認證檢驗實驗室。主要提供各種污染及高科技產業超微量污染物檢測及表面分析/精細結構鑑定(ICP/AA、GC/MS/MS、GC/FID/TCD/FTP/ECD、FE-SEM/EDS、HR-TEM、FTIR、XPS、EA、UV-Vis、XRD 等精密貴重儀器項目)優質服務。

榮獲臺灣高等法院檢察署認定之國內極少數司法警察機關調查環境刑事案件採樣檢測鑑定機構之一，並積極從事污染防治與控制相關之研究，協助政府機構、公民營企業執行產品純度、飲用水、水質水量、土壤、地下水、廢棄物、全球電子資訊產品材料污染檢測及奈米級樣品微結構 / 表面超微量分析等各項檢驗服務及技術諮詢，並於100年4月榮獲「教育部校園安全衛生管理系統」示範學校實驗室。

 

服務項目

•  固體廢棄物採樣 / 毒性溶出分析 

•  奈米級樣品微結構特性 / 粒徑分析 

促進企業與學校 / 研究機構合作

•  提供企業所需之技術諮詢 / 資訊服務 

•  提供分析檢驗方法 

•  提供相關環保問題解決方案 

•  供廢棄物資源回收製程改善建議 

增進環境分析品質保證

•  發展開創性檢測分析方法或技術

環境污染處理 / 整治技術研究 

•  擁有專業檢驗團隊 

元智大學環境科技研究中心在本校化材系林錕松教授領導下，實驗室軟、硬體設備具有國家級實驗室的水準，團隊成員為專職檢驗專才及校內教授與校外各領域專業權威人士共同組成提供策略及環保諮詢。

 • 未來展望

進入TAF體系與國際認證接軌，並建立揮發性有機物 (VOCs)、農藥及全球電子資訊產品材料污染檢測之分析能力；及提供環保及污染改善技術諮詢服務，促進官、產、學合作共同完成各種開發計劃之環境監測專案或委託環境檢測案件，並承接環保署、環檢所或業界檢驗技術新方法研發計畫。
