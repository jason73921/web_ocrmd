# 網站文章資料_元智桃市府辦閱讀營 訓練思辨_1102

## 一、文章基本資料

- **文章ID**: 1102
- **文章標題**: 元智桃市府辦閱讀營 訓練思辨
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【中央社/記者卞金峰桃園11日電】 


為鼓勵閱讀，元智大學與桃園市府合作，今起舉辦為期三天的「桃園市暑期經典閱讀營」，課程有校園野菜採摘、漢字審美、唐詩吟詠以及故事闖關遊戲。 

元智大學與桃園市政府日前簽署合作備忘錄，因而合辦「桃園市暑期經典閱讀營」，閱讀營招募對象是桃園市境內國小四、五、六年級學生，名額只限60名。 


課程由元智大學通識教學部主任梁家祺帶領，今天導讀《河馬教授說故事：大自然裡的生命教育》一書。梁家祺透過「校園中的野菜」活動，以實際展現的植物種子，讓學生到元智大學的校園裡摘採，如昭和草、咸豐草、蛇莓等，從認識、瞭解、烹煮到食用，帶領學生上一堂真正能認識身邊周遭可食用植物的知識旅程，這是一堂不一樣的植物課程。 

元智中語系助理教授黃智明，運用「手繪本五格書創作」的方法，教導學生認識饒富趣味的漢字教學法。他選擇導讀的書籍是《拉拉的自然筆記》，黃智明表示，在現今的時代裡，漢字已逐漸由單純的實用性轉向審美性質，成為一種兼具藝術與抽象符號的歷程。他希望透過活動，能夠帶領同學從經典中學習大自然的美景，也透過早期的象形文字，讓同學瞭解漢字書寫的可變化性。 


《唐詩三百首》是由元智通識教學部陳巍仁教授所選定帶領學生認識的經典，他以「人情與天籟─我的唐詩音樂劇」作為主題，並運用詩詞吟唱概論、基本調式教學（宜蘭酒令調、福建流水調、天籟吟調）、情境式唐詩教唱與編劇演出（崔顥〈長干行〉、張籍〈節婦吟〉），讓學生們愛上唐詩、吟詠出唐詩的韻味。 


麥麗蓉教授導讀《魯賓遜漂流記》，並設計出「生存大挑戰」活動，使用闖關遊戲的方法，將《魯賓遜漂流記》書中的核心精神與價值設計於各關遊戲中，讓學生在遊戲中體現失去至重新建構的歷程。 


王怡云教授導讀《愛麗絲夢遊仙境》，運用「愛麗絲大遊行」的活動，將書中角色與內涵設計於桌上遊戲中，引導學生創作，並在遊戲過程中再次經歷《愛麗絲夢遊仙境》的文學趣味。
