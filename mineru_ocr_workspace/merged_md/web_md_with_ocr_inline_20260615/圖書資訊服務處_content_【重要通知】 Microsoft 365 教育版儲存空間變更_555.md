# 網站文章資料_【重要通知】 Microsoft 365 教育版儲存空間變更通知 _555

## 一、文章基本資料

- **文章ID**: 555
- **文章標題**: 【重要通知】 Microsoft 365 教育版儲存空間變更通知 
- **文章網址**: https://www.yzu.edu.tw/admin/is/index.php/tw/news-m365-112121101
- **文章簡介**: 微軟M365 教育版儲存空間變更通知（詳細說明請參考微軟官方公告）：
1.    自113/02/01起，微軟限制每個使用者只有100 GB的OneDrive儲存空間。
2.    自113/08/01起，微軟將大幅調降學校的總儲存空間為100 TB，包含OneDrive、SharePoint（Teams）和Exchange（Email）。
- **文章全文**: 為因應微軟M365 教育版儲存空間變動，請務必配合下列事項：
1. 因微軟只提供學校100 TB的總儲存空間，本處為了有效配置每個使用者每個服務之儲存空間，因此不會提供每個人100GB的OneDrive儲存空間。
2. 若您的OneDrive空間已超過100GB，建議每位教職員及學生盡速自行備份下載，並於113/01/31前刪除雲端上之檔案，以免後續導致帳戶無法使用。
3. 現階段不會主動刪除帳號、檔案及信件，但請務必自行備份相關資料，本處不負責提供備份或保留資料之服務。後續將評估全校使用者數量及可配發之儲存空間，屆時會再公告周知。
4. 如何查詢OneDrive使用空間，請參考：如何查詢M365 OneDrive使用空間.pdf

若有問題請洽網路媒體組聯繫(校內分機 2325）
