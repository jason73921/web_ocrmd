# 網站文章資料_中華民國企管菁英協會講座「美，從一杯茶開始」活動紀實_210

## 一、文章基本資料

- **文章ID**: 210
- **文章標題**: 中華民國企管菁英協會講座「美，從一杯茶開始」活動紀實
- **文章網址**: https://www.yzu.edu.tw/alumni/index.php/tw/
- **文章簡介**: 
- **文章全文**: 元智校友所屬之企管菁英協會99年10月30日於元智大學60304教室邀請到茶學專家金春枝老師舉辦了一場色香味俱全的講座「美，從一杯茶始」。一開始，老師以沉香薰陶每位聽眾，不僅是為了開場做準備，同時也讓聽眾的心神能安定下來，老師也補充到沉香是和茶最搭的香。

        金老師對於茶十分地講究，她堅持自己所販售的茶在每一批出貨前都要經過SGS檢驗合格，並且是沒有任何一項檢驗項目是被勾選、檢驗出來的，即使是在安全範圍內也沒有。雖然檢驗的費用累積起來也是一筆可觀的數目，但是老師認為「做檢驗就是要保護別人和自己的健康」，老師也說，好的茶農是「看天氣、看茶」來做茶的。

        接著，老師開始進行茶學的介紹，她提到蔣勳的「美的覺醒」其中的「眼、耳、鼻、舌、身」套用在茶上是非常貼切地。在茶的領域裡，眼所看見的即為色，也就是湯色、器之美以及空間的美；耳所聽聞的便是聲，也就是煮水聲、大自然聲以及音樂；鼻所呼吸的便是香，也就是茶香和幽蘭香；舌所嚐到的即是味，也就是回甘、兩頰生津及神清氣爽；身所感覺到的即是觸，也就是握杯的細緻、敦厚；另外，老師多提及了「意」，所代表的是享用的方法，老師覺得享用茶的方式應該是要「透過一杯茶的時間好好讓自己的身體放鬆」。

        在提及這六美後，老師也分別補充了一些要點：「好茶」在品茗的同時，應該會有淡淡的蘭花香；茶具的部分，台灣製造的陶瓷茶具的杯口應是觸感圓滑；而高溫煉製後的茶具較無重金屬殘留，分辨是否為高溫煉製的方法即是在杯子沒有裝茶的時候輕敲杯身，若有清脆聲響便是高溫煉製的茶具，使用起來讓人比較放心；老師也提到了觀看茶杯可知製作者的個性，像是老師的朋友所製作出來的茶杯就如作者本身般敦厚；另外還有「杯顯茶色」的道理，像是紅茶就應該用牙白杯、骨瓷杯，如此一來更能顯紅，讓這杯紅茶看來更好喝。

        到底什麼是呈現茶的美感的方式呢，金老師說茶席、沖泡方法以及各類茶沖泡的溫度便是方法。好的茶，除了三峽的碧蘿春(70度C)、桃竹苗的東方美人(85度C)之外，第一泡應該使用100度C的水來沖泡，如此才能淬煉出茶的美，同時也舒展茶葉；而茶具的挑選，也是十分重要，老師提到，她有一個十分喜愛的茶杯，在茶杯裡頭畫了一幅山水畫，讓她在品茗的時感受到「一杯水器，一杯山水」的雅興。

        一杯茶，對金老師來說，是「內聚一杯茶，外放大千世界」。曾經到過許多國家進行茶的授課教學的金春枝老師，她說她在國外進行茶會時，特別嚴厲地要求與會人員在席間必須保持安靜，靜靜地品茗。對老師而言，自己先尊重自身的文化，才能去要求別人來尊重自己的文化。

        最後，老師提到茶與美的關係，便是由生活來連接的，在生活上我們常常會有品茗以及接觸美的機會，老師也套用了蔣勳的話，「美常常是停留在一種感覺的狀態」，美其實是一個剎那間的感覺，品茗的同時，也應該要培養美感，如此才能品出茶之美。
