# 網站文章資料_《金融》深化兩岸金融科技創新，上海商銀辦校園競賽_2396

## 一、文章基本資料

- **文章ID**: 2396
- **文章標題**: 《金融》深化兩岸金融科技創新，上海商銀辦校園競賽
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20171120-7
- **文章簡介**: 【時報記者任珮云台北報導】

由上海商業儲蓄銀行偕同大陸上海銀行、香港上海商業銀行與IBM共同舉辦的第一屆「滬港台金融科技創新Hackathon校園競賽–台灣傳奇賽」，成績揭曉！由來自元智大學與中央大學提出的「金融社群媒體分析平台」奪得第一名殊榮，交通大學的「基於影像深度學習之身分認證及防詐欺ATM」以及政治大學的「iLoan」則分別榮獲第二名與第三名獎項；上海商業儲蓄銀行將贊助機票與住宿費，讓前三名獲獎團隊成員前往上海參加12月2日舉辦的上海菁英賽，使台灣的傳奇隊伍可與來大陸及香港的菁英團隊競賽，角逐兩岸三地金融傳奇殊榮與百萬獎金，深化金融創新服務能量。

此次競賽方式是先在台灣、香港與上海等三個地方，針對「社交金融」、「數位化金融」、「惠普金融」、「區塊鏈技術」、「人工智慧應用」與「物聯網金融」等六大主題進行第一輪競賽，其後，獲得傳奇獎的兩岸三地優勝隊伍將聚集在上海，進行金融傳奇總決賽。(待續)
