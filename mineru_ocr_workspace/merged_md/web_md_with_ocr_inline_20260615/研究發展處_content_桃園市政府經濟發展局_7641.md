說明：

一、本府建置之「桃園城市數據視覺化平台」，係整合市府各局處關鍵性數據開放式資料應用平台，涵蓋交通、環境、 水情及民生經濟（如用水、用電）等多元領域。平台提供直覺化的視覺化操作介面，並以 API 形式開放資料介接， 建構橫向聯繫之資訊整合及資料匯流機制。

二、為加速資料應用創新，市府積極鼓勵產官學界廣泛運用本平台所提供之 API 資源，強化學術研究、政策研析及產業創新應用之能量。期透過開放資料共享機制，促進在地智慧治理與產業加值發展，共同打造具前瞻性與韌性的智慧城市。

三、旨揭平台網址為https://dataview.edb.tycg.gov.tw/，頁面即可下載CSV、PDF、EXCEL格式資料檔案進行運用，另如需API介接，請申請會員(我的E政府)，另本平台介接資料皆無需費用，請多加利用。

四、檢附平台數據介紹簡報；若平台使用過程中有相關問題， 請聯繫本局委辦公司(藍星球資訊股份有限公司)呂珮瑄小姐(電話:02-89787198分機294)。
