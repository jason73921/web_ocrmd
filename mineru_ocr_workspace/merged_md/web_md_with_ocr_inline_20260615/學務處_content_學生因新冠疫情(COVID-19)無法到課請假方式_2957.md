# 網站文章資料_學生因新冠疫情(COVID-19)無法到課請假方式_2957

## 一、文章基本資料

- **文章ID**: 2957
- **文章標題**: 學生因新冠疫情(COVID-19)無法到課請假方式
- **文章網址**: https://www.yzu.edu.tw/admin/st/index.php/tw/?option=com_content&view=article&id=2957
- **文章簡介**: 說明：

一、本校學生若受疫情影響須「確診隔離」「自主防疫」或「居家檢疫」時，應遵照中央流行疫情指揮中心發布相關通報，並請依以下列方式完成請假程序。

     （一）請至學生   個人Portal系統請假（或手機APP），並點選「網路請假」，假種為「病假」。

     （二）請假事由選填如下：

              1. 受疫情影響須「確診隔離」。

              2. 受疫情影響須「自主防疫」。

              3. 受疫情影響須「居家檢疫」。

     （三）檢附證明文件：「嚴重特殊傳染性肺炎之各類通知書」或其他相關證明（不受三日內繳交之規定，證明傳送至請假作業承辦人信箱）。

二、因新冠疫情(COVID-19)無法到課，請假不受缺課扣考、勒令休學規定限制。請同學務必遵守規範，避免受罰。

三、學生請假作業承辦人員（簡麗芬）聯繫方式：

 　  （一） 電話：03-4638800#2240

      （二） E-mail：jean0125@saturn.yzu.edu.tw

四、相關說明:

      （一）針對確診或快篩陽性個案，實施7天居家照護，期滿無症狀可入班 上課。

     （二）針對與確診者(及快篩陽性個案)摘下口罩共同活動15分鐘以上人員，快篩陰性者可上課，如有症狀應儘速就醫。

     （三）針對與確診者(及快篩陽性個案)全程配戴口罩共同活動人員，需進行自我健康監測。

     （四）居家隔離不入校(住宿生不離開隔離宿舍)，自主防疫期間不入班上課。

     （五）111學年度第一學期各級學校開學防疫措施：https://reurl.cc/Qbo8d9
