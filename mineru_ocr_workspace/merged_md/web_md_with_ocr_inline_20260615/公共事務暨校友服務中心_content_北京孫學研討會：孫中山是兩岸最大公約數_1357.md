# 網站文章資料_北京孫學研討會：孫中山是兩岸最大公約數_1357

## 一、文章基本資料

- **文章ID**: 1357
- **文章標題**: 北京孫學研討會：孫中山是兩岸最大公約數
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【中時電子報/記者陳君碩】 


由中國宋慶齡基金會、台灣中山學術文化基金會共同主辦的第四屆「孫中山與宋慶齡」研討會，10月14日在北京宋慶齡故居舉行。中國宋慶齡基金會副主席齊鳴秋在開幕式上致辭表示，孫中山是兩岸共同追溯歷史、面向未來的最大公約數。

中新社報導，此次研討會為紀念孫中山先生誕辰150周年，特別選定主題為「孫中山思想：歷史、現實、未來」。大陸參與的，包括中國社科院近代史研究所、北京大學、北京師範大學、中山大學、河南大學，台灣參與的則有台灣大學、金門大學、元智大學、中國文化大學等高校及研究機構共20多名兩岸學者參加研討。

中國宋慶齡基金會副主席齊鳴秋在開幕式上致辭表示，孫中山和宋慶齡的精神和思想，是兩岸中華兒女共有的精神財富，是兩岸共同追溯歷史、面向未來的最大公約數。在紀念孫中山先生誕辰150周年前夕，兩岸共同舉辦研討會，是對兩位世紀偉人最好的紀念，也是兩岸歷史文化學術的一件盛事。

研討會現場，兩岸40歲以下的青年學者共提交論文10篇，聚焦孫中山的法治、監察、福利、民生等方面的思想，回溯孫中山學醫從醫、北上、赴台以及與廖仲愷家族往來的史實，探討孫中山研究在大陸、台灣的現狀、困境與趨勢。

為推動兩岸孫中山、宋慶齡研究的可持續發展，研討會由兩岸青年學者發表論文，兩岸資深專家當場進行評論。
