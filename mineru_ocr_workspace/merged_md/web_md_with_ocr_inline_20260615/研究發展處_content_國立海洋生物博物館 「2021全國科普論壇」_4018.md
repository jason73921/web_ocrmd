# 網站文章資料_國立海洋生物博物館 「2021全國科普論壇」_4018

## 一、文章基本資料

- **文章ID**: 4018
- **文章標題**: 國立海洋生物博物館 「2021全國科普論壇」
- **文章網址**: https://www.yzu.edu.tw/admin/rd/index.php/tw/?option=com_content&view=article&id=4018
- **文章簡介**: 主旨：有關本館主辦之「2021全國科普論壇」改為線上舉辦之相關事宜，詳如說明，敬請查照。

說明：

一、依教育部110年7月23日臺教社(三)第1100092723號文辦理。

二、本館因應當前新冠肺炎疫情嚴峻且未來發展難以預測，為 避免發生群聚感染之風險，本館將「2021全國科普論壇」 更改為線上舉辦，原擬舉辦之相關活動其中包括會前參訪活動、科學市集與會後展示館參觀活動一併取消。

三、原訂論壇會議日期，將改於110年9月25日(六)當日舉辦， 修改之會議行程，如附件。

惟因發表時間有限，口頭報告時間將會縮減，僅會依評審意見擇優部分投稿者進行口頭 發表，詳細論文發表議程將另行公告於活動官網。

四、本次活動仍將舉辦「館長圓桌會議」，為能夠提升會議交 流之效力，將公開收集各方對於未來科普推廣之發展方向與策略之建議，本館將在110年8月31日之前收齊各方會議，並從中彙整出討論議題於圓桌會議中提請討論。任何 有興趣提供建議者，請將建議議題在期限之前以電子郵件寄到yhchen@nmmba.gov.tw 陳勇輝 先生收，並在主旨上註 明「科普論壇圓桌會議之建議討論議題」。

五、本館與部屬四大館所煩請科普相關業務單位彙整之後，以 館為單位提出1~2議題寄至本館yhchen@nmmba.gov.tw 陳勇輝 先生收，來信請註明承辦窗口聯絡方式，以便會議中進 行討論。

六、2021全國科普論壇活動網站: http://www.jetkey.tw /ScipopForum
