Term Frequency (TF):

$$
\mathrm { T F } ( x ) = \log _ { 1 0 } \bigl ( 1 + c ( x ) \bigr ) o r c ( x )
$$

c(x) is $\#$ times x appears in the document

$$
\mathrm { I D F } ( x ) = ~ \log _ { 1 0 } \left( \frac { N _ { d o c s } } { D F ( x ) } \right)
$$

DF(x) is the number of documents x appears in.
