# 網站文章資料_嘻哈新生代奎仔出輯 GAME TIME比賽開始！_3033

## 一、文章基本資料

- **文章ID**: 3033
- **文章標題**: 嘻哈新生代奎仔出輯 GAME TIME比賽開始！
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20180723
- **文章簡介**: 【Yahoo奇摩娛樂新聞】

曾於2017年代表台灣，前往深圳參加中國大型饒舌比賽Listen Up的奎仔，挾著初生之犢不畏虎的氣勢，於今年6月大學畢業之際，發行首張個人專輯《GAME TIME》。

「GAME TIME」是比賽開始的時間，也可以解讀成遊戲時間。對奎仔而言，比賽就如同遊戲一般，可嚴肅也能戲謔面對。人生亦是如此，電玩遊戲的虛擬就不是真實嗎？人生如夢一場，選擇遊戲人間並盡情享受，但仍認真面對每一場挑戰。這就是奎仔的GAME TIME。

大小地下表演經驗豐富的奎仔，有著嘻哈新生代間公認，穩健而無懈可擊的饒舌技術。能快意順暢地揮灑韻腳，也可透過聲線之多變營造聽眾的情境想像。這亦反映在其新專輯的曲風多變之上。(待續)


【關於奎仔】
奎仔，嘉義人，喜歡看漫畫，打單機遊戲，黑皮工作室創辦人之一，大學北上桃園念書，因緣際會加入元智嘻研社，開始音樂之路，作品不願意受框架設限，喜歡以一種玩世不恭的第三者心態去呈現，有多次表演經驗，曾得到中國舉辦的Listen up台北站季軍，代表台灣至深圳比總決賽，也是元智大學106級畢業歌創作者之一，在2018年6月底發行首張個人專輯。
