# 網站文章資料_航空城要拆福海宮廟埕 信徒要求完整保留_2220

## 一、文章基本資料

- **文章ID**: 2220
- **文章標題**: 航空城要拆福海宮廟埕 信徒要求完整保留
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20170918-3
- **文章簡介**: 【自由時報/記者謝武雄／桃園報導】

位於大園竹圍福海宮因為在桃園航空城都市計畫區內，寺廟主體位於航空城計畫區範圍內，原本要被拆除，在信眾爭取下，交通部同意不拆，但僅保留周邊20米範圍，與眾希望保留廟埕周邊100米不符，為此，福海宮今天舉辦文化座談會，邀請多位專家學者及中央、地方民意代表出席，會中大家一致呼籲尊重寶貴的宗教文化資產，維持廟埕周邊百米規劃。

中央大學土木系教授林志棟表示，福海宮根本不影響未來第三跑道飛航安全；元智大學社會政策系教授劉阿榮建議將福海宮開發為宗教文化產業園區，讓福海宮成為航空城守護神的新角色。(待續)
