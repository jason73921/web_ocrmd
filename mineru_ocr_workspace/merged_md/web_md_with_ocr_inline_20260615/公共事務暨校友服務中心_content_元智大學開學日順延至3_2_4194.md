# 網站文章資料_元智大學開學日順延至3/2_4194

## 一、文章基本資料

- **文章ID**: 4194
- **文章標題**: 元智大學開學日順延至3/2
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20200205-4
- **文章簡介**: 因武漢肺炎疫情蔓延，元智大學配合國家防疫政策，避免校園發生群聚感染，維護全校教職員工生健康，開學日期延至3/2 ，並取消3/7校慶相關活動。

元智開學日順延兩週調整為3/2(一)，學期結束日順延至7/3(五)，共計18週。另，3/7（六）為「元智大學31周年校慶活動暨運動會」，為落實防堵疫情擴散之各項防治措施，以降低群聚事件發生的機會，今年校慶相關活動皆取消辦理。

此外，因應武漢肺炎疫情元智大學並提供安心就學措施，內容包括如有延遲返校上課者，由開課單位轉知授課教師，再個別提供必要之課業輔導或其他加退選作業。若無法返校或因隔離無法繼續學習者，得視個案需要，由教務處通知開課單位，請其協調授課教師以錄影非同步，或提供線上教材檔案、個案成績評量方式等作法讓學生修課。另外，諮商與就業輔導組將配合陸生輔導單位之轉介，適時提供學生諮商輔導或職涯輔導，以協助渡過困境。

元智大學表示，非常時期請大家配合政府防疫，武漢肺炎病毒除口沫傳染外，接觸感染也是傳染原因，學校提醒教職員生勤洗手，必要時請戴口罩，多運動增強免疫力，少出入公共場所，如有身體不適請記得戴口罩並就醫。

*108學年度行事曆依據109.02.05 108-13電子行政會議修訂通過，行事曆連結（行事曆尚需報教育部備查後實施）
