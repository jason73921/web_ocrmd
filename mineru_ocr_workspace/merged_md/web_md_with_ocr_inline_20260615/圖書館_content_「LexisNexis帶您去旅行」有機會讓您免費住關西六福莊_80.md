# 網站文章資料_「LexisNexis帶您去旅行」有機會讓您免費住關西六福莊生態渡假旅館乙晚_80

## 一、文章基本資料

- **文章ID**: 80
- **文章標題**: 「LexisNexis帶您去旅行」有機會讓您免費住關西六福莊生態渡假旅館乙晚
- **文章網址**: https://www.yzu.edu.tw/library/index.php/tw/
- **文章簡介**: 
- **文章全文**: 即日起於LexisNexis有獎徵答活動網站回答問題並填寫資料，即有機會獲得關西六福莊生態渡假旅館乙晚、咖啡機、膠囊咖啡機、隨行杯果汁機、7-11禮卷等豐富大獎。

◎活動期間：即日起至2014年11月30日

◎活動方式：凡是在 「LexisNexis帶您去旅行」活動網站參加本活動，進行答題並填寫相關資料送出，即完成參加手續。

※資料庫連結路徑：元智大學圖書館→館藏資源→電子資料庫→輸入Lexis即可進入Lexis.com或LexisNexis Academic找到活動答案哦！

◎活動獎項：



【注意事項】請務必完整填寫各欄位資料，資料缺漏者將取消參加資格。

        1. 獎品於活動結束後兩週內以e-mail通知中獎者，通知後一週內無回咅或未能領獎者視同放棄，獎項將以郵寄方式發送。 必要時主辦單位得請得獎者出示學生證或教職員服務證明以確認身分。

        2. 得獎者若同一人重複中獎，將取最高之獎項送出。獎品將依所得稅法相關規定辦理，逾期未領取者視同放棄。

        3. 主辦單位保有本活動資格審核最終權利，並保留以其他等值獎品替代本活動原公佈獎項品項之權利。

        4. 參加活動者即視同認可並接受本辦法之各項規定。
