# 網站文章資料_2019機器人智機化應用競賽 台灣大學電機所奪冠_3932

## 一、文章基本資料

- **文章ID**: 3932
- **文章標題**: 2019機器人智機化應用競賽 台灣大學電機所奪冠
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20190902
- **文章簡介**: 【工商時報 文╱蔡榮昌】

歷經半年的激烈競爭，經過初審、複審的層層考驗，由工業局主辦，精機中心負責承辦之「2019機器人智機化應用競賽」， 日前提（24）日頒獎典禮後得獎隊伍終於揭曉，由第一名國立台灣大學電機奪冠。

因應政府智慧機械發展政策，今年度競賽以「工業機器人智機化應用」為主題，藉由競賽激發國內大專院校學子創新思維，設計具創意性、實用性的機器人產品及智機化應用。並藉由學界在創意思考之優勢，將產業需求與所面臨的問題，透過工業機器人智機化應用與融入使用情境，以機器人實際動態呈現。

2019機器人智機化應用競賽得獎名單如下：第一名國立台灣大學電機所，作品名稱：智慧遊龍。第二名國立台北科技大學自動化科技研究所，作品名稱：立體視覺結合力感之智慧機器人去毛邊系統。第三名建國科技大學自動化工程系暨機電光系統研究所，作品名稱：視覺智慧澆鑄系統-以六軸機器手臂重量感測器。

佳作淡江大學機械與機電工程系，作品名稱：皮卡丘。佳作元智大學機械工程系，作品名稱：開心製造所。佳作國立高雄大學電機工程系，作品名稱：智手遮天。佳作國立成功大學機械工程系，作品名稱：手上長眼。（待續）
