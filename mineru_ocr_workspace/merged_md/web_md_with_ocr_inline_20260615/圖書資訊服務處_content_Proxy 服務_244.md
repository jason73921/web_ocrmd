# 網站文章資料_Proxy 服務_244

## 一、文章基本資料

- **文章ID**: 244
- **文章標題**: Proxy 服務
- **文章網址**: https://www.yzu.edu.tw/admin/is/index.php/tw/proxy
- **文章簡介**: 
- **文章全文**: 透過 HTTP Proxy Server 的 Cache 功能，可減少 www client 端直接連往 www server 的頻率，藉此可降低網路流量, 並增快 client 端擷取資料的速度。

自動設定與手動設定兩種設定擇一即可。如果沒有特殊需求, 自動設定會比手動有彈性且方便。而且自動設定會自動幫你選擇最適當的 proxy 伺服器，在速度和適應性上面會較好。

如何讓瀏覽器自動設定 proxy：


Firefox 使用者 

 請點選 Firefox->選項

 或是 請點選 工具 -> 選項

 在 "進階" 功能的 "網路" 頁籤下，按下 "設定" 的按鈕，

 請點選 "Proxy 自動設定網址(URL)" 並在空格裏面填上 http://proxy.yzu.edu.tw/proxy.pac ，並按下確定按紐。




Internet Explorer 6.x 以上 使用者



打開 工具 選單裏的 Internet 選項(網際網路選項)，

 進入連線的畫面，再進入區域網路設定 ，

 在 "使用自動組態指令碼" 上打鉤，並在網址欄位內 填上 http://proxy.yzu.edu.tw/proxy.pac ，按確定即可。


切記，勿將自動偵測設定打鉤。
