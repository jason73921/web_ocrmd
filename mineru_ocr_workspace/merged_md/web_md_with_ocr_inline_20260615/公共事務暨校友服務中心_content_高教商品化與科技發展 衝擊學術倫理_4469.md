# 網站文章資料_高教商品化與科技發展 衝擊學術倫理_4469

## 一、文章基本資料

- **文章ID**: 4469
- **文章標題**: 高教商品化與科技發展 衝擊學術倫理
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20200724
- **文章簡介**: 【中央社記者陳至中台北23日電】

中國國民黨高雄市長補選參選人李眉蓁涉論文抄襲，連日來引起討論。學界人士認為，高教商品化的趨勢，以及科技發展的衝擊，學術倫理的要求須與時俱進。

元智大學校長吳志揚接受中央社記者訪問表示，科技發展讓論文抄襲、複製、引用更為容易，但另一方面，也發展出越來越精細的比對軟體，方便教授審核時運用。現在各大學都有相關規範，要求學術論文發表前都要跑過軟體，避免抄襲、剽竊等問題。

吳志揚認為，隨著技術越來越發達，學術論文越透明越好，即便是碩士論文，也都該全文公開接受檢驗。畢竟論文本來就是寫出來讓人看，甚至還希望別人多多引用，而不是鎖在櫃子裡藏起來。

吳志揚表示，學術倫理是永恆不變的議題，「誠實是最好的策略」；元智大學在所有研究所、大學部都有學術倫理相關課程，包括實驗室筆記，也都有一定的要求，讓學生儘早熟悉學術規範。他相信隨著各界的重視，抄襲的情況應會有所改善。（待續）
