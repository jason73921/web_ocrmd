# 網站文章資料_東海大學「第二十四屆國際氣膠科技研討會-2017細懸浮微粒(PM2.5)管制與氣候變遷論壇」第二輪徵稿通知_1149

## 一、文章基本資料

- **文章ID**: 1149
- **文章標題**: 東海大學「第二十四屆國際氣膠科技研討會-2017細懸浮微粒(PM2.5)管制與氣候變遷論壇」第二輪徵稿通知
- **文章網址**: https://www.yzu.edu.tw/admin/rd/index.php/tw/?option=com_content&view=article&id=1149
- **文章簡介**: 一、 旨揭論壇，由臺灣氣膠研究學會、本校環境科學與工程學系共同舉辦。該研討會將提供國內外產官學界之氣膠相關領域交流平台。會議日期訂為106年9月8日至9日於本校管理學院舉辦。   二、 該論壇將涵蓋大氣氣膠與氣候變遷、大氣氣膠與空氣品質、生物氣膠與室內空氣品質、空氣污染物之控制策略與技術、氣膠監測與評估、PM2.5管制策略與源解析、PM2.5暴露與健康效應等相關重要議題。   三、 詳細資訊請參閱該研討會官方網頁：http://www.taar.org.tw/conference/2017。
