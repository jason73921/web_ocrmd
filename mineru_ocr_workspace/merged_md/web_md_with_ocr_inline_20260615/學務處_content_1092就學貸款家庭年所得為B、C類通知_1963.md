# 網站文章資料_1092就學貸款家庭年所得為B、C類通知_1963

## 一、文章基本資料

- **文章ID**: 1963
- **文章標題**: 1092就學貸款家庭年所得為B、C類通知
- **文章網址**: https://www.yzu.edu.tw/admin/st/index.php/tw/?option=com_content&view=article&id=1963
- **文章簡介**: 1092就學貸款家庭年所得為B、C類通知已於3月29日發e-mail及簡訊通知，請詳閱e-mail。

一、就學貸款家庭年所得為B類者：即家庭年收入逾114萬元至120萬元，自償還期起算日起之利息，由主管機關(教育部)及借款學生各
　　負擔半額利息，且應自貸款撥款日次月起按月繳交。

       敬請於4月9日(五)前繳交同意書至生輔組，以利後續作業。

       下載：B類通知書暨回函同意書

二、就學貸款家庭年所得為C類者：即家庭年收入逾120萬元，請於4月9日(五)前繳交「同意書」及「另一個兄弟姐妹就讀1092學期之
　　高中職(含)以上的就學證明文件」(即本學期完成註冊並蓋有註冊章的學生證影本或繳費證明影本)至生輔組，否則不符資格，將取
       消貸款。

       下載：C類通知書暨回函同意書

同意書可由年滿20歲者可自行簽妥後繳回；未滿20歲者除了自行簽名外，仍須由法定監護人、代理人(或保證人)簽名或簽章後繳回或傳真03-4353511至生輔組。

若針對家庭年收入有疑慮者，請於4月9日(五)前至國稅局申請「父、母親及學生本人之108年各類所得清單」繳交至生輔組申復。

以上敬請知悉，謝謝！
