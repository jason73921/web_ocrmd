# 網站文章資料_新明青創基地程式教育團隊進校園 機器人教育更有趣_4276

## 一、文章基本資料

- **文章ID**: 4276
- **文章標題**: 新明青創基地程式教育團隊進校園 機器人教育更有趣
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20200406-2
- **文章簡介**: 【聯合報/記者鄭國樑/即時報導】

青年局表示，資訊世代的衝擊，全世界都在夯程式設計，而且已然成為新趨勢。聚忻科技顧問公司進駐新明青創基地，提出教育理念為「玩中做、做中學、學中用」，用有趣的程式教育激發學生學習動力。

聚忻科技並且運用「STEAM教育」與「曼陀羅思考法」，以情境式教學法讓學生們在程式設計之前，先進行開創式的程式邏輯發想，再透過網頁編寫程式，讓學生能夠依照自己的想法去操控麥坤小車、寶比機器人等教具，創新有趣的程式課程受好評，贏得許多家長及學校青睞，也成功轉型學校教育為新創事業。

聚忻科技顧問去年進駐桃園新明青創基地，藉基地的宣傳效益，平均每星期開設2至3次課程、聚會及討論，不到一年的時間已培育出13位助理教師以及2位專業講師，成員包含明新、元智及健行三所大專院校的學生。（待續）
