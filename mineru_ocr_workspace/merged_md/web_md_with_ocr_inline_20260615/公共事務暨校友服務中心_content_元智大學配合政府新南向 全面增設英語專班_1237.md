# 網站文章資料_元智大學配合政府新南向 全面增設英語專班_1237

## 一、文章基本資料

- **文章ID**: 1237
- **文章標題**: 元智大學配合政府新南向 全面增設英語專班
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【中央社】

因應全球化時代以及新政府的新南向政策，元智大學自明年(106)起，各學院全面增設英語專班(學士英語學程)，每學院招生名額30-40名左右(共130名)。未來並結合企業產學合作學程，提供學生學雜費，培育產業人才。另因應政府新南向政策，規劃東南亞中心，設置「亞太文學與文化學程」以開拓學生之國際視野。 

元智大學為全國首創雙語大學，為建置更好的國際學習環境、實施專業課程英語教學、培養具國際觀之管理人才，元智大學自96學年度起，增設管理學院英語專班，首屆即吸引成績優異的學生申請入學，頗獲各界好評。明年起元智將全面增設各院英語專班，以迎接政府之新南向政策。 


全球化的時代，英語還是最重要的溝通語言，元智大學校長吳志揚表示，因應政府新南向政策，許多國際或是在地大廠溝通上仍以英語為主，學生還是要有全球化的概念，尤其星 、馬、菲律賓、泰國、印度等國家，在品牌、商標或宣傳、標示，還是以英文為主，落實英語的學習還是最基本。 


吳志揚說，近日教育部提出的政策，僅是就過去政策建立的基礎與盤點，但台灣要走向世界，走向全球化，除配合東南亞國家開設東南亞語文課程，或是提供獎學金招攬東南亞學生外，還需有更遠大的格局。吳志揚說，因為東南亞各國語言非常多元，英語溝通還是最基本，元智大學是全國首創雙語教學，1/3以上全英語授課，自明年開始，各院全面增設英語專班或學程，就是要重點招生，配合國家政策，結合企業，培育人才，以迎接新南向之人才所需。 


此外，迎接新南向政策，元智人文社會學院中語系也規劃「東南亞中心」以及「亞太文與文化學程」，吳志揚表示，教育部已將東南亞地區視為我國推動華語文教育的重點地區之一，桃園是臺灣新移民第二大的聚居地，僅次於新北市，成為該市的「第五大族群」，社會影響力不容忽視。元智東南亞中心的師資來自海內外，有其特殊的文化與背景，以其教學人力與文化之優，對於推動東南亞的研究與教學，相信更能得心應手。 

元智英語專班的未來發展特色，將建構專各院英語專班之「國際業師系統」，爲學生開拓國際視野、建立國際人脈網絡與國際文化的學習與了解， 藉由國際業師、以及海外企業實習的課程安排，協助政府在新南向人才的培育。另外，未來將搭配集團企業東協佈局之人力需求，開設專班以及業師輔導之專業實習，達到產學人才培育之合作，學生就學與就業無縫接軌，積極往東協國家招生，並提升企業在地價值與競爭力。
