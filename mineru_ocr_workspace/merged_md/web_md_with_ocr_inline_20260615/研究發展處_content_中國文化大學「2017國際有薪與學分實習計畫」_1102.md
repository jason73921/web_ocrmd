# 網站文章資料_中國文化大學「2017國際有薪與學分實習計畫」_1102

## 一、文章基本資料

- **文章ID**: 1102
- **文章標題**: 中國文化大學「2017國際有薪與學分實習計畫」
- **文章網址**: https://www.yzu.edu.tw/admin/rd/index.php/tw/?option=com_content&view=article&id=1102
- **文章簡介**: 主 旨:有關中國文化大學推廣教育部擬至貴校辦理2017大專院校學生「國際有薪與學分實習計畫」免費講座並提供五名免費代辦名額予貴校學生事宜。

說 明:

一、「國際有薪與學分實習計畫」是依照泰國與阿拉伯聯合大公國相關規定訂定之教育與文化交流計畫，透過此計畫提供本國學生至當地四至五星級旅館實習機會，提升職場英語應用能力、學習星級旅館管理實務之技能、體驗高度國際化之商業文化、擴展國際視野與加速進入職場免於畢業即失業窘境。

二、計畫參加泰國實習資格:大專院校在校學生與畢業未滿一年之青年，可實習3或6或12個月。計畫參加阿拉伯聯合大公國實習資格:大專院校在校學生與畢業未滿兩年之青年，可實習6或12個月。

三、欲參與此計畫之學生須通過徵選與面試，錄取後將依據學生之科系、經歷、英文程度等條件，媒合實習工作。

四、中國文化大學推廣教育部將提供五名免費代辦名額予貴校學生赴泰國與阿拉伯聯合大公國實習三或六個月，經旅館人事單位網路面談錄取學生後，將依據學生之科系、經歷、英文程度等分配實習工作。

五、中國文化大學推廣教育部希至貴校舉辦「國際有薪與學分實習計畫」免費講座，懇請惠予協助辦理相關事項，如:租借教室或協助佈達相關訊息於貴校網站。若貴校學生對此計畫有興趣者，請聯絡:李小姐，電話:(02)27005858轉8585，e-mail:cylee@sce.pccu.edu.tw，網址:http://studyabroad.sce.pccu.edu.tw/

六、若貴校不方便舉辦免費講座，懇請惠予張貼「國際有薪與學分實習計畫」之網站連結:http://studyabroad.sce.pccu.edu.tw/至貴校網站。

七、本計畫由中國文化大學推廣教育部策畫與長泰事業股份有限公司執行。
