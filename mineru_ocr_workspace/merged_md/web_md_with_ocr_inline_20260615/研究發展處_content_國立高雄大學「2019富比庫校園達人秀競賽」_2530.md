##  特殊字元處理方式

1. 字元上方出現 ഥ 請在字元後加上 "\"

2. 上標字元，請在文字後面加上 $1 1 \wedge 1 1$

3. 下標字元，請在文字後面加上 $" \wedge \wedge "$

![](images/f3bf0522a57ec45c3e76ced95d8aec2f4dd6a07ca6ed32b4142bd46d1c114a20.jpg)

 辨識正確率計算方式

1. 以編號(Number)所對應的內容(Content)做比對後，逐㇐字元比對完全正確，則視為正確個數(K)。

2. 編號(Number)重複不列入正確個數計算。

3. 正確 $\begin{array} { r } { \vec { \mathfrak { L } } _ { \mathrm { * } } ^ { \sharp } = \frac { \overline { { \mathbb { E } } } ^ { \sharp } \sharp \sharp \langle \vec { \theta } | \oplus \rangle } { \underline { { \xi } } _ { \mathrm { * } } ^ { \sharp } \sharp \langle \vec { \bf R } \rangle } } \end{array}$ ，四捨五入至小數點第二位。
