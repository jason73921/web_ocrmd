# 網站文章資料_【生輔組公告】社團法人中國工程師學會114年度「優秀工程學生獎學金」、「沈怡學金」、「莫衡先生紀念獎學金」_5843

## 一、文章基本資料

- **文章ID**: 5843
- **文章標題**: 【生輔組公告】社團法人中國工程師學會114年度「優秀工程學生獎學金」、「沈怡學金」、「莫衡先生紀念獎學金」
- **文章網址**: 
- **文章簡介**: 一、 「優秀工程學生獎學金」候選人應符合下列條件：
(一)具有領導才能並有具體優良事蹟之大專校院工程科系學生，且為最後壹學年就讀之本國籍在學學生（不含研究生）。
(二)在學期間之學業總平均在80分以上，操行成績均為甲等。
(三)獎金金額及名額：每年遴選 10名為限，每名30,000元及獎牌，每校限推薦l名。
二、 「沈怡學金」候選人應符合下列條件：
(一)學校院工程相關科系之優秀學生；水利、土木、城鄉建築等科系學生優先考慮。
(二)目前於大學校院就讀之本國籍在學學生。
(三)前一學年學業成績總平均及操行成績皆在80分以上者。
(四)具備家境清寒證明；以區鄉鎮公所核發之證明為優先。
(五)獎金金額及名額：每年遴選 2名為限，每名30,000元及獎牌，每校限推薦l名。
三、 「莫衡先生紀念獎學金」候選人應符合下列條件：
(一)目前於公私立大學校院大學部就讀之在學學生。
(二)述校院中之土木、水利、運輸、環工及測量等相關科系之優秀學生具有傑出事蹟者。
(三)前一學年度學業及操行成績總平均皆在80分以上者。
(四)獎金金額及名額：每年遴選 2名為限，每名50,000元及獎牌，每校限推薦l名。
四、 申請期程：即日起至114年3月15日止。
