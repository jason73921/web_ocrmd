# 網站文章資料_少子化 51大專院校寄存名額近6000_7075

## 一、文章基本資料

- **文章ID**: 7075
- **文章標題**: 少子化 51大專院校寄存名額近6000
- **文章網址**: 
- **文章簡介**: 【人間福報/記者曾博群綜合報導】

少子化衝擊下，教育部允許大專校院以寄存名額取代減招。根據最新統計，113學年共有51校申請寄存共5898個招生名額，近5年總寄存名額逾3萬7000個，私校成重災區。教團形容，寄存僅是製造「滿桌」假象，高註冊率不等於熱門科系；也有大學校長分析，寄存高峰已過，除非因註冊率過低恐遭教育部扣減名額，學校也不會主動申請。

教育部自103學年推出寄存名額制度，大專校院可申請減招名額，減少的名額寄存在教育部，當學校招生狀況好轉，或想調配招生名額時，可再取回。113學年共有51所學校申請，包含2所公立大學寄存29名，14所私立大學寄存2687個；4所國立技專寄存305個，31所私立技專寄存2877個；整體較去年9021個名額減少。

元智大學校長廖慶榮說，111學年適逢考招新制，各校申請入學大量缺額，導致112學年寄存名額創新高，屬特殊現象，今年各校缺額大減，新申請寄存名額也下降。
