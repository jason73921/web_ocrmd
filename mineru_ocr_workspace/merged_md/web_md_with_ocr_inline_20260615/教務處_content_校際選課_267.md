1142學期外校生至本校校際選課辦理時間：115/2/23(一)~115/3/3(二)16:00

校際選課相關作業時程（含選課、停修），請確實依據兩校作業規定及時程辦理。如有衝堂以零分列計，本校系統所填之上課時間，請務必轉換為本校節數時間填寫。



依據專科以上學校遠距教學實施辦法 第七條規定：學生修習遠距教學課程成績及格，且符合大學法施行細則及專科學校法施行細則學分計算之規定者，由學校採認其學分，並納入畢業總學分數計算。採計為畢業總學分數之遠距教學課程學分數，不得超過畢業總學分數之二分之一。



學生是否可修習外校遠距教學課程之校際選課，由各系所自行決定。





一、元智、中原與中央大學跨校選課合作聯盟注意事項





    1. 元智、中原與中央大學簽署校級校際選課協議，自105學年度起生效。

    2. 合作聯盟校際選課以三校學生為對象，並以學期課程為限，不包含暑期課程。

    3. 本校學生選修他校之科目、學分總數，依本校校際選課辦法相關規定辦理。

    4. 聯盟學生校際選課依原肄業學校規定繳費，免繳開課學校學分費。即全額學雜費生註冊完成後，無須再繳交校際選課學分費；非全額學雜費生則須至本校總務

        處繳交校際選課學分費，非繳至開課學校。暑期校際選課學分費，修習校際選課學生皆須繳交學分費至開課學校。

    5. 本校學生每學期選讀他校之課程學分數，限佔該學期所修學分數總數三分之一（含）以內，但大學部四年級（含）以上、碩士班級博士班學生不受此限制。

    6. 聯盟所屬各校上課節次時間不同，學生校際選課，其上課及往返時間不得與選修本校科目時間衝堂，一經查出有衝堂情形，衝堂之科目均以零分計算，不得異

        議。

    7. 中原大學開課查詢系統：https://itouch.cycu.edu.tw/active_system/CourseQuerySystem/

        中原大學課表時間：http://acdm.cycu.edu.tw/H2000/elective_msg/table_load/class_tab1.htm 

       (元智學生登入元智首頁／個人Portal ／學習檔案／校際選課申請校際選課時，請務必將他校課表時間對應至元智課表時間，輸入時以元智課表時間為準，俾便

       與元智課表查詢是否衝堂，如有衝堂以零分列計，並請正確輸入他校課號班別，俾便課務組再次查核)

      中央大學開課查詢系統：https://cis.ncu.edu.tw/Course/main/news/announce


       中央大學課表時間：https://pdc.adm.ncu.edu.tw/Course/course/COUR_U.pdf

    8. 學生修習一般學期課程之全額學雜費生，請註冊完成後再行列印校際選課單。

    [註]:外校生在中央大學人工加退選期間辦理加選者，另須繳交手續費100元。

    [註]:外校生至中央大學修習在職專班課程(課號共六碼，第三碼為A)、師資培育課程(課號共六碼，前二碼為EP) ，須另繳交學分費至中央大學出納組。

    [註]:本校已與台北商業大學簽定策略聯盟合約書，雙方課程互選免收學分費(學期間學分生延畢生須繳交學分費)。





二、本校同學至他校校際選課程序： (因疫情關係，他校做法可能有所改變，請自行詢問他校)




    1. 所有大學與獨立學院均可前往校際選課（須詢問系上是否承認該門課）。

    2. 自行詢問他校校際選課選課及繳費期限。

    3. 登入元智首頁／個人Portal ／學習檔案／校際選課申請校際選課。

    4. 開課單位或系主任審核結束後，系統自動發信至學生信箱，請同學自行列印核准之書面申請書，課務組已電子同意無須到校蓋章，由學生持該單逕向他校辦理

        選課手續，完成外校程序後，再將第二聯繳回課務組確認加選(第二聯請盡量用郵寄方式於期限內寄回課務組，因疫情關係，他校做法可能有所改變，請自行詢

        問他校，也可不用郵寄方式。)，經課務組審核第二聯後，不符合規定者，例如衝堂等事項，則此單會依規定最終審核不同意。須符合開課單位之規定，課務組

   之”電子同意”始生效。

    5. 至他校辦理繳費登記等程序後，攜第二聯（本校教務處存聯）回校備查(第二聯請盡量用郵寄方式於期限內寄回課務組(因疫情關係，他校做法可能有所改變，

  請自行詢問他校)。

    6. 校際選課單需於期限內繳回(第二聯請盡量用郵寄方式於期限內寄回課務組)，逾期者選課程序未完成視爲選課無效。





三、他校同學至本校校際選課程序：




    1.他校學生選讀本校開設之課程（教學計畫書），須先經其肄業學校之書面同意（校際選課單）。

    2. 於本校公告之「電腦選課時間」內，至本校教務處課務組查詢課程選課情形或經本校開課學系同意。

    3. 依照他校學生書面校際選課單，至本校各單位處室辦理，並於出納組繳完選課費用。

    4. 送至課務組協助選課，逾期不予受理。

    [註]: 相關選課日程可於本校行事曆查詢
