# 網站文章資料_【轉知】新北市立圖書館2019年「我的閱讀我做主」參與式預算活動訊息_1006

## 一、文章基本資料

- **文章ID**: 1006
- **文章標題**: 【轉知】新北市立圖書館2019年「我的閱讀我做主」參與式預算活動訊息
- **文章網址**: https://www.yzu.edu.tw/library/index.php/tw/20190517-1
- **文章簡介**: 說明：
一、旨揭活動為推廣圖書館創新閱讀方式，邀請民眾發揮專業與想像，規劃設計獨特有趣的「創意閱讀活動」，將閱讀進行不同的連結與發想，前三名獲票數最多提案，每案有10萬元執行提案經費，將由提案人參與討論計畫執行與經費運用方式，並由圖書館實際辦理活動。
二、本活動提案徵件日期為5月20日起至6月20日止，提案者年齡不限制，歡迎踴躍參與提案，請貴校協助宣傳與推廣，詳細活動內容及相關資訊請逕至新北市立圖書館網站（https://www.library.ntpc.gov.tw/htmlcnt/cc51986fd35045f9ac6c206db52c3782）查詢。
