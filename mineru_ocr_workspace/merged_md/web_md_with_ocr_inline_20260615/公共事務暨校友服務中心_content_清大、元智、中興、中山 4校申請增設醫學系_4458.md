# 網站文章資料_清大、元智、中興、中山 4校申請增設醫學系_4458

## 一、文章基本資料

- **文章ID**: 4458
- **文章標題**: 清大、元智、中興、中山 4校申請增設醫學系
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20200720
- **文章簡介**: 【公共電視/林曉慧 陳柏諭 台北報導】

涉及醫師總量管制，目前醫學系每年招生限制在大約一千三百人，今年包括清大、元智、中興與中山四所大學申請要增設醫學系。清大表示希望要培養有現代工程知識背景的醫師，元智大學則是要和同集團的亞東醫院合作籌建醫學院。

 

元智大學校長吳志揚表示，「他們又是一個教學的醫院，所以在這個各項資源、師資設備，或者是對整個醫療的發展等等，各個面向，他們應該是不亞於其他的醫學系。」

目前全台有13所大學醫學系，每年招生上限約為1300人。為了確保醫學教育品質，教育部表示，最新規定是校方得提自我評估報告，轉醫學院評鑑委員會預評，提意見給各校修正，再和衛福部一起進行專業審查。預計今年10月、11月後才會有結果，獲通過的學校，最快110學年招生。（待續）
