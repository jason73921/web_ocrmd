# 網站文章資料_遠傳電信 全台灣首次5G核心網路展示 Network Slicing及NFV關鍵技術_1810

## 一、文章基本資料

- **文章ID**: 1810
- **文章標題**: 遠傳電信 全台灣首次5G核心網路展示 Network Slicing及NFV關鍵技術
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【自由時報】

遠傳電信持續創新腳步，在2016年9月發表下載速度可達1Gbps之Gigabit LTE網路後，緊接著在2016年12月再引進全台第一個5G原型機網路，體驗超過5 Gbps飆網速度之後，遠傳今日展示全台灣第一個5G核心網路（5G Core），其中包括網路切片(Network Slicing), 虛擬化網路(Network Function Virtualization, NFV)、分散式雲端(Distributed Cloud) 等5G關鍵技術，展現遠傳電信在5G技術創新能力。 


遠傳電信5G實驗室全台首次5G核心網路展示 

遠傳5G核心網路透過物聯網應用來展示，並結合愛立信、工研院及元智大學合作項目，包括一、關鍵型任務物聯網(Mission Critical IoT) 的無人機操控，其要求網路效能能滿足超低時延及可靠性；二、有多樣性感應裝置的海量型物聯網(Massive IoT)應用，包括智慧家庭及智慧健康照護等應用中之各種感測器及能與人對話、互動的機器人幫手，以及與元智大學合作的智慧農業等；三、高速寬頻應用，現場手機播放需要大頻寬支援的4k串流影片，播放時影像維持順暢高品質。藉由NFV網路高彈性組網能力，能夠在單一實體網路上，切分出三個各自獨立之虛擬邏輯網路，同時提供三種不同網路特性需求的應用，彼此間不受影響，例如在關鍵型任務物聯網應用下，可以讓切出之核心網更靠近基地台端，藉以降低時延。 


5G核心網路可應用於任務型無人機操控 網路效能可滿足低時延及可靠性 

遠傳電信執行副總經理饒仲華表示：「藉由5G關鍵技術網路切片(Network Slicing)能夠讓營運商透過單一網路，針對不同應用及不同客戶需求提供專用的虛擬網路。因應未來5G多元應用，對於網路將有不同程度的要求和考量包括成本、數據量、連結裝置數量、可靠度和時延程度等等，因此網路需具備足夠的彈性以做出調適，符合不同產業應用的需求。在這次成功展示5G核網技術後，遠傳將於今年導入Network Slicing 的第一步NFV於目前4.5G現網上。」 


台灣愛立信總經理何可申表示：「根據最新的愛立信行動趨勢報告指出，預計到2022年，將會有290億台連網裝置，其中物聯網裝置將達到180億。在新應用案例的帶動下，物聯網裝置預期將以21%的複合年成長率增加。為此愛立信領先市場提供5G核心系統以及完整的產品組合，以支援遠傳電信在5G方面的測試並展示5G關鍵技術，以期5G能夠早日在台深耕，因應不同產業提供具備彈性的網路，也為消費者帶來最新的服務。」

遠傳5G實驗室成立之初，就期許做為未來中、下游軟、硬體廠商各項測試實驗的用途。2017年遠傳仍引領電信業邁入下一個5G的重要里程碑， 5G實驗室網路已可提供未來高階手機 (網速可達1Gbps的手機)實測，目前已有國際手機大廠洽談測試合作計畫，有助於台灣5G加速啟動，帶動5G上下游產業鏈活絡。
