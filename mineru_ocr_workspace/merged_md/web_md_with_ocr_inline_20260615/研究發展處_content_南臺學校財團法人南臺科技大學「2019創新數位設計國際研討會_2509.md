# 網站文章資料_南臺學校財團法人南臺科技大學「2019創新數位設計國際研討會」_2509

## 一、文章基本資料

- **文章ID**: 2509
- **文章標題**: 南臺學校財團法人南臺科技大學「2019創新數位設計國際研討會」
- **文章網址**: 
- **文章簡介**: 主旨：檢送本校創新產品設計系舉辦「2019創新數位設計國際研討會」活動訊息，敬請惠予公告並鼓勵所屬相關人員踴躍與會，請查照。
說明：
一、活動主旨：聚焦數位科技與設計的趨勢之研究與應用，以各種設計思維與創新設計，厚植設計力，協助臺灣擺脫新興國家的紅海攻略，提升產業價值，提振競爭優勢。讓臺灣在全球化的過程中，可以結合科技、文化、設計與行銷，找出傳統產業的新價值與科技創新的方向與未來，立足台灣，放眼全球，並邀請國外知名學者蒞會參與研討，促進設計創新之國際交流，歡迎國內設計科系師生共襄盛舉。
二、活動資訊：
(一)活動時間：108年5月31日（星期五）08：00-16：00。
(二)活動地點：本校E棟（13F）國際會議廳。
