# 網站文章資料_國立臺北教育大學 辦理「第十三屆發展研究年會-後疫情時代的區域發展轉型與大學社會實踐」_3813

## 一、文章基本資料

- **文章ID**: 3813
- **文章標題**: 國立臺北教育大學 辦理「第十三屆發展研究年會-後疫情時代的區域發展轉型與大學社會實踐」
- **文章網址**: https://www.yzu.edu.tw/admin/rd/index.php/tw/?option=com_content&view=article&id=3813
- **文章簡介**: 主旨：檢送本校社會與區域發展學系與台灣發展研究學會將於 110年10月30至31日（六、日）共同辦理「第十三屆發展研究年會-後疫情時代的區域發展轉型與大學社會實踐」研討會，檢附徵稿啟事1份，敬請惠予公告並鼓勵貴單位所屬或學校師生踴躍投稿，請查照。

說明：

一、本校社會與區域發展學系與台灣發展研究學會共同辦理研討會，以「後疫情時代的區域發展轉型及大學社會實踐」為主題，關注後疫情時代下，發展研究之各項議題，並研討當前全球化、新興科技、氣候變遷對區域不均發展之影響以及檢視後疫情時代下，國內大學又如何投入於社會實踐，來推動區域發展轉型進行探討。本系誠摯邀請不同學 門領域學者及博碩士生，針對十項徵稿主題（詳如附件），惠賜論文，並共同參與研討。

二、研討會重要日程：

(一)摘要繳交截止：110年6月26日。 

(二)摘要錄取通知：110年7月10日。

(三)全文繳交截止：110年10月07日。

(四)最佳論文獎申請：110年09月30日。

(五)研討會舉行：110年10月30-31日（六、日）。

三、本校社會與區域發展學系網址：https://social.ntue. edu.tw/
