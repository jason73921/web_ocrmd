# 網站文章資料_他抓得住美麗 文飛墨舞盡是文學地景_1929

## 一、文章基本資料

- **文章ID**: 1929
- **文章標題**: 他抓得住美麗 文飛墨舞盡是文學地景
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20170522
- **文章簡介**: 【Hinet新聞】

地景，可以是詩、可以是畫、可以是藝術、可以是文學，更可以是歷史，知道千塘之鄉的桃園市最大的陂塘在那裡？知道古老漢詩，書寫的桃園美景在那裡嗎？正在桃園市土地公文化舘展出的「文學地景-桃園生活書寫特展」，正在用詩詞抓住桃園美麗的瞬間永恆。


進入文學地景的展場，首先被地上的詩與兩大巨型捲軸作品所震攝。這場別開生面的展覽，藉由臺藝大古蹟系老師陳板，引用描述桃園的吳濁流漢詩等文學作品的書法、桃園作家邱傑的辨天池詩作及元智、中央大學詩社成員的新詩，融合古今詩作，介紹桃園各地景美麗風采，讓繁忙的現代手機生活裡，能在吉光片羽中，捉住古老桃園記憶中的永恆美景，對照今昔轉變，透過桃園在地創作的詩作及書法，結合文學、老照片及詩品，呈現桃園在地特色的文化饗宴。


出身自陂塘環繞八德的桃園市長鄭文燦表示，桃園陂塘有很多特色，提供文學家書寫桃園地景，尤其是陂塘文化，讓桃園產生與農業、生活等相關水域空間，文學家筆下等陂塘文化，與我們想的不一樣，希望大家來體會。


對桃園文史知之甚詳的鄭文燦說，桃園大圳於1917年開鑿完成，石門大圳則是民國50年代配合石門水庫完成，桃園大圳水利工程也是日籍工程師八田與一規畫設計，此2條大圳系統有很多支線，而且都有編號，以邱傑老師寫的1-1辨天池，當時是全桃園最大的一口陂塘，也是目前陽明社區及陽明公園所在地。


辨天池位於桃園後站介壽路與建興街之間，面積寬廣，有18甲之大，水深丈餘，游魚成群，是過去桃園台地最大的陂塘，曾為桃園名勝之一。 池中有辨天池、大黑二島，一在池之濱，一在池中央，間隔僅百五。有小舟可橫渡，島上樹木蔥茸、鳥語花香、清幽雅靜、風景宜人，因此，每年端午龍舟均在此舉行。因中正國際機場興建時，政府將機場所在地之大園鄉居民安遷於辨天池，因而填平池塘興建國民住宅，成為現今的陽明社區。


除了桃園的舊風貌，新的桃園也正在努力茁壯，所以這次展覽也呈現了現在青年對桃園的許多想像，包含元智不成文詩社及中央松林詩社，創作了許多有關桃園生活地景的小品詩，像是平鎮區相當著名的晴耕雨讀小書院，以及具時代性的中壢大時鐘，還有就在文化館後面的虎頭山，這些題材都是因為新生代創作者對桃園土地的感動，觸發他們的文學創作靈感。


負責策劃展覽的桃園市文化基金會表示，桃園市土地公文化館自去年11月14日試營運，並於今年2月27日開幕，由於展出內容豐富多元，參觀民眾直線上升，從今年1月到現在為止參觀人數已突破25萬人次了。
