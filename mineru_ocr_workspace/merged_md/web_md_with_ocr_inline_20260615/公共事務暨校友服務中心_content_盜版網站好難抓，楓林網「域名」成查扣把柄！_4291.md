# 網站文章資料_盜版網站好難抓，楓林網「域名」成查扣把柄！_4291

## 一、文章基本資料

- **文章ID**: 4291
- **文章標題**: 盜版網站好難抓，楓林網「域名」成查扣把柄！
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20200421
- **文章簡介**: 【遠見雜誌文/邱于瑄】

近日楓林網查封事件爆發，成了台灣OTT產業遲來的正義。盜版影音網站存在台灣多年，但法源不完備、追查不易，始終窒礙難行。此次查封楓林網，警方首度針對「域名」進行扣押，直接阻止盜版網站營運，將成為未來封網趨勢。

台灣最大盜版影音網站楓林網被抄，連帶如小鴨影音、gimy TV等盜版網站紛紛關站自保。「為何要奪走我放鬆的夥伴？」「我以後要去哪裡看片？」網友在Dcard、PTT等平台哀鴻遍野。

然而，KKTV資深行銷經理宋守珏曾說：「少一個盜版網站，可以讓我們多拍1000集《麻醉風暴2》。 」台灣OTT協會也曾指出，盜版機上盒或是影音平台，每年預估使OTT產業損失高達283億元。

盜版網站在台已有多年歷史，一直以來，盜版網站會將伺服器設在國外，或是租用境外網址，並在不同國家多點架設，成立多個備援網站，分散風險，且盜版網站最常使用層層轉址的方式，導至不同伺服器，使得執法單位難以追查。

此外，元智大學資訊傳播學系助理教授葉志良表示，因為伺服器分散各國，若想查封盜版網站，常需與他國司法互助，然而與台灣簽署司法互助國家少之又少。他說：「我們提出司法互助給這些國家，通常他們都不會搭理我們，因為對他們來說，只是多了更多事。」（待續）
