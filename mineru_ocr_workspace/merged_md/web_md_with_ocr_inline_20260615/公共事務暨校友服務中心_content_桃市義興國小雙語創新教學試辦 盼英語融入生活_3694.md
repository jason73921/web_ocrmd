# 網站文章資料_桃市義興國小雙語創新教學試辦 盼英語融入生活_3694

## 一、文章基本資料

- **文章ID**: 3694
- **文章標題**: 桃市義興國小雙語創新教學試辦 盼英語融入生活
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20190429
- **文章簡介**: 【大紀元報/記者陳建霖/桃園報導】

桃園自106學年度起，推動「雙語創新教學計畫試辦學校」計畫，選擇義興、快樂、內壢、青埔國小和青埔國中等5所學校，投入約1,978萬元外師經費，擴大外師聘任名額，補助國中小雙語學校約309萬元，提升校內軟硬體設施，進行雙語試辦教學。

其中義興、快樂國小1至2年級及內壢、青埔國小1年級都至少有三分之一課程進行雙語教學，桃園市長鄭文燦26日視察的義興國小更是透過中文、英語、程式語三種語言學習，結合人工智慧、機器人教育、科技教育、程式教育、圖書館智慧教育等多元課程，巧妙將英語融入生活和教學中，發展成為自然語言，讓學生敢勇於開口說英語，落實雙語教學向下扎根目標。

鄭文燦也說，桃園各級學校規劃許多創新教學課程，元智大學規劃三分之一課程使用全英文教學，也與美國麻省理工學院合作辦理「MIT-YZU科技文化英語體驗營」，提供全程英語營隊活動，讓學生在英語環境下更有效率的學習語言；此外，平鎮國中也設立南桃園首座自造教育及科技中心，發展多元科技教育，提供學生更創新且新穎的教學課程。（待續）
