# 2020 第十四屆波蘭國際發明展暨發明競賽

# 參展辦法

『波蘭國際發明展』為經濟部智慧財產局公告認定著名國際發明展，是波蘭唯一官方所舉辦的發明展，由波蘭總統授權，波蘭專利局、高等教育部，以及波蘭發明協會等共同舉辦，同時也是屬於IFIA國際發明聯盟總會公告之國際發明展，敬邀各界發明先進參加本展覽，本次展覽因應全球新冠肺炎(COVID-19)疫情，改為線上辦理。

一、主辦單位：台灣國際發明得獎協會、中華創新發明學會、國際創新發明聯盟總會

二、展覽名稱：2020第十四屆波蘭國際發明展

三、活動地點：波蘭華沙科技大學

四、成績公告：2020年10月 21日（三）

五、報名日期：即日起至2020年 8月31日（一）前截止

[1] 請下載中英文報名表、摘要(為word文件,勿轉成其他格式)，填妥後回傳至報名信箱：choice@mail2000.com.tw
