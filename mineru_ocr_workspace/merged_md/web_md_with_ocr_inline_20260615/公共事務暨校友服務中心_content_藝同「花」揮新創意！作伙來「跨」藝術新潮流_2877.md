# 網站文章資料_藝同「花」揮新創意！作伙來「跨」藝術新潮流_2877

## 一、文章基本資料

- **文章ID**: 2877
- **文章標題**: 藝同「花」揮新創意！作伙來「跨」藝術新潮流
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20180521
- **文章簡介**: 【勁報記者張亞痕/臺中報導】

一場跨地域、跨校際、跨領域的藝術輕旅行「2018藝術新聲」展覽，由11校13系全國藝術科系新秀自即日起至6月6日，在大墩文化中心大墩藝廊（一）至（六）及動力空間7個展場隆重登場，看臺灣新生藝術家如何引領當代藝術新潮流！

20日的展覽開幕式現場由臺中市副市長張光瑤、文化局副局長施純福、總策展人陳一凡教授，以及國立臺灣美術館館長蕭宗煌、中華民國畫廊協會理事長鍾經新等貴賓，為本次展覽儀式以灌溉的特殊方式共同創作「2018藝術新聲之芽」。

張光瑤指出，多元化是藝術的最大特色，藉由藝術新聲的展覽平臺，可以看到越來越多藝術設計與電子產品結合的面貌，藝術的美感為當代科技產品加值；而現代人的生活藉藝術的涵養而豐富。文化局亦表示，透過灌溉「藝術之芽」的開幕儀式，象徵藉由「藝術新聲」的展覽平臺，產、官、學界一起支持這些年輕學子們為臺灣藝術界的未來注入嶄新的生命力並成長茁壯！

總策展人陳一凡指出，「藝術新聲」是一窺臺灣新生藝術家創作的最好切片，「2018藝術新聲─11校13系畢業生推薦展」，展覽規模逐年擴展，從原本的7校7系，迄今已擴大至11校13系。

而為鼓勵當代藝術的自由、多元與開放，促進青年學子的藝術創作，臺中市政府文化局大墩文化中心特別匯集全國由北而南的各大藝術校系，在此展現年輕新秀的藝術創新理念，包括了：臺北藝術大學美術學系及新媒體藝術學系、臺灣藝術大學美術學系及雕塑學系、臺灣師範大學美術學系、元智大學藝術與設計學系、清華大學藝術與設計學系、東海大學美術學系、臺中教育大學美術學系、彰化師範大學美術學系、嘉義大學視覺藝術學系、臺南藝術大學材質創作與設計系、以及高雄師範大學美術學系等。(待續)
