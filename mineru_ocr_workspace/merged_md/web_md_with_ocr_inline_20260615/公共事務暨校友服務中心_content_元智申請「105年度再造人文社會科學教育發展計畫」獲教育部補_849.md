# 網站文章資料_元智申請「105年度再造人文社會科學教育發展計畫」獲教育部補助_849

## 一、文章基本資料

- **文章ID**: 849
- **文章標題**: 元智申請「105年度再造人文社會科學教育發展計畫」獲教育部補助
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【中央社】

元智大學向教育部申請「105年度再造人文社會科學教育發展計畫」，管理學院以「提升全球移動力、培育學生就業力」、人文社會學院以「「強化社會實踐力,重現人文社會教育價值」再造規劃案」， 2月26日經教育部核定通過，並分別各獲180萬元補助，為所有申請補助次高的學校。 

元智大學管理學院以「提升全球移動力、培育學生就業力」為主題，獲180萬元補助，元智管理學院院長曾芳美表示，此計畫以學生學習為主體，設計社會企業與產業整合課程，並藉由國際認證、企業合作、產業互動、海外交流、專題競賽等，爰引國內外學校及廠商資源，將學生培養成產業愛用人才及社會菁英。 


曾芳美表示，元智管院具備AACSB認證，國際化之推展成效卓著，並運用此優勢洽談學術合作、締結聯盟與國際交流；104年更延攬美國傑出學者Fred加入以充實研究能量。結合實務方面，則與遠東集團及國內外台商合作之實習課程、企業參訪、就業媒合與職涯發展諮詢等，並積極邀請企業高階主管演講及授課。此次獲得教育部補助，亦可展現管院在提升國際化與學用合一之努力獲得肯定。 

元智大學人文社會學院以「強化社會實踐力，重返人文社會教育價值」為主題獲180萬元補助，人社院院長洪泉湖表示，在強化社會實踐力方面，本院各系所在教學、研究與服務方面將朝向社會、社區、地方政府、企業、國際，以加強學術與社會之接軌。在重返人文社會教育價值方面，本院各系所將走出象牙塔，在課程設計、教學方法與師資聘用上將考量社會需求與市場趨勢。 


洪泉湖說，此計劃預期使人社院師生的社會實踐能力大為提升，藉由教師帶領學生面向社會，走入社區，邁向國際，結合在地產業與地方政府，預期將提升桃園地區之能見度、參與度與貢獻度。
