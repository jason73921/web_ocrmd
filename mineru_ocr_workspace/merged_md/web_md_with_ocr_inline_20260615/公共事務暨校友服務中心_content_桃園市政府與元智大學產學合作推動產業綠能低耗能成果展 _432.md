# 網站文章資料_桃園市政府與元智大學產學合作推動產業綠能低耗能成果展 _432

## 一、文章基本資料

- **文章ID**: 432
- **文章標題**: 桃園市政府與元智大學產學合作推動產業綠能低耗能成果展 
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【工商即時/校園報導/記者傅秉祥/桃園報導】

元智大學於今日(12/16)下午2:30，假桃園市政府13樓公務人力培訓中心將舉辦『104年度桃園市綠能產業推動計畫成果發表會』，希望藉由綠(節)能產業的發展，可以減少對能源的依賴、降低二氧化碳排放、進而提升產品競爭力與企業形象。

桃園市政府為推動市內產業朝三低一高(低汙染、低耗能、低用水及高附加價值)發展，邀請元智大學燃料電池中心承辦「桃園市綠能產業技術服務團隊」，協助桃園市廠商提升綠能關鍵技術、建立自主化節能管理能力、與節能(水) (或環保)標章申請。以奠定永續發展基礎，發展更具智慧化、效率化、科技化的產業與產能。

元智大學機械系教授，亦是綠能團隊團長鐘國濱表示，「桃園市綠能產業技術服務團隊」歷經六個多月的計畫執行，進行30餘家廠商的輔導案例並完成25家輔導工作。接受綠能關鍵技術提升輔導業者共8件、接受節水節能技術輔導共12件、接受綠色產品/綠色工廠認證輔導共5件(其中1家重新取得環保標章)。

今日成果展之主席-桃園市經發局長 張昌財博士表示，根據團隊盤查桃園市內七大類綠能產業計41家業者。其中氫能與燃料電池業者6家(22.22%)、電動車輛業者9家(23.68%)、能源資通訊業者7家(22.86%)，高於其他綠能產業之占比率(桃園/台灣) 顯示桃園市內綠能產業之豐沛動能。其中，元智大學已初步整合氫能燃料電池與電動車輛業者，分別發展以華德動能公司為首之『氫能燃料電池增程交通運輸工具聯盟』與富堡能源公司為首之『先進能源增程無人飛機產學研發聯盟』，進行技術之上下游垂直整合與跨技術之平行整合，以促成具規模之綠能產業與行銷國際市場為發展方向，華德動能公司之蔡篤雄董事長與富堡能源公司之陳嘉鴻總經理將於今日進行輔導成果分享。

桃園市之工業產值超過新竹科學園區，且多數為高耗能(水)之傳統產業。面對未來國際間更嚴格之低碳生產製程要求，桃園市府更將致力於節能製程(商品)之推動。因此，今日成果展之亮點將是接受綠能團輔導之優良節能技術業者-台灣昶電股份有限公司致贈節電商品給桃園市政府，預計為裝設之空調系統節電五成以上，並藉由綠能團推廣至市內業者，為桃園市產業之綠色發展盡最大努力。



桃園市經發局長張昌財博士(左)、元智大學機械系教授兼綠能團隊團長鐘國濱(中) ，一起致贈獎牌給廠商台灣昶電科技公司。
