# 網站文章資料_尤努斯獎創新創業競賽廣獲國內外學子參賽 3組隊伍脫穎而出_1082

## 一、文章基本資料

- **文章ID**: 1082
- **文章標題**: 尤努斯獎創新創業競賽廣獲國內外學子參賽 3組隊伍脫穎而出
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【勁報/記者羅蔚舟/新竹報導】 


首屆尤努斯獎-社會創新與創業競賽，吸引包含伊利諾大學、哥倫比亞大學等國內外55所大專院校、逾200位青年學子組隊參賽，時興的新住民、青年投入社企及資源回收等議題，成為最受青睞的創業組簡報主題，最後由「衣想世界」等3組隊伍脫穎而出，除抱走2萬元獎金，並有機會於8月25日從尤努斯博士手中拿到最高15萬元獎金。 


由勞動部勞動力發展署桃竹苗分署與國立中央大學尤努斯社會企業中心合作辦理的「2016社會企業發展與管理國際研討會」暨「尤努斯獎：第一屆社會創新與創業競賽」在中央大學盛大登場，立委余宛如、中央大學校長周景揚、財團法人尤努斯基金會董事長蔡慧玲、財團法人商業發展研究院副院長王建彬、台灣三星股份有限公司總監洪悅容等關心社企發展的專家學者、工作者與青年朋友等各界人士逾300人與會。 


勞動部勞動力發展署桃竹苗分署長丁玉珍表示，桃竹苗分署近年來不斷致力於跨領域整合產官學研多元資源，落實推動在地社會企業，鼓勵青年學子以行動參與社會議題、關心社會問題。本競賽於4月1日至5月14日進行徵件，社會創新服務組與社會企業創業組等兩組共計有63組隊伍報名參加，包含知名學府國立台灣大學、國立交通大學以及國際知名的哥倫比亞大學、伊利諾大學等海內外學校。 


其中深受大專院校青年學子高度關注的「尤努斯獎：第一屆社會創新與創業競賽」，來自各大專院校的16組晉級青年團隊在競賽中充分展現出對社會創新服務與社會企業創業議題的蓬勃活力與熱忱，獲得「社會創新服務組」優勝的2組團隊是代表東海大學的「無與輪比」與由中央大學同學組成的「Herbalive」，各獲得2萬元獎金。 


至於晉級「社會企業創業組」決賽的3組團隊分別是由金門大學、中山醫大、中山大學等同學組成的「衣想世界」，以及實踐大學代表隊「Global Scavengers」，第三組入圍的是代表元智大學的「幫 TRUE」，3組團隊除可分別獲得2萬元晉級獎金外，也取得了由桃竹苗分署提供的冠軍15萬獎金、亞軍8萬獎金及季軍5萬獎金的8月25日決賽門票，屆時尤努斯博士也將來台親自評選出最終冠軍隊伍並頒獎。
