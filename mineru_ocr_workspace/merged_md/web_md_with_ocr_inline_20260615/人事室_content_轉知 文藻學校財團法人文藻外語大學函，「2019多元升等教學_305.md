# 網站文章資料_轉知 文藻學校財團法人文藻外語大學函，「2019多元升等教學實踐暨技術應用報告研討會 」徵稿簡章_305

## 一、文章基本資料

- **文章ID**: 305
- **文章標題**: 轉知 文藻學校財團法人文藻外語大學函，「2019多元升等教學實踐暨技術應用報告研討會 」徵稿簡章
- **文章網址**: https://www.yzu.edu.tw/admin/ho/index.php/tw/?option=com_content&view=article&id=305
- **文章簡介**: 文藻學校財團法人文藻外語大學　函

 主旨：檢送本校「2019多元升等教學實踐暨技術應用報告研討會 」徵稿簡章，敬請師生踴躍投稿參與。

說明：

一、配合教育部「推動教師多元升等制度重點學校計畫」，為 引導教師專業多元分工，關注教學現場，進而提升教學品 質，促進學生有效學習，並落實大學社會責任，對在地區 域或社會投入教研能量，達成高等教育深耕之目標，特辦 理本研討會提供教學實踐研究與技術應用創新研發成果發表場域。

二、徵稿辦法：

(一)本研討會採簡版論文審查，稿件將邀請相關領域專家學 者進行審查，詳見徵稿簡章。

(二)投稿者請將投稿論文(簡版論文)、投稿基本資料表、著 作授權書，108年3月31日24時前，以電子郵件寄100213 @mail.wzu.wdu.tw，信件主旨標示：投稿「2019多元升 等教學實踐暨技術應用報告研討會」。

(三)完成投稿程序者將會收到確認信件，審查結果將於108 年4月15日以E-mail方式通知。

(四)論文通過審查，於期限內繳交全文並於108年6月14日研 討會上完成口頭發表者，將於研討會後由本校集結出版 論文集(具ISBN書號)。

(五)為促進教學實踐相關研究風氣與獎勵優秀論文，特舉辦 優秀論文徵選，以拔擢優秀研究人才、發揮教學創新與 落實應用價值，其上述收錄之論文，將擇優推薦，經推 薦論文投稿於本校「語言與國際研究期刊」，可加速審 稿程序。

三、徵稿主題：

(一)創新教學與課程設計

(二)教學媒體與科技應用

(三)高等教育學生學習成效與評量工具相關研究

(四)高等教育產學應用之連結

(五)校務研究(如學生學習成效、學生就業與職能發展、招 生相關議題)

(六)跨文化相關之研究

(七)其他教學實踐相關議題

四、重要日期：

(一)論文投稿截止日期：108年3月31日（日）

(二)論文審查通知日期：108年4月15日（一）

(三)論文全文繳交截止日期：108年5月24日（五）

(四)研討會舉行日期：108年6月14日（五）
