# 網站文章資料_元智攜手亞台青 百萬獎金大賽助力創業_1815

## 一、文章基本資料

- **文章ID**: 1815
- **文章標題**: 元智攜手亞台青 百萬獎金大賽助力創業
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【中時電子報/胡恆睿】 

元智大學與亞台青（成都）海峽青年創業園，4月27日將於桃園市元智大學校區舉辦「兩岸青年創新創業大賽」，除提供百萬創業獎金外，另提供免機票、食宿的5天成都市創業者交流之旅，比賽報名將於4月10日截止。

今年，賽事設元智大學、成都2個分賽區。各個賽區決出優秀隊伍，將於2017年5月23日至27日在成都市參加大賽總決賽。亞台青（成都）海峽青年創業園總經理郭弘揚表示，成都不但是西部特大城市，鐵路貿易對接歐洲，目前又成為四川自貿區中心，市場消費潛力大，產業發展前景好。希望參賽者可以透過參與這次大賽，享受到元智大學與亞台青創業園正規的兩岸創業孵化專業機構的輔導，將是一個很好的創業助力。
