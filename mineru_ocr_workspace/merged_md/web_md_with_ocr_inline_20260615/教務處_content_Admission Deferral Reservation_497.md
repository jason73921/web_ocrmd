- **文章簡介**: Application flow






1. Only new students could apply and need to complete all procedures before registration.

2. Download Application Form of YZU Admission Postponement

3. After all procedures are completed, return to the union service counter.

4. Apply for admission deferral reservation must through the Academic Affairs Meeting. You still have to go through the register if the application is unsure to be passed. (After the application passed, would reimburse the tuition.)
