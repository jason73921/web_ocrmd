# 網站文章資料_鄭文燦：桃園鐵路地下化進入實質規劃階段_2128

## 一、文章基本資料

- **文章ID**: 2128
- **文章標題**: 鄭文燦：桃園鐵路地下化進入實質規劃階段
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20170810
- **文章簡介**: 【MyGoNews林湘慈/綜合報導】

桃園市長鄭文燦2017年8月9日主持市政會議時表示，桃園鐵路地下化可行性研究報告於7月31日獲行政院核定，即將進入綜合規劃及工程設計等實質設計階段，總經費964億元，其中市府負擔229億元、自償性經費為181億元，主體工程由交通部所屬單位辦理，市府則負責周邊都市發展，鐵路地下化預定2025年完成。市府會做好各項準備工作，也會積極向中央爭取加速核定綜合規劃及施工期程，減少市民在施工期間的不便。

鄭文燦也說，其他車站也各具特色，包括內壢站的開發將結合內壢北營區計畫，發展為「內壢之心」，完善內壢的生活圈；中原站將結合中原營區的開發，推動「中原創業村」，讓中原大學周邊形成新的文創聚落；平鎮站將建置轉運中心，交通會更方便；永豐站則結合周邊包括元智大學、衛福部桃園醫院等，發展成為桃園與中壢間新的都市核心；中路站鄰接中路地區，也會配合建設更多公共設施。鄭文燦責成都發局及交通局，及早做好都市計畫的設計規劃。(待續)
