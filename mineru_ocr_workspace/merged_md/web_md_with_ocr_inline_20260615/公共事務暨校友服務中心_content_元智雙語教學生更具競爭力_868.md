# 網站文章資料_元智雙語教學生更具競爭力_868

## 一、文章基本資料

- **文章ID**: 868
- **文章標題**: 元智雙語教學生更具競爭力
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【中國時報/余祥/台北報導】

政治大學規定新聘教師需要開設英語授課課程，引發教授反彈；反觀元智大學目前有3分之1的學程都以英語授課，校長吳志揚表示，英語已經不再是美國的英語，而是世界的英語，學好英文絕對是通往國際化的第一步，教育部甚至應該提供經費讓大學廣設全英文專班，網羅世界一流人才來台就讀。

元智大學十多年前就開始循序漸進推動英語授課，對於政大擬開設更多英文課程，吳志揚表示，非常樂觀其成，不然元智很孤單，開設英語課程不但有利於招募更多的國際生，打開台灣的大門，也能讓學生的英文能力提升，降低學生與世界溝通的阻礙，未來更有競爭力。

吳志揚說，元智管理學院的各系，因基礎科目同質性高，因此低年級時就可共同開設全英文課程，到了高年級再分流，其他大學也可視各院所性質，進行英語課程的整合，元智的經驗可提供其他大學參考。

以英文授課的元大組織學群教授翁晶晶說，因為校園雙語教學環境佳，新進老師自然而然就能融入這個氛圍，且聘請老師前也都有英語面試，因此老師多半對開設英語課程有基本認知，較不會造成反彈。

翁晶晶同時分享自身的教學經驗，她說，全英語課程不但能夠加強學生的英語能力，還能在課程中帶入國際觀和視野，因此她認為英文是很好的工具，讓學生在課程中訓練出國際化的視野。

另外，開設英文課程能吸引外籍生就讀，還能產生「鯰魚效應」，激起台生發言和學習的意願。許多學生都回饋說，原本對英文很恐懼，但一學期後也覺得開口說英語是很正常的事了。 

另位也是以英語授課的元智國際企業學群助理教授翁華鴻則認為，對學生而言，最困難的應該是要記得英文的專有名詞，但他都會告訴學生，未來若希望出國工作或是從事與外國客戶接觸的相關工作，都必須要懂得這些專有名詞，甚至能與人對答如流，這是必要的功夫、必經的努力。
