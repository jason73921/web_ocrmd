- **文章全文**: 邀請元智在地學生及來自中國大陸姊妹校的140餘位同學參加。140餘位學生來自不同地區，分別有大連理工大學、河南大學、華中師範大學、貴州大學、貴州師範大學、蘭州交通大學、興義民族師範學院、泉州師範學院等。

本次活動主題以創新創業為主軸，透過講座進行探討四大主題，包括領導力、創新力、凝聚力、執行力，課程安排三位管理學院教授、三位創業中的青年創業家進行分享，加強學生創業的新觀念與想法，開啟對於未來創業夢的一小步。

簡忠仁教授的組織心理學
講座邀請管理學院吳相勳教授、劉悅新教授、簡忠仁教授為前三場專業課程，透過個案分析、創業觀念、組織心理學等專業教材，提供學生創業的不同面向與觀念，後三場講座分別邀請幻象設計黃力恩總監、Like it Formosa 步行英語導覽鍾炘儒執行長、校友Youtuber JR LEE RADIO進行分享。由創業家分享自身經歷、挫折、心路歷程，更刺激參加講座同學們對於自身喜好、定位、熱情的反思與展望。

校友JR LEE RADIO的線上分享
Like it Formosa 步行英語導覽鍾炘儒執行長
