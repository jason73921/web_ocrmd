# 網站文章資料_中華大學『2019華大黃金蟲草創新創業競賽』_2337

## 一、文章基本資料

- **文章ID**: 2337
- **文章標題**: 中華大學『2019華大黃金蟲草創新創業競賽』
- **文章網址**: https://www.yzu.edu.tw/admin/rd/index.php/tw/?option=com_content&view=article&id=2337
- **文章簡介**: 主旨：檢送本校創新創業中心舉辦『2019華大黃金蟲草創新創業競賽』，歡迎貴校師生報名參加，請查照。
說明：
一、為推廣黃金蟲草商業應用，期盼透過校園團隊間相互交流切磋，聚焦黃金蟲草之創意商品開發及微型創業發展，開啟同學們的靈活創意思維、激發創新應用能力及培養創業家精神，特辦理本屆「2019華大黃金蟲草創新創業競賽」。
二、競賽報名日期自即日起至108年4月19日(五)下午5時，開放線上填寫報名表單(https://goo.gl/forms/CuHRUkPelQqdewGV2)與繳交紙本報名表。
三、108年4月26日(五)下午五點前將公布入圍決賽團隊於本校創新創業中心官網http://ec-sos.chu.edu.tw/。
四、108年5月8日(三)「截止繳交決賽簡報檔案」，請於下午5點前寄到活動聯絡人信箱，E-mail：chu.ecsos@gmail.com，逾期恕不受理。
五、108年5月15日(三)「辦理競賽決賽」，活動流程與地點將公告於創新創業中心官網。
六、寄送地址：30012新竹市香山區五福路二段707號(中華大學創新創業中心 陳俐榮專員收，專線：03-518-6693)。
七、檢附相關附件，敬請貴校師生踴躍報名參加。
