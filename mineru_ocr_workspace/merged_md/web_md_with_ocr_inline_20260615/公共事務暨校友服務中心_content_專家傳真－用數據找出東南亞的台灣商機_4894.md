# 網站文章資料_專家傳真－用數據找出東南亞的台灣商機_4894

## 一、文章基本資料

- **文章ID**: 4894
- **文章標題**: 專家傳真－用數據找出東南亞的台灣商機
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20210127
- **文章簡介**: 【工商時報】

東南亞市場快速成長，是未來全球經濟成長的主要引擎之一，不少台商響應政府的新南向政策，對於東南亞市場躍躍欲試，其中，許多是採用跨境電子商務的模式，也就是以速度較快，成本與風險相對較低的方式進入東南亞零售市場。

跨境電商市場策略的第一步是決定好主要經營的市場與選擇產品，也就是找出符合自己優勢的「選品策略」。那麼，究竟在東南亞各國電商市場，有哪些台灣產品是熱賣暢銷的？月銷量可以達到多少？如果對這個問題沒有概念，也不曾進行研究，那就是缺乏行銷的知識，成功的機率就不高。在多年來的廠商輔導與訓練課程中，我們發現這是許多業者的通病。

電子商務講求數據分析，選品正是從數據分析開始，就讓我們以在東南亞各國領先的蝦皮網站之新加坡站為例，採用能夠分析各國市場的「蝦皮分析神器」去採集數據，找出以Taiwan為產品名稱上架銷售，歷史銷售件數最多的前五項商品的名稱，歷史銷量與最近一個月的銷售額以及轉化率等重要情報，藉此，一探台灣產品的競爭力。查詢時間是在2021年1月23日，在公布結果之前，先邀請您花個一分鐘想想，您覺得賣最好的會是那些產品？為什麼？先考考自己的商業判斷敏銳度以及對東南亞市場的了解。（作者：朱訓麒元智大學助理教授）（待續）
