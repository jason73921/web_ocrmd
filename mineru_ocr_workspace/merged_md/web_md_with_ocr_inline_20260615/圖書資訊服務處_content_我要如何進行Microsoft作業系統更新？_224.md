## 自動更新登錄檔下载：

1.請邀挥下列之更新時間，例如邀挥08:00，则系統就會方

## 請點選一個電腦有開機時間

2.下载方式請選播下列任一時間，按右键另存新檔，將下下執行，重開概以後即可自動於排定的時間更新。

3.依照設定，每隔12小時将會與何服器比對有無最新的更新，開概3分遍衡也會與饲服器比對最新的更新。

4.此排程時間無法重複擇，系统将會以最後一次執行登檔之中定的時間為準。

5.請透挥電题會開碳的时間，並安装所有要求更新之项目。

![](images/7106ee3f1289dacaff192eca1ca0ade4306798dffc18d22c913f63aa3950c7fb.jpg)

## 無法自動更新作業系统者，請容試先安装以下雨们更新誉：

## Windows Server Update Services

## 使用须知：

1.凡於校内的使用者替可以使用本服務。

2.請使用者加入下列的系统登檔，以便使用自動更新。

3.本自勤更新服務通用於繁體中文、简體中文、日文、韓文、英文版之Windows(2000、XP、2003、Vista）、Exchang(2000、2003、2007）、Oe全系列（Oo2003之前的版本：如Oce2000、OffceXP需在更新时放入原始安装光碟。）

4.其Windows版本，如Windows98、WindowsMe，建議使用微軟WindowsUpdate進行更新。
