# 網站文章資料_國立成功大學「2018年維生基礎設施（交通系統）領域氣候變遷產業交流研討會」_1975

## 一、文章基本資料

- **文章ID**: 1975
- **文章標題**: 國立成功大學「2018年維生基礎設施（交通系統）領域氣候變遷產業交流研討會」
- **文章網址**: 
- **文章簡介**: 說明：1.活動介紹

為強化交通運輸產業掌握氣候變遷調適作法與技術，本次研討會邀集國內交通領域產官學研界單位等相關人士共同討論，進行經驗分享與雙向回應交流，激盪各界先進應用理念吸引跨領域先進活絡投入，評估我國交通系統(公鐵水空)因應氣候變遷調適措施可行性並配合實際案例研析，以實踐氣候變遷調適專業行動策略，推動綠色永續運輸之願景與提升氣候變遷的知能。
