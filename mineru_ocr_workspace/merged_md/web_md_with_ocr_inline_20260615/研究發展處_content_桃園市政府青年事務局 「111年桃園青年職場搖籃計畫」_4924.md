# 網站文章資料_桃園市政府青年事務局 「111年桃園青年職場搖籃計畫」_4924

## 一、文章基本資料

- **文章ID**: 4924
- **文章標題**: 桃園市政府青年事務局 「111年桃園青年職場搖籃計畫」
- **文章網址**: https://www.yzu.edu.tw/admin/rd/index.php/tw/?option=com_content&view=article&id=4924
- **文章簡介**: 主旨：有關本局辦理「111年桃園青年職場搖籃計畫」（下稱本計畫），請貴校育成中心協助宣傳週知，請查照。

說明：

一、為協助優秀青年人才與桃園在地企業有效對接，本局辦理 「111年桃園青年職場搖籃計畫」，邀請100家在地企業， 提供600個以上職缺數額，並輔以職涯發展相關資源，期透過本計畫有效提升青年競爭力，順利接軌職場。

二、本計畫適用對象為18歲至35歲之青年族群，不限戶籍地點，計畫內容如下：

       （一）職涯輔導：本計畫提供多種多樣職涯輔導資源，包含職涯課程、職場體驗、專業培力課程、職涯導航、履歷診 療等，協助青年完成就業準備。

       （二）就業媒合：青年透過計畫網站可投遞履歷申請面試，經正職150小時或兼職100小時以上之實務訓練後，將有轉正機會。

三、為利求職青年瞭解旨揭計畫，請貴校協助將計畫相關資訊置於所屬網站及社群平臺，或以電子郵件等方式轉知貴校畢業青年知悉，計畫網站：https://reurl.cc/XVrKRg。

四、隨函檢送旨揭計畫相關宣傳資料一份，本計畫A1宣傳海報將另行寄送，再請貴校協助張貼宣傳。
