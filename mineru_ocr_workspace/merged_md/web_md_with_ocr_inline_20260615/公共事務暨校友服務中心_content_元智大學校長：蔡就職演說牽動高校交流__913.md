# 網站文章資料_元智大學校長：蔡就職演說牽動高校交流	_913

## 一、文章基本資料

- **文章ID**: 913
- **文章標題**: 元智大學校長：蔡就職演說牽動高校交流	
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【中評社/記者黃文杰報導】 


元智大學吳志揚校長(左)與西安電子科技大學鄭曉靜校長(左)共同簽署學生交流專案協定書。(照片：私立元智大學提供)

中評社桃園4月11日電（記者 黃文杰）元智大學校長吳志揚接受中評社訪問表示，這次利用四月春假走了一趟西安，拜訪大陸姊妹校也參加西安交通大學120周年校慶，不論台灣或大陸高校校長普遍都在關注，事實上也都在等民進黨“總統”當選人蔡英文520就職演說以及未來新政府政策方向，這可能牽動兩岸的高校交流。 

元智大學由校長吳志揚率團，於4月初春假期間拜訪大陸西安姊妹校如西北大學、西北工業大學與西安電子科技大學，其中西北大學是“211工程”重點建設，被譽為“中華石油英才之母”、“經濟學家的搖籃”、“作家的搖籃”，西安電子科技大學（簡稱西電）也與元智關係密切，因緣於元智電機通訊學院與西電通信工程學院合作已久。 


吳志揚與西北大學校長郭立宏共同簽署雙聯學制協定書，也與西安電子科技鄭曉靜校長簽署雙聯學制協定書，以本科生與碩士生開始進行雙聯學位，落實兩校合作細節，也來到西北工業大學，與西北工業大學校長汪勁松簽署校際合作框架協議之續約，雙方期盼能加深交流深度，相互學習與發展。 


最後一站出席西安交通大學120周年校慶活動，吳志揚來自台灣許多大學校長或代表一起致賀，彼此也討論未來兩岸高校交流。 


吳志揚接受中評社訪問表示，這次包括走訪元智在西安的姊妹校，基本上並未提到交換生或交流生政策是否改變，不過台灣高校或大陸高校都有關注，也都在等520之後新政府的政策方向。 


“大家都覺得擔心，只是擔心，尚未有明顯強大的氛圍，會讓兩岸高校交流速度放慢或中止”，他說。 


吳志揚表示，必須坦白說，很多大學校長都不知道520之後會怎樣，大陸高校也在等“上面”的氛圍，不敢隨便做出太大改變。 


他說，正因為大陸高校都在等中共中央最新的“出台”政策，大陸也知道台灣政情已經改變，很多事情“都在關注”。 


吳志揚說，兩岸經過這麼多年的互動，彼此有了信任，可以預期的是，民進黨政府也不會排斥民間，特別是兩岸高校繼續交流，只要政治上不會太大衝擊，基本上認為短期之間改變不大，也不會因為換了政府，就會改變兩岸政策。 


元智大學校長吳志揚(左)與西北工業大學汪勁松校長(右)共同簽署雙聯學制協定書。(照片：私立元智大學提供) 


元智大學校長吳志揚(左)與西北大學校長郭立宏(左)共同簽署雙聯學制協定書。(照片：私立元智大學提供) 


元智大學11日表示，校長吳志揚與西北大學校長郭立宏共同簽署雙聯學制協定書，吳志揚表示，未來與西北大學將加強雙方人才的培養，並有教師科研交流，以本科生與碩士生開始進行雙聯學位，以基礎和進階學分加總採認，以達成學分數轉換且符合畢業標準，落實兩校合作細節。 


西安電子科技(簡稱西電)是大陸地區最早設立資訊理論、資訊系統工程、雷達、微波天線、電子機械、電子對抗等專業科系的高校之一，成為大陸IT學科的啟蒙濫觴，領先群倫而以電子與資訊學科成為該校優勢特色。而元智的電機通訊在台灣也是熱門系所，並有遠東集團遠傳電信做為產學合作之合作夥伴。 


元智大學吳志揚校長與西電鄭曉靜校長簽署雙聯學制協定書，以本科生與碩士生開始進行雙聯學位，落實兩校合作細節。雙方校長皆認為兩岸高校擁有共同的文化底蘊和歷史傳承，在學生聯合培養方面極具優勢。元智校長吳志揚表示，西安電子科技大學與元智大學關係密切，因緣於元智電機通訊學院與西電通信工程學院合作已久，未來兩校在現有合作的基礎項目，將積極落實研究計劃，提高學生的交流能力，推動兩岸人才培養。 


另外，此行也來到西北工業大學，並與西北工業大學校長汪勁松簽署校際合作框架協議之續約，雙方期盼能加深交流深度，相互學習與發展，對於未來兩校的合作，雙方都拋出合作議題，加深合作關係與項目。
