# 網站文章資料_【資產盤點】113/3/22(五)前各單位同仁需完成資安資訊資產盤點工作_577

## 一、文章基本資料

- **文章ID**: 577
- **文章標題**: 【資產盤點】113/3/22(五)前各單位同仁需完成資安資訊資產盤點工作
- **文章網址**: https://www.yzu.edu.tw/admin/is/index.php/tw/1130314-1
- **文章簡介**: 配合「教育部高教深耕計畫-資安專章」執行，敬請各位單位同仁於3/22(五)前協助進行資安資訊資產盤點工作。

資訊資產目標包括：「陸製資產」、「IOT設備（定義為可連網或可進行資料存儲之設備）」及目前單位內擁有的「資訊系統」全面盤點。

【注意事項】


3月22日（週五）下班前完成填寫『盤點表單』（填報方式請詳見盤點表單之說明）。請仔細填寫相關資訊，以確保盤點工作的準確性。

資訊資產定義：

陸製資產&IOT設備：可以「連接網路」或「可進行資料存儲」之設備皆須進行盤點，包括但不限於辦公場域內之「個人電腦」、「筆記型電腦」、「多功能事務（印表）機」、「無線AP」、「路由器」、「Storage儲存設備(NAS、外接移動硬碟)」、「監視設備」。

資訊系統：貴單位於業務執行時所使用到之資訊系統（包含自主開發及外部採購）。

禁止使用及採購「大陸廠牌資通訊產品」相關規定
