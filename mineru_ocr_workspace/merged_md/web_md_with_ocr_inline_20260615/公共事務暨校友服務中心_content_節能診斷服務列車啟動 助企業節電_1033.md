# 網站文章資料_節能診斷服務列車啟動 助企業節電_1033

## 一、文章基本資料

- **文章ID**: 1033
- **文章標題**: 節能診斷服務列車啟動 助企業節電
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【經濟日報/李炎奇】 


高溫氣候不斷發生，全球暖化日益嚴重，以及可預期未來能源價格的上漲趨勢，企業營運成本與風險不斷增加，尤其對於我國為數高達百萬家的產業中小能源用戶，更是不小的衝擊。

為了協助產業中小用戶渡過能源難關，同時善用大專院校節能服務能量，經濟部能源局規劃設置節能診斷服務中心(Energy Diagnostic Center，簡稱EDC)，透過具節能專業的大專院校教授帶領學生方式，提供學校所在地區之產業中小用戶適切的節能診斷服務，除就近協助產業中小用戶節能降低成本外，也透過培養學生節能診斷實務經驗，縮短學用落差，培訓我國節能技術綠領人才。

經濟部能源局已於今(105)年5月遴選出11家大專院校，將於北、中、南、東各區設立11個節能診斷服務中心，包括台北科技大學、台灣科技大學、台灣大學、元智大學、中原大學、聯合大學、勤益科技大學、雲林科技大學、崑山科技大學、中山大學及宜蘭大學。

為進一步對國內產業中小用戶廣宣EDC成立訊息，經濟部能源局於6月1日配合105年度節能月活動，在台大醫院國際會議中心辦理EDC成立啟動儀式，未來將提供廣大中小能源用戶免費節能診斷服務，協助其降低成本與提高營運績效，提升中小企業的綠色競爭力。
