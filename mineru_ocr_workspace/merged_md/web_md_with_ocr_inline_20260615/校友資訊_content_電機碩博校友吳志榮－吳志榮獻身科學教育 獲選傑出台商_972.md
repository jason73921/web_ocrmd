# 網站文章資料_電機碩博校友吳志榮－吳志榮獻身科學教育 獲選傑出台商_972

## 一、文章基本資料

- **文章ID**: 972
- **文章標題**: 電機碩博校友吳志榮－吳志榮獻身科學教育 獲選傑出台商
- **文章網址**: https://www.yzu.edu.tw/alumni/index.php/tw/20230612
- **文章簡介**: 【工商時報 / 蘇崇愷】

大陸台商、武漢力翰科普發展有限公司董事長吳志榮，深耕大陸科學教育市場多年，不僅帶領力翰把業務拓展到大陸60座城市，更熱中參與公共服務，擔任武漢市台資企業協會副會長，協助台商、台青在大陸發展。吳志榮稍早憑藉亮眼的創業成績，榮獲第十屆傑出大陸台商獎。

出生於嘉義縣六腳鄉的農家子弟吳志榮，19歲考上公務員，在台北市教育局的社教機構負責小學科學教育工作12年，期間繼續深造取得元智大學電機博士學位。吳志榮在校期間擔任元智大學科學教育研究中心執行長，並於2007年孵化力翰，成為以科學教育為主的企業。

吳志榮表示，力翰目前的核心高層團隊成員，都來自當年的元智大學科學教育研究中心。力翰的定位是「科學普及」，客群是每一位小孩（該公司主要教學對象為小學生），這樣的市場最大，有別於傳統補習班教育模範生、考試提分的定位。

在吳志榮的帶領下，力翰發展出全方位、軟硬體兼具的科學教育業務。該公司不僅能派遣專業師資到校園授課，還能為學校培訓老師、開設科學教室，甚至是幫助學校展示科技課程、舉辦科學冬夏令營等。吳志榮指出，在多年努力下，已在大陸多位校長的心中留下「科學教育找力翰」的印象。

2021年7月，大陸推出教育「雙減」政策。吳志榮表示，「雙減」政策把學生從硬式教育中解放出來，對於從事素質教育的力翰而言影響不大。吳志榮表示，為響應該政策，力翰2022年成立旅行社，以因應兒童寒暑假走出戶外的需求，2023年大陸防疫解封，將是旅行社展開業務的第一年。

另一方面，大陸教育部近日印發「基礎教育課程教學改革深化行動方案」的通知，提出加強實驗教學，強化學生動手操作實驗等。對此，吳志榮表示，上述政策與力翰團隊長期推廣的工作完全契合，是一大利多。

展望未來，吳志榮表示，已經啟動建設1,000多坪的集團大樓，具有研發中心等功能，預計兩年後啟用。另外，力翰已經獲得兩岸科學教育市場的肯定，未來想要打破由歐美引領教育潮流的現狀，在大陸把力翰打造成具有「寶島」元素的世界級教育企業。

針對武漢市台資企業協會會務，吳志榮指出，疫情期間到當地發展的年輕人變少，希望能扭轉外界對於武漢疫情的刻板印象，讓台商、台青重拾信心。吳志榮以自身為例，指出兩岸同屬中華文化圈，市場很大，且武漢位於高鐵樞紐、交通便利，鼓勵台商和台青到當地發展。

逾20年科教經驗 成業務利器

武漢力翰科普發展有限公司董事長吳志榮分享創業三大心法，分別是專業技能、存在感、客情維護。吳志榮指出，自身擁有超過20年的科學教育經驗，能夠說服重視理論的各校校長。

針對存在感與客情維護，吳志榮表示，力翰透過科學秀、公開課，甚至在期刊上發表文章，展現出自家科學教育實力，進一步取得客戶信賴。此外，吳志榮指出，自己的微信加了許多位校長好友，透過主動分享，可以讓校長們更了解力翰。

吳志榮說：「對於學校而言，和教育業者合作的試錯成本高，因此學校會長期觀望，但力翰有信心、從容以對，學校往往和力翰合作一次就能建立信任感。」
