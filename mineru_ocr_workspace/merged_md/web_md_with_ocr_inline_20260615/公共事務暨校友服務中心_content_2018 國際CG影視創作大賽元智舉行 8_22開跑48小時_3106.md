# 網站文章資料_2018 國際CG影視創作大賽元智舉行 8/22開跑48小時3D Live動畫製作總決賽_3106

## 一、文章基本資料

- **文章ID**: 3106
- **文章標題**: 2018 國際CG影視創作大賽元智舉行 8/22開跑48小時3D Live動畫製作總決賽
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20180823
- **文章簡介**: 【中央社】

國內動畫軟體大廠甲尚科技(Reallusion　Inc.)與元智大學共同主辦的『ASIAGRAPH Reallusion Award 2018國際CG影視創作大賽』48小時總決賽，自8月22日開跑，來自美國、台灣、中國、日本、馬來西亞及印尼的15個決賽隊伍將匯聚元智大學，展開48小時Live總決賽，角逐30萬元首獎、二獎及三獎9萬元的高額獎金及Faceware及孚心科技(FOHEART)贊助的動態捕捉軟硬體。其中元智大學資訊傳播學系張世明教授指導下有兩隊進入決賽，為全球唯一入圍兩隊的參賽學校。

今年參與決賽的隊伍包括有知名國際影視學校-美國Georgia State University、台灣藝術大學、元智大學、亞洲大學等；中國南京藝術學院、吉林藝術學院學院；印尼Tarumanagara University 、Politeknik Negeri Media Kreatif Jakarta等，馬來西亞Universiti Teknologi MARA Shah Alam、Universiti Teknologi MARA Alor Gajah及日本名古屋工學院、名古屋學藝大學等。（待續）
