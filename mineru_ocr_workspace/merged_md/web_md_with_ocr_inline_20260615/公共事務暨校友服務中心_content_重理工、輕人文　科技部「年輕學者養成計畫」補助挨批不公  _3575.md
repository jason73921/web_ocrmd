# 網站文章資料_重理工、輕人文　科技部「年輕學者養成計畫」補助挨批不公  _3575

## 一、文章基本資料

- **文章ID**: 3575
- **文章標題**: 重理工、輕人文　科技部「年輕學者養成計畫」補助挨批不公  
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20190220-1
- **文章簡介**: 【旺報/李侑珊】

科技部補助研究計畫，長期被批評「重理工、輕人文」，世新大學社會心理系主任詹昭能近日統計，在107年度「年輕學者養成計畫」共78件研究案中，在「哥倫布計畫」共計29件研究案中，人文社會領域僅2件，占6.89％，49件「愛因斯坦培植計畫」則只有7件人文社會領域研究通過，占14.29％，且私立大學或申請僅兩校，同時還存在「重公立、輕私立」的問題。

詹昭能指出，打開「年輕學者養成計畫」檢定清冊，若以計畫名稱來看，依舊以理工類為最大宗，若以比較寬鬆的標準而論，屬於人文社會的研究案有9件。

詹昭能認為，像是在「愛因斯坦培植計畫」當中的研究案，像是：｢可解釋的人工智慧細菌通訊在行動邊緣運算｣、｢解放智慧型手錶：利用豐富觸覺回饋增進智慧型手錶之多媒體體驗｣、｢邁向研究資料基礎建設：資料共享實務與資料之學術影響力探索｣、｢氣候變遷下未來城市之洪水韌性評估與建構-一個跨領域之整合性時空模擬架構｣等4案，看似與社會研究相關，卻很難歸屬於人文社會計畫，若加以排除，總計78件計畫僅有4件屬之，僅占5.13％。

詹昭能提到，科技部補助研究計畫不只存在「重理工、輕人文」的問題，「還明目張膽的『重公立、輕私立』｣，其中僅有兩件來自於私立大學，都屬於「愛因斯坦培植計畫」，分別是元智大學電機工程學系（所）助理教授彭朋瑞、長庚大學資管系助理教授曾意儒，詹反問：「私立大學年輕學者真的這麼不成材嗎？｣(待續）
