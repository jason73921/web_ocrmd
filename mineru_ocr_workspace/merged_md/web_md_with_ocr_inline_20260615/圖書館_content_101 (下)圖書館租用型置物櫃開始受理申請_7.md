# 網站文章資料_101 (下)圖書館租用型置物櫃開始受理申請_7

## 一、文章基本資料

- **文章ID**: 7
- **文章標題**: 101 (下)圖書館租用型置物櫃開始受理申請
- **文章網址**: https://www.yzu.edu.tw/library/index.php/tw/101 (下)圖書館租用型置物櫃開始受理申請
- **文章簡介**: 
- **文章全文**: 因應同學需求，圖書館將於下學期起增設置物櫃，預計將提供96個租用櫃，申請說明如下：

1. 租賃對象：校內學生。

2. 租用費用：以一學期計價，每一置物櫃收費450元。

3. 使用期限：本學期租期為102.2.1~102.7.15。

4. 申請方式：

 採登記制，於102.1.15前洽圖書館流通櫃台登記。

 於102.1.16抽簽決定名單與櫃位。

 於102.1.21前完成繳費；逾期未繳費視同放棄，由後補名單遞補。

5. 有意申請者，請速洽圖書館流通櫃台王小姐(分機2321)辦理。
