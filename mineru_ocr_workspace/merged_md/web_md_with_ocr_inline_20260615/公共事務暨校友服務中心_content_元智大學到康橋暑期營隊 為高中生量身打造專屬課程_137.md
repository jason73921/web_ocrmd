# 網站文章資料_元智大學到康橋暑期營隊 為高中生量身打造專屬課程_137

## 一、文章基本資料

- **文章ID**: 137
- **文章標題**: 元智大學到康橋暑期營隊 為高中生量身打造專屬課程
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【中央社】

元智大學管理學院大學部受康橋高中邀請，為康橋高中學生量身打造大學基礎課程，於今年7月中旬暑假期間舉辦為期一周的「2015決戰商場」暑期營隊，由元智大學管理學院副院長李弘暉帶隊，安排元智大學商管各領域菁英教師親自前往新北市新店區康橋高中授課，讓高中學生提早接觸大學的課程，確認未來生涯方向，更規劃讓學員於課程中發現，商業與管理並非生硬的理論，幫助康橋學生探索商管學程核心知識。 

元智大學管理學院係全台唯一以「主修」替「學系」，與世界一流大學同步課程設計，並因應國際社會產業趨勢，提供學生更多元、具特色及競爭力之學程課程。元智大學於管理領域之專業，受到多項國際獎項肯定，此次與新北市康橋高中合作，透過元智大學的教授團的規劃課程，深入淺出引導同學們思考。一個好的管理者需要具備什麼能力？行銷手段背後的意義是什麼？如何展現最完美的社交技巧？經濟是如何影響我們的生活？跳脫平常聽講的上課模式，藉由影片分享、小組討論、上台發表、互動桌遊等，打破師生間距離，熱絡的上課氣氛，不再沉悶單調，且能培養獨立思考的能力。 


一周營隊課程由李弘暉副院長開啟學員管理的新思維，並邀請元智管院教授江藍龍、徐樂文等師資，生動活潑的行銷講解與商業溝通的傳授，讓學生體驗不一樣的大學課程。這次的暑期營也安排桌遊的設計課程、理財的學習，以深入淺出方式帶領高中生不一樣的學習，除安排由元智大學管理學院各領域頂尖教授擔任講師，營隊期間也安排兩次企業參訪，分別至雄獅旅遊集團總部與太平洋自行車公司，利用實地走訪、企業分享的實例，讓學習更加生活化。為期一周的營隊活動，帶給康橋同學專業知識、新的思考方向外，也學習在各種議題上尋求創新的解決方案。
