# 網站文章資料_桃園航空城股份有限公司敬邀出席「2019桃園航空城論壇暨ICF聯合工作坊」活動_2832

## 一、文章基本資料

- **文章ID**: 2832
- **文章標題**: 桃園航空城股份有限公司敬邀出席「2019桃園航空城論壇暨ICF聯合工作坊」活動
- **文章網址**: https://www.yzu.edu.tw/admin/rd/index.php/tw/?option=com_content&view=article&id=2832
- **文章簡介**: 主旨：敬邀　貴校出席本公司108年11月20日(星期三)假台北諾富特華航桃園機場飯店舉辦之「2019桃園航空城論壇暨 ICF聯合工作坊」活動，請查照。

說明：

一、智慧城市為桃園航空城之發展目標，桃園市政府於108年6 月榮獲國際智慧城市論壇(ICF)評選為「2019年度全球智慧 城市」TOP 1，藉由參與全球智慧城市活動，將桃園市政府 推動的科技、人文、觀光等面向相關計畫成果展現於國際 舞台，加強與產官學界的緊密合作。循此，「2019桃園航 空城論壇暨ICF聯合工作坊」，尋求國內外產業及專家的經 驗分享及建議，以提供未來桃園航空城更完整規劃。

二、貴校為「桃園航空城大學聯盟」成員，誠摯邀請　貴校出 席本次活動。

三、「2019桃園航空城論壇暨ICF聯合工作坊」於108年11月20 日(星期三)假台北諾富特華航桃園機場飯店（桃園市大園 區航站南路1-1號）舉辦，報名方式如下：

(一)報名期間：即日起至人數額滿止。

(二)報名方式：本活動採網路報名，請逕行於線上報名，網 址：event.gvm.com.tw/201911_tycaerotropolis/。

(三)連絡電話：遠見雜誌，呂先生(02)2517-3688。
