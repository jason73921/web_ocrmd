# 網站文章資料_活得久還要活得好　新北開辦松年大學提供長者快樂學習_135

## 一、文章基本資料

- **文章ID**: 135
- **文章標題**: 活得久還要活得好　新北開辦松年大學提供長者快樂學習
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【中華電視公司／記者黃村杉／新北報導】

新北市政府因應高齡化社會，研考會辦理系列「活躍樂齡論壇」，討論主題聚焦在長者身心全方面照護需求，國立中正大學林麗惠副教授指面對高齡化社會的來臨，社會觀念已由「關心少數失能長者」翻轉為「關心多數健康長者」；另元智大學徐業良主任則重新演譯老祖宗的智慧，分享從老人福祉科技到智慧生活產業研究等，開啟長者照護方式新想像。

主委林純秀指出，市府因應高齡化趨勢，開辦松年大學提供年滿58歲以上長者快樂學習場域，鼓勵要在松年大學讀到「博士班」畢業；並指市長朱立倫也以行動鼓勵高齡學習風氣，每年都親自參加松年大學結業典禮，期許長者「越學越年輕、越來越快樂」，完成新北市「在地就養，健康樂活」服務版圖。

國立中正大學林麗惠副教授表示，面對高齡化社會的來臨，社會觀念已由「關心少數失能長者」翻轉為「關心多數健康長者」，活得久還要活得好，除了提供優質社會福利及健全醫療體系，更應該建構高齡學習網絡，讓長輩快樂學習而忘記年齡。

林麗惠並以「關心今天的老人，就是關心明天的自己」與聽眾共勉，指長者退休後往往容易成為「等吃飯、等睡覺、等垃圾車」的三「等」公民，樂齡學習鼓勵長者走入學校及社會，達到成功老化的快樂銀色生活。

元智大學徐業良主任重新演譯老祖宗的智慧，將曾子曰「樂其心，不違其志，樂其耳目，安其寢處」解釋為「科技提供長者平安環境及獨立生活，擁有快樂的心情」。無論是穿戴式裝置、服務型機器人或遠端照護系統，不該只有冰冷的數據，而應該重新連結長者與家人之間的情感。

徐業良舉例說明，當監測系統顯示媽媽的血糖值正常時，子女可以即時按讚鼓勵，不但關心長者的身體健康，也撫慰情感需求；並指二十四孝「嚙指痛心」講述母親嚙其指而曾參心痛的故事，可能就是遠距照護系統的先驅，獲得全場熱烈的掌聲。
