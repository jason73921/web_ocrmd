# 網站文章資料_不用口試就能錄取？23所大學試辦「逕予錄取」_1586

## 一、文章基本資料

- **文章ID**: 1586
- **文章標題**: 不用口試就能錄取？23所大學試辦「逕予錄取」
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【三立新聞台/生活中心/綜合報導】 


今年的大學學測將在1月20、21日舉行，在個人申請的管道上，多了「逕予錄取」這個入學方式，只要書面資料過關，各系可以不用口試直接錄取，但最多只能拿出2成名額，目前共有23所大學參與。 


大學招聯會執行秘書、清大教務長戴念華表示，為了準備口試，學校跟學生都要耗費大量心力準備，若能免口試直接錄取，對雙方都沒有壞處。

大學招聯會副召、靜宜大學的校長唐傳義表示，免口試錄取在國外行之有年，認為「逕予錄取」是可行的。此外，唐傳義認為逕予錄取較適合理工領域的招生，若是教育、社會等需考慮學生特質的系所，仍須進行口試決定是否錄取。參與逕予錄取的校系，會在書面資料審核後，公布第一批免口試就能錄取的名單，並在口試後公布第二批錄取名單。

招聯會表示，今年試辦的大學有23所，包括大同、大葉、中山、中華、元智、玄奘、東吳、東海、南華、真理、政大、高雄大學、清華、台南大學、聯合大學、華梵、開南、實踐、台北市立大學、銘傳、靜宜、中央及長庚等校。此外，參與「逕予錄取」的校系，今年最多只能拿出2成的名額，將視試辦狀況，決定明年是否進行調整。
