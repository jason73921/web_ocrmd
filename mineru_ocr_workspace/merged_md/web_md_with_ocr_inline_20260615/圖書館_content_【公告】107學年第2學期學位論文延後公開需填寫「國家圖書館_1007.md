# 網站文章資料_【公告】107學年第2學期學位論文延後公開需填寫「國家圖書館學位論文延後公開申請書」，夾附於紙本論文內_1007

## 一、文章基本資料

- **文章ID**: 1007
- **文章標題**: 【公告】107學年第2學期學位論文延後公開需填寫「國家圖書館學位論文延後公開申請書」，夾附於紙本論文內
- **文章網址**: https://www.yzu.edu.tw/library/index.php/tw/20190523-1
- **文章簡介**: 說明：


 1. 依據新版學位授予法第16條：國家圖書館保存之博士、碩士論文應提供公眾於館內閱覽紙本，或透過獨立設備讀取電子資料檔。但涉及機密、專利事項或依法不得提供，並經學校認定者，得不予提供或於一定期間內不為提供。本校乃配合辦理，若研究生有上述理由需延後公開者，請填寫「國家圖書館學位論文延後公開申請書」，夾附於紙本論文內 (連同授權書一併夾附)。

2. 依著作權法第15條：學位授予法撰寫之碩士、博士論文，著作人已取得學位者，推定著作人同意公開發表其著作。但研究生若有申請專利、投稿期刊、涉及國家機密、依法不得提供之上述理由需延後公開者，需加填申請書。

3. 紙本上架公開日期到期時，國家圖書館內的獨立設備將同步開放公眾讀取電子資料檔。此與論文電子檔的網路授權無關，即使網路授權狀態為不公開，國家圖書館仍會依法讓公眾透過獨立設備讀取電子資料檔。

4. 「國家圖書館學位論文延後公開申請書」檔案位置：博碩士論文系統→表單下載→學位論文延後公開申請書。
