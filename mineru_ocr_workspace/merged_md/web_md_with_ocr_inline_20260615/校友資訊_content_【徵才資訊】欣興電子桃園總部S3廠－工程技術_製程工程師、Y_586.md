= YIP TD良率整合工程師 =

1. 關鍵產品良率議題整合

2. 量產前樣品良率提升

3. 擬定良率改善策略、規劃Roadmap，並追蹤對策有效性

4. 異常與良率分析，並層別問題

5. 負責客戶端良率改善窗口

6. 104職缺資訊：https://www.104.com.tw/job/6kulf

 

= YIP TD良率解析工程師 =

1. 載板產品低良率分析、切片，解析異常真因。

2. 異物及材料失效分析，善用分析儀器(EDS/FTIR 等…)釐清異物來源。

3. 良率改善方案提出並執行

4. 跨部門溝通協調、資料彙整。

5. 104職缺資訊：https://www.104.com.tw/job/794o4?jobsource=company_job

 

**工作職能

1. 理工背景或具備工工、統計、數據分析背景經驗者

2. 為載板產品外商客戶對應窗口，需不怕以英文對應(無多益者，可先提供校內相關證明)

3. 具良好邏輯，能尋絲剝繭分析問題

 


 

公司簡介：

欣興電子成立於1990年，總公司位於桃園龜山工業區，為聯電責任企業群，是電路板（PCB）、積體電路載板（IC Carrier）產業的世界級供應商。

公司研發總部坐落桃園市龜山區，目前分為PCB事業部、載板事業部、IC代工預燒測試事業部，海內外集團員工人數共有2萬5千人。

本公司注重創新、研發、培養人才、團隊合作，努力提昇經營績效，並以市場導向、客戶為尊的服務，十餘年來成長迅速且穩健，年年獲利，也屢獲客戶的佳評。
