# 網站文章資料_AOI論壇展覽 熱議AI深度學習邊緣運算導入對策_3207

## 一、文章基本資料

- **文章ID**: 3207
- **文章標題**: AOI論壇展覽 熱議AI深度學習邊緣運算導入對策
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20181004
- **文章簡介**: 【經濟日報 曹松清】

由自動光學檢測設備聯盟、工研院共同舉辦的「全國AOI論壇與展覽」，於10月4日於交通大學登場。

這次活動聚焦時下最夯的AI深度學習、邊緣運算等議題，現場吸引國內外數百位專家參與盛會。成功營造產學研互動交流場域，並凝聚產學研之研發力量，共同推動台灣AOI產業蓬勃發展。

主辦單位自動光學檢測設備聯盟會長亦為工研院量測中心主任林增耀表示，近來工業4.0、AI、物聯網蔚為風潮，在此趨勢下，機器視覺需加入強大的即時資料收集、處理與分析能力，滿足智慧製造需求。今年活動以「智慧科技驅動AOI產業新未來」為主題，探討智慧運算AOI、深度學習AOI、先進檢測AOI等三大議題。

並邀請工研院、台灣大學、中央大學、中興大學、元智大學、Google、SAS台灣、致茂電子及達明機器人等12位產學研專家接續演講，分享「眺望趨勢 ‧ 量測設備未來10年發展重點」、「AOI and IoT / Edge and Cloud 產線應用」、「探討深度學習在電腦視覺應用上的疑問」、「AOI參與整廠協作之實務建議」等議題。期盼借鏡專家的前瞻論點，協助設備開發者、技術研發者了解技術趨勢，洞察產業潮流，激勵AOI產業迎向技術升級課題。(待續）
