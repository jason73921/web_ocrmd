# 網站文章資料_【住宿訊息】公告109學年度大學部學生宿舍（第三宿舍）床位抽籤作業_1843

## 一、文章基本資料

- **文章ID**: 1843
- **文章標題**: 【住宿訊息】公告109學年度大學部學生宿舍（第三宿舍）床位抽籤作業
- **文章網址**: 
- **文章簡介**: 公告日期：109年3月31日（星期二）

 

主旨：公告109學年度大學部學生宿舍（第三宿舍）床位抽籤作業。

說明：

一、抽籤前請詳閱下列應注意事項：

(一)依據「元智大學學生宿舍管理規則」第八條規定，申請之床位一次為一學年，即應住滿一學年（包含上、下學期），除休學、退學、轉學、實習、出國交換及特殊原因經核准者外，均不得申請退宿，住宿未滿一學年而自願或勒令退宿者，一律不退還住宿費及保證金，學生不得異議。〈109-2非特殊原因逕行退宿者，仍需繳納住宿費，否則無法完成註冊手續。〉

(二)中籤者須在公告時間內繳交保證金2,000元（4/13～4/16）以及「住宿同意書」( 4/30截止收件)，已繳交保證金而放棄者，所預繳之保證金概不退還，且不得要求任何補救措施。

(三)抽完籤欲放棄者，請自行再上Portal取消之，若於4/9系統關閉後有任何異動應《立即主動》告知宿服組，如造成權益受損，學校恕不負責。

(四)舊生住宿區（第三宿舍）凌晨2~6時斷網，無法接受者，請勿申請住宿。

(五)請妥慎評估自我「團體生活適應能力」以及「健康狀態是否適宜住宿」後再決定是否參加校舍抽籤。

二、抽籤資格：大一～大三在學之大學部學生。

三、下列學生請勿參與抽籤：

(一)已申請優先住宿並經核准者。(二)宿服組推薦之樓長及宿自會幹部。(三)資服處推薦之網管人員。(四)體育室推薦之校隊代表。(五)境外生（含外籍學位生、僑生、陸生、雙聯生），不需抽籤但須提出優先住宿申請(106.107)。

四、抽籤日期：109年4月6日10時起至109年4月9日16時止。

五、抽籤方式：進個人Portal點選『住宿申請』選項，並依指示完成抽籤程序，籤號在10000至99999之間，床位依所抽籤號由小到大依序分配，額滿為止，申請者不得異議。

六、中籤結果公告日期：109年4月10日（星期五）。

七、床位分配方式：

(一)中籤者可自組室友，未自組室友者由宿服組權責分配。

(二)寢室分配由宿服組全權決定。

(三)開放抽籤床位數：預估男二舍395床，女二舍615床（實際人數以宿服組公告之中籤名單為準）。
