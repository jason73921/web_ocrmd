# 網站文章資料_國立臺灣大學「MOST GASE X SEMI 科技女力講座」_3675

## 一、文章基本資料

- **文章ID**: 3675
- **文章標題**: 國立臺灣大學「MOST GASE X SEMI 科技女力講座」
- **文章網址**: https://www.yzu.edu.tw/admin/rd/index.php/tw/?option=com_content&view=article&id=3675
- **文章簡介**: 主旨:

科技部全球事務與科學發展中心（MOST GASE）謹訂於本（110）年3月12日（星期五）舉辦「MOST GASE X SEMI 科技女力講座」，敬請惠予協助公告，請查照。

說明:

一、為鼓勵女性青年投入科學發展，促進科學與科技界中的性別平等，科技部全球事務與科學發展中心將舉辦旨揭講座，

邀請在科技業與科研領域中的成功女性企業家與領導人物，分享其學思歷程，使青年們在求學或求職時，不再受限於傳統的性別刻板印象，

打開更寬廣的職涯選擇，並鼓勵更多女性投身科研領域與科技產業。

二、講座資訊如下：

（一）時間：110年3月12日（星期五）14時至17時。

（二）地點：臺北市信義區菸廠路88號臺北文創大樓6樓D廳。

（三）參與對象：全國大專院校及高中職學生、教師、職員、一般民眾。

（四） 報名方式：即日起至110年3月10日受理報名（報名網址：https://www.surveycake.com/s/20aoN）。

三、如需進一步資訊，請參考該中心官網（網址：https://gase.most.ntu.edu.tw/events/49），

或本案聯絡人：曾小姐，Tel：02-3366-2121#119，Email： gasezippytseng@ntu.edu.tw。
