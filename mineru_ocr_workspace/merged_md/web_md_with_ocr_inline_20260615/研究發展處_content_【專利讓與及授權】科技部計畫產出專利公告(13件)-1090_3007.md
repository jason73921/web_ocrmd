建構於適應性網路模糊推論系統之三層串接式認知引擎之方法及裝置



I474728



本發明揭示一個建構於適應性網路模糊推論系統(adaptive network-based fuzzy inference system,ANFIS)之三層串接式認知引擎(cognitive engine)方法及裝置，使認知無線電(cognitive radio,CR)行動通訊網路可以因應網路環境之變化，彈性、可靠而有效率的使用無線行動通訊網路之頻譜與功率資源之智慧型網路資源分配控制器，包含第一層之ANFIS資料率控制器(ANFIS rate controller,ARC)、第二層之ANFIS功率控制器(ANFIS power controller,APC)與第三層之ANFIS干擾控制器(ANFIS interference controller,AIC)，可偵測網路話務性能、傳接機性能、用戶車速、通道環境變化、傳接機訊干雜比，訓練ANFIS控制器參數，分別適應性控制傳接機的資料率、發射功率與天線零點數目之指配，使認知無線電行動通訊網路的用戶可產生最佳化之最大系統平均通達率、最小發射功率與干擾的多媒體通訊網路服務品質(quality of service,QoS)。



2015/02/21~ 2032/09/20

圖框速率提升轉換方法及其架構



I646836



本發明提出一種圖框速率提升轉換方法，包括一雙向三維遞迴搜尋演算法、一自適應搜尋範圍的雙向動態估計演算法以及一交疊區塊動態補償演算法。雙向三維遞迴搜尋演算法快速地產生初始向量，自適應搜尋範圍的雙向動態估計演算法進一步提升初始向量的精準度，接著交疊區塊動態補償演算法進行動態補償，以改善區塊效應以及提升影像品質。本發明採用預測即補償的作法，大幅提升預測的精準度，運算量大幅減少，並且不占用過高的硬體成本。此外，本發明亦提出一種圖框速率提升轉換硬體架構。



2019/01/01~ 2037/06/04

I635740

THREE LAYER CASCADE ADAPTIVE NEURAL FUZZY INFERENCE SYSTEM (ANFIS) BASED INTELLIGENT CONTROLLER SCHEME AND DEVICE



US9,077,491 B2



Intelligent technique is an effective method to perform the network resource management. A three layer cascade adaptive neural fuzzy inference system (ANFIS) based intelligent controller is proposed for the mobile wireless network to optimize the maximum average throughput, minimum transmit power and interference for multimedia call services. The proposed intelligent controller is designed with a three layer cascade architecture, which mainly contains and ANFIS rate controller (ARC) in the first layer, an ANFIS power controller (APC) in the second layer and an ANFIS interference controller (AIC) in the third layer. The design aim of the proposed three layer cascade ANFIS cognitive engine is maximizing the average throughput of the mobile wireless network, while minimizing the transmit power and interference power.



2015/07/07-2033/10/02

高色轉換效率的有機發光二極體結構



M518412



一種高色轉換效率的有機發光二極體結構，係包括一玻璃層、一第一色轉換層、一第二色轉換層以及一氧化物式透明導電層。該玻璃層係為玻璃材質。該第一色轉換層係設置於該玻璃層之上，具有複數第一大奈米粒子、複數第一小奈米粒子以及一第一染料，該等第一大奈米粒子之直徑係大於該等第一小奈米粒子之直徑，該第一色轉換層具有一第一大奈米粒子濃度、一第一小奈米粒子濃度以及一第一染料濃度，該第二色轉換層係設置於該第一色轉換層之上。藉由上述之結構，本創作可提高色轉換層之色轉換效率。

聯合運動補償與空間錯誤隱蔽之操作方法



I578769



本發明為一種聯合運動補償與空間錯誤隱蔽之操作方法，其方法包含邊界平滑處理以及運動補償映射雙線性內插處理，先將損壞的像素所在之宏區塊進行邊界平滑處理，藉以形成邊界平滑像素，再利用邊界平滑像素，執行運動補償映射雙線性內插處理，以獲得恢復像素。其中宏區塊是由多個像素所構成且排列成正方形，邊界平滑處理係採用邊界平滑濾波器的處理方式，而運動補償映射雙線性內插處理，則使用垂直像素及水平像素之線性濾波器取得恢復像素，此外本發明採用共用於運動補償之空間錯誤隱蔽模式，以大幅降低硬體使用率及提高硬體共用率。



2017/04/11~ 2036/03/10
