# 網站文章資料_元智大學與中華精測企業專班開班_2459

## 一、文章基本資料

- **文章ID**: 2459
- **文章標題**: 元智大學與中華精測企業專班開班
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20171212-5
- **文章簡介**: 元智大學與中華精測科技合作開辦「中華精測EMBA企業碩士學分專班」課程，建立企業與學術的前瞻合作模式，協助企業人才培育與擴展國際化視野。12月初正式舉辦始業式，並由中華精測總經理黃水可與元智大學校長吳志揚共同主持。課程安排將以6大主軸為方向，由元智66位博士師資充份支援，搭配各業界的專業業師共同授課。

元智大學校長吳志揚表示「中華精測EMBA企業碩士學分專班」的合作模式，是標準的產學合作，為企業量身打造的客製化模式，由元智規劃課程、師資與場地，中華精測推薦主管與資深幹部參加，共同培育企業的優秀人才。

吳志揚說，中華精測總經理黃水可為元智優秀的校友，公司整體的動能與能量非常強，執行效率更值得佩服，未來希望雙方合作緊密，並能縮短學用落差，而且讓已在職者也能有繼續終身學習與進階的機會。

元智大學規劃「組織行為與領導」、「團隊建立與管理」、「專案管理」、「作業管理」、「創新與技術管理」及「全球產業分析」等6大方向課程，由元智大學66位博士師資充份支援，搭配各業界、專業領域業師共同授課。6門課18個學分，每門課54小時，採EMBA多教師授課傳授專業與實務並重的課程，並以企業包班的方式安排學員的上課時間將持續到2018年春季。
