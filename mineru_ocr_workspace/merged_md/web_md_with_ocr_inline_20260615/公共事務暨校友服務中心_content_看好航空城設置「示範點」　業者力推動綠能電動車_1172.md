# 網站文章資料_看好航空城設置「示範點」　業者力推動綠能電動車_1172

## 一、文章基本資料

- **文章ID**: 1172
- **文章標題**: 看好航空城設置「示範點」　業者力推動綠能電動車
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【台灣好新聞/記者葉志成／桃園報導】 


航空城公司董事長張昌財拜訪綠能電動車業者，雙方就桃園航空城未來產業專區發展及規畫溝通交流，希望政府能積極推動電動機車。

國際高端車廠相繼投入電動車市場，電動車成為汽機車業者轉型環保綠能的最佳投資。桃園航空城公司董事長張昌財表示，拜訪電池大廠統振公司，與在金門地區積極推動電動機車的易能電網科技，有意在桃園航空城設置「示範點」，以作為進軍桃園地區先行「試點」，二公司看好綠能電動車的發展潛力，雙方就桃園航空城未來產業專區發展及規畫溝通交流，也希望政府能積極推動電動機車，更可減少空氣污染，維護國人健康。

桃園航空城公司為積極進行「預招商」，董事長張昌財拜訪統振公司及易能電網科技，希望符合「三低一高」產業－兩家公司都能到航空城投資。航空城公司董事長張昌財近期上任後主動出擊、拜訪，尋求國內和桃園市內符合航空城未來產業要求廠商到航空城設廠和投資。

在與電池交換系統營運商易能電網科技及統振公司高層交換意見時，易能電網科技總經理王章平說，國內目前大約還有二百萬輛二行程機車，這些機車很容易造成空氣污染，如果能全數來更換成電動機車，不但對台灣地區空氣污染源減少很多外，對國人健康也是一大幫助。 


航空城公司董事長張昌財拜訪綠能電動車業者，雙方就桃園航空城未來產業專區發展及規畫溝通交流，希望政府能積極推動電動機車。

他還說，以台灣生產機車實力和在東南亞台灣機車發展，若能轉換成電動機車，應該對目前台灣經濟低迷會有很大的幫助，所以，他對台灣推動綠能交通很有興趣和信心，他也說，他們曾與台中市政府合作推動的綠能交通網計畫，日前也開始在金門地區，專營電動機車與電動輔助自行車的銷售，成效不錯，目前更積極進行「ezSWAP電池交換站」設置。

主導電動機車交換式電池的統振公司董事長陳敦仁也指出，很高興得知張董事長早在二十年前在元智大學教書時就已提出用機械手全自動交換電池的概念，電池 交換系統模式導入後，不但能解決更換電池方便，更可利用智慧軟體置入後，可以監測機車續航力和以更多元化方式來核算使用電池計費問題，他們未來自動交換式電池，不但會投入電動機車、電動輔助自行車及相關周邊商品外，他們也會以各種不同組合方式，投入到電動車等。會中的共識是電動機車的推行除了要有鼓勵的紅蘿蔔也要有處罰的鞭子，也就是相對的要有政策規定汰換時程，如此電動車的推動才能達到事半功倍。

張昌財表示，航空城公司現在工作就是積極來進行招商和行銷工作，以目標管理行銷手法，先釐清符合高科技、綠能、高效益企業後，主動去拜訪、邀請到航空城來投資，尋求這些合於航空城推動產業專區工廠盡量在航空城土地徵收計畫完成後這些廠商就能進入設廠、生產，以爭取最快的生產效能。除了生產廠商的引入之外，也大力推展創新、研發為企業尋找新出路。 


航空城公司董事長張昌財拜訪綠能電動車業者，雙方就桃園航空城未來產業專區發展及規畫溝通交流，希望政府能積極推動電動機車。
