# 網站文章資料_財團法人台灣永續能源研究基金會「2021第14屆台灣永續獎」_3818

## 一、文章基本資料

- **文章ID**: 3818
- **文章標題**: 財團法人台灣永續能源研究基金會「2021第14屆台灣永續獎」
- **文章網址**: https://www.yzu.edu.tw/admin/rd/index.php/tw/?option=com_content&view=article&id=3818
- **文章簡介**: 主旨：

敬邀　貴校參加「2021第14屆台灣永續獎」評選活動，鏈 結聯合國永續發展目標，展現社會影響力，請 查照。

說明：

一、聯合國於2015年提出17項永續發展目標(Sustainable Development Goals, SDGs) 及 169 個具體目標，成為至 2030年前國際合作的指導原則，國際社會的永續議題並持 續升溫，為鼓勵企業將永續理念納入決策方針，台灣永續 能源研究基金會14年來持續舉辦「TCSA台灣永續獎」 (Taiwan Corporate Sustaibability Awards, TCSA)評選 活動，其理念與永續發展目標一致。

二、「TCSA台灣永續獎」歷屆參獎企業及機關(構)累計達364 家，其總營業額與2019年國內 GDP比值為145%，為臺灣最 具影響力之永續獎項，並被譽為ESG績效最佳指標。

三、本年度獎項區分為五大類，說明如下:

(一)永續發展目標行動：表揚各界在實踐聯合國 17 項永續發展目標之卓越行動方案(即個別案例) 。

(二)永續傑出人物：表揚致力善盡社會責任，推動組織邁向 永續發展相關工作付出貢獻卓著、足堪典範之個人。

(三)永續綜合績效：評選於環境永續(E)、社會公益(S)及公 司治理(G)三面向兼具突出綜合績效，善盡社會責任，並 具發展願景、策略之典範大學。

(四)永續單項績效：表揚組織大學在特定單項永續議題之績 優表現並具有標竿作用及領導性地位者，參獎者須呈現 整體推動策略與成果 ( 非僅個別案例 ) 。

(五)永續報告：表揚永續報告資訊揭露完整、可信度高、溝 通成效佳，足以彰顯善盡社會責任之大學。

四、承上，為鼓勵大學推動社會責任實踐方案，善用學校資 源，促進地方創生成效，並接軌國際實踐聯合國永續發展 目標，本屆增設「永續發展目標行動獎」，以凸顯大學在 SDGs績效上之優異表現，誠摯邀請 貴校共襄盛舉，參與本 獎評選活動，踴躍競逐最高榮譽、形塑品牌形象，高度鏈 結SDGs，並持續擴大社會影響力。

五、本獎頒獎典禮將分別於9月8日結合首屆「2021 台灣永續發 展目標論壇」頒發永續發展目標行動獎，其餘獎項於11月 17日結合「2021 第4屆全球企業永續論壇」頒發，懇請貴 校踴躍報名，樹立高教永續典範。

六、本獎採E化線上報名作業，自本年5月3日起開放報名。評選 辦法已公告於官方網站(tcsaward.org.tw)，敬邀 貴單位 踴躍報名參加，共同善盡社會責任，提升臺灣永續競爭 力。
