# 網站文章資料_元智大學開發物聯網技術　改善人類未來生活品質_1904

## 一、文章基本資料

- **文章ID**: 1904
- **文章標題**: 元智大學開發物聯網技術　改善人類未來生活品質
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20170509-3
- **文章簡介**: 【蘋果即時/突發中心張沛森／桃園報導】

隨著物聯網（IoT）技術的起飛，各種自動化智慧化的應用與管理系統，將改善人類未來的生活品質。元智大學電機系教授郭文興帶領網際網路實驗室，長期鑽研物聯網技術，已開發出多項深具潛力的應用面向，並研發「YZU IoT Gateway」，應用於智慧家庭。

「YZU IoT Gateway」具備高度客製化模組化彈性的物聯網管理框架，可橋接用戶、設備與服務業者，統合來自不同品牌的市面產品、各式各樣的感測器與雲端的資料來源，並透過多樣化的操作介面提供整合管理功能，設備商不再需要費心設計APP，服務業者也可提供整合性的服務，一般或專業使用者便能享受高品質低成本的物聯網整合應用。

網際網路實驗室電機所碩二江銘洋同學表示，以智慧家庭為例，安裝此系統的機上盒可透過藍芽或Wi-Fi連結家中各種電器硬體，並接收網路上的資訊。

使用者從整合畫面便能輕鬆得知天氣、家中的溫濕度等訊息，並可以控制電燈、電扇和等家電，更可以設定各種自動化功能。

例如，設定在用戶回家時自動開啟電燈，一氧化碳過高時啟動警報器。這樣的系統便可減少操作負擔，增進用戶安全，並節省消耗的能源。元智大學表示，目前此技術已獲得多項國內外專利，也與產官學界進行各種合作，與交通大學的物聯網雲端團隊測試系統架接，與工研院共同開發數位匯流相關技術，受民間產業委託執行多項產學計畫與技術轉移。
