## 一、文章基本資料

- **文章ID**: 411
- **文章標題**: Notice for Conflicting Exams
- **文章網址**: https://www.yzu.edu.tw/admin/aa/index.php/tw/conflicting-exams
- **文章簡介**: Please review your exam schedule to ensure that there are no conflicts. 

If find any, please file a request for changing in scheduled exam in advance

 and refer to the following details.

Exam Conflicts cross Departments


When to file a request : According to the YZU Academic Calendar and announcement by Curriculum Section

NOTE: YOUR REQUEST WON’T BE ACCEPTED AFTER THE DEADLINE.

Where to file a request：At the Curriculum Section, Office of Academic Affairs ( Room 1210, Building 1) (Ext.2259)

Information required for filing a request: course code, class, and course name.

Office Hours: 09:00~16:30


Exam Conflicts within One’s Own Department


When to file a request：According to the YZU Academic Calendar and announcement by Curriculum Section.

Where to file a request：At the department office offering course to file an exam conflict adjustment request.
