# 網站文章資料_為畢業生找出路 元智校長給企業一封信_4421

## 一、文章基本資料

- **文章ID**: 4421
- **文章標題**: 為畢業生找出路 元智校長給企業一封信
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/2020062203
- **文章簡介**: 桃園電子報記者劉筱寧  2020-06-19

新型冠狀病毒（COVID-19）肆虐，全球經濟嚴重受到衝擊，根據世界銀行報告顯示，預估2020年的全球經濟產出將大幅萎縮5.2%，創下自1870年以來，最嚴重的崩跌。儘管各國政府分別以不同名目的紓困方式支撐國內經濟總量，但疫情之影響，全球製造業、服務業及航空業等嚴重受挫，失業人口將持續攀升。

今年的畢業生求職將面對一大挑戰，受新冠肺炎疫情影響，不少公司暫緩徵才計劃、凍結內部人事，部分產業如旅遊業、航空業、製造業等更是受挫。勞動部針對今年應屆畢業生也推出「青年就業獎助計畫」，提供就業獎勵金，鼓勵應屆畢業生在困境中勇敢的面對職場。元智大學每年都會為畢業生謀出路，今年因為疫情更是積極，校長吳志揚寫信給各校友主管，請企業在疫情後，面對新一波新常態的經濟，如有人力資源的需要或調整，請給元智的畢業生一個發揮長才與打開視野的機會。

吳志揚表示，新冠肺炎全球大流行不但帶來健康的安危，影響社會和經濟原本的節奏，也打亂了今年畢業生的求職夢，許多企業徵才計畫暫告停滯，人力資源將面臨重新調整，今年的畢業生將面臨求職的嚴峻考驗，但危機往往也是轉機的開始，因為疫情改變了原有經濟活動，也改變人類的思維，這波疫情意外助長新科技應用的普及，未來的經濟活動將面臨一個嶄新的生活模式與樣態。

吳志揚表示，元智向以資通訊聞名，在人工智慧、物聯網、大數據等方面著力極深，學生對外參加獎項也有非常亮眼成績表現，除為國內第一所具國際聲望的英語教學外，前(106)年首創國內以新雙語(即英語與程式語言)教學，全校必修程式語言課程(中語系也不例外)，符合大數據數位時代及人工智慧崛起之所需，也是未來經濟模式的重點。(待續)
