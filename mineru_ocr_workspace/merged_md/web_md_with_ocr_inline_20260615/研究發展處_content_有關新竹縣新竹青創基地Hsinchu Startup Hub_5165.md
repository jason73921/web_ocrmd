# 網站文章資料_有關新竹縣新竹青創基地Hsinchu Startup Hub 112年進駐 團隊招募評選案_5165

## 一、文章基本資料

- **文章ID**: 5165
- **文章標題**: 有關新竹縣新竹青創基地Hsinchu Startup Hub 112年進駐 團隊招募評選案
- **文章網址**: https://www.yzu.edu.tw/admin/rd/index.php/tw/?option=com_content&view=article&id=5165
- **文章簡介**: 說明：

一、為扶植新竹縣青年創新創業，整合資源、加強媒合，給予青年創業工作者支援與輔導，特成立新竹青創基地Hsinchu Startup Hub，期望成為青年創新創業後盾，鼓勵其根留竹縣發展，創造竹縣永續創業能量。

二、基地占地280坪，內設有共享辦公空間、新創辦公室、共享學習教室、聯合會議室、小型會議室、洽談室等空間，並邀請專家學者提供創業諮詢輔導、辦理各類培力課程、媒合會、成果博覽會等，提升青年創業各階段面臨問題的解決能力，也加速提升個人及企業競爭能力，並打造多元化之創新創業聚落。

三、報名資格：凡團隊成員中，至少1人年齡45歲以下、設籍、或居住新竹縣，即可申請進駐。

四、申請方式：即日起至112年1月13日（五）止，至報名網頁（https://reurl.cc/zrLZpN）完成進駐申請。

五、詳細活動內容，請見新竹青創基地官方網站（https://startup-hc.tw/）或新竹縣政府教育局青年事務科粉絲專頁查詢。

六、檢附活動海報暨新竹青創基地進駐懶人包各乙份。
