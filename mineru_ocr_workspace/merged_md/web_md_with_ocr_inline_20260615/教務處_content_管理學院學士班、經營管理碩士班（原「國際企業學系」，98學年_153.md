# 元智大學國際企業學系碩士班修業規定

（九十三學年度入學新生適用）

93.03.11 92-7 國企系系務會議通過

93.04.14 92-8 國企系系務會議通過

93.11.17 93-4 國企系系務會議通過

94.03.17 93-8 國企系系務會議通過

94.10.20 94-4 國企系系務會議通過

1.最低畢業學分：36學分 + 碩士論文

2.選修本學程課程注意事：

(1)必修科目：共12 學分。

◎有特殊經驗之研究生可專案申請改變必修科目，並由選修科目替補。

“國際行銷管理”先修課為”行銷管理”，“國際財務管理”先修課為”財務管理”未曾修習者可在大學部補修，若大學部課程與碩士班一年級必修課衝堂或兩門大學部課程衝堂，則可至二技補修。

◎跨國商業溝通為先修課程，需修過國企所暑假開設課程方可畢業。

(2)選修科目(共24 學分)：

「數量方法」：多變量分析及計量經濟學為二選一之必選。

「研究方法」：競爭策略、財務研究方法、全球產業分析及行銷研究為四選二之必選。

「進階研究」：本課程需於選定指導教授後方可進行選課

市場競爭分析、全球行銷策略、策略行為分析、全球資產管理、跨國營運管理、產業趨勢分析、廠商競爭力分析、顧客關係策略管理、高科技事業經營、國際經濟理論與政策、合作策略(至少選修 9學分)

「專業科目」：定價與授權策略、創新與技術管理、全球競合策略、國際化策略、高科技行銷、消費者行為、行銷傳播管理、國際零售業管理、國際金融、投資學、衍生性金融商品、風險管理、區域市場研究、國際經貿組織研究、哈佛國際競爭策略、策略聯盟(至少選修 6學分)

(3)自由選修科目：或經指導教授或國企系主任同意之碩士班課程。

3.研究生應於碩士修業第一學年結束前(約三月底)選定碩士論文指導教授；需經指導教授同意並至所上完成登錄程序，且應依規定完成碩士論文口試。

4. 學生需於升研二那年的八月三十一日前，自行報名通過TOEFL/ITP考試紙筆測驗 520分以上(電腦測驗 190分以上)，或 GMAT前 70%以上，或 GRE前 70%以上，或多益(TOEIC)675 分以上，或全民英檢中高級。未達規定分數者必須繳交兩次英文考試成績至系辦，方可於研二以下述修課方式替代考試成績：至大學部修習 4學分之外語課程（「外語研習 III」、「外語研習 IV」、「商用英文」、「商業溝通」四選二），且修課成績達 80分以上。於研一先修課的同學，畢業前仍需繳交兩次考試成績。
