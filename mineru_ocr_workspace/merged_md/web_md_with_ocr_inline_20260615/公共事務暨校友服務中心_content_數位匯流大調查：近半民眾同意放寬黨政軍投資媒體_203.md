# 網站文章資料_數位匯流大調查：近半民眾同意放寬黨政軍投資媒體_203

## 一、文章基本資料

- **文章ID**: 203
- **文章標題**: 數位匯流大調查：近半民眾同意放寬黨政軍投資媒體
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【NOWnews/政治中心／綜合報導】 


台灣數位匯流發展協會11日公布「2015上半年數位匯流大調查」，這是2012年以來連續第八次調查，台灣民眾對媒體收視環境不滿意度持續上升，51.2%不滿，40.9%滿意，不滿比例較去年同期上升5.8個百分點。

調查發現，由中嘉購併案引發外界討論的廣電三法「黨政軍退出媒體」應放寬，有49.4%表示支持廢止「黨政軍條款」對政府投資媒體限制，34.0%認為仍應有限制；至於在媒體數位化，頻道大幅增加情形下，有54.6%民眾支持政府或政黨可以擁有頻道宣揚政治理念，37.5%仍表反對。

調查也發現，在「媒體可能染紅」的議題上，62.4%民眾擔憂大陸藉由台商購併媒體而影響台灣輿論，32.2%不擔心；56.8%表示應該禁止大陸台商回台購併台灣媒體，31.1%則認為不應管控購併的商業行為。

至於全台最大有線電視中嘉網路交易案，有52.4%民眾支持禁止中嘉網路賣給具有大陸台商背景的業者，34.0%認為不應該限制購併的商業行為，13.6%無明確意見。

在其他數位匯流相關議題上，調查發現，86.4%民眾認為台灣媒體目前有逐漸被財團壟斷的情形發生，77.2%民眾覺得情況嚴重；另有69.5%擔憂未來媒體被特定集團壟斷，顯示民眾對媒體壟斷議題更加關注。

在有線電視滿意度上，有65.0%收視戶對有線電視重播率不滿，52.9%覺得目前費用太貴。至於NCC推動的分組付費，有61.3%支持，30.7%反對；對於開放新業者經營有線電視方面，則有69.5%考慮更換，21.4%無意願改變。

至於網路使用上，67.7%民眾有使用智慧型設備行動上網，較去年同期增加了13.4個百分點；使用4G上網的比例，則由去年底的26.1%上升到54.0%，大幅增加27.9個百分點。對於行動上網速度滿意度，則有68.1%肯定，30.7%不滿意，滿意比例較去年底上升了12.6個百分點。

國立台灣大學電機工程學系林宗男教授認為，媒體作為第四權應落實監督政府責任，政府以及政黨不應干涉媒體立場；國人可以考慮在不違反「黨政軍退出媒體」的前提下，對「黨政軍退出媒體」條款進行適度符合現況之修改。最後，呼籲立法機關及有關單位盡速通過「反媒體壟斷法」。

元智大學資訊管理學系周韻采教授則認為，近年媒體業者經營不易，很難在不受政黨影響下具有完全自主性。別讓「要求政黨退出媒體」成為空泛的口號，制定政黨置入性行銷、發包媒體等相關規範，才是實際、具體的作為。

國立台北科技大學智慧財產權研究所江雅綺助理教授則表示，依據此次數位匯流大調查的結果，在許多議題上民眾多數傾向支持改革。例如：黨政軍條款的開放，讓資金能夠充分挹注，以及在壟斷議題上，贊成「媒金分離」以確保言論多元的價值。

「數位匯流大調查」是台灣數位匯流發展協會，委託趨勢民意調查公司，調查共計完成1501份有效樣本，抽樣誤差在百分之九十五的信心水準下，為正負二點六個百分點，調查時間是7月30日至8月7日晚間進行。
