# 網站文章資料_私大校長建議碩士不寫論文　教部：「限專業領域」研究後提供參考規範_6415

## 一、文章基本資料

- **文章ID**: 6415
- **文章標題**: 私大校長建議碩士不寫論文　教部：「限專業領域」研究後提供參考規範
- **文章網址**: 
- **文章簡介**: 【ETtoday/記者許敏溶／高雄報導】

去年台大國發所碩士在職專班接連爆發論文抄襲被撤銷學位事件，也讓外界質疑大學未認真把關碩士論文，在今（6日）大學校長會議上，有多位私立大學校長提案，建議碩士在職專班學習國外，取消學位論文，改採多修課方式取代，教育部則表示，碩士在職專班以專業實務報告替代學位論文部分，將補助私立大學校院協進會進行專案研究，提供參考格式規範讓大學參考。
去年九合一選舉前後，政治人物博碩士論文抄襲被撤銷學位事件連環爆，其中又以碩士在職專班最多，包括台大國發所、中華大學、德明財經科大等，引發外界質疑。

不過，在今天的全國大學校長會議上，在進行提案討論時，包括多位私立大學校長建議，應該取消碩士在職專班論文，改以多修課方式取代。

元智大學校長廖慶榮指出，碩士在職專班的實務課程，應該比照國外，直接多修一些課來取代撰寫論文，碩士不必寫論文；東海大學張國恩也認為，在國外很多名校已經取消碩士論文規定，教育部應該讓大學有更彈性，利用修課取代撰寫碩論；德明大學校長盧秋玲則說，以學術專業性來說，寫論文應該在博士生，而不是碩士班，應該給學校更多自主性，碩士可以專業實務報告替代論文。（待續）
