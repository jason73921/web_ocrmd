# 網站文章資料_申請加入中華民國元智大學校友總會Q&A_818

## 一、文章基本資料

- **文章ID**: 818
- **文章標題**: 申請加入中華民國元智大學校友總會Q&A
- **文章網址**: https://www.yzu.edu.tw/alumni/index.php/tw/q-a
- **文章簡介**: 申請加入中華民國元智大學校友總會Q&A

 

元智大學自民國78年創校，為整合各系系友會、地區校友會以及未來將成立的各國校友會，在廖慶榮校長及熱心學長姐的鼓勵與支持下，推動成立「中華民國元智大學校友總會」。

中華民國元智大學校友總會現正熱烈招募團體會員中，敬邀各類型校友組織踴躍加入我們，匯集大家的力量，將元智校友組織發展茁壯！

 

Q.加入校友總會有哪些好處？

A.校友總會將以服務與加強校友互動為主要任務，與校友攜手協助母校校務發展，促進海內外校友聯繫與交流，鏈結校友企業與母校產學合作資源。母校亦提供各項優惠福利（如圖書借閱、健康休閒中心、終身教育課程折扣等），並持續研擬更加優惠及便利之服務。

 

Q.申請加入校友總會的條件有哪些？如何加入？

A.校友總會之成立旨在整合所有校友會，故其入會以團體會員為主，贊助會員/榮譽會員等個人會員為輔。

如您為各系系友會、地區校友會或是興趣型組織等各類型校友組織的會長/領導人，邀請您填寫團體會員入會申請表，簽名掃描後回傳至 alumni@saturn.yzu.edu.tw或傳真03-4637488即可。

 

Q.我不但是校友，也是公司的負責人，可以申請加入嗎？

A.校友總會非常歡迎擔任公司負責人的校友，以公司團體名義申請加入，申請方式同上。

 

Q.贊助/榮譽會員是什麼？有限制條件嗎？

A.凡贊同本會宗旨且贊助本會經費、資源之個人或團體，可以申請以贊助會員加入，贊助經費沒有限制金額；對本會工作有特殊貢獻者，則可以榮譽會員身分加入。請填寫贊助會員入會申請表、榮譽會員入會申請表簽名掃描後回傳，首次審查會員名冊將由籌備會委員進行，其後則交給理事會審查。

惟依章程草案規定，贊助會員、榮譽會員無表決權、選舉權、被選舉權與罷免權，請知悉。

 

Q.需要繳交哪些費用？何時需要繳交？

A.團體會員首次入會費5,000元，常年會費5,000元，故第一次加入共計需繳納10,000元。

 

Q.會費將用在哪些地方？

A.校友總會為獨立之社團法人，其會費的收取主要用在總會各項校友活動、母校活動支援和總會相關行政費用支出。其年度收支預算表及決算表、工作計畫及報告等，均會在每年度的會員大會中與會員討論及報告。
