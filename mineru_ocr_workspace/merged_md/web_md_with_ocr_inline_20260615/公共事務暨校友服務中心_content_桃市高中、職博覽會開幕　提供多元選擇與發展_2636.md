# 網站文章資料_桃市高中、職博覽會開幕　提供多元選擇與發展_2636

## 一、文章基本資料

- **文章ID**: 2636
- **文章標題**: 桃市高中、職博覽會開幕　提供多元選擇與發展
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20180312
- **文章簡介**: 【台灣好新聞/記者黃駿騏/桃園報導】


桃園市長鄭文燦10日上午前往桃園市立綜合體育館，出席「107年度桃園市高中高職博覽會」開幕式，鄭市長表示，今（107）年桃園市高中高職博覽會主題為「實驗」和「創課」，共有39個學校參加，每個學校皆展現最佳特色，吸引優秀學生，也希望透過這項活動，提供學生們多元選擇以及未來多元發展。

鄭文燦指出，根據統計，包括免試入學、特色招生或新增五專名額在內，107年度學校招生人數多於報考學生人數，1個學生有1.08個錄取名額，可提供學生多元選擇。今年桃園市高中高職博覽會共有39個學校參加，除了市立高中職及私立高中，也邀請中正預校、龍華科技大學、台北科技大學以及新生醫護管理專科學校等學校參加，希望讓大家看到高中、高職及五專最新的趨勢。

鄭文燦提及，為鼓勵高中生追求學習發展，今年將選出40位學生，在暑假期間赴美國史丹佛大學及柏克萊大學進行營隊體驗學習，其中20個名額開放評選競爭、16個名額提供給市立高中、4個名額提供清寒學生。若試辦成果良好，未來將擴大辦理。同時，市府亦和元智大學、美國麻省理工學院進行合作計畫，開放高中生、大學生在桃園市內大學舉辦營隊體驗學習。包括立法委員陳學聖、桃園市議員邱素芬、林俐玲、黃婉如、市府教育局長高安邦皆一同出席活動。
