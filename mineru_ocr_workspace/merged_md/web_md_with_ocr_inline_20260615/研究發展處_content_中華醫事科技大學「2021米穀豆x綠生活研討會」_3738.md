# 網站文章資料_中華醫事科技大學「2021米穀豆x綠生活研討會」_3738

## 一、文章基本資料

- **文章ID**: 3738
- **文章標題**: 中華醫事科技大學「2021米穀豆x綠生活研討會」
- **文章網址**: https://www.yzu.edu.tw/admin/rd/index.php/tw/?option=com_content&view=article&id=3738
- **文章簡介**: 主旨:

農委會農糧署與中華醫事科技大學共同舉辦「2021米穀豆x綠生活研討會」，敬請各校惠予公告。

說明:

一、活動內容：農委會農糧署與中華醫事科技大學共同舉辦 「2021米穀豆x綠生活研討會」，運用「食米、在地、低 碳、環保」4大特色，

結合餐飲示範講座與研討會論文成果發表之學術研討會。透過產官學食品及廚藝專家指導，持續協助臺灣優質農產，以建構綠色經濟循環，落實全民綠色生活。

二、主辦單位：行政院農委會農糧署、中華醫事科技大學民生學院、餐旅管理系

三、協辦單位：社團法人臺灣穀物產業發展協會

四、活動日期：中華民國110年5月19日(三)~ 5月20日(四)

五、活動時間：08:30~17:00，詳見議程表(如附件)。

六、辦理地點：

(一)、2021米穀豆x綠生活研討會：中華醫事科技大學民生科技大樓9樓餐旅多元專業示範教室。

(二)、2021米穀豆x綠生活研討會壁報論文發表：中華醫事科 技大學民生科技大樓10樓食尚智慧食宴室。

(三)、校址：717 臺南市仁德區文華一街89號。學校位置圖、交通指南如附件一所示。

七、參加對象：食品暨餐飲業者、高中職及大專院校教師。

八、報名方式：請點選下列網址或掃描附件中QRcode，研討會可報名1日或2日(https://forms.gle/MLs5zy1FQ1WbrFdT6)。

研討會名額有限，依各校系科、單 位及報名順序錄取，額滿為止，報名成功將寄發通知至您 的電子信箱。

九、其他活動事項請參考附件。
