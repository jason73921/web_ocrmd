# 網站文章資料_陌生領域從零開始-看SDN新手劉恩成如何躍身國際_1936

## 一、文章基本資料

- **文章ID**: 1936
- **文章標題**: 陌生領域從零開始-看SDN新手劉恩成如何躍身國際
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20170523-4
- **文章簡介**: 【中央社】

劉恩成，一個在兩年前剛踏入SDN（軟體定義網路）領域，還只是新手的他，目前已躍居全球僅七家SDN認證實驗室中，台灣唯一認證之國際窗口。

畢業於元智大學通訊博士班的劉恩成，就讀期間所做的學術研究為基於無線網路上的多媒體品質討論，而SDN對其而言是個新領域，有著新技術和新的知識，碰巧遇上Mobilheroes通訊大賽針對SDN的創新應用競賽，抱著挑戰的心態，劉恩成決定參賽。

SDN（軟體定義網路），劉恩成將其比喻成紅綠燈的明暗秒數控制權，從每一隻紅綠燈需要單獨設定，集中到決策中心統一指派。因此，決策中心可以依據每個紅綠燈回報的情況，即時調整每個紅綠燈的明暗秒數，有效解決汽車在道路上遇到的各種問題。將其轉換成專業名詞後，紅綠燈，意即交換器；決策中心，為控制器；而汽車，指的就是每個人發的封包。

透過專題實作，劉恩成找到自己的方向

回憶起求學經歷，劉恩成自大二起，因不懂自身科系未來走向，對自己的未來規劃感到十分困惑，但遇上當時的指導教授賴文彬後，他從一個不知道方向的學生，變成一個能腳踏實地、穩步前進的人，也從不懂規劃到能帶領好一個團隊，藉由專題實作，劉恩成快速累積自己的技術能量、時間管理，累積了不少知識，而他更從與學長姐的互動中，看到前輩的背影，開始能安排一個可行的未來。而後透過參加那場比賽，讓劉恩成得以參與國際論壇展示其作品，也因此受到交通大學網路測試中心主任許郡泓的賞識，得到踏入ONF（Open Networking Foundation，開放網路基金會）的機會，成為台灣唯一SDN認證實驗室的主要國際窗口。

三個月從零開始，與時間競賽

談起參加比賽的過程，劉恩成認為最大的困難在於時間壓力，當時距離比賽僅有三個月的時間，在這種兼具技術與知識的比賽中，除了實作上的技術困難，要如何呈現出一個同時具備技術實力以及知識深度的成果，是不小的挑戰。大量的英文文件閱讀能力、高品質的程式實作與軟硬體兼通的嵌入式技術，皆是劉恩成團隊遠遠不足的。劉恩成認為，能獲得全國亞軍的殊榮，通訊系與通訊研究中心提供的相關資源對他們幫助很大，能讓他們心無旁騖地投入比賽，他特別感謝指導教授賴文彬及甘堯江，認為元智大學在提攜學生方面，不遺餘力。

累積技術奠基知識，讓國際看見台灣

現身為台灣唯一認證實驗室之國際窗口，劉恩成期望未來能繼續帶著自身的經驗與知識朝國際發聲，讓台灣有機會在相關的領域發光，成為繼科技島後，能輸出專業知識的知識島，跳脫國際對於台灣的傳統思維；除此之外，更藉由產官學的互相搭配，培養專業人才，形成一個良性循環的人才島，期望人才相互合作，創造未來。

善用手邊工具，迅速累積實力

走過同樣的路，對於後進的學弟妹，劉恩成最大的建議是直接面對GOOGLE搜尋引擎。對於查得到、看得見的東西，要想辦法藉由搜尋快速掌握，累積實力；上手後，再開始處理那些搜尋不到的東西；那些搜尋不到的，才是有機會發光發熱的地方；放下搜尋引擎，踏出屬於自己的那一步，追尋自己的夢想。
