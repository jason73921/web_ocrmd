# 網站文章資料_教部補助型計畫將增行政管理費 支應水電、通訊_7395

## 一、文章基本資料

- **文章ID**: 7395
- **文章標題**: 教部補助型計畫將增行政管理費 支應水電、通訊
- **文章網址**: 
- **文章簡介**: 【中央社/記者許秩維台中25日電】

教育部長潘文忠今天表示，為減輕學校執行計畫時水電、通訊等支出負擔，補助型計畫將增列行政管理費項目，以業務費15%為編列基礎，委辦型計畫則取消行政管理費60萬元上限。

113年全國大專校院校長會議今天在台中市朝陽科技大學舉行第1天議程。

教育部長潘文忠表示，過去有大學校長反應，教育部補助型計畫時通常只編列人事費和業務費，但水電、通訊等費用也是一筆不小的支出，為減輕學校的負擔並支持計畫的運作，教育部將調整補助型計畫，增列行政管理費項目，以業務費的15%為編列基礎，至於委辦型計畫，則會取消行政管理費60萬元的上限，讓業務單位跟學校更有彈性規劃。

元智大學校長廖慶榮今天在報告高教留才攬才策略時提到，可從薪資待遇和制度環境著手，例如彈性薪資、宿舍等待遇，還有暢通的晉升管道、良好的教研環境等；政府也應提供協助，例如公立大專教師的學術加給今年調整15%就是好的政策，建議未來每幾年就調整一次專業加給，對大學延攬人才一定會有幫助。（待續）
