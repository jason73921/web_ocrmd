# 【顧問養成】輔導顧問實務分享—顧問產業現況 x 產官學鏈結

## 講座防疫辦理規劃

1. 於入場前量測體溫，規定額溫低於攝氏37.5 度才可入場

2. 課前準備酒精消毒，並提供給工作人員與參加教師使用

3. 每位參加教師與工作人員皆須填寫個人資料蒐集、處理及利用之告知暨同意書

4. 座位採梅花座管理，降低感染風險

5. 參加人數控制在50人以下

6. 授課時須戴口罩

7. 茶會點心以個別盒裝方式發放，無共用餐盤
