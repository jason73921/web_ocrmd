# 網站文章資料_科技部「108年度生命科學新進教師學者研習營」_2636

## 一、文章基本資料

- **文章ID**: 2636
- **文章標題**: 科技部「108年度生命科學新進教師學者研習營」
- **文章網址**: 
- **文章簡介**: 主旨：本部生命科學研究發展司訂於本(108)年9月16日、27日分別假集思臺大會議中心蘇格拉底廳、國立成功大學國際會議廳第二演講室(光復校區)舉辦「108年度生命科學新進教師學者研習營」，請惠予協助張貼海報並轉知所屬踴躍參加，至紉公誼。 說明： 　一、本部為培訓科學技術人員，促進科學技術人員之進用與交流，並鼓勵科學技術人員創業，特由生命科學研究發展司(下稱生科司)邀請國內知名學者及業界先進，以專題講座方式，傳授分享其計畫撰寫、團隊養成及創新研究之經驗與心得閱歷，引領新進教師學者以開闊多元的視野開展生涯。 　二、凡申請本部生科司「新進人員研究計畫」之新進學者，於計畫申請時須勾註是否曾參加過本研習營以為參考。本研習營參加對象以新進學者為優先，名額有限，請把握機會踴躍報名參加。 　三、本研習營採線上免費報名，請考量機關所在地及人數限制，就北部場(限150人)及南部場(限150人)擇一報名(不得重複報名)，請由本部首頁「最新消息」之「活動訊息」，點選本研習營活動訊息進入報名網頁，或逕至以下網址報名：https://reurl.cc/zvjAV(線上作業系統於108年7月29日開放)。 　四、檢附研習營海報1份，併附相關網址供參：

(一)本部首頁：https://www.most.gov.tw/

(二)本部生科司首頁：https://www.most.gov.tw/bio/ch

(三)研習營活動訊息：https://reurl.cc/zvjAV
