# 網站文章資料_【國立交通大學】與旺凌/網聯辦理「全球物聯網創新應用競賽」延長報名_2583

## 一、文章基本資料

- **文章ID**: 2583
- **文章標題**: 【國立交通大學】與旺凌/網聯辦理「全球物聯網創新應用競賽」延長報名
- **文章網址**: 
- **文章簡介**: 主旨：本校國際產學聯盟總中心與旺凌/網聯辦理「全球物聯網創新應用競賽」延長報名至2019年7月31日，請貴校協助轉知各大專院校學生踴躍報名參加，敬請查照。 

 說明：

  一、旨揭活動係由科技部指導辦理，透過網聯通訊的物聯網晶片及物聯網平台，舉辦本次物聯網創新應用競賽，希望能推廣物聯網晶片的應用，並促進學生能學以致用，將學習到的知識化為實際產品，贏得高額獎金，期以多方交流討論激發創新應用能量。  

二、時程：

  (一)2019年06月01日：開始網路報名 

 (二)2019年07月31日：報名截止 

 (三)2019年8月：培訓/技術諮詢期間與成果設計方向(AI/主控/雙模/低功耗)連結

  (四)2019年8月：實作專案 

 (五)2019年9月中：進行初賽篩選（進入決賽團隊有技術諮詢機會及評審建議）

  (六)2019年10月中：進行決賽 (取前3名及佳作5名) 

 三、培訓地點：國立交通大學（新竹市東區大學路1001號）

  四、獎勵辦法：

  (一)第一名：獎金15萬 + 獎狀

  (二)第二名：獎金10萬 + 獎狀

  (三)第三名：獎金5萬 + 獎狀

  (四)佳　作：獎品 + 獎狀

  五、報名網站：http://netlinkcompetition.blog.nctu.edu.tw
