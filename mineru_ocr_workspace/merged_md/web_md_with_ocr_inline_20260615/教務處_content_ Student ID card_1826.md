When is the student ID card typically issued for freshmen?


 

Generally, it will be distributed on the first day of school, and class representatives will be asked to collect them on behalf of their respective classes.




 

How to apply for a replacement if a student ID card is lost?


 






Please make sure to check your personal PORTAL for the photo. If there is no photo or if you want to change the original photo, please bring the electronic photo file to the Registration Section.

Go to the personal portal and select "Student ID Card" to complete the procedures for reporting loss and replacement. 

After completing the operation, download and fill out the "Loss and Refund Application Form," "Affidavit For Losing Student ID Card," and "Reissue/Data Correction of Student ID Card," totaling three forms.

The cost for replacing (or changing) the student ID card is NT$200.

Students should go to the Registration Section counter for processing.

After seven working days, go to the Registration Section counter to collect the replacement student ID.
