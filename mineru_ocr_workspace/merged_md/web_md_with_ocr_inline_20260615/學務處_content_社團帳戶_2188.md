社團帳戸

## 社團負責人換人了，社團帳戶怎麼辦？

A：可以申請【帳戶異動】，就是更換帳戶的負責人。請依照以下步驟辦理\~

1.填寫申請資料：【社團遠東商銀帳户異動申請單】、【社團遠東商銀帳户異動申請書】  
2.將以上2樣資料，繳交至課外組。(課外組會協助發公文）

3.待公文發下來了，課外組會通知社團負責來課外組領取公文。

4.請攜带：

• 公文正本

•帳戶異動申請單、申請書

社團大章+新任户名/負責印章

•雙證件正本+影本

社團帳户申請/異動表單

• 存摺

5.電話預約申辦時間（遠銀桃園分行03-3396339分機10479-廖小姐）

6.備妥以上物件、至遠東商銀桃園分行辦理

7.將新存簿的封面影本繳交至課外組(重要!)

8.完成了！

注意：帳户負责人必须年滿18歲才能辦理喔\~

![](images/d3c716416fd39ec207b55cf62319b3da9b82ea77bb53840554cf66249b7bce8b.jpg)

## 社團一定要有帳戶嗎？

A：如果社團有帳戶的話\~

1.社員缴交的社費，可存在社團帳戶，不用煩惱社費要放在哪裡\~

2.如果社團要用社費辦活動，領取的社费還有存簿明细可查，確保每分錢的去向。

3.如果社團有申請活動補助費、有申請校外指導老師費、或是得獎了有獎勵金或補助費等，學校可以直接將補助費匯款至社團帳戶，匯款時間及金額也清楚明瞭。

4.社團帳戶的有無，也是社團評鑑的評分項目喔，這麼好拿的分數，怎能錯過呢？而且擁有社團存簿的明細，在製作社團評鑑的經費資料時也不會慌了手腳喔～

還沒有社團帳戶的社團還不趕快辦起來？！

在課外組官網表單下載區或掃上方QR code

![](images/325132afc8368c3c9fd130ab972064931c2850884401d02c315bc9beac282d05.jpg)

## 社團開新帳戶，要準備什麼？

A：開立新帳户請照以下步驟辦理：1.填寫申请资料：【社團遠東商銀帳戶開戶申請單】【新開立遠東商銀帳號申請書】

2.將以上2樣資料，繳交至課外組。(課外組會協助發公文)

3.待公文發下來了，課外組會通知社團負責人來課外組領取公文。

4.請攜帶：

• 公文正本

• 開戶申請單、申請書

• 社團大章+新任戶名/負責人印章•雙證件正本+影本

5.備妥以上物件至遠銀桃園分行辦理（需事先電話預約03-3396339分機10479-廖小姐)

6.將存簿的封面影本缴交至課外組。  
7.完成了～

注意：帳户負責人必须
