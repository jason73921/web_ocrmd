# 網站文章資料_社團法人臺灣跨文化健康照顧學會「人體研究計畫諮詢取得原住民族同意與約定商業利益及其應用辦法」_3864

## 一、文章基本資料

- **文章ID**: 3864
- **文章標題**: 社團法人臺灣跨文化健康照顧學會「人體研究計畫諮詢取得原住民族同意與約定商業利益及其應用辦法」
- **文章網址**: https://www.yzu.edu.tw/admin/rd/index.php/tw/?option=com_content&view=article&id=3864
- **文章簡介**: 主旨:

有關本會辦理110年度原住民族委員會「人體研究計畫諮詢取得原住民族同意與約定商業利益及其應用辦法」，教育訓練乙案，敬請惠予公告，並鼓勵所屬同仁報名參加，請查照。

說明:

一、依據原住民族委員會「人體研究計畫諮詢取得原住民族同意推動計畫專案管理中心勞務採購案」（案號：109147）契約書辦理。

二、本會「110年人體研究計畫諮詢取得原住民同意推動計畫」 專案管理中心，為提升研究人員對原住民族傳統文化與自主權利之意識，

並能落實人體研究法第十五條保障研究對象之權益，以達到族群共享及共同參與為目的，特舉辦「人體研究計畫諮詢取得原住民族同意與約定商業利益及其應用辦法」之教育訓練。

三、時間：110年5月26日（三）9:00-17:00

四、地點：國立陽明交通大學 陽明校區第一會議室（地址：台北市北投區立農街二段155號）

五、報名方式：即日起至110年5月18日止，採線上報名，人數30名，額滿為止。詳情公告於專管中心官網(https://www.crbtzuchi.org)。

六、本活動完全免費，全程出席並簽到、簽退及參加課後測驗者核予6小時受訓證明，通過課後測驗核發2小時受訓證明（證書一律課後寄送）。

七、檢附議程表及交通位置圖乙份，如附件。
