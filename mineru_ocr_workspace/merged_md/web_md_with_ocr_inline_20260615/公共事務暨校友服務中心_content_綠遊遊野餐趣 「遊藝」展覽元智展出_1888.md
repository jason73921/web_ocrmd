# 網站文章資料_綠遊遊野餐趣 「遊藝」展覽元智展出_1888

## 一、文章基本資料

- **文章ID**: 1888
- **文章標題**: 綠遊遊野餐趣 「遊藝」展覽元智展出
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20170502
- **文章簡介**: 【中央社】
元智大學藝術中心以校園遊樂場為發想，自4月27日起至6月23日於元智大學校園內及圖書館美學之道舉辦「遊藝」藝術展覽，企圖打破傳統對藝術認知的拘束，讓藝術以玩味有趣的方式，重新進入我們的生活中。

元智大學藝術中心鄒淑慧主任表示，在我們熟知的藝術空間裡，氣氛總是安靜拘謹，甚且多半觀眾可能看不懂作品所要表達的是什麼，為翻轉這種有距離感的陌生氣氛，突破傳統展覽空間的形式，決定從關乎教育的校園開始做起。校園是學生最密切的學習場域，卻也是最多束縛的空間環境，透過啟動藝術家與觀眾的奇異想像，希望改造校園氛圍，讓它不再只有上課讀書與充滿考試壓力，而是成為一個輕鬆好玩、無牆無距離的開放式美術館。

本次展覽特別邀請八位不同藝術創作領域的藝術家，透過天馬行空的創意思維，帶領觀眾從室內到戶外，從平面、影像到立體創作，碰撞出不一樣的生命活力，刺激失去創意的生鏽腦袋，重新活化每個人潛藏的藝術DNA。除此之外，本展也同時展出元智大學藝設系學生以及桃園市南門國小師生、親子的共同創作，以不同的創意實際行動，顛覆既定的藝術想像，反轉校園每個角落，讓觀眾在看展覽時就像在遊樂場輕鬆嬉戲一般，處處都有驚奇的發現。

八位藝術家分別是黃柏勳、卓霈欣、陳興仲、黃惠成．游尊鈞、劉哲榮、鄭炳和、米各說及桃園市南門國小師生與家長；黃柏勳的作品，主要探討創作者自身與其所處生活環境之相互關係，樣貌繽紛、形式鮮明，迷離夢幻般的色調，透露出內心深處所感受到的漂移、孤獨與難以言說的不確定性。卓霈欣以鉛筆做為主要創作媒材，以人物為創作主題，從日常生活與內心的感觸出發，每幅作品都是與自我內心的對話，也期望能引起觀眾記憶與生活裡的共鳴。陳興仲的作品以鮮麗色調勾勒出高雄城市特色，頗具童畫之趣。作者於作品之中隱藏許多令人玩味的符碼及視覺語彙，增添更多層意涵，期使作品能與觀眾產生互動。

黃惠成使用大量符號，影像經拆解重構並組織成新的圖像，這些「曾經的實體」，透過交混（hybrid）作用，產生互話，賦予彼此意義，隱現港人生活就是被壓縮的城市現象。游尊鈞透過鮮明、活潑的色彩，將生活中感到煩悶、無趣的灰暗感翻轉成明亮的世界，激起觀眾內心的赤子之心。劉哲榮的作品主要探討人類在面臨保育及方便性所衍生出的最後形式，創作不僅是內在思維的辯證與反映，更在於探索自身經歷、社會環境中的交錯現象及未來的前瞻性。

鄭炳和係豐年祭時，有感於人們對天地的尊敬，對祖靈的追遠，對長者的倫理，對環境的秩序，這種天人合諧的文化，激發創作靈感，透過環保機械人表達。米各說則以動畫為主，透過不同動物的故事寓意，表達獨特的創作風格與觀點、溫暖的調性及逗趣的內容，為不一樣的聲音提供分享的管道，找到屬於各自的自在空間。桃園市南門國小師生、親子的共同創作，從環保觀念結合藝術創意出發，透過天馬行空的想像力，合力將報廢木頭課桌椅變身為令人驚豔的創作，充滿奇麗的異想與趣味，讓校園有了不一樣的風情故事。
