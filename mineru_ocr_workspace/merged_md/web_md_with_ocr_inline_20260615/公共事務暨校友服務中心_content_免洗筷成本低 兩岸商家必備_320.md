# 網站文章資料_免洗筷成本低 兩岸商家必備_320

## 一、文章基本資料

- **文章ID**: 320
- **文章標題**: 免洗筷成本低 兩岸商家必備
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【中時電子報】 


免洗筷成本低廉，是兩岸餐飲業者最愛。免洗筷便宜得超乎想像，在大陸市場一大箱3500雙免洗筷僅賣300元台幣，生產業者不諱言，生產過程用化學物替筷子「整容」是業內公開祕密。

儘管大陸各地宣導禁用免洗筷，免洗筷仍在餐飲界大行其道，成為大大小小餐廳內必備「道具」，成本便宜，替商家省去不必要開銷。

快餐業者坦言，顧客反而不愛用重複使用的消毒餐具，一個月差不多要用掉5000多雙免洗筷。重複使用餐具，業者必須另外支付消毒、清洗、人力成本。免洗筷供應商1雙才2分錢（人民幣，下同）左右，換算下來買3500雙只要65元，好一點的3600雙105元。

免洗筷批發商也坦言，大陸市售免洗筷又分多種「檔次」，最好的外頭有塑膠袋包裝，每10雙或20雙有獨立包裝，上頭也印有生產日期，多提供給超市。另一種大布袋裝的提供給餐館，最差的就是提供給路邊小吃店、早餐店的，沒有任何包裝，也沒有生產日期。

製造商也不諱言「不加工竹筷哪能這麼整齊一致！」透露這些筷子多半都用化工原料、硫磺加工過，如果不用硫磺燻、雙氧水漂白就是黑色的，賣相不佳。

環保署統計，台灣每年約用掉45億雙免洗筷，總長度足以繞地球20圈。在各地夜市、小吃攤、早餐店，免洗筷隨處可見。

台灣多所大專院校、中小學近年開始宣導禁用免洗餐具理念，台大、政大、元智、東吳等院校，要求校內餐飲業者不得提供免洗碗筷，或是採「付費制」，索取免洗餐具要付1元台幣。但整體而言，宣導力度仍不夠，校內學生要取得免洗筷仍十分容易。(旺報)
