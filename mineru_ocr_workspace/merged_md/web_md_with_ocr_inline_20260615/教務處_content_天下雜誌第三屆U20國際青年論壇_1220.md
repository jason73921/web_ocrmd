# 網站文章資料_天下雜誌第三屆U20國際青年論壇_1220

## 一、文章基本資料

- **文章ID**: 1220
- **文章標題**: 天下雜誌第三屆U20國際青年論壇
- **文章網址**: https://www.yzu.edu.tw/admin/aa/index.php/tw/u20
- **文章簡介**: 正當全球企業加速遵循ESG永續指標，尋求新興的永續商業模式之際，有一群地球新聲代、20歲上下的青年（U20，Unique 20），面對氣候危機早已是日常，更自然關注永續發展。

2022年U20國際青年論壇邁入第三屆，持續廣邀12~30歲對「促進環境永續」、「減緩社會不平等」、「地方創生」三大議題有所行動或想法的青年進行徵件，希望號召台灣有行動力的青年、產業、政府共同解答這個問題，入選就有機會站上國際論壇，與政府、企業共同倡議對談。

【徵件活動說明】


活動網址：https://csr.cw.com.tw/feature/2022U20

活動目的：讓青世代大聲說出自己的永續地球觀、減緩社會不平、地方創生等行動，一同跟企業、政府溝通，開展跨領域、跨世代交流

徵件對象：12-30歲的青世代青年，且符合以下任一條件


           - 你在任一領域（逆轉氣候變遷、減緩社會不平等、地方創生）已有些許影響力，在學校社團或是朋友之間，讓身邊的人們動起來

           - 你開始了行動的一小步，希望更多人知道你所關注的議題，並且加入你的行動

           - 你已經展開完整行動，渴望突破同溫層，有機會與企業或政府單位合作


徵件截止日期：2022/7/15（五）14:00

獎勵辦法：
1.有機會成為CSR@天下報導案例，請由此觀看學長姐報導。
 2.入選團隊(總共15組)，可參加3大主題工作坊，永續企業家、青年創業家將與你面對面分享經驗，並針對你的行動方案提出建議，也有機會合作。
 3.工作坊後將選出6組青年在10月初「地球的新聲」U20國際青年論壇上發表行動理念，並獲得由信義房屋社區一家、天下雜誌提供的獎金，每組獎金各1.5萬。過去的青年代表演講影音請由此連結參考：https://reurl.cc/9GbgYY

報名流程：https://reurl.cc/VDmk2y（點選完成基本資料後，可獲得完整徵件報名文件）
