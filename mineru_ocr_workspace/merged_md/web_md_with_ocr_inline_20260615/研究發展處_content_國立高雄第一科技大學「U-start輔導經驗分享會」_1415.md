# 網站文章資料_國立高雄第一科技大學「U-start輔導經驗分享會」_1415

## 一、文章基本資料

- **文章ID**: 1415
- **文章標題**: 國立高雄第一科技大學「U-start輔導經驗分享會」
- **文章網址**: 
- **文章簡介**: 說明：

一、 為形塑發展特色，本校在2010年宣示轉型為「創業型大學」，透過建置完整的創新創業生態系統，近年已累積四次教育部U-start計畫全國獲選隊數第一，並且104年~106年連續三年獲選隊數全國第一，同時扶植多家校園師生衍生企業，期透過本次U-start輔導經驗分享會，將本校推動創新創業生態系統與師生創業輔導經驗與全國大專院校進行分享，邀集全國大專院校共同交流校園創新創業氛圍建立與成效擴散。



二、 分享會訊息如下：

  (一)時間：106年12月11日(一)下午1時至5時20分

  (二)指導單位：教育部技術及職業教育司、教育部青年發展署

  (三)活動對象：全國大專院校推動創新創業相關教師職員

  (四)地點：第一科大圖書資訊館B1樓創意教學室(824高雄市燕巢區大學路一號)

  (五)線上報名網址：https://ppt.cc/feoSxx
