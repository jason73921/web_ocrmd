Wifi use precautions:


This account is only for alumni to apply, please provide complete information to confirm your identity.

It takes about one week to verify Wi-Fi Application, we will get back to you by E-mail.

One person can only apply for one account. The valid period is limited to one season. After the use permission expires, a new application is required.

Before connecting to the wireless network, please make sure that your computer system is not infected with computer viruses and that legal anti-virus software has been installed.

The use of campus network should comply with the "Taiwan Academic Network Management Regulations" and the "Yuan Ze University Network Usage Management Measures".

It is prohibited to use the Internet to transmit threatening, obscene, and unfriendly information.

The use of P2P software is prohibited.
