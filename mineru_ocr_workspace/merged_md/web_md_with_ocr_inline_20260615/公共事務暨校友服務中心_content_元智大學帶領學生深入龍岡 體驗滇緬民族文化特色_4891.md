# 網站文章資料_元智大學帶領學生深入龍岡 體驗滇緬民族文化特色_4891

## 一、文章基本資料

- **文章ID**: 4891
- **文章標題**: 元智大學帶領學生深入龍岡 體驗滇緬民族文化特色
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20210126-1
- **文章簡介**: 【桃園電子報記者/諶志明】

元智大學日前舉辦第三屆「異域歲月：青年田調人才培力工作坊」三天二夜活動，帶領學生深入龍岡區域，住在「龍岡清真寺」、拜訪穆斯林家庭，體驗與臺灣主流文化不同的料理與滇緬少數民族特色。

元智大學表示，龍岡是中壢的軍事設施集中區，不僅許多軍事單位設於此，1950年代以來更有大量眷村設立，成為中壢除市區、內壢之外，另一個主要的人口聚集區，也是臺灣眷村密度最高的地方。龍岡近年以「米干節」打出名號，但民眾尚未普遍認識當地豐富而且值得發掘的地方特色，以及來自異域的動人故事，往往以一嚐米干湯頭的鮮美，做為瞭解當地文化的第一步，但你吃過大薄片、豌豆粉、破酥包等滇緬美食嗎？這些獨特料理撐起了當地的滇緬美食經濟。

元智大學通識教學部的桃園文化廊道USR計畫，在主任梁家祺帶領下，安排學員住宿龍岡清真寺宣禮樓，進行在地文化講座、文化料理體驗、分組田調觀察、簡報與書寫競賽等各種課程，這一屆還加入了提案教學，期許學員在進行田野調查訪談之後，能對該主題所理解的現況，進而提出改善或推廣的方案，活用所學到知識與技巧。梁主任表示，以夜宿龍岡清真寺的號召力非常強，並由熟悉在地文化的業師帶領學員探索龍岡地區，因而獲得校內學生熱烈響應。
