# 網站文章資料_人工智慧檢視人流 獲產學應用價值獎_2489

## 一、文章基本資料

- **文章ID**: 2489
- **文章標題**: 人工智慧檢視人流 獲產學應用價值獎
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20171220-2
- **文章簡介**: 【中央社記者許秩維台北19日電】

元智大學團隊用監視攝影機，搭配電腦視覺與人工智慧技術，輔助博物館或展覽空間進行遊客行為的追蹤分析，助節省人力和設備支出，獲最佳產學應用價值獎。

元智大學今天發布新聞稿表示，元智大學資工系學生林毓宸及姚雅馨，在教師葉奕成指導下，與國家實驗研究院和基隆海洋科技博物館合作，參加全國大專校院開源軟體程式創意設計競賽，以作品「那些年我們一起追的骨架：用開源工具透視博物館人流」獲最佳產學應用價值獎。

元智資訊工程系教師葉奕成表示，作品主題就是使用開源工具來透視博物館人流，透過電腦視覺與人工智慧技術，輔助博物館或展覽空間進行遊客行為的追蹤、分析跟記錄，希望協助展覽及策展者能更進一步的瞭解參觀者以及參觀狀況。(待續)
