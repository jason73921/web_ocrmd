# 網站文章資料_【函轉】全民即日起踴躍參加全國首創「書香社區認證」活動_727

## 一、文章基本資料

- **文章ID**: 727
- **文章標題**: 【函轉】全民即日起踴躍參加全國首創「書香社區認證」活動
- **文章網址**: https://www.yzu.edu.tw/library/index.php/tw/2017071902
- **文章簡介**: 為協助我國各地社區致力於建構書香社會、促進閱讀風氣及培養定期閱讀習慣，國家圖書館特別推動「書香社區認證計畫」，並建置「書香社區認證網站」，規劃健全的評比指標與評核制度，各地民眾皆可自由登入認證網站，填寫各項指標資料。經由各地社區居民之熱烈參與，一方面作為每位民眾自我檢視閱讀情形及接續精進閱讀素養的參考；另一方面則經過統計分析，呈現全國各地的閱讀地圖，有助於政府與民間團體合作規劃各項閱讀政策，協助各地社區將閱讀習慣帶入在地居民的日常生活中。
「書香社區認證計畫」之推動主要係鼓勵各地社區於推廣圖書及促進閱讀的工作上進行創新與變革，期以創造臺灣豐富性與多樣性的閱讀文化，以閱讀軟實力展現國家競爭力。
因此特別歡迎每位民眾即日起踴躍上網（http://scc.ncl.edu.tw/book/）進行閱讀認證，國家圖書館亦將統計分析全國民眾分齡認證結果，辦理閱讀績優社區表揚及頒獎大會、頒發績優認證社區獎項，藉以鼓勵各地社區積極提升民眾閱讀力。
書香社區認證網址：http://scc.ncl.edu.tw/book/
