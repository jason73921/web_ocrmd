# 網站文章資料_財團法人雪心文教基金會  「111年大專院校學生暑期及學期實習辦法」_4597

## 一、文章基本資料

- **文章ID**: 4597
- **文章標題**: 財團法人雪心文教基金會  「111年大專院校學生暑期及學期實習辦法」
- **文章網址**: https://www.yzu.edu.tw/admin/rd/index.php/tw/?option=com_content&view=article&id=4597
- **文章簡介**: 主旨：本會為提供大專院校學生至非營利組織實習機會，規劃111年大專院校學生暑期及學期實習辦法，詳如說明，敬請 貴校協助轉知學生參加。

說明：


一、本會成立宗旨秉持故雪廬老人(李炳南教授)慈悲度眾的精神及行事，延續雪廬老人弘法利生的工作。
積極規劃推動社會教育與終身學習等各項公益文教活動，更積極培育大專青年人才從事志工教育服務，提供暑期及學期實習的機會給予學生。二、實習地點：本會自有場地「雪心文教基金會」(臺中市東區立德街197號，(04)2280-8260)及「雪廬山」(苗栗縣卓蘭鎮坪林里象山50號，電話(04)-2592-0410)，或本會各縣市連體機構。實習內容及相關申請辦法，請參閱本會網站(http://www.shiushin.org.tw/)。
三、申請方式：請至本會網站上填寫「學生實習個人履歷表」(請填寫期限至111年6月5日晚上12:00截止)，填妥後請交回本會，由本會「策畫推廣部」安排線上面試(另行通知面試日期及GoogleMeet網址)，確認錄取後，須由學校系辦公室委函向本會提出實習申請。
四、暑假實習期間：自民國111年6月27日起至8月31日止。
五、本會承辦人：張力中專員電話：(04)2280-8260#915，電子郵件：kindlytess@gmail.com。
六、檢附：111年大專院校學生暑期及學期實習辦法1份(如附件一)，敬請轉知學生參閱辦理。
