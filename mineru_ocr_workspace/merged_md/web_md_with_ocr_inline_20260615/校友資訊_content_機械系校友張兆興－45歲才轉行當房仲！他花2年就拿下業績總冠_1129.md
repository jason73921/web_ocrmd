# 網站文章資料_機械系校友張兆興－45歲才轉行當房仲！他花2年就拿下業績總冠軍 祕訣曝光_1129

## 一、文章基本資料

- **文章ID**: 1129
- **文章標題**: 機械系校友張兆興－45歲才轉行當房仲！他花2年就拿下業績總冠軍 祕訣曝光
- **文章網址**: https://www.yzu.edu.tw/alumni/index.php/tw/20241225
- **文章簡介**: 【經濟日報／楊伶雯】

 

45歲進入房仲業，2年內拿下業績總冠軍，入行4年後取得博士學位，身為房仲業高材生的台灣房屋八德興豐直營店店長張兆興，堪稱時間管理大師，新人時期連續3個月每天扛著廣告A字架，拿著文宣，從桃園至大溪每天換十字路口站崗，走遍八德區大小公園及市場，晚上下班揹著開發信，從八德跑步7公里到大溪，沿途更換社區發放DM，工作當健身，僅9個工作周完成39.5件合格專任委託，奪得台灣房屋集團進案王第二名。




出身軍眷書香世家，張兆興就讀軍校時已讀到碩士，17餘年軍旅生涯中，他始終謹記父親曾說家族世代均從事教職工作，為延續傳統，他轉任軍訓教官，展開7年的教職工作。「但當初投考軍校薪水2萬1，我母親說連她在傳統市場做生意，隨便都有5萬，讓我發現如果靠領薪水，永遠沒辦法財富自由。」。於是才40多歲他就從教職退休，因一部描述「老鷹的一生」影片帶給他強大震撼與啟發，內容述說老鷹在30歲時，會歷經一年的時間更新腳爪、啄，以及笨重的舊羽毛，當老鷹再度銳變後，能再度展翅高飛30年。

「為迎接下半人生更強大的開始，我決心加入台灣房屋。」張兆興表示，選擇台灣房屋開啟事業第二春，是因進公司前也有投資買賣房子的經驗，認為自己對房地產並不陌生，進公司後才知道，受過完整的教育訓練，才真正彌補了過去所欠缺的專業。公司首重交易安全，一定要能保障雙方才進行買賣，不讓客戶承擔風險，每每看到客戶圓夢般的喜悅，帶給他更多的使命感與增添工作價值感。

45歲轉行，起初並不容易，因當時適逢政府打房交易量縮，他每天從早到晚如軍人般自律的排滿開發行程，不僅業績亮眼，在工作忙碌之餘同時攻讀博士學位，完成期刊發表、發明專利及博士論文，成為元智大學機械工程博士。「其實最要感謝台灣房屋提供新人5萬元豐厚底薪，才能讓新人安心衝刺與成長。」張兆興回憶自己邊讀書邊工作的新人時期，對公司滿懷感謝之情。

軍人出身的他自律嚴謹，對工作實事求是，正直負責的態度讓他擁有許多回流客，許多客戶聽從他的建議買房後，由小資族一路由小換大，透過買屋賣屋的換屋過程，逐漸達成財富自由。張兆興觀察，其實透過房地產致富者，一開始都未必是有錢人。能成功致富，他們共同擁有的，就是相信專業，以及勇於出手的魄力。

「常常我建議後客戶馬上就買，我都問他們怎能決定這麼快？客戶說因為跟台灣房屋交易很安心，有一種自然的幸福感。」張兆興說，透過台灣房屋集團提供強大資源與專業分析，讓買屋賣屋如同老鷹般，精準出擊。
