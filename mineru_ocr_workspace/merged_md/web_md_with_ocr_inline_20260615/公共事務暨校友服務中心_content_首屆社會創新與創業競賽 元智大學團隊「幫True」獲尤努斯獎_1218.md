# 網站文章資料_首屆社會創新與創業競賽 元智大學團隊「幫True」獲尤努斯獎亞軍殊榮_1218

## 一、文章基本資料

- **文章ID**: 1218
- **文章標題**: 首屆社會創新與創業競賽 元智大學團隊「幫True」獲尤努斯獎亞軍殊榮
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【中央社】

元智大學管理學院組織管理學群徐樂文及翁晶晶老師，指導學生企管大三黃翊寧、英專大三陳又誠及英專大二張沁瑜，組成元智大學團隊「幫True」，參加「2016社會型企業青年論壇暨尤努斯獎：第一屆社會創新與創業競賽」榮獲亞軍，全國第二名，獎金8萬元。 


這次競賽是由勞動部與中央大學共同合作舉辦，並邀請諾貝爾和平獎得主尤努斯博士親自擔任評審並頒獎，創業競賽以社企之父尤努斯為名，打破舊有社企框架，從新塑造對於社企的價值。期許在政府的推動下，將社會企業的概念擴大推廣，並擁有相關的配套措施協助青年利用創新的商業模式投入社企及創業。 

元智大學管理學院英專班二年級張沁瑜表示，元智大學團隊「幫True」，是以改善食物浪費問題與改善市場不公為出發點，秉持著零污染、零浪費、零剝削、零失業率的「四零」原則，以合理的價格向果農收購落果或是外觀不良的水果，再加以加工；加工的程序以人工為主，徹底的利用物料，將污染與浪費降到最低。 


張沁瑜說，重視環境發展的同時，幫True也釋出工作機會到市場，提升社會的就業率；最重要的是改善交易市場的不公，不再讓中盤商的壟斷影響果農的生計。幫True的商 


業模式以尤努斯博士的七大原則為基準，將經營重點放在社會回饋與改善而非營利，希望能夠透過這樣的運作模式能對社會盡一份力，一同讓這個社會變得更好。 

元智英專班三年級陳又誠談到，社企在台灣是一項非常新穎的企業，能夠同時解決社會問題且要有一定的獲利，可創造雙贏局面。企管系三年級黃翊寧說，這次的比賽除了是一種經驗外，擴大瞭解社會的其他層面，這場比賽最珍惜的是夥伴間的情誼，與激發出的創意火花，並從中體驗社會企業的重要。 

元智管院指導老師徐樂文表示，「賺錢做好事,做好事賺錢」難的其實不只是「做好事」，而是 「如何賺錢」讓它在「做好事」的同時，也能系統性地創造出「利潤 」“profitable”。這個比散訓練學生的創意，建立一個生態系統(eco-system)， 讓不起眼的水果(浪費)，鏈結到(污染)、(剝削)與(失業)，得以建立平台(platform)，產生系統性的良性循環 (Loop)，進而產生出(利潤)，再把此(利潤)投入創造出解決核心問題的(規模)(Scale)。整個比賽最難之處， 當然是起心動念(做好事)是最重要。



元智大學團隊「幫True」獲尤努斯獎亞軍殊榮，指導老師徐樂文、英專班大三陳又誠、企管大三黃翊寧、英專大二張沁瑜(由左到右). 


諾貝爾和平獎得主尤努斯博士(中)親自擔任評審並頒獎
