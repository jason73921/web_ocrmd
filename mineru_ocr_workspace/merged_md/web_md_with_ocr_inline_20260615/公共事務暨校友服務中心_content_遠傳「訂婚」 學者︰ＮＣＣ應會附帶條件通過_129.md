# 網站文章資料_遠傳「訂婚」 學者︰ＮＣＣ應會附帶條件通過_129

- **文章標題**: 遠傳「訂婚」 學者︰ＮＣＣ應會附帶條件通過
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【自由時報/記者蔡穎/台北報導】

遠傳將「繞道」併購中嘉網路，國家通訊傳播委員會（ＮＣＣ）昨表示，尚未收到併購申請案，無法評論，未來審查時將依《有線廣播電視法》檢視股東結構，以及遠傳在併購案中的角色；至於「黨政軍條款」問題，ＮＣＣ副主委虞孝成說，每一個併購案的條件不同，投資結構要等申請書送來才知道，「現在無法評論」。 

元智大學大數據創新中心教授葉志良分析，三大電信業者中僅遠傳獨缺有線電視系統，與中嘉策略結盟並不讓人意外；反倒是遠傳僅握有債權，如何實質影響中嘉決策引人好奇。

他表示，私募基金如凱雷、安博凱過去來台併購有線系統卻不積極建設，炒作心態引人詬病，希望ＮＣＣ以健全產業發展為出發點審慎處理併購案。

一位不願具名的學界人士指出，此併購案是「雷聲大、雨點小」，初步看來僅為更緊密的策略聯盟，且遠傳出資不到兩百億元也遠低於外界預期；該知情人士說，依照ＮＣＣ過去審查旺中案、台固案等有線系統併購經驗，應會對私募基金加上附帶條件才通過，例如要求數位化具體時程、質押比和網路投資承諾等。

電信同業看待併購案也顯得異常「淡定」。一位高層表示，眾多電訊傳播業者還在尋找新的商業模式，現在網路上什麼服務都有，國內業者真正的競爭對手是亞馬遜、蘋果和中國視頻網站等；他表示，「黨政軍條款」是不合時宜的法規，呼籲各界促成「廣電三法修法」，及早放寬投資限制。
