# 藝術中心展覽

《串臉藝起來》

人人生而平等，平等的根本之道在於包容一切、無分好壞美醜。本展覽目的在於推廣目前全球盛行的「脸部平權觀。神经纖维瘤（Neurofibromatosis，簡称NF）朋友因病情影響導致臉部面容有別於常人，在社會中常被視為邊緣人，因此希望透過此展覽，呼大眾，懷抱同理心和關懷心，平等對待NF朋友，不再以異樣眼光看待他們。且讓我們在萬物平等的藝術世界裡，攜手共同創作那些閃耀著愛的符號，鼓勵他們勇敢走出灰調人生，書出属於每個人獨特且期待被分享的心路故事。並願我们以堅石的信念碰撞出宇宙的新生命光芒，照亮更開闊更不一樣的視野及展望，讓美麗的彩虹橋在你我心靈之間築起，也讓台湾變成一個更懂得尊重和包容的友善社會。

開幕茶會：105年11月24日（四）13：00

茶會地點：元智大學五館一楼圖書館前廣場

展覽時間：105年11月24日（四）至106年1月18日（三）

開放時間：週一至週六9：00-17：00週日休館

展覽地點：元智大學圖書馆美學之道

主辦單位：藝術中心藝術與設計學系藝術管理碩士班圖書資訊服務處

协辦單位：社團法人台灣神经纖维瘤協會

補助單位：教育部「再造人文社會科學教育發展計畫
