# 網站文章資料_亞洲大學「大數據與人工智慧國際高峰論壇—智慧醫療與金融科技」_960

## 一、文章基本資料

- **文章ID**: 960
- **文章標題**: 亞洲大學「大數據與人工智慧國際高峰論壇—智慧醫療與金融科技」
- **文章網址**: https://www.yzu.edu.tw/admin/rd/index.php/tw/?option=com_content&view=article&id=960
- **文章簡介**: 主旨：本校謹訂於106年4月13日舉辦「大數據與人工智慧國際高峰論壇—智慧醫療與金融科技」，詳如說明，請惠予轉知所屬單位踴躍報名，請查照。

說明：

一、 在全球產業高度關注「大數據與人工智慧」之下，我國政府也正大力推動相關的政策與投資，促使臺灣能在此領域中扮演關鍵的角色。醫療及健康產業一直以來，是臺灣的極大優勢，且亞洲大學在與亞洲大學附屬醫院的整合優勢下，期望能在智慧醫療與智慧健康領域與各位先進一同合作累積研究能量與人才。同時，「金融科技」發展勢不可擋，新興金融科技興起已衝擊傳統金融機構的經營，金融業轉型已勢在必行。金管會發布的「金融科技發展策略白皮書」，更提出11項重要施政目標，加速深化金融產業與資通訊產業的跨業合作，共同打造完善的金融科技生態圈。

二、 亞洲大學因應全球市場脈動，訂於106年4月13日（星期四）一日，假台中金典酒店舉辦「大數據與人工智慧國際高峰論壇—智慧醫療與金融科技」。會中特邀請臺灣衛福部、美國史丹佛大學、新加坡南洋理工大學、金管會、立法委員、安侯建業KPMG、勤業眾信Deloitt、臺灣金融科技公司Fusions360、臺灣證券交易所、中研院等專家，深入剖析全球趨勢、案例與臺灣的機會和挑戰。

三、附件：邀請函、DM
