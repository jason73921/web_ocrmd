就礙找碴
活動方式：為元智無障礙環境健檢

活動時間：5/8~5/19

活動地點：圖書館Facebook粉絲專頁







身心障礙者的處境為何？外界的障礙在哪裡？同理心為障礙者設想，在校園環境內找出障礙地點，例如：校園內高起門檻、水溝蓋間隙、標誌等，未符合建築物無障礙設施設計規範的設施，希望能改善校園無障礙環境設施，使每一個人均能於校園內通行無礙。

在本篇下只要按讚+留言貼上校園環境內(1)障礙地點、(2)障礙說明、(3)障礙照片，並於文末加註您的學(帳)號，不得複製他人留言與照片，與前面留言者同一張障礙照片不予計算，活動結束抽出Family Mart 100元卷計15名；威秀影城電影票一組1名；休閒流行布包1名

## 認識身心障礙有獎徵答

你對「障礙」與「身心障礙者」了解多少，來吧! 看看你能答對幾題?

以下描述你認為正確的請選「O」，錯誤請選「X」。

一、( 0 )台灣的身心障礙人口有一百一十七萬多人，佔總人口比例 4.97%。

解答說明：截至105年台灣衛福部的人口統計，台灣領有身心障礙手冊的身心障礙人口有一百一十七萬多人，佔總人口比例 4.97%。

二、( X )台灣有約 6萬人有嚴重的視覺障礙，在與他們互動時，由於他們看不見，也會影響聽力，因此對他們說話時要特別大聲一點。

解答說明：台灣有約6萬人有嚴重的視覺障礙，在與他們互動時，可注意:

1. 他們並不會因為看不見，而影響聽力，說話時不需特別大聲，以正常的語調和速度直接與其交談即可。

2. 請不要跟正在工作的導盲犬玩。

3. 引導視障者時可讓他(她)搭你(妳)的手臂。如果遇到階梯或其他障礙，請暫 停一下，確認後再往前行。

4. 進入或離開房間請記得告知。

三、( X )所有的身心障礙者都對自己的障礙感到遺憾與痛苦。

解答說明：並不是所有的身心障礙者都對自己的障礙感到遺憾或痛苦，有些身心障礙者不但不會感到遺憾，甚至覺得自己的生命經驗因此更豐富。

四、( X )對於耳聾聽力障礙的人而言，主要問題不是他們無法聽到，而是他們所面臨的挑戰，如果與他們溝通時有口譯員在旁協助，請直接與口譯員溝通，而不是與該聾人交談。

解答說明：如果與他們溝通時有口譯員在旁協助，請直接面對聾人交談，而不是直接與口譯員溝通。並請避免站在光源或窗戶前面，可能會使你的臉變皺，難以讓聾人清楚地看到你。

五、( 0 )有學習障礙的人並不代表弱智。事實上，大多數學習障礙的人具有正常的智力，完全有能力執行不受其殘障阻礙的複雜任務，透過合適的教學策略可以幫助學習障礙的學習者學習適應和執行與同齡者相當的學術水平。

解答說明：學習障礙或學習困難（Learning disability），又稱特殊學習需要，是指儘管智商沒有問題，也沒有發展遲緩，但是在聽力、會話能力、閱讀能力、書寫能力、計算能力、推理和推論能力這些特定領域上，學習或使用上卻出現明顯問題而引致學習困難。

雖然學習障礙因為腦部功能的不同，使其在資訊的接收和處理上異於常人，而無法在一般傳統教學下充分學習，但是透過有效的教學策略，學習障礙者一樣可以超越障礙，發揮潛在的能力。

六、( X )人們應該主動來協助身心障礙者，只要看到身心障礙者，不需要詢問太多，就應該主動的來幫助他們。

解答說明：幫助身心障礙者之前一定要先詢問，你需要幫忙嗎?我要怎麼樣協助你?因為每一個身心障礙者的狀況與需要是不一樣的，不要以自已的想法去推論。

七、( X )要儘量鼓勵憂鬱症者不要想太多，並與精神分裂症者爭辯他的想法是有問題的，才是對他們最好的幫助。

解答說明：不要想「起動」憂鬱症的當事人，不要想「壓制」燥鬱症的當事人，也不要和精神分裂症的當事人「爭辯」他的妄想內容，幫他們學習這些行為是因疾病所引起的，告訴他們這不是他的錯，如果憂鬱一直去不掉，也不是他差勁，這種支持性的表達，會解除許多當事人的焦慮、內疚和自責，即使當事人有否認的反應，我們也要予以了解。

八、( X )一旦被診斷為智能障礙，將終生都會維持原本的智能障礙程度。

解答說明：人的心智功能(mental functional)程度是會改變的，特別是輕度智能障礙者。在實施密集的教育方案之後，有些人也會進步並改善障礙的程度。

九、( X )身心障礙者在工作中受挫的原因，經常是由於缺乏足夠的工作技能。

解答說明：身心障礙者在工作中受挫的原因，並非只是缺乏足夠的工作技能，很多時候是來自不友善的環境，例如: 雇主的歧視、工作環境的障礙、沒有合適的工作內容。

十、( X )身心障礙者的困境與障礙，都是由於他們身體的缺陷影響所造成的。

解答說明：他們的困境與障礙實際上來自於社會，一旦社會將以上這些障礙排除，身心障礙者的困境與障礙就不會存在。

第一種是物理性的障礙，我們眼睛所能見的各種『有障礙環境』。人類已經可以太空旅行，但使用輪椅的身障者卻還出不了大臺北。

第二種是態度性的障礙，這種障礙存在於人心，看不見，但能感受得到這種不平等的歧視。我們每個人嘴巴上都會說要支持弱勢，但心裡的接納程度卻不見得如此。

第三種是制度性的障礙。舉例來說，障礙者的升學處境，並沒有被限制不可以去唸大學，但是必須看該所大學有沒有開缺、看該系所決定他們要收什麼障別。
