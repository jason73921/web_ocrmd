# 網站文章資料_開南大學「2019當前各國企業福利制度回顧與發展學術國際研討會」_2484

## 一、文章基本資料

- **文章ID**: 2484
- **文章標題**: 開南大學「2019當前各國企業福利制度回顧與發展學術國際研討會」
- **文章網址**: 
- **文章簡介**: 主旨：本校辦理「2019當前各國企業福利制度回顧與發展學術國際研討會」徵稿啟事，請查照。
說明：
一、目的：本研討會目的瞭解各國企業辦理福利制度之內涵及實施現況、企業組織特性與福利間之關聯、探討各國企業組織特性與綜合福利水準之關係及推動福祉政策及工會團體參與企業福利運作情況，提供政府推動勞動福祉政策評估機制及工會福利制度架構及研提保障國內勞動權益的具體建議。
二、會議子題：
(一)各國企業組織推動福利制度之個案研究。
(二)各國政府推動勞動福祉政策評估機制之研究。
(三)工會團體參與企業福利運作架構之研究。
三、會議時間：中華民國108年5月31(星期五)。
四、會議地點：開南大學至誠樓A105國際會議廳。
五、徵文對象：
(一)國內外大學之教授及碩博士班在學研究生。
(二)對本議題有興趣之專家、企業工作者。
六、投稿時間：稿件全文自即日起至2019年5月25日止，以郵戳為憑，向開南大學進修及推廣教育處投稿。
七、通訊地址：桃園市蘆竹區開南路1號（進修及推廣教育處）。
