# 網站文章資料_黑天鵝來了？ 川普將讓世界大地震_1432

## 一、文章基本資料

- **文章ID**: 1432
- **文章標題**: 黑天鵝來了？ 川普將讓世界大地震
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【聯合報/曾敬豪、魏妤庭、黃信璁】 


美國大選川普領先。出口民調顯示，共和黨在參眾兩院席次也領先。川普如果當選將是衝擊全球政經市場最大的黑天鵝。川普在選戰中高喊「美國優先」的態度，對國際局勢將帶來什麼樣的變化？他的決策影響亞洲局勢，更影響台海的穩定。在這個重要時刻，為你解析美選後，總統當選人將為世界帶來什麼樣牽一髮動全身的變化。 


亞太地區／撤軍拉高區域衝突 反TPP放緩亞洲經濟成長 

● 撤軍亞洲，軍備競賽恐爆發 

● 反TPP，或導致亞洲經濟成長放緩 


川普揚言要求盟邦負擔美軍防務費用，否則考慮將美軍撤離日韓，他表示這些國家應當自我防衛。一旦美國退出亞洲，亞太情勢恐怕更加複雜。 


在東北亞，因應朝核危機，日本和韓國可能發展核武自衛；南海地區，越南、印尼可能與日本、澳洲、印度合作，以面對中國的主權聲索。亞太地區可能爆發軍備競賽與結盟對抗以求自保，將大大提高衝突與戰爭的可能性。 


此區域國家最關心川普反跨太平洋貿易夥伴協定（TPP）的立場，此舉對許多東亞出口導向的國家不利，包括台灣。日本野村證劵指出，川普的保護主義，將導致亞洲國家「經濟成長放緩」等問題，韓、菲、中將受重創。白宮經濟顧問委員會警告，若TPP不通過，美國將蒙受貿易轉移損失，及喪失相關規則制定權。 


中國大陸／經濟手段代替軍事圍堵 美中大打貿易戰 

● 不與中國做軍事抗衡 

● 以經濟手段大打貿易戰 


川普「美國優先」的政策，他將聚焦在建設美國本身，呼應選戰中「使美國偉大」的口號。 


戰略安全方面，川普不會對中國採抗衡、圍堵的立場，中國社科院認為，這點對中國有利。但川普可能會以經濟手段使中國就範，例如對中國課徵45%的關稅，威脅中國出口，以弭平一年約3000多億美元的貿易逆差，此舉讓美國仍站在戰略至高點。 


TPP若不通過，將對中國較有利，新加坡李光耀公共政策學院院長馬凱碩表示，「中國會成為經濟影響力的中心。」 但川普的其他主張，仍讓中國很頭痛。若川普真祭出報復性關稅，以及要求海外企業回美國否則課以重稅，都對中國經濟造成不小挑戰，《紐時》擔憂雙方將大打貿易戰。 


台灣／美國優先 撤軍與貿易保護影響台灣大 

● 縮減亞洲武力，台海穩定受影響 

● 反TPP恐讓美元貶，台股隨之波動 


高喊「美國優先」的川普，將縮減在亞洲武力，美國協防台灣難度將升高，屆時中國對台武力威脅大增，台灣可能成為美國的談判籌碼。但川普在衡量利益後，或許也有機會增加對台軍售的數量。 


一般認為，川普的不穩定性比柯林頓高出許多，他較有可能偏離過往美國用來指導兩岸關係的三公報一法，即「一中政策」。 


川普主張貿易保護主義，且對中貿易態度強勢，其反對TPP的方針恐怕會讓美元貶值，影響美股起伏，帶動台股波動，這對依賴國際貿易與資金的台灣來說，未來四年的處境將更加艱辛。 


歐俄地區／北約國自付軍事費 恐釀下一個烏克蘭 

● 解除對俄制裁，針對敘利亞問題合作 

● 要求北約自付軍事費 


川普在選戰中展現親俄態度，他曾表示，美國應當接受俄併克里米亞，上台後，便可能正式承認。他也認為與俄交好「不是壞事」，在敘利亞問題將與俄國合作，並解除美國對俄制裁。元智大學胡逢瑛教授表示，原本美俄僵持不下的問題（如戰略武器的數量），也可能順利談成。 


如同東亞盟邦須自我防衛的狀況，川普同樣要求北約會員國付擔更多軍事費用，盟國必須「先盡義務」。此立場將不利北約波羅的海的成員國面對俄國擴張的威脅，東歐可能出現下一個「烏克蘭危機」。 


本屆美國大選多聚焦TPP議題，較少談到跨大西洋貿易及投資夥伴協定（TTIP）。預估川普將持一貫立場反對TTIP，不再繼續與歐談判。川普曾支持「英國脫歐」，與他「美國優先」政策主張一致，皆採先本土後國際的作法。 


中東地區／與俄聯手 「要把IS炸到翻過來」 

● 面對恐怖主義、極端組織，採強硬立場 

● 連合俄國轟炸IS，以「軍事主義」解決問題 


中東地區情勢複雜，領土、民族、宗教、派系衝突、經濟利益、戰略位置及境外強權等因素，讓這地區極不穩定，加上恐怖主義、極端組織的蔓延，讓問題雪上加霜，影響全球，成為川普勢必得處理的棘手事項：對抗「伊斯蘭國」（IS）：目前聯軍正圍攻IS佔領的摩蘇爾，也將對IS的「首都」拉卡展開攻勢。川普在選戰中展現對IS的強硬立場，他表示會編列預算攻擊IS的生財項目，「要把IS炸到翻過來」，並將派遣地面部隊或與北約合作。《經濟學人》分析，他是以「軍事主義」解決相關問題。 


敘利亞內戰：2011年，在民眾要求政府進行民主改革卻反遭流血鎮壓後，衝突漸升高為武裝起事對抗阿薩德政權。美國要求阿薩德下台，但俄國卻支持他。美國指控俄敘兩國不斷轟炸阿勒坡，造成大量人員死亡與人道危機。川普不認為指責俄國有任何意義，也不反阿薩德政權。他將與俄國合作，共同轟炸IS，此舉恐讓敘國反抗軍在過程中遭到濫炸，不利反抗軍。 


以巴問題：以色列與巴勒斯坦因領土主權等問題，已困擾許久、衝突不斷，而以色列對美國又具有較大的影響力。在此議題上，川普單方面支持以色列的強硬立場，視巴勒斯坦才是麻煩製造者，稱會把耶路撒冷視為以色列的國都。 


中美洲地區／反自由貿易與移民 墨西哥首當其衝 

● 反自由貿易，中美洲自貿協定岌岌可危 

● 建造邊界牆控管美墨邊境 


川普多次批判並揚言廢除北美自由貿易協定（NAFTA）。如果廢除，墨西哥的經濟將受到很大衝擊，同時還需煩惱川普威脅的35%關稅。川普反自由貿易協定的傾向，讓中美洲自由貿易協定（CAFTA）岌岌可危。 


另外受川普上台波及的還有古巴，歐巴馬努力恢復的雙邊關係正常化，將回到副總統潘斯說的對古「禁運」狀態。 


對於移民議題，2017年1月上任後，川普不改其「美國優先」的主張，他將強力控管美墨邊界，將與墨西哥進一步協調邊境管控的問題，包括建造美墨國界的邊界牆、攔截非法移民電匯款、增加簽證費與關稅等申辦費用，增加整體非法移動的成本。 


美國本土／高壓管移民 槍枝放寬帶來潛在國土危機 

● 美國優先，經濟政策將趨保守 

● 嚴密監控穆斯林，提高外國雇傭的難度 


根據國際貨幣基金（IMF）對2016年及2017年的預測，明年的全球經濟成長（3.4%）會比2016年（3.1%）表現略佳。目前全球經濟復甦力道仍不足，處在「悶經濟」狀態。世界貿易組織（WTO）最近則下修明年全球貿易成長預測，估計將介於1.8%~3.1%間，低於早先在4月的評估3.6%。 


柏克萊加州大學教授艾肯格林表示，美、中、歐皆未走出內部困難，現下缺乏引領世界前進的「火車頭」。新總統川普反全球化的貿易政策，不利於全球經濟成長，以鄰為壑的方式，林偉修教授也不看好能幫助美國經濟成長，許多機構與專家亦曾表達對前景感到憂心忡忡。 


九一一事件及奧蘭多夜店槍擊案皆發生在美國本土，川普歸咎結果於歐巴馬與柯林頓的政策。若川普實現競選諾言，穆斯林族群將相當不好過，他們可能會被禁止入境、嚴密監視、另建資料檔案，或遭遣返出境。川普將展開驅逐500萬非法移民的行動，尤針對有犯罪事實與過期滯留者。 


他希望終結透過屬地主義成為公民的管道，認為這是吸引非法移民的最大原因，更要以思想檢核篩選真正認同美國的移民。提高申請簽證與工作許可的困難度，增加企業雇用外國人與移民的成本，保障本國人的工作權。 


川普強調「法律與秩序」，卻對槍枝管制寬鬆，這樣一方面高壓的做法，一方面允許購買攻擊性武器的作法，能讓美國更安全，恐不容樂觀。
