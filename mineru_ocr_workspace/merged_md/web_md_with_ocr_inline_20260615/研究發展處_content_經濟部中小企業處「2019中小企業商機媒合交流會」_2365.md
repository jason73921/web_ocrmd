# 網站文章資料_經濟部中小企業處「2019中小企業商機媒合交流會」_2365

## 一、文章基本資料

- **文章ID**: 2365
- **文章標題**: 經濟部中小企業處「2019中小企業商機媒合交流會」
- **文章網址**: https://www.yzu.edu.tw/admin/rd/index.php/tw/?option=com_content&view=article&id=2365
- **文章簡介**: 主旨:

敬請協助轉知「2019中小企業商機媒合交流會」徵展訊息予以中小企業踴躍報名參展，請查照。

說明:

一、本處預定於今(108)年6月25日(星期二)假臺大醫院國際會議廳及8月23日(星期五)假臺南大億麗緻酒店辦理「2019中小企業商機媒合交流會」，協助中小企業順利尋求策略合作夥伴與商機媒合管道，增加中小企業新產品曝光及行銷機會，進而創造市場新商機。

二、本次商機媒合會徵展報名日期自即日起至本(108)年4月10日截止，徵募符合「智慧科技」應用之創新技術及產品，相關訊息、參展辦法及報名表電子檔等，請逕至「中小企業商機媒合整合服務網」網站(httpx://www.technomart.org.tw/)查詢及下載使用，或洽詢財團法人工業技術研究院楊小姐，電話:03-5913811，Email:candyyang@itri.org.tw
