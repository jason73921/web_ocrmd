# 網站文章資料_看準綠能商機接軌國際 產學研成果首赴英國綠能展發表_1909

## 一、文章基本資料

- **文章ID**: 1909
- **文章標題**: 看準綠能商機接軌國際 產學研成果首赴英國綠能展發表
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20170510
- **文章簡介**: 【中央社 】

為提升國際能見度，並宣揚我國政府發展綠能科技之決心，第二期能源國家型科技計畫(NEP-II)辦公室今年首度組團至英國蘇格蘭參加「All-Energy」展，展期為5月10日至11日。 「All-Energy」展為結合再生能源商品、技術展覽及研討會之綜合性國際能源展會，自舉辦以來，歷年皆吸引多國政府及廠商出席參觀，是開創各國綠能商機的重要場域。　　

本次參展，將由NEP-II計畫執行長─台灣大學吳光鐘教授率同光陽工業、智晶光電、鼎樫科技及綠源科技等廠商，以及台大、成大、中央、中山、元智、屏東科大、資策會、台經院、核研所、工研院、國家奈米元件實驗室等學研單位共同參與。我國參展代表團於展場中設立「臺灣國家形象館(Energy@Taiwan)」，以「節能」、「創能」、「儲能」及「系統整合」四大展區呈現研發成果。例如目前已商品化、可提升能源使用效率之「外轉子馬達風扇」；透過搭配各類太陽能設備，即能應用於不同場域，可依據日照的移動轉換軌跡之「追日系統」；曾獲得第12屆國家新創獎，可生產纖維酒精及生質化學品之「纖維解聚糖化技術」；獲頒百大科技研發獎（2016 R&D100 Award），可廣泛應用於可攜式電子產品之「可高速充放電鋁電池」；獲得德國紐倫堡國際發明展金牌獎、第13屆國家新創獎，並於2016台北國際發明暨技術交易展競賽獲3金2銅，可作為自給式電力源之「固態氧化物燃料電池」；以及可搭配太陽能、儲能進行需量控制，有效提升用電效率之「微電網系統技術」。　　

本次參展，除積極對外展示我國能源研發成果之外，將舉辦研討會，加強與國際能源組織及業者之互動，創造合作契機；參展代表團亦將安排與英國相關領域部門互相交流，建立國際合作網絡，期能協助我國廠商創造海外商機，同時吸引國際大廠投資臺灣，進一步帶動我國綠能發展。
