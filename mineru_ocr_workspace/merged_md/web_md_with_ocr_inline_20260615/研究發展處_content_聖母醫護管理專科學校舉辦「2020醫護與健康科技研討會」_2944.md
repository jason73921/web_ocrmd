# 網站文章資料_聖母醫護管理專科學校舉辦「2020醫護與健康科技研討會」_2944

## 一、文章基本資料

- **文章ID**: 2944
- **文章標題**: 聖母醫護管理專科學校舉辦「2020醫護與健康科技研討會」
- **文章網址**: https://www.yzu.edu.tw/admin/rd/index.php/tw/?option=com_content&view=article&id=2944
- **文章簡介**: 主旨：

本校於109年05月16日(星期六)舉辦「2020醫護與健康科 技研討會」，檢送徵稿訊息惠予公告，請鼓勵貴屬同仁及 師生踴躍投稿，並參與研討會，請查照。

說明：

一、研討會宗旨：配合教育部高教深耕計畫，並鼓勵教師從事 與教學相關之專題研究，並發表研究成果，以提升學術研 究風氣。藉此研討會邀請各專家學者先進交換實務經驗， 增進學術界與實務界合作互動與交流，掌握醫護與健康科 技領域上之最新發展與未來趨勢，分享研究成果。

二、研討會重要時程：

  (一)論文摘要投稿截止日：2020/04/06(一)17:00止

  (二)摘要審查結果公告日：2020/04/29(三)13:00後

  (三)口頭論文全文繳交截止日：2020/05/06(三)

  (四)壁報論文海報收件截止日：2020/05/14(四)

  (五)研討會報名截止日：2020/05/08(五)17:00止

(六)研討會日期：2020/05/16(六)

三、研討會主題：

  (一)醫護科技及健康照護研究。

  (二)健康產業與餐飲管理。

  (三)醫學美容科技應用。

  (四)幼兒保育與健康照護。

  (五)資訊科技健康應用。

  (六)醫護與健康人文探討。

四、論文發表方式：「口頭論文發表」與「壁報論文發表」。

五、投稿及研討會報名網址：http://www.smc.edu.tw/health/ (或於本校網站點選「2020醫護與健康科技研討會」圖示)投稿後，參與本校研討會者需另行再進入「研討會報名區」報名。

六、證明、論文集發放、徵稿格式及要求、論文發表、壁報張貼收回注意事項及交通住宿參考資訊等，請參閱研討會計畫書及相關附件。

七、本活動備有茶水及餐點，請自行攜帶環保杯。

八、活動聯絡人：李盈青小姐，電話(03)989-7396，分機505或 502。
