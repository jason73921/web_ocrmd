# 網站文章資料_元智推程式語言必修課　盼畢業生學會寫App_1118

## 一、文章基本資料

- **文章ID**: 1118
- **文章標題**: 元智推程式語言必修課　盼畢業生學會寫App
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【國立教育廣採電台】 


為了加強學生的競爭力，元智大學首創新雙語課程，除了原有的雙語教學，也把程式語言列為新雙語，今年起試辦成為各院系的通識必修課，希望培養有程式設計能力的跨領域人才，讓學生畢業後個個都會寫App。

程式語言成為學習趨勢，為因應大數據、工業4.0、物聯網時代，元智大學首創推出新雙語課程，除了原有的雙語教學，今年起將試辦把程式語言列為通識必修4學分，讓各學院加入程式語言設計課，元智大學校長吳志揚表示，今年入學新生將試辦修習通識必修的程式設計課，預計明年全面正式上路，也希望不論是人文還是理工科系的學生，都要在畢業前具備寫APP的能力。

除了推動新雙語課程，元智大學同樣提供新生獎勵措施，只要學生的學測總級分或指考採計科目（至少3科）原始總分在前15%者，第一年學雜費比照國立大學收費標準，第二至四年成績符合標準者，將可繼續享有補助。
