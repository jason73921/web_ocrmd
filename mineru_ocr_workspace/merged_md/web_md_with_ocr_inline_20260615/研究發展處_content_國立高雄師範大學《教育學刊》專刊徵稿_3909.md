# 網站文章資料_國立高雄師範大學《教育學刊》專刊徵稿_3909

## 一、文章基本資料

- **文章ID**: 3909
- **文章標題**: 國立高雄師範大學《教育學刊》專刊徵稿
- **文章網址**: https://www.yzu.edu.tw/admin/rd/index.php/tw/?option=com_content&view=article&id=3909
- **文章簡介**: 主旨:

檢送本校教育學系《教育學刊》專刊徵稿，敬請轉知貴校教師、博士班研究生踴躍投稿，請查照。

說明:

一、《教育學刊》榮獲TSSCI收錄期刊、國際資料庫EBSCO及 PROQUEST收錄。

二、近年來，「素養導向」在教育領域中備受重視。各次領域諸如教育基礎理論、教育政策與行政、課程與教學，心理輔導、測驗統計等，皆能對此主題進行探究。

本刊將於57 期（2021年12月出刊）、58期（2022年6月出刊）發行以 「素養導向」為主題之專刊，徵求相關論文稿件。

三、專刊主題：素養導向教育趨勢之探究與實作。

四、相關議題：全球素養、公民素養、科技素養、媒體識讀、永續經營、素養導向師資培育、108課綱三面九項等。

五、徵稿要點：請詳閱本刊徵稿要點與撰稿須知。

六、徵稿期限：57期為2021年8月31日、58期為2022年2月28 日。

七、徵稿對象：現任(或曾任)各大學校院或學術研究機構之教師、專任研究人員及博士班研究生。

八、檢附《教育學刊》專刊徵稿海報及徵稿要點供參。
