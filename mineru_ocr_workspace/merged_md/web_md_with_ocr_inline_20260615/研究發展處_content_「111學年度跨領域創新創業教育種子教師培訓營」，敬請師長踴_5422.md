# 111學年度跨領域創新創業教育種子教師培训

配合教育部「大專校院推動創新創業教育計畫」本研習將透過創新創業教育工作者的教學經驗分享提供教師尋找教學靈感，並協助教師結合自身專長進行創新教學 C

## 研習對象：

(一) 大專院校教師。

(二) 創新創業推動教職員工者。

## 研習證明：

1.全程參與者可獲得研習證明

2.具公務人員資格者可認列學習時數。

3.本校新進教師，可列入新進教師教學研習時數。

## 報名方式：

研習課程皆採線上方式辦理，

原則上不限制參與人數，

歡迎有興趣的教師與行政人員共同參與。
