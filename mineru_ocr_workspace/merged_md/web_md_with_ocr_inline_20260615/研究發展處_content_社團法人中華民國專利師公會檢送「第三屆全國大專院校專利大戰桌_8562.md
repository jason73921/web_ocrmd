# 第三屆全國大專院校專利大戰桌遊競賽（含深耕課程）

# 活動報名簡章

## 壹、競賽目的：

為推廣專利知識、提升公眾對於智慧財產權重要性的認識，透過專利、商標與產品開發的桌上策略遊戲競賽，讓參賽者體會到智慧財產權在產品開發中的重要性，激發對創新與智慧財產權保護的關注和興趣。

本競賽旨在為國家培育優秀的專利人才，藉由寓教於樂的方式，讓大專院校學生在競賽過程中深入理解專利申請、商標註冊、侵權規避等實務議題，建立正確的智慧財產權觀念。同時，透過策略思考與決策訓練，培養學生在產品開發過程中的專利布局能力，使其未來投入產業時，能夠運用智財策略提升企業競爭優勢，進而促進我國科技產業之創新發展與國際競爭力。

此外，本活動結合深耕課程，由專業專利師親自授課，提供學生與業界專家交流的機會，促進產學合作，縮短學用落差，為我國智慧財產權人才培育奠定扎實基礎。

## 貳、主辦單位：

社團法人中華民國專利師公會、國立臺北科技大學、國立中興大學、國立成功大學、國立臺灣大學、國立中興大學科研產業化平台、國立成功大學產學創新總中心。
