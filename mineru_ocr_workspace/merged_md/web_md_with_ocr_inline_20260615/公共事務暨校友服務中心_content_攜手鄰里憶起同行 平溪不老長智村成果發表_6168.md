# 網站文章資料_攜手鄰里憶起同行 平溪不老長智村成果發表_6168

## 一、文章基本資料

- **文章ID**: 6168
- **文章標題**: 攜手鄰里憶起同行 平溪不老長智村成果發表
- **文章網址**: 
- **文章簡介**: 【國立教育廣播平台/王芓諭臺北】

為了讓失智症長者能夠在社區中安心生活，新北市衛生局在平溪區結合社區網絡，連結社區中的商家、失智據點、社區據點、警消等，並在今天(7日)舉辦「不老長智村成果發表」，讓在地長者都能得到妥善照護，並作為失智友善示範區。
 
根據世界衛生組織統計，全球每3秒就新增1位失智症患者，目前臺灣約有29萬名失智者，因此如何讓民眾正確認識、協助與關懷失智者及其家庭，讓他們可以在社區中安心生活且能融入社會，是首要目標，因此新北市為了讓長者獲得更完善的社區照顧，選擇於人口高齡佔比最高的平溪區，以衛生所為主體連結在地服務資源，搭配失智友善計畫，並參與歐盟智慧健康長照計畫，建置「不老長智村」。
 
平溪區在地商家「温泉商店」李老闆，因為母親為失智症患者，使得李老闆對失智症者有極大的包容，甚至找出一套方法來面對失智症客人，考量長者常因疾病關係會重複購買，或是家屬抱怨為何不阻止長者購買，然而參與失智友善訓練後，李老闆開始學習如何協助失智長者及家屬，包含主動與家屬確認購買意願，若家屬不需要且尚未開封，便可拿回店內退貨。
 
另外，元智大學成為歐盟智慧健康長照計畫合作夥伴，機械工程學系研究生黃楷淇表示，整合各個國家的智慧健康產品，來協助失智長者提升生活品質，像是今天展出的Tessa是荷蘭發明的，主要是陪伴型機器人，搭配Tiper訓練手指的壓力，同時測驗手指的力度，至於元智大學發明的WhizToys，透過燈光與螢幕APP的搭配，可訓練長者腿部的反應能力。（待續）
