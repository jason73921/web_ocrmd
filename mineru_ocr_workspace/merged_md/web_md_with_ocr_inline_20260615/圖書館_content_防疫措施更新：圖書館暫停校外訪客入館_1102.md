# 網站文章資料_防疫措施更新：圖書館暫停校外訪客入館_1102

## 一、文章基本資料

- **文章ID**: 1102
- **文章標題**: 防疫措施更新：圖書館暫停校外訪客入館
- **文章網址**: https://www.yzu.edu.tw/library/index.php/tw/20200302-1
- **文章簡介**: 入館：

配合校園疫情管控措施，自3/3起暫停校外訪客入館（持有本館所核發借書證者不在此限），同時暫停受理校外人士（含校友及圖書館之友等）借書證新申請。[3/2更新]

進館讀者建議全程配戴口罩(請自備)；如有發燒或持續咳嗽等現象，請勿入館並及早就醫。

入館前請使用酒精消毒（設置於門禁入口處）。




館舍環境消毒：

每日進行入出館門禁推桿、電子書牆螢幕及電腦鍵盤消毒作業。

每日進行閱覽座位等消毒，若讀者有疑慮，另備有酒精提供讀者自行擦拭。




借還書：

櫃台設有圖書滅菌機，借書讀者請多加利用。

還書請多利用還書箱，減少人流與接觸。

陸生（含港澳）及其他因配合防疫之必要，無法於3/2歸還圖書館借書之師生，將於還書後豁免罰款。
