# 網站文章資料_元智與醫院攜手培育護理公費生 畢業取得證照保障就業_7529

## 一、文章基本資料

- **文章ID**: 7529
- **文章標題**: 元智與醫院攜手培育護理公費生 畢業取得證照保障就業
- **文章網址**: 
- **文章簡介**: 【國立教育廣播電台/記者曾鈺羢 臺北報導】
 
元智大學與亞東醫院、聯新國際醫院合作，今天(12日)簽訂「建教合作備忘錄」及「護理公費生培育方案」，與亞東醫院的護理公費生培育方面，113學年度將有18個名額，提供每位學生每學年12萬元的獎學金補助，公費生畢業取得證照後，可直接進入亞東醫院工作。

至於聯新護理公費生部分，113學年有10個護理公費生名額，第一、二年每學期分別有6萬元補助，第三年每學期提供8萬元、第四年每學期提供9萬元；此外，護理公費生於在學期間，如取得要求的相關證照、技能檢定或其他有助於護理專業能力養成與提升的優異表現者，提供最高新台幣2萬元的補助，4年將提供最高60萬元獎助學金，學生畢業後順利考取護理師證照，直接保障進入聯新醫院工作機會。（待續）
