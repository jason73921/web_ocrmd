# 網站文章資料_經濟部智慧財產局「2018APEC著作權集管組織對微中小型企業之最佳授權實務工作坊」_2014

## 一、文章基本資料

- **文章ID**: 2014
- **文章標題**: 經濟部智慧財產局「2018APEC著作權集管組織對微中小型企業之最佳授權實務工作坊」
- **文章網址**: https://www.yzu.edu.tw/admin/rd/index.php/tw/?option=com_content&view=article&id=2014
- **文章簡介**: 主旨：本局訂於本(107)年度10月23日及10月24日於張榮發基金會國際會議中心辦理「2018APEC著作權集管組織對微中小型企業之最佳授權實務工作坊」，請貴校院惠予轉知師生踴躍報名參加及協助公告周知，如說明，請查照。

說明：

一、有鑑於著作權集體管理組織是所有影視音娛樂、文創商業等產業鏈中不可或缺的重要組成部分，在現今數位科技發展下，集管組織如何建立效率及便捷性的授權模式，以減少與微中小型企業間的授權成本，助其提升產品服務效能及創造產業新價值，將是促進產業發展的重要關鍵。本局於去(106)年獲亞太經濟合作會議（APEC）同意補助「著作權集管組織對微中小型企業最佳授權實務指南」之計畫案，將透過調查與分析各經濟體集管組織授權狀況與政策發展、專家及實務人員經驗交流，訂定可廣泛適用且具備執行效益之手冊，並辦理旨揭工作坊，藉此強化及提升集管組織及微中小型企業在數位時代之競爭力，建立更優質的授權環境。

二、旨揭工作坊將邀請國際著名集管團體日本JASRAC、澳洲APRA AMCOS及Copyright Agency，以及韓國、美國實務與學界專家，分別針對「日本卡拉OK授權實務暨最新授權趨勢」、「澳紐音樂授權實務」、「韓國線上音樂授權實務運作暨最新授權趨勢」及「美國音樂著作授權制度之變革暨現代法案分享」等議題，共同探討國際集管團體現行實務及最新發展趨勢。

三、有關工作坊議程及報名方式請參閱本局智慧財產權研討會登錄中心網址http://activity.tipo.gov.tw，並預訂於本(107)年10月5日上午10時起開放網路報名，額滿為止（本案報名聯絡人:本局著作權組林先生，電話02-2376-7159）。

四、隨函檢附工作坊議程表1份，請惠予轉知師生踴躍報名參加。
