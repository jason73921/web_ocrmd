# 網站文章資料_桃園勞工學苑學分班招生中 歡迎踴躍報名進修_4954

## 一、文章基本資料

- **文章ID**: 4954
- **文章標題**: 桃園勞工學苑學分班招生中 歡迎踴躍報名進修
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20210401
- **文章簡介**: 【桃園電子報記者諶志明】

桃園市政府勞動局辦理勞工學苑課程，今年持續補助大專校院辦理學分班，歡迎戶籍地或工作地於桃園市之在職勞工踴躍報名參加。

勞動局表示，今年勞工學苑學分班包含龍華科技大學「電子電路繪圖、模擬、與佈局」、元智大學「行動裝置程式設計大學學士學分班」、銘傳大學「危機處理─不法侵害與新冠肺炎應對」、中原大學「整合行銷與設計一」及「工程圖學」、健行科技大學「韓語基礎及初級檢定班」及「木作家具設計實務班」等共7班，每班3學分，自2月起熱烈招生中，預計5月起陸續開課，詳情可至勞工學苑資訊網，或逕洽各開課單位查詢；後續另有產業應用班7班及生活應用班20班即將陸續招生，歡迎有興趣的勞工朋友密切注意網站。(本廣告由就業安定基金補助)
