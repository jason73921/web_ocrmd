# 網站文章資料_國立虎尾科技大學「2017校慶全國學生專題製作競賽暨成果展示」徵稿_1314

## 一、文章基本資料

- **文章ID**: 1314
- **文章標題**: 國立虎尾科技大學「2017校慶全國學生專題製作競賽暨成果展示」徵稿
- **文章網址**: 
- **文章簡介**: 國立虎尾科技大學

主旨：檢送本校辦理「2017校慶全國學生專題製作競賽暨成果展示」辦法1份，敬請貴校推薦優秀專題作品2組報名參加，請查照。

說明： 

一、為鼓勵學生積極從事專題研究，培養創新思考模式，以提升學術研究能力與實務發展技能；透過校際競賽交流，獎 勵績優專題製作成果，特舉辦本活動。

二、參賽對象：全國公私立大專校院之全職在學學生（含在臺 外籍學生），每隊成員最少4人為原則。

三、參賽組別、類別：分為1.研究所組及2.大專組，成員中如 有研究所學生則須報名研究所組；專題競賽區分為4類， 分類如下：A.工程類、B.電資類、C.文理類、D.管理類。 

四、報名期間：自106年10月18至106年11月3日。詳情請至主 辦單位網站查閱：http://tech.nfu.edu.tw/bin/home.php。 

五、如對本競賽有相關疑義，請洽本校研究發展處實習組邵小姐(電話：05-6315022 )。
