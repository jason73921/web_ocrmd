## 一、文章基本資料

- **文章ID**: 185
- **文章標題**: 我要如何設定我的電腦才能使用無線網路?
- **文章網址**: https://www.yzu.edu.tw/admin/is/index.php/tw/2015-08-06-05-51-17
- **文章簡介**: 
- **文章全文**: Windows XP：


點選開始→控制台→控制台 


點選"網路連線"


在"無線網路連線"的圖示上按右鍵選"內容"


在中間的列表中找到"Internet Protocol (TCP/IP)"，反白後按"內容" 


選擇"自動取得IP"及"自動取得DNS伺服器位址" 

按"確定"以關閉所有視窗 


請對 "無線網路連線" 圖示上按右鍵選擇"檢視可用的無線網路" 


請點選"YZUWLAN"的選項，並點選連線。 


開啟瀏覽器，自動跳入認證畫面，並輸入您的e-mail帳號及密碼。
  


-->
Windows 7：


開始 -> 控制台


請點選 "檢視網路狀態及工作"


請點選 "變更介面卡設定"
 

請點選"無線網路連線" -> 連線


請點選 " YZUWLAN " 當中的 " 連線 "
 

開啟瀏覽器，自動跳入認證畫面，並輸入您的e-mail帳號及密碼。
  


 Android 作業系統：


在「Home」的畫面中按下「MENU」，並選擇「 設定 」。

按下「無線通訊控制」。

按一下「Ｗi-Fi」 開啟Wi-Fi。

Wi-Fi開啟後（如下圖，右邊ICON顯示為綠色），按一下「Wi-Fi設定」。

在「Wi-Fi網路」選項中按下「YZUWLAN」。

Wi-Fi設定完成。

開啟瀏覽器，會自動跳進認證畫面，在「帳號」欄及「密碼」，輸入您的e-mail帳號及密碼


APLLE IPod、IPad、IPhone：


在主畫面中按下「設定」。

在「Wi-Fi網路」,右推開啟「Wi-Fi」,「選擇網路」中按下「YZUWLAN」。

返回主畫面，按下「Safari」（或其他瀏覽器亦可）。

瀏覽器會自動跳進認證畫面，在「帳號」欄及「密碼」，輸入您的e-mail帳號及密碼
