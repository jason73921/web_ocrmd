# 元智大學與天津理工大學

# 學術交流與合作協議書

## 一、協議宗旨

元智大學與天津理工大學在雙方友好協商的基礎上，通過建立及實施兩校合作交流計畫，增進彼此瞭解與友誼，加強兩校的合作與聯繫，以促進雙方教學、科研、管理等方面的發展，提升兩校教育及學術水準。

## 二、交流內容

雙方同意進行下列各合作事項

1. 教師互訪

雙方互相薦派教師進行考察訪問、學術演講、合作研究、短期客座講學等。

2. 學生交流

雙方選派學生進行互訪、交流、短期研修。短期研修達一學期的學生，互相核發學習證明。

3. 聯合舉辦學術研討會

雙方以平等互惠為基礎，在可能的範圍內就管理、科技、人文、藝術、教育及其他相關學術課題共同舉辦學術研討會。

4. 合作研究開發

雙方針對相關領域專案進行合作研究，並共同爭取民間或政府基金支持，相互推薦對方的研究成果。

三、有關教師的互訪、交流、合作、學生的交流以及合作進行科學研究的經費、題目及成果的利用等，雙方根據具體情況分別協商確定。

五、本協議書一式兩份，雙方各執一份。本協議書經雙方簽署後立即生效，有效期五年。若雙方均未在協議期滿前六個月以書面形式通知對方終止協定，則該協定每期可順延五年。
