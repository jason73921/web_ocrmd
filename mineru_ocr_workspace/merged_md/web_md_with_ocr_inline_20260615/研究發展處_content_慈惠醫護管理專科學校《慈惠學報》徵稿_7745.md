# 網站文章資料_慈惠醫護管理專科學校《慈惠學報》徵稿_7745

## 一、文章基本資料

- **文章ID**: 7745
- **文章標題**: 慈惠醫護管理專科學校《慈惠學報》徵稿
- **文章網址**: https://www.yzu.edu.tw/admin/rd/index.php/tw/?option=com_content&view=article&id=7745
- **文章簡介**: 主旨：本校《慈惠學報》第二十一期自即日起徵求稿件，敬請轉知貴校教師同仁惠賜論文，以光篇幅，請查照。

說明：

一、《慈惠學報》內容涵蓋醫學、護理、幼兒、保健、管理、 人文社會科學及自然工程科學之理論和實務相關領域之學術性論文，歡迎全國各大專院校教師暨學術機構研究人員來稿。

二、本校《慈惠學報》已登錄國際標準期刊號ISSN 22224572，預計於114年12月份出版，截稿日期為114年10月15 日止。

三、投稿者均須簽署「投稿資料表」與「著作授權同意書」並酌收新台幣2,000元整之審稿費，審稿費請用郵政匯票，抬頭為慈惠醫護管理專科學校。

四、稿件格式依慈惠學報徵稿要點，其他投稿相關事宜，逕洽本校研究發展組郭加恩組長，聯繫電話(08)8647367轉147。

五、投稿資料表、投稿稿件（紙本一式三份）及著作授權同意書、郵政匯票，請郵寄至926屏東縣南州鄉三民路367號慈惠醫專研發組收，投稿稿件電子檔請mail至 tzuhui110@gmail.com。
