# 網站文章資料_52台播什麼？學者籲：排頻權在系統商、尊重市場機制_4736

## 一、文章基本資料

- **文章ID**: 4736
- **文章標題**: 52台播什麼？學者籲：排頻權在系統商、尊重市場機制
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20201119
- **文章簡介**: 【台灣數位匯流網記者吳易芸、方文/台北報導】

中天新聞台換照不過，根據執照到期日只能播送到12月11日，未來有線電視52頻位成為兵家必爭之地，國家通訊傳播委員會（NCC）主委陳耀祥今天（18日）在記者會上呼籲，系統業者給予公廣集團機會，他初步想法是會與系統溝通討論。不過，此話立刻引發業界熱議，「排頻權是市場商業機制，身為主關機關的NCC不適宜暗示所屬意的頻道。」

知情人士表示，陳耀祥這席話是脫稿演出，並未在委員會討論方向。世新大學廣電系兼任副教授、前NCC委員何吉森表示，主委話說得不清楚，其實只是希望，但頻道規劃權在系統業者，而頻道新增涉及營運計劃變更，仍應經NCC許可。業者只要不違反有廣法29條3項，及其授權之「頻道規劃及其類型變更許可辦法」即可。

元智大學資訊傳播學系助理教授葉志良表示，頻位是市場機制協商而來，政府不應該有太多角色可以介入，主委的話是建議，但主管機關最後仍要尊重市場機制，尊重系統商的排頻權。
葉志良表示，NCC應該要全盤檢討萬年頻道表，也就是針對2位數頻位，朝向3位數擴大區塊化進行，區塊化可以增加消費者收視的便利性，這才是一勞永逸的作法。（待續）
