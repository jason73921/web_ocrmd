# 網站文章資料_即時互動音樂劇　DoReMi天團等你來挑戰！_1423

## 一、文章基本資料

- **文章ID**: 1423
- **文章標題**: 即時互動音樂劇　DoReMi天團等你來挑戰！
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【愛台客／綜合報導】 


獲得文化部105年「文化創意產業補助計畫」的三立電視，將跨界與今年榮獲金鐘獎「動畫節目獎」的原金國際及台灣本土歌謠大師簡上仁一起合作，創新製作出最新的兒童節目「魔法列車DoReMi」，節目透過新科技的運用，打造生動活潑的動畫主角，並結合傳統的台語童謠元素，讓小朋友可以體驗台灣早期生活型態及故事傳承與本土音樂的多元薰陶！

熟料「魔法列車DoReMi」未演先轟動，主持人Dodo、Rere、Mimi所組成的「DoReMi天團」，即將於11月13日在「嘉義縣表演藝術中心」先登場。這回他們跟著本土音樂家-簡上仁老師，又跳又唱，詮釋多首精彩的台灣童謠，而且此次更結合新科技呈現多媒體體驗，由元智大學資訊傳播學系的師生跨產業操作，甲尚科技與首羿國際 提供最新的製作系統，讓觀眾能置身在動畫的虛擬世界中，並隨心所欲現場互動，跟動畫的主角ReRe勁歌熱舞，比看看誰的舞蹈跳得最夯、最逗趣。相信透過DoReMi風趣活潑的演出，保證讓大家回味無窮。

11月13日「台灣民謠經典再現」節目，將由簡上仁老師以及其所領軍的田園樂府樂團現場伴奏曲曲動人的台灣民謠，並邀請台灣風合唱團、大地兒童合唱團、雯翔舞團及美聲音樂家范廣慧、姜宜君、朱育陞等來詮釋新舊的台灣民謠，曲曲洋溢泥土芬芳，將唱出台灣歌謠新希望。現場演出免費觀賞，歡迎大家索票。
