# 網站文章資料_國立虎尾科技大學 「2023全國大專暨高中職學生專題製作競賽暨成果展示」_5804

## 一、文章基本資料

- **文章ID**: 5804
- **文章標題**: 國立虎尾科技大學 「2023全國大專暨高中職學生專題製作競賽暨成果展示」
- **文章網址**: https://www.yzu.edu.tw/admin/rd/index.php/tw/?option=com_content&view=article&id=5804
- **文章簡介**: 主旨：檢送本校辦理「2023全國大專暨高中職學生專題製作競賽暨成果展示」辦法，敬請貴校推薦優秀專題作品報名參加，請查照。

說明：一、為鼓勵學生積極從事專題研究，培養創新思考模式，以提升學術研究能力與實務發展技能；透過校際競賽交流獎勵績優專題製作成果，特舉辦本活動。

二、參賽對象：全國公私立大專校院及高中職之全職在學學生。

三、競賽參賽組別與類別區分：組別分為1.研究所組2.大專組3.高中職組，成員中如有研究所學生則須報名研究所組；類別分類如下：A.工程類、B.電資類、C.文理類、D.管理類。

四、報名期間：即日起至112年10月20日。敬請鼓勵貴校師生踴躍參加，並惠予與會人員公假。

五、隨文檢附活動競賽辦法及申請表，詳情請至本校研發處網頁-證照競賽區查閱(https://tech.nfu.edu.tw/index.php/zh/race/3596-2023-71)。

六、如對本競賽有相關疑義，請洽本校研究發展處實習組王小姐(電話：05-6315022)。
