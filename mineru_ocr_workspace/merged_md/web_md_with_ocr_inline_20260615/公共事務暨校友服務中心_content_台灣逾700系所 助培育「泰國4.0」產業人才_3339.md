# 網站文章資料_台灣逾700系所 助培育「泰國4.0」產業人才_3339

## 一、文章基本資料

- **文章ID**: 3339
- **文章標題**: 台灣逾700系所 助培育「泰國4.0」產業人才
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20181113
- **文章簡介**: 【中央廣播電台/黃美寧】

駐泰代表處今天(12日)公布『台灣培育「泰國4.0」人才的教育與獎學金』手冊，彙整中央研究院及30所台灣的大學可培育「泰國4.0」十大產業人才的702個系所。除了泰國學生可以申請台灣各大學提供的總數大約1,200個獎學金（www.studyintaiwan.org），駐泰代表處也將優先推薦到台灣的大學就讀「泰國4.0」十大產業相關系所的泰國學生20-25名，可以獲得台灣三個部會提供的全額獎學金。

泰國政府刻正積極推動「泰國4.0」經濟發展計畫，以升級既有產業為基礎帶動發展未來產業，優先發展之10大目標產業為新一代汽車、智慧電子、精緻農業、高附加價值觀光、新一代食品、自動機械及產業用機器人、數位經濟、生質燃料及生物化學、醫療照護、及航太與物流等產業。

駐泰處公告30所台灣的大學、可培育「泰國4.0」十大產業人才的702個系所，來自幾乎所有的頂尖大學及其他公私立大學。除了上次公布的17所頂尖大學之外，這次名單增加不少私立大學的系所，包括臺北醫學大學、中國醫藥大學、淡江大學、元智大學等等，以提供更全面的大學系所供泰國學生參考與申請就讀。( 待續）
