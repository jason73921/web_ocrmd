# 網站文章資料_科技部南部科學園區管理局109年度「南部科學園區研發精進產學合作計畫」徵稿_2977

## 一、文章基本資料

- **文章ID**: 2977
- **文章標題**: 科技部南部科學園區管理局109年度「南部科學園區研發精進產學合作計畫」徵稿
- **文章網址**: https://www.yzu.edu.tw/admin/rd/index.php/tw/?option=com_content&view=article&id=2977
- **文章簡介**: 主旨：本局公開徵求109年度「南部科學園區研發精進產學合作計畫」，請查照並轉知所屬踴躍提出申請。

說明：

一、依據「科技部科學園區研發精進產學合作計畫實施要點」辦理。

二、本(109)年度徵求政策方向為鼓勵具體技轉學研成果商品化、具高經濟效益或具高創新及發展潛力之計畫，並設定年度優先徵求主題，歡迎踴躍提出申請。

三、公開徵求期程自即日起至109年4月20日(星期一)下午5時止，本年度計畫執行期間為1年，執行起始日以109年7月1日起算。

四、有意申請本年度計畫者，請詳閱「徵求公告」及「申請手冊」等資料，並請逕自本局網站(https://www.stsp.gov.tw/ tw/)點選「廠商服務>園區獎補助計畫>研發精進產學合作 計畫>最新消息」下載。

五、計畫諮詢窗口：鄭小姐，06-5051001分機2154，annacheng@learnmore.com.tw；林小姐06-5051001分機 6331，mesha@learnmore.com.tw。
