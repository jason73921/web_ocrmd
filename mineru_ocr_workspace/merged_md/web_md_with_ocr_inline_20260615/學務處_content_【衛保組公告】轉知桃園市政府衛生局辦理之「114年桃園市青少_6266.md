# 網站文章資料_【衛保組公告】轉知桃園市政府衛生局辦理之「114年桃園市青少年菸害防制知識答題抽好禮活動」，鼓勵未滿20歲學生踴躍參與。_6266

## 一、文章基本資料

- **文章ID**: 6266
- **文章標題**: 【衛保組公告】轉知桃園市政府衛生局辦理之「114年桃園市青少年菸害防制知識答題抽好禮活動」，鼓勵未滿20歲學生踴躍參與。
- **文章網址**: https://www.yzu.edu.tw/admin/st/index.php/tw/?option=com_content&view=article&id=6266
- **文章簡介**: 一、為保護桃園市青少年免於菸（煙）品危害及降低其使用率，桃園市政府衛生局製作衛教宣導影片，希冀強化青少年對菸（煙）品危害識能，並使學生瞭解菸害防制法相關規定。

二、旨揭活動資訊如下：

(一)活動網址：https://forms.gle/TZGv5byGCs1pwzZa7。

(二)活動時間：114年6月10日至114年9月30日。

三、抽獎訊息：每人僅限參加1次，如發現重複參加或冒用他人姓名參加活動，將取消得獎資格。

四、獎項數量：超商禮券（100元50名）。預訂於114年10月17日將符合抽獎資格之資料，以電腦系統隨機亂數產出中獎名單。

五、中獎名單將於桃園市政府衛生局無菸網（https://nosmoking.tycg.gov.tw/）最新消息上公佈，不另行通知或取得參加者同意。詳細資訊可參考前開網站活動消息。
