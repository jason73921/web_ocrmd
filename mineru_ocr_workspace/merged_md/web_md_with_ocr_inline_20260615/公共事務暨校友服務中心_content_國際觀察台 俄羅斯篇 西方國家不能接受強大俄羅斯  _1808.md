# 網站文章資料_國際觀察台 俄羅斯篇 西方國家不能接受強大俄羅斯  _1808

## 一、文章基本資料

- **文章標題**: 國際觀察台 俄羅斯篇 西方國家不能接受強大俄羅斯  
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/

- **文章全文**: 【人間福報】

三月二十八日，俄羅斯總統普亭簽署命令，禁止國外不受俄羅斯歡迎的組織在俄組成法人機構。美國方面立刻表示關切，是針對美國非政府組織而來，認為俄羅斯侵害民主自由和人權。

美俄在許多問題上針鋒相對，美國鷹派極盡可能要把俄羅斯口誅筆伐，其目的在於延長對俄羅斯的經濟制裁，並且團結西方的意識形態和軍事同盟陣線。 


日前，俄羅斯一家反腐基金會發動千人示威遊行，過程中發生警民衝突，組織者納瓦爾尼被莫斯科特維爾區法庭判處二萬元盧布的罰金與拘役十五天。西方立刻對俄發動輿論攻擊。然而，對俄羅斯民眾而言，街頭民主不是他們希望看到的決策過程。 


俄羅斯民眾關心的是：誰應該為俄羅斯的國家安全和人民福祉負責？他們早已經厭倦了街頭民主的方式來推翻政權。俄羅斯政府要向民眾負責解決任何他們關心的問題，這是普亭之所以成為強人的原因，不是他要成為獨裁專制者，而是民眾希望國家領導人要在問責的體制中，拿出解決問題的魄力。這是普亭獲得多數民眾支持的原因。 


西方輿論總喜歡以俄羅斯侵害人權自由和貪汙腐敗為藉口，公開支持顏色革命。去年俄羅斯的非政府組織國外資金有三百七十億盧布，今年初已經暴增到七百億盧布，還有增長的趨勢。 


以前索羅斯基金會幫助過很多俄羅斯的學校印書和出版，現在俄羅斯政府也將其列為不受歡迎的機構。俄羅斯政府一開始會以某些禁令來規範這些組織的行為，要有足夠積累的事件和損害，才會迫使俄政府做出禁止的決策。 


這些都是俄羅斯民眾希望政府做到他們願望的結果，西方卻喜歡倒果為因，原因就是西方國家不能接受強大的俄羅斯，影響他們在世界中的地位和利益。 


當前俄羅斯正在反恐，光是情報機構掌握參加伊斯蘭組織的聖戰士，就至少有八千名。 


俄羅斯所有的規範，必須是建立在保護本國國民生命財產安全，和維護國家社會秩序的安全基礎上，不可能讓街頭民主去解決俄羅斯當前的國家任務和困境。 


俄羅斯民眾對於普亭解決貪汙問題是殷切的，認為政府的速度還是太慢，這讓有心製造顏色革命者找到了軟肋。 


然而，普亭以法規來處理所有問題是不可避免的趨勢，他不會輕易破壞俄羅斯以血淚換來的憲政體制。這也說明，任何外國法人在俄羅斯若存有僥倖的心態，都將是難以被接受的。 


三月二十六日的大規模群眾遊行，是針對總理麥維德夫涉貪控訴而來，普亭不喜歡在施壓狀態下決策人事。故普亭難以棄軍保帥，因為當前普亭以開發北極資源增加財源，不會讓國內政治陷入鬥爭，而使人民喪失信心。 


但此事件卻引起國會內部對總理質疑，明年總統大選之前，似乎政黨間角力已經展開。胡逢瑛（桃園市／元智大學助理教授）
