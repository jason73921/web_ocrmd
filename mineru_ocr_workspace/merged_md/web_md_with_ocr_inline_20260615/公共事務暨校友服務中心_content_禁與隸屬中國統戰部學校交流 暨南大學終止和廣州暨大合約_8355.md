# 網站文章資料_禁與隸屬中國統戰部學校交流 暨南大學終止和廣州暨大合約_8355

## 一、文章基本資料

- **文章ID**: 8355
- **文章標題**: 禁與隸屬中國統戰部學校交流 暨南大學終止和廣州暨大合約
- **文章網址**: 
- **文章簡介**: 【公視/林曉慧 沈志明】

教育部禁止與隸屬中國統戰部的學校交流，且學歷不再獲採認。國立暨南大學校長指出，學校有和廣州暨南大學簽署交流備忘錄，但既然對岸的暨大已歸屬統戰部門，不在學術交流範圍，不會再推薦學生海外研修的範圍。也有大學校長表示，一定配合政策，但也希望政府說清楚，與中國各校交流的紅線。

中國211工程的重點建設大學，國際排名不差、學歷也受台灣採認，廣州暨南大學目前約有1500名台灣學生。台灣的國立暨南大學在內，有多校與其簽署交流備忘錄。校長武東星說，教育部禁止與隸屬中國統戰部的學校交流，全面檢視後，將終止與其合約。

私立元智大學校長廖慶榮則表示，「211、985工程，畢竟在學術上絕對是大陸前100所的學校。學校直屬統戰部，確實我想站在台灣立場可能也不適合交往了。」（待續）
