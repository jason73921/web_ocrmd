# 網站文章資料_成大跨維綠能材料中心 活絡南台灣綠色經濟_2973

## 一、文章基本資料

- **文章ID**: 2973
- **文章標題**: 成大跨維綠能材料中心 活絡南台灣綠色經濟
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20180626
- **文章簡介**: 【經濟日報 張傑】

成功大學新設「跨維綠能材料（Hi-GEM）研究中心」，由材料系講座教授黃肇瑞擔任計畫總主持人，並以關鍵材料的開發為目標，中心研究成果將就近於沙崙綠能科學城實地驗證，可望帶動相關產業鏈發展，提升台灣關鍵材料掌握度及活絡綠色經濟脈動。

成大「跨維綠能材料（Hi-GEM）研究中心」以發展新世代膠固態「安全電池」為主軸，結合最新穎的材料基因與人工智慧電腦輔助新材料設計，及高階臨場實驗分析系統，應用在新型高安全儲能與產能元件上，並搭配電力系統整合及工業設計，實現關鍵材料開發、新元件與系統建置，及往產品設計與產業化目標邁進。

目前，成大「跨維綠能材料（Hi-GEM）研究中心」已建構極完整的「綠能材料」研發團隊，集結校內跨領域成員，包括工學院材料系與化工系、理學院物理系與光電系、電資學院電機系、規劃設計學院工業設計系，並跨校整合台南大學綠能系、台灣大學材料系與元智大學化材系。(待續)
