# 網站文章資料_大專資訊應用服務創新競賽 賴副總統期許產學攜手培育人才_4725

## 一、文章基本資料

- **文章ID**: 4725
- **文章標題**: 大專資訊應用服務創新競賽 賴副總統期許產學攜手培育人才
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20201113-4
- **文章簡介**: 【工商時報/文 陳宗慶】

經濟部工業局、教育部資科司及中華民國資訊管理學會共同主辦的「第25屆大專校院資訊應用服務創新競賽」，11月7日舉行決賽，共有346隊次、超過2,000名師生參與決賽。雖然COVID-19疫情持續，本項競賽已連續辦理8年的亞太交流組也不停歇，透過遠距連線的方式，計有印尼、馬來西亞、新加坡、斯里蘭卡、緬甸、泰國、澳洲等7個亞太地區經濟體學生團隊線上參賽，共同競逐138個獎項205萬元總獎金。

與時俱進的研究主題也是本項競賽的重要特色，今年全球籠罩在COVID-19疫情中，有不少團隊針對疫情提出解決方案，比如中央大學的作品「急救車智能點班小幫手」，提出以「影像辨識技術」判斷藥品剩餘數量，並以「資料數位化」、「圖表視覺化」的方式進行記錄與管理，協助護理人員提升工作效率，也能有更多心力投入在病患身上。而今年很夯的「數位轉型」議題，也是競賽提案重點之一，比如元智大學及交通大學共同創作的「基於孿生神經網路之工業影像錯件偵測系統」，就是想改善工廠「自動光學檢測」之不足，提出「錯件偵測」的優化方案，採用不同種類的小樣本學習模型、影像優化演算法、文字特徵提取技術，來提升「錯件偵測」的準確度，降低產線損失，進而提高工廠的營收。

在公私部門積極投入下，今年共有10家企業及政府機關參與設組出題，主題涵蓋資訊安全、智慧聯網（AIoT）、雲端服務、AR/VR等科技應用，有效引導師生朝此領域研究創作，培育產業所需的科技人才。

現場也設立徵才媒合攤位，關貿網路公司、凌群電腦公司、宅妝公司、元大銀行、台灣亞馬遜網路服務公司、金財通商務科技服務公司、新加坡商鈦坦科技公司、1111人力銀行、時代基金會、電腦技能基金會等10家企業及合作單位積極展現網羅優秀人才的決心，吸引在場學生團隊的高度興趣，期待能促成更多產學媒合成果。獲獎隊伍名單於11月9日公布於競賽官網：https://innoserve.tca.org.tw/ 。
