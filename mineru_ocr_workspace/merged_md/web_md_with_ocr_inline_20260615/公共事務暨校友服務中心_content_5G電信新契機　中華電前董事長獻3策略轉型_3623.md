# 網站文章資料_5G電信新契機　中華電前董事長獻3策略轉型_3623

## 一、文章基本資料

- **文章ID**: 3623
- **文章標題**: 5G電信新契機　中華電前董事長獻3策略轉型
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20190319
- **文章簡介**: 【蘋果日報/林彤潔/台北報導】

今年電信業最炙手可熱的人才莫過於「大人物」人才，台灣通訊學會舉辦「通傳產業競爭之人才與資本策略」演講，探討電信及傳播產業如何因應5G時代制定人才與資本策略。對電信業者而言，5G是新的契機，中華電前董事長呂學錦則祭出電信業轉型3策略，包括結合垂直產業與新創公司發展、以併購(M&A)方式擴大事業範疇，以及企業瘦身結合併購策略擴展新領域合作機會。

元智大學教授周韻采分析，近期海內外電信事業高階主管換血潮，包括中華電信、遠傳電信、AT&T、Verizon、DOCOMO幾乎都是從電信領域內部拔擢；相較台灣大哥大邀請非電信背景的林之晨上任新總座，預計將台哥大轉型為科技公司的用人策略令人驚艷，也是世界少見。（待續）
