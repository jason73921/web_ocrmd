# 網站文章資料_元智藝設系學生設計手部握力復健裝置 _228

## 一、文章基本資料

- **文章ID**: 228
- **文章標題**: 元智藝設系學生設計手部握力復健裝置 
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【工商時報/記者傅秉祥/桃園報導】 


在醫院復健科或骨科門診，常常看到因手部或手腕運動傷害或是職業傷害的族群，病人常誤以為是關節出問題，其實是手部肌腱發炎或是神經失調所致，除了配合醫師治療吃藥之外，手部的復健更是不可少。元智大學藝術設計系大學部三年級同學陳亮先，結合機械原理與藝術設計的理念，設計手部握力復健裝置，作為手傷患者復健之使用。

元智大學藝術設計系副教授林楚卿表示，元智大學早在幾年前已實施跨院跨領域之學習，實施最早的為藝術設計系與機械系跨域學習。機械系的同學必須修讀藝術設計的課程，而藝設系的同學也必須學習機械的基本原理，機械與藝術的結合，讓機械設計因藝術美學而不再呆板，讓藝術設計應用機械原理而更增添生活互動的樂趣。 


陳亮先表示，以達文西投石機的發明，作為發想設計的開始，這套手部握力復健裝置，選擇投石機當作研究機構的素材，參考玩具做出了縮小原尺寸比例的木製投石機，之後再繼續發展成自己的設計，從投石機的特性選出了「緊繃」「放鬆」「反覆」三個重點，轉化成了手部握力復健。 


陳亮先表示，投石機是毀滅性的武器，我把破壞性質的層面弱化，轉換成比較溫柔的方式去思考設計。最初使用的是五金材料，但因受到材料限制，發展空間也有限，之後利用3D列印的技術，依患者手部大小做調整，製作出符合人體工學的零件，降低手作的誤差，提供患者比較無負擔的復健裝置，以達手部復健的成效。 


手部握力的復健裝置原發想，是提供「能自主控制握力」的患者使用，也就是橈神經失調的患者，手部神經受傷主要會使患者有垂手症狀，手腕下垂導致神經壓迫，所以握力會下降，藉由固定手腕讓患者能正常施力，藉以重複施力的動作，讓握力慢慢達到復健功能。這個設計同時兼顧復健與遊戲搭配，在手指前端裝上感應器，藉由握的動作感應，累積分數或是獲得獎勵，讓患者能將復健視為一件有趣的事情，降低排斥心理的壓力。 


另外，藝術設計系三年級同學陳昀志，以達文西仿生模仿鳥類翅膀的飛行機器發想，並參考飛行機器的機構及仿生的動態特徵，發展出具達文西意象的作品，設計仿生掃地裝置。陳昀志表示，本設計以飛行器的撲翼系統作為設計重點，運用翅膀在飛行時，展開與收闔的動作來加以改變，以仿生的方式轉化為掃地機的機制。操作方式則以人力的推動，來觸發裝置上的連動，將飛行機器的動態美感展現出來，並且讓動力轉換成機構的模式，是這個設計的重要關鍵。陳昀志說，在製作過程上以木材卡接的技法為主，膠為部分補強，整體全為手工裁切。而作品中特別以飛行器的撲翼系統作為設計重點，運用翅膀在飛行時，展開與收闔的動作來加以改變，以仿生的方式轉化為掃地機的機制。 


陳昀志說，此作品全為以木材為主，可以自行DIY設計，與小孩的學步機有類似的原理，對於小家庭或是單身女生最為適用。
