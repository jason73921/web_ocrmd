# 網站文章資料_110-1就學貸款家庭年所得為B、C類通知_2212

## 一、文章基本資料

- **文章ID**: 2212
- **文章標題**: 110-1就學貸款家庭年所得為B、C類通知
- **文章網址**: https://www.yzu.edu.tw/admin/st/index.php/tw/?option=com_content&view=article&id=2212
- **文章簡介**: 就學貸款家庭年所得為B、C類通知已於10月29日發e-mail通知，請詳閱e-mail。

一、就學貸款家庭年所得為B類者：

       即家庭年收入逾114萬元至120萬元，自償還期起算日起之利息，由主管機關(教育部)及借款學生各
       負擔半額利息，且應自貸款撥款日次月起按月繳交。

       敬請於11月5日(五)前繳交同意書至生輔組，以利後續作業。

       下載：B類通知書暨回函同意書

二、就學貸款家庭年所得為C類者：

       即家庭年收入逾120萬元，若學生本人及其兄弟姊妹有二人以上就讀高級中等以上學校，具正式學籍者，
       請於11月5日（五）前繳交「同意書」及相關證明文件（另一個兄弟姐妹就讀110-1學期之高中職(含)以
       上的就學證明文件，即蓋有110-1學期註冊章的學生證影本或繳費證明影本），俾憑辦理就學貸款，自
       償還期起算日起之利息，由借款學生負擔全額利息，且應自貸款撥款日次月起按月繳交。

       若學生無兄弟姊妹就讀高級中等以上學校者，則不符申請就學貸款資格，請儘快至總務處補繳學雜費。

下載：C類通知書暨回函同意書

同意書可由年滿20歲者可自行簽妥後繳回；未滿20歲者除了自行簽名外，仍須由法定監護人、代理人(或保證人)簽名後繳回或傳真03-4353511至生輔組(活動中心2樓)。

若對財稅中心查調所得結果有疑慮者，請洽詢所屬國稅局，申請最近一年度（109年）父、母及學生之綜合各類所得清單，11月5日（五）前繳交至生輔組，以利協助申覆。

生輔組 敬啟
