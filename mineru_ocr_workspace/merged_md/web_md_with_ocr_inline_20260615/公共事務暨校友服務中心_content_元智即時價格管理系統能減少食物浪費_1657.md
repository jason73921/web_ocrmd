# 網站文章資料_元智即時價格管理系統能減少食物浪費_1657

## 一、文章基本資料

- **文章ID**: 1657
- **文章標題**: 元智即時價格管理系統能減少食物浪費
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【工商時報/記者傅秉祥/桃園報導】 


據聯合國統計，全球平均每年有13億噸糧食浪費，卻有8億多人處在挨餓中，有46%糧食浪費則發生在食物供應鏈的最終端(亦即零售商與消費者)，其中生產過程中的損失大多發生在開發中國家，而在零售和消費過程中的浪費往往發生在中高收入的國家，因為已開發國家，消費者浪費食物通常都是因為購買過量，最後把沒吃的丟棄就造成浪費。


元智大學機械系教授鐘國濱(圖中)實驗室團隊研發「即時價格管理系統」，實驗室團隊成員包含博士生葉佳鎮、余浚瑋、馬嘉慶、蘇奕儒及碩士生林政隆等人，利用電腦系統根據商品條形碼計算出對應商品的價格與時間關係式，其中價格對時間關係式藉由初始價格、終止價格、製造日期及期限日期而定義，可以讓我們根據自身的經濟條件採購食物，有效管理而不至於造成食物浪費。 


全球每年造成食物浪費約1/3(約13億噸)可供飢餓人口吃上四年!現行解決食物過剩問題，可以利用食物銀行的輸送，但通常會有募集過產、或是過剩及賣相不佳的食品後，再進行儲存、分類與分配，雖然可減少食物的浪費，但也會有期限過後便棄之的結果。鐘國濱表示，此研發利用商品條形碼、電腦系統、輸入設備及顯示設備。商品條形碼包含有商品的初始價格、終止價格、製造日期及期限日期，期限日期在製造日期之後。以掃描商品條形碼並將商品的購買日期記錄至電腦系統中，接著電腦系統將購買日期代入價格對時間關係式中，以獲得對應購買日期的即時商品售價，以及顯示設備與電腦系統電性連接，並將即時商品售價顯示於顯示設備上。 


鐘國濱表示，一般市面上的商品皆為固定價格，以食品而言，離製造日期越遠的商品，越不容易受到挑選，且一旦超過了期限日期，店家只能將超過期限日期的商品毀損。鐘國濱說，一般快過期的商品通常會被擺放至展示櫃的前側(或上層)，但此方法被動且效果有限，聰明的消費者也多會選擇製造日期較接近採購日期之商品(較新鮮)，往往有可能因為消費者選擇底層(較新鮮)的商品，而導致商品掉落的危險。 


鐘國濱表示，以3C產品而言，時間也會造成該類型商品的價格浮動。一般店家通常以人力來重新標價，時間一久反而容易造成錯誤標價，且無法隨著商品上市時間做出實時判斷，導致店家難以正確決定目前的商品定價，也會造成消費者購物時的迷失。 


鐘國濱認為制定商品的價格隨時間而有所變動，引導消費者依據自身經濟條件，挑選較新鮮(製造日期接近採購日期、經濟條件好)或即將過期(製造日期遠離採購日期、經濟條件較差)之商品，致使商品於保存期限內都能獲得不同經濟條件之消費者青睞與消費利用，避免浪費。 


此研發主要引導製造商(廠商)依其產品特性訂出變動價格(隨時間變化)，透過此技術欲達成製造商(上游)減少庫存回收浪費，上架通路商(中游)易於管理，對於消費者(下游)能各取所需(依自身之消費能力與偏好)。目前此研發也正申請專利中，相信未來能造福更多的消費者，也可減少食物的過剩浪費。
