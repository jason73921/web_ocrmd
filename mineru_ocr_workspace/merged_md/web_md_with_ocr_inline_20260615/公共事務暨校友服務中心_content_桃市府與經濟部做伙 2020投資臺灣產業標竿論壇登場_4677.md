# 網站文章資料_桃市府與經濟部做伙 2020投資臺灣產業標竿論壇登場_4677

## 一、文章基本資料

- **文章ID**: 4677
- **文章標題**: 桃市府與經濟部做伙 2020投資臺灣產業標竿論壇登場
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20201026-5
- **文章簡介**: 【台灣好新聞/記者黃駿騏/桃園報導】

桃園市政府經發局24日與經濟部工業局共同主辦「2020投資臺灣產業標竿論壇」，廣邀回臺投資之桃園市企業等多名業界代表共同交流分享，期望透過本次論壇聽取產官學界各方意見，引導企業升級轉型，創新未來尋找方向。

「2020投資臺灣產業標竿論壇」於台北諾富特華航桃園機場飯店舉，桃園市政府副市長高安邦以「投資臺灣之契機」為主題，說明美中貿易競爭與新冠疫情影響下，牽動全球製造供應鏈重組，產業重新洗牌定位。在資訊電子業方面，今年主要國家皆卯足全力進行5G基礎建設，桃園市也配合「亞洲矽谷計畫」，結合桃園ICF首獎之智慧城市能量，積極推動智慧產業。

桃市府經發局長郭裕信在下午首場專題演講中，以「智慧製造升級未來力」為題，說明配合中央推動「亞洲矽谷計畫」，建置虎頭山創新園區，打造全台首座結合「車聯智駕中心」與「資安物聯網中心」的測試研發基地，具體以科技連結資源，建造loT智慧園區。

論壇最後由桃園市政府經濟發展局郭裕信局長與元智大學校長吳志揚共同總結，透過經驗交流，共同推動智慧城市，發展智慧及創新科技相關應用，推動桃園產業數位轉型，如智慧製造、智慧工廠，及5G 技術應用，促進經濟發展，帶動台灣產業轉型。
