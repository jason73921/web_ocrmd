# 網站文章資料_境外生返台先住防疫旅館　大學校長：暑假空宿舍多_4400

## 一、文章基本資料

- **文章ID**: 4400
- **文章標題**: 境外生返台先住防疫旅館　大學校長：暑假空宿舍多
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20200619-1
- **文章簡介**: 【三立電子報/記者簡若羽／台北報導】

因應新冠肺炎疫情趨緩，教育部昨（18日）宣布，開放港澳、越南、泰國等11個低風險國家或地區的境外生返台，以應屆畢業生優先為原則，返台後須配合入住校外防疫旅館14天。不過，元智大學校長吳志揚指出，防疫旅館費用為兩千至六千不等，令不少境外生感到壓力，，而時值暑假期間，校內獨棟宿舍數量充足，入住一天僅百元，盼政府優先規劃，安排學生於宿舍進行檢疫。

教育部宣布境外生可分批返台，首批優先開放越南、香港、澳門、泰國、帛琉、澳洲、紐西蘭、汶萊、斐濟、蒙古、不丹等11個低風險國家或地區學生入境，以應屆畢業生優先，返台後須配合入住校外防疫旅館14天，若學校有校外獨棟或租賃宿舍，須先請地方衛生單位檢視是否符合防疫規格，經教育部實地確認後，才能納為校外居家檢疫場所。

不過，防疫旅館要價不菲，恐對境外生造成經濟負擔。元智大學校長吳志揚受訪時表示，首批預計有190多位境外生將返台，而目前調查防疫旅館，為兩千到六千不等，不僅對學生造成壓力，對防疫量能來說可能也是一大衝擊，認為應善用各大學現有資源。

吳志揚舉例，元智校內有一百多間宿舍，暑假期間閒置的獨棟空間更多，足以容納檢疫學生，且過去一天只收一百元清潔費，不會造成經濟壓力。雖然佔境外生大宗的陸生尚不能返台，吳志揚認為，目前入境的人數與檢疫量能相對應，大量開放也無法容納，政府逐步分地區分階段進行是正確的。（待續）
