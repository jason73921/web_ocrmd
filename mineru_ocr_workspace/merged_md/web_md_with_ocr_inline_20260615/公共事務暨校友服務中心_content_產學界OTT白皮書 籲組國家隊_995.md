# 網站文章資料_產學界OTT白皮書 籲組國家隊_995

## 一、文章基本資料

- **文章ID**: 995
- **文章標題**: 產學界OTT白皮書 籲組國家隊
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【工商時報/林淑惠/台北報導】 


第一份由產學界代表提出的「OTT產業政策白皮書」昨天出爐，建議新政府組成OTT國家代表隊，扶植台灣本土OTT產業，以翻轉整個影視產業日漸衰敗的危機、另啟新契機。

另外，產學界代表直言，在OTT領域，台灣最大勁敵其實正是對岸的大陸，解決之道、建議應該回到兩會會談，把OTT當做一個重要議題討論。

「OTT產業政策白皮書」是由元智大學大數據與數位匯流創新中心邀請學界及產業菁英所共同撰寫，元智大學資訊傳播學系助理教授葉志良主撰，前NCC主委蘇蘅、前NCC委員劉崇堅、魏學文，以及前衛星電視公會秘書長鍾瑞昌以及學界代表共同參與撰寫。未來白皮書將送到行政院、經濟部、文化部、科技部及國發會等相關單位，同時提出四大建言，提供新政府參考。葉志良表示，Netflix、愛奇藝、Dailymotion等跨國OTT業者，正挾帶著雄厚的資金、內容以及技術進入台灣市場，本土有線電視甚至影視產業正面臨生存保衛戰，環顧歐盟、日、韓、中國早就將OTT視為戰略級產業，並制定相關政策引導產業發展，台灣已大幅落後，更應儘早訂定國家級政策急起直追。

葉志良強調，台灣市場雖小，但基礎網路設施建全、行動上網普及率高而且速度快、影視內容多元而且在全球華人市場扮演重要地位，這多項優勢，正是吸引Netflix、愛奇藝、Dailymotion落腳台灣的主要原因，外人都如此看中台灣，台灣更不能「做小」自己的能量。蘇蘅認為，台灣不缺影視人才，但近年這些優秀人才卻大量且快速外流，再者，很多國家都已組成國家隊在打國際市場，台灣卻還是單打獨鬥的局面，她建議，政府的跨部會產業政策應該儘快出爐，並儘快在匯流五法鬆綁相關法條。由五大電信公司組成的台灣電信產業發展協會秘書長劉莉秋則以唐三藏西方取經為例，暗喻政府開放跨國OTT業者來台，卻忘了要對跨境OTT業者規範一個「緊箍咒」，呼籲新政府應要求跨境業者在台設立登記，以建立公平競爭環境。
