# 網站文章資料_2016年第七屆大專院校物聯網暨RFID專題競賽結果出爐_1046

## 一、文章基本資料

- **文章ID**: 1046
- **文章標題**: 2016年第七屆大專院校物聯網暨RFID專題競賽結果出爐
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【GS1 Taiwan】 


GS1 Taiwan為促進物聯網暨RFID之應用，特辦“2016年第七屆大專院校物聯網暨RFID專題競賽”，並於6月3日舉行決賽，此次分為應用實作組，與創意專題組，各隊以實際作品與生動簡報展開激烈的競爭，並由來自學術單位老師以及業界專家共6名評審委員進行評分。 


在精彩的決賽後，應用實作組由逢甲大學以「S-Parking」主題獲得首獎，國立澎湖科技大學以「應用於活水產品產銷履歷之RFID讀取器天線」主題獲得特優獎。創意專題組則由和春技術學院以「NFC保全巡檢系統」主題獲得首獎，元智大學以「綠色寵物」主題獲得特優獎，實踐大學高雄校區以「停車 Easy Go」獲得佳作。GS1 Taiwan今年第七年舉辦專題競賽，獲得各大專院校的響應，今年參賽隊伍共13隊報名，入圍決賽共有5隊，在一輪精彩的比賽後，其排名為： 


應用實作組： 


1. 首獎：逢甲大學『S-Parking』- S-Parking 


2. 特優獎：國立澎湖科技大學『健倫與他的小夥伴們』- 應用於活水產品產銷履歷之RFID讀取器天線 


創意專題組： 


首獎：和春技術學院『IOT巡邏兵』- NFC保全巡檢系統 


2. 特優獎：元智大學『Pet Master』- 綠色寵物 


4. 佳作：實踐大學高雄校區『The Parking Life』- 停車 Easy Go 


GS1 Taiwan為培養國內EPCglobal標準及物聯網人才，積極扮演國際與台灣產官學研業界的橋樑角色，經過七年的經營，仍大量投注資源人力於此項活動，主要是有感於眾多學校師生的熱情投入，也看到效果在產學合作間擴散，期待有助於此產業的未來發展。
