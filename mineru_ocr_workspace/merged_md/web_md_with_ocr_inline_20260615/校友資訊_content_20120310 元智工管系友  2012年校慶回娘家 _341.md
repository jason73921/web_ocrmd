# 元智工管系友校慶回娘家

元智工管系藉由23週年校慶活動，於2012年3月10日，假元智大學活動中心陶邑餐廳，舉辦工管系系友會，邀請系友回娘家參與盛會·當日出席踴躍，深刻感受到系友們對系上的熱情與關心！

鄭春生主任·蔡篤銘教授·陳啟光教授、丁慶榮教授、林瑞豐教授、周金枚教授等多位師長及同仁到場與系友敘舊、談天，系友會張啟昌會長率系友們開心回味求學時的點點滴滴，也與師長們分享工作近況及關心系上近來成長與轉變，學長姐與學弟妹間密切分享傳承經驗，並交換工作中的酸甜苦辣，不時可聽到大家爽朗的笑聲及驚呼聲·藉由此次活動，除增進系友對元智的熟悉度，期許建立元智大學親切且務實的畢業校友形象，並盼在未來選擇新進員工時，學長姐提攜元智校友為第一錄用選擇。
