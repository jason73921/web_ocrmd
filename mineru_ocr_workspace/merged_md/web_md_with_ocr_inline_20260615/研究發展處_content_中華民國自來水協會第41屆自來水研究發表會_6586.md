# 網站文章資料_中華民國自來水協會第41屆自來水研究發表會_6586

## 一、文章基本資料

- **文章ID**: 6586
- **文章標題**: 中華民國自來水協會第41屆自來水研究發表會
- **文章網址**: https://www.yzu.edu.tw/admin/rd/index.php/tw/?option=com_content&view=article&id=6586
- **文章簡介**: 主旨:「第41屆自來水研究發表會」訂於1 1 3年1 1月14日(星期四)假輔仁大學濟時樓舉行,歡迎共襄盛舉,請查照。

說明:

一 、中華民國自來水協會為增進產、官、學界對自來水相關技術及管理研究與應用之經驗交流,特舉辦「自來水研究發表會」,以提昇供水技術、管理效能與服務品質。

二、本屆發表會徵稿包括四大主題: ( A )工程技術( B )營運管理( C )水質安全( D )供水服務,主題具體內容請參閱「第41屆自來水研究發表會徵文通告」海報。

三、惠請於徵文通告掃描QR CODE填寫報名資料,並於1 13年7月26 日(星期五)前,將500字以內之研究報告摘要E一mai l至本籌備會 (zcan@mail.water.gov.tw)。另於113年8月9日(星期五)前通知審查結果,正式研究報告則於113年8月28日(星期三)前截止收件。

四、    有關自來水研究發表會最新資訊,請至自來水協會網站參閱，網址:http://www.ctwwa.org.tw/最新消息或至台灣自來水公司第十二區管理處網站下載,網址: https://www.water.gov.tw/dist12/ 訊息公告/本處新聞。

五、    檢附「第41屆自來水研究發表會徵文通告」海報1份 ,、請張貼公告,予以宣傳。
