# 蒙特梭利老人失智照護工作坊實施計畫

1. 依據：財團法人陳建志毅宇未來教育基金會委請國立政治大學承辦蒙特梭利全球影響力計畫，為推廣蒙特梭利教育，及其應用於老人失智照護領域，於109年3月辦理工作坊，推廣其理念並培力相關人才。

2. 目的：

1、 促進教育工作者、家長及一般社會大眾對蒙特梭利運用於長照教育的理解。

2、 培力相關教育及長照工作者實踐蒙特梭利理念。

3. 辦理單位：

主辦單位：財團法人陳建志毅宇未來教育基金會、政大臺灣蒙特梭利教育基地。

4. 辦理時間：109年3月13日(星期五)至109年3月15日(星期日)共3天。
