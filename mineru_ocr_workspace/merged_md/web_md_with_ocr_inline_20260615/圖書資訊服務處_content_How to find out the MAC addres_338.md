- **文章網址**: https://www.yzu.edu.tw/admin/is/index.php/tw/how-to-find-out-the-mac-address
- **文章簡介**: XP


Click  Start  ->  All programs  ->   Accessories -> Command Prompt




    Type in “ ipconfig/all ”  and press enter key

    Find the MAC address next to the label Physical Address  




 

Windows 7


Click  Start -> Command Prompt 

     


    Type in “ipconfig /all” and press enter key

    Find the MAC address next to the label Physical Address
