# 111 年度桃園市政府衛生局母嬰親善醫療院所暨母嬰照護醫護人員研習會報名簡章

一、為使醫護人員具備正確的母乳哺育及照護相關知識，並提供正確的母乳哺育觀念予孕產婦及其家人，辦理本研習會(視COVID-19新冠肺炎疫情狀況調整辦理時程、場次及報名人數)。

二、報名對象：本市推廣母乳哺育之醫護人員或有興趣參訓者。

三、報名方式：採線上報名，不接受現場或臨時報名（限本市醫護人員及母嬰親善醫療院所為優先報名），線上、實體課程每場次各150名(額滿為止，倘因 COVID-19新冠肺炎疫情影響則以線上方式辦理，辦理時程將另行公告於本局網站)，相關資訊請逕至桃園市政府衛生局網站首頁>業務資訊>健康促進服務>婦幼衛生查詢。

四、課程講義、線上課程軟體及會議代碼將另行以電子郵件提供予與會者。

五、報名費用：免費。
