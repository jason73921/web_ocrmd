# 網站文章資料_【宿服組公告】114學年度學生宿舍抽籤作業辦法_5950

## 一、文章基本資料

- **文章ID**: 5950
- **文章標題**: 【宿服組公告】114學年度學生宿舍抽籤作業辦法
- **文章網址**: https://www.yzu.edu.tw/admin/st/index.php/tw/?option=com_content&view=article&id=5950
- **文章簡介**: 公告日期：114年3月17日（星期一）

 

主旨：公告114學年度學生宿舍抽籤作業辦法

說明：

一、為保障您的權益，抽籤前請詳閱下列應注意事項：

(一) 學生取得住宿資格，以住滿「114學年上、下兩學期」並應繳交兩學期住宿費為原則（無正式學籍學生除外），住宿費「以年定價」，分上、下學期各繳納一次，與學雜費一併繳交。住宿期限未滿欲中途退宿者需提出申請，未核准中途退宿者，一律不退還住宿保證金及當學期住宿費或應繳交次學期住宿費。但因特殊原因如畢業、休學、退學、轉學、實習、出國交換及不可抗力之原因經核可者，不在此限。校內宿舍寢室設計皆為４人一間，對聲音敏感、有睡眠障礙、身心暨有疾病易受團體生活影響者，建議避免申請校內宿舍，以保障您的身心健康。 

(二) 中籤後續配合事項：正式取得住宿資格要件→ 3/25～3/27限期繳交住宿申請切結書以及住宿保證金。相關事宜請詳閱後續中籤公告，摘要如下。

1. 住宿申請切結書建議提前備妥，以免中籤後繳交不及。（表單下載）

2. 112-2起恢復住宿保證金預繳機制，請逕行至〈Portal→免到校註冊→住宿保證金〉查詢繳款帳號後繳費。繳款帳號每人不同，切勿使用他人繳款帳號繳納自己的保證金。

3. 以上兩者缺一皆視同自動放棄，當事人不得異議及要求任何補救措施。

(三) 放棄注意事項：

1. 3/23住宿申請系統關閉前：請逕行返回Portal「取消申請」。

2. 中籤且已繳交住宿保證金後：請至宿服組辦理放棄住宿資格手續，不退還已繳之住宿保證金。

3. 7/10資料轉檔前：有任何異動應《立即主動》告知宿服組，如權益受損，責任自負。

4. 7/10資料轉檔後：不再受理非特殊原因床位放棄申請並續依宿舍管理規則辦理相關事務。

 

二、抽籤資格：

(一) 大一～大三在學之大學部學生

(二) 碩一至碩二在學之碩士班學生

(三) 下列學生請勿參與抽籤：

1. 已申請優先住宿並經核准者。 2. 宿舍自治會幹部。 3. 資服處推薦之網管人員。 4. 體育室推薦之校隊代表。 5. 境外生（含外籍學位生、僑生、陸生、雙聯生），不需抽籤但須提出續住申請（已於2月完成辦理）。

 

三、開放抽籤期程：114年3月21日(星期五)10:00起至114年3月23日(星期日)23:59:59止。

 

四、抽籤方式：

(一) 抽籤作業程序

(二) 登入個人Portal點選『住宿申請』選項，並依指示完成抽籤程序，電腦將隨機給號，籤號在10000至99999之間。

(三) 取得籤號後依地區分數作加權，再依加權後的號碼由小到大排序，排序在釋出名額內者即中籤。

(四) 加權分數：高雄、屏東、台東、花蓮 ：20% │ 彰化、南投、雲林、嘉義、台南：15% │ 宜蘭、苗栗、台中：10% │ 基隆：5%。

(五) 籤號排名將根據後續抽籤結果而變動，最終排序請以抽籤公告的結果為準。

 

五、中籤公告日期：114年3月24日（星期一）。

 

六、床位分配方式：

(一)開放抽籤床位數：男生310床，女生320床。

(二)主要住宿區：大學部以男女二舍為原則，碩士班以男女一舍為原則。

(三)中籤者可自組室友，未自組室友者由宿服組權責分配。

(四)寢室分配由宿服組全權決定。

 

七、後續床位作業預告：

3/24 中籤公告 │ 3/25~3/27 繳交住宿申請切結書及住宿保證金 │ 4/8 公告一抽正式取得住宿資格名單 │ 4/14 公告第二次抽籤作業辦法。

# 元智大學住宿申請切結書

學生資料

□大學部 □碩士班 □博士班 申請學期： 114-1 及\_ 114-2

姓名： 系所班： 年級： 現年

學號： 手機號碼：

緊急連絡人姓名： 聯絡電話：

本人於住宿期間，願意履行並遵守下列住宿規定。

## 一、 承諾配合事項

1. 同意住宿學生宿舍 1 學年（包含上、下兩學期）；除特殊原因如畢業、休學、退學、轉學、實習、出國交換及不可抗力原因經核可者外，均不得申請退宿、退費。

2. 簽署住宿同意書後，一學年內非經同意決不可無故退宿，請妥慎考慮團體生活自我適應能力以及健康狀態是否適宜住宿。若違約自願放棄住校，未繳納住宿費者無法完成註冊程序，已繳納住宿費者無法退費。

3. 寒暑假住宿須另行申請及繳費（續住之研究生除外）。

4. 2,000 元保證金將於辦理退宿驗收項目合格後無息歸還，驗收項目：住宿期間是否違規、床位是否清空整潔、退宿手續是否完成等。

## 二、 承諾住宿規定

1. 住/退宿時間以公告為準，特殊原因放棄床位者須至宿服組辦公室辦理相關手續。

2. 每學期因違反住宿相關規定扣滿 15 點（含）以上者，取消住宿資格且不退還住宿費及保證金。

3. 愛惜使用宿舍公物，因故意或過失致毀損宿舍內設施或裝備物品者，願負修繕或損害賠償之責。

4. 配合宿舍清館時間，辦理離/退宿程序。

5. 遵守本校「學生宿舍管理規則」及「學生宿舍生活須知」相關規定。

申請人(學生)： （簽名） 保證人(家長)： （簽名）
