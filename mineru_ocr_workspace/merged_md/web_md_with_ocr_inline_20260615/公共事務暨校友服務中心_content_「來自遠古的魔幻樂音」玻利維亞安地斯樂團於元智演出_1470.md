# 網站文章資料_「來自遠古的魔幻樂音」玻利維亞安地斯樂團於元智演出_1470

## 一、文章基本資料

- **文章ID**: 1470
- **文章標題**: 「來自遠古的魔幻樂音」玻利維亞安地斯樂團於元智演出
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/

- **文章全文**: 【Yahoo新聞】

元智大學藝術中心每年翻新活動主題，盼藉由不同主題的演繹，將不同形式的藝術活動引入校園，以活化校園文化氛圍，創造校園藝術生活品質。11月23日晚上七點，邀請到玻利維亞安地斯樂團，於元智大學六館有庠廳進行演出，主題為《來自遠古的魔幻樂音》，歡迎社區民眾踴躍前來聆賞這一場深具民族風情特色的音樂會。

元智大學藝術中心主任鄒淑慧表示，元智大學藝術中心每年翻新活動主題，今年主題為「在轉角遇見‧藝術好好玩」，是以玩耍藝術的心情帶領人們探索生活與美的各種可能，發掘非典型藝術之有機空間及無限想像。本學年表演活動便從「藝術快快樂樂，走進校園的美好生活」的重點契入，上下學期各規劃一場具有創意活力的表演，讓全校師生能夠從活動中，玩味表演藝術，體認生活文化。

11月23日晚上七點，特別邀請到玻利維亞安地斯樂團於元智大學六館有庠廳進行演出，主題為《來自遠古的魔幻樂音》。安地斯樂團在玻利維亞已成立多年，為了發揚藝術傳承文化，他們努力將印加文化音樂推及於亞洲地區，甚至全世界。團員皆來自安地斯山脈中不同的國度，因對音樂的喜好與執著而組成安地斯樂團，該團自從1997年受外交部邀請來台做親善表演後，陸續參加中央與地方政府單位的文化交流活動，包括各縣市的藝術節表演。無論是在慶典中與大夥兒熱熱鬧鬧地唱歌跳舞，或者是純音樂性表演活動中，他們都能讓合諧而歡樂的安地斯樂曲觸動聽眾的心弦。

本次表演內容是從印加傳統出發，其優美的旋律中暗藏著神秘色彩，空靈而飄邈的樂符帶領觀眾穿越時空，聆聽過去印加王朝傳統國度的古老樂章，從智利到秘魯，再到玻利維亞，讓人體會安地斯山脈純真的風土民情，以及感受拉丁民族的熱情。千百年流傳下來的樂音魔力，透過數十種傳統高山樂器包括了排笛、木簫、拉丁吉他、十弦琴、直笛以及拉丁打擊樂器等，不但帶出印加音樂特有的蒼涼、輕快、哀愁、壯闊、樂天的特性，還有異國的浪漫，讓裊裊笛音和律動鼓聲將聽者化身為蒼鷹，隨著笛音飛翔在安地斯山的神奇幻想中。

歡迎社區民眾踴躍前來聆賞這一場深具民族風情特色的音樂會，現場直接感受印加音樂的浪漫、熱情、豐富、多樣的迷人特性，不必遠飛到半個地球之外的南美洲。
