# 網站文章資料_ 國際新航向 元智「千人出國計畫」送學生國際交流學習_7449

## 一、文章基本資料

- **文章ID**: 7449
- **文章標題**:  國際新航向 元智「千人出國計畫」送學生國際交流學習
- **文章網址**: 
- **文章簡介**: 【國立教育廣播電台】
 
元智大學因應校園國際化，推動「大專校院學生雙語化學習計畫（EMI）」全英語課程學習環境，將籌辦「千人出國計畫」，預計選送至少1000名學生，提供獎助學金，鼓勵學生積極參與國際交流學習。
 
元智大學教務長謝建興表示，學校積極推動全英語授課，110年管理學院獲教育部第一期「大專校院學生雙語化學習計畫（EMI）」重點培育學院。112年管理、資訊和電機通訊三個學院同時榮獲教育部第二期「大專校院學生雙語化學習計畫」重點培育學院的殊榮。
 
元智大學教務長謝建興指出，推動的「千人出國計畫」除了配合教育部的學海系列計畫，進一步推動學生到海外研習進修的機會，透過元智與合作學校的緊密合作，學生可參與短期進修、暑期海外研習營、海外移地教學、海外實習課程以及國際志工服務隊等多元化的學習機會(十天以上)。此外，學生可參與國際研討會議並進行學術發表，豐富學生學習經歷。三個重點學院合計每年提供超過300個到海外交流機會，溪望元智下一期成為教育部重點培育大學。（待續）
