# 網站文章資料_元智管院「2016年千里馬盃模擬創業競賽」競賽獲銅牌_1562

## 一、文章基本資料

- **文章ID**: 1562
- **文章標題**: 元智管院「2016年千里馬盃模擬創業競賽」競賽獲銅牌
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【大紀元/記者徐乃義／桃園報導】

元智大學管理學院教授徐樂文，指導管理學院大三同學呂佳璇、林如芸、曾安翹、黃俞文、李佩瑩，組成「少年你說的隊」，以「『CA』llecting C ─ 群眾集『卡』」參加「2016年千里馬盃模擬創業競賽」社會企業創業組，獲銅牌獎。

「2016年千里馬盃模擬創業競賽」希藉競賽凝聚全國各地熱血的青年創業家，發揮年輕人創新創意的力量，並加入國際創新創業組，加強與國際之鏈結，彼此交流學習與經驗分享，促進青年的創業有更廣闊的空間，帶來更好的創業知識、機會，以實踐青年的創業夢。 

元智大學管理學院客座助理教授徐樂文表示，「CA」llecting C ─ 群眾集「卡」是以「卡路里共享」的概念，內容包含一個集卡路里的卡和手機App。藉由與健身房業者合作，累積客戶在運動時消耗的卡路里數，當達成階段性的目標時，此社會企業組織會將此轉換給弱勢需要營養早餐的孩童，相應份數的營養早餐，同時和運動手環(紀錄卡路里數)等相關企業合作，將群眾集卡的理念拓展至各地，以幫助更多需要幫助的弱勢孩童及其營養早餐(基金)。徐樂文表示，主辦單位與桃園青年事務局都在陸續作後續創業的連結，學生也田野調查許多周邊的健身房及早餐店，希望能擴及更遠及偏鄉小學等。

徐樂文表示，其實整個企畫就是創造一個連結，從整合桃園健身房出發、早餐店，進而擴充到全台。它是一個全民健身與公益觀念的激勵連結，健身房因此會員變多，一般民眾又可因此消耗多餘卡路里而健身，而偏鄉學童因此又有營養早餐或早餐基金，營造三贏之局面。
