# 網站文章資料_救國團桃園市團委會慶祝六十四週年團慶_1534

## 一、文章基本資料

- **文章ID**: 1534
- **文章標題**: 救國團桃園市團委會慶祝六十四週年團慶
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【中央社】

救國團桃園市團委會於十月三十日(星期日)早上十時至十二時於桃園市平鎮區青少年平鎮區青少年活動中心辦理慶祝本團成立六十四週年團慶大會，邀請各界貴賓及本會各義工夥伴一同參加慶祝，上午在會中除了表揚熱心團務工作的義工夥伴，當日下午辦理全縣義工暨眷屬聯誼活動。 


典禮由本會主任委員吳志揚主持，邀請本市各界貴賓及義工五00人參加；大會程序和以往不同，參加人員報到後，可以欣賞到精采的新興高中爵士拉拉舞蹈、中大壢中儀隊、元智大學薩克斯風社團及本會終身教育師生恰恰舞蹈等表演，也可以看到救國團各義工單位今年辦理各項社會服務公益活動的影片成果，表現出本團公益服務的精神；顯現出救國團經歷了六十四年仍然是現代社會中活力的代表；團慶大會中共有社會團務義工184人及學校兼任幹部24人接受表揚，頒發獎章及證書，感謝這些夥伴平日對公益服務的貢獻。 


本年學校團務兼任幹部受獎人員共24人。社會團務各區隊會義工夥伴獲得勞績獎章有31人，獲得服務獎章有55人，獲得熱心服務獎有98人。 


當日下午義工暨眷屬聯誼活動採連線闖關方式進行，共分16站進行趣味活動闖關及摸彩活動，更加增添活動的熱鬧，本會社會團務義工展現出平日所練習，有活力四射的舞蹈表演；團慶大會及義工聯誼活動於下午十六時結束。
