# 網站文章資料_美日韓難擠入僵屍頻道表 學者憂不利國際形象_3509

## 一、文章基本資料

- **文章ID**: 3509
- **文章標題**: 美日韓難擠入僵屍頻道表 學者憂不利國際形象
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20190117-3
- **文章簡介**: 【台灣壹週刊/撰文何醒邦】

有線電視頻道20年來變動率低於1成，被稱為「萬年不變的頻道表」，本土新頻道或國際新頻道，想要擠進前80台的「前段」頻道分潤，都有一定困難度。

有線電視系統業者凱擘，在2018年5月向NCC（國家通訊傳播委員會）申請移頻案，打算將衛視合家歡（原45台）移至145台、壹電視新聞台（原49台）移至149台、JETTV（原80台）移至45台、寰宇新聞台（原222台）移至49台、WAKUWAKU JAPAN（原212台）移到80台。但NCC以考量地方政府大多反對、各機構收視率報告及消費者權益，在本月初將此案打回票。

之後，立委段宜康在臉書指出，日本的WAKUWAKU JAPAN和南韓頻道tvN，因與系統業者的授權金談不攏，應該快看不到了，但龍祥因為定頻在64台，靠著不斷重播，就可每年收到5千萬元的權利金。此番言論，也讓「萬年頻道表」問題再度浮出檯面。

元智大學資訊管理學系教授周韻采則指出，2005年，當時行政院新聞局長林佳龍制定全國一致的有線電視頻道表，沿用至今14年，使得劣幣驅逐良幣，有線電視重播又難看；而頻道變更的僵固性，如同死水一灘，不僅沒競爭力，也影響消費者收視權益，且如果境外頻道紛紛離台，也不利於台灣的國際形象。（待續）
