# 網站文章資料_推動長照 莫計較本勞外勞_1514

## 一、文章基本資料

- **文章ID**: 1514
- **文章標題**: 推動長照 莫計較本勞外勞
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【伊佳奇】 


「長照2.0」才剛試辦，照護人力嚴重不足，民進黨立委吳玉琴日前在立法院提出，要求勞動部3個月內檢討修正《就業服務法》中，規定85歲以上輕度失能老人可聘外籍看護的條文，她認為這項條文影響長照2.0的推動，且排擠本國家事服務員的工作權，但此舉卻完全漠視85歲以上輕度失能老人被照護的權益，若老人基本權益都忽視，如何奢談老人福利。

去年8月，勞動部提案修正《外國人從事就業服務法第46條》，在上屆立法院獲得三讀通過後，公告85歲以上輕度失能老人可聘僱外籍看護。根據勞動部提供修法的相關資料，該部民調顯示，6成有老人的家庭認為若沒人照顧失能老人，家中長輩健康恐會惡化，另外高達近9成支持開放輕度失能長輩也可聘外籍看護。同時，該部也調查曾受訓取得服務資格本國看護的意見，有5成反對，但沒意見及不反對者合計也有5成。 

85歲以上輕度失能老人在生活照護上，因失能已逐漸生活無法自理，是需要他人協助，如果不送機構照護，是以社區化與在地化的方式提供照護，可協助家庭提供服務者包括居家照服員、長照2.0社區據點服務、外籍看護等。

衛福部早已坦誠表示，推動長照2.0政策，明年開始的計畫服務對象要成長44%，達73萬8000人，照服人力保守估計尚有1萬3000人的缺口。事實上，在長照1.0階段，照服人力一直持續呈現不足狀態下，往往無法提供足額的法定居家服務時數。

此外，長照2.0才開始試辦，要將舊有8項服務增加9項到17項，新試辦的A、B、C級各服務單位在整合上需要時間磨合，若能維繫原有的服務能量已屬不易，遑論立即接手新的服務對象與增加新的服務項目。 

立委提出這項提議時，若能先確保有足夠的本國籍照服員或長照2.0社區服務能量，對於85歲以上輕度失能老人在生活照護上無後顧之憂，深信大部分民意會傾向接受，但如果無法提出保證，僅為本國籍照服員的工作權，及目前仍是「看得到、卻用不到」的長照2.0來修法，使得85歲以上輕度失能老人在生活照護上頓失人力照護，所衍生的後遺症，將付出更多的社會成本。

長照2.0政策目的是滿足被照護者的生活照護需求，目前參與長照2.0的民間團體是負責提供服務，如果修法目的僅在保護這些團體的利益與生存，漠視長照2.0的目的與被照護者的生活照護權益，本末倒置的修法行為，將忽視100多萬家庭的照護需求，政府決策時應慎思。（作者為元智大學老人福祉科技研究中心顧問）
