# 網站文章資料_【轉知】財團法人車輛安全審驗中心函，有關調查國內有意願辦理車輛安全性瑕疵檢測之機構或專家學者名單_479

## 一、文章基本資料

- **文章ID**: 479
- **文章標題**: 【轉知】財團法人車輛安全審驗中心函，有關調查國內有意願辦理車輛安全性瑕疵檢測之機構或專家學者名單
- **文章網址**: https://www.yzu.edu.tw/admin/ho/index.php/tw/1005
- **文章簡介**: 財團法人車輛安全審驗中心 函

主旨：有關調查國內有意願辦理車輛安全性瑕疵檢測之機構或專家學者名單乙案，詳如說明。

說明：

一、交通部先前函請本中心徵詢彙整國內有意願辦理車輛安全性瑕疵檢測之機構或專家學者名單並陳報在案，考量該名單自前次更新迄今已逾2年，為求資料正確性擬進行更新； 另因應汽車買賣定型化契約應記載及不得記載事項自今 (110)年7月1日起施行之修正內容，爰就旨揭之受理案件種 類增列「綜合研判」並重新調查受理案件類型等事宜，惠 請各單位協助更新旨揭名單與資訊，俾利提供交通部後續 參考辦理。

二、惠請貴單位依前述說明，協助調查內部專家、學研單位或 相關機構，有無意願受理車輛安全性瑕疵案件之檢測、鑑 定或綜合研判(意指在不涉及檢測技術、場地、設備能量， 且未涉車輛工程或零件專業技術亦無須探究瑕疵成因為前提下，協助確認瑕疵存在性)，並請於110年10月29日前依不同車輛專業領域(如汽車、機車、自行車)填寫「車輛安 全性瑕疵檢測機構或專家學者名單資料表」(如附件一)回 復本中心彙整，俾利辦理後續作業。

三、有關「車輛安全瑕疵性檢測機構或專家學者名單資料填寫表」可依下述方式擇一回復： (一)填寫附件二表格資料後，以傳真方式回復。(傳真電話： 04-7810401)； (二)請於車輛安全資訊網/便民服務/下載專區/文件下載電子 檔，並填妥表格資料後，以E-MAIL方式回復本案聯絡 人。(下載網址：https://www.car-safety.org.tw /car_safety/TemplateOneList?OpID=402車輛安全性瑕 疵檢測機構或專家學者名單資料填寫表.xls)

四、本中心依交通部所交辦蒐集彙整「車輛安全性瑕疵檢測機 構或專家學者名單資料表」之個人資料，以提供有車輛安 全性瑕疵檢測需求之主管機關或民眾參考，故請願意受理 車輛安全性瑕疵檢測鑑定案件之單位聯絡人或專家學者個 人，填寫同意書(如附件二)併同該名單資料表提供予本中 心辦理，本中心將依個人資料保護法等規定辦理。

五、本案聯絡人：林永洲，聯絡電話：04-7812180轉7253、EMAIL：yongjhou0724@vscc.org.tw。
