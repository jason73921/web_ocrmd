# 網站文章資料_國立嘉義大學「2019年全國物理教育聯合會議」_2578

## 一、文章基本資料

- **文章ID**: 2578
- **文章標題**: 國立嘉義大學「2019年全國物理教育聯合會議」
- **文章網址**: 
- **文章簡介**: 主旨：本校訂於108年8月22日(星期四)至8月24日(星期六)在蘭潭校區舉辦「2019年全國物理教育聯合會議」，歡迎各大專院校學術先進踴躍報名參與，請查照。
說明：
一、2019年全國物理教育聯合會議將於108年8月22日(星期四)至8月24日(星期六)舉辦，會議地點為嘉義市東區國立嘉義大學（蘭潭校區國際會議廳、圖書館演講廳、綜合教學大樓等會場），會議報名及大會網址如海報二維QR碼。
二、今年會議主題包含近期熱門教育議題(探究與實作、教師增能、科普推廣、大學課程改革、物理演示等)，更包含逾100隊全國高中物理探究與實作競賽獲獎學生團隊參與發表研討，及嘉義區各級學校代表共同加入物理教育推動聯盟，三日活動內容充實且豐富，學術先進欲投稿發表者，請參考大會論文摘要徵稿訊息。
三、截稿時間：108年7月30日（星期二）(細節請參考大會網址)，若有相關問題，請洽大會聯絡主席蘇炯武老師(05-271-7990)或電子物理學系辦公室(05-271-7911)。
