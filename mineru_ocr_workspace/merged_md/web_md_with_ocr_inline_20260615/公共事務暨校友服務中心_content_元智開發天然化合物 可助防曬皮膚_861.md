# 網站文章資料_元智開發天然化合物 可助防曬皮膚_861

## 一、文章基本資料

- **文章ID**: 861
- **文章標題**: 元智開發天然化合物 可助防曬皮膚
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【中央社/記者許秩維】

元智大學團隊針對天然化合物研究開發，利用其抗輻射、保護DNA結構等特性，除了發展美白抗皺、防曬保濕的化妝品，也可應用在化療上，幫助保護皮膚細胞。 

元智大學生物科技與工程研究所長魏毓宏帶領的團隊，投入「天然化合物―四氫嘧啶之生產與應用關鍵技術開發」計畫，未來可應用在生技化妝品產業，還可作為癌症化療時的細胞保護劑，該計畫獲得科技部3年補助總經費新台幣1500萬元。 


魏毓宏表示，團隊針對天然化合物「四氫嘧啶(Ectoine)」進行研究開發，發現四氫嘧啶不僅可穩定細胞內蛋白質和其他細胞的結構，也能減少紫外線引起的DNA鏈斷裂、抑制動物體內「酪胺酸酶」活性，還能結合水分子，有助於可滋潤皮膚。 


魏毓宏指出表示，癌症化療時，有時會因化療輻射造成皮膚組織壞死或皮膚出現變紅、癢、脫皮、色素沉著、乾燥等情況，而四氫嘧啶具保護生物細胞特性，即使皮膚長期暴露在輻射下，四氫嘧啶也能抗輻射並保護細胞，因此可應用在癌症化療時，作為保護正常細胞的保護劑。 

元智大學生技所與化材系也整合組成跨領域研發團隊，未來將根據四氫嘧啶的特性，以生物製程手段，開發可應用於美白、保濕、抗皺等生技化妝品，以及作為癌症化療時細胞保護劑。
