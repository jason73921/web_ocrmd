# 網站文章資料_元智大學校園景點故事徵選結果出爐 _417

## 一、文章基本資料

- **文章ID**: 417
- **文章標題**: 元智大學校園景點故事徵選結果出爐 
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 元智大學日前舉辦校園景點故事徵選，徵文結果已出爐，第一名作品「腳尖接腳跟」由元智大學管理學院經管所學生沈家帆獲得；第二名作品「有限的空間，無限的延伸」由元智管理學院企管系三年級陳宇康，筆名「心活泉」所獲得；第三名是目前就讀桃園市立永豐高中二年級的黃予璿，作品為「那抹夏天的私心」。

元智校園景點故事徵選於2015年11月初舉行，為期約兩週的徵文，共有50件作品參賽，參賽者有校友、在校學生，社區民眾以及高中生。而且文筆都非常熟練，故事也扣人心弦。獲得第一名是元智經營管理所學生沈家帆，他以腳尖與腳跟的碰觸，描寫元智六館之心情故事；而第二名陳宇康則是以雕塑家李再鈐先生之作品「無限延續」為題發揮，意境深遠；第三名黃宇璿，永豐高中二年級學生，他以校門口林蔭大道為引，描寫元智景點絲絲入扣。

元智大學公事室主任蔡佩瓊表示，校園景點故事的徵文，除了讓大家發覺元智校園之美外，更重要的是，訓練年輕人寫作能力與故事的串聯。寫作需要長期的訓練，短時間一蹴而成不太可能，多讀、多寫才是最實在的方法，元智大學的經典50，要求學生讀經典，即是植基在這樣的基礎上。

蔡佩瓊說，本次活動參與者踴躍，參賽者文筆都相當好，故事敘事也深富情感，其中獲得第三名的是位高中生，他的文筆獲得幾位評審共同的讚賞，值得年輕學子學習效法。

這次比賽參賽者除需於「元智大學電子報」粉絲團按讚才可報名。獲獎者第一名可得5,000元禮券，第二名獲得4,000元禮券；第三名獲得3,000元禮券；佳作每名各得1,000元。得獎者並將安排元智電子報採訪，參賽者之詩文，未來將請元智藝設系學生，以裝置藝術方式呈現於校園內，以供觀賞。
