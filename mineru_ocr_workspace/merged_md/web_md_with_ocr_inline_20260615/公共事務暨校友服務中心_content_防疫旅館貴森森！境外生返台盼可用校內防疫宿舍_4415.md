# 網站文章資料_防疫旅館貴森森！境外生返台盼可用校內防疫宿舍_4415

## 一、文章基本資料

- **文章ID**: 4415
- **文章標題**: 防疫旅館貴森森！境外生返台盼可用校內防疫宿舍
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20200619-16
- **文章簡介**: 【旺報/簡立欣】

中央流行疫情指揮中心17日宣布有條件讓境外生返台，不過要求境外生要集中在校外檢疫旅館完成檢疫程序再返校，元智大學校長吳志揚18日表示，其實校方之前都已經準備好宿舍可以讓境外生集中檢疫，且校外檢疫旅館費用也偏高，建議指揮中心可以讓學校盡量使用校內資源。

吳志揚表示，學校對於境外生返台都有相關準備，宿舍也都準備好，教育部高教司司長朱俊彰之前也都親自到校訪視。不料最近新聞出來，又要求境外生要使用校外檢疫旅館，校方只好緊急去跟桃園市政府協調，還已經問到雙北的防疫旅館，但有些價格偏高，有些甚至要7、8000元，，遠遠不止目前各校每人每日防疫宿舍費用約2000元，負擔較大。（待續）
