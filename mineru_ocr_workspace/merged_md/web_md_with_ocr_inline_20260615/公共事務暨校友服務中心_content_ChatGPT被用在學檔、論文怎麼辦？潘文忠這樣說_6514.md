# 網站文章資料_ChatGPT被用在學檔、論文怎麼辦？潘文忠這樣說_6514

## 一、文章基本資料

- **文章ID**: 6514
- **文章標題**: ChatGPT被用在學檔、論文怎麼辦？潘文忠這樣說
- **文章網址**: 
- **文章簡介**: 【中華日報/記者王誌成／台北報導】

聊天機器人ＣｈａｔＧＰＴ掀起熱潮，若被高中生代筆使用在學習歷程檔案，恐將衝擊大學考招公平性。教育部長潘文忠二十一日表示，ＡＩ（人工智慧）可作為學習輔助，但難以表現出個人的長處和亮點，面對教師面試，教師最想看到的是學生如何陳述自己的學習過程。

ＣｈａｔＧＰＴ也被廣泛討論可運用於論文寫作、課堂報告。香港大學日前開出第一槍，向師生發出內部郵件，禁止所有課堂、作業使用ＡＩ工具，否則將被視為潛在抄襲或欺騙。台灣國內大學仍抱持「觀望」態度，但仍未討論到禁止的程度。

外界擔心有學生會用ＣｈａｔＧＰＴ來製作備審資料、高中學習歷程檔案。潘文忠認為，人工智慧確實有它的長處，但就學習歷程檔案而言，是學生在學習過程中的歷程，這個過程需經過學校老師的審核認證，學生最終提交給至個人申請，大學也會兼顧學測成績，推甄過程除了學習歷程資料，也包含筆試、面試，也有部份科係採取實作，才能更詳細了解學生的綜合表現。

元智大學教務長謝建興表示，學界可正面看待ＡＩ的發展，即便能用ＣｈａｔＧＰＴ寫學習歷程，口試時教授深入追問幾個問題，很容易就能知道是不是學生自己寫的，ＣｈａｔＧＰＴ目前只能提供文字敘述，但許多校系要求圖表、甚至是影像的呈現，目前技術無法做到，至於更深入的學問探究，還是得靠老師和學生自己。（待續）
