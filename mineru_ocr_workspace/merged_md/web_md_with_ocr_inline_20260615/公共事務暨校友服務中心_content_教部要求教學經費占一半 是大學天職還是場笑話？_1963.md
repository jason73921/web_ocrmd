# 網站文章資料_教部要求教學經費占一半 是大學天職還是場笑話？_1963

## 一、文章基本資料

- **文章ID**: 1963
- **文章標題**: 教部要求教學經費占一半 是大學天職還是場笑話？
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20170601-3
- **文章簡介**: 【聯合報/記者吳佩旻、林良齊/台北報導】

教育部檢討過去大學教師普遍「重研究輕教學」，「高教深耕計畫」要求各大學用於教學經費不得少於五成；對此台大教務長郭鴻基表示，教部此舉等同「讓大學高中化」，在高等教育是場笑話；北科大代理校長黎文龍則認為，教學是學校的天職，大多經費本該用於教學。

郭鴻基批評，教育部不該將教學與研究二分法畫分。他指出，教授應隨時接觸最新訊息，探究最新研究，才能有足夠的教學能量，難以想像老師不做研究，該如何把學生教好。他認為，老師帶著學生做研究，能讓學生從做中學、挑戰未知，建議將研究與教學合而為一。

清華大學教務長戴念華也認為，研究與教學不可偏廢，單一著重哪一個都不好，應該讓教學與研究結合。他擔憂競爭者多，未來經費恐被稀釋，而學校又是銜接半導體等產業人才重鎮，企業看中的就是清大的研究能量，如果過度強調教學，恐怕讓產業人才「失了根」。他認為，教育部應該就各校強項去推展，讓適合研究的學校著重於研究。

「大學不是中研院！」元智大學校長吳志揚則表示樂見其成，他指出，科技部主要補助大學進行研究，教育部則制定教學及考招對策，但過去教育部和科技部的角色太類似，教育部應回頭檢視核心工作，將對教學的重視程度，反映在經費挹注上，讓教學的比重高於研究。他強調，大學的首要工作就是培育人才，老師最要緊的就是教好書。

台科大副校長江行全也說，原本在頂尖大學及典範科大計畫已規範教學經費運用，對台科大不是問題。針對教部規畫將經費的兩成撥付給各校，他也表示贊同。江行全並肯定「高教深耕」經費編列，偏重過去沒有強調的教學及社會責任。

至於教育部提出將教學成果列入教師升等考量，黎文龍說，有些學校太重視論文發表，忽略教學，教育部重視教學的方向是正確的，未來會有以更創新的方法引導學生。江行全表示，將鼓勵教師在網路上公開教學、教材及教法，這對其他學校也有益。
