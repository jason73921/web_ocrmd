# 網站文章資料_從新聞開罰事件看系統移頻的合理正當性_4643

## 一、文章基本資料

- **文章ID**: 4643
- **文章標題**: 從新聞開罰事件看系統移頻的合理正當性
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20201006-1
- **文章簡介**: 【台灣數位匯流網記者方文、陳怡雯／台北報導】

針對有線電視上下游授權交易買賣，不少專家傳播、經濟與法律學者齊聲呼籲主管機關國家通訊傳播委員會（NCC）呼籲尊重、回歸市場機制，不宜強制要業者談到授權，元智大學資訊傳播助理教授葉志良表示，NCC對於頻道授權的態度一直是「商業協商」，基於這個原則之下，NCC無辦法能強制業者一定要談到授權，而過去常以「消費者保護」強壓系統，也肯定會出問題。
事實上，NCC聲稱捍衛維護觀眾視聽權益應更要著眼在廣播電視內容上能否帶給觀眾更多元、符合公共利益以及傳遞更正確資訊內容給觀眾的標準。而不是僅以地方政府聲音與意見為由認定損及消費者收視權益，去否准系統業者申請移頻。

法學者劉定基、黃銘輝多次直指，有線電視系統業者的頻道規劃權限，涉及編輯權的行使，屬於言論自由保障的範疇。過多審查管制恐阻礙市場商業機制與競爭運作。（待續）
