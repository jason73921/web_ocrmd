# 網站文章資料_110學年上學期學生汽車通行證辦理事宜_190

- **文章簡介**: 各位同學好！

擬辦理110學年上學期(110.08.01～111.01.31)汽車識別證者，請先行上網填妥申請資料並列印申請表。110年10月4日起，攜申請表及相關證件至聯合服務櫃檯繳費辦理。

辦理程序：

(1)上網 ( http://140.138.35.130/GeneralOffice/park/ ) 填妥汽車識別證申請表印出簽名，並貼妥駕照、行照影本。

(2)至聯合服務櫃檯，由受理單位核對、登錄資料。

(3)申請人完成繳費。

(4)申請人領取車輛識別證。



 

非於上述收費時間辦理者，請於週一至週五上班時間，先與總務處警衛隊劉逢相先生聯繫辦理時間（03-4638800分機2261），俾便配合辦理。

備註：

1、本次所辦車輛識別證使用時間為110/8/1～111/1/31，逾期作廢。

2、逾當日繳費時間，請擇他日辦理，或可備妥相關資料及現金（每學期一般生收費4,000元、在職專班2,500元）託人代為繳費辦理。 (因疫情考量~學生110學年度維持未調漲前之收費標準)

3、建議以「班」為單位彙整資料，統一辦理停車證，以利繳費並迅速取得車證使用。

 

總務處 敬啟110.09.02   + 110.09.29修正
