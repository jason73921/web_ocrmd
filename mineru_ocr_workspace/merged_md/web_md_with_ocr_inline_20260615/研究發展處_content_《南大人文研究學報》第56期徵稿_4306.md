# 網站文章資料_《南大人文研究學報》第56期徵稿_4306

- **文章ID**: 4306

- **文章簡介**: 主旨：

檢送本校人文學院《南大人文研究學報》第56期徵稿啟事，敬請惠予公告，並鼓勵貴校相關系所師生惠賜稿件，以光篇幅。

說明：
一、《南大人文研究學報》第56期將於111年10月出版，該期即日起徵稿。
二、徵稿領域分「一般論文」與「臺南學專輯」。
(一)一般論文：舉凡人文相關之論著、調查報告、專題研究、書評、學術趨勢等學術性文稿均歡迎。

(二)臺南學專輯：１、關乎「大臺南地區」(民國之後臺南行政區之總稱)，文化、宗教、社會、歷史、風俗、民情之相關研究，皆屬之。２、歷史的縱軸時間上，研究過去時光，比如日治時期的臺南州區域，包含部分現在的嘉義、雲林地區，研究主軸是放在現代臺南的重心上，旁類觸及其他區域都沒有關係，可以視作相關的研究部分。

三、檢附《南大人文研究學報》第56期徵稿稿約及作者基本資料各乙份如附件，有意投稿者亦可至本學報網站（網址：http://www.liberal.nutn.edu.tw）人文研究學報專區查閱、下載。
