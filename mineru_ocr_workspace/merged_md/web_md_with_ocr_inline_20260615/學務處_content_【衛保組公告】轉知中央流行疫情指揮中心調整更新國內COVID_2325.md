# 網站文章資料_【衛保組公告】轉知中央流行疫情指揮中心調整更新國內COVID-19疫苗接種政策_2325

## 一、文章基本資料

- **文章ID**: 2325
- **文章標題**: 【衛保組公告】轉知中央流行疫情指揮中心調整更新國內COVID-19疫苗接種政策
- **文章網址**: https://www.yzu.edu.tw/admin/st/index.php/tw/?option=com_content&view=article&id=2325
- **文章簡介**: 指揮中心考量國內外疫情風險、國內疫苗供應現況及參酌各國疫苗接種政策，並依據衛生福利部傳染病防治諮詢會預防接種組(ACIP)於本年3月24日第2次臨時會議決議事項，調整國內COVID-19疫苗接種措施及期程如下：

 

 (一)12至17歲青少年Moderna COVID-19疫苗接種建議 

１、自本年3月25日起同意Moderna COVID-19疫苗用於滿12至17歲青少年族群之基礎劑及基礎加強劑接種；不同廠牌之mRNACOVID-19疫苗可交替使用。 

２、依Moderna COVID-19疫苗中文說明書核准劑量與接種時程，建議免疫功能正常對象接種2劑，並建議免疫功能不全對象接種包括基礎加強劑在內之3劑，每劑0.5ml (100mcg)。 

３、根據國外研究及監測數據顯示，延長各劑接種間隔可降低心肌炎發生比率，且不影響抗體及細胞性免疫， 爰建議第1劑與第2劑間隔至少12週(84天)，基礎加強劑與最後1劑基礎劑間隔至少4週(28天)。 

４、基於國際間監測青少年接種mRNA疫苗後心肌炎或心包膜炎之通報情形顯示，青少年族群接種Moderna疫苗後，心肌炎/心包膜炎之發生率高於其他年齡層，也較接種BNT疫苗者高且接種第2劑後發生比率高，爰接種前請提供家長及青少年相關資訊，以充分評估接種效益後接種。 

５、接種前需充分告知接種Moderna COVID-19疫苗之保護效益、可能產生之副作用(含心肌炎/心包膜炎發生風險)及接種後應注意事項，並取得家長同意，經醫師評估後始得接種。

 ６、依據mRNA疫苗臨床試驗及上市後監測資料，接種疫苗 曾出現極罕見的心肌炎和心包膜炎病例，且較常發生在接種第2劑之後以及年輕男性。接種疫苗後28天內若發生疑似心肌炎或心包膜炎的症狀，例如：胸痛、胸 口壓迫感或不適症狀、心悸(心跳不規則、跳拍或顫 動)、暈厥(昏厥)、呼吸急促、運動耐受不良(例如走幾步路就會很喘、沒有力氣爬樓梯)等症狀，請務必立即就醫，並告知醫師相關症狀、症狀發生時間、疫苗接種時間，以做為診斷參考。

 

 (二)COVID-19疫苗與其他疫苗之接種間隔

１、考量接種實務調整，自本年4月6日起COVID-19疫苗與其他疫苗可同時分開不同手臂接種，亦可間隔任何時間接種，以利接種後反應之判別。 

２、接種前應充分告知民眾，不同疫苗接種後可能產生的不良反應症狀及可能發生與持續的時間，以及接種後應注意事項。

 

(三)對於民眾於國內外接種世界衛生組織(WHO)緊急使用清單 (EUL)之COVID-19疫苗或參與國內製造經緊急授權使用疫苗臨床試驗者，於國內實施追加劑政策（本年1月7日） 前，具有國內外之COVID-19疫苗接種紀錄3劑(含)以上者，最後1劑已超過12週以上，自本年3月25日起同意可經醫師評估後再提供1劑追加劑接種。於國內實施追加劑政策(本年1月7日)後，完成過去國內外接種疫苗紀錄補 登3劑(含)以上者，最後1劑為國內本年1月7日後接種且與前1劑間隔達12週者，則該劑次視為追加劑，不另提供追加劑接種，並應修正全國性預防接種資訊管理系統 (NIIS)之紀錄。
