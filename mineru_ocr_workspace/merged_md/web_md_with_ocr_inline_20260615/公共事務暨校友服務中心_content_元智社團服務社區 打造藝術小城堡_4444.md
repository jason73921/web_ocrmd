# 網站文章資料_元智社團服務社區 打造藝術小城堡_4444

## 一、文章基本資料

- **文章ID**: 4444
- **文章標題**: 元智社團服務社區 打造藝術小城堡
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20200701-1
- **文章簡介**: 元智學生社團長期與社區結合，透過舉辦營隊活動，深耕地方服務，今年首度推出全新計畫，「藝術小城堡」系列活動，透過繪本故事導讀，帶領桃園市特教班的學童與幼兒園的小朋友，將寓言童話融入創意勞作，發揮想像力，探索美學新天地，打造屬於自己的藝術小城堡。

元智大學課外組黃郁蘭組長表示，今年首度推出全新計畫，為桃園華勛國小特教學生與幼兒園小朋友，設計連續6週的「藝術小城堡」美學課程。特別邀請洪明露、徐紫翎等幾位在藝術及幼教與特教美術的專業老師授課，也招募近十位元智學生，從旁輔導小朋友完成作品，發揮青年服務熱忱。

授課老師洪明露，曾榮獲台灣藝術最高殊榮之一的「高雄獎」，他認為運用不同的媒材特性，輸出對物體的印象，可提升幼童認知概念，內化美學知識。另一位授課老師徐紫翎，不但經營藝術工作室，也長期投入幼教與特教美術教學。徐紫翎說，特殊教育需要投入更多愛心與耐心，有些孩子對文字敏感度不夠，卻對藝術情有獨鍾；透過創作抒發孩子內在情緒，也能訓練提升專注力。

活動總召人文社會學院英專班學生詹仁傑說，參與學校深耕服務地方，藉由特教學童美術課程，可體會他們在學習歷程上的努力與辛苦，學生跨域學習和服務地方的社團，不但服務社會同時也提升個人的價值；資管系二年級葉毓庭說，在協助特教學生陶土製作時，小朋友都很會表達自己獨特的想法，完成的作品也讓人驚艷。資管系二年級孫廷翰認為這個活動提供雙向學習管道，一方面讓幼童學習用色彩和形狀，延伸繪本的故事，也從創作過程受到肯定；另一方面，我們也有機會走出校園，協助社區發展，發揮社會責任，大學生活也更有意義。小城堡」內容豐富，九次課程共四大主題，包含「手作陶土」與「黏土捏塑」課程，訓練手眼肌肉協調，建立3D立體空間意象。「蠟筆塗鴉和水彩運用」培養調色基礎概念，同時認識幾何圖像。「畫布彩繪」運用壓克力顏料，在油畫布上感受運筆上色的流動效果，揮灑多樣混色的創意畫作。

華勛國小吳鳳嬌校長特別感謝元智大學，努力深耕地方，善盡社會責任。除了合作多年的「艾妲資訊培力」和「寰宇文化領航」兩項USR計畫，「藝術小城堡」更昇華到心靈層次，提供學童藝術饗宴，增添生活色彩。吳鳳嬌校長規劃於11月華勛30周年校慶和明年親職教育日，與元智合辦「藝術小城堡」特展，集結本系列學童作品，以立體作品框呈現「藝術小城堡」，發表這次豐碩的教學成果。
