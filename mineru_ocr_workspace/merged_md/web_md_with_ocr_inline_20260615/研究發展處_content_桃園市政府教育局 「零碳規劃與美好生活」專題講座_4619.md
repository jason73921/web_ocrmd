# 網站文章資料_桃園市政府教育局 「零碳規劃與美好生活」專題講座_4619

## 一、文章基本資料

- **文章ID**: 4619
- **文章標題**: 桃園市政府教育局 「零碳規劃與美好生活」專題講座
- **文章網址**: https://www.yzu.edu.tw/admin/rd/index.php/tw/?option=com_content&view=article&id=4619
- **文章簡介**: 主旨：函轉南華大學與財團法人人間文教基金會、台泥企業團， 聯合邀請英國社會科學院楊威院士於111年6月14日進行 「零碳規劃與美好生活」專題講座，惠請協助宣傳並鼓勵所屬同仁踴躍報名參加，請查照。

說明：

一、依據教育部111年5月30日臺教資(六)字第1110053750號函 辦理。

二、英國籍楊威院士，學術專業地位崇高，在2011年創建了借 鑒中醫理論的21世紀田園城市空間規劃方法，在國際上引領該理念的研究、普及與實踐，並取得理論和實踐相結合的重要成果，被授予英國社會科學院院士，並擔任2021年英國皇家規劃學會主席。

三、活動時間：111年6月14日(二)上午9時30分至11時30分。

四、活動方式：因應疫情將採線上舉行，報名截止後以Email發送會議網址，將視新冠肺炎疫情，再行決定是否辦理實體課程。

五、報名時間：即日起至6月12日止。

六、報名網址：https://forms.gle/FHgu9Q9V6G2qCQSG7。

七、本講座「環境教育人員認證展延研習」申請中，將可採計環境教育人員認證時數2小時。

八、隨函檢送講座海報(附件)，敬請廣為宣傳。
