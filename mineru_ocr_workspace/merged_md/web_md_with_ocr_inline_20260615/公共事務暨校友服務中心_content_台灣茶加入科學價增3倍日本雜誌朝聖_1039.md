# 網站文章資料_台灣茶加入科學價增3倍日本雜誌朝聖_1039

## 一、文章基本資料

- **文章ID**: 1039
- **文章標題**: 台灣茶加入科學價增3倍日本雜誌朝聖
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【大紀元2016年06月02日訊】

國內學者和業者合作，一改過去台灣+茶烘焙方式，利用科學技術讓好的成分增加，去除澀味，不但提升茶葉價格2至3倍，更使台灣烏龍茶與世界名茶齊名，還有日本雜誌專文報導。

國內產業以中小企業為多，極需研發能量挹注，科技部民國102年起透過產學技術聯盟合作計劃（產學小聯盟），鼓勵學術界研究人員以過去研發成果為主軸，提出協助與服務產業界為目標的計劃。

至104年止，產學小聯盟共核定補助136個產學技術聯盟，產學共同研發436件新技術，開發439件新產品，吸引2500多家廠商加入。並預定6月3日上午10時在國立台北科技大學國際會議廳舉辦成果發表會，展出11個聯盟成果。

其中，中興大學教授曾志正執行的“ 台灣+茶製程技術產學聯盟”，輔導茶農研發的新型茶品“熷烏龍茶 ”，還吸引日本partners雜誌經由市場人士口碑引薦，來台專訪本聯盟會員，並專文介紹“?烏龍茶”為世界上極品好茶。

曾志正指出，烏龍茶的兩大特點就是養生跟好喝，透過實驗發現茶中的“茶飢素”及“體歸靈”，分別有去油膩及抗老化的功效，他認為，廣泛栽種於台灣中部山區的青心烏龍就含有特有的“茶飢素”，是台灣農業具優勢的資源，只要研發技術門檻高的精品茶，就能行銷全世界。

“ 熷烏龍茶 ”是一種烏龍茶經反複烘焙及陳放所轉化出風味完全不同的精製茶品，透過實驗室科學分析，驗證這款茶具有“咖啡因明顯減少、回甘度明顯增加、茶澀度明顯去除、新產生安神功效”等特色，經輔導茶製程後，具特色風味，讓茶農的茶葉每斤價位由新台幣3000元提升至6000元至1萬元不等。在高端茶市場蔚為風潮，要加入聯盟的茶農絡繹不絕。

這個聯盟也輔導茶農研發“生態野放茶”，並取得歐盟、美國、日本以及加拿大有機認證，於英國參展，獲得好評。 


另外，元智大學教授徐業良執行的“老人福祉科技產學聯盟”，聯盟所擁有的7項應用技術，已獲得22項國內外發明專利。其中，聯盟會員的溫感釋壓泡棉市佔率佔全世界第五，透過聯盟的合作平台，將國內所研發的商品拓展至國外市場。

科技部表示，產學小聯盟在服務創新的突破、廠商需求的挖掘、核心技術的多元應用及媒合會員垂直跨業整合等面向，已協助會員廠商將核心技術成功導入產品並帶來實際收益。
