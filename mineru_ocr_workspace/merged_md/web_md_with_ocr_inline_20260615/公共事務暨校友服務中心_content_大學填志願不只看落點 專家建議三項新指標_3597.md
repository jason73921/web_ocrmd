# 網站文章資料_大學填志願不只看落點 專家建議三項新指標_3597

## 一、文章基本資料

- **文章ID**: 3597
- **文章標題**: 大學填志願不只看落點 專家建議三項新指標
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20190304-3
- **文章簡介**: 【青年日報/記者黃朝琴/臺北報導】

學測成績公布，大學甄選3月開跑，考生該如何填志願？升學專家建議，面對國際化的浪潮，除選校、選系的傳統觀念，依分數落點選校系，還可打破排名思維，參考「國際學習」及「跨域學習」，甚至「跨校學位」3項選填志願新指標，尋找屬於自己的第一名。

升學資訊平台「大學問網站」今天召開記者會公布「大學國際學習和跨域學習趨勢調查」，針對全國149所大專校院發出問卷，在國際學習面向，全國起碼有90所大學有國際交換，70所大學有海外實習，90所學校設有雙聯學位，跨域學習面向，全國大學校內跨域有1601門課，校外跨域有2666門課，還有1068門微學分課程。

「大學問」地毯式檢視全臺大學課程，依據全球產業趨勢及跨域、跨校、國際學習的機會，今天也推薦「2019十大值得關注的大學科系」，包括臺科大全校不分系、北教大的兒英教育學系、陽明醫學系B組、元智電機通訊英語學士班、嘉義大學園藝學系、靜宜西班牙語文學系、臺灣海洋大學海洋環境資訊系、東吳巨量資管學院學程、嶺東科大數媒設計系，以及正修科大的土木與空間資訊系；臺清交頂大皆未上榜。（待續）
