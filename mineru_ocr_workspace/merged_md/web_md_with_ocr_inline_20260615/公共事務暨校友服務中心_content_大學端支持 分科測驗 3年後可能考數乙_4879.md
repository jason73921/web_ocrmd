# 網站文章資料_大學端支持 分科測驗 3年後可能考數乙_4879

## 一、文章基本資料

- **文章ID**: 4879
- **文章標題**: 大學端支持 分科測驗 3年後可能考數乙
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20210120
- **文章簡介**: 【人間福報/台北訊】

一○八課綱強調適性發展，高中數學科提早在高二分流，一一一學年起，大學學測考數Ａ和數Ｂ，改為分科測驗的指考仍考數甲但刪除數乙，引發反彈。大學招聯會表示，已邀集大學主管開會，多數與會者支持指考考數乙，未來將由教育部彙整高中端意見，綜合評估才會做成決議。修改考科要於考試前三年公告，最快一一三學年分科測驗有望納數乙，即將到來的一一一、一一二學年仍不考數乙。

大考中心日前公告，一一一學年起指考改為分科測驗，科目由十科減為七科，不再考共同科目國文、英文，並刪除數乙。考科包括數甲、物理、化學、生物、歷史、地理、公民七科。據了解，刪數乙是為減少考科，降低學生升學負擔。消息一出，「消失的數乙」能否重回考場，成全國家長和高中教師關注焦點。

招聯會執行祕書戴念華表示，整體來說大學端支持指考考數乙，主因是不考數乙，大學難鑑定考生數學能力，商管等學群也面臨沒有合適考科可參採的問題。元智大學教務長、管理學院教授曾芳美說，管院的財金、資管系尤其重視數學與邏輯，往年都採國文、英文、數乙三科，改制後失去合適的對應科目。（待續）
