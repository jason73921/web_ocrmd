# 網站文章資料_賴副總統出席資訊應用服務創新決賽 期許產學攜手培育人才_4721

## 一、文章基本資料

- **文章ID**: 4721
- **文章標題**: 賴副總統出席資訊應用服務創新決賽 期許產學攜手培育人才
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20201110
- **文章簡介**: 【經濟日報/陳華焜】
經濟部工業局、教育部資科司、中華民國資訊管理學會共同主辦第25屆大專校院資訊應用服務創新競賽，11月7日舉行決賽，共有346隊次、超過2,000名師生參與決賽。

雖然COVID-19疫情持續，本項競賽已連續辦理8年的亞太交流組也不停歇，透過遠距連線的方式，計有印尼、馬來西亞、新加坡、斯里蘭卡、緬甸、泰國、澳洲等7個亞太地區經濟體學生團隊線上參賽，共同競逐138個獎項205萬元總獎金。

副總統賴清德對於臺灣科技產業發展及科技教育相當重視，特別蒞臨現場為全體師生加油打氣。副總統表示，未來科技是要軟硬兼施，用創新提供服務，而這個競賽名稱即點出未來科技是的由資訊、應用、服務、創新等四大方向來來組成，如此才有可能創台灣第二次經濟奇蹟。台灣正推動「六大核心戰略產業」，其中包含強化資訊及數位產業發展，結合5G時代、數位轉型、及國家安全的資安產業，正好與大專校院資訊應用服務創新競賽近幾年的主軸方向不謀而合，大會邀請指標企業和政府單位設立「新興科技領域」的競賽主題，向全國的學生徵求解答，評審陣容高達5成是業界高階主管，這樣的賽制讓作品更能具體解決問題，為產業尋找人才及創意，是本項活動舉辦成功及價值所在。

經濟部次長林全能表示，工業局自93年參與主辦，便致力將此一競賽建構為產學合作交流平台，而自己也從第17屆起連續6年擔任本競賽的評審委員，每年團隊的創新都令人非常驚豔，也獲益良多，期望透過競賽拔尖的過程，讓企業發掘創意、尋找人才，形成產學雙贏，多年來累計促成產學合作911件，截至108年企業持續採用率高達72%。另外，經濟部也輔導優秀的作品朝商品化、事業化發展，並協助媒合跟創投及大企業合作，已協助團隊取得超過2億元的資金挹注，是非常有意義的成果，比如愛酷智能科技股份有限公司，107年剛創立的時候，即參與經濟部的商品化、事業化輔導，發展人工智慧及數據分析技術，協助企業智慧升級，今年2月已完成Pre-A輪融資，募資金額約為新台幣7,000萬元，目標成為台灣行銷科技第一品牌。不僅如此，競賽已走向國際化，自101年起設立「亞太交流組」，9年來邀請10個經濟體、144個團隊參賽，今年也以跨國視訊方式，與國際保持科技交流。

與時俱進的研究主題也是本項競賽的重要特色，今年全球籠罩在COVID-19疫情中，有不少團隊針對疫情提出解決方案，比如中央大學的作品「急救車智能點班小幫手」，提出以「影像辨識技術」判斷藥品剩餘數量，並以「資料數位化」、「圖表視覺化」的方式進行記錄與管理，協助護理人員提升工作效率，也能有更多心力投入在病患身上。而今年很夯的「數位轉型」議題，也是競賽提案重點之一，比如元智大學及交通大學共同創作的「基於孿生神經網路的工業影像錯件偵測系統」，就是想改善工廠「自動光學檢測」的不足，提出「錯件偵測」的優化方案，採用不同種類的小樣本學習模型、影像優化演算法、文字特徵提取技術，來提升「錯件偵測」的準確度，降低產線損失，進而提高工廠的營收。（待續）
