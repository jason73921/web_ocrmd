# 網站文章資料_薩德危機 中美應尋求合作_1716

## 一、文章基本資料

- **文章ID**: 1716
- **文章標題**: 薩德危機 中美應尋求合作
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【旺報/胡逢瑛】 


東北亞的局勢正在加劇惡化。部署薩德顯然代表美國對朝問題強硬派有可能再次主導朝鮮半島的局勢。現在共和黨的川普執政，當前其外交部署主要特點都是準備與中國進行軍事較量，作為經貿談判籌碼的強硬派當道。倘若對朝鮮半島問題也是啟用強硬派，就代表未來中美在朝鮮半島上的談判，會更加艱巨且不利中國的周邊安全。

美對朝強硬派抬頭 

美國對北韓的強硬派基本主張的特點有3：第一，中國和俄羅斯都需要配合美國加強對北韓進行經濟制裁，認為當前北韓有資金製造導彈表示中俄從中作梗；第二，歐巴馬時期造成北韓試射導彈能力增加，且頻率倍增，必須更加有效敦促中俄積極執行聯合國安理會的經濟制裁決議；第三，六方會談結束就是談判路徑的失敗，導致北韓已經具備打擊美國本土的導彈能力，因此，必須部署薩德系統，以確保美國本土安全。 

當前朴槿惠下台，這個期間又是美國準備部署薩德的期間，這難道又是一場推翻朴槿惠政權作為美國全權主導南韓政權所發動的顏色革命的一環？南韓國內正處於代理總統治國的空窗期，這樣一來，美國堅持部署薩德勢必影響南韓總統大選之後的外交方向。這是否更加意味著親美和親中方向的決戰？

當前中國反韓的民族情緒興起，這是最容易把南韓的民族主義激起而轉向投奔美國懷抱的結果。倘若中國不能同時掌握南北韓對話的主導能力，未來中美摩擦會更加激烈。

俄羅斯對南北韓問題一貫主張政治訴求和外交途徑，當前美國部署薩德的舉措等於不考慮俄羅斯在遠東的利益，同時也不考慮中國在朝鮮半島的利益，這樣顯然不利於中美俄三方共同合作解決區域衝突問題。但是，美國強硬派會自認為保護美國本土安全的利益以及落實美日韓盟友的利益，這樣更加團結美日韓在東北亞的同盟陣線。儘管美國部署薩德違反《美蘇限制中導條約》之規定，但是，俄羅斯當前不會願意再開啟新的美俄戰場，更何況融合南北韓是符合俄羅斯的利益。

減少民族主義對立 

現在中國要如何與美國進行談判？王毅外長提出的「停止核導活動」和「停止美韓大規模軍演」的「雙暫停」思路僅是拋磚引玉，不過目前中國尚未證明足以同時說服兩照雙方遵從這一訴求的能力。

當前中國首先應該避免把南韓推向美國，親美派往前衝只是作為激怒中國的手段。其次，積極在美俄韓三方進行斡旋，找出共同利益的基礎，證明中國具有協調能力的意願和行動。再者，北韓彰顯軍力代表希望各方承認其大國地位，中國如能使各方回到談判軌道，讓美國確認不會有立即受攻擊的危險，這才能說服美國暫緩部署薩德系統。

中國倘若要積極部署反薩德系統當然也是回應的一種方式，不過那會增加軍備競賽的對立氣氛，也會失去斡旋的主導地位，使中國陷入新一輪冷戰當中。那麼，川普一旦啟用對朝強硬派處理薩德和對朝經濟決議的執行力度，中國勢必成為攻擊對象，等於宣告重啟談判無望。

美朝核框架協議顯示美國曾經想主導朝鮮半島，但是身陷阿富汗和伊拉克戰爭泥潭中的美國無法獨自擔保朝鮮半島安全問題，六方會談顯示中國有了主導協調的地位，南韓有了影響白宮決策的強硬聲音。

然而，歐巴馬的「亞太再平衡」政策對中國進行圍堵意圖甚明，使中國無法繼續在朝鮮無核化問題上與美國積極合作。

當前川普上任，對中俄也都是挑戰和契機。美國鷹派抬頭對中國的挑戰在於，美國將更加強硬處理亞太事務，中國任何的強硬舉措都會給川普新政局增加難以緩和的障礙。持續積極對川普當局進行開誠布公的高層溝通仍是最直接有效的路徑。 （作者為元智大學助理教授）
