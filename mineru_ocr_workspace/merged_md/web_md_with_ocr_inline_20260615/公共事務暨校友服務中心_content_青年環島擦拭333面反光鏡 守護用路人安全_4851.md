# 網站文章資料_青年環島擦拭333面反光鏡 守護用路人安全_4851

## 一、文章基本資料

- **文章ID**: 4851
- **文章標題**: 青年環島擦拭333面反光鏡 守護用路人安全
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20210106
- **文章簡介**: 【中央社 記者許秩維台北7日電】

在「反光鏡菩薩」張秀雄的啟發下，宋侑芩和余依軒兩名大學生組團，環島3週走遍台灣19個縣市，擦拭333面反光鏡，用行動守護眾多用路人的安全。

教育部青年發展署推動「青年壯遊台灣─尋找感動地圖實踐計畫」，鼓勵青年自行規劃多元面向的壯遊企劃，開拓視野、培養自主及團隊合作能力，同時探索台灣在地文化特色，入選團隊還可獲得實踐獎金。

元智大學學生宋侑芩和台灣師範大學學生余依軒組成「咪露de澎」團隊，受到長年擦拭反光鏡被稱為「反光鏡菩薩」的張秀雄啟發，兩人決定加入守護台灣用路人安全的行列，帶著長柄刷，開啟騎機車擦拭反光鏡的環島之旅，並獲得感動地圖實踐計畫成果競賽最大獎「感動五燈獎」。

宋侑芩和余依軒從桃園出發，沿路擦拭反光鏡，常遇到居民好奇觀看，還有人誤以為這是新職業，途中曾遭遇狂風暴雨、颱風天、機車故障、手機泡水等狀況，要前往蘭嶼時，也因天氣不佳而導致航班取消，甚至還出了一場車禍，雖然兩人飽受驚嚇，卻沒有阻止她們繼續的決心。（待續）
