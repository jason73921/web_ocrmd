# 網站文章資料_104學年度轉學考招生名額異動公告8_138

## 一、文章基本資料

- **文章ID**: 138
- **文章標題**: 104學年度轉學考招生名額異動公告8
- **文章網址**: https://www.yzu.edu.tw/admissions/index.php/tw/104-8
- **文章簡介**: 
- **文章全文**: 說明：104學年度轉學考招生名額依最新學生在學狀況統計後，除以下三學系異動外，其餘維持4月份招生簡章公告之名額，此次統計為最後確定之招生名額。



1. 藝術與設計學系２年級：招生名額增加1名，共3名。

2. 電機工程學系３年級：招生名額增加1名，共6名。

3. 資訊傳播學系設計組２年級：退伍軍人外加名額原0名，更改為1名。

4. 資訊傳播學系科技組２年級：退伍軍人外加名額原1名，更改為0名。
