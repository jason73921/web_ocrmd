# 網站文章資料_2019警衛隊將清理廢棄無人之自行車，有主車輛請繫上紅色布條_177

## 一、文章基本資料

- **文章ID**: 177
- **文章標題**: 2019警衛隊將清理廢棄無人之自行車，有主車輛請繫上紅色布條
- **文章網址**: https://www.yzu.edu.tw/admin/ga/index.php/tw/2019
- **文章簡介**: 108/12/16前，請將您的自行車綁上紅色布條，俾利區分使用中與廢棄、違規未領之車輛。

逾期未繫紅色布條之自行車，將視為無主之廢棄車，移至學校大門右側鳳凰樹下集中處理。

紅色布條請就近至警衛室、男女宿舍櫃台或總務處領取使用。謝謝您！

一、違規遭鎖車之自行車車主請於上班時間至會計室繳交違規處理費，憑發票至警衛室請警衛開鎖以取回您的愛車，
 並同樣需於12月16日前綁上紅色布條。

二、12月16日起，校內各館廢舊未綁紅色布條之自行車將先集中放置於各館集中處（如下），並陸續移至學校大門右側鳳凰樹下予以集中。

大門：集中放於入學校大門右側鳳凰樹下。
 一館：集中放於後(東北側)方草坪(靠六館)。
 二館：集中放於北(北側門)方(靠圖書館旁樟樹下) 。
 三館：集中放於東大門右側(東南方樹下草坪)及西側門右側(西北方樹下)。
 六館：集中放於自行車停車場右側(東北方樹下草坪)。

宿舍區：集中放於活動中心前靠體育館右側圍牆旁及自行車停車場旁。
 體育館：集中放於自行車停車場南側門水泥柱旁。

三、集中處理後之未綁紅色布條自行車輛將於109年1月10日起依廢棄物集中處理，所有人不得異議。

總務處 警衛隊108.11.19
