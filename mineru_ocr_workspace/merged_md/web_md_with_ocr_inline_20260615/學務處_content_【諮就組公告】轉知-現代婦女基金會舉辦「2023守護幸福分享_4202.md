# 網站文章資料_【諮就組公告】轉知-現代婦女基金會舉辦「2023守護幸福分享愛」募款活動_4202

## 一、文章基本資料

- **文章ID**: 4202
- **文章標題**: 【諮就組公告】轉知-現代婦女基金會舉辦「2023守護幸福分享愛」募款活動
- **文章網址**: 
- **文章簡介**: 一、現代婦女基金會已成立36年，每年協助約7千多個受暴家庭，在台灣，家暴通報件數已經超過15萬件，平均每3分鐘就有一件家暴通報發生。為募集服務經費，特於每年年底期間辦理「守護幸福分享愛」活動，即日起至11月17日前，捐款800元致贈阿默經典乳酪蛋糕。
二、敬邀貴校共同發起團體捐款支持行動。
三、本項募款所得將運用於「受暴家庭扶助計畫」，支持本會每年需要被協助的受暴家庭與孩童，提供安全協助、危難救助、身心療癒、關係輔導等服務，並提供諮商師等資源幫助目睹兒少創傷復原，穩定學習。
四、歡迎透過本會「守護幸福分享愛」活動網頁線上捐款 https://reurl.cc/8N59nj，或傳真回覆紙本捐款單捐款(詳附件)。本活動業經主管機關衛生福利部核可在案，勸募字號：衛部救字第1121361231號。
五、本案承辦人：現代婦女基金會資源發展組劉斐平小姐，連絡電話：(02)2391-7133#305，傳真：(02)2391-7129，Email：fifiu@38.org.tw
