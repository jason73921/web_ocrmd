# 網站文章資料_107學年度博士班備取生報到注意事項_391

## 一、文章基本資料

- **文章ID**: 391
- **文章標題**: 107學年度博士班備取生報到注意事項
- **文章網址**: https://www.yzu.edu.tw/admissions/index.php/tw/107-4
- **文章簡介**: 說明：

一、 各系所備取生遞補名單將自107.06.04起，於每週一上午10時在本校招生網站公告（http://www.yzu.edu.tw/admissions/最新消息），開學日(107.09.10)前兩週，改為每週公告兩次，請備取生務必主動上網查詢，另亦會以e-mail通知遞補上之備取生。

二、 下載：備取生報到注意事項及委託書，報到專區網址：https://exam.yzu.edu.tw/NewNetapply（點選「107博士班」→查詢報名結果與成績→輸入身分證字號及密碼→查詢成績→報到專區）查詢。

三、 各系所錄取生新生注意事項，請自行參考各系網頁公告。
