# 網站文章資料_中華民國工業區廠商聯合總會「109年度產業園區產業輔導服務人員跨域加值精鍊」_3176

## 一、文章基本資料

- **文章ID**: 3176
- **文章標題**: 中華民國工業區廠商聯合總會「109年度產業園區產業輔導服務人員跨域加值精鍊」
- **文章網址**: https://www.yzu.edu.tw/admin/rd/index.php/tw/?option=com_content&view=article&id=3176
- **文章簡介**: 主旨:

本會訂於新北市產業園區服務中心(北區班)及高雄市左營區蓮潭國際會館(南區班)舉辦「109年度產業園區產業輔導服務人員跨域加值精鍊」課程，請協助廣宣並踴躍報名參加。

說明:

一、時間:

北區班:109年7月31日、8月7日、8月14日(五)。

南區班:109年8月21日、8月28日、9月4日(五)。

二、地點:

北區班: 新北市產業園區2樓會議室(新北市新莊區五工路95號)。

南區班: 蓮潭國際會館1樓R108會議室(高雄市左營區崇德路801號)。

三、本次跨域加值精鍊課程特邀產、學、研具輔導實績及經驗之專家擔任講師，針對現行及有志於產業園區服務及輔導工作之大專院校輔導團隊、產業圈服務中心及協進會幹部以及其他產、學、研工作團隊，提供產學合作、政府計畫及專案管理之跨域整合知識。本課程舉辦目的在促進產業園區升級轉型，增進服務人員跨域整合知識及技能。

四，本會活動免費參加，招訓簡章及報名表如附件。請有幸區報名者將報名表於7月17日(五)前傳真04-235585998或e-mail:ipmf@hibox.hinet.net

或直接前往中華民國工業區廠商聯合總會官網:http://www.muga.org.tw/booking.php報名；王先生連絡電話:04-23585158分機33
