# 網站文章資料_元智輔助平臺　助指考生選填志願_111

## 一、文章基本資料

- **文章ID**: 111
- **文章標題**: 元智輔助平臺　助指考生選填志願
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【教育廣播電台/徐詠絮】 


104年大學指考成績單明天（17）寄發，元智大學推出「科技輔助選填志願互動平臺」，整合各校系基本資料、就業、社群網路討論及就讀感想等資訊，提供指考考生選填志願參考。元智大學表示，這次是利用大數據分析技術，建構「科技輔助選填志願互動平臺」(http://unipop.yzu.edu.tw/)，平臺也提供落點分析服務，只要輸入成績就可點選有興趣的落點校系，直接進入PK功能，為幫助考生更瞭解平臺使用，元智大學19號、20號兩天在校內五館地下室5002教室，舉辨說明會，除了介紹平臺系統，也提供志願選填諮詢服務。
