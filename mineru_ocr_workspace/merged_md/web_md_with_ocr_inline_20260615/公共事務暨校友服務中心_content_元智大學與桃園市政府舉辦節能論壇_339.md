# 網站文章資料_元智大學與桃園市政府舉辦節能論壇_339

## 一、文章基本資料

- **文章ID**: 339
- **文章標題**: 元智大學與桃園市政府舉辦節能論壇
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【中央社】

為擴大市內廠商節能活動之參與，元智大學10/30假桃園市政府舉辦『104年度桃園市綠能產業技術服務團-節能技術實務經驗分享暨課程講座』，並邀請計畫執行專家-紀培鴻博士與黃有智博士分別講授工廠節能概論與節能減碳實務經驗分享。 


桃園市政府為推動市內產業朝三低一高(低汙染、低耗能、低用水及高附加價值)發展，發展更智慧化、效率化、科技化的產業，奠定永續發展基礎，攜手元智大學邀請產學研機構之專家學者於今年中(2015年六月)成立「桃園市綠能產業技術服務團隊」，協助市內廠商提升綠能關鍵技術、建立自主化節能管理能力、與節(水)能(或碳足跡)標章申請。希望藉由綠(節)能產業的發展，可以減少對能源的依賴、降低二氧化碳排放、並進而提升產品競爭力與企業形象。 


歷經四個多月的計畫執行，元智大學已進行十餘家廠商的輔導案例並累積相當之實務經驗。以接受輔導業者-弘馳股份有限公司(桃園觀音鄉)為例，服務團團長元智大學機械系教授鐘國濱表示，該公司營業項目為廢棄物電子產品中提煉貴重金屬，傳統提煉過程中需先經高溫處理，接著歷經灑水急速冷卻降溫過程，此先後動作分別意謂高耗能與高經濟價值高熱蒸氣浪費。因此根據本服務團之建議報告，該公司可於提煉製程末端加裝小型蒸氣發電機藉以回收高熱蒸氣發電，再將此電力用於前端高溫處理所需的耗電，如此可降低公司可觀之電力支出，並可大幅減少二氧化碳排放並有利於後續碳足跡標章之申請通過。 


此論壇更力邀工業技術研究院之林榮堂顧問講授熱能設備節能技術課程。由於各領域業界報名參與論壇踴躍，藉由本論壇之舉辦，預期未來將為桃園市的綠(節)能發展帶來可觀之效益。
