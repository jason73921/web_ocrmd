# 網站文章資料_中國醫藥大學 「2021永續智慧創新黑客松」_4112

## 一、文章基本資料

- **文章ID**: 4112
- **文章標題**: 中國醫藥大學 「2021永續智慧創新黑客松」
- **文章網址**: https://www.yzu.edu.tw/admin/rd/index.php/tw/?option=com_content&view=article&id=4112
- **文章簡介**: 主旨：中國醫藥大學辦理「2021永續智慧創新黑客松」競賽，敬請學生踴躍報名參加。

說明：

一、本校攜手逢甲大學、亞洲大學、靜宜大學、東海大學、大葉大學及嶺東科技大學7校、14個學院及中心共同舉辦 「2021永續智慧創新黑客松競賽」活動，將於2021年12月11日線上舉行，總獎金高達55萬，鼓勵跨校、跨領域與新 創學生團隊一同來挑戰。

二、本競賽將透過18間企業與NGO的真實命題，落實以永續價值 為驅動力的創新發展，優化未來產業之永續經營與服務創 新，為後疫情時代的智慧生活，開啟無限創意與想像。

三、主要議題與參與命題廠商包括：

(一)銀髮智慧照護、居家樂活、未來醫院議題：中醫大附設 醫院、上銀科技、大毅建設、嘉新水泥、台達電子、由 田新技、福倫藥局。

(二)永續與創新行銷：家樂福、信義房屋、樺懋精品磁磚、 偉孚興業、維豐橡膠、維豐文化藝術基金會。

(三)智慧商業與管理議題：義隆電子、康淳科技、Nu Pasta。

(四)偏鄉教育創新議題：博幼基金會、歐旻慈善基金會。

四、本次黑客松競賽將於10月1日至30日完成線上報名徵件，11月19日繳交提案計畫書，12月11日進行線上決賽。

五、大會網站：https://sustainabilityhackathon.fcu.edu. tw/。
