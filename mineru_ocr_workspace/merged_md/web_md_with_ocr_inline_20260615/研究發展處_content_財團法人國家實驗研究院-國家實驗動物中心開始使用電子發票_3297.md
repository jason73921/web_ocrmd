# 網站文章資料_財團法人國家實驗研究院-國家實驗動物中心開始使用電子發票_3297

## 一、文章基本資料

- **文章ID**: 3297
- **文章標題**: 財團法人國家實驗研究院-國家實驗動物中心開始使用電子發票
- **文章網址**: 
- **文章簡介**: 主旨：本院國家實驗動物中心（以下簡稱本院動物中心）各項銷售服務所開立發票自109年10月1日起使用電子發票，請查照。

說明：

一，本院動物中心配合政府電子發票政策，自109年10月1日起使用電子發票，期能加速發票送達時間，減低成本及落實節能減碳愛地球的目標。

二，電子發票施行初期109年10月1至11月30日止，本院動物中心將採雙軌服務，於開立發票後除提供貴單位自行連結下載及列印外，同步提供紙本列印及郵寄之服務，確保貴單位能如期收到發票。

三，自109年12月1日起，本院動物中心所開立之電子發票，皆由系統抓取貴單位會員資料之E-mail信箱進行郵寄，請貴單位自行連結，下載及列印，本中心不再提供紙本郵寄服務。

四，請貴單位務必隨時維護會員資料E-mail之正確性，以確保如期收到發票，維護貴我雙方權益。

五，貴單位如發現未收到電子發票之情形，請參照下列查詢作法：

（一）請至本院動物中心對外服務平台（https://service.nlac.narl.org.tw）：會員登入一會員資料→帳務帳單查詢→以帳單編號查詢→列印發票。

（二）或至財政部電子發票整合服務平台（https://www.einvoice.nat.gov.tw/）：登入帳號密號查詢/下載/列印發票

（三）請再次確認會員資料E-mail之正確性，如有異動請隨時更新。

六，如有任何問題，請洽服務窗口尤梅珠小姐，聯絡電話02-8753-7252。
