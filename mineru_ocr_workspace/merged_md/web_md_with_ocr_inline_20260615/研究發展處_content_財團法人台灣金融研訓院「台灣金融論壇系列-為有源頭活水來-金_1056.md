# 網站文章資料_財團法人台灣金融研訓院「台灣金融論壇系列-為有源頭活水來-金融挺創新創業研討會」_1056

## 一、文章基本資料

- **文章ID**: 1056
- **文章標題**: 財團法人台灣金融研訓院「台灣金融論壇系列-為有源頭活水來-金融挺創新創業研討會」
- **文章網址**: https://www.yzu.edu.tw/admin/rd/index.php/tw/?option=com_content&view=article&id=1056
- **文章簡介**: 主旨：謹訂於106年6月20日(星期二)辦理台灣金融論壇系列-「為有源頭活水來-金融挺創新創業」研討會，敬邀 貴單位相關人員蒞臨參與，敬請 查照。

說明：

一、政府正積極推動「亞洲‧矽谷」等五大創新產業，做為驅動臺灣下世代產業成長的核心，同時戮力健全創新創業生態系，鼓勵創新創業，輔以連結國際、連結未來及連結在地，促進產業升級，期望能以創新創業驅動經濟成長。在政府力促產業創新升級的宏大藍圖中，新創事業自草創之初至發展成熟，各階段皆須有充裕且穩定的資金供給，有鑑於此，本院特舉辦本次研討會，邀請專家分享專業見解與建言，探討面向涵蓋完善創新創業生態系統之國家戰略、活絡創新創業投資環境之機制設計、協助創新創業的資金配套與創投經驗等。冀能匯聚各方意見，透視潛在問題並尋求解決方向，以供主管機關與業者參考。檢附研討會實施要點如附件一。

二、請於106年6月16日前，至台灣金融研訓院本活動網頁線上報名(網址http://service.tabf.org.tw/tw/User/9140096/)，或依式填具參加人員報名表(格式如附件二)，以傳真(02-2363-9111)或電子郵件(regosd@tabf.org.tw)方式傳送本院客戶服務中心彙辦。

三、如有任何疑問或洽詢事項，請賜電本院客戶服務中心（電話：02-3365-3615)辨理。
