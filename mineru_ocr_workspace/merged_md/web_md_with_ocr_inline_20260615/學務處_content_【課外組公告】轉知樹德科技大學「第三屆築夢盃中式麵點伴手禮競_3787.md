# 網站文章資料_【課外組公告】轉知樹德科技大學「第三屆築夢盃中式麵點伴手禮競賽」，歡迎本校學生踴躍參加~_3787

## 一、文章基本資料

- **文章ID**: 3787
- **文章標題**: 【課外組公告】轉知樹德科技大學「第三屆築夢盃中式麵點伴手禮競賽」，歡迎本校學生踴躍參加~
- **文章網址**: https://www.yzu.edu.tw/admin/st/index.php/tw/?option=com_content&view=article&id=3787
- **文章簡介**: 一、 競賽主旨：

永續築夢造福健康為傳承中式麵食文化技藝及促進伴手禮產業創新、創意及創造之能力提升，「樹德科技大學」與「美國小麥協會」及「奇美食品股份有限公司」、「統清股份有限公司」聯合舉辦「2023 第三屆築夢盃中式麵點伴手禮競賽」，邀請全國對中式麵點有熱情的業者、社會人士、各大專院校及高中職學生等技藝高手，以永續食農、健康飲食為發想，同場交流競技傳承華人麵食技藝。
二、 本競賽邀請全國對中式麵點有熱情的業者、社會人士、各大專院校及高中職學生等技藝高手，以永續食農、健康飲食為發想，同場交流競技傳成華人麵食技藝。
三、 參賽對象：

1.高中(職)學生組：高中(職)對烘焙、餐飲、食品相關學系之在學生。

大專社會組：年滿18歲以上，對烘焙、餐飲、食品相關學系之大專在學生，與對中式麵食有興趣之社會人士、糕餅業者及師傅等從業人員。

2.組隊條件：

高中(職)學生組：每隊2人為一組參加，依報名方式審核通過初審。

大專職業社會組：每隊2人為一組參加，依報名方式審核通過初審。
四、報名日期：即日起至112年7月14日（星期五）。

        決賽名單公布：112年7月21日（星期五）。

        決賽日期：112年8月12日（星期六）。
