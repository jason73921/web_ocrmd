# 網站文章資料_元智大學榮獲第六屆國家企業環保獎「銅級獎」_328

## 一、文章基本資料

- **文章ID**: 328
- **文章標題**: 元智大學榮獲第六屆國家企業環保獎「銅級獎」
- **文章網址**: https://www.yzu.edu.tw/index.php/tw/20240927-1
- **文章簡介**: 元智大學以「以創新教學及重點研究為特色，擁有創新、多元、智慧及人文的永續發展校園，且具國際聲望之優質大學」為校務願景，首次申請國家企業環保獎即獲銅獎肯定。頒獎典禮9月6日在中油大樓國光會議廳舉行，由環境部彭啓明部長親自頒發獎座，由永續發展暨社會責任中心王瑄主任代表領獎，當天下午也到總統府接受蕭美琴副總統接見。

 

    元智大學致力於推動綠色校園及永續發展，積極響應政府政策，增加綠電發電量，出租屋頂提供廠商設置屋頂型太陽能發電系統，亦自建太陽能光電系統，減少二氧化碳排放，達到節能減碳的目標。

「永續發展校園」一直是元智大學整體校務發展願景中重要的一環，2009年起即建置「校園電力監控及節能管理系統」，同年通過ISO 14064-1溫室氣體盤查驗證、2011年成為全球第一所通過ISO 50001能源管理系統驗證大學，2022年持續推動通過新版ISO 50001:2018國際驗證及ISO 14064-1:2018溫室氣體盤查驗證，持續投入節能，積極發展永續校園；配合政府「台灣永續發展核心目標」之政策，依據近、中、遠三階段，逐步由低碳邁向校園淨零碳排目標。

 

    關於教室部分，利用空調/照明配合課表自動化管理，實施「有課取電，沒課斷電」的措施、宿舍空調及供社團借用場域，採用插卡取電的方式、推行步行運動。另外，為提升飲用水品質及節能，汰換飲水機為具環保及節能標章之機型等，使教職員對節能減碳之推動有感。

    此外，除專責組織推動節能計畫及措施，亦重視日常生活環境教育。於宿舍宣導及執行節電節水措施，各單位推行節能環保/淨零綠生活活動，如淨灘活動、專題演講、雲端電影院、二手書交流、認識綠色存股、棲地維護等活動，持續將永續理念傳達予教職員，提升教職員生永續意識。

    元智大學積極投入節能、減塑行動，透過各項能源、環境認證，確保執行有效，期間榮獲各項獎項的肯定。未來，將繼續致力於環保作為，並不斷探索創新方法來保護地球家園，與更多機構和組織攜手合作，共同實現環境永續。
