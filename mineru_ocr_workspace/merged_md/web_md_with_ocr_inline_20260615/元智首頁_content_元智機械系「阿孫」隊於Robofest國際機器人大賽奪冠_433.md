# 網站文章資料_元智機械系「阿孫」隊於Robofest國際機器人大賽奪冠_433

## 一、文章基本資料

- **文章ID**: 433
- **文章標題**: 元智機械系「阿孫」隊於Robofest國際機器人大賽奪冠
- **文章網址**: https://www.yzu.edu.tw/index.php/tw/20260410-3
- **文章簡介**: 由元智大學機械工程學系陳松郁老師指導的「阿孫」隊，在第27屆Robofest國際機器人大賽台灣選拔賽中表現亮眼，勇奪第一名，展現台灣青年於智慧健康科技領域的創新能量。

 

「阿孫」隊由李奕辰、黃詠緹、魏宇均及余若琪組成。團隊以結合「跌倒偵測」與「趣味復健」的機器人作品脫穎而出，突破傳統被動監測模式，透過互動式訓練引導長者進行身心活動，不僅提升安全性，也有助於改善獨居長者的生活品質，在長照人力不足的現況下提供創新解方。

 

指導老師陳松郁表示，此次獲獎是機械、設計與系統整合的成果，學生自賽前即投入機構設計與系統開發，歷經長時間反覆測試與調整，展現高度專注與執行力，也感謝系上完整課程與資源支持，讓學生得以將理論轉化為實作成果。

 

團隊成員李奕辰表示，從初期機器人無法啟動、輪子故障，到逐步排查供電與馬達驅動問題，最終完成AI跟隨與避障功能，讓作品成功運作，過程雖艱辛但收穫豐富，也體會到面對瓶頸時回歸基礎的重要性；黃詠緹指出，團隊將課堂所學的機械設計與控制理論實際應用於作品中，不僅提升專業能力，也培養了解決問題與團隊合作的經驗。

 

余若琪則提到，開發過程面臨馬達扭力不足、結構調整及系統供電不穩等問題，甚至導致設備頻繁當機，但團隊彼此支持、即時協助，是克服困難的關鍵；魏宇均表示，在撰寫跌倒偵測程式過程中，歷經多次除錯與設備異常，團隊始終互相扶持，共同找出問題並解決，最終讓系統穩定運作並成功發出警示，展現高度默契與合作精神。

 

機械系主任江右君表示，元智大學機械工程學系長期重視實作教學，提供經費與創客空間，鼓勵學生參與校外競賽，培養跨領域整合能力。「阿孫」隊的優異表現，不僅展現學校教學成果，也突顯台灣青年在智慧醫療與機器人領域的發展潛力，未來可望持續在國際舞台發光發熱。
