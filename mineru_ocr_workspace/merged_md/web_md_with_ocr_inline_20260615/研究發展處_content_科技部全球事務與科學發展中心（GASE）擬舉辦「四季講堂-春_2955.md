# 網站文章資料_科技部全球事務與科學發展中心（GASE）擬舉辦「四季講堂-春季場」講座_2955

## 一、文章基本資料

- **文章ID**: 2955
- **文章標題**: 科技部全球事務與科學發展中心（GASE）擬舉辦「四季講堂-春季場」講座
- **文章網址**: https://www.yzu.edu.tw/admin/rd/index.php/tw/?option=com_content&view=article&id=2955
- **文章簡介**: 主旨：科技部全球事務與科學發展中心（GASE）擬舉辦「四季講堂-春季場」講座，敬請惠予協助公告，請查照。

說明：

一、為提升國內青年學子於科研領域上的國際視野，並為其搭建與國際上具學術影響力專家之間的橋樑，旨揭中心特舉辦《四季講堂-春季場》講座，邀請西元1995年諾貝爾生理學或醫學獎得主克里斯汀·紐斯林-沃爾哈德（Christiane Nüsslein-Volhard）來臺演講，與有志於追求科學道路的年輕學子分享諾貝爾獎背後的故事。

二、《四季講堂-春季場》資訊如下：

（一）時間：109年2月6日（星期四）15時30分至17時。

（二）地點：本校博雅教學館101教室。

（三）參與對象：全國大專院校及高中職教師、職員、學生，以及對本次講座主題有興趣之一般民眾。

（四）報名方式：即日起至109年2月5日受理線上報名。線上報名網址(https://reurl.cc/Rd3GpG)，詳細資訊請參考該中心網站(網址：https://gase.most.ntu.edu.tw /events）。

三、當天活動將同時辦理《GASE科學眼計畫》，期望青年學子於當天接受國際大師面對面啟發後，透過文字呈現其對於相關領域的反思，並藉由此計畫激發其與同儕投入科學研發領域之興趣及動機。

四、本案聯絡人：洪小姐，電話：02-33662121#24。
