# 網站文章資料_【宿服組公告】暑假期間熱水注意事項_6089

## 一、文章基本資料

- **文章ID**: 6089
- **文章標題**: 【宿服組公告】暑假期間熱水注意事項
- **文章網址**: 
- **文章簡介**: 公告日期：114年6月19日

 

1.宿舍於暑假期間6/29(日)-8/29(五)19:00-21:00提供熱水（時間若調整另行公佈），請同學盡量在這段時間盥洗完畢，若無法在該時段完成，其他時段請至下列各點使用電熱水器，但請安靜勿喧嘩，尊重是一種美德，感謝您。

　宿舍熱水器位置

　　男一宿1樓B區：3間淋浴間

　　男一宿2至7樓B區：每樓層各1間浴室

　　女一宿1樓：2105房旁5間淋浴間

　　女一宿2至9樓南區：每樓層各2間浴室

 

2.熱水器連續使用下，需要時間重新製熱，同學使用前可先查看熱水器上的溫度表，評估自己可接受的溫度後再使用，避免水溫不足影響品質。

3.熱水器儲水容量有限，若一次性用水量過大，將影響後續人員之使用，請節約用水，謹慎使用。

4.同學若遇熱水器故障，請盡速向宿服組反應或是上網填報修喔！

 

 

學生事務處宿舍服務組
