# 網站文章資料_105學年度轉學考招生名額異動公告_238

## 一、文章基本資料

- **文章ID**: 238
- **文章標題**: 105學年度轉學考招生名額異動公告
- **文章網址**: https://www.yzu.edu.tw/admissions/index.php/tw/105-5
- **文章簡介**: 說明：105學年度轉學考招生名額依最新學生在學狀況統計後，除以下五個學系異動外，其餘維持4月份招生簡章公告之名額，此次統計為最後確定之招生名額。

1. 光電工程學系２年級：招生名額增加1名，共6名。

2. 通訊工程學系２年級：招生名額增加1名，共3名。

3. 管理學院學士班２年級：招生名額增加1名，共15名。

4. 應用外語學系３年級：招生名額增加1名，共2名。

5.藝術與設計學系３年級：招生名額增加1名，共5名。
