# 網站文章資料_學雜費漲幅0.53%　大學怨調幅太低、申請憂社會觀感_6661

## 一、文章基本資料

- **文章ID**: 6661
- **文章標題**: 學雜費漲幅0.53%　大學怨調幅太低、申請憂社會觀感
- **文章網址**: 
- **文章簡介**: 【nownews/記者李琦瑋／台北報導】

教育部今（11）日表示，112學年度大學學雜費基本調幅上限為0.53%，學校若欲調整學雜費，應依照相關程序辦理，最遲於6月5日提出申請案報教育部審議。部分大學抱怨，調幅太低、跟不上通膨，一學期才漲1、200塊，但申請程序繁複，又要面臨社會觀感不佳的風險，多傾向不申請。

教育部表示，於3月31日發文給各大專校院，依「專科以上學校學雜費收取辦法」規定，學雜費基本調幅參酌學校教學成本及受教者負擔能力，並依行政院主計總處公布的消費者物價指數年增率、平均每戶可支配所得年增率、受僱員工薪資年增率進行核算，經過計算後，112學年度學雜費基本調幅上限為0.53%。

欲調整學雜費的學校，應依「專科以上學校學雜費收取辦法」規定，完成校內相關資訊及研議公開程序後，於公告期限內提出申請，經教育部核定者，才可辦理學雜費調整，112學年度申請大學學雜費調整案最遲於6月5日報教育部。

元智大學主秘吳和生也說，0.53%調幅太低，建議配合通貨膨脹，調高幅度，若以元智大學一學期學雜費5萬5上下來算，一學期可調漲291元，學校會討論要不要申請，但看教育部政策，雖說讓大學申請，但據以往經驗，很多都不會過。

他坦言，申請程序複雜，即使通過，調漲幅度過低沒效益，但更大的機率是教育部面對外界壓力，根本不會讓學校調漲，申請學校還得面臨社會觀感不佳、輿論撻伐，其實是吃力不討好的事情。（待續）
