# 網站文章資料_遠東商銀獲頒財富管理新星獎_3240

## 一、文章基本資料

- **文章ID**: 3240
- **文章標題**: 遠東商銀獲頒財富管理新星獎
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20181016
- **文章簡介**: 【工商/呂清郎】

遠東商銀在國際知名財資雜誌（The Asset）舉辦財富管理與私人銀行評鑑中，獲「台灣區最佳財富管理銀行新星獎」，頒獎典禮在香港舉行。

財資雜誌採取全方位的評核標準，針對財務績效、行銷策略、特色分行、集團綜效及品牌形象等五大構面進行評鑑，開放全亞洲地區各國銀行參與角逐。由十個質化及量化的指標來衡量銀行的各項表現，通過初選的銀行需再經過會議及簡報等評核，評審團由香港來台，實地參觀遠東商銀分行，肯定遠東商銀在科技化、高齡化及全球化的大時代趨勢下，能有創新的表現，具內涵的樂活理財新價值。

遠東商銀整合遠東集團多角化資源，提供財富管理客戶多方面的生活服務與理財諮詢，包括每年與遠東飯店聯手籌畫VIP客戶米其林饗宴、與元智大學合作開辦「樂齡理財學苑」、與亞東醫院合辦VIP客戶健檢等，成為客戶追求樂活的夥伴。此外，亦運用新型態數位社群工具與客戶連結，推出全台首創分行財富管理LINE@，提供客戶生活圈在地商家優惠資訊，創造客戶、商圈店家與銀行的三贏新局面。（待續）
