# 2020全國三分鐘生科論文口說競賽

# 資格、規則與獎勵

## 一、參賽資格

1.生命科學相關系所在校大學部學生、碩、博士班研究生

2.生命科學相關系所主聘教師聘僱仍在職之專、兼任研究助理及博士後研究員

3.符合以上之一資格且具有個人研究成果者（論文內容以個人研究為限）

4.參加過本活動並得獎者，不得再參加

## 二、分區範圍

北區：北北基桃

中區：竹苗中彰投及金門

南區：雲嘉南高屏及澎湖

東區：宜花東

## 三、參賽規則

1. 參賽順序由主辦方抽號決定。

2. 從踏入比賽指定框內，即開始計時，僅有三分鐘，超過者一律以失格論。

3. 僅可口頭報告，不得展示任何器材輔具。

4. 僅可顯示單張靜態投影片(不得以其他任何視覺及音效器材輔助表現)。

5. 中文為指定語言。

6. 比序同分時由評審議決，依評審宣布之結果為最終決定。

## 四、獎勵方式

1. 各區預賽獎金:由各區決定

2. 全國決賽獎金:

第一名，獎金3萬元整

第二名，獎金2萬元整

第三名，獎金1萬元整

優選三名各獎金8000 元整

網路票選人氣王一名獎金 5000 元整
