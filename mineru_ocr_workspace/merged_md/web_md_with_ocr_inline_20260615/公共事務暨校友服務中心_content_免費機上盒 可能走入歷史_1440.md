# 網站文章資料_免費機上盒 可能走入歷史_1440

## 一、文章基本資料

- **文章ID**: 1440
- **文章標題**: 免費機上盒 可能走入歷史
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【中國時報/記者李盛雯】

你家還有在看有線電視嗎？如果有，你家有幾個機上盒？你可以想像，隨著數位匯流發展和OTT時代來臨，未來可能不會再有免費的機上盒。

21世紀基金會主辦的「影視產業現況評估與規管架構研議研討會」今天舉行，國家通訊傳播委員會(NCC)七個委員當中，來了四個，委員過半的出席率被業者視為政策風向球的大放送，委員們的發言也格外受到業者關注。

NCC委員陳耀祥、何吉森、郭文忠、陳憶寧、台大教授葉疏、景文大學教授莊春發、元智大學教授周韻采、21世紀基金會副董事長高思博、台灣有線寬頻產業協會理事長彭淑芬、衛星公會祕書長陳依玫、五大MSO高層等產官學界人士齊聚一堂，對於有線電視費率鬆綁及如何因應OTT海嘯，熱烈討論。

NCC委員何吉森提到，隨著有線電視全面數化化，民眾家中「真的需要那麼多機上盒嗎？」現在每戶送兩個免費機上盒，有人反映未來是否可以在更換機上盒時允許收費？他知道有人家裡有10幾個機上盒，因為不用付費，一個房間來一個；但這樣的做法未來可能會有改變，有線電視系統業者可能不再提供民眾免費機上盒，過去的「以戶付費」有考慮往「以機付費」發展。

台大教授葉疏表示，對系統業者而言，全面數位化可申請關閉類比頻道，節省數位、類比兩種訊號雙載的頻寬資源，提供多頻道視訊及寬頻聯網服務。數位化過程中，業者的投資金額和人力成本大增，免費的數位電視 盒非一次性，很可能兩三年後就需更換，成本高昂、預期效益年數短，增加當期折舊費用，是業者很大壓力。

葉疏說，數位化過程中節目播放成本及節目版權成本增加，每戶節目版權成本成長是「非常有限的」；收入增加有限，數位紅利卻尚未顯現，過去被視為金雞母的有線電視系統業者獲利看衰。他建議應允許消費者依不同需求採不同收費，用良性競爭給予業者「較大的彈性空間」與誘因，才能促成正向競爭。

何吉森數度提及最近吵得火熱的授權費議題，授權費是頻道業者重要的固定收入，但政府要鼓勵的是內容產業;如何將錢導向內容、如何宣導民眾拒看盜版、如何讓業者在投入資金的同時，不致陷入「又要馬兒好、又要馬兒不吃草、最後馬兒死光光」的慘境，與會者認為鬆綁費率是必要之道。

向來有話直言的景文大學教授莊春發說，政府應思考依據業者數位化程度、調高基本收視費，根據業者數位化品質的提升給予適當基本費率調整，既然業者要花錢投資，就不能不讓業者將成本反映在收費上。

莊春發呼籲，政府應相信自由市場，「不要管太多」，否則把業者逼死，只是凸顯政府的刻薄，官員不需要很忙，對於市場，「就放下吧」，把市場的還給市場。

台灣有線寬頻產業協會理事長彭淑芬期望未來治理方向兼顧消費者權益的同時，要考慮產業永續發展，特別是鬆綁管制的可能性，產業先求生存，才有本錢談未來。
