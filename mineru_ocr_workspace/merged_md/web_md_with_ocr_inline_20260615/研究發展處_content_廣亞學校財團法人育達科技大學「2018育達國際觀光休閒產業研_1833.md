# 網站文章資料_廣亞學校財團法人育達科技大學「2018育達國際觀光休閒產業研討會」_1833

## 一、文章基本資料

- **文章ID**: 1833
- **文章標題**: 廣亞學校財團法人育達科技大學「2018育達國際觀光休閒產業研討會」
- **文章網址**: https://www.yzu.edu.tw/admin/rd/index.php/tw/?option=com_content&view=article&id=1833
- **文章簡介**: 主旨：本校休閒創意學院和休閒運動管理系訂於107年10月25日舉辦「2018育達國際觀光休閒產業研討會」，敬請惠予公告並鼓勵貴校相關人員報名參加，請查照。

說明：

一、隨著全球化的觀光休閒活動蓬勃發展，邀請國內外觀光、餐旅和休閒運動領域專家學者共襄盛舉，共同探索全球業態之創新與改變，進而促進觀光、餐旅和休閒運動產學界之永續發展，以因應全球化產業的影響，同時希望透過本次研討會的交流分享，建立跨國間的產官學界知識交換與共享平台，為產業人才開啟躍上全球新境界。

二、謹訂於107年10月25日（星期四)假本校綜合大樓三樓多媒體會議室（綜317會議室)。
