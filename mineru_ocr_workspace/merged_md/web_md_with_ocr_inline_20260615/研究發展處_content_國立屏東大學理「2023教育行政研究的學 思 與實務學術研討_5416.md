# 國立屏東大學教育行政研究所

「教育行政研究的學思與實務」 5.13

學術研討會 屏東大學民生校區五育樓國際會議廳

## 研討内容

(一)關於地方政府或學校層級與教育政策執行的研究成果。

(二)關於校長領導或教師領導、教師協作與學校教育的研究成果。

(三)關於非營利組織與學校(非學校)型態實驗教育的研究成果。

(四)關於教育經費投入與政策成效產出的研究成果。

(五)關於中央與地方政府或學校特殊教育評鑑的研究成果。

(六)關於公共化(准公共化)教保服務政策與執行成效的研究成果。

(七)關於幼兒園混齡教育與師資結構的研究成果 C

(八)其他有關於各類教育主題的研究成果。

## 研討會報名

1、報名期間：至2023年4月25日中午12時止，且視報名狀況提前截止報名。

2、報名網址：https://webap.nptu.edu.tw/aes/

(報名時，請以名稱或日期擇一查詢可快速搜尋本活動）

3、請發表人及參與研討會者務必線上報名，以利統計人數。
