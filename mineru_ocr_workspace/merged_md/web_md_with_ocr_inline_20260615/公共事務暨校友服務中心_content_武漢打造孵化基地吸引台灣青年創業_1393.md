# 網站文章資料_武漢打造孵化基地吸引台灣青年創業_1393

## 一、文章基本資料

- **文章ID**: 1393
- **文章標題**: 武漢打造孵化基地吸引台灣青年創業
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【澳廣視】

作為大陸中西部首家面向台灣的創業孵化器，提供軟硬件配套，目前已有 20多家台灣企業進駐，主要以商貿類為主。 台灣的元智大學科學教育研究中心在去年落地武漢發展。教學總監王祥寧表示，最初到內地發展時遇到不少困難，與武漢政府接觸後，發現當地提供不少支援，目前在武漢各區開設了多間辦學機構，現在有 20多位台灣人員，計劃未來繼續在台灣招聘 100多人到武漢工作。 (黃嘉靜於湖北武漢報道)
