- **文章全文**: 一、亞馬遜網路服務公司（Amazon Web Services） 為全球最大之雲端運算平台，其AWS Academy與AWS Educate平台授權各校免費使用，以培育雲端運算產業亟需之專業人才。AWS亦提供相關線上教材、線上實驗(labs.)及免費雲端運算資源使用，可供學校教師教學之應用與參考。

二、旨揭活動邀集國內、外專家學者及從業人員分享相關知能學習與實務運用經驗，並可實際操作 AWS 雲端運算平台，增加與會者新興科技相關知能。

三、活動資訊如下：

(一)本活動全程免費。

(二)活動時間：民國113年9月28日（星期六）上午9時至下午5時30分

(三)活動地點：瓶蓋工廠台北製造所（115028台北市南港區南港路二段13號）

(四)報名方式：線上報名。請至本活動官方網站報名（https://awscmd.tw/）

四、活動備有茶水餐點，為響應環保，請自行攜帶飲水容器。
