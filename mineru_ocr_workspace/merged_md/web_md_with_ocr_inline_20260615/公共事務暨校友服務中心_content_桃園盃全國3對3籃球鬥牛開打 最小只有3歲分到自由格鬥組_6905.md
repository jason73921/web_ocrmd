桃園市第7屆桃園盃全國三對三籃球賽今天開打，總獎金60萬元是國內少見高額獎金的鬥牛賽。市長張善政指出，比賽適逢暑假12組賽別合計2128支隊伍、7448名選手報名，職籃現役與前職棒好手也到場對戰各路好手。今年年齡最小只有3歲在「自由格鬥組」，最年長在「長青男子組」88歲都熱力拚鬥，挑戰全國規模最大、暑期超敢鬥三對三籃球賽 。

桃園盃全國三對三籃球賽今明兩天（22、23日）預賽，29日（周六）複賽，將於桃園高中光電籃球場、桃園區三民運動公園籃球場，及龍潭區國道三號高架橋下籃球場等地同時開打，30日決賽在元智大學體育館舉行，今年安排的場地都是有頂遮及室內比賽，讓大家不論大太陽或下雨都能盡情享受比賽。（待續）
