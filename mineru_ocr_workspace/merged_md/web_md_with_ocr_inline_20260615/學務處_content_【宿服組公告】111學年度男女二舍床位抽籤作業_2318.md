# 網站文章資料_【宿服組公告】111學年度男女二舍床位抽籤作業_2318

## 一、文章基本資料

- **文章ID**: 2318
- **文章標題**: 【宿服組公告】111學年度男女二舍床位抽籤作業
- **文章網址**: https://www.yzu.edu.tw/admin/st/index.php/tw/?option=com_content&view=article&id=2318
- **文章簡介**: 公告日期：111年4月8日（星期五）

 

主旨：公告111學年度男女二舍床位抽籤作業。

說明：

一、抽籤前請詳閱下列應注意事項：

(一)請妥慎評估自我「團體生活適應能力」以及「健康狀態是否適宜住宿」後再決定是否參加校舍抽籤。

(二)依據「元智大學學生宿舍管理規則」第八條規定，申請之床位一次為一學年，即應住滿一學年（包含上、下學期），除休學、退學、轉學、實習、出國交換及特殊原因經核准者外，均不得申請退宿，住宿未滿一學年而自願或勒令退宿者，一律不退還住宿費及保證金，學生不得異議。

(三)欲參加抽籤者建議提前備妥住宿同意書（需家長簽名），以免逾期。

(四)抽完籤欲放棄者，請自行回到Portal取消之，若於4/17系統關閉後有任何異動應《立即主動》告知宿服組，如權益受損，責任自負。

二、抽籤資格：大一～大三在學之大學部學生。

下列學生請勿參與抽籤：

(一)已申請優先住宿並經核准者。

(二)宿舍自治會幹部。

(三)資服處推薦之網管人員。

(四)體育室推薦之校隊代表。

(五)境外生（含外籍學位生、僑生、陸生、雙聯生），不需抽籤但須提出優先住宿申請。

三、抽籤日期：111年4月15日(星期五)10時起 至111年4月17日(星期日)24時止。

四、抽籤方式：進個人Portal點選『住宿申請』選項，並依指示完成抽籤程序，籤號在10000至99999之間，床位依所抽籤號由小到大依序分配，額滿為止，申請者不得異議。

五、中籤結果公告日期：111年4月18日（星期一）。

六、床位分配方式：

(一)以男女二舍為主要住宿區。

(二)中籤者可自組室友，未自組室友者由宿服組權責分配。

(三)寢室分配由宿服組全權決定。

(四)開放抽籤床位數：預估男二舍400床，女二舍580床。

七、後續床位作業預告：

4/18 中籤公告 │ 4/18~4/22 繳交住宿保證金及同意書 │ 4/25 第二次抽籤。
