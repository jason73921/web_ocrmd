- **文章簡介**: 【科技新報/作者陳冠榮】
 
Google Cloud 31 日推出「雲端人才搖籃計畫」，繼 7 月底與元智大學合作後，宣布再與東海大學、高雄科技大學展開合作，共同推動雲端學程，並協助導入 Google Cloud Platform（GCP）應用平台，完善雲端教育環境。這項計畫預計在 3 年內協助 1,500 位學生取得 Google Cloud 雲端基礎認證，獲得認證的學生也將列入 Google 人才庫，成為 Google Cloud 合作夥伴未來徵才的優先考量對象，助力產學接軌，不斷為台灣產業注入人才活水。

在「雲端人才搖籃計畫」中，Google Cloud 將提供課程教材、教學簡報、測驗題庫、創新實驗課程以及全球雲端認證計畫等，協助學生精進雲端專業知識並考取受國際認可的證照。這項計畫分兩階段進行：第一階段將完成 Google Cloud 雲端運算基礎課程（Google Cloud Computing Foundation，GCCF）的所有規劃及種子教師培訓，涵蓋大數據、人工智慧、機器學習等重要應用領域；第二階段將開辦正式學程，培育對象以合作院校的大二學生為優先，並同步進行 Cloud Digital Leader（CDL）與 Associate Cloud Engineer（ACE）兩項同樣為全球認證的進階課程。

元智大學校長廖慶榮表示：「透過與 Google Cloud 合作，不僅讓學生們循序漸進地從基礎到進階學習雲端知識以及培訓出種子教師，透過 Google Cloud 雲端技術工具的挹注，也讓學校行政、研發單位有更多的資源加速打造雲端教育環境，達成智慧校園轉型的目標。」（待續）
