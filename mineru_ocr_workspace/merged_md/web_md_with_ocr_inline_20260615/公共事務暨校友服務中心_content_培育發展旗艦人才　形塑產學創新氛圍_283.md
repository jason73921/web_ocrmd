# 網站文章資料_培育發展旗艦人才　形塑產學創新氛圍_283

## 一、文章基本資料

- **文章ID**: 283
- **文章標題**: 培育發展旗艦人才　形塑產學創新氛圍
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【台灣好新聞/記者葉志成／桃園報導】 


在多變又具風險的環境，青年需要更多更多具創新能力的人才，以因應未來職涯的各種挑戰。鑒於各式人才斷層及學用落差問題日益嚴重，桃園市政府青年事務局特規劃辦理於TYC-Campus創新產業人才培育發展旗艦型計畫，透過主題學程、產業實作及輔導，並引入國際資源，希望培育具創意、國際觀及問題解決能力的青年，形塑桃園市產學創新氛圍。104年10月8日特邀請本市所有大學之育成單位及青年學子共同參與記畫啟動記者會，期望促進未來產業及社會轉型。

桃園市政府邱俊銘副秘書長表示：「桃園市有六都最年輕的人口結構，市內有29個工業區，涵蓋各產業，另16所大專校院，都是相當優異的資源。而在產業變化劇烈的時代，產學必需積極接軌，青年才能在學習適用於產業之職能及創意，方能在未來促進產業轉型。本計畫由中原大學負責執行，策略性地結合桃園的大專校園創新育成資源，連結各重點及創新產業，目的就是培育跨領域創新人才，為青年打造其所必需能力，許青年一個面對各種環境挑戰的基本能力。」計畫以5大服務主題為主軸，分別是開拓、培育、連結、實作、提升，從發掘產業及人才需求開始，開設主題學程及國際聯結之活動，再辦理學生實作，並在實作中提升其能力。

青年事務局指出，本計畫將逐步整合各類產學資源，主題學程除了產學方面的技術性培育外，同時建立良好的工作態度及團隊合作機制，透過跨域式及主題式設計交流活動，提供參加者相互連結、拓展國際視野的機會。另外，不同於一般的被動學習的實習課程，計畫中提供青年學生至創新產業及新創公司實作的機會，在實作中加入了實習單位內部及外部的指導，積極培養學生主動學習的動機，提升青年發掘問題與解決問題的能力，對其未來職場就業力及創新能量必大有助益，亦能協助解決桃園地區產業轉型斷層問題。

本旗艦型計畫由中原大學執行，另有一般型子計畫由元智大學執行，再搭配其他大專校院創新育成單位，桃園在交通區位、產業聚落、學研體系及人力資源上，都具成為亞洲矽谷的良好基礎，這一系列的計畫都希望建立一種產學連結的網絡環境，讓彼此能夠相互交流支援，讓青年在未來有更寬廣的發展空間。旗艦型計畫將陸續在104年9月~12月舉辦「工具機、車輛及自行車」、「軟硬體設計與數位資訊應用」兩個專業領域系列學程；一般型子計畫則有「社會企業」及「觀光休閒」2個系列學程，可充份配合桃園市轄內產業聚落及具潛力產業類型。計畫並將設置「TYC-Campus創新產業人才培育發展計畫網站」作為桃園地區大專校院青年於創新人才培育之資訊平台。最後於104年12月10日辦理「TYC-Campus計畫聯合成果發表展」，讓實習團隊進行成果發表及展示，使產業及人才創新達到雙贏局面，達到成果擴散目的。
