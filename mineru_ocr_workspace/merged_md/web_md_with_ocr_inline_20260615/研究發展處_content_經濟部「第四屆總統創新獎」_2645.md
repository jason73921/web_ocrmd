# 網站文章資料_經濟部「第四屆總統創新獎」_2645

## 一、文章基本資料

- **文章ID**: 2645
- **文章標題**: 經濟部「第四屆總統創新獎」
- **文章網址**: https://www.yzu.edu.tw/admin/rd/index.php/tw/?option=com_content&view=article&id=2645
- **文章簡介**: 主旨：第四屆「總統創新獎」自即日起至本(108)年9月30日止受理報名，敬請貴單位協助轉知徵選訊息，並鼓勵所屬單位踴躍報名參選，詳如說明，請查照。

說明：

一、「總統創新獎」係鼓勵產、官、學、研等之個人或團體，積極追求在產品、技術、管理、組織、行銷、服務或文化等多元創新與價值創造,帶動國家整體經濟發展與強化國際競爭。

力優勢。

二、第四屆「總統創新獎」自即日起至本(108)年9月30日(星期一)止受理各界推薦及報名,為讓各界瞭解徵選作業與相關規定，自本年8月26日(星期一)起,規劃於臺北、新竹、臺中、臺南、高雄及花蓮辦理參選說明會,請惠予協助公告與轉知第四屆「總統創新獎」徵選訊息，並鼓勵所屬單位踴躍報名參選。

三、檢附「總統創新獎」參選須知(附件1)、徵選海報(附件2)、

參選說明會宣傳單(附件3)等資料，敬請貴單位協助於所屬網站公布第四屆「總統創新獎」徵選訊息，相關參選訊息請至「總統創新獎」獎項網站査詢(https://www.rocpia.tw/)ㆍ

四、有關受理推薦及報名，與詢問徵選作業等相關事宜,請逕洽總統創新獎委辦單位:中華民國產業科技發展協進會，服務電話:(02)2325-6800分機881(魏小姐)、分機891(林小

姐)、分機885(黄先生)、分機880(鄭先生)，電子信箱：rocpia-info@mail.hinet.net。
