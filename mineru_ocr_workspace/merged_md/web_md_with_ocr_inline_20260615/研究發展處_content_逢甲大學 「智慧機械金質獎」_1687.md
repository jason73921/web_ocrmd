# 網站文章資料_逢甲大學 「智慧機械金質獎」_1687

## 一、文章基本資料

- **文章ID**: 1687
- **文章標題**: 逢甲大學 「智慧機械金質獎」
- **文章網址**: 
- **文章簡介**: 主旨：經濟部工業局主辨之107年「智慧機械金質獎」典範案例甄選活動開放受理，敬請協助轉知及訊息曝光，鼓勵廠商踴躍報名參加甄選，請查照。
說明：
一、為發掘標竿企業以及建立產業精進典範，樹立國內產業發展之指標，鼓勵產業由精密機械升級轉型為智慧機械，特規劃智慧機械與智慧製造等典範案例甄選，頒發【智慧機械金質獎】，作為企業觀摩學習對象，擴大推動智慧機械創新產業之效益，使獲獎企業量能擴散，帶動國內產業投入研發，加速導入智慧製造生產模式。
二、受理期間與聯絡窗口：

(一)受理期間：自公布日起至107年5月31日止，逾期恕不受理。

(二)聯絡窗口：逢甲大學產學合作處-林靜慧小姐，電話：(04)2451-7250分機6815。

(三)收件地址：臺中市西屯區文華路100號(逢甲大學產學合作處)。
三、甄選活動詳細內容，請參閱活動網站 http://www.smartm.org/org/。
