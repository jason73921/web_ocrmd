# 網站文章資料_2020基隆山海工作營Ｖ5.0 創作成果展_4508

## 一、文章基本資料

- **文章ID**: 4508
- **文章標題**: 2020基隆山海工作營Ｖ5.0 創作成果展
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20200810-1
- **文章簡介**: 【台灣新生報/記者王世明/基隆報導】

為解決基隆青年外流、空間閒置及產業沒落等問題，尋找基隆西岸太平山城的地方創生契機，基隆市政府與元智大學第五年合作，從七月三十一日起舉辦為期九日的「２０２０基隆山海工作營Ｖ５．０」，並於八日（星期六）舉辦創作成果展，包括市府都發處徐燕興處長、市議員莊敬聖及眾多到場一同共襄盛舉，感受基隆特色山海景色，也一同見證基隆的改變。

基隆市府表示，基隆港西岸太平山城是基隆市地方創生四大場域之一，市府除了積極推動相關硬體建設，包含西岸纜車規劃、都市計畫道路開闢、山海散步道改善與太平國小整改建之外，也透過舉辦軟體活動，像是設計與建築者之家展覽、都市設計及再生議題交流講座、山海工作營與小市集等，討論及實驗山城生活新可能，翻轉城市，以接續明年的城市博覽會。

元智大學陳冠華教授則表示，這次「山海小市集」共招募七十攤，市集招募一下就秒殺報名，競爭非常激烈，今年不只擴大規模，更讓市集更加接地。除此之外，搭配山城創生主題，特邀阮慶岳老師辦理「山徑躊躇」新書發表會。陳教授進一步說明中山一路一一三巷已經成為非常著名山上文化小徑，隨著上山小徑可以感受基隆特色山海景色，市集也展現太平山城聚落多樣面貌以及實驗未來的各種可能。（待續）
