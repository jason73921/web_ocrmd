# 給就學貸款畢業的學生貸款屆期請依約還款

恭喜畢業的學生，順利完成學業！

為保有自己的信用，別忘了依約還款，若有還款困難，可善用緩繳貸款本金及延長償還貸款期限等措施。祝福你們～擁有幸福健康的人生

依據《高級中等以上學校學生就學貸款辦法》第11條「申請本貸款之學生於各階段學業完成後，應依規定向承貸銀行償還貸款……。」第12條「學生或保證人未依貸款契約償還借款者，由承貸銀行依法追繳，並將資料送請財團法人金融聯合徵信中心建檔，列為金融債信不良往來戶，並揭露至貸款完全償還為止；已償還者，由承貸銀行通知財團法人金融聯合徵信中心註銷紀錄。

學生於開始分期償還後，有下列情形之一者，得向承貸銀行申請緩繳本金，緩繳期間每次為一年，並以十二次為限；緩繳期間之利息，由各級主管機關負擔：一、年收入未達前條第三項中央主管機關所定一定金額。

二、持有低收入戶、中低收入戶證明 。 。

依據《高級中等以上學校學生就學貸款作業要點》第十八點「學生於開始償還貸款之前一年度，平均每月所得未達新臺幣五萬元（前一年度如有就學或服義務兵役之緩繳期間不予列計）及為低收入戶或中低收入戶，得申請緩繳貸款本金（已逾應償還起算日或已開始還款者，應先償還逾期期間之已到期本息、違約金後，始得申請），如學生有符合本辦法第七條第一項第二款第一目或第二目之子女，每有一名子女得再增加新臺幣一萬元。

依前項規定申請緩繳貸款本金最多以申請十二次為限，每次申請緩繳期限為年，貸款到期日並隨緩繳期限順延。其緩繳期間之利息，由各級主管機關負擔。……」

查詢就學貸款金額可透過台灣銀行官方網站的服務專區之「還款明細查詢」功能查詢你的就學貸款還款明細，或至台灣銀行網路銀行／行動銀行查詢。

元智大學學生事務處

生輔組敬啟
