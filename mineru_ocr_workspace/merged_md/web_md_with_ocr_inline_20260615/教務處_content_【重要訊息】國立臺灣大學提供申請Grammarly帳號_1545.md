# 網站文章資料_【重要訊息】國立臺灣大學提供申請Grammarly帳號_1545

## 一、文章基本資料

- **文章ID**: 1545
- **文章標題**: 【重要訊息】國立臺灣大學提供申請Grammarly帳號
- **文章網址**: https://www.yzu.edu.tw/admin/aa/index.php/tw/2024-01-08-3
- **文章簡介**: 國立臺灣大學EMI教學資源中心（以下簡稱EMI TLC）秉持資源共享、共學共益的理念，將於113年再次開放全國大專校院『專任教師』申請免費英語編修軟體Grammarly帳號。

EMI TLC為使資源更加充分運用、觸及更多師生，113年起將不再提供已獲教育部「大專校院學生雙語化學習計畫」第一期第2階段之重點培育學校(院)申請，因此本中心僅收『人社院及工院專任教師』之帳號申請！

請有意願申請Grammarly帳號之專任教師於113/2/16(五)前於google表單填寫並提交以下資料，本中心彙整完畢後將交由EMI TLC開通帳號。

逾期恕不受理，敬請見諒。謝謝！


可申請資格：人文社會學院及工程學院之專任教師(限額46名)。

提供資料：姓名、系所名稱、職稱、學校Email、專長領域。

申請期限：即日起至113年2月16日（五）止(額滿將提前截止)。


※名額有限，將以符合資格之前46位專任教師為申請名單送交EMI TLC，請有興趣之教師儘早申請，謝謝！

 

--------------------------------------------------------------------------------------------------------------------------------------------------


注意事項：



務必提供學校Email，若提供個人Email，如Gmail，EMI TLC將不予受理亦不另行通知。

EMI TLC將統一於113年3月22日(五)開通帳號，若實際開通日期有異動將另行通知。

帳號開通後，申請教師將收到由Grammarly所寄發之邀請註冊信，點選「Join Now」，並填入同申請時填寫之學校Email並自訂密碼，即可開始使用。

帳號開通後若3個月未使用或3個月內未完成註冊者，EMI TLC將會刪除帳號，不另行通知。

Grammarly使用期限：帳號註冊日至114年1月17日。若實際使用期限有異動將另行告知。
