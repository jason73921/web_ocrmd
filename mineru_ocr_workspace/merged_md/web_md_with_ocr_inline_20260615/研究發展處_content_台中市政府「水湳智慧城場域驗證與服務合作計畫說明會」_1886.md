# 網站文章資料_台中市政府「水湳智慧城場域驗證與服務合作計畫說明會」_1886

## 一、文章基本資料

- **文章ID**: 1886
- **文章標題**: 台中市政府「水湳智慧城場域驗證與服務合作計畫說明會」
- **文章網址**: 
- **文章簡介**: 主旨：臺中市政府訂於107年8月17日（星期五）辦理「水湳智慧城場域驗證與服務合作計畫說明會」。

說明：

一、臺中市政府訂於107年8月17日（星期五）辦理「水湳智慧城場域驗證與服務合作計畫說明會」，邀請產、官、學界與會。

二、臺中市政府將水湳智慧城定位為「低碳、智慧、創新」之智慧應用示範區，為結合民間創研能量，打造水湳智慧城為創新服務與智慧應用方案之實驗場域，預計於本（107）年8月中旬公告「水湳智慧城場域驗證與服務合作計畫」，為利計畫順利推廣，特辦理推廣說明會。

三、計畫說明會訊息如下：

（一）時間：107年8月17日（星期五）09:00~11:00

（二）地點：臺灣大道市政大樓惠中樓401會議室（臺中市西屯區臺灣大道三段99號）

（三）報名方式：請於107年8月16日17:00前線上報名，如提前額滿將停止報名。

（四）會議資料：請至臺中市政府資訊中心官方網站水湳智慧城專區下載。

（五）備註：因席位有限，建請各與會單位至多派2人與會，如有意參加計畫說明會，除逕於線上報名外，惠請告知研發處邱宇君小姐，將提供開會通知單俾憑免費停車。

四、聯絡人：水湳智慧城專案辦公室，TEL：04-22289111#22124、22126，E-mail：spmo@taichung.gov.tw 。
