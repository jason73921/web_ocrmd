# 網站文章資料_保命防跌 監測浴廁濕度溫度發警訊_1509

## 一、文章基本資料

- **文章ID**: 1509
- **文章標題**: 保命防跌 監測浴廁濕度溫度發警訊
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【青年日報】

濕滑的浴廁潛藏危機，元智大學團隊設計「PREtector浴警」，可感測濕度、溫度等，分析浴廁環境是否會提高跌倒率，並發出警訊，有助於維護居家安全。

元智大學學生梁庭浩、沈和寬、胡家綺等人組成團隊，在元智大學大數據與數位匯流創新中心教師潘人豪指導下，設計「PREtector浴警」，提升年長者的居家照護安全，獲得今年「智慧城市應用服務設計競賽」佳作與遠傳專題獎。

潘人豪表示，PREtector結合「實體監控裝置」和「雲端平台」，只要在浴廁中裝設監控裝置，並在平台上開通帳號串聯裝置後，裝置就能感測環境中的濕度及溫度等各種環境因素，透過即時分析裝置接收到的數據，瞭解是否會造成使用者跌倒率提高，若達警告值則發出警訊。

跌倒是長者主要事故傷害之一，考量長者對穿戴式裝置排斥，潘人豪表示，在不接觸身體前提下，設計在環境裡藉由物體掉落地面產生的震動及聲波頻率，進行交叉比對主動偵測狀況，判斷是否有跌倒發生，並透過中央管理平台，讓遠端照護人員監測，可在緊急事件發生時進行即時通話，幫助釐清現場狀況，提供主動救援，達到緊急照護目的。

潘人豪說，團隊希望透過「PREtector」，能有效減輕意外風險，進一步將目標由機構內的高齡照護環境，延伸到個人居家環境及服務業場域中，提升所有民眾在使用浴廁時，能獲得更佳的安全防護。
