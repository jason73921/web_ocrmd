# 網站文章資料_元智大學資傳系在ISARCH國際比賽大放異彩 三組專題榮獲互動組Winner獎項_342

## 一、文章基本資料

- **文章ID**: 342
- **文章標題**: 元智大學資傳系在ISARCH國際比賽大放異彩 三組專題榮獲互動組Winner獎項
- **文章網址**: https://www.yzu.edu.tw/index.php/tw/20241205-1
- **文章簡介**: 元智大學資訊傳播學系學生在國際舞台上再創佳績！該系三組畢業專題作品「愛無能」、「NO.2」以及「Multifly」在2024年ISARCH國際比賽中脫穎而出，榮獲互動設計組Winner獎項，展現學生在互動設計與創新應用上的卓越實力。

 

由黃絜如與張韶宸二位教授所指導的「愛無能」，是由方羿捷、郭久慶、黃薰儀、林意軒、李苡秀五位同學共同創作。作品利用爬蟲技術蒐集與「愛無能」相關的詞彙，並以動態視覺呈現情感缺失的概念。使用者先完成互動測驗，得知自身的「愛無能」指數，並在展場中透過AI生成專屬圖像，結合共鳴的主題標籤，深入探索情感的內涵。

 

作品「NO.2」則由林俊廷、陳品如、陳芷媛、張翔喻、周庭豪、黃巧鈴等六位同學製作，指導老師為黃絜如與張家榮。透過互動網頁，將食物消化後的碳排放量、產電量及環境影響可視化，結合AI辨識技術與資訊設計，幫助使用者理解飲食行為對永續發展的影響，並提供環保飲食建議。以趣味方式倡導環境友善飲食，呼籲大眾攜手為地球永續努力。

 

由黃絜如老師所指導，林廷諭、王羽茹、劉羽晴、詹侑庭、趙蔆瑄、黃宥瑄六位同學共同完成的「Multifly」，作品以AI技術與醫學知識結合，將皮膚斑紋轉化為獨特藝術形式，呈現皮膚多樣性之美，並與異位性皮膚炎基金會合作，讓大眾重新審視身體特徵的價值。

 

元智大學資傳系主任鄧進宏對學生的卓越表現給予高度肯定，他表示，這些成果不僅為元智大學爭光，更展現了年輕世代在創新設計與社會關懷上的無限潛力與熱情。

 

校方強調，此次的成就不僅是對學生努力的最佳肯定，更象徵著元智大學在教育與創新發展上的重要里程碑。未來，學校將持續投入資源並提供多元平台，支持學生參與更多國際競賽，鼓勵他們在專業領域不斷突破，為社會帶來正面且深遠的影響。

 

此次學生在ISARCH國際比賽中的卓越表現，充分展現元智學子的創新實力與國際競爭力。期待這群優秀的年輕人，未來能在更多國際舞台上發光發熱，為台灣的創意產業注入嶄新的活力與動能！
