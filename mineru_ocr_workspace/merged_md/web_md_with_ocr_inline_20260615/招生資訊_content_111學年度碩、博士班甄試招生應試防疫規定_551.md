# 元智大學因應「嚴重特殊傳染性肺炎」防疫111 學年度碩、博士班甄試招生應試防疫規定

本校 111學年度碩、博士班甄試口試依簡章原訂日期(110/11/6-9間)如期舉行，詳細口試日期請參閱簡章「學系分則」，個別口試時間及資訊將於 110/11/3公告於各系所網頁。為因應「嚴重特殊傳染性肺炎」防疫，以下應試防疫規定煩請考生務必配合辦理：

1. 居家隔離、檢疫及自主健康管理等 3類考生，事前請考生主動通報本校：

(1). 口試當日若為「居家隔離」、「居家檢疫」或「社區監測通報採檢個案尚未接獲結果」者，請配合留在家中，不得到校應試，考生可依自身身體狀況及環境設備依本校「元智大學招生入學視訊面試規範」第一條，檢附證明提出申請視訊面試。考生須簽署本校「元智大學視訊面試應試切結書」，併同本校「元智大學招生入學視訊面試申請表」於報考系所 (學位學程)面試日2天前，以電子郵件、傳真等方式送達招生系所（學位學程）。未依規定提出申請，一律不予受理。

(2). 口試當天考生如無法進行視訊面試，請於應試日期前掃描傳真或E-mail提供主管機關開立之「居家隔離通知書」、「居家檢疫通知書」等證明，並填寫「元智大學研究所招生報名費退費申請表」，可申請報名費全額退費。傳真電話:03-4630997；E-mail：tutoanda@saturn.yzu.edu.tw；連絡電話：03-4638800 分機 2315。

(3). 口試當日若屬於需「自主健康管理」者，或應試當天若有「發燒」、「嚴重呼吸道症狀」、「腹瀉」、「嗅覺喪失」等身體不適症狀者，應於試前主動告知本校各系所承辦人員，以利安排至「隔離考場」應試。各系所承辦人員連絡電話：總機 03-4638800，分機如下:

醫學研究所碩士班，分機:6601。

中國語文學系碩士班，分機:2707。

社會暨政策科學學系碩士班，分機:2161。

藝術與設計學系藝術與設計管理碩士班，分機:3301。

文化產業與文化政策博士學位學程，分機:2701。

電機工程學系(甲組)，分機:7101、7102。

工業工程與管理學系，分機:2501。

化學工程與材料科學學系，分機:2551。

機械工程學系，分機:2451。

管理學院經營管理碩士班，分機:6051。

管理學院財務金融暨會計碩士班，分機:6062(財務金融學程)、6061(會計學程、金融科技學程)。

管理學院博士班(學術組)，分機:6052。

資訊管理學系，分機:2791。

2. 考生入校請主動出示應考証，並全程佩戴口罩應試：因考量試場為相對群聚及密閉空間，考生除報到試務人員核對身分時，須配合暫時取下口罩，辨識身分後再戴回；其餘時間，請考生全程佩戴口罩應試。

3. 提前報到，體溫量測：請依各系所規定時間提前辦理報到(詳見各系所網頁公告)，以配合檢測及執行量體溫等防疫工作。

4. 考生應落實自我健康狀況監測：如應試當天若有身體不適症狀者，應全程配戴口罩、保持社交距離，並主動告知監(試)務人員，依試務人員指示進入規定試場應試。

5. 因應「COVID-19」疫情，本校校園目前仍暫停對外開放。故口試當天

(1). 考生進入校園需主動出示「應考證」，方得入校應試。 請考生出門前務必備妥「應考證」及相關身分證件，以利應試。應考證列印方式: 進入元智大學報名系統網址：https://exam.yzu.edu.tw/NewNetapply/ （選擇「111 碩士班甄試」或「111 博士班甄試」/點選「查詢報名結果與成績」/輸入身分證字號、密碼/點選報考系所「點選進入」/點選右上方「應考證」列印）。

(2). 目前暫不開放親友陪考：因本校校園目前暫停對外開放，原則上不開放親友陪考。但考生如有特殊陪考需求，原則上得申請1 位親友陪考。申請人需於口試日前一日中午前，填妥「陪考人員實聯制申請表」，E-MAIL回傳系所申請。考試當天陪考人於校門口出示「陪考人員實聯制申請表」正本入校。洽詢電話：總機03-4638800，第一條第3 點各系所承辦人員。

6. 本公告未盡事宜，悉依政府各項最新防疫規定、本校簡章及相關招生規定辦理。
