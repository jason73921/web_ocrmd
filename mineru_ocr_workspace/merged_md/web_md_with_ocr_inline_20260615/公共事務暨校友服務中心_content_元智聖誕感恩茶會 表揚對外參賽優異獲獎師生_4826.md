# 網站文章資料_元智聖誕感恩茶會 表揚對外參賽優異獲獎師生_4826

## 一、文章基本資料

- **文章ID**: 4826
- **文章標題**: 元智聖誕感恩茶會 表揚對外參賽優異獲獎師生
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20201224-3
- **文章簡介**: 【大紀元記者陳建霖台灣桃園報導】

元智大學聖誕感恩茶會表揚一年來對外參賽成績優異獲獎之師生，參加者約兩百人，校長吳志揚親自頒獎，場面熱絡溫馨。今年邀請捐款十萬元以上之校友及教職員參加，依「捐資教育事業獎勵」辦法，申請教育部部長之獎狀或獎牌，由校長頒發之。

吳志揚表示，感恩茶會為感謝師生對外參賽獲得獎項，為校爭取榮譽。元智向以資通訊聞名，在物聯網、大數據分析以及人工智慧等方面研究實力很強。因應人工智慧時代來臨，元智的學生實作能力很強，老師指導認真負責，在業界名聲很好，吳志揚說，感恩茶會是鼓勵師生對外多參加比賽，增強自身實力，為校爭取榮譽，除了相互切磋，增廣見聞外，對於學生的職涯也有十足幫助。

今年邀請捐款校友加入，吳志揚說，校友是學校永遠的家人及資產，感謝校友及老師同仁的捐款，對於教育長期的關心與支持，同時校友在職場上也提供學生許多實習的機會，職涯上提攜學弟妹給予協助，讓整個產業人力資源更加活絡有競爭力，以促進國家整體經濟的發展。（待續）
