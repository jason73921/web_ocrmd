- **文章簡介**: 公告日期：111年11月24日(星期四)

 

 

1.至櫃台填寫網路修繕本，並詳述問題狀況。

2.在Portal報修，顧客服務→選擇問題類別「學生宿舍網路」，並詳述問題狀況。

資服處收到案件後，會派人前往處理。

 

！顧客服務系統內問題類別不要填錯囉！

如果類別填成一般宿舍修繕，還要經過案件類型分類，再轉移案件給資服處，處理時間上會慢許多，記得網路問題要選「學生宿舍網路」喔！



  

1.Go to the counter to fill in the Network Repair book and write down the problem in detail.

2.Report the case in the customer service system on Portal, question category  select ‘student dormitory network’ and describe the problem situation.

After we received the case, the Office of Library and Information Services(OLIS) will send someone to handle it.

 

～ Do not fill in the wrong problem type in the customer service system ～

If the problem type category is filled in as general dormitory repairs, the case will need to classify then transferred to the OLIS. The processing time will be much slower. Remember to select "Student Dormitory Network" for network problems!
