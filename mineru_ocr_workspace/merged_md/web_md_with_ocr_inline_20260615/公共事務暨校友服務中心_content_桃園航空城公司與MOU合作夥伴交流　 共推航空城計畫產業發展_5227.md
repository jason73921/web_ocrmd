# 網站文章資料_桃園航空城公司與MOU合作夥伴交流　 共推航空城計畫產業發展_5227

## 一、文章基本資料

- **文章ID**: 5227
- **文章標題**: 桃園航空城公司與MOU合作夥伴交流　 共推航空城計畫產業發展
- **文章網址**: 
- **文章簡介**: 【經濟日報 / 吳佳汾】

桃園航空城計畫進入實質開發階段，優先產專區招商行動緊鑼密鼓展開，因此桃園航空城公司於今(30)日邀請MOU合作夥伴及桃園市政府地政局、經濟發展局，針對桃園航空城計畫開發進度與產業規劃特色與有意進駐企業，進行交流對談。

桃園航空城公司董事長陳錫禎於會中致詞表示，航空城公司至今已簽署49家MOU合作夥伴，透過「桃園航空城合作備忘錄夥伴交流座談會」分享並廣納產業界寶貴意見，期待MOU合作夥伴未來進駐後，帶動航空城計畫整體經濟成長與發展。

第一梯次的「桃園航空城合作備忘錄夥伴交流座談會」與會企業包含台杉投資管理顧問股份有限公司、億鴻系統科技股份有限公司、台灣航空貨運承攬股份有限公司、臻鼎科技控股股份有限公司、台灣準時達國際物流股份有限公司、台灣歐西瑪股份有限公司、車麗屋汽車百貨股份有限公司、泵乂科技股份有限公司、創奕能源科技股份有限公司、元智大學以及桃園市政府地政局、經濟發展局共同與談，產官企齊聚一堂，為實現「機場經濟學、智慧航空城」而努力。

主要業務為「桃園航空城開發計畫招商」與營運「亞洲‧矽谷創新研發中心」的桃園航空城公司董事長陳錫禎於會中提到「亞洲‧矽谷創新研發中心」與桃園航空城計畫相輔相成，打造青埔特區成為5G高科技的智慧應用實證場域，建構創新創業生態系，落實桃園智慧城市的建設願景。透過本次活動，讓產業與政府面對面，促成招商進駐，官企產業合作交流，塑造桃園航空城的智慧城市發展模式。陳錫禎強調，未來將舉辦多場產業交流座談會，持續邀請MOU合作夥伴同聚一堂，促進產業整合升級，帶動航空城整體發展，將產業創新能量推向國際，達到「桃園模式、世界城市」願景。

桃園市政府地政局長蔡金鐘於會中分享航空城計畫土地開發與徵收最新期程，包括產專區分區劃設面積、最小建築基地面積及道路最小面寬等土地開發最新資訊，讓MOU合作夥伴進一步了解航空城計畫產專區相關資訊，未來若有機會正式進駐產專區，能更有效規劃產業用地及建築面積。

桃園市政府經發局副局長黃穗鵬於會中分享航空城招商及產業規劃，包括第三、四種產業專用區發展產業之項目、優先產專區之產業規劃及招商作業期程等最新資訊。希望未來以桃園資通訊產業及製造業的堅強實力，透過高科技產業廠商進駐，促進航空城計畫產業發展，並期許順利進入招商作業階段。

本次座談會出席人員有桃園航空城公司董事長陳錫禎、總經理許又銘、副總經理羅碧霞、曾鈐龍；桃園市政府地政局長蔡金鐘、經濟發展局副局長黃穗鵬；台灣航空貨運承攬股份有限公司董事長許旭輝、台杉投資管理顧問股份有限公司總經理翁嘉盛、億鴻系統科技股份有限公司董事長張家豪、臻鼎科技控股股份有限公司總經理事特助李葆元、台灣準時達國際物流股份有限公司總經理邱尚謙、台灣歐西瑪股份有限公司董事長鄧江榮、車麗屋汽車百貨股份有限公司董事長卓豐閔、泵乂科技股份有限公司總經理林明輝、創奕能源科技股份有限公司海外市場業務管理師賴怡真、元智大學知識服務與創新研究中心畢家豫博士、李詩政教授與桃園市政府相關局處等貴賓。
