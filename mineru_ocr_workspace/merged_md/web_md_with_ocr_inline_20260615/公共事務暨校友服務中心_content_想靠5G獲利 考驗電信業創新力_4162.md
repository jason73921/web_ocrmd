# 網站文章資料_想靠5G獲利 考驗電信業創新力_4162

## 一、文章基本資料

- **文章ID**: 4162
- **文章標題**: 想靠5G獲利 考驗電信業創新力
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20200110
- **文章簡介**: 【聯合報/記者黃晶琳、戴瑞瑤】

五大電信業者爭搶第五代行動通訊5G頻譜，其中三點五GHz頻段每人每Ｍ暫時得標金已創世界最高，卻配上全球最低4G吃到飽資費基礎，未來進入5G時代，消費者用好、用滿的習慣難倒退，電信業者若還要每年維持穩定獲利，短線考驗業者財務調度，長線考驗業者開發5G創新商業模式的能力。

目前三點五GHz每單位（十Ｍ）標金已經衝上三十六點六一億元，假設某業者拿下八十Ｍ，標金也高達二九三億元，即使拉長二十年使用年限費用攤提，自拿到執照的那一刻起，就得面對每個月超過一點二二億元的頻譜費用，更遑論還有數百億元的5G網路建設投資。

回顧二○一三年我國4G行動頻譜競標創下一一八六點五億元天價，當年也被批評恐讓消費者買單；但隔年4G開台後，業者先強打千元以上分級付費，但隨市場競爭激烈，一路降低吃到飽門檻，到了二○一七年甚至出現四九九元超低價吃到飽。當年參與決策人士透露，4G標金出現天價後，當時擔任行政院政務委員的張善政也在立院表示，未來4G的基本入門費率，應不高於3G費率；事後回顧，業者為了快速回收成本，反而拚命建設、擴大用戶數、爭取現金流，帶動4G產業發展、政府稅收也豐收，民眾行動上網普及化。

目前全球都看好5G趨勢，美中甚至爆發科技戰，這次電信業者競標5G也出現類似當年的戲碼，隨著5G標金飆高，業者有輸不起的壓力。

元智大學資訊管理學系教授周韻采直言，5G成本已超乎預期，雖短期從業者財報看來不會有影響，但後續隨著執照、設備開始折舊等，5G虧損會逐漸明顯，且台灣的行動用戶已經飽和，如果業者能做的只是把4G用戶移轉到5G，對業者來說將是長期利空。儘管大家對5G時代充滿遠景，但現在最讓業界沒把握的是「5G創新商業模式在哪裡？」何況面對民眾已習慣低價、高傳輸量的環境，電信業者要如何制定資費，還能年年維持穩定獲利，在5G的殺手級應用出現前，電信高層坦承「很頭疼」。（待續）
