# 網站文章資料_重賞學生飆創意 旺宏科學獎遇疫情不喊卡_4295

## 一、文章基本資料

- **文章ID**: 4295
- **文章標題**: 重賞學生飆創意 旺宏科學獎遇疫情不喊卡
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20200422-1
- **文章簡介**: 【自由時報/記者楊綿傑/台北報導】

武漢肺炎疫情未歇，年輕學子可以趁不適合出門時動動腦做科學專題，還有機會拿高額獎金！第19屆旺宏科學獎宣布今年持續舉辦，即日起至下月底有興趣的高中生都可以報名，今年預估總獎學金高達500萬，甚至有機會被名校看見。

旺宏教育基金會說，為推廣科普教育，基金會往年於競賽報名期間，都偕同科學獎召集委員及評審們拜訪學校及舉辦校園科普講座，但今年受疫情影響，配合中央防疫政策，將原先實體拜訪活動轉至線上進行，透過主動致電，與校長們溝通交流意見。

旺宏教育基金會指出，歷屆科學獎獲獎同學透過實作與參賽的完整歷練，在大學推甄過程中更顯優勢。入圍決賽的同學可以獲得5位科學獎召集委員聯合具名的推薦信函，以上屆獲獎的高三同學為例，逾半數透過特選與繁星計畫進入心目中的理想頂尖大學或科系，以往更有因獲獎申請至海外如美國麻省理工學院（MIT）、耶魯（Yale）及康乃爾（Cornell）大學、法國索邦（SORBONNE） 等大學就讀的例子。

評審團隊方面，旺宏教育基金會說，今年召集委員團隊更為堅強，包括召集人交通大學講座教授吳妍華、交通大學副校長林一平、元智大學校長吳志揚及中正大學校長馮展華，更邀請到中央研究院副院長周美吟加入召集委員行列，與國內頂尖科學專家組成評審團隊
根據競賽資料，旺宏科學獎若奪得旺宏獎及金牌獎，分別可獲得大學4年獎學金新台幣48萬元及40萬元，希望透過高額獎金激勵，吸引優秀學子狂飆科學創意。相關資訊可上旺宏教育基金會網站 查詢。
