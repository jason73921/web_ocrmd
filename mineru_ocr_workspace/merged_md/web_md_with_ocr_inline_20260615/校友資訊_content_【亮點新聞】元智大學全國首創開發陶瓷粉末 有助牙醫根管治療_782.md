# 網站文章資料_【亮點新聞】元智大學全國首創開發陶瓷粉末 有助牙醫根管治療_782

## 一、文章基本資料

- **文章ID**: 782
- **文章標題**: 【亮點新聞】元智大學全國首創開發陶瓷粉末 有助牙醫根管治療
- **文章網址**: https://www.yzu.edu.tw/alumni/index.php/tw/20220223-2
- **文章簡介**: 【中時/李侑珊】



元智大學化學工程與材料科學系孫安正教授研究團隊，研製出齒科活髓治療之覆髓術用生醫陶瓷粉末，成為台灣首位成功案例，有助牙醫從事根管治療。(李侑珊攝)

牙髓發炎進行根管治療侵入性手術，治療時間耗時，又會帶給患者許多不適。元智大學化學工程與材料科學系孫安正教授研究團隊，在新悅美學牙醫診所的臨床研究量能資助下，成功研製出齒科活髓治療之覆髓術用生醫陶瓷粉末，成為台灣首位成功案例，替齒科醫用材料領域，奠下未來自給自足的基礎。

孫安正表示，活髓治療的目的是希望保留牙髓的活性，使牙髓恢復健康，並預防牙髓相關症狀，降低做根管治療（俗稱抽神經）的可能性。

孫安正提到，牙髓組織存在於牙齒內部一個細長的狹窄空間裡，末端與牙齒外血管相連而獲取養份，但是當牙髓組織因外傷或是蛀牙而暴露於口腔，遭到冷熱溫差或細菌攻擊時，將引發牙髓神經的發炎反應，即牙髓炎，這時候就會感到牙疼。

孫安正指出，醫師有了活髓治療這套技術，針對某些較輕微的牙髓炎，可以只要移除發炎部分，得以保留未受感染的牙髓，既可讓醫師省時省力、也可減輕治療痛苦。

孫安正說，治療時，醫師使用機具器械，將包覆在牙髓外的象牙質跟琺瑯質做些許去除後，再將器械深入牙髓組織將發炎的部分袍除。牙髓組織是一條極似橡皮筋的軟組織，被手術袍除牙髓的傷口部位，必須做材料覆蓋的覆隨保護動作，之後再於覆髓處之上，補上樹脂或是銀粉，就算是完成整個療程。

孫安正表示，目前市面上正在商用的活髓治療填充材料有許多種，其中較有效且無毒的就是人造生物陶瓷粉末，具有良好的生物相容性，植入牙髓組織，有助於牙髓恢復健康，並減少根管治療的風險。

即便台灣牙科密度高，人造生物陶瓷粉末卻全部仰賴進口，孫安正因此與新悅美學牙醫診所的醫師群合作，經過2年的研究，成功研製出屬於台灣的齒科活髓治療之覆髓術用生醫陶瓷粉末原型。

孫安正說，目前實驗室研發出表面顏色、硬化時間、機械強度與X光下的分辨率均媲美現有市售材料，於體外實驗還同時確立了生物相容性、生物親和力與生物抗菌性等特點，目前該研究已進入動物實驗階段，雖然離真正商品化還有一段漫長的路要走，但仍是期待團隊的研究成果，能為台灣自有生物陶瓷材料的應用與開發，奠下一個里程碑。
