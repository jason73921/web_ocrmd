# 網站文章資料_96名台灣大學生結束重慶實踐對大陸發展印象深刻_3078

## 一、文章基本資料

- **文章ID**: 3078
- **文章標題**: 96名台灣大學生結束重慶實踐對大陸發展印象深刻
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20180807
- **文章簡介**: 【中新社重慶8月6日電/記者韓璐】

“這次回到台灣後，我會和家人商量，希望能到大陸工作。”6日，即將在重慶結束社會實踐的台灣彰化北斗明道大學大四學生林鈞毅告訴中新社記者，他對再回到大陸工作充滿期待。

當日，第六屆台灣大學生重慶實踐活動舉行總結會，96名在重慶西永微電園、重慶銀行、重慶水務集團、重慶高速集團等12家企事業單位實習的台灣大學生相互交流體會，分享收穫。

“實習是每個學生走向社會、走上工作崗位必不可少的重要環節。所以早在一年前，我就決定要利用暑假來大陸實習，看看大陸跟台灣有什麼（在實習上）不一樣的地方。”來自台灣元智大學的韓凱奇，今年在重慶三峽擔保集團小微擔保項目助理崗位實習。(待續）
