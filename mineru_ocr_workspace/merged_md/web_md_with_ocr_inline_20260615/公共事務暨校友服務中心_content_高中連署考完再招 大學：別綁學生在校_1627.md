# 網站文章資料_高中連署考完再招 大學：別綁學生在校_1627

## 一、文章基本資料

- **文章ID**: 1627
- **文章標題**: 高中連署考完再招 大學：別綁學生在校
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【大紀元2017年02月07日訊】

逾百所高中教育人員連署新考招案，呼籲教完考完再招生，部分大學仍傾向招聯會方案，有校長認為學習未必只能在學校，高中不應該綁架學生那麼多時間。

大學招聯會去年公布考招改革方案，逾百校高中教育人員連署，建議等教完考完再招生，高三寒假可以先考學測，6月中考加深加廣學科，7月再啟動大學推甄申請，以維護高中完整學習。

雖然有部分大學支持高中端的連署提案，但也有大學持不同看法。

元智大學校長吳志揚表示，招聯會的方案已經將現在的3、4月的招生時程延後處理，對學生來說影響應該減少，高中端教師認為招生延到暑假，學生就能完整學習，這只是假設性答案，以台灣現況來看，越晚招生，只是讓學生反覆練習各種考題，且心裡一直懸著，相信很多學生也不贊成。

高中端認為高三下甄選影響到高中生完整學習，但吳志揚指出，根據大學端觀察，上大學後表現優異的很多都是透過甄選錄取的學生，所以完整學習是否影響到學生的表現，他也持保留態度。

在討論考招方案時，吳志揚認為，大家都忽略時代的變遷和科技對教育產生的改變，現在磨課師等線上課程已經是學習新潮流，學生未必要待在教室或學校才能學習，只要上網，隨時隨地都能學習知識，學校不應該再把學生綁在教室或學校。

吳志揚指出，現在勞工都有一週40小時的工時規劃，還有一例一休，但學生每週被綁在學校的時間太多，甚至下課後還要去補習班，「簡直比勞工還過勞」，高中應該培養學生自主和終身學習的能力，讓學生有機會去學習他們感興趣的東西，否則從小綁太緊，到大學已欠缺學習動力。

陽明大學校長梁賡義認為，學生從國小到高中已經學習10多年，只差這半年應該不至於影響太大，應該讓學生有機會放鬆，也能提前準備學大學需要的東西。

成功大學校方也表示，招聯會方案已將招生時程延後，應該會減低對學生學習的影響，且招生若集中在暑假才進行，大學端作業時間也會嚴重被擠壓。（轉自中央社）
