# 網站文章資料_桃園市政府青年事務局「前瞻矽谷 物聯網創新產品交流」_1040

## 一、文章基本資料

- **文章ID**: 1040
- **文章標題**: 桃園市政府青年事務局「前瞻矽谷 物聯網創新產品交流」
- **文章網址**: 
- **文章簡介**: 主旨：本局訂於106年5月23日至26日舉辦「前瞻矽谷 物聯網創新產品交流」，請惠予協助宣導周知並轉知貴屬踴躍報名參加，請查照。

說明：

一、為促進台灣與國際新創團隊接軌，本局邀請6位矽谷創業家至桃園與國內創業團隊同臺交流，帶動桃園地區培育創新產業青年人才風氣，建立國際友好關係，開拓國內青年國際視野。

二、本活動各場次時間及地點資訊如下：

(一)  物聯網創新契機與挑戰

1.     時間：106年5月23日(星期二)下午1時30分至4時50分。

2.     地點：高鐵桃園站前國際金融雙星大樓(桃園市中壢區高鐵站前西路一段286號4樓)。

3.     創新展品介紹：Swift Motion智能勞動防護背心/LoRa解決方案應用/智慧電錶。

(二)  物聯網自造契機：

1.     時間：106年5月24日(星期三)下午1時30分至4時30分。

2.     地點：桃園市政府青年事務局展演廳(桃園市中壢區環北路390號1樓)

3.     創新展品介紹：LoRa解決方案應用/智慧電錶。

(三)  物聯網自造契機(IoT x Maker)

1.     時間：106年5月25日(星期四)下午1時30分至4時30分。

2.     地點：桃園市政府青年事務局展演廳(桃園市中壢區環北路390號1樓)

3.     創新展品介紹：隨手完樂器─智慧電吉他玩具/零時空汙觀測器/打造智慧型吉他效果器。

(四)  物聯網創新跨域連結

1.     時間：106年5月26日(星期五)下午1時30分至4時30分。

2.     地點：桃園市政府青年事務局展演廳(桃園市中壢區環北路390號1樓)

3.     創新展品介紹：智慧電動滑板車/植栽物聯網環境偵測器/令人驚豔的超智慧行李箱。

三、本活動採線上報名方式，可透過兩種管道報名：

(一)  「桃園市TYC物聯網數位學院」線上報名，網址：http://tyc.iot.tycg.gov.tw/iot/index8.asp。

(二)  「活動通」線上報名，網址：http://www.accupass.com/event/register/1705150816501850093312。

四、請貴單位協助宣傳周知，並鼓勵所屬踴躍報名參加。
