# 網站文章資料_東吳與台經院 元智與興采產學合作_370

## 一、文章基本資料

- **文章標題**: 東吳與台經院 元智與興采產學合作
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【大紀元記者黃筠芸、陳建霖台灣連線報導】

東吳大學與台灣經濟研究院20日簽訂合作備忘錄，將在巨量資料經濟專業研究領域等方面合作與交流。另一方面，元智大學與興采集團簽約，合作項目包括學生企業實習、獎助學金以及畢業後的就業問題。

東吳大學與台灣經濟研究院選定「政策研究」、「調查統計」、「產業分析」與「景氣預測及資訊發布」等方面做為研究主題。以東吳大學的研究基礎，結合台經院的研究團隊，協助政府及產業界解決問題，貢獻學術創新研究及人才培育。

台經院院長林建甫表示，東吳大學巨資學院與台經院互惠合作，不管在兩岸、新興市場、產業研究上，透過完整資料庫數據分析，將能持續推動台灣一般經濟及產業經濟研究，促進台灣經濟繁榮，同時為培育國家資料科學家盡一分心力。

另外，元智大學與興采集團簽署「產學合作意向書」，由代理校長徐澤志、管理學院院長曾芳美代表，與興采集團董事長陳國欽簽約。徐澤志表示，雙方簽約內容以學生的企業實習為主，興采集團看重元智培育人才的用心及教學品質，雙方建立產學合作關係，以教學、實習兩面向為重點，為國家社會培育更多優秀人才。

興采集團從事機能性紡織品的研發、製造及銷售。陳國欽表示，非常榮幸能與元智大學師生合作，期許雙方一同貢獻社會。
