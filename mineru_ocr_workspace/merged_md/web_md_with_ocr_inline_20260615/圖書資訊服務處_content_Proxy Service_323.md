# 網站文章資料_Proxy Service_323

## 一、文章基本資料

- **文章ID**: 323
- **文章標題**: Proxy Service
- **文章網址**: https://www.yzu.edu.tw/admin/is/index.php/tw/proxy-service
- **文章簡介**: HTTP proxy service provides cache functions to accelerate service requests by retrieving content saved from a previous request made by the same client or even other clients. Caching proxies keep local copies of frequently requested resources, allowing us to significantly reduce their upstream bandwidth usage and costs, while significantly increasing performance.

Proxy service can be configured manually or automatically. Auto-configuration is recommended as it is more flexible and convenient. The most suitable proxy server will be selected automatically in the auto-configuration mode with better performance and adaptability.

How to configure your proxy settting :

Mozilla / Firefox


Click FireFox  -> Options.


    or  Click Tools -> Options

     


    Click on the Advanced category.

    Click on the Network tab.

    Click on Settings under Connection. 



    Make sure that the checkbox next to Use  "Automatic Proxy configuration URL"  is checked

    In the Address text field, type in http://proxy.yzu.edu.tw/proxy.pac


    Click on OK

    Click on OK 




Internet Explorer 6.x/ 7.x /8.x


Open Internet Explorer

    Click on Tools

    Click on Internet Options
 

    Click on Connections

    Click on the LAN Settings button 



    Make sure that the checkbox next to Use automatic configuration script is checked

    In the Address text field, type in http://proxy.yzu.edu.tw/proxy.pac


    Click on OK

    Click on OK 




Attention: Please don"t click "automatically detect settings".
