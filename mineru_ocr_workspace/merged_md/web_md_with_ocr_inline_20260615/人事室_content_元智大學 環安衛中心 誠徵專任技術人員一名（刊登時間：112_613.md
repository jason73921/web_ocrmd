# 網站文章資料_元智大學 環安衛中心 誠徵專任技術人員一名（刊登時間：112.3.17）_613

## 一、文章基本資料

- **文章ID**: 613
- **文章標題**: 元智大學 環安衛中心 誠徵專任技術人員一名（刊登時間：112.3.17）
- **文章網址**: https://www.yzu.edu.tw/admin/ho/index.php/tw/kan-deng-shi-jian-112-3-17
- **文章簡介**: 一、 資格條件：

1. 國內外大學(以上)畢業。

2. 具環安專業1年，一般行政2年以上經驗尤佳。

3. 有下列情事之ㄧ者，本校不得聘(僱)用：

(1) 性別平等教育法第27-1條第1項至第3項規定不得進用之情事。

(2) 曾犯性侵害犯罪防治法第二條第一項之性侵害犯罪，經有罪判決確定。

(3) 曾經學校性別平等教育委員會(以下簡稱性平會)或依法組成之相關委員會調查確認有性侵害行為屬實。

(4) 曾經學校性平會或依法組成之相關委員會調查確認有性騷擾或性霸凌行為，有終止契約及終身不得擔任教育從業人員之必要。

(5) 曾經學校性平會或依法組成之相關委員會調查確認有性騷擾或性霸凌行為，有終止契約之必要，且議決一年至四年不得擔任教育從業人員，於該管制期間。

(6) 曾經主管教育行政機關認定符合補習及進修教育法第九條第六項第二款之情事。

(7) 曾經主管教育行政機關認定符合補習及進修教育法第九條第六項第三款之情事，且於該認定一年至四年不得聘用或僱用期間。

(8) 曾經主管機關認定有教育人員任用條例第31條所定情事。

二、 專長條件：

1. 具環保(乙級廢水處理專責人員以上、乙級廢棄物處理專責人員以上、室內空氣品質專責人員(必)、貯存系統污染監測人員(必))、職業安全衛生管理員以上(必)等環安衛相關證照優先及實務經驗者佳。

2. 具撰寫計畫及簡報有經驗者佳，並熟悉學校行政工作者優先。

3. 具團隊精神、工作態度積極、細心負責、配合度高、抗壓性佳、善於溝通協調，能獨立處理負責業務者。

4. 熟悉Microsoft Office軟體應用。

三、 工作內容：

1. 秘書相關行政事務處理、各式報告簽呈之撰寫、預算之編列、帳務管理、專案規劃、文件雙語化工作之執行。

2. 負責環境保護相關業務為主及協助職業安全衛生管理相關業務。

3. 環安衛相關行政業務。

4. 其他主管交辦事項。

四、 工作地點：本校校區。

五、 待遇：依本校相關人事規定標準給付 36,000~41,000元，希望到職日期：112年4月1日。

六、 應徵方式

1. 有意願者，請於112年3月31日(星期五)前（以郵戳為憑掛號郵寄），備妥履歷表(含大頭照)、自傳、最高學歷證件(國外學歷須經外館認證)、專長證照與成績單，及其他有利應徵審查之相關佐證資料(以上均為影本)，信封上請註明應徵『環安衛中心專任環安衛技術人員』，寄至320315桃園縣中壢市遠東路135號　元智大學環安衛中心收，由主聘單位先行書面審查。

2. 連絡人：張蕙安小姐，電話：03-4638800轉2065。

七、 條件符合者，送達人事室彙整後，E-mail或電話通知筆試（科目：公文、資訊概論、英文、環境保護及職業安全衛生相關專業）及安排口試時間。

八、 備註

1. 本徵才訊息同步刊登於各大就業網站。

2. 請註明白天聯絡電話、手機號碼及電子郵件信箱帳號，檢附之證件不齊者恕不受理報名。

3. 先行書面審查，合者擇優面談，未獲遴用者，恕不再通知及退件。

4. 本項職缺除正取名額外，另視情況增列備取人員 1-2名，候補期間自甄選結果確定之翌日起算 3 個月內有效。
