# 網站文章資料_油漆式速記多國語言雲端系統_849

## 一、文章基本資料

- **文章ID**: 849
- **文章標題**: 油漆式速記多國語言雲端系統
- **文章網址**: https://www.yzu.edu.tw/library/index.php/tw/201805-4
- **文章簡介**: 內容簡介：

油漆式多國語言雲端學習系統包含相當豐富多元的版本，除了有配合政府最新南向政策的東南亞四國語言之外，更包含外籍生來台灣求學所需要的華語學習版本。我們在這個系統中還增加了許多老師與學生互動的後台機制，方便老師管理帳號、查詢成績及考場管理，在未來機制中也會加入聊天室、學生PK單字大賽等有趣互動功能。目前已開放試用版本(區分為七類)：

A.全腦(圖像+句型+情境)學習類：全腦速記新多益字彙、全腦速記英檢初級字彙、全腦速記英檢中級字彙、全腦速記學測指考必考字彙、全腦速記統測必考字彙、全腦速記美國大學學術水平測驗(SAT)必考字彙、全腦速記越語初級(北越)字彙、全腦速記泰語初級檢定字彙、全腦速記印尼語初級檢定字彙、全腦速記西班牙語初級檢定字彙、全腦速記法語初級檢定字彙、全腦速記德語初級檢定字彙、全腦速記韓語初級檢定字彙、全腦速記日語N5級檢定字彙。

B.升學類版本：學測指考必備(8500字)、統測必備單字全壘打版(8000字)、統測必考單字(3000字)、國中必備基礎單字、國中會考必背單字(2500字)、美國研究生管理科考試必考字彙(GMAT)、美國研究生入學考試必考字彙(GRE)、美國大學學術水平測驗必考字彙(SAT)

C.英語檢定類：超右腦全民英檢初級檢定字彙、全民英檢中級檢定字彙、全民英檢中高級檢定字彙、超右腦新多益檢定必考字彙、商用英文多益必考字彙、商用英文新多益分類必考字彙、雅思英語(IELTS) 必考字彙、托福英語(TOEFL) 必考字彙 、軍事檢定英語(ECL) 必考字彙、英檢必備片語(含狄克生片語)、公職人員檢定英文。

D.專業英語類：會展英語、電商英語、醫護英語、生技英語、物理治療英語、運動休閒英語、餐旅英語、博弈英語、科技英語、商用英語、財會金融英語、電腦英語、觀光領隊導遊英語、烘培英語、航空英語、化學英語、資管英語、國際政治英語。

E.第二外語類：日文N1級檢定、日文N2級檢定、日文N3級檢定、日文N4級檢定、超右腦日文N5級檢定、醫護日語、觀光日語、商用日語、餐旅日語、超右腦印尼語初級檢定、超右腦越南語(南越)初級檢定、超右腦越南語(北越)初級檢定、超右腦泰語初級檢定、超右腦馬來西亞初級檢定、超右腦韓語初級檢定、超右腦西班牙語初級檢定、超右腦法語初級檢定、超右腦德語初級檢定、超右腦俄語初級檢定。

 F.外國人學繁體中文系列：美國人學繁體中文初級檢定、越南人學繁體中文初級檢定、泰國人學繁體中文初級檢定、印尼人學繁體中文初級檢定、馬來西亞人學繁體中文初級檢定、日本人學繁體中文初級檢定。

G.外國人學簡體中文系列：美國人學簡體中文初級檢定、越南人學簡體中文初級檢定、泰國人學簡體中文初級檢定、印尼人學簡體中文初級檢定、馬來西亞人學簡體中文初級檢定、日本人學簡體中文初級檢定。
