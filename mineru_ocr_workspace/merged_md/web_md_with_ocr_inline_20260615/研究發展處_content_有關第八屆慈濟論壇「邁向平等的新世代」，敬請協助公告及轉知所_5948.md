# 【第八屆慈濟論壇】邁向平等的新世代

根據世界經濟論壇所發布的《2023 年全球風險報告》（Global RiskReport 2023）指出，全球已進入 COVID-19 之後的「新常態」（NewNormal）；疫情尚未完全遏止，2022 年初又爆發俄烏戰爭，更加重糧食及能源危機。

加上長年極端氣候的影響，導致經濟發展嚴重失衡，世界財富與權力掌握在極少數人手上。由巴黎經濟學院世界不平等實驗室所發布的《2022 年全球不平等報告》（World Inequality Report）指出，全球最富有的 10%人口掌握全球 76%的財富，最貧窮的 50%人口只擁有 2%的財富。財富的極度分配不均，導致社會動盪不安、歧視頻仍，「降低不平等」成為人類在二十一世紀所面臨最大的挑戰之一。

為促進環境、經濟、社會之永續發展，聯合國提出十七項永續發展指標，其中第十項「降低不平等」（Reduced Inequalities）指出，增強並促進所有人的社會、經濟與政治包容性，無論其年齡、性別、身心障礙、種族、族群、族裔、宗教、經濟或其他任何區別，都應該受到平等對待。經由適當的政策，確保機會平等，鼓勵社會創新與社會企業發展，促進全球合作，期能邁向平等永續的公正社會。

本屆慈濟論壇以「邁向平等的新世代」為主題，首先針對就業、居住、無家者關懷、能源、糧食等面向，邀請全球於此領域之專家、學者及實踐者共同交流研討，以環境、經濟、社會之永續發展為面向提出建言。
