# 網站文章資料_【衛保組公告】訊息轉知-桃園衛生局公布111年端午應景食品抽驗結果_2557

## 一、文章基本資料

- **文章ID**: 2557
- **文章標題**: 【衛保組公告】訊息轉知-桃園衛生局公布111年端午應景食品抽驗結果
- **文章網址**: 
- **文章簡介**: 大家 好

因應端午節來臨，桃園市衛生局為把關端午應景食品，自4月起至各超市賣場、傳統市場、食品原料行、粽子製造業及專賣店等場所，抽驗各式粽類成品及配料（生鮮肉、鹹蛋黃、蝦米、花生、乾香菇、菜脯、甜餡料、調味醬）等總計108件，檢驗項目包含微生物、硼酸、乙型受體素、防腐劑、蘇丹色素、漂白劑、甲醛、甜味劑及黃麴毒素等項目，其中1件完整包裝冷凍肉粽檢出微生物超標與規定不符，已移源頭衛生局辦理。詳細抽驗結果公布於衛生局網站及桃食安心資訊平台(http://food-safety.tycg.gov.tw/)「新聞稿」區供民眾參考。
      本次抽驗108件產品中，1件完整包裝冷凍肉粽檢出腸桿菌科超標，不符食品中微生物冷凍食品類衛生標準，違反食品安全衛生管理法第17條規定，衛生局已請販售業者將違規產品下架，移請製造商之管轄衛生局針對製程品管及貯運衛生後續查辦，並要求改善，避免不良情事發生。


      衛生局提醒民眾，隨著天氣日漸炎熱及梅雨季節剛結束，暖溼是細菌的溫床，食物保存及衛生尤其重要，購買粽子時，應選擇信譽良好之廠商，並留意製作及販售環境衛生，熟食者購買後充分加熱再食用；另選購食材時以來源、標示清楚的食品優先選購，不買外觀過於鮮豔、有刺鼻味的食品，且依外包裝標示之儲存條件妥善保存及儘速食用完畢，以確保產品新鮮及食用安全。如有因消費問題或任何爭議，可撥打0800-285-000或1950消費者服務專線，向衛生局或本市消費者服務中心申訴。
