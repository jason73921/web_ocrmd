## 一、文章基本資料

- **文章標題**: 研揚、遠傳、雲協、大同世界科技 深耕5G時代LPWAN技術應用	
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20181222
- **文章簡介**: 【CTIMES/SmartAuto 報導】

研揚科技，昨20日與雲協、遠傳、大同世界科技共同主辦「雲協LPWAN SIG研討會」，結合雲端、端點及邊緣產品，聯合打造5G物聯網新時代。會中研揚科技針對AI邊緣運算及智慧路燈進行演講與展演。

研揚科技近年來針對AI人工智慧及物聯網相關產品開發研究不遺餘力。會中研揚科技客戶委製產品處協理賴立凱就部門最新研發的AI Core X來進行產品應用說明。AI Core X是款Intel Movidius Myriad X Mini-PCIe模組，搭配UP squared來進行AI邊緣運算。
同時AI Core X可搭配Intel OpenVINO SDK，並支援TensorFlow及Caffee frameworks來進行深度學習。目前已經使用在工地安全、智慧零售、交通等應用環境中，另外一個在現場展示的物聯網應用則為智慧城市中的智慧路燈的應用。目前智慧路燈已經安裝在台北市多個路段，例如︰仁愛路、信義路、高架橋、大安森林公園、健康公宅、快速道路等地方。

研揚所研發的智慧路燈，可利用遠端控制平台，結合雲端運算、AI邊緣運算加入分析，讓整套系統的資訊傳輸及分析更為快速準確，可提升用電效益，更能提供能源使用及環境影響的數據參考資料。

這次研討會除了研揚之外，微軟、遠傳、安謀等大廠，也在會中分享各家廠商在物聯網世代的應用及解決方案。讓與會人員在物聯網應用中，又加深一層的了解與發想。在這次更新「物聯網生態圈手冊2.0」中，除了有更多重量級的業者加入外，同時許多國內知名大專院校也加入物聯網研究行列，例如：台大、成大、中興、台北、中正、元智、大同、逢甲等十餘所大專院校來共襄盛舉，旨在結合產、官、學、研等不同角度，將物聯網的發展往更深及更廣的方向前進，也為國家培育更多相關人才與提供更多資源給教育單位。（待續）
