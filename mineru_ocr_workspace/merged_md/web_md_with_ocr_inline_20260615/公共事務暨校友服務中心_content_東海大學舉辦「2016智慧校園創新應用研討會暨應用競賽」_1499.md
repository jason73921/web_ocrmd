# 網站文章資料_東海大學舉辦「2016智慧校園創新應用研討會暨應用競賽」_1499

- **文章全文**: 【中央社】 


東海大學昨(29)日舉辦「2016智慧校園創新應用研討會」,及「2016智慧校園創新應用競賽」。本次「2016智慧校園創新應用研討會暨應用競賽」由東海大學、台灣中部科學園區產學訓協會、台灣臺中軟體園區發展產學訓聯盟、國立中山大學雲端計算與服務跨校資源中心共同主辦。 


本次研討會受到各界關注與重視，由東海大學校長王茂駿主持，包括科技部中部科學工業園區管理局陳銘煌局長、教育部資通訊軟體創新人才推升計畫何建明顧問、經濟部加工出口區管理處中港分處 梁又文處長、臺中市政府數位治理局籌備處蕭景燈資訊長等貴賓都特別蒞臨指導。 


東海大學校長王茂駿表示，隨著網路與數位技術的快速發展，像是物聯網、巨量資料、雲端運算、還有人工智慧、IR及VR等都在開發應用，加上友善介面 及不同的智慧裝置，包括智慧產品、智慧機械、智慧工廠、智慧家庭等，對於工作、學習、生活、學習還有社交都帶來極大的方便與改變，也帶來新的需求與機會。這些對於台中軟體園區的發展也帶來新的契機，本次就產學訓聯盟的夥伴，大家就新的技術、系統進行經驗交流。 


近年物聯網、巨量資料、雲端運算、綠能管理、智慧城市等議題為各界關注之焦點，各大專校院順應此趨勢的發展，乃積極結合相關資訊技術規劃並逐步打造智慧校園，以提供更便利、智能應用及雲端平台運算與儲存的各項資訊服務和大數據校務研究分析。 


2016智慧校園創新應用競賽，自9月開放全國大專校院在校學生報名，共有38校83個隊伍325人參與競賽，11月29日進入決賽入圍共20校29個隊伍112人，競爭激烈，結果出爐由元智大學 coding fighter奪冠獲得六萬元獎金、高雄應用科大360環環相Call亞軍獲四萬元獎金、而修平科技大學校園抓漏尖兵獲得季軍獲得兩萬元獎金。 


本研討會特別針對智慧校園相關技術與創新應用進行研討與案例分享，議題包含：「物聯未來，智慧製造進行式」、「智慧校園空間資訊平台應用」、「大數據應用與分析」、「學習平台發展趨勢：雲端化、行動化、數據分析」、「智慧物聯網之網路架構大未來」、「大數據時代高速儲存的挑戰」、「教育部資通訊軟體創新人才推升計畫–雲端種子教師課程發表」、「智慧校園案例分享–行動裝置管理」、「智慧校園競賽作品觀摩」等。智慧校園的發展應與學生的創意發想與實作結合，可創造更便利並符合需求的資訊服務融入校園生活，甚至擴大至智慧城市應用。 


本研討會邀請產、官、學、研、訓各界的資訊主管、相關領域的學者專家，以及全國大專校院具有創新能量與技術實力的學生競賽隊伍，透過最新技術及創新應用進行分享，增進彼此經驗交流，期望透過本會議激發更多的創新能量與產學媒合，在出席來賓與講者經驗分享和討論中，協助各校加速推動智慧校園建設與服務。
