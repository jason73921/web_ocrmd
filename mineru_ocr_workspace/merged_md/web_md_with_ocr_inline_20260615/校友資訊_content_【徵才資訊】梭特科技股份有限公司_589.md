【公司簡介】

梭特科技股份有限公司創立於民國99年，致力於半導體精密設備研發/製造/銷售與服務，專注在Pick & Place 和 Die Attach 取放技術之鑽研，Chip Sorter 和 Die Bonder 產品應用在LED 、LD、IC 後段製程 (Backend / Assembly Process ) ，在既有取放技術的架構下，不斷延伸與客戶、通路商與同業之間的合作，開發出各式創新和性價比高的設備，志在成為世界級取放自動化設備及技術的專業領導提供者。

R&D Software Engineer R&D軟體工程師

【工作任務】

在R＆D軟體的工程團隊，我們是負責設計、開發、優化機台程式，以滿足客戶要求。 我們必須依客戶要求，提供必要支持。 我們與製造、業務和技術支持單位等團隊合作，以縮短產品交貨時間，幫助進行模組或系統鑑定，並減少故障排除和系統恢復的時間成本。

【職位描述】


能與外部單位溝通，了解及整理軟體需求。

設計、開發，優化、維護機台軟體（通信，自動/順序控制和GUI）。

設計並實施不同角色且多面向的單元測試。

針對客戶應用程序、製造/工程團隊要求的問題提供快速應對措施。

實施系統性能評估和撰寫軟體設計文件。

撰寫機台軟體功能之技術手冊。


【工作特點】 

透過專案管理手法與敏捷框架的工作方式，讓每個人在工作中都可以建立自己的成就感與工作價值、個人與團隊的共同成長。


Value is Key
