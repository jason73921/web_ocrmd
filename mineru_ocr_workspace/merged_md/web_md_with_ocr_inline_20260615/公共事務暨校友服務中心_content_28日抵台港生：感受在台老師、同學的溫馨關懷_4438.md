# 網站文章資料_28日抵台港生：感受在台老師、同學的溫馨關懷_4438

## 一、文章基本資料

- **文章ID**: 4438
- **文章標題**: 28日抵台港生：感受在台老師、同學的溫馨關懷
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/2020062907
- **文章簡介**: 【大紀元2020年06月28日訊】

臺灣開放11地境外生入境，4名來台就讀的香港籍畢業生今天（28日）傍晚搭機抵台，遺憾無法參加大學畢業典禮；雖然在等待過程焦慮不安，卻充分感受在台老師、同學的溫馨關懷。

中央流行疫情指揮中心日前宣佈，優先開放11個低風險國家、地區應屆畢業生分批返台就學，而從26日起陸續有學生來台，辦理畢業等相關手續，今天傍晚也有4名香港籍學生搭機入境。

據中央社報導，中央大學莫姓畢業生接受中央社記者訪問時表示，在疫情期間等待4個月重回臺灣，內心很憂心，對於無法參加大學畢業典禮覺得很可惜。

元智大學張姓女畢業生被問到等待來台時間有多久，毫無猶疑喊出“140天”，對於“無限期等待，感到相當焦慮”，因為有很多包括租屋、畢業等程式都必須要在臺灣處理，只想趕快回到臺灣。

她說，非常感謝房東沒有催著要趕快處理租房事情，也非常體諒，而一些老師、同學們也不斷透過網路期盼她“趕快回來”，讓她覺得很溫馨；在完成離校手續後，就會先回香港工作。

她也說，在臺灣讀了4年書，卻沒有能參加大學畢業典禮的確相當遺憾，但事情發生有種種因素，在這段時間也經歷很多，香港僑生也在這段時間結合力量，相互關注、扶持，也得到很多。

元智大學鐘姓畢業生說，在等待返台過程中內心很焦慮，也很不安；對於無法參加畢業典禮覺得很可惜，“也無法請學校再補一次畢業典禮”；這次疫情非預期，只能接受，“雖然沒有參加畢業典禮，但是回憶還是在的，與同學、朋友、老師感情也不會因此沖淡。”

至於問到未來是否會留在臺灣工作，她透出興奮表情說“希望留在臺灣”。

臺灣開放11地境外生入境，首波8名學生26日搭機抵台，而今天有4名香港籍學生來台，教育部也特別在桃園國際機場接機大廳設置輪值單一櫃檯，協助確認學生入境後順利銜接交通接送到防疫旅館，避免境外生四處散開造成防疫破口。

中央流行疫情指揮中心日前宣佈，優先開放11個低風險國家、地區應屆畢業生分批返台就學，包括越南、香港、澳門、泰國、帛琉、澳洲、紐西蘭、汶萊、斐濟、蒙古、不丹，符合資格共2238名。
