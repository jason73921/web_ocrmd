# 網站文章資料_歡迎教職員參加「111年度私校教職員線上自主投資宣導說明會」_509

## 一、文章基本資料

- **文章ID**: 509
- **文章標題**: 歡迎教職員參加「111年度私校教職員線上自主投資宣導說明會」
- **文章網址**: https://www.yzu.edu.tw/admin/ho/index.php/tw/111-2
- **文章簡介**: 財團法人中華民國私立學校教職員退休撫卹離職資遣儲金管理委員會 函

主旨：函請貴校教職員參加「111年度私校教職員線上自主投資宣導說明會」，至紉公誼。

說明：

一、為使私校教職員更加瞭解自主投資及退休理財之規劃等相關資訊，本會將於111年7月26日及7月28日舉辦線上宣導說明會，歡迎私校教職員踴躍報名參加。

二、本會採Teams視訊方式，共分三場次以不同主題分眾舉辦， 說明如下：

(一)111年7月26日(二)上午10:00-11:30，敬邀服務5年內之教職員參加，主題為「對於私校退撫儲金您夠瞭解了 嗎？-如何善用自主投資平臺及對帳單掌握未來的退休金」。

(二)111年7月26日(二)下午2:00-3:30，敬邀對投資組合有興趣之教職員參加，主題為「私校退撫儲金投資組合介紹 與理財觀念分享」。

(三)111年7月28日(四)下午2:00-3:30，敬邀規劃退休之教職 員參加，主題為「退休私校教職員的專屬保障-以分期請領規劃退休生活」。

三、有關報名及相關訊息請至本會官網（右下方「相關連結」 之「111年度宣導說明會線上報名」），報名截止日期為 111年7月15日，如有任何疑問，歡迎來電詢問02-2396-2880#18。

四、請貴校人事單位轉知教職員，廣為周知，毋任感荷。
