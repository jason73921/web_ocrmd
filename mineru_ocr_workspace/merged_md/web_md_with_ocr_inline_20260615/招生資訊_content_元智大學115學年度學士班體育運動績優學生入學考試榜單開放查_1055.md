# 網站文章資料_元智大學115學年度學士班體育運動績優學生入學考試榜單開放查詢、暨志願選填作業通知_1055

## 一、文章基本資料

- **文章ID**: 1055
- **文章標題**: 元智大學115學年度學士班體育運動績優學生入學考試榜單開放查詢、暨志願選填作業通知
- **文章網址**: https://www.yzu.edu.tw/admissions/index.php/tw/115sport04
- **文章簡介**: 1.依據本校115年4月30日 115學年度第12次招生委員會議決議。本錄取名單如有爭議，以本校招生委員會之決議為準。

2.榜單查詢網址：https://exam.yzu.edu.tw/NewNetapply （點選「招生管道」→榜單查詢）查詢。

3.個人成績查詢網址：https://exam.yzu.edu.tw/NewNetapply （點選「招生管道」→查詢報名結果與成績→輸入身分證字號及密碼→查詢成績）查詢。考生個人成績單一律網路查詢，本校不另郵寄紙本成績單。

4.複查成績時間：115年5月4日至115年5月5日截止收件（以郵戳為憑），逾期概不受理。複查成績辦法請參閱簡章第27頁。「成績複查申請書」

5.選填志願序：請於規定時間115年5月6日至115年5月7日下午17:00止填寫「就讀志願序申請表」（簡章第29頁）以電子郵件傳送至aaregi@g.yzu.edu.tw
 主旨請註明: 115運動績優就讀志願序申請表+運動項目+名次+姓名。
 主旨範例: 115運動績優就讀志願序申請表 籃球 第1名 王○○。

※未於115年5月6日至115年5月7日下午17:00止，回傳「就讀志願序申請表」者，視同放棄錄取資格，不予分發，考生不得異議，且不得以任何理由要求補救。
