# 新竹市政府 公告

發文日期：中華民國108年2月14日

發文字號：府授環綜字第10800309451號

附件：

主旨：公告新竹市政府遴選環境影公開接受公會行政機關公益團體等之推薦。

依據：新竹市政府環境影響評估審查委員會專家學者遴選作業要點第2點。

公告事項：

一、公告事項：公告新竹市政府遴選環境影響評估審查委員會專家學者委員，公開接受公會行政機關大專院校學術研究機構、環保公益團體等之推薦，及受理推薦時間。

二、推薦資格：本府環評專家學者委員須具備環評相關學術專長及實務經驗，並符合下列資格之一者：

(一)從事環評相關項目之技師，滿5年以上者。

（二）大專院校教授、副教授。

（三)曾任各機關環評審查委員會委員，滿2年以上者。

(四)行政機關簡任級以上從事環評相關項目職務，滿3年以上者。

（五)曾任學術研究機構環評相關項目研究員。

（六)曾任環保公益團體負責人，滿3年以上者。

(七)其他具有特殊專長，經專案認定者。

三、推薦期限：即日起至本（108）年3月15日止；推薦表請上本市環境保護局網站（http://www.hccepb.gov.tw）下載，並請於期限內填妥及簽名後函復該局。

四、相關推薦事宜請洽本市環境保護局綜合計畫科曾思嘉技士，電話：（03）5368920分機1004，傳真（03）5368894。
