- **文章簡介**: 【經濟日報/記者彭慧明/即時報導】

Google Cloud今日宣布推出「雲端人才搖籃計畫」，將透過雲端科技賦能台灣人才國際競爭力。首波與元智大學、東海大學以及高雄科技大學共同推動雲端學程，協助院校導入Google Cloud Platform（GCP）應用平台，完善雲端教育環境。此計畫預計在三年內協助 1,500 位學生取得國際認可的Google Cloud雲端基礎認證，獲得認證的學生將列入Google人才庫，成為Google Cloud 合作企業夥伴未來徵才的優先考量對象，助力產學接軌，不斷為台灣產業注入人才活水。

除規劃上述課程資源外，Google Cloud將透過與合作學校導入GCP應用、建置數位課程的雲端資源、促進資訊系統雲端化等作法，協助學校進一步建置雲端教學環境。例如Google Cloud 協助元智大學研發AI雲端服務，未來可以在校內提供大數據應用服務結合物聯網IoT的智慧應用；東海大學則將學生服務系統移轉至Google Cloud，並運用Google Cloud雲端運算與AI工具優化並開發創新線上學生服務，協助合作院校邁向智慧校園。（待續）
