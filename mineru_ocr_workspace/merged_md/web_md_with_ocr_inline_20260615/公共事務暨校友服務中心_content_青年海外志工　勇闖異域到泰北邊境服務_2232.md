# 網站文章資料_青年海外志工　勇闖異域到泰北邊境服務_2232

## 一、文章基本資料

- **文章ID**: 2232
- **文章標題**: 青年海外志工　勇闖異域到泰北邊境服務
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20170921-1
- **文章簡介**: 【蘋果日報】

教育部青年署指出，今年暑假教育部補助82個海外青年志工隊中，有20隊不畏艱辛揹起行囊前往泰北邊境服務，其中玄奘大學志工服務隊透過中華救助總會引薦，前往泰國山地村的海育中學，服務隊以過去3年服務於泰國美華國小的工作模式，轉移至此，從事教學服務(華語、數學和社會)及環境美化(如彩繪校舍牆面)。中華大學志工服務隊首次與華人磐石領袖協會合作，前往光華中文學校，學校位於泰國清萊府萊掌村，為一所曾經面臨廢校命運的學校，但因華人磐石領袖協會協助重新運作，並規劃10年計畫，期待透過培育當地自身的能量，減少對外依賴，最後能夠自行掌理校務，獨立運作。台中教育大學和元智大學的志工服務隊，今年來到泰國大同中學已進入第10年，長期的深耕及陪伴讓大同中學日漸成長茁壯。雖然2個團隊所前往的服務據點及期程相同，但彼此運用所學專長，區隔服務的主軸和對象。青年署指出，青年海外志工服務需要更多充滿熱情的青年一起加入行列，更多詳細資訊請上教育部青年發展署官網及iYouth青年國際圓夢平台（https://iyouth.youthhub.tw/main.php）查詢。（生活中心／台北報導）待續
