# 網站文章資料_培育新南向人才 元智增設英語專班_1226

## 一、文章基本資料

- **文章ID**: 1226
- **文章標題**: 培育新南向人才 元智增設英語專班
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【中央社記者許秩維台北5日電】 


因應全球化和政府新南向政策，元智大學今天表示，106學年各學院將全面增設英語專班，新增130個名額，同時規劃東南亞中心，並設置亞太文化學程。 

元智大學校長吳志揚表示，現在許多國際或在地大廠溝通仍以英語為主，學生必須具備全球化概念，加上政府力推新南向政策，包括新加坡、馬來西亞、菲律賓、泰國等國，在品牌、宣傳上都以英文為主，因此落實英語學習還是最基本。 


為培育新南向人才，吳志揚表示，過去元智首創雙語教學，管理學院也設立英語專班，106學年起，各學院將全面增設英語專班（學士英語學程），預計新增130個名額，加上管院原有的50個名額，總計180個名額，未來還將建構各學院英語專般的「國際業師系統」，並規劃海外企業實習。 


除了英語專班外，配合新南向政策，元智人文社會學院也規劃東南亞中心和亞太文學習文化學程。吳志揚指出，東南亞是我國推動華語文教育的重點地區之一，元智東南亞中心的師資來自海內外，希望以教學人力和文化的優勢，全力推動東南亞的研究與教學。
