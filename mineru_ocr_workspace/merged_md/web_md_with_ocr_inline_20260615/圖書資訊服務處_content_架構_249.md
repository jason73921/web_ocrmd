- **文章全文**: 無線網路，讓行動學習好方便

你曾經夢想在二館旁邊的草地上，一邊享受著暖暖的冬陽，一邊上網找資料，或者在微風徐徐的午後，帶著你的Notebook或Tablet PC，坐在六館前的廣場，恣意地遨遊在網路的國度裡嗎？這已經不是夢想，從今天起，在元智校園的每個角落，你都可以無限（線）上網，歡樂學習。
 元智校園網路已經遍佈各館各樓層，但在室內、外開放空間內，如果也能夠使用網路，對全校教職員及學生將更方便。因此，資服處從90學年度開始，即積極規劃建構校園無線網路。除了各館公共教室、圖書館、行政辦公區、及活動中心已佈設無線網路外，戶外區域包括圓環及操場區亦已架設完成，加上部分系所自行建置的區域，在校園內幾乎已經達到無縫隙漫遊。無線科技日新月異，採漸進式的規劃較符合經濟效益，資服處分階段逐步投資建置以及積極爭取與廠商合作畫。

無線網路架構：


整個無線網路包含安全認證設備；RADIUS Server ；Access Point(AP)以及網路連結設備。廣佈於各棟建築物內外，涵蓋了整個校園。詳細分述如下:









無線網路示意圖









Access Point
 元智無線區域網路，目前以802.11b為主，連線速度達到11Mbps，未來隨著WLAN相關技術的標準化，陸陸續續會導入校園內，其傳輸速率將達到54Mbps，甚至100Mbps。為整合異質性AP，在規劃上並未硬性規範AP的特性，只要是符合Wi-Fi規格的產品，皆可整合至網路內。

安全機制：
 因 Access Point 的 WEP安全機制，其本身設計並未考慮週到，有很大的安全顧慮，且是以手動方式來更新code，對規畫者與使用者而言，極為不便。另外雖可以考慮以網路卡卡號來杜絕非法使用者，但無線網路講究行動無礙，隨時連線。在管理上是相當難以執行的一環。因此規畫以闡道連結方式，將所有使用者的連結，全數導向Secure Server/Access Manager設備上，此種方式較令人質疑部份為傳輸流量(Throuput)大小，所幸規畫時即考量到這因素，所選擇的產品，在設計採分散式的架構，且有相當好的應用經驗。當網路架構很小時，只要一台Secure Server，同時擔任認證與重新導向的功能。可是當負載增加時，只要增加Access Manager設備以分散流量的負載，所需的設備成本是相當的少，而且Access Manager只扮演重新導向的角色。安全認證仍有Secure Server 來執行。在認證機制上，支援AD/NT Domain；LDAP；802.1X；RADIUS等方式，規畫以減少使用負擔與管理工作為原則，採取結合NT Domain與RADIUS認證機制。使用者只要以電子郵件的帳號密碼通過認證即可，不需額外申請使用許可。

網路漫遊：
 方便好用、設定簡易是使用者對行動網路最基本的要求。您只要自備個人電腦和無線網卡，不需要申請即可使用無線網路。可以在校園內任何一個地方上網，想要更換地點時，不會因為連線之基地台的改變而導致訊號中斷。在複雜的校園環境中，並非將SSID值設成一樣，就可達到漫遊的目的，還要規畫到當使用者所使用的網路設定，是可以在任何系所都可以使用的，而且非私有的IP，因此，規畫不需任何修改，甚至可忽略這一環節，只要開機通過身份認證即可使用網路。

無限應用：
 元智校園無線網路已經陸續建構完成，搭配校園主幹Gigabit 相輔相成，未來將配合無線網路科技發展，整合系所自行建置的部分，繼續擴充與提昇效益。在各校紛紛標榜無線網路覆蓋率百分之百的時候，只希望帶給元智一個先進、實用的網路學習環境！
