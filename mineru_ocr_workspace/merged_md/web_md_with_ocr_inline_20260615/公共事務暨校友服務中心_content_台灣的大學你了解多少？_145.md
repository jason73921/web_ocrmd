# 網站文章資料_台灣的大學你了解多少？_145

## 一、文章基本資料

- **文章ID**: 145
- **文章標題**: 台灣的大學你了解多少？
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【青塔網】

很多人都知道寶島台灣有不少的好大學，然而對於台灣的大學，我們中的大多數並不了解，這兩年越來越多的台灣高校開始到大陸招生，大家也開始更多關注台灣省的大學，而且相信台灣的大學到大陸招生的規模會逐年擴大，本期青塔就來扒一扒祖國寶島台灣的大學。

台灣的大學將大幅減少 

目前台灣共159所大學，其中公立51校、私立108校。然而台灣的大學在未來一段時間將大幅減少，之前台灣《中國時報》已經報導，台灣少子化現象嚴重衝擊台灣的高等教育，到2023年大學新生只剩18萬人，台當局教育部門決定加速推動大學整併及退場，估計8年內砍掉近50所公私立大專院校，目前三分之一大學即將面臨整併或退場，但是這也意味著招生壓力不斷顯現的情況下，各大台灣高校將來勢必會不斷擴大在大陸的招生規模。

在台灣上大學“十分”容易 

以前台灣高校錄取率低，許多落榜考生為了能上大學不得不復讀重考。現在大學增多，一些私立學校甚至招不滿學生，應屆畢業生只要想讀書，哪怕成績再低也肯定有大學可上，願意重考的學生自然就大幅下降了，這也造成了台灣真正願意复讀的學生已經越來越少了，2010年的時候，考生最低原始總分只需10分(3科)就可申請上大學，因而被坊間譏笑為在台灣上大學“十分”容易。當年，台灣高校錄取率高達97%。

在台灣醫學類專業超熱門 

台灣的高等學校一般分為普通大學及技職院校,普通大學的學生來源以普通高中生居多，而技職院校則以高職(中專、中職)與綜合高中生為主，也有招收少數的普通高中學生,所以通常普通大學的學生在基礎學科方面會比技職院校好上不少，技職院校生則是實作技能較佳,但整體上普通大學會比技職院校的學生總體素質稍好一些。在台灣大部分的國立學校都比私立學校要好上一點，但是一些老牌的私立大學也非常不錯，另外在台灣類似於醫學中醫、牙醫、藥學等專業的分數都是高得嚇人，只要有牽扯到生醫相關的專業，分數都普遍比較高，例如去年台大醫、陽明醫、台大牙醫、陽明牙醫都須滿級分(75級分)，其他學校醫學系、牙醫系至少73級分，頂尖高分群有2599人次搶進11個醫學系，頂尖考生可謂搶著進醫學專業，這和國內會出現北大醫學院錄取分數線不如北大本部的情況截然不同。

五大、四中、雙科 

在台灣學子眼中及一般民眾普遍認知來說，在台灣最有名的國立學校為五大(台灣大學、清華大學、交通大學、成功大學、政治大學，簡稱台清交成政)、四中(中央大學、中山大學、中興大學、中正大學，簡稱中央中山中興中正或央山興正)、雙科(國立台灣科技大學、國立台北科技大學，簡稱台科北科)，另外陽明(陽明大學)、台北(台北大學)、台師(台灣師範大學)，皆有各自擅長的專業，在台灣的名氣也極響。

台灣的文理科名校情況 

在台灣文科類較強的大學有台灣大學和政治大學，接下來是新竹清華大學、新竹交通大學與成功大學,後面三所大學主要強項還是在理工科系，商學科係不多，因此商學類的資源相對弱勢一點，但是也不差。另外中央大學、高雄中山大學、中興大學、台北大學和中正大學在文科方面也相對比較強勁，都有各自較為突出的文法商學科系。再接下來就是一些老牌私立大學，如輔仁大學、東吳大學、淡江大學、東海大學、逢甲大學、實踐大學、銘傳大學、世新大學等，這些學校由於校史較悠久，所以在知名度與校友等方面比台灣的新興的國立大學更好一點。

從理工科的角度看，綜合實力最強的同樣是台灣大學，當然其他高校也都各有擅長的學科，在台大後面則是新竹清華大學與新竹交通大學，新竹交通大學在電氣工程、電子工程與信息技術專業上更有優勢，其他方面是新竹清華佔優勢，接下來則是成功大學，在理工方面緊追新竹清華交通大學，另外陽明大學、中央大學、高雄中山大學、中興大學、中正大學、台灣科技大學也都非常不錯，四所中字輩大學一樣都有各自突出的理工科系，陽明大學則是相對專注在生物、醫學等專業領域。另外台灣科技大學為技職院校的龍頭，在工科領域方面也十分優秀，另外技職院校的二三甲，台北科技大學及雲林科技大學在台灣的工科評價也相當不錯，最後以私立老牌大學來說，理工科較好的為長庚大學、元智大學、中原大學、淡江大學、逢甲大學和大同大學等。
