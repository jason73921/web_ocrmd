# 網站文章資料_是德EXM無線測試儀簡化物聯網裝置驗證與製造測試_108

## 一、文章基本資料

- **文章ID**: 108
- **文章標題**: 是德EXM無線測試儀簡化物聯網裝置驗證與製造測試
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【EET電子工程專輯】

是德科技(Keysight Technologies Inc.)日前宣布旗下EXM無線測試儀增添新的WLAN 802.11ah和802.11af功能，以協助工程師在產品開發的設計驗證和製造階段，對更多物聯網(IoT)無線裝置和格式進行測試。此外，這些新技術大幅擴充了EXM現有的無線連接、蜂巢式以及應用與發展範圍，以支援更多元的無線裝置與標準。

台灣是德科技總經理張志銘表示：「物聯網產業正蓬勃發展，產業分析師預測到2020年，這項新的連網技術將可連接280至330億個物件。EXM具備多樣化功能，讓裝置製造商得以滿足市場對最新的無線裝置和智慧型裝置(IoT裝置)日益成長的需求。藉由增添全新功能，EXM無線測試儀再度證明我們的平台可全面保障製造商之設備投資，並可隨著其需求變化，輕鬆進行平台擴充。」

此外，是德科技並宣佈與元智大學電機學系簽署合作備忘錄，共同成立「積體電路量測實驗室」，結合企業及學界的研發能量，促成產學合作機制，攜手強化積體電路設計與量測技術發展。

是德科技行銷處副總經理羅大鈞強調，「是德科技多年來在教育領域投注資源不遺餘力，使科技成為提升教育品質的重要助力。」

元智大學在台灣是德科技的審慎評估下，共同成立「積體電路量測實驗室」列為台灣是德科技2015年度的重要計畫之一。 藉由成立「積體電路量測實驗室」，是德科技與元智大學將定期協商研究方向，訂定產學合作計畫及相關事宜，讓元智大學學生在正式進入職場前，有更多的實機操作以及磨練的機會此外。台灣是德科技也將支援元智大學電機系教育培訓，提升教學品質及優化專業科技人才。
