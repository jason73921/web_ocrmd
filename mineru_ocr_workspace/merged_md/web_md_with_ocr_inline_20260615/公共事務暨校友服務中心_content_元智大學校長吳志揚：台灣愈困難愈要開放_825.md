# 網站文章資料_元智大學校長吳志揚：台灣愈困難愈要開放_825

## 一、文章基本資料

- **文章ID**: 825
- **文章標題**: 元智大學校長吳志揚：台灣愈困難愈要開放
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【中評社/記者黃文杰】 


私立元智大學校長吳志揚接受中評社訪問表示，台灣應該要有更開放的心態，讓全世界華人“幫你生小孩”，按照人口比例原則，你（台灣）的聰明才智不可能超越世界強國，政策開放，才能吸引人才，愈是困難時刻，愈要開放，如果逆境時還選擇緊縮，套用閩南語就是：“等死”。 


吳志揚，美國芝加哥大學數學系博士，曾任位於嘉義的中正大學校長，今年2月1日正式就任元智大學校長。 


因為“吳志揚”三個字，跟國民黨前桃園縣長、現任不分區“立委”吳志揚同名同姓，還沒有上任，就鬧了不少笑話，不過校方安排校長與媒體聚會，“數學家吳志揚”展現不凡的數字概念。

中評社問兩岸學術交流部分，特別是過去擔任南部公立大學校長，現在換到北部私立大學，怎樣看新政府上台之後的陸生政策？

吳志揚說，他算是很早就進入大陸，進行學術互訪交流，還摸索出一套“T”字策略。 


英文字母“T”策略，就是大陸沿海，由南的廣東中山、福建廈門到北京清華甚至哈爾濱，各主要大學拜訪交流，接著以長江軸線，包括重慶、武漢到南京上海，正好構成“T”字策略。 


他說，新政府上台，對於陸生政策應該不會立即做出重大改變，陸生跟陸客政策，兩者不必然相等，就等大陸先觀察新政府的態度與作法。 


中評社跟數學家校長訪談，會發現這位校長引述的例子，都有數字故事。 


吳志揚站在人口比例，來看兩岸高教交流或競爭，大陸是你台灣的60倍，如果台灣智力提升3倍，也還是20比1。
