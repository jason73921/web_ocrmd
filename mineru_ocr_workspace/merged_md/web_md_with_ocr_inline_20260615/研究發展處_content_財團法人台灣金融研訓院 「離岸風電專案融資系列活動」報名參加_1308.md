# 網站文章資料_財團法人台灣金融研訓院 「離岸風電專案融資系列活動」報名參加_1308

## 一、文章基本資料

- **文章ID**: 1308
- **文章標題**: 財團法人台灣金融研訓院 「離岸風電專案融資系列活動」報名參加
- **文章網址**: https://www.yzu.edu.tw/admin/rd/index.php/tw/?option=com_content&view=article&id=1308
- **文章簡介**: 主旨：訂於本(106)年11月14日(星期二)~11月16日(四)舉辦「離岸風電專案融資系列活動」，敬邀 貴屬相關人員踴躍報名參加。

說明：

一、台灣雖擁有打造離岸風電廠所需具備的各種地理環境條件，惟離岸風力發電所需資金需求龐大，須有金融體系的支持，開發始得以成功，其中「專案融資」正扮演不可或缺的角色。有鑑於此，本院與北陸能源股份有限公司及玉山能源股份有限公司共同舉辦「離岸風電專案融資系列活動」，透過論壇及研習班(共兩場次)方式，邀請加拿大北陸能源公司及國際知名綠能公司等專家來台，針對離岸風電專案融資實務、專案合約、避險機制及國際規範等議題交流分享，盼以國際經驗幫助台灣金融業者更深入瞭解專案融資，使離岸風電產業能獲得更多金融支持，活絡台灣綠能經濟發展動能，成就金融與產業雙贏新格局。

二、舉辦時間：

*第一場(免費論壇):2017年11月14日（星期二）09:00-16:00，此場次僅開放網路報名，提供中英同步口譯。

*第二場(收費研習班):2017年11月15日（星期三）~11月16日（星期四）09:00-16:00。

三、本活動詳細參加對象、所需費用、議程安排及報名方式相關說明請詳附件實施要點及報名表。
