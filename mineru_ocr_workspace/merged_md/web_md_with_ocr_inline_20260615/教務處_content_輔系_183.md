如何得知輔系的相關法規？

 




請參考「元智大學各學系學生修讀輔系辦法」

 






請問大六的學生可否以未修完輔系的原因延長修業年限？

 




不可以，不得在法令規定年限外請求留校補修輔系科目與學分。

 




 

如何辦理輔系申請？

 


 

至個人Portal「雙主修、輔系」頁面申請。



 

審核流程說明 (Application Procedures )



學生線上申請(Student Application Online)
→ 學生所屬系秘書 (Dept. Secretary)
→ 學生所屬系主任 (Dept. Chair)
→ 申請學系秘書 (Double-Major / Minor Dept. Secretary)
→ 申請學系系主任 (Double-Major / Minor Dept. Chair)
→ 註冊組複核(Double Review by Registration Section)
→ 學生查詢確認申請結果(Double Review by Registration Section)





 

如果到大四還修不完輔系學分，但想畢業該怎麼辦？

 


 

至個人Portal「雙主修、輔系」頁面申請撤銷。
