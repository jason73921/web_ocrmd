# 網站文章資料_中央研究院「109年中央研究院院區開放參觀活動」_3408

## 一、文章基本資料

- **文章ID**: 3408
- **文章標題**: 中央研究院「109年中央研究院院區開放參觀活動」
- **文章網址**: https://www.yzu.edu.tw/admin/rd/index.php/tw/?option=com_content&view=article&id=3408
- **文章簡介**: 主旨：

檢送本院「109年中央研究院院區開放參觀活動」海報，惠請張貼並宣傳，請查照。

說明：

一、本院於今（109）年10月31日（星期六）上午9時至下午4時 舉辦院區開放參觀活動，今年主題科普演講將由本院陳建仁院士，

於上午10時30分在人文社會科學館3樓國際會議廳主講「科學與人文的結合：臺灣COVID-19防疫的成功模式」。

二、今年因應新冠肺炎疫情，大幅減少實體活動場次，並採事先報名制，報名者才能進場館參觀。活動網站於9月23日（星期三）上線，

並將於10月5日（星期一）上午10時正式 開放線上報名。

三、活動當日欲以遊覽車方式至本院之團體，請至活動網站進行「遊覽車登記」（登記期限至10月26日止）。

四、詳細資訊請參考活動網站：https://openhouse.sinica. edu.tw。

五、檢附活動海報電子檔、活動當天遊覽車路線圖及臺北市各景點大客車停車位一覽表。
