# 網站文章資料_臺灣大專校院專任教師年齡結構趨向老化_2920

## 一、文章基本資料

- **文章ID**: 2920
- **文章標題**: 臺灣大專校院專任教師年齡結構趨向老化
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20180606-1
- **文章簡介**: 【中國新聞網】

中新社臺北6月4日電 臺灣當局教育主管部門統計處近日公佈“2017學年大專校院專任教師概況”，顯示年齡50歲以上者占全體教師的比例達54.4%，較2001學年的18.3%增加36.1個百分點。

綜合台當局教育主管部門官網資訊及《中國時報》等臺灣媒體4日報導指出，按年齡觀察，2001學年臺灣大專校院專任教師年齡大多集中在35歲至39歲，2006年集中遞增為40歲至44歲，2011年再增加為45歲至49歲，2017學年續後移至50歲至54歲。教師年齡分佈結構趨向老化，主因師資員額增長停滯，原有教師年齡日漸增長，補進年輕教師人數有限所致。

臺灣元智大學校長吳志揚認為，教師年齡結構老化確實衝擊臺灣高等教育，但目前臺灣大專校院教師薪資仍較低，若情況再不改善，想延攬年輕、優秀人才的難度會越來越高。(待續)
