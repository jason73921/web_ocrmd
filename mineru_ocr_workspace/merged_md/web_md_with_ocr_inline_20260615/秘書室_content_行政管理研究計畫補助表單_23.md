# 元智大學行政管理研究計畫補助合約

甲方：

乙方：

茲經雙方協議，乙方所提之「 」(編號： )

由甲方在 學年度「行政管理研究計畫 項下予以補助，其條件如下：一、本計畫執行期間自民國年　月日至　年月日止，期滿不得以任何理由申請延長。

二、本計畫補助金額共計新台幣元，其詳細項目所附補助經費明細表為準，由乙方核實動支，不得移作他用；其動支情形，甲方得隨時派員查核。單據應黏於黏存單，註明科目及用途，並經計畫主持人簽章後，送請甲方辦理核銷手續。至計畫期限屆滿後，如有結餘，均不得再行支用。

三、乙方在計畫執行中，如因實際需要必須變更預算時，須提出變更項目對照表述明理由，徵得甲方書面同意後為之，但以一次為限。

四、乙方應每季提報計畫執行進度，經甲方審核及彙整後提行政會議報告，甲方須瞭解本計畫之執行情形時，乙方應盡力協助，詳予說明，並提供參考資料。

五、乙方應於本計畫執行期限屆滿後一個月內備妥研究成果報告及摘要一式五份送甲方辦理結案手續，如不照前述規定辦理，甲方得自次學年度起不再給予其他研究計畫案之補助。

六、在計畫執行中及研究完成後，未經甲方同意，乙方不得將執行情形及研究成果對第三者發表。如擬在國內外刊物發表者，其文稿須先經甲方書面同意。如不照上述規定辦理，甲方得不再給予其他專題研究之補助。

七、乙方如未能履行上述各項條件之一者,甲方得停止補助並追加還已核銷之款項，又如發現預期結果無法達成或研究工作不能進行時，得隨時結束計畫，並收回未支用之款項。

八、本合約一式五份，甲乙雙方各存一份，本校會計室、計畫主持人所屬系所及學院各存一份。

九、本計畫應於執行結束後公開展示具體成果。
