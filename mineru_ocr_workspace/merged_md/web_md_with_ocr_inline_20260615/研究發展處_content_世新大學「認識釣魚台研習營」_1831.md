# 網站文章資料_世新大學「認識釣魚台研習營」_1831

## 一、文章基本資料

- **文章ID**: 1831
- **文章標題**: 世新大學「認識釣魚台研習營」
- **文章網址**: https://www.yzu.edu.tw/admin/rd/index.php/tw/?option=com_content&view=article&id=1831
- **文章簡介**: 主旨：檢送本校辦理107年度「認識釣魚台研習營」，請惠予協助公告並鼓勵貴屬高中及大專院校教師參加，敬請查照。

說明：

一、 臺灣鄰近海域風波不斷，為加強國民對臺灣海域爭議與臺灣漁民處境之理解，以及強化師生對國際局勢的認識，謹訂於本（107）年7月28日(六)、29(日)兩天假蘇澳區漁會3樓多媒體室辦理「認識釣魚台研習營」。 

二、 本次研習營議程規劃豐富，不僅認識釣魚台議題、漁民作業處境、東亞四海國際局勢之分析，另有蘇澳樂齡非洲鼓隊表演以及在地走訪行程，精彩可期。詳細議程請參見附件。 

三、 參與兩日研習營為免費，並提供午餐與茶點。然收取活動費用540元，僅付在地鮮食晚餐與兩日保險，敬請事先報名。報名網址如下: https://goo.gl/vSydY1。 

四、 本次研習營全程參加者，頒發國高中小老師16小時教師研習時數、大專院校老師16小時研習證書。其中研習時數以全國教師在職進修資訊網之裁定為準。 

五、 活動聯絡人：卓淑惠，電話： 02-2236-8225#3012。信箱：diaoyutai.tw@gmail.com。
