- **文章簡介**: 【經濟日報 曹松清】
 
叡揚資訊偕同台灣微軟日前舉辦暑期Chatbot開發黑客松，邀請大專院校學生參與。此活動透過工作坊實際操作和競賽演練，使參賽者與專家零距離互動，深入瞭解Chatbot原理、機制和多面向的應用，培養專業技能，成為未來職場方向，或求職創業的加分利器。

元智大學資管系大四學生汪文豪、陳宜湞、陳侑辰與鄭淂元，參加「叡揚資訊暑期Chatbot開發黑客松」，脫穎而出獲得「第一名」、「評審青睞獎」、「人氣獎」及「實用獎」共四個獎項，是參賽團隊中唯一獲得四獎項的團隊，成績斐然；另外一組團隊資管系大四呂芷婷、杜旻庭、陳詩婷及大三的呂欣澄、曾謙文以作品「做伙食」獲得亞軍殊榮。

獲得冠軍團隊的鄭淂元表示，團隊開發的聊天機器人「Davis帶位仕」是透過叡揚的iota C.ai對話服務平台與微軟提供的Azure服務為主而開發的聊天機器人，團隊以MVC開發的餐廳管理介面伺服器架設於Azure上，並使用Azure SQL Database進行資料的儲存及處理。

獲得亞軍團隊的資管系呂芷婷表示，作品「做伙食」以食譜為主題，提供使用者方便的操作介面，讓機器人推薦符合各種需求的食譜。呂芷婷說，這個APP的使用方式與一般的對話問答不同，「做伙食」讓使用者可自行輸入冰箱食材，透過AI的斷詞系統，分析使用者的意圖，同時利用同義詞管理，讓使用者能快速的找到適合做的料理。（待續）
