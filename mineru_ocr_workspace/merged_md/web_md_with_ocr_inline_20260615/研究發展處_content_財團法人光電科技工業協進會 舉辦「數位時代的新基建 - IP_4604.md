說明 : 

一、本會(PIDA)與台灣智慧資本(TIC) 共同舉辦「數位時代的新基建 - IP 思維提升論壇」，以「中美地緣政治下的 IP 未來趨勢」，「台灣學研界IP循環現況」，「從永續經營-倡 議營業秘密管理的創新」，「台灣產業界改變的企圖」等議題為主。 

二、「數位時代的新基建 - IP 思維提升論壇」資訊

1. 日期 : 2022年6月9日(星期四) 

2. 時間 : 下午 14:00~下午17:00 

3. 地點 : 緯育 TibaMe 台北訓練中心 

4. 地址︰台北市中山區南京東路三段219號5樓  (捷運綠線南京復興站，8號出口)

三、目的與效益 :

1. 宣傳智慧財產經濟內循環之強大潛能,讓台灣成為智財強國。     

2. 真實呈現台灣智財經濟內循環的真實量能,守住並獲取台灣本來就該有的智財經濟。

3. 此論壇擬連結產學研等單位,促進跨界交流。同時引起台灣在世界上長期是智慧財產弱勢的關注。

四、敦請協助轉發此論壇訊息於研究發展處或相關處室,並鼓勵相關人員參與此論壇。

五、「數位時代的新基建 - IP 思維提升論壇」活動內容如附件所示。

六、本活動將遵循中央疫情指揮中心與地方政府之防疫政令辦理
