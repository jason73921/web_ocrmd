# 網站文章資料_【亮點新聞】元智大學首辦校內露營 觀星、攀樹活動一籮筐_955

## 一、文章基本資料

- **文章ID**: 955
- **文章標題**: 【亮點新聞】元智大學首辦校內露營 觀星、攀樹活動一籮筐
- **文章網址**: https://www.yzu.edu.tw/alumni/index.php/tw/20230502-1
- **文章簡介**: 【中時 / 蔡明亘】

桃園市元智大學首創在校園操場搭35個帳篷，今明2天邀教職員生、校友、眷屬及觀音根滿慈善基金會、桃園家扶中心等，齊聚元智，除了維繫彼此情誼，凝聚向心力及認同感外，也讓沒參加過露營的人，體驗露營樂趣。活動中也簽署「元智低碳十宣言」，身體力行，創零碳綠能生活。



元智大學4月29至30日首度辦理校內露營活動，安排多元豐富活動。（元智大學提供／蔡明亘桃園傳真）



校方盼透過露營活動，維繫彼此情誼，凝聚向心力及認同感外，也讓沒參加過露營的人，體驗露營樂趣。（元智大學提供／蔡明亘桃園傳真）

元智大學體育室、學務處、秘書室、校友總會及福委會共同規畫在校露營，於操場搭35個帳篷，約150人參與。由元智大學原資中心邀請校內原民生共同規畫安排原住民迎賓舞及原住民傳統美食外，搭配親子運動等，讓元智教職員生及校友們在忙碌生活中，品味元智之美，一同駐足校園，拉近彼此距離，共度2天歡樂時光。



2天露營活動精采又豐富，有原住民迎賓舞蹈開場，還有動手體驗製作搗麻糬與竹筒飯。（元智大學提供／蔡明亘桃園傳真）

元智大學校長廖慶榮表示，2天露營活動精采又豐富，有原住民迎賓舞蹈開場，動手體驗製作搗麻糬與竹筒飯，還有羽球及匹克球親子體育競賽等，除運動強身外，更可促進親子關係。元智不但重視大學永續發展與社會責任，為發揮社會影響力，更積極推動USR計畫，以善盡社會責任，與在地連結，培育優秀人才。

廖慶榮指出，活動中也簽署元智低碳十宣言，報到時即讓參與者簽署承諾，內容包括構建環境教育校園，創造校園永續文化、制訂環境保護及廢棄物減量，落實校園生態環境等，每個人都有能力拯救地球，只要在生活上做些改變，累積起來就有巨大影響力。

廖慶榮說，登山社特別與桃園市童軍會呂理祿訓練員共同辦理攀樹及走繩活動，結合環境生態及ESG永續理念，將「親近樹、認識樹、善待樹」的概念帶給同仁及眷屬，增進大家對森林永續的瞭解，大家在攀爬上樹之餘，還能夠對於樹木及生態環境更加了解。

晚間則安排元智大學通識教學部星象專家俞伯傑老師，介紹4月的星象與星座故事，躺在地上或坐在椅子上，一邊欣賞美麗的星空，一邊聽故事；隔天收帳前，邀請參與者進行校園淨化環境，不留下垃圾，有美好的環境才有美好的生活。

參與活動的元智大學張姓同仁表示，雖在元智上班，但第一次在操場上露營，既新鮮又有趣，是難得的回憶，元智校園優美，不用到山上就能感受，真的太幸福了。

機械系校友謝愉隆分享，畢業13年重回母校參與露營活動，非常興奮及開心，除了體驗原住民活動外，親子體育競賽更可增進家人間的情感，透過與元智人一起玩樂，除了凝聚母校的向心力，更增進彼此情誼。值得一提的是，對於低碳宣示覺得深具意義，帶著家人一同宣誓，透過平日的小動作，讓地球永續生存，為地球盡份心力，真是難能可貴的經驗。
