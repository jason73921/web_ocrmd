桃園市元智大學以培育具全球移動力的數位跨域人才為目標，深化學生數位競爭力，近期引進Google Cloud雲端學程計畫，盼強化學生技術與專業能力。元智大學校長廖慶榮表示，經系統性人才培育，此計畫到第3年預估可培育至少500人通過基礎認證。

元智大學未來將採用Google Cloud Platform（GCP）培養學生成為數位人才。廖慶榮指出，Google Cloud將提供課程內容、教學簡報、測驗題庫、創新實驗與全球雲端認證計畫等資源，完成課程並通過測驗的學生可獲結業證照，列入Google數位人才探索計畫，將成未來Google Cloud與其合作夥伴的徵才優先考慮對象。

校方說明，Google Cloud雲端學程計畫分3年執行，第1年雙方將完成Google Cloud雲端計算基礎課程的所有規畫與種子教師培訓；第2年正式開辦學程，培育對象以元智學生優先，同時進行2個進階課程，包含Cloud Digital Leader（CDL）與Associate Cloud Engineer（ACE），進階課程適合有志於數位應用領域發展者。

廖慶榮表示，經系統性人才培育，此計畫第3年可培育至少500人通過基礎認證，同時元智將成Google Cloud的生態系夥伴，盼透過不同產學合作模式，深化人工智慧與機器學習應用。另此計畫也涵蓋智慧校園建置，將建立數位課程雲資源、資訊系統雲端化，最後將校園當成課程專案實驗場域，支援AI人才培育實戰能力所需環境，對學生的國際化技術力與就業競爭力帶來極大助益。
