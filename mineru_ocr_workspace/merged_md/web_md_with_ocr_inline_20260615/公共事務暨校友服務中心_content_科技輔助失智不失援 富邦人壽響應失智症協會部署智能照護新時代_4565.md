# 網站文章資料_科技輔助失智不失援 富邦人壽響應失智症協會部署智能照護新時代_4565

## 一、文章基本資料

- **文章ID**: 4565
- **文章標題**: 科技輔助失智不失援 富邦人壽響應失智症協會部署智能照護新時代
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20200916
- **文章簡介**: 【商業週刊】

失智症的發生率隨著年齡增長而增加，有鑑於此，長期關注失智高齡議題的富邦人壽，積極響應台灣失智症協會倡議「失智照護科技協作」之理念，攜手舉辦「2020年台灣失智症聯合學術研討會─失智照護科技應用之超前佈署」，以深入淺出的觀點分享：善用智能科技的應用，讓高齡者受到更適合的照顧，照護家屬也能緩解身心，打造出友善的照護環境。

「失智症不是一個家庭的事，而是全體民眾應共同面對的社會議題！」對於失智高齡議題自來十分關注的富邦人壽總經理陳俊伴表示，希望社會各界能夠及早正視即將來襲的高齡海嘯共伴效應帶來的失智議題。臺灣失智症協會賴德仁理事長也提到，「失智者的診斷及認知促進非常重要，尤其臺灣是科技島，藉由與各界科技人士合作，共同用科技來照護失智者們。」而衛生福利部長期照顧司祝健芳司長表示，「目前失智症服務量能已大幅提升，未來政府將再加強服務品質的提升，並持續改善支付制度、資訊系統與行動方案，而照顧與科技的結合也是衛生福利部很重視的一環。」

創新輔具加溫科技 打造有感的身心治療術
陪伴失智者們治療是段漫漫長路，增加照護人力並非最好解方，加上2025年台灣即將邁入高齡化社會，智能科技輔助照護已是刻不容緩的議題。所幸，隨著日新月異的科技發展，物聯網（IoT）、虛擬實境（VR）、藍芽定位（Beacon）及穿戴裝置等新興智能科技的運用，讓智慧醫療變得可行，而這些科技輔具也皆以病人為中心進行優化，像是陪伴機器人更加智慧，在聊天過程中打動長者的心，實現懷舊治療的智慧化，達到延緩失智症的可能性。當失智者願意放下對科技的戒心，走進運動與社交生活，他們的身心便開始活絡，達到運動加認知訓練的治療目的。

而科技輔具不只鼓勵長者們提升認知訓練，也關懷心理健康；以板橋榮家失智專區為例，便是結合非藥物活動與高齡智能科技，打造實景擬真的「懷舊火車站」，彷彿親臨火車站，坐在車廂座位上，望著由智能科技所投射出來的火車窗外實景，一解長者鄉愁之餘，具溫度的科技也撫慰了長者心。

臺大人工智慧與機器人研究中心主任傅立成則分享人工智慧將可預測阿茲海默症，透過大數據及人工智慧，可將長者過往的生活資料數位化後作為輸入，再藉由機器人用輕鬆聊天的方式，達到延緩失智症的可能性。以樂齡科技健身房為例，元智老人福祉科技研究中心主任徐業良，分享樂齡科技是如何輔助長者多元互動運動及認知促進的同時，他也不忘強調，「智能科技真的只是配角，照護者才是最好的設計者！」（待續）
