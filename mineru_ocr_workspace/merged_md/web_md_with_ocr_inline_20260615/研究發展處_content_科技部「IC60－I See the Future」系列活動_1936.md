# 網站文章資料_科技部「IC60－I See the Future」系列活動_1936

## 一、文章基本資料

- **文章ID**: 1936
- **文章標題**: 科技部「IC60－I See the Future」系列活動
- **文章網址**: 
- **文章簡介**: 主旨：為慶祝積體電路發明60週年，本部啟動IC60系列活動，敬邀各界人士共襄盛舉，請查照轉知。

說明：

一、本（2018）年適逢IC發明60週年，本部規劃「IC60－I See the Future」系列活動，邀請各界人士認識IC，啟發青年學子對此領域的興趣，讓台灣重要的半導體產業得以傳承。

二、旨揭系列活動內容如下：

(一)「IC DAY廠商參訪」:串連晶片中心與上中下游廠商，策劃一日IC體驗參訪活動，開啟年輕學子對於IC產業的好奇與想像。

(二)「IC60大師論壇」：以「回顧過去，啟發未來」為主軸，邀請多位對IC發展歷程有貢獻的大師、前輩，分享經驗及傳承；並透過跨界論壇，探討半導體新興發展趨勢，鼓勵青年投入科研領域。

(三)「IC60特展」：以創意展覽呈現積體電路發展歷史、科技影響力及未來展望，特展內容包含「積體電路發展歷史區」、「半導體製程體驗區」、「產業合作夥伴區」三大展示區域。

(四)「IC常設展」：9月1日起在新竹科學園區奈米電子大樓建置IC常設展展場，包括開放申請參觀半導體無塵室，做為台灣半導體科技的科普中心，並開放國中以上的學校或團體預約參觀。

(五)「IC之音專訪CEO」：專訪見證台灣半導體電子產業發展的重要人士，也回顧備受懷念的相關領域典範人物、關鍵推手。8月起陸續在空中頻道播出，讓這些當事人帶領聽眾親身體驗台灣奇蹟的誕生過程。

(六)「IC60主題影片」：以「IC產業在台灣」、「IC產業發展回顧」及「IC現在與未來之應用」等主題製作多支IC影片，帶領國人一窺60年來台灣IC產業發展的艱辛歷程及豐碩收穫。
