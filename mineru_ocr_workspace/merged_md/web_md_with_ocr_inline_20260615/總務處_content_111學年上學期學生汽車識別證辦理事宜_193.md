# 網站文章資料_111學年上學期學生汽車識別證辦理事宜_193

## 一、文章基本資料

- **文章ID**: 193
- **文章標題**: 111學年上學期學生汽車識別證辦理事宜
- **文章網址**: https://www.yzu.edu.tw/admin/ga/index.php/tw/111-1
- **文章簡介**: 各位同學好！

擬辦理111學年上學期(111/08/01～112/01/31)汽車識別證者，請先行上網填妥申請資料並列印申請表。111年6月14日起，攜申請表及相關證件至聯合服務櫃檯繳費辦理。

辦理程序：

(1)上網 ( http://140.138.35.130/GeneralOffice/park/ ) 填妥汽車識別證申請表印出簽名，並貼妥駕照、行照影本。

(2)至聯合服務櫃檯，由受理單位核對、登錄資料。

(3)申請人完成繳費。

(4)申請人領取車輛識別證。

 

聯合服務櫃檯現場收費時間規劃如下：

 

非於上述收費時間辦理者，請於週一至週五上班時間，先與總務處警衛隊劉逢相先生聯繫辦理時間（03-4638800分機2261），俾便配合辦理。

備註：

1、本次所辦車輛識別證使用時間為111/08/01～112/01/31，逾期自行作廢。

2、請備妥相關資料及現金：一般生收費4,400元、在職碩專班2,800元，可託人代為繳費辦理。

3、建議以「班」為單位彙整資料，統一辦理停車證，以利繳費並迅速取得車證使用。

4、入校車輛相關規範及收費標準請詳參本校校區車輛管理辦法。

 

總務處 敬啟 111.06.10公告      111.09.12再次提醒
