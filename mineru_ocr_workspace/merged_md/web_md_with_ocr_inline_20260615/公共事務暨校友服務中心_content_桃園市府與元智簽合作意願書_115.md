# 網站文章資料_桃園市府與元智簽合作意願書_115

## 一、文章基本資料

- **文章ID**: 115
- **文章標題**: 桃園市府與元智簽合作意願書
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【中央社記者卞金峰桃園20日電】

桃園市政府今與元智大學簽訂多項合作意願書，內容含「大數據中心與智慧型政府建構計畫」、「國民中小學經典閱讀」、「公務人員才能培育」及「網路美術館及讀書會雲端服務計畫」等議題。 


儀式由桃園市長鄭文燦及元智大學校長張進福共同主持，希望藉由元智大學豐富的學術資源，協助桃園市政府推動各項市政。 


「大數據中心與智慧型政府建構計畫」由市府研究發展考核委員會與元智大學大數據與數位匯流創新中心共同合作，鄭文燦表示，桃園市需要轉型為具未來性的智慧城市，大數據的思維與技術正是市府政策推動所急切需要的。 


他也舉美國紐約市的市民熱線311為例，說明大數據分析應用如何將民眾的不滿需求轉化為施政重點，讓政策擬定與推動的精準度大幅提升。市府也擇定研考會所負責的1999與市政信箱作為優先合作與釋出的資料項目。 


鄭文燦指出，在確保民眾個資安全的前提下，藉由與元智大學大數據分析團隊的合作，協助市府分析出民眾最關切的市政問題，以及各項議題的趨勢與可能的關聯性，不斷提升市政品質與市民滿意度，讓民眾對市政更加有感。 


桃園市府教育局與元智大學通識教學部合力推動經典閱讀，推薦桃園經典好書100本；桃園市將導入元智大學推展「社區行動閱讀教室」資源，舉辦經典講座、成立親子共讀經典讀書會等，以整合大學資源，充分發揮「大手牽小手」的精神。 


為培育具有卓越管理及變革領導能力的高階文官，以及強化公務人員數位資訊運用能力，桃園市府人事處與元智大學終身教育部合作辦理「公務人員才能培育計畫」，針對市府所屬機關、各區公所的高階文官與公務人員，分別開設領導才能發展及數位資訊運用專班，期能透過相關訓練課程，優化市府公務人力素質，提升服務品質，進而強化地方治理效能。 


桃園市府文化局與元智大學大數據與數位匯流創新中心將共同合作「雲端服務計畫」，包括推動雲端讀書會、網路美術館典藏資料儲存的服務、研擬數位落差相關研究議題，透過網路雲端服務，市民可於任一時間、地點瀏覽桃園市豐碩的藝術成果。
