# 網站文章資料_台商吳志榮：從臺灣公務員到行走兩岸的創業者_1473

## 一、文章基本資料

- **文章ID**: 1473
- **文章標題**: 台商吳志榮：從臺灣公務員到行走兩岸的創業者
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【華夏經緯網】

“不管是遇到挫折還是失敗，沒有創過業都將是我的遺憾。”

2007年，在經濟危機席捲全球的形勢下，吳志榮毅然決定辭掉臺北市教育局的工作，投身創業。“如果不創業，很多理想就無法實現”吳志榮就是這樣，從臺灣一個普普通通的公務員，成為行走兩岸的創業者。 

那一年， 和許多貪圖穩定的年輕人不同，天生就酷愛冒險的吳志榮，決定不再安於現狀。而在旁人眼中，這是一場毫無勝算的賭博。

吳志榮說，19歲他就進入臺北市教育局工作，因為被安排分管科普，便與科學結下了不解之緣。在他的眼中，科學並不是一個學科，而更像一種生活態度。

“我們的目的不是為了讓孩子拿高分，而是要讓他們真正愛上科學。”吳志榮創辦的力翰教育十分注重培養孩子的動手能力，他認為解決未來社會的需求，必須從科學教育入手。

“兩岸一脈相承，如果能夠服務兩岸的孩子會讓我感到更幸福。”2012年，公司邁入第五個年頭，經過大量的市場調研，吳志榮決定把他在臺灣的創業模式和教育理念帶到大陸來。

一步一腳印，走進了大陸校園 

萬事開頭難，回憶起在臺灣和大陸這兩段創業經歷，吳志榮依然感慨萬分，兩岸的創業環境、文化氣氛大不相同，“但無論在臺灣還是大陸創業，一開始要想讓別人接受我們，都是不容易的。”

吳志榮說，科學教育是比較新興的行業，它沒有之前的模式可以遵循，要想讓兩岸的中小學接受他們的教育理念，他們必須四處走訪、調研，說服校長、老師引進外面的師資。“在臺灣創業初期，300多家學校，我們都是這樣一步一腳印走出來的。”

“與臺灣不同，在大陸創業要跟著政策走。”吳志榮說，特別是教育這個領域，要想讓自己的公司順利進駐到大陸公辦學校，必須付出比在臺灣更多的努力，因為這其中不僅要說服校長和老師，更要得到教育局的高度認可。

想要脫穎而出，終要回歸產品。吳志榮早年在臺灣元智大學讀博士研究生的時候，就積累了豐富的科研成果，臺灣多年的創業經歷更讓他們的教學產品日趨完善，“在臺灣，從創業初期到現在，力翰可以說是唯一把科學教育企業化經營的成功案例。” 吳志榮的頗感欣慰。

由於獨特的教學方式大大提高了孩子們的學習積極性。吳志榮的公司度過了創業初期的迷茫，得到了大陸公辦學校的認可，也成為了行業中不可替代的佼佼者，“人家願意讓你進駐校園，這意味著你的師資水準一定遠超校內的科學老師。”

大陸市場有多大？ “驚為天人”

“驚為天人”吳志榮用這四個字形容初來大陸的感受。

大陸市場有多大？“我們在大陸用兩年的時間，就達到了在臺灣花十年的規模。”吳志榮說，市場環境正是大陸與臺灣創業最大的差異。

早在五年前，臺灣的市場就已經逐漸飽和。在吳志榮的計畫之內，全臺灣有三分之二的小學都已經是他們的合作學校，但吳志榮說，臺灣小學只有1000多家，而大陸中小學有21萬家，這代表著巨大無窮的市場。 

這期間，他們還去過深圳和上海，但幾經輾轉，吳志榮最終選擇今年要將公司總部移到武漢。武漢歷來被稱為“九省通衢”之地，得天獨厚的地理位置，讓它成為中國經濟地理的“心臟”。吳志榮認為，武漢雖然不是一線城市，但四通八達、優勢凸顯，況且這裡的房租是上海的一半，每年可以省下近100萬的成本。
