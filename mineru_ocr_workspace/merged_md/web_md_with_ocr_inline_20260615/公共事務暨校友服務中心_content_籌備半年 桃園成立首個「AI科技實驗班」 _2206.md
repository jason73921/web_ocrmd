# 網站文章資料_籌備半年 桃園成立首個「AI科技實驗班」 _2206

## 一、文章基本資料

- **文章ID**: 2206
- **文章標題**: 籌備半年 桃園成立首個「AI科技實驗班」 
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20170912-1
- **文章簡介**: 〔自由時報/記者魏瑾筠/桃園報導〕

桃園市立永豐高中「AI科技實驗班」今天開班，教育局長高安邦、市議員呂林小鳳、祥儀企業總經理彭紹松出席開班剪綵，15名高一新生這3年中，將接受一系列物聯網、APP及程式設計等專業課程，永豐高中校長林裕豐表示，科技實驗班是桃園第一個，課程規劃上與普通班有非常大的差異，更加重視學生專業上的訓練，升學課程的比例比一般少，為此學校也特別舉辦家長說明會，期盼家長、學生能夠顛覆以往的觀念，也支持這樣的選擇。

林裕豐表示，實驗班學生在未來3年，除了專業課程、英文多益要達到550分以上，也要考取相關證照、積極參與各項競賽，師資群來自鄰近的中原、中央、元智等大學，目前已有20多位教授群，在升學的規劃上為了讓學生無後顧之憂，其中也與元智大學合作「3+4」模式，未來學生可選擇元智大學申請入學，也可依其他升學管道申請一般大學；另與桃園在地的祥儀企業合作，將產、官、學連結起來，也補強科技產業中「高中端」的人才訓練，另，學校也將在明年完成「物聯網中心」。(待續)
