# 網站文章資料_化材系校友王獻儀－與「老龐」並列冠軍 「岩漿火鍋」蔥燒牛肉麵脫穎而出_1104

## 一、文章基本資料

- **文章ID**: 1104
- **文章標題**: 化材系校友王獻儀－與「老龐」並列冠軍 「岩漿火鍋」蔥燒牛肉麵脫穎而出
- **文章網址**: https://www.yzu.edu.tw/alumni/index.php/tw/20241017
- **文章簡介**: 【民視即時新聞】

2024台北國際牛肉麵節的鮮食組比賽的結果出爐，今年拿下冠軍的有三家店，其中老龐家傳牛肉麵，再度蟬聯，特別的是，另外兩家都並非是專賣牛肉麵的業者，包括麻辣鍋出名的岩漿火鍋，以及以水餃鍋貼打天下的八方雲集，讓消費者也覺得不可思議！

倒出濃郁湯頭，麵條緩緩滑落，牛肉塊隨之墜入碗中，這碗是2024台北國際牛肉麵節，冠軍之一，岩漿火鍋的蔥燒牛肉麵。！

民眾：「很香辣，對，然後有一點麻度在裡面，所以是滿過癮的吃起來，對它的口味比較特別，我比較喜歡重口味的感覺。」

岩漿火鍋主廚王獻儀：「因為麻辣鍋跟牛肉麵，其實有滿多類似的地方，我用麻辣鍋的做法來做牛肉麵，當然就是味道上，層次會豐富一點，湯到滷完整個大概要4個小時左右。」

除了岩漿火鍋外，來自美食之都台南的老龐家傳牛肉麵，以特製牛排骨牛肉麵，同樣拿下冠軍，更特別得是，用水餃鍋貼打天下的八方雲集，自家的牛肉麵，竟然也獲得評審欽睞，同樣名列冠軍，只是獲得金牌的三家，竟然有兩家不是專賣牛肉麵。
