# 網站文章資料_財團法人工業技術研究院「2017年創新智慧應用大賽」_1192

## 一、文章基本資料

- **文章ID**: 1192
- **文章標題**: 財團法人工業技術研究院「2017年創新智慧應用大賽」
- **文章網址**: https://www.yzu.edu.tw/admin/rd/index.php/tw/?option=com_content&view=article&id=1192
- **文章簡介**: 主旨：為協助大學校院推廣研發技術，促進產學合作交流，特辦理2017年「創新智慧應用大賽」，得獎作品將展示於10月台北國際電子展－AIoT智慧生活體驗館，與國際廠商互動，敬請鼓勵 貴校 教師、學生籌組團隊參加。

說明：

一、2017年「創新智慧應用大賽」由外貿協會、電電公會主辦，工研院執行辦理，報名截止日期為九月四日(一)。

二、人工智慧(AI)與物聯網(IoT)崛起，AIoT將逐漸改變人類生活，位居全球資通訊產業關鍵位置之台灣，在產業及生活型態變革的洪流中也不容缺席，2017首屆之「創新智慧應用大賽」遂應運而生。藉由本競賽，希冀挖掘出台灣產業在AIoT應用於「智慧生活」得以耀眼全球的創新吸睛案例，並藉此刺激產學研的跨界合作與商業模式創新，透過持續研發與創新，實質提升台灣產業競爭力。
