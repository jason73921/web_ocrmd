# 網站文章資料_【住宿訊息】公告107學年度學生宿舍（新宿）床位「第二次抽籤」作業規定_1442

- **文章ID**: 1442
- **文章標題**: 【住宿訊息】公告107學年度學生宿舍（新宿）床位「第二次抽籤」作業規定
- **文章網址**: 
- **文章簡介**: 公告日期：107年4月17日（星期二）

 

主旨：公告107學年度學生宿舍（新宿）床位「第二次抽籤」作業規定。

說明：

一、登記時間：4月23日上午9:00～4月27日下午3:00止。

二、抽籤時間：4月23日上午11:00～4月27日下午5:00止。

三、抽籤方式：需本人親至宿服組辦公室辦理抽籤登記始取得抽籤資格，完成登記2小時之後再自行登入個人portal抽籤取得排名序號，未至宿服組辦理抽籤登記者，所抽序號一律無效，當事人不得異議。





先 >> 登記（取得抽籤資格）


➛➛➛

後 >> 抽籤（取得籤號之排名序號）






登記地點：宿服組辦公室

方式：本人取表登記並繳回

開放登記時間：4/23…09:00

結束登記時間：4/27…15:00


 

注意：完成登記2hr.後抽籤

方式：登入個人portal抽籤取號

開放抽籤時間：4/23…11:00

結束抽籤時間：4/27…17:00






四、抽籤資格：大一～大三之在學生。

五、登記地點：宿服組辦公室（位於男1舍1樓）。

六、本次抽籤床位數：男舍：85床；女舍：65床。

七、遞補順序：以電腦抽籤籤號之排名序號由小至大遞補。

八、名單首次公告日期：107年4月30日（星期一），後續如再有缺額產生，將一直以本次抽籤序號由小至大遞補到本(107)年5月18日為止，未獲遞補學生不得異議。


九、繳交保證金日期及地點：5月2日上午9:00起至5月4日下午4:00止逕至本校1館1樓總務處（或財管組）繳納，收費時間：每日上午9:00～12:00，下午1:00～4:00。

十、其他應注意事項：

(一)中籤者必須由本人進住，不得冒名頂替，違反者將取消床位並依校規懲處。

(二)宿舍申請，以一學年（上、下學期）為原則，除因畢業、休學、退學、轉學、出國交換、長期校外實習或特殊原因經核准者，宿期限未滿而自願及勒令退宿者，一律不退還保證金及住宿費。

(三)中籤者須在公告時間內繳交保證金2,000元，否則視同放棄住宿資格，所收取之保證金於107學年下學期住宿費中扣抵。已繳交保證金而放棄床位之同學，所預繳之保證金概不退還，且不得要求任何補救措施。

(四)大一新生住宿區維持凌晨0~6時斷網，舊生住宿區（新宿）則在凌晨2~6時斷網，無法接受者，請勿申請住宿。
