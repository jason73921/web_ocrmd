# 網站文章資料_科技部 「銀漿塗佈製程活動感知薄床墊開發及非察覺性睡眠監測研究和居家長者照護AIoT應用」研究計畫案_3859

## 一、文章基本資料

- **文章ID**: 3859
- **文章標題**: 科技部 「銀漿塗佈製程活動感知薄床墊開發及非察覺性睡眠監測研究和居家長者照護AIoT應用」研究計畫案
- **文章網址**: https://www.yzu.edu.tw/admin/rd/index.php/tw/?option=com_content&view=article&id=3859
- **文章簡介**: 主旨：

貴機構機械工程學系（所）徐業良教授申請補助延攬大陸科技人士白麗君參與「銀漿塗佈製程活動感知薄床墊開發 及非察覺性睡眠監測研究和居家長者照護AIoT應用」研 究計畫案，業經核定以「博士級研究人員」資格補助 (MOST 110-2811-E-155-501)，補助項目為研究費、年終 獎金、離職儲金、保險費等，請查照。

說明：

一、補助期間自110年6月1日起至111年7月31日止，月致研究費新臺幣63,000元整（自補助期間內實際到職之日起按月核給）。

二、申請機構應與受延攬人簽訂契約，明定雙方權利義務關係，參與之研究計畫完成或停止時，即終止補助。申請機構於接獲本部核定函後，得於到職前1個月內於本部延攬科技人才資訊系統進行請款作業，同時備函檢附單位領據、 撥款申請名冊與核定清單影本到部請款，並於補助期滿或中途離職後3個月內，依相關規定繳交受延攬人工作報告及檢據辦理經費結報，如有餘款並應繳回，或有不足者，應附領據由本部補發差額。

三、受延攬人申請來臺居（停）留事宜，請逕向主管機關辦理，於請撥補助款項時，應檢附下列許可證明文件影本之ㄧ，俾憑撥款：

(一)依大陸地區人民進入臺灣地區許可辦法，取得之來臺從事專業活動許可。

(二)依大陸地區人民在臺灣地區依親居留長期居留或定居許可辦法，取得之依親居留或長期居留許可。

四、受延攬人在臺居（停）留期間，不得涉及違反我政府之政 治活動、或有妨害我國家安全或利益之虞之相關研究或活動。

五、大陸人士在(來)臺相關資訊可至內政部移民署網站 (http://www.immigration.gov.tw/mp.asp?mp=1)查詢，請 轉知並提供貴機構相關生活資訊供來臺大陸科技人士參考。

六、檢附經費核定清單乙份供參並請轉知徐業良教授。
