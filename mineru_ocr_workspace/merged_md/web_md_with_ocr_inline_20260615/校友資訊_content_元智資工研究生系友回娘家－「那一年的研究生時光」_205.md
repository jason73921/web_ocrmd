# 網站文章資料_元智資工研究生系友回娘家－「那一年的研究生時光」_205

## 一、文章基本資料

- **文章ID**: 205
- **文章標題**: 元智資工研究生系友回娘家－「那一年的研究生時光」
- **文章網址**: https://www.yzu.edu.tw/alumni/index.php/tw/
- **文章簡介**: 
- **文章全文**: 元智大學資訊工程學系於1月30日，舉辦鍾添曜、黃依賢老師研究室畢業生回娘家活動，於龍潭老地坊餐廳舉辦，近百位畢業研究生與在校生同歡。 



資工系大家長黃依賢主任與鍾添曜老師親臨與校友同聚，黃依賢表示往年的系友活動主要對象為大學部畢業生，今年藉由實驗室的歷年畢業生及在學研究生大聚會，將系友通訊網擴展至研究所，不但可增加資工系友群組，也期待能帶給在學的同學們增廣見聞的機會。 



黃主任與鍾老師於餐敘中與畢業生們相談甚歡，系友們彼此分享工作經驗，也將唸書時的點點滴滴與心得分享給學弟妹，許多畢業生都很懷念在實驗室共處的那段時光呢！ 



此外，餐會後的回函調查，獲得許多畢業生們的寶貴建議，也同時更新了畢業生動態及聯絡資料庫，系友們除了給予學弟妹們許多未來就業的資訊，並勉勵大家一起努力為系上爭光。 



本次資工研究生系友回娘家，內容豐富，意義非凡，活動最後在畢業生們的歡笑及對系上誠摯的祝福中劃下完美句點。
