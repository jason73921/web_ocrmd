- **文章全文**: ThERE Cafe& Live House深藏城市中的音樂舞台



「從音樂獨立思考，將音樂作為媒介，往下扎根，進而影響一些人。」孫詮，元智國際企業系97級畢，ThERE Cafe& Live House共同創辦人之一，因喜歡音樂，從學生時期開始接觸音樂，與樂器行老師一同構想擁有展演空間的咖啡館，建立推廣獨立音樂平台，於2011年夢想之路就此展開。

 ThERE Cafe& Live House現今已邁入第四年，從剛開始默默無名，邀請樂團遭拒等，至今已有規律的樂團表演時間，樂團也會主動聯絡，頗具盛名的人氣樂團也在此辦過巡迴演唱會。 

 凡事起頭難，「雖然ThERE Cafe& Live House表面上有很多樂團表演，像是一個獨立音樂據點，開店初期從零開始，所觸及的對象僅於小眾範圍內，四年來一直存在的問題就是無法更加普遍化、普及化。」這個圈子(獨立音樂)的人熟知此地，但隔壁鄰居可能不知道ThERE Cafe& Live House是一個能夠容納兩百人規模的Live House，每個星期五晚上都神秘地在都市叢林中搖滾。

 展演售票也使ThERE Cafe& Live House與客人間產生難題，「因售票的因素，開店到現在真的做得很辛苦，不了解的客人覺得票價應包含餐點費，但ThERE Cafe& Live House希望讓更多人了解Live House是為了音樂而存在的空間，觀眾所購買的票價是為了樂團的音樂價值。」音樂中有創作者獨立思考所想表達的意念，每個人看到、聽到的也都將產生不同的化學反應

 孫詮表示，從小到大都在桃園生活，藉由對人文、本土關注，希望能為桃園文化盡一份心力，文化就是生活的一部份，每件小事都是值得去觀察的。回想大學時期，學生時期應加強思考人文與在地關注，但每個階段有不同學習到的事物，這也都是可以累積的經驗與價值觀。

 ThERE Cafe& Live House提倡音樂創作，藉由於桃園提供一個使獨立音樂自由發聲的空間，讓在地的音樂能量慢慢累積。文化需要時間堆積，使更多人對自身的生活環境開始關注並且思考真正想要的，讓在地能量發光發熱。

 ThERE Cafe& Live House於2014年主辦「唱所欲言」，以講座的方式推廣創作，創作是需要很長時間的培養與累積，藉由許多創作者從生活、土地、家鄉、社會等各種面向來分享，希望能給聽眾多一些創作的養分。

 透過桃園在地人文演講，於都市努力擁抱彼此的獨立音樂，以及親手製作的美食甜點，ThERE Cafe& Live House產生與家鄉連結的能量，進而將在地生活融入音樂，關懷周遭人、事、物。

 ThERE Cafe& Live House希望利用Live House讓每個人都能聽到年輕樂團不同的聲音、對台灣的想法，亦或是內心與夢想的對話。孫詮表示，未來希望ThERE Cafe& Live House能有機會舉辦更多不同形式的活動、講座，關係人文、在地、社會等議題與青年對話；也致力於生活化，使更多學生願意參與。

 夢想是進取的起點，不懈的努力才是邁向成功的階梯，藉此孫詮提醒元智學弟妹，並不是每個人都能走創業這條路，思考好自己的本質與所想，凡事學會獨立思考，不要人云亦云！
