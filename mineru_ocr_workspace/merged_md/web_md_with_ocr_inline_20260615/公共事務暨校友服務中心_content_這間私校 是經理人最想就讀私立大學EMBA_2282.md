# 網站文章資料_這間私校 是經理人最想就讀私立大學EMBA_2282

## 一、文章基本資料

- **文章ID**: 2282
- **文章標題**: 這間私校 是經理人最想就讀私立大學EMBA
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20171013-1
- **文章簡介**: 【聯合報/記者馮靖惠/即時報導】

元智大學設立全台第一個EMBA學位，不僅為成立時間最久的EMBA，也是私校EMBA排名中的第一名。根據Cheers雜誌針對3000大企業經理人2018年EMBA就讀意願最新調查，元智EMBA連續2年蟬連獲得各行各業經理人心目中的私校首選。

元智大學EMBA主任謝志宏教授表示，課程、師資、校友資源以及最重要的辦學理念是元智吸引學生就讀主因。元智管理學院擁有AACSB國際商管學院認證，為提供學生更多元的課程選擇，整合全院66位專任老師開課不分系，並邀請IBM、GE等外商高階經理人及工研院、商發院高階主管進行個案教學。增加「共時授課」課程，透過教授與業界師資的理論與實務的交流，為學生帶來更多產業洞察與啟發。

另外，校友旁聽EMBA課程免費，提供終身學習的服務並連線台北開設創新認證學分課程，組織校友互聯網。謝志宏表示，在選擇EMBA時，除了師資、課程之外，人脈也是一個重要考量。元智EMBA目前擁有超過2000位校友，所在產業遍及金融、物流、航空、製造及軍公教各領域，同學之間的交流透過社團和聯誼會等活動，讓畢業學長姐與在學的學弟妹擁有充分的時間交流，豐富多元的人脈就是元智EMBA特色。(待續)
