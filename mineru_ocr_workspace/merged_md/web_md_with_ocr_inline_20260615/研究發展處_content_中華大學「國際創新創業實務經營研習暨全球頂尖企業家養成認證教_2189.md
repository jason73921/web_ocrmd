# 網站文章資料_中華大學「國際創新創業實務經營研習暨全球頂尖企業家養成認證教師研習」_2189

## 一、文章基本資料

- **文章ID**: 2189
- **文章標題**: 中華大學「國際創新創業實務經營研習暨全球頂尖企業家養成認證教師研習」
- **文章網址**: https://www.yzu.edu.tw/admin/rd/index.php/tw/?option=com_content&view=article&id=2189
- **文章簡介**: 主旨：檢送中華大學管理學院與碁峰資訊股份有限公司合作辦理「國際創新創業實務經營研習暨全球頂尖企業家養成認證教師研習」活動實施計畫乙份，請鼓勵貴校教師踴躍報名參加，請查照。

說明：

一、為配合教育部高等教育深耕計畫，厚植學生創新創業教育，本校管理學院特與碁峰資訊股份有限公司，訂於107年12月20日，假本校管理學院金融資訊中心 M323電腦教室，

舉辦「國際創新創業實務經營研習暨全球頂尖企業家養成認證教師研習」活動。

二、活動免費，午餐自理，請鼓勵貴校與創新創業實務授課相符合之教師參加，於107年12月20日(星期四)前上網報名(僅能容納60人，額滿為止，請儘速報名)。

三、報名網頁: https://bit.ly/2KW28FY

四、教師研習活動實施計畫及報名表如附件。

五、聯絡人：碁峰資訊-劉道弘先生，聯絡電話：(02)27882408轉875，

E-mail：nick_liu@gotop.com.tw。
