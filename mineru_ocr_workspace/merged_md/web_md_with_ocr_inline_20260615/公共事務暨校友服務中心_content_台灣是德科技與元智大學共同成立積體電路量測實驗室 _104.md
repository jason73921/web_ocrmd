# 網站文章資料_台灣是德科技與元智大學共同成立積體電路量測實驗室 _104

- **文章標題**: 台灣是德科技與元智大學共同成立積體電路量測實驗室 
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【CTIMES 編輯部 報導】 


台灣是德科技（Keysight Technologies Taiwan）與元智大學電機學系於2015年7月14日簽署合作備忘錄，「積體電路量測實驗室」正式啟用，此有效結合企業及學界的研發能量，促成產學合作機制，更攜手強化積體電路設計與量測技術發展。此合作備忘錄讓台灣是德科技有機會支持教育單位的方針落實，這次簽署將有利於公司在技術研究與交流、人才培養及儲備等方面的進一步發展，對公司未來市場拓展、業績成長有積極意義。 



圖左為元智大學電機通訊學院院長林志民教授，圖右為台灣是德科技行銷處副總經理羅大鈞 

台灣是德科技行銷處副總經理羅大鈞強調：「作為企業公民，是德科技多年來在教育領域投注資源不遺餘力，使科技成為提昇教育品質的重要助力。」元智大學在台灣是德科技的審慎評估下，共同成立「積體電路量測實驗室」列為台灣是德科技2015年度的重要計畫之一。 


藉由成立「積體電路量測實驗室」，台灣是德科技與元智大學將定期協商研究方向，訂定產學合作計畫及相關事宜。讓元智大學學生在正式進入職場前，有更多的實機操作以及磨練的機會此外。台灣是德科技將秉持著致力於積體電路設計與量測科技人才的培育的精神，支援元智大學電機系教育培訓，提昇教學品質及優化專業科技人才。 

元智大學電機通訊學院院長林志民教授提到：「元智大學電機系與台灣是德科技成立「積體電路量測實驗室」於元智大學七館70604室，所建置的重點設備，主要深化積體電路量測及相關積體電路設計等核心課程的教學品質，嘉惠本系莘莘學子，讓學生擁有一優質的教學設施與環境，培育學生瞭解積體電路設計與量測的重要性。並同時讓教學與研究接軌，作為元智大學電通學院積體電路領域未來的研究主軸佈局。」 


揭牌典禮後，接著安排網路分析儀產品應用課程，由台灣是德科技資深專案經理蔡怡杏進行專題演講，有助於實驗室使用者瞭解產品及應用的實機使用操作，透過產學研交流，激發創新研究靈感。
