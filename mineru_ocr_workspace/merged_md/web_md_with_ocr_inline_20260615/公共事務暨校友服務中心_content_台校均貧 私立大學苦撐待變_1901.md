# 網站文章資料_台校均貧 私立大學苦撐待變_1901

## 一、文章基本資料

- **文章ID**: 1901
- **文章標題**: 台校均貧 私立大學苦撐待變
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20170509
- **文章簡介**: 【中天快點TV】

大陸教育部直屬大學年預算動輒百億人民幣起跳，看在台校眼中情何以堪！台灣大專校院可說是「均貧」，除了「五年五百億」、「教學卓越計畫」、「典範大學計畫」可以拿到教育部補助，國立大學有教育部撐，私立大學除了學費就只有3種情況可以苦撐待變：宗教團體辦學、財團辦學或是開醫院。
根據教育部「104學年度各校學雜費占平均每生教學成本」，學雜費占教學成本最低的前12校，第一名是台北醫學大學，一年學雜費11萬1686元，每位學生教學成本卻高達40萬7147元，學費收入僅占27.43%。如果不是有附設醫院財務支撐，根本辦不下去。

第二名長庚大學，一年學雜費10萬1777元，每位學生教學成本35萬6253元，學費收入僅占28.57%；第三名慈濟大學，一年學雜費9萬817元，每位學生教學成本30萬6373元，學費收入僅占29.64%。長庚大學背後是台塑企業，慈濟大學背後是宗教團體，才得以彌補每生每年20多萬差額。

四到十名依序是佛光大學（佛光山贊助）、高雄醫學大學（有醫院）、元智大學（遠東集團贊助）、南華大學（佛光山贊助）、華梵大學（宗教團體贊助）、中國醫藥大學（有醫院）和中原大學，除了中原大學之外，都有企業或宗教團體做「靠山」。
