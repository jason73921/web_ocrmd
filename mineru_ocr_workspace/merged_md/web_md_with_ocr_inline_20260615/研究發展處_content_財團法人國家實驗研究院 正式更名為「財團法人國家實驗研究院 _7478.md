# 更名通知書

敬啟者：

本中心之中文全名原為「財團法人國家實驗研究院台灣儀器科技研究中心」，經主管機關（中華民國國家科學及技術委員會）核准，變更名稱為「財團法人國家實驗研究院國家儀器科技研究中心」（將原名稱中「台灣」變更為「國家」。本中心英文全名原为TaiwanInstrumentResearchInstituteNationalAppliedResearchLaboratories，亦同步變更為NationalCenterforInstrumentationResearch，NationalInstitutes ofAppliedResearch。更名後，本中心之中文簡稱變更為「國儀中心」，英文簡稱變更為「NCIR」。

自民國114年3月25日起，敬請配合將本中心之相關資料更改為「財團法人國家實驗研究院國家儀器科技研究中心」。原統一編號、地址、銀行帳號、聯絡電話及傳真皆維持不變。基此，本中心更名前所簽署之合約繼續有效，不因更名而影響雙方之權利義務，特此說明。

如對更名事宜有任何疑問，歡迎隨時與我們聯繫，也祈盼您繼續給予本中心鼓勵與支持！
