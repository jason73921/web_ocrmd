# 網站文章資料_【轉知】國立臺灣大學提供申請英語編修軟體Grammarly帳號及停止授權對象_1544

## 一、文章基本資料

- **文章ID**: 1544
- **文章標題**: 【轉知】國立臺灣大學提供申請英語編修軟體Grammarly帳號及停止授權對象
- **文章網址**: https://www.yzu.edu.tw/admin/aa/index.php/tw/2024-01-08-2
- **文章簡介**: 主旨：本校EMI教學資源中心提供英語編修軟體Grammarly帳號使用權限給全國大專校院，請查照。

說明：

一、該中心秉持資源共享、共學共益的理念，並為了鼓勵更多教師運用資源及協助各校師生提升英語寫作能力與文章編修效率，開放全國大專校院申請Grammarly帳號。

二、本申請請各校責成相關單位彙整申請表單後統一寄至該中心Email，個別申請者本校不予受理。

三、為使資源更加充分運用並觸及更多師生，113年起將不提供給已獲教育部「大專校院學生雙語化學習計畫」第1期第2階段之重點培育學校及EMI教學資源中心申請；原本已開通之重點培育學校及EMI教學資源中心之教師亦將於113年1月17日起停止授權。

四、聯絡人：羅蘊宜專員（02-33669790#225）。
