# 網站文章資料_國立虎尾科技大學 辦理教師增能系列活動-「研究倫理研習」課程資訊_4646

## 一、文章基本資料

- **文章ID**: 4646
- **文章標題**: 國立虎尾科技大學 辦理教師增能系列活動-「研究倫理研習」課程資訊
- **文章網址**: https://www.yzu.edu.tw/admin/rd/index.php/tw/?option=com_content&view=article&id=4646
- **文章簡介**: 主旨：檢送本校教學發展中心與國立中正大學人類研究倫理中心合作辦理教師增能系列活動-「研究倫理研習」課程資訊，敬請惠予公告並鼓勵貴校師生踴躍參與，請查照。

說明：

一、為能有效提升於研究過程中對應個人與群體權利義務的認知，並於計畫執行前、中、後等階段均能獲得相對應之保障，特辦理「研究倫理研習」課程。(附件)

二、研習課程內容：

(一)議題一：研究倫理概念及為什麼需要申請倫審。

(二)議題二：研究倫理申請及審查之秘笈。

(三)議題三：教學實踐的研究倫理考量與審查重點。

三、研習課程主講人：

(一)議題一、二主講人：國立中正大學心理學系副教授、倫理審查委員會委員暨人類研究倫理中心主任鄧閔鴻主任。

(二)議題三主講人：國立中正大學教育學研究所教授暨倫理審查委員會委員王雅玄委員。

四、研習活動時間：111年6月29日(三)9:00-12:00。

五、研習活動地點：本次活動採線上形式辦理(採用Cisco Webex應用程式)，並於活動前一日寄發線上會議連結網址。

六、活動報名時間：即日起至6月28日(二)下午5時止。

七、活動報名網址：https://iact.nfu.edu.tw/activities /actinfo/9b7c8d13e4b2f08895fb7bcead930b46 /actinfo/9b7c8d13e4b2f08895fb7bcead930b46 

八、檢附活動議程供參，相關資訊亦同步公告於本校教學發展 中心網站(https://ctld.nfu.edu.tw/)，全程參與者並完 成簽到/退者將由國立中正大學人類研究倫理中心核發研習時數。

九、活動承辦人：本校教學發展中心柯雅羚助理，連絡電話 (05)6315123，電子信箱lingko223@nfu.edu.tw。
