# 網站文章資料_校友會會員之權益_140

## 一、文章基本資料

- **文章ID**: 140
- **文章標題**: 校友會會員之權益
- **文章網址**: https://www.yzu.edu.tw/alumni/index.php/tw/2018-03-12-01-22-25
- **文章簡介**: 親愛的校友 您好：

母校為扶植校友組織之發展，特別配合校友會施行對校友有益之福利政策，以鼓勵校友加入校友會。

有任何問題歡迎您隨時指教，謝謝！

 

圖書館借書服務

　- 線上申請（校友會校友免收3,000元保證金） 

　- 取件地點：元智圖書館出納流通櫃台申請（服務專線03-4638800*2321）


　- 取件日期：約3天左右

　- 相關資訊：http://www.yzu.edu.tw/library 

 

終身教育部課程折扣優惠

　- 凡校友會會員憑校友證，至母校終身教育部報名修課，除免報名費外，且享有學費七折之優惠（需「適用自辦課程折扣」之課程）

　- 詳見終身教育部官網


 

免費或酌收優惠價參加校友會舉辦之活動或聚餐

 

申請健康休閒中心使用證注意事項

　- 辦健康休閒中心使用證（一年5,560元、半年3,330元、一年內有效30格票3,030元、10格票1,150元、單次票150元）

　- 因健康休閒中心無使用信用卡扣款機制，需現金繳納，但可由他人代為辦理。

　- 詳見元智大學健康休閒中心官網>票證申請

　- 申請辦法請參閱「健康休閒中心管理實施細則」
