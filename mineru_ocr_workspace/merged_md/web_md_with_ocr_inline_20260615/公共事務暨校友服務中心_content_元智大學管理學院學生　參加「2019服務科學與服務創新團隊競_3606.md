# 網站文章資料_元智大學管理學院學生　參加「2019服務科學與服務創新團隊競賽」榮獲全國冠軍_3606

## 一、文章基本資料

- **文章ID**: 3606
- **文章標題**: 元智大學管理學院學生　參加「2019服務科學與服務創新團隊競賽」榮獲全國冠軍
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20190308-1
- **文章簡介**: 在元智大學管理學院鄒蘊欣老師的帶領下，行銷碩士學程的學生陳冠真、簡少軍、蕭余芷、鄭亞瑄，參加「2019服務科學與服務創新團隊競賽」，歷經四個月的努力，從全國大專院校100個團隊中脫穎而出，以「寵物醫聯網」的服務科學與創新企劃案，奪得全國冠軍，抱回獎金二十萬元。

「2019服務科學與服務創新團隊競賽」由台灣服務科學學會主辦，溫世仁文教基金會與看見台灣基金會協辦。今年全國參賽團隊有超過500位師生參賽。元智行銷四位同學所組成的「小福星」團隊歷經全國北、中、南三區進行分區初賽；最後爭取到全國12強團隊，闖入全國總決賽。全國總決賽以「智慧服務的創新模式」為題，強調以「設計思考」(design thinking)的觀點與方法，洞察需求、定義問題、創意發想，最後提出改善既有服務流程或創造創新商業模式。

指導老師管理學院行銷學群鄒蘊欣表示，由於競賽時程長達四個月，對參賽的同學來說，不僅是時間、智識與經驗的付出，更是對自我與團隊毅力的考驗。元智「小福星」團隊從一開始的創意不斷被挑戰與批判，到學習觀察自己及週遭親友在生活上所產生的不便，以使用者觀點的角度，提出「寵物醫聯網」的企劃案。

參賽同學鄭亞瑄表示，此企劃從寵物的醫療需求做為痛點切入，透過金融、醫療、區塊鏈及智慧服務等應用，整合寵物醫療、寵物保險及寵物用品，打造寵物生態圈的智慧服務創新商業模式。「小福星」的企劃案突顯科技運用及服務創新，讓團隊在競爭激烈的環境中，獲得評審青睞，高分榮獲金牌獎之殊榮。

鄒蘊欣說，元智管院行銷碩士學程一直強調培養學生的實務與產業經驗，在教學的過程中，強調產業實務與理論的碰撞、對話與結合，此外也鼓勵學生批判性思考，從他人批判、自省與資訊整合的過程中，創作出具高度實踐性的創意企劃，過程中，四位同學的簡報內容修改不下百次，不斷鍛練團隊在整合與分析資訊的能力，如今獲得評審青睞，實至名歸。

元智大學管理學院於今(2019)年獲得英國泰晤士報評鑑為全球五百大商管學院，在教學方針與研究能量方面，不僅強調理論與實務的結合，更積極培養學生的國際移動力與產業實踐能力，從課程規劃了大數據行銷，以及鼓勵學生培養跨域能力即可窺見。鄒蘊欣表示，此次參賽的同學除行銷管理專才外，還有外文及傳播，以及餐飲實務管理的背景，在在都是成就學生金牌作品的養份。
