n   講題：學術期刊投稿基本原則與技巧

*注意事項*

n   本辦公室將於活動開始前一天(2月17日)寄發觀看影片之網址及測驗試題，請至報名時填寫之信箱確認。2月18日至2月22日期間24小時皆可上網觀看。*若當天下午4點前，仍未收到課程資訊，請寄信詢問吳怡潔專案工作人員(z10503061@email.ncku.edu.tw)

n   報名者請於課程開放時間內完成觀看課程與測驗，時間截止後將關閉影片與測驗卷。

n   本次課程開放期間只計算前三次測驗分數，測驗題目共10題，1題10分，總計100分。測驗分數達80分以上者，將取得學術倫理教育時數2.5小時，於活動結束後兩週內核發(校內人士將直接匯入平台, 校外人士將以Email寄送電子證書 )。

n   未完成測驗或未達80分者，恕不發放時數證明，且不另行通知。
