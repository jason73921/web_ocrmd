## 一、文章基本資料

- **文章ID**: 178
- **文章標題**: 108學年下學期學生汽車通行證辦證時間調整
- **文章網址**: https://www.yzu.edu.tw/admin/ga/index.php/tw/108-2
- **文章簡介**: 【公告】108學年下學期學生汽車通行證辦證時間調整

各位同學好！

因應本學期延後開學，謹調整臨櫃辦車證時間

擬辦理108學年下學期(109.02.01.～109.07.31.)汽車識別證者，

新學期第一週～聯合服務櫃檯現場收費時間調整如下，敬請多加利用（若時間允許，可於現場取證）。



非於上述收費時間辦理者，請於週一至週五上班時間，先與總務處警衛隊劉逢相先生聯繫辦理時間（03-4638800分機2261），俾便配合辦理。

總務處 敬啟  109.02.05


各位同學好！

擬辦理108學年下學期(109.02.01～109.07.31)汽車識別證者，請先行上網填妥申請資料並列印申請表。

109年1月7日起，攜申請表及相關證件至聯合服務櫃檯繳費辦理。

辦理程序：

   (1)上網 ( http://140.138.35.130/GeneralOffice/park/ ) 填妥汽車識別證申請表印出，並貼妥駕照、行照影本。

   (2)至聯合服務櫃檯，由受理單位核對、登錄資料。

   (3)申請人完成繳費。

   (4)申請人領取車輛識別證。（若時間允許，可於現場取證）

聯合服務櫃檯現場收費時間規劃如下：



備註：

  1、本次所辦車輛識別證使用時間為109.02.01～109.07.31，逾期作廢。

  2、逾當日繳費時間，請擇他日辦理，或可備妥相關資料及現金（每學期一般生收費4,000元、專班2,500元）託人代為繳費辦理。

  3、建議以「班」為單位彙整資料，統一辦理停車證，以利繳費並迅速取得車證使用。

  4、入校車輛相關規範請詳參本校校區車輛管理辦法。

總務處 109.01.02敬啟
