# 網站文章資料_50大學師連署 籲兼任教師一體適用勞基法_1559

## 一、文章基本資料

- **文章ID**: 1559
- **文章標題**: 50大學師連署 籲兼任教師一體適用勞基法
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【自由時報/記者蕭玗欣/台北報導】 


高等教育產業工會今發起連署，要求將「未具本職」兼任教師納入勞基法，讓所有兼任教師一體適用勞基法。日前勞動部和教育部將「未具本職」兼任教師納入勞基法草案規範，但工會痛批，此舉為各校開了後門，有學校以書面或口頭明示、暗示，要求各兼任教師自行去找到本職証明，藉以排除教師適用勞基法，否則將不予續聘。該連署目前已有超過50位大學兼任教師連署支持。

連署聲明開頭以「兼任教師一體適用勞基法—致手上握有權力讓我們獲得基本尊嚴的人們」為題，發起人是一群高達1萬3000多名在各大專院校流浪兼任、開課數多達2萬餘堂課的老師，聲明表示，這群兼任教師的存在，是非典型僱佣在高教場域的體現，也是長年以來產業結構無法進階轉型、國家財政無法有效開源導致教育經費未能隨時代走向而更多提升、終致各校不願聘足專任教師的結果。

聲明指出，目前的草案強調只適用於那些「不具本職」的人，並以個別極端的個案用以說明為何不將所有兼任教師一體適用勞基法，「不但法理上站不住腳，同時也為各校開了後門，各種亂象已經陸續出現。」 

高教工會表示，有學校透過書面或口頭明示、暗示，要求各兼任教師自行找到校外的本職證明，就此排除教師適用勞基法，否則就威脅不予續聘，若有不願意配合自行「証明」具本職的教師，校方就刪減其授課時數，兼任教師賴以維生的鐘點費也就跟著減少。 


高教工會也說，過去幾週以來，曾不斷向教育部檢舉各校亂象的案例，提供各種書面証據，但至今仍未得到教育部和勞動部回覆，要求教育部和勞動部在擬定新草案時，將「不具本職」刪除，讓所有兼任教師一體適用勞基法。

目前連署兼任教師包括文藻外語大學日語系丁紀祥、世新大學通識教育中心方喜恩、元智大學藝術與設計學系牛俊強、元培科技大學應用外語系王平、世新大學資訊傳播學系王立業等50多位教師，連署人數持續增加中。
