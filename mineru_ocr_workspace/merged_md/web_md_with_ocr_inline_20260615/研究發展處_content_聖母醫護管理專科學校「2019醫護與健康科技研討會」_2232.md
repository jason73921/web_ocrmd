# 網站文章資料_聖母醫護管理專科學校「2019醫護與健康科技研討會」_2232

## 一、文章基本資料

- **文章ID**: 2232
- **文章標題**: 聖母醫護管理專科學校「2019醫護與健康科技研討會」
- **文章網址**: 
- **文章簡介**: 主旨：聖母醫護管理專科學校於108年5月18日(星期六)舉辦「2019醫護與健康科技研討會」，檢送徵稿訊息，惠予公告，請鼓勵貴屬同仁及師生踴躍投稿，並參與研討會，請查照。

說明：

一、研討會宗旨：配合教育部高教深耕計畫，並鼓勵教師從事與教學相關之專題研究，並發表研究成果，以提升學術研究風氣。藉此研討會邀請各專家學者先進交換實務經驗，增進學術界與實務界合作互動與交流，掌握醫護與健康科技領域上之最新發展與未來趨勢，分享研究成果。 

二、研討重要時程： 

論文摘要投稿截止日：2019/04/15(一) 

摘要審查結果公告日：2019/04/30(二) 

口頭論文全文繳交截止日：2019/05/10(五) 

壁報收件截止日：2019/05/16(四) 

研討會報名截止日：2019/05/13(一) 

研討會日期：2019/05/18(六) 

三、研討會主題： 

護理與醫學科技研發。 

健康產業與餐飲管理。 

醫學美容科技應用。 

幼兒保育與健康照護。 

資訊科技健康應用。 

醫護與健康人文探討。 

四、論文發表方式：「口頭論文發表」與「壁報論文發表」。
