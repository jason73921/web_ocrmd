# 網站文章資料_產學合作 遠銀、元智、成大推首創的「菜巿場投資法」_5111

## 一、文章基本資料

- **文章ID**: 5111
- **文章標題**: 產學合作 遠銀、元智、成大推首創的「菜巿場投資法」
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/202107129-2
- **文章簡介**: 【翻爆】

針對渴望投資理財自主投資民眾，卻面對龐大雜亂或片段的投資參考訊息，通常會有無所適從的感覺，因此今年剛獲頒Asiamoney台灣最佳數位銀行的遠東商銀，積極發展智能金融服務，聯手元智大學、成功大學合作，研發設計AI智能RM（Relationship Manager），成為台灣第一個將自然語言處理及情感分析技術，應用在基金投資的創新「大數據菜巿場投資法」。

遠東商銀表示，這個技術是運用彙整大量財經新聞、媒體報導等網路大數據，為客戶轉換成簡單易懂的投資訊息，利用專利人工智慧演算法篩選同一類基金市場中表現較佳基金，提供客戶在線上自主投資時，輔助決策參考資訊，這個創新合作俗稱「大數據菜巿場投資法」，為業界獨家創新技術，已獲3項專利、4個國內外獎項肯定。

遠東商銀指出，AI智能RM具備兩個特色，一是「秒懂市場」，即運用元智大學獲獎的專利AI語意分析技術，收集各大網路新聞媒體的財經新聞並判斷其內容語意的正負向，彙集成14個基金投資市場分類的「投資溫度」，客戶可以在幾秒內得知14個市場的樂觀程度。（待續）
