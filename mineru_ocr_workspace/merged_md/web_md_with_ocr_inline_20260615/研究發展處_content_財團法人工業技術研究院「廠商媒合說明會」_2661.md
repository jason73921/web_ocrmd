# 網站文章資料_財團法人工業技術研究院「廠商媒合說明會」_2661

## 一、文章基本資料

- **文章ID**: 2661
- **文章標題**: 財團法人工業技術研究院「廠商媒合說明會」
- **文章網址**: https://www.yzu.edu.tw/admin/rd/index.php/tw/?option=com_content&view=article&id=2661
- **文章簡介**: 主旨：本院執行科技部108年度海外人才橋接方案(LIFT 2.0)，將於108年9月12日辦理「廠商媒合說明會」，地點於新竹市光復路二段321號1館609教室，歡迎貴單位參加並請協助公告周知，請查照。

說明：

一、科技部108年度海外人才橋接方案(LIFT 2.0)於本年度搭建LIFT TAIWAN線上線下人才/來臺就業交流平臺，鏈結海外學人返/來臺就業，促成海外學人返/來臺貢獻所學，與國內產學研機構進行面對面交流。

二、本方案訂於108年10月21日至11月1日舉辦「108年度第二梯次海外學人國內交流會」，預計邀請70位以上海外博士級(或同等)人才返/來台參訪國家重大建設、國內產學研機構，並於活動期間舉辦廠商交流媒合會，檢附本次媒合資訊如附。

三、為使國內產學研機構更了解本次媒合活動形式，將於108年9月12日辦理「廠商媒合說明會」，地點於新竹市光復路二段321號1館609教室，報名網址為https://college.itri.org.tw/edm/D2/004/02/edm.html ，歡迎貴單位參加並轉知相關單位，及協助廣宣本次活動，請查照。

四、相關事宜可逕洽海外人才橋接方案計畫辦公室；陳小姐(03-5743982/ E-mail: itri535110@itri.org.tw)
