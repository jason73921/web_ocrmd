# 網站文章資料_107學年度碩士班、博士班甄試入學備取生報到注意事項_350

## 一、文章基本資料

- **文章ID**: 350
- **文章標題**: 107學年度碩士班、博士班甄試入學備取生報到注意事項
- **文章網址**: https://www.yzu.edu.tw/admissions/index.php/tw/107-4
- **文章簡介**: 說明：

一、各系所備取生遞補名單將自106年12月18日起至107年2月26日止（春節期間2/19暫停公告），於每週一上午10時在本校招生招生報名查詢系統網頁公布備取生遞補名單（https://exam.yzu.edu.tw/NewNetapply→點選「107碩士班甄試」或「107博士班甄試」→報到查詢），請備取生務必主動上網查詢，另亦會以e-mail通知遞補上之備取生。

二、下載：備取生報到注意事項及委託書報到專區網址：https://exam.yzu.edu.tw/NewNetapply（點選「107碩士班甄試」或「107博士班甄試」→查詢報名結果與成績→輸入身分證字號及密碼→查詢成績→報到專區）

 三、 各系所錄取新生注意事項，請自行參考各系網頁公告。
