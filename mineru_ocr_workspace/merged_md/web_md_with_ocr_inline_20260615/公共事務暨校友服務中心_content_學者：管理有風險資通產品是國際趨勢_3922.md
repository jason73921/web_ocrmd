# 網站文章資料_學者：管理有風險資通產品是國際趨勢_3922

## 一、文章基本資料

- **文章ID**: 3922
- **文章標題**: 學者：管理有風險資通產品是國際趨勢
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/2019082001
- **文章簡介**: 聯合晚報 記者賴于榛／台北報導

行政院近日將公布危害國家資通安全產品禁購清單，學者們紛紛表示，禁購清單當然有必要，因為網際空間（Cyberspace）的威脅，没有地域性，有風險的資通訊產品應明文管理是國際趨勢，且禁止華為應毫無疑慮，但接下來公布的清單是否會借資安之名打擊大陸廠商，將是各界關注重點。

東吳大學法律研究所兼任助教授萬幼筠指出，針對關鍵基礎建設的安全部分，世界各國都有立法，尤其IoT（物聯網）設備，只要能夠聯網的部分，都已是加強管制的重點。

元智大學資訊管理系教授周韻采也贊成行政院的做法，過去曾是NCC籌備委員的她表示，馬政府時代雖然沒有像蔡政府訂出「各機關對危害國家資通安全產品限制使用原則」，但透過一些技術性卡關，不讓中國大陸有資安疑慮的產品進到台灣，她肯定民進黨政府這次將資安疑慮的原則說的更明確，沒有模糊地帶也讓電信業者有原則可遵循。(未完待續)
