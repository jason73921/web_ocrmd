# 網站文章資料_衛生福利部臺中醫院「全人醫療暨整合服務研討會」徵稿_1118

## 一、文章基本資料

- **文章ID**: 1118
- **文章標題**: 衛生福利部臺中醫院「全人醫療暨整合服務研討會」徵稿
- **文章網址**: https://www.yzu.edu.tw/admin/rd/index.php/tw/?option=com_content&view=article&id=1118
- **文章簡介**: 主旨：106年第六屆「全人醫療暨整合服務研討會」預定於106年9月29日(五)假衛生福利部臺北醫院辦理，投稿日期延至106年7月14日，敬請貴單位惠予公告並鼓勵踴躍投稿，請查照。

說明：

一、為提升學術論文研究風氣，以提供學術研究交流平台，以「醫療照護/醫學教育類、病人安全/品質類、智慧醫療/E化類、醫院管理/健康管理類、長期照護/高齡友善/社區醫療類」等五大類為主題，期待透過研討會各講者精采之

專題發表、醫界菁英經驗傳承、醫療院所同仁及大專院校的踴躍參與，帶給與會者一場耳目一新的學術饗宴，進而提升醫療品質與教學研究，使全民健康福祉獲得更大的助益，朝向優質的全人醫療目標邁進。

二、投稿期間：延至106年7月14日止。

三、投稿方式：一律將稿件投稿至http://edu.mohw.gov.tw年會專區。

四、聯絡人：衛生福利部臺北醫院許純敏專員/陳彥伶專員，電話:02-22765566轉2816/2820或醫管會教研小組陳蔚菱專員，電話:04-22294411轉3603。
