# 網站文章資料_我見我思－李明哲的試煉_1865

## 一、文章基本資料

- **文章ID**: 1865
- **文章標題**: 我見我思－李明哲的試煉
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20170424
- **文章簡介**: 【中時電子報/周韻采】

在李明哲被大陸拘留近1個月裡，海協會曾透過李俊敏傳遞訊息給家屬，卻被妻子李凈瑜以非典型方式回應。她發表聲明：「如果我繞過政府去接受一個掮客的條件，我就是出賣了台灣的尊嚴。」這樣直接嗆聲大陸當局的做法，引起諸多討論。

有人認為非典作法完全無助李明哲脫困，甚至懷疑李凈瑜是否真心要救夫。另一派則是支持李凈瑜對抗中共威權，彰顯台灣人的尊嚴，呼籲所有台灣人都應做李凈瑜的後盾。

不管非典做法是否真地能營救李明哲，在無法得知他本人意願的狀況下，我們身為旁觀者只能尊重他的法定代理人──妻子與父母──的決定。然而，我也同時反對將抵抗中國的慾念投射在李明哲身上，讓他去承擔整體台灣人的尊嚴。這樣集體意志的沸騰，跟當年台灣少棒隊遠征威廉波特，希冀一群孩童去成就台灣人的榮耀，有什麼不一樣？我們每個人都只能代表自己，要個人去代表集體的意志，是殘忍且不道德的。

倒是李凈瑜聲明中的一段話讓我玩味再三：「我不會讓我先生為獲得自由而失去尊嚴，像狗一樣活未來的一生。」這句話讓人熱血沸騰，但也武斷地詮釋李明哲的真正想望。換言之，李明哲是不被妻子允許如法國導演高達的電影《賴活》般，一樣沒有尊嚴地向中國認罪且苟活著。在此，李明哲本人已失去話語權。我想像著，當李明哲重新踏上台灣土地的那一刻，他被大批媒體堵麥，他或許深情地望著妻子一眼，然後堅定地譴責中國，絕不向強權屈服。然而，會不會也有這樣的可能？事過境遷後，李明哲也難免抱怨，妻子讓他受了許多額外的折磨，卻還要為她武斷的決定背書。

在美麗島事件淡出台灣人的記憶多年後，艾琳達又回到台灣作博士後研究。我與她是淡淡交往的朋友，但一次聊天時，她說她不想做曾經的施明德女人，她要做艾琳達。我感覺她對施明德有怨懟，但男人民主鬥士的光環讓她無法對台灣人說出她的苦處。我認識的美麗島受難者家庭，女人們扛著代夫出征旗號得到政治平反，然而男人出獄後卻都勞燕分飛，眾人除了唏噓外，又能替他們的愛恨情仇承擔什麼？

當大陸逮捕李明哲後，李凈瑜毫無選擇地必須表態，滿足大眾要向大陸喊話的躁鬱。或許她也還沒想到，這些挺她的聲音，在未來與李明哲的重聚時，就像幽幽存在的第三者，不斷翻攪他們的記憶與愛戀，讓他們再也無法回到從前了。大陸不人道地加諸這對夫妻不應該的人性試煉，但可悲的是，這些要台灣人尊嚴的支持者是對他們的感情世界無能為力的。到頭來，他們還是得踽踽獨行走完情字這一程。（作者為元智大學資訊管理學系教授）
