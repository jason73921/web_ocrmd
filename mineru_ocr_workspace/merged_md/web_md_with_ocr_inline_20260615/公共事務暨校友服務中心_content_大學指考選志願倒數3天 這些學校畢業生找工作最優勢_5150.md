# 網站文章資料_大學指考選志願倒數3天 這些學校畢業生找工作最優勢_5150

## 一、文章基本資料

- **文章ID**: 5150
- **文章標題**: 大學指考選志願倒數3天 這些學校畢業生找工作最優勢
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20210823-5
- **文章簡介**: 【聯合報/記者陳素玲／台北即時報導半導體】

大學指考選志願倒數3天，畢業生出路比「人頭」也比「拳頭」。「104玩數據」上週依三大燙金產業最常邀約畢業生總量(人頭)排序，本週再依平均每位畢業生獲得幾次邀約(人均實力)、以及畢業後五年真正進入產業工作(錄取實力)排序，結果發現，理組為首的半導體、軟體網路偏愛國立理工學校與三大技職，文組商管的金融業以老牌私校和技職較占優勢。

「 104玩數據」分析最近五年應屆畢業生共80.6萬份履歷表，依新鮮人最想去的前十大產業，再依平均月薪排序，得出三大燙金產業是半導體業、金融業、軟體網路業。近日再針對2021年上半年，三大燙金產業的五大主要職缺(不含作業員)邀約近2.7萬名應屆畢業生，依平均每人獲得幾次邀約進行校排。

上半年是企業的校園徵才、以及畢業季徵才高峰前段。三大燙金產業的五大主要職務上半年主動邀約今年應屆畢業生(含役畢)，按最高學歷「人均」學校排序。 結果發現，半導體前十名幾乎是國立大學的天下，除了頂大幫(台、成、清、陽明交大)、中字輩(中正、中山、中興、中央)、三科大(台科大、北科大、雲科大)，淡江是唯一入榜的私校。

至於軟體網路業，除上述國立大學之外，新面孔有元智、長庚、華夏科大、暨南。金融業若是金融營業員，前五名人均邀約次數較多的畢業生分別是中央、元智、東海、世新、東吳，有更多私校及技職入榜，可見名學光環之外，新鮮人實作力，仍受產業重視。（待續）
