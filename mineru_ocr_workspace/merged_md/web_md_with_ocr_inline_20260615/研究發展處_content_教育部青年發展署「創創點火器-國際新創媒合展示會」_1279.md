# 網站文章資料_教育部青年發展署「創創點火器-國際新創媒合展示會」_1279

## 一、文章基本資料

- **文章ID**: 1279
- **文章標題**: 教育部青年發展署「創創點火器-國際新創媒合展示會」
- **文章網址**: https://www.yzu.edu.tw/admin/rd/index.php/tw/?option=com_content&view=article&id=1279
- **文章簡介**: 主旨：本署定於本(106)年11月1日辦理創創點火器「國際新創媒合展示會」，惠請貴單位協助推薦新創團隊或公司參展，請查照。

說明：

一、為提供新創團隊與創投媒合及交流機會，並促進國際新創風氣，本署定本年11月1日10時30分至17時於松山文創園區辦理創創點火器「國際新創媒合展示會」，將邀請65家新創團隊或公司參展，以及6家國外加速器參加，參展主題分為「智慧‧物聯」、「行動‧雲端」、「生活‧商務」、「網路‧創新」4大領域。

二、檢送旨揭活動簡介(如附件1)及報名表(如附件2)，歡迎推薦貴管計畫輔導之創業團隊或公司參展，並請協助於本年10月13日前回傳報名表至本案承辦單位(聯絡人：呂小姐，電話：(02)2505-6789轉5124)。
