# 網站文章資料_【數位匯流修法與立法】學者：NCC需要擴權才足以因應匯流時代規管_3053

## 一、文章基本資料

- **文章ID**: 3053
- **文章標題**: 【數位匯流修法與立法】學者：NCC需要擴權才足以因應匯流時代規管
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20180730-4
- **文章簡介**: 【匯流新聞網/記者胡照鑫/台北報導】

行政院通過「數位國家，創新經濟發展方案」（簡稱DIGI+方案），從2017年到2025年要投入1700億經費，持續優化台灣寬頻網路涵蓋率，佈建高速寬頻網路建設，推動數位經濟。針對後匯流時代的法規要如何修改與制定，才能符合產業發展需求？新台灣國策智庫邀集學者專家召開「數位匯流修法與立法」圓桌論壇會議，探討如何讓台灣相關法規跟上網路時代的快速變動。

新台灣國策智庫執行委員、台灣數位匯流發展協會理事長吳世昌表示，台灣數位匯流提了20年，但是進展相當緩慢，根據數位匯流協會長期調查，國人知道「數位匯流」這個名詞與意涵的比例相當低，這些跟修法速度跟不上都有一定程度相關。

吳世昌認為，目前除了由國家通訊傳播委員會(NCC)已經提出的電信管理法及數位通訊傳播法草案正在立法院審議中，包括廣電三法要如何修法？如何跟數位匯流融合？都是相當重要的議題。

元智大學資訊傳播系助理教授葉志良表示，台灣人行動上網流量亞洲第一、世界第二，在後數位匯流時代的網路管理更顯重要，現行法規只能防君子不能防小人，因為像是境外業者、侵權網站全都管不到，建議NCC的組織需要調整，將功能擴大，甚至要有部分網路管制權力，才能真正做到規管。(待續)
