# 網站文章資料_20191220 十年之聚 共享時空橫跨之美 元智「同學會」藝術展_706

## 一、文章基本資料

- **文章ID**: 706
- **文章標題**: 20191220 十年之聚 共享時空橫跨之美 元智「同學會」藝術展
- **文章網址**: https://www.yzu.edu.tw/alumni/index.php/tw/20191220
- **文章簡介**: 元智大學藝術中心於12月20日中午12時30分舉行「同學會」展覽開幕式，由陳曉朋、游孟書、涂維政、李明學、蘇孟鴻五位臺灣當代年輕且創作成熟的藝術家，以及十三位元智校友聯手，於元智大學圖書館中庭，以師生聯展方式展出22件作品。

元智大學林志民副校長於開幕時表示，元智大學藝術中心透過各種展演活動，豐富校園藝文色彩；藝術與設計系則致力於培育藝術人才，深耕校園藝術教育。兩者相輔相成，發揮雙乘的效果。感謝五位參展的老師，他們都是台灣當代知名且具份量的藝術家，也都曾在本校渡過幾載的教學歲月，奉獻他們的藝術專業與教育熱情。這次展覽同時也有來自不同領域的十三位畢業校友一同參與，他們的創作成績燦然，不但彰顯老師的教學成果，也可謂是藝設十年的總成果展現。感謝他們將各自學習與成長的果實回饋學校。

2020年是元智大學藝設系畢業十週年，「同學會」以此爲基調，以師生聯展呈現十年跨度的聚首，誠邀了陳曉朋、游孟書、涂維政、李明學、蘇孟鴻五位臺灣當代年輕且創作成熟的藝術家，以及葉天棋、王君懿、張卉欣、葉韋鉦、蔡濟安、洪晧珉、張書甄 林恩崙、林家禾、陳泳丞、錢畇竹、林昭安、徐婷十三位已經在不同領域發光發熱的畢業校友異同參展，透過跨領域創作呈現藝術與設計兩極光譜的再見與回顧，從而延展許多可能與發展。

從垂手可得的生活經驗中滋生出對這個世代的反饋，即是五位藝術家相似之處，而背後蘊育的過程使他們的作品各具特色：陳曉朋游刃有餘於幾何圖象之操作，將有意義的形式有機地轉換成抽象表現，走出西方抽象繪畫歷史中的全新途徑；游孟書擅用玩具物件以翻鑄再削熔手法，傳達自我對文化社會的見解和內在與現實的衝突；涂維政擅長糅合時間和空間，從大眾觀念的「舊的」、「過去的」再創造，產生主流與非主流價值觀之間的對立觀點，促使觀衆從中感知作品所要表達的批判力道；李明學擅以日常生活中的現成物，透過不同媒材要素的組合和安排，表現生活中的黑色幽默，產生新與舊的辯證思考；蘇孟鴻揮灑自如於花鳥山水與春宮圖之間，影射藝術潛藏的危機，使華麗的色彩更顯得虛無，間接反諷和批判藝術的價值。此外，校友們的創作媒材更是多元，不再只是純藝術觀賞，也與許多設計面有所結合，呈現藝術與設計結合的世代進行式的意義，為展覽帶來無限的可能性與想像。

元智大學藝術與設計系兼任藝術中心主任陳冠華表示，這次展覽是藝設系成立以來首次舉辦的師生聯展，可說十年作育英才的珍貴果實，在開放的空間、以開放的形式，展出跨世代不同的觀點、語彙與風格，雖未必是劃時代，但絕對是「畫」時代的重要展覽，期望觀眾在近距離觀賞時，能感受作品所傳遞的理念與能量，進而激盪出個人的情感與思想。
