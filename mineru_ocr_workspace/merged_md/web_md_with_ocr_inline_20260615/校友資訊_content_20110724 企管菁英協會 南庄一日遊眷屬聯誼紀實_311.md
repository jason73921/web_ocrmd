# 南庄一日遊會員暨眷屬聯誼紀實

中華民國企管菁英協會於2011年7月24日，舉辦南庄一日游-會員暨脊屬聯誼·本次活動係由該會理事黄英茵、劉志奇負責主辦·由於已許久未辦理大型之會員聯誼活動，因此本次活動會員參與特別熱烈，報名情形相當踴躍，原訂40個名額，最後一共來了76人。本次活動地點特別選在好山好水的苗栗縣南庄鄉百香農場舉辦·安排精彩活動内容包含南庄老街巡禮、擂茶DIY比賽、客家麻麻耀DIY…等。一路上，知識淵博、深受會員歡迎的許評常務理事和有「菁英協會名嘴」之稱的新任理事長蔡國圳分別擔任兩部游覽車的主持人，會員與眷屬們都認為難得有機會能夠全家出遊，一來有益身心，二來增進與家人互動機會、三來也達到會員聯誼目的，真是一舉數得。
