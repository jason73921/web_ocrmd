# 網站文章資料_台教育部補助私校調薪 校方僅需負擔一成_7201

## 一、文章基本資料

- **文章ID**: 7201
- **文章標題**: 台教育部補助私校調薪 校方僅需負擔一成
- **文章網址**: 
- **文章簡介**: 【大紀元記者林紫馨台灣台北報導】

私立大學校院協進會17日舉辦2023私立大學校院校長會議，討論私校經營與發展。因應公立大專教師調薪，教育部鼓勵私立大專跟進，拉近公、私校教職員的待遇差距。教育部高教司長朱俊彰表示，預計明年編列21億元補助款，符合條件的私校只須負擔一成多，其餘由政府協助。

元智大學校長廖慶榮表示，上一次調薪，政府說補貼七成，但須符合一定條件，認真計算下來，大概還是有45%由學校負擔。他推估，因應明年調薪，以教育部目前宣布的補助方式，私校自行負擔比率還是會超過三成。

廖慶榮呼籲，教育部要確定上述補貼是長期預算，因為一旦停止或中斷，很多學校將無法支撐教職員的薪資，建議政府應考慮比照準公幼模式，設計準公共化大學制度，讓更多學生獲得平等的受教權。（待續）
