# 網站文章資料_國立成功大學舉辦106年度科技部工程領域新進教師暨研究人員研習會_1319

## 一、文章基本資料

- **文章ID**: 1319
- **文章標題**: 國立成功大學舉辦106年度科技部工程領域新進教師暨研究人員研習會
- **文章網址**: https://www.yzu.edu.tw/admin/rd/index.php/tw/?option=com_content&view=article&id=1319
- **文章簡介**: 轉知　

國立成功大學為協助新進教師學者瞭解研究資源取得及研究生涯規劃，舉辦「106年度科技部工程領域新進教師暨研究人員研習會」，廣邀貴校新進教師參加。
 說 明：   
 一、 國立成功大學自105年起協助科技部工程技術研究發展司辦理旨揭研習會，邀請學界先進分享其計畫撰寫、計畫申請及團隊養成之經驗與心得閱歷，引領新進學者以更開闊多元的視野開展其學術研究生涯，並追求學術卓越的熱忱且樂在其中。
 二、 本會議訂於106年11月10日（週五）13時10分至16時50分，假臺北市和平東路二段106號科技部科技大樓2樓13會議室。
 三、 檢附旨揭活動議程表如附件，活動報名請上網站https://goo.gl/forms/eEYTGTtAN7ohiIg73。若有疑問，請不吝來電或來函詢問。（承辦人王靖丰秘書，電話06-2757575轉61200，電子信箱cfwang＠mail.ncku.edu.tw）
 四、 為求活動順遂，惠請貴校協助公告宣傳，至紉公誼。
