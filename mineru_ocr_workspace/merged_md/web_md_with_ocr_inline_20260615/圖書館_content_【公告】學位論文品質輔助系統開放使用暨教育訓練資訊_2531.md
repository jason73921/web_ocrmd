# 網站文章資料_【公告】學位論文品質輔助系統開放使用暨教育訓練資訊_2531

## 一、文章基本資料

- **文章ID**: 2531
- **文章標題**: 【公告】學位論文品質輔助系統開放使用暨教育訓練資訊
- **文章網址**: https://www.yzu.edu.tw/library/index.php/tw/is-2026041691046
- **文章簡介**: 為協助本校應屆畢業碩博士生提升學位論文品質，圖書館配合國家圖書館辦理「學位論文品質輔助系統」（原學位論文相似檢測輔助系統），自即日起開放使用，提供學生進行論文相似內容之檢視與比對。

本系統比對資料來源為國家圖書館「臺灣博碩士論文知識加值系統」，涵蓋歷年中外文電子學位論文（含公開及未公開），與本校既有Turnitin系統之比對範圍不同，建議可搭配使用，以提升檢測完整性。惟本系統僅供學生自我檢測參考，本校論文檢測方式、標準及最終認定，仍以各系所規定為準。

一、使用對象與期間


對象：本學期（114學年度第2學期）應屆畢業碩博士生

期間：115年4月27日至8月31日


二、使用方式
學生以個人Portal帳號登入本校博碩士論文上傳系統，點選「論文檢測」，上傳完整論文PDF檔，完成後可線上查看或下載檢測報告。

三、使用規範


每位學生可使用5次檢測次數

前3次約30分鐘內完成，第4至5次需間隔24小時

若相似度異常偏高，系統將限制該篇文稿操作並列入使用次數


四、教育訓練資訊


管理者場：4月22日（星期三）15:00（4月20日截止報名）點擊報名

學生場：4月29日（星期三）及5月13日（星期三）15:00（4月27日截止報名）點擊報名


五、相關資訊


系統網址：https://wass.ncl.edu.tw

系統使用手冊：學位論文相似檢測輔助系統 - 使用者操作手冊.pdf

系統問題回報：請填寫國家圖書館客服表單( https://forms.gle/qBsdTXVjXwyMbojTA ) ，或洽詢專線電話(02)2361-9132 分機526或504。


敬請應屆畢業碩博士生踴躍使用，並善加運用系統輔助論文撰寫與修正，以提升學術品質。
