# 網站文章資料_經濟部智慧財產局「發明競賽」_6529

## 一、文章基本資料

- **文章ID**: 6529
- **文章標題**: 經濟部智慧財產局「發明競賽」
- **文章網址**: 
- **文章簡介**: 主旨：

敬請貴部(會)協助將2024台灣創新技術博覽會「發明競 賽」徵展訊息轉知各相關學研單位，並鼓勵優秀研發創作 踴躍報名參展，以共同促進專利技術商品化與產業化，至 紉公誼。

說明：

一、本展去年規模創下歷史新高，共有來自23國近500家廠商參 展、使用1,151個攤位，展出超過千項發明專利與創新技 術，發明人及廠商共襄盛舉，3天展期共吸引國內外近5萬 人次進場觀展。

二、2024台灣創新技術博覽會將於本(113)年10月17日至19日於 台北世貿一館盛大展出，預計邀請600家以上國內外發明人 及廠商參展，其中發明競賽展區邀請最近5年內(108年7月1 日至113年6月30日)取得專利或申請中的專利作品參展，並 同步舉辦發明競賽，藉由競賽的肯定，促進得獎作品交易 商機。

三、前述發明競賽報名期間自113年4月26日至7月19日止，展覽 相關訊息、參展報名簡章電子檔，請逕至本展專屬網站(網 址：https://www.inventaipei.com.tw/zh-tw/menu /9F7FD125309115F0D0636733C6861689/info.html)，即可 查詢及下載報名表格等相關資料。
