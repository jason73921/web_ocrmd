# 網站文章資料_胡逢瑛》普丁藉金正恩融入亞洲 _1235

## 一、文章基本資料

- **文章ID**: 1235
- **文章標題**: 胡逢瑛》普丁藉金正恩融入亞洲 
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【中時電子報】

日前北韓繼續試射飛彈，朝鮮半島的緊張關係持續上升。聯合國安理會於今年3月對北韓祭出了第2270號決議加強經濟制裁，這說明了要實現南北韓統一變得越加不可能，經濟制裁最關鍵的衝擊就是重創了北韓原本已經惡化的經濟狀況。然而金正恩政權迄今仍然屹立不搖，加上南韓決定部署薩德系統，使得東北亞形勢更加波譎雲詭。

金正恩做得太過火 

安理會通過決議加強對北韓的制裁，當然是來自於中國和俄羅斯的支持，這對於兩國而言是非常煎熬的決定，代表處理朝鮮半島和平問題的方向進入新的轉折點，也就是透過談判解決朝核危機也變得難以實現。主要原因是金正恩做得太過火，希望迫使金正恩下台。

然而南韓決定部署薩德反導系統，而反導致新的地緣對峙。顯然，經濟制裁沒有迫使金正恩放棄飛彈試射與核武，並且這位獨裁者還繼續以飛彈試射的挑釁方式製造朝鮮半島的危機。這樣一來，日韓部署薩德反導系統（終端高空區域飛彈防禦系統）的理由就變得非常充足。

這將引起中俄兩國邊境安全的危機，特別是中國的反對，整體上更加惡化東北亞的情勢，尤其是中美地緣政治對峙會更加激烈。

如此一來，中俄關係也形成新的挑戰。俄羅斯的態度在於如何緩和朝鮮半島的局勢，並且獲得協調地位的重要角色，這勢必引起中國方面的猜疑。2015年還是俄羅斯和北韓之間的友誼年，兩國還預期在2020年以前完成10億美元的貿易量。現在聯合國安理會的經濟制裁決議不但使兩國經貿關係受阻，並且還有美日韓飛彈系統的威脅。

因此，俄羅斯當前的作法勢必會加強與日韓之間的關係。俄羅斯總統普丁於8月將總統辦公廳主任換成日本專家安東‧萬諾，就是對外釋放出重要的訊息。

俄羅斯希望整合俄中日韓地緣經濟以提升俄羅斯的協調地位。俄羅斯試圖平衡中國對於北韓問題的獨斷地位。在俄羅斯遠東地區日本與南韓投資的擴大是明顯的徵兆，這顯示俄羅斯試圖透過俄中與俄日韓兩組地緣經濟整合的手段來避免東北亞戰爭發生。

俄盼加強經貿互動 

北韓飛彈危機的持續升高會促使俄羅斯拉攏日本與南韓，這將為中俄關係增加了新的不信任因素。俄羅斯高層勢必更加細緻化處理中俄關係。俄羅斯為了使自己快速融入亞洲，定會加強與亞洲各國之間的經貿互動。

台灣或可藉由俄羅斯轉向亞洲政策與東方經濟論壇等平台，促進台俄之間的互動以增加了解；台俄若是增加經貿聯繫，多邊運用軟實力外交，以提高自身在亞太地緣中的價值，倘若再搭配政府的南向政策，並且準確拿捏兩岸關係如何穩定的定位方向，或可重現亞洲轉運站與不沉航空母艦的傳統地緣優勢。（作者為元智大學助理教授、莫斯科國立國際關係大學博士）
