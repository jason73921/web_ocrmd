# 網站文章資料_【生輔組公告】114-2學期就學貸款申請注意事項_6451

## 一、文章基本資料

- **文章ID**: 6451
- **文章標題**: 【生輔組公告】114-2學期就學貸款申請注意事項
- **文章網址**: https://www.yzu.edu.tw/admin/st/index.php/tw/?option=com_content&view=article&id=6451
- **文章簡介**: 申請期限：
115/01/15~02/24
115/01/15~02/28（限延畢生）

收件方式：可現場繳交至活動中心2樓生活輔導組或郵寄至生輔組，逾期恕不受理


收件時間：
115/01/15-02/11　週一至週五　9:00~16:30
115/02/23-03/2　週一至週五　9:00~17:00（02/24開學日收件至18:00)

申請流程：個人Portal登入→免到校註冊→「就學貸款申請登入」→填妥送出→列印或另存PDF「就學貸款申請憑單」或「註冊明細」→國內臺灣銀行分行或線上對保（首次申請限臨櫃辦理；第二次申請者，如連帶保證人或個人、關係人資料基本資料不變，且無申貸生活費、海外研修費者，請善用線上對保方式，可節省更多時間）

繳交資料至生輔組：
1.銀行就學貸款申請／撥款通知書(學校聯)(須簽名或蓋章)
2.賃居契約影本 (申貸校外住宿費者)

注意事項：
1.台灣銀行就學貸款對保作業至2月底截止，因第三階段選課或其他因素導致金額有異動需求者，請於2/26前來電或來信生輔組辦理異動作業。
2.新生首次須至臺灣銀行臨櫃辦理，相關規定及資料準備請參閱臺灣銀行網站及注意事項。
3.可貸款項目為：學雜費、電腦及網路使用費、平安保險費、書籍費(3,000元)、住宿費(校內住宿者依所居住宿舍金額收費；校外賃居學生最高可申貸13,800元)。
4.若有餘額須繳交者，請至個人Portal登入→免到校註冊→註冊明細查詢→下載新的註冊明細帳號→ATM轉帳或便利商店或信用卡繳款。
5.【重要】享有「就學優待學雜費減免」及「大專校院弱勢助學金」資格，請先扣除「就學優待學雜費減免」及「大專校院弱勢助學金」金額後，再辦理貸款程序。
6.【建議】領有公教人員子女教育補助費或其他政府相關補助者，請先扣除公費或子女教育部補助費後之差額再辦理貸款。
7.延畢生可於第三階段選課後（2/28前），再依據選課後的註冊金額辦理就學貸款申請。（臺灣銀行對保作業至2月底截止）；若註冊有異動無法至台銀辦理，請盡速與生輔組聯繫，切勿再等候，以免逾時台銀對保時程而無法辦理。
8.為期迅速安全核撥款項，請至個人portal登錄學生本人之銀行、郵局或其他金融機構帳號，個人匯款一律免扣匯款手續。退款事宜請洽總務處財管組。
9.依規定學生若於選課確定後有多貸款金額要退還臺灣銀行。另，學生該學期有休、退學等因素，其註冊餘款依規定不可退還學生本人，須由學校退還給臺灣銀行。
10.請申貸學生務必留意本組郵件及簡訊，確認是否符合就學貸款可申貸資格。
11.上列未盡事宜處，請依教育部部頒最新相關辦法及要點為主。
