# 網站文章資料_川普在中東極限遊戲三大目的_4163

## 一、文章基本資料

- **文章ID**: 4163
- **文章標題**: 川普在中東極限遊戲三大目的
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20200113
- **文章簡介**: 【聯合報】

伊朗伊斯蘭革命衛隊聖城旅指揮官蘇雷曼尼，3日遭美國無人機襲擊身亡。美國總統川普下令擊斃的手段不但冒險粗魯，勢必加劇中東局勢更加混亂；此次行動，除了彰顯美軍準確命中敵人的武力恫嚇之外，同時暴露美軍在全球各地軍事基地的困難處境。

川普仗著目前無人敢向美國正式宣戰的霸權心態，選擇在地區強權最弱，但戰略資源最豐富的中東地區下手。美國現在自產頁岩油，無須在乎石油價格升跌。美國現在需要的是把現有戰略地點發揮最大效果，來實現川普維持美國軍事和經濟第一的手段。川普選擇對伊朗下手，條件最好：其一，伊朗是川普在國家安全戰略報告中指稱的「流氓國家」，更加使他的支持者崇拜他的氣魄，增加總統權威感，提升連任機會。其二，美國的中東盟友以色列、沙烏地阿拉伯近年也採取多元平衡政策，與中、俄關係友好，美國需要為美軍存在製造紛爭。其三，在伊拉克軍事基地駐守美軍更加有理，在激化兩伊矛盾同時，還可控制波斯灣與通往印度洋之間的阿拉伯海域，實現廣義的「印太戰略」。

襲擊蘇雷曼尼事件不能使伊朗向美國宣戰，但可激怒伊朗報復攻擊。伊朗只會增加對胡塞民兵和真主黨的支持，發動對沙烏地阿拉伯和以色列更多攻擊，造成盟國有苦難言。美國在去年10月默許土耳其的「和平之泉」軍事行動，已助長區域強權採取軍事行動來解決邊境安全問題。（待續）（作者為台灣元智大學助理教授）
