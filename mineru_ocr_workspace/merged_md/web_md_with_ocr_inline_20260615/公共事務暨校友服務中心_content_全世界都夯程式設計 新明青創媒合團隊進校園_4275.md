# 網站文章資料_全世界都夯程式設計 新明青創媒合團隊進校園_4275

## 一、文章基本資料

- **文章ID**: 4275
- **文章標題**: 全世界都夯程式設計 新明青創媒合團隊進校園
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20200406-1
- **文章簡介**: 【大紀元記者徐乃義台灣桃園報導】

資訊世代的衝擊，全世界都在夯的程式設計，已然成為新趨勢，進駐桃園市政府青年事務局新明青創基地的聚忻科技顧問有限公司，提出教育理念為「玩中做、做中學、學中用」，用有趣的程式教育激發學生學習動力，並且運用「STEAM教育」與「曼陀羅思考法」，以情境式教學法讓孩子們在程式設計之前先進行開創式的程式邏輯發想，再透過網頁編寫程式，讓孩子們能夠依照自己的想法去操控麥坤小車、寶比A1等相關教具，創新有趣的程式課程大受好評，贏得許多家長及校方的青睞，也成功轉型學校教育為新創事業。

聚忻科技創辦人馮建中老師擁有多年教育經驗，自身投入教育事業，也培養出許多優秀的子弟兵，近年來，除了種子師資培育，更積極拓展國小A1教育的管道與舞台，目前已進入上大、新坡、頭洲、忠福、新明、林森、快樂及新湖國小等近10間學校授課，也帶領學生參加國內外競賽增進實務經驗，更於「108年國際新創機器人節－TIRT 全能機器人國際邀請賽」競賽中，由馮建中總導師領軍，帶著新坡、新明、林森國小課後社團的學生，分別於國中、國小組中鹽水車STEAM挑戰賽、寶比2.0迷宮挑戰賽及機械手臂MARKER挑戰賽中取得第一名的佳績。

聚忻科技顧問有限公司於108年進駐桃園新明青創基地，不到一年的時間已培育出13位助理教師以及2位專業講師，成員包含明新、元智及健行三所大專院校的學生。預計在今年5月至新明國小開設A1競賽社團課程，課程內容包含無人機教育、A1尋跡避障、三對三機器足球競賽等系列主題課程；暑假7、8月期間，更將前往龍潭高中開設工匠技藝教室課程(主軸為機器人與創客教育)，逐步進入校園，為機器人教育開啟師生創業之篇章。
