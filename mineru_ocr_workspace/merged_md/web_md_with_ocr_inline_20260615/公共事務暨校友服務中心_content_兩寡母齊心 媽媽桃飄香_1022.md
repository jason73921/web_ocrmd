# 網站文章資料_兩寡母齊心 媽媽桃飄香_1022

## 一、文章基本資料

- **文章ID**: 1022
- **文章標題**: 兩寡母齊心 媽媽桃飄香
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【中國時報/賴佑維／桃園報導】 


復興區媽媽桃果農高麗枝10多年前和丈夫返回故鄉務農，但丈夫1年前在果園心肌梗塞過世，2甲大的果園剩她和70歲的母親秋香阿嬤照料。兩人強忍悲痛，在果園努力工作，讓媽媽桃飄香。

高麗枝說，10多年前長輩年事已高，不便到果園工作，因此和丈夫回到部落務農。原本日子還過得去，丈夫也很疼她，從來不讓她做粗重的工作，但去年丈夫因為心肌梗塞，在果園過世，只剩她和母親秋香阿嬤及小五的女兒。她說，為了忘記悲傷，只好不斷努力耕作果園，靠媽媽桃療傷。

秋香阿嬤說，今年因為寒害影響，五月桃產量大減，以自己果園為例，去年套袋12萬顆，今年只剩3萬。她說，生活勉強過得去，但明年的肥料錢不知道怎麼辦。高麗枝樂觀的說，要把媽媽桃賣光光，盡量減少損失。

三光里長林正光說，去年在桃園市原住民族行政局協助下，果農開始有了產地直銷的概念，三光國小也和部落家長合作，推出親師生農場，成功行銷媽媽桃。今年各界資源也紛紛湧入，除了扶輪社協助架設網站外，三光國小也把設備轉移到里辦公室，甚至有學生成立青春桃氣護衛隊，利用周末到部落，協助果農包裝。

青春桃氣護衛隊負責人顏妃吟說，成員大多為元智大學學生和桃園市內國高中生，大家利用周末，約20多人協助果農包裝等。林正光補充，原本媽媽桃深受盤商剝削，但去年行銷成功，現在盤商都知道不能到三光里，果農也終於得到應有的利潤，但希望能打出口碑，讓更多人知道媽媽桃的名氣。
