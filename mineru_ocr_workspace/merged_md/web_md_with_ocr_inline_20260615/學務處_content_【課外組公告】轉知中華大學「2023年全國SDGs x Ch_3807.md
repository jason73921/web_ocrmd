# SOGS ChatGPT四格漫畫競赛

## 活動主題

請使用「聯合國永續發展目標SDGs」作為故事主題，利用指令ChatGPT生成故事内容，再以AI繪固軟體輔助製作出四格漫畫，請列出指令證明，並說明故事核心。

![](images/faf9790f579716b015d44a42643ae075b0228d9b2e22e4b4ae7aaed319895557.jpg)

## 微件时間

即日起至112年08月31日（四）23:59截止。

報名资格

凡全臺大專校院之在籍學生均可報名參加。

## 獎勵方式

特優（遴選1名）：新台幣3.000元奖金及奖狀乙帧。  
優選（选選1\~5名）：新台幣各1.000元獎金及奖狀各乙帽。  
佳作（选選1\~5名）：新台幣各500元奖金及奖狀各乙帧。

於缴交時間內將作品上傳聯絡信箱，收到回信即報名成功。
