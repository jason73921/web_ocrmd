# 網站文章資料_中菲將成戰略命運共同體_1418

## 一、文章基本資料

- **文章ID**: 1418
- **文章標題**: 中菲將成戰略命運共同體
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【中時電子報/胡逢瑛】 


菲律賓總統杜特蒂於10月18日至21日訪問中國，緊接著25日訪問日本3天。美國在意的是中國的做法，陸媒更轉述美媒彭博新聞社18日刊文，認為中國在南海採用多管齊下的微妙戰略，並且中國參與國際事務更加節制和老到。

俄羅斯的轉向亞洲戰略進入平衡。菲律賓和日本都有這種緊迫感。杜特蒂上任後的首次國事訪問設定在北京，向中國傾斜不僅是外交思維的轉變，是否也隱含著冒著美國不滿而下台的風險？杜特蒂向中國和俄羅斯的靠攏能夠獲得地緣空間上的緩解和執政改革的機會嗎？中菲的口頭協議存在嗎？是否這次訪問在書面協議之後有口頭交易的共識。

杜特蒂自中國返菲之後聲稱菲國漁民可以在黃岩島附近捕魚。但沒有見到雙方正式的協議。

這也顯示了中菲雙邊在對話和口頭承諾上發展出的某種共識關係，這應該是中國處理爭議性問題的過渡性做法，包括兩岸關係。

倘若在爭議領土上要做出書面協議，會使僵局無法打開。書面協議確認之後就有永久占領或是永久失去的問題。例如，中俄在簽署領土協議上認知不同，19世紀，中國視為的不平等條約被俄國視為合乎國際法則。21世紀初期，中俄為了發展兩國永久信任關係，雙方都要承擔領土割讓的歷史評判。

中菲擱置南海爭議，要把眼光放在未來的和平相處上，這使得美國亞太再平衡戰略有被斷鏈的可能。川普代表著美國縮減國外軍事開銷的一部分輿論，未來美國不論總統是誰，圍堵戰略都可能面臨調整。因為圍堵中俄兩國不但提供兩國結盟的理由，當俄羅斯反擊的時候，克里米亞收回了，敘利亞塔爾圖斯常設海軍基地也建立了。這些不是俄羅斯主動擴張，而是美國支持反對派推翻政權之後，俄羅斯順勢推展地緣擴張戰略，這是俄國傳統的做法。

美國的戰略緊逼反而使中國的戰略威脅擴張到南海。中菲的信任關係仍受考驗，書面協議背後的口頭交易則是建立互信的緩衝。中國希望降低南海的軍事衝突，那麼，未來支持菲律賓以觀察員身分參加上海合作組織是不錯的選擇。中國把菲律賓納入到建造人工島礁的一帶一路計畫中，將使中菲成為戰略命運共同體。中國正在強化對話協商作為穩定區域局勢的橋梁。（作者為元智大學助理教授）
