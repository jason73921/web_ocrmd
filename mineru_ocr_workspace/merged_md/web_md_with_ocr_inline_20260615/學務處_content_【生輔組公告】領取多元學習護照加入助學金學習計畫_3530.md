# 網站文章資料_【生輔組公告】領取多元學習護照加入助學金學習計畫_3530

## 一、文章基本資料

- **文章ID**: 3530
- **文章標題**: 【生輔組公告】領取多元學習護照加入助學金學習計畫
- **文章網址**: https://www.yzu.edu.tw/admin/st/index.php/tw/?option=com_content&view=article&id=3530
- **文章簡介**: 元智多元學習護照是一種助學金的扶助措施，本學期已核定就學優待、大專弱勢助學金的同學，請儘速到生輔組領取護照加入助學金學習計畫。

學務處生輔組期待您的參與~

領取多元學習護照資格及活動相關請參下列網址：https://www.yzu.edu.tw/admin/st/index.php/tw/higher-education-publicity?view=article&id=1434:2018-03-28-09-03-37&catid=2

或【多元服務讚常見問題集】cloud：https://reurl.cc/vk7ljN

【對象資格_依教育部規定】112年度起適用                               

低收入戶學生／

中低收入戶學生／

特殊境遇家庭子女、孫子女學生／

未符合學雜費減免資格但獲教育部弱勢助學金補助學生／

身心障礙學生及身心障礙人士子女／

家庭突遭變故經學校審核通過者／

原住民學生 ／

懷孕學生、扶養未滿 3 歲子女之學生 ／

請符合以上資格者，請至生輔組/活動中心二樓領取多元學習護照
