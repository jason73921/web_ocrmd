# 學生懲戒公告

中華民國110年5月5日

109 學輔懲字第 010 號

主旨：公告本校系陈、賴、余、陈、黃等五位同學懲戒核定案。

說明：

一、本案依109學年第2次學生評議委員會議決議並經學務長核定。

二、陳陈、賴、余、陳、黃等五位同學因違反學生紀律規範與處理程序實施辦法第三條第三項第四款：行為不當影響公共秩序之安寧或妨害他人者。經學生評議委員會議決議，核予「書面警告」處分，以兹警惕。

三、如不服本校之懲處，應於收到學校通知函之次日起二十個工作日內以書面向學務處提出申訴，逾期不予受理，申訴及處理程序請參閱「元智大學學生申訴處理辦法」。

學務處生輔組敬啟
