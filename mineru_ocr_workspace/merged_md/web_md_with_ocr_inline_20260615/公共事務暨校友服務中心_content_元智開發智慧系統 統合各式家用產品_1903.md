# 網站文章資料_元智開發智慧系統 統合各式家用產品_1903

## 一、文章基本資料

- **文章ID**: 1903
- **文章標題**: 元智開發智慧系統 統合各式家用產品
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20170509-2
- **文章簡介**: 【中央社記者魯鋼駿桃園市8日電】

元智大學電機系近期研發「YZU IoT Gateway」系統，可橋接用戶、設備與服務業者，讓使用者透過手機就能控制家電，或設定各種自動化功能。

元智大學電機所碩二生江銘洋表示，以智慧家庭為例，安裝此系統的機上盒可透過藍芽或Wi-Fi連結家中各種電器硬體，並接收網路上的資訊。設備商不再需要費心設計APP，服務業者也可提供整合性的服務，一般或專業使用者便能享受高品質低成本的物聯網整合應用。

江銘洋說，使用者從整合畫面便能輕鬆得知天氣、家中的溫濕度等訊息，並可以控制電燈、電扇等家電，設定各種自動化功能。例如，設定在用戶回家時自動開啟電燈，一氧化碳過高時啟動警報器。

電機系副教授郭文興則強調，市面上現有各品牌的數位家庭方案，只能連接所屬品牌的裝置，或是只能使用特定一兩種通訊標準，使用者或管理者難以任意連接各種類型設備，自定顯示畫面，或視需要擴充閘道器的功能。

郭文興說，此技術已獲得多項國內外專利，也與交通大學的物聯網雲端團隊測試系統架接，與工研院共同開發數位匯流相關技術，受民間產業委託執行多項產學計畫與技術轉移。1060508
