# 【編劇課之二】編劇技巧：怎麼編出一個好劇本

編劇為一門綜合專業，需要多種能力配合，參與本課程，除了學會編劇，還可以增進以下能力：

1. 表達能力：同一個故事，可以有一百種以上的說法，學會說故事的同時，也在激發各種不同的表達能力。

2. 組織能力：劇本其實是非常講架構和組織的，在編劇中會學到怎麼解剖一個故事，可以大幅增進組織能力。

3. 邏輯能力：編劇重視因果，什麼原因會導致這樣的結果？這劇情合理嗎？可以鍛鍊邏輯能力。

4. 想像力與創造力：編故事可以讓想像力天馬行空的奔馳，大幅增進想像能力，進而得到更多創造力。

5. 換位思考：編劇需考慮不同角色的人物設定，什麼樣的人會做什麼樣的事？說什麼的話？學習編劇等於學習不同人物的思考方式，也能幫助學員理解他人立場，培養同理心。

6. 情緒抒發：編劇可以創造一個自己喜歡的新世界，編寫自己期待中的人物，得到美好的結局，簡言之便是滿足幻想，暫時脫離理性的現實綑綁，得到片刻紓壓。

學習編劇有這麼多好處，歡迎開始學習。另外，我們也將從市場面提及學習編劇如何多方位轉型，讓學員瞭解編劇市場現況與前景，開啟斜槓或打造第二人生。課程中也會讓學員了解編劇的出路、以及可能的職缺喔！
