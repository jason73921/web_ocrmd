# 網站文章資料_超額教師 桃市教局助培養第二專長_1623

## 一、文章基本資料

- **文章ID**: 1623
- **文章標題**: 超額教師 桃市教局助培養第二專長
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【自由時報／記者陳昀／桃園報導】

受到少子化的衝擊，各級學校多面臨減班情形，市議員謝彰文反映，許多學校為了避免教師超額而改聘代理教師，但代理教師流動性高、超額教師又無處可去，恐影響學生的受教權；教育局長高安邦表示，桃園市減班情形趨緩且已增加正式教師員額，教育局也和三所大學合作，鼓勵教師培養第二專長，因應不同科目的教學。 


謝彰文指出，自一○三學年度起，學生數明顯下降，三年來全市國中合計減少二六九班、八八九二名學生，國小也減少二一三班、七○二一名學生，因此國中、國小各有八十三名、十八名超額教師。

她表示，許多學校為了避免教師超額而改聘代理教師，一○五學年度國中、國小分別有五五○位、六七八位代理教師，人數頗多，但代理教師流動性大，頻繁更換可能造成學習不連貫，應降低代理教師的比例，代理教師的增聘與超額教師的安排應取得平衡，尤其國中超額教師更需要有對應的職缺，否則恐影響學生的受教權。

依教師法規定，超額教師必須優先介聘或遷調，但教師只能介聘到有同科目缺額的學校，若全市學校都沒有該科缺額，又無其他適合工作調任，超額教師恐面臨資遣命運，國小教師不分科，國中超額教師面臨的壓力相對較大，尤其是家政、美術等科目。

高安邦表示，教育局嚴格控管員額，鼓勵可能超額的教師儘早培養第二專長，呼籲教師視各校情形安排進修，教育局也和中央大學、中原大學、元智大學合作開設培訓課程，讓教師取得研習時數或修完學程。 

另通過編制調整 增加正式員額

教育局去年底通過教師編制調整案，增加正式教師員額，除了每班至少兩名教師，也從原本的「每九班可增置一名教師」改為「每六班可增置一名教師」，下學年研擬再改為「每四班可增置一名教師」，希望減緩超額教師壓力。
