# 網站文章資料_[轉知] 教育部青年發展署主辦「青年好政論壇-桃園場」_1296

## 一、文章基本資料

- **文章ID**: 1296
- **文章標題**: [轉知] 教育部青年發展署主辦「青年好政論壇-桃園場」
- **文章網址**: 
- **文章簡介**: 一、教育部青年發展署為擴大青年參與，今(106)年暑假擴大 於全國各地共計辦理23場青年好政論壇與38場青年審議活動，其中於桃園市以「如何打造青年社會創新的友善環境？」為主題，以審議民主模式邀請青年學子共襄盛舉，以追求共善的對話與溝通，每場青年審議的結論報告，青年署將送至相關部會及縣市政府作為未來政策規劃參考。

 二、青年好政論壇由教育部青年發展署主辦，遠見天下文化出版股份有限公司為承辦單位，本會為青年好政論壇-桃園場合辦單位。

 三、「青年好政論壇-桃園場」招募30位年滿18-35歲的青年公民，即日起受理報名，報名請洽(02)25420959關懷生命協會蕭業庭專員。

四、「青年好政論壇-桃園場」辦理時間:106年7月23日(日)9:00-17:00。 辦理地點: 桃園終身學習中心802大會議室(桃園市成功路2段7號)。 

五、青年好政論壇也將搭配2場的審議民主活動供青年朋友參加，本會特別將審議民主與動物福利結合，提供關心公共政策也對動物福利有興趣的青年朋友，一個暢所欲言的最佳平臺，藉此強化青年公民養成機制。

 六、審議活動報名網站：https://goo.gl/qpVND3
