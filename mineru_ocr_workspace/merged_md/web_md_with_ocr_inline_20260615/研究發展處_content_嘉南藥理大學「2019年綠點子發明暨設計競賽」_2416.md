# 2019 年綠點子發明暨設計競賽

# 2019 Green Idea Invention and Design Fair July 29\~August 1, 2019, Tainan, Taiwan

## 報 名 簡 章

一、 指導單位：科技部高瞻計畫辦公室

二、 主辦單位：國際發明學會、嘉南藥理大學、中華創意創新發展協會

五、 協辦單位：嘉南藥理大學創新育成中心、台灣數位媒體設計學會、中山大學、

福州大學廈門工藝美術學院、龍岩學院、寧德師範學院

六、 競賽宗旨：近年來快速的科技發展，雖然帶來經濟的繁榮，增進了人類的生

活品質，但也帶來一些新的問題，如對環境生態的衝擊。隨著全球經濟發展，

能源使用量急速增加，造成嚴重的全球暖化之問題。因此，能源與環境保護

是目前各國政府極為重視的議題。有鑒於此，特舉辦 2019 年綠點子(green

idea)發明暨設計展。綠點子即綠色科技，希望能提供未來解決能源與環境保

護議題的創新發明與設計。同時，為提升產學合作與創新研究之風氣，並鼓

勵優秀人才積極投入創新研發工作，希望能藉此發明暨設計展宣揚優秀發

明人之智慧專利創作，促進國際交流，提高專利產品之行銷及市場價值，進

而促成產學研三方進一步的交流與合作。

3. 設計類組：各專門領域之創新設計作品。

4.設計類包含對物品之全部或部分之形狀、花紋、色彩或其結合，透過視覺訴求之創作。

5.參賽對伍如不克出席現場參賽，可委託主辦單位代展。
