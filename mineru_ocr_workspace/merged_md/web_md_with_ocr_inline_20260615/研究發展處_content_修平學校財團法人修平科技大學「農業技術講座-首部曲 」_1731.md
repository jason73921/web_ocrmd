# 網站文章資料_修平學校財團法人修平科技大學「農業技術講座-首部曲 」_1731

## 一、文章基本資料

- **文章ID**: 1731
- **文章標題**: 修平學校財團法人修平科技大學「農業技術講座-首部曲 」
- **文章網址**: https://www.yzu.edu.tw/admin/rd/index.php/tw/?option=com_content&view=article&id=1731
- **文章簡介**: 主旨：檢送本校將於107年5月31日舉辦「農業技術講座-首部曲」，歡迎貴校師生報名參加，請查照。
說明：
一、本校大學社會責任計畫「具適地性服務的農業創生系統」為提升夥伴農友及師生農業專業知識，此次邀請戴振洋老師講解「蔬菜設施栽培」與「有機蔬菜栽培」應用主題，協助農民於農作技術的自我精進，透過講座能延伸學校師生對有機農業的認知。
二、活動內容：
(一)時間：107年5月31日（星期四）上午9時00分至14時00分（9時至9時30分報到）。
(二)地點：修平科技大學A607會議室。
(三)報名方式：採線上報名，即日起至107年5月30日（星期三)中午12時止，報名網址：<活動海報>。
三、檢送附件1_活動海報、附件2_交通方式。
