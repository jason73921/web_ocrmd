# 網站文章資料_宜蘭縣金門同鄉會張太白頒三采文化獎學金及敬老金_1489

## 一、文章基本資料

- **文章ID**: 1489
- **文章標題**: 宜蘭縣金門同鄉會張太白頒三采文化獎學金及敬老金
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【大成報記者于郁金/宜蘭報導】

宜蘭縣金門同鄉會暨三釆文化公司張輝明董事長11月26日於宜蘭大學校區內樂米餐廳辦理105年獎學金及敬老金頒獎典禮活動；宜蘭金門同鄉會理事長張太白親臨主持並致詞勉勵在學金門子弟要更優秀向學，頒獎典禮圓滿順利完成。

應邀觀禮者有金門縣政府駐台北部服務處主任孫允理、陳再旺、戴河生(代表金門縣長陳縣長福海問候鄉親)、臺北市金門同鄉會總幹事張自福(代表臺灣金門同鄉會總會總會長李台山)、宜蘭縣金門同鄉會理監事等人，宜蘭縣金門同鄉會理事長張太白勉勵獲獎學生繼續努力向學，以後有能力，也來回饋家鄉與鄉親子弟。

11月26日頒贈典禮在宜蘭大學校區內樂米餐廳舉行，計有6位70歲以上長輩獲贈敬老金，包括許金玉(80歲以上)、蔡春生、吳克由、黃獻榮、薛永通、洪維來；獎學金有元智大學蕭辰合、宜蘭大學陳冠中、羅東國中范毓宸、國華國中黃潁蘋、黃鈺斌等。

三采公司董事長張輝明原籍金門縣金沙鎮人，在臺灣出版界闖出一片天的，2014年起，提供宜蘭縣金門同鄉會獎學金，鼓勵優秀好學的旅台鄉親子弟；張輝明於宜蘭縣金門同鄉會設獎學金，緣起於他的金門縣金沙國中同學、宜蘭縣金同會理事長張太白之牽線，除了鼓勵金門學子，亦由於張輝明是白手起家，艱苦創業有成典範，張太白期望藉由他成功的故事，啟發旅台金門同鄉會子弟奮發向上的動力。 


民國47年出生的張輝明，滿月即遇823砲戰，父母帶他搶搭登陸艦到臺灣避戰禍；戰後再返金門定居，從小學到高中參加美術比賽都拿第1名；金門高中畢業考取海洋學院(現海洋大學)水產製造系，念了3個月沒興趣，重考考上師大美術系；大學期間，他到廣告公司、百貨公司擔任美工助理，26歲退伍後，任職協和工商美工科老師約1年；27歲任職臺北市立士林高商廣設科，開啟他日後10年教書之路。

曾獲選「金石堂2012年出版風雲人物」張輝明在30歲那年，為因應教學需要，於寒暑假期間，即自寫、自畫、自編教材；32歲正式成立「三采文化」，以美術實用工具書為主，37歲離開教職，全心投入出版事業；張輝明從1張桌子起家的出版手工業，發展成商業美術應用參考書專家，將「三采文化」推上圖解工具書的第1把交椅，奠定市場地位。

宜蘭縣金門同鄉會平常服務鄉親更是不遺餘力，除結合家鄉傑出企業家辦理獎助學金及敬老金頒發活動外，同時對旅台金門鄉親之關懷慰問，可從這些年來的鄉親服務案件，就可以看得出宜蘭縣金門同鄉會的努力，是有目共睹的。張太白理事長表示，幫鄉親解決困難與疑難雜症，也是宜蘭縣金門同鄉會任務之一，他責無旁貸，也請大家在有需要時，及時對有難鄉親伸出援手。

宜蘭縣金門同鄉會簡介： 

1、於民國99年6月20日成立，由宜蘭縣頭城國中退休教師李增河經過半年多的走訪，尋覓30餘位金門鄉親做為發起人，歷經3次籌備會，才成立的第14個旅台金門同鄉會。 

2、宜蘭縣金門同鄉會原有「金聲蘭語」網站：希望藉此平台得以「傳播金門人的聲音，訴說蘭陽地的話語」。 

3、成立金門故事館：藉由書報雜誌和影音資訊，讓鄉親得以深入了解有關金門的(人、事、時、地、物)。 

4、宜蘭縣金門同鄉會標誌：將金門莒光樓、宜蘭龜山島做成連結。 

(209行善家族記者于郁金)
