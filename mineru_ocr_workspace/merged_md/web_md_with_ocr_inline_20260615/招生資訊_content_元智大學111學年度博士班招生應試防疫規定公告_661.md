# 元智大學 111 學年度博士班招生應試防疫規定

公告日期:111/4/28

本校 111學年度博士班口試依簡章原訂日期(111/5/3-5間)如期舉行「到校口試」。為因應「嚴重特殊傳染性肺炎」防疫，以下應試防疫規定煩請考生務必配合辦理：

1. 符合本校「招生入學視訊面試規範」第一條各款因素並附有證明者，可依規定申請視訊面試：考生須簽署本校「元智大學視訊面試應試切結書」，併同本校「元智大學招生入學視訊面試申請表」於報考系所（學位學程）面試日 2 天前，以電子郵件、傳真等方式送達招生系所（學位學程）。未依規定提出申請，一律不予受理。詳細規定請參見本校「招生入學視訊面試規範」辦理。

2. 口試當日若屬於需「自主健康管理且無症狀」者，或應試當天若有「發燒」、「咳嗽」、「呼吸道症狀」、「腹瀉」、「嗅覺喪失」等身體不適症狀者，應於試前主動告知本校各系所承辦人員，以利安排至「隔離考場」應試。

3. 各系所承辦人員連絡電話：總機03-4638800，分機如下:

管理學院博士班，分機:6052。

資訊管理學系博士班，分機:2791。

文化產業與文化政策博士學位學程，分機:2701。

電機工程學系博士班(甲組)，分機:7101、7102。

工業工程與管理學系博士班，分機:2501。

化學工程與材料科學學系博士班，分機:2551。

機械工程學系博士班，分機:2451。

4. 考生入校請主動出示應考証，並全程佩戴口罩應試：配合本校校門防疫管制措施，考生煩請配合主動出示應考証及證件入校。考生除試務人員核對身分時，須配合暫時取下口罩，辨識身分後再戴回；其餘時間，請考生全程佩戴口罩應試。

5. 提前報到，體溫量測、手部消毒：請依各系所規定時間提前辦理報到(詳見各系所網頁公告)，以配合檢測及執行量體溫、手部消毒等防疫工作。

6. 考生應落實自我健康狀況監測：如應試當天若有身體不適症狀者，應全程配戴口罩、保持社交距離，並主動告知監(試)務人員，依試務人員指示進入規定試場應試。

7. 不開放陪考：非必要人員請勿到校，以避免造成群聚或交叉感染。考生若有特殊需求，請事前申請。

8. 本公告未盡事宜，悉依政府各項最新防疫規定、本校簡章及相關招生規定辦理。
