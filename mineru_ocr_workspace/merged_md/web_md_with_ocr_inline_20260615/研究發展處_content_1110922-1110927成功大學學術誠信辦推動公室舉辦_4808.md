n   備註：

n   本辦公室將於活動開始前一天(9/21)寄發觀看影片之網址及測驗試題，請至報名時填寫之信箱確認。9月22日至9月27日期間24小時皆可上網觀看，過此時間將關閉測驗卷。

n   此課程觀看之影片為2022年7月21日學術誠信系列講座之錄影影片，若先前已取得該場次的認列時數，請勿再行報名此場次，若發現將直接取消時數認列不另行通知。

n   參與學員於前三次測驗中任一次測驗達80分以上者，核發2小時學術倫理研習時數證明。第四次測驗之後的分數將一律刪除不計。

n   核發時數預計於活動結束後一週內完成審核，校內人士將直接匯入學術倫理教育時數登錄平台，校外人士將寄發研習證明電子檔至報名信箱。

n   此活動不需簽到退，若收到系統寄送提醒簽到退之信件，請直接忽略。
