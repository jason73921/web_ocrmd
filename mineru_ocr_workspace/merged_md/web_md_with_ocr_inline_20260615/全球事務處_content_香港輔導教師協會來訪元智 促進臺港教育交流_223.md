# 網站文章資料_香港輔導教師協會來訪元智 促進臺港教育交流_223

## 一、文章基本資料

- **文章ID**: 223
- **文章標題**: 香港輔導教師協會來訪元智 促進臺港教育交流
- **文章網址**: https://www.yzu.edu.tw/admin/gao/index.php/tw/xiang-gang-fu-dao-jiao-shi-xie-hui-lai-fang-yuan-zhi-cu-jin-tai-gang-jiao-yu-jiao-liu
- **文章簡介**: 由中華青年交流協會主辦，邀請香港輔導教師協會於2023年4月9日至2023年4月12日來台參訪，於4月12日返港前最後一站來到元智大學。
- **文章全文**: 本次香港輔導教師協會由協會何玉芬主席，同時亦是迦密愛禮信中學校長領隊，率領16名香港中學校長、輔導老師，並由中華青年交流協會鄭婷文秘書長、台北經濟文化辦事處專案主任朱文潔、副主任廖潮鏗偕同來訪元智。元智大學則由全球事務處李婷副國際長、企業書院魏煒圻書院長、管理學院AACSB辦公室翁華鴻主任接待，並邀請四名不同科系的香港學生參與座談。 中華青年交流協會秘書長鄭婷文表示，本次參訪團來元智拜會是因元智大學管理學院獲得AACSB國際商管學院認證，並且作為全台第一個「新雙語」大學為兩大吸引參訪團拜會之亮點。

座談會中由全球事務處李婷副國際長以親切的廣東話介紹元智大學概況、遠東集團支持、學術交流跨域等特色；企業書院長分享元智大學在學生職涯、實習之規劃與願景；AACSB翁主任分享管理學院十多年來經營，致力打造元智管理學院成為全球認證之商管學院，皆獲得在場嘉賓肯定。


座談會中雙方交流

除此之外，在綜合交流中，何玉芬校長亦邀請本校四名在校港生分享在元智大學的學習特色。其中管理學院王同學分享管理學院課程相當活潑，亦有報告作業以演戲方式進行，認為元智大學提供一個自由而且多元的學習平台；而資訊學院王同學則分享自己是大學畢業，先工作後才決定申請碩士班入學，因為看見元智在課程設計上更能幫助學生與職涯接軌，並看見學校在實習計畫上的努力，故在眾多資訊科系學系的學校上選擇了元智大學就讀。

何玉芬校長表示，雖然過去沒有拜訪過元智大學，但一直聽說校園相當美麗，今日走訪以後，發現不僅校園美麗，在學校教育的軟實力、硬實力上也相當紮實，邀請元智大學教授們可以來香港各高校拜訪。


何玉芬主席率香港輔導教師協會來訪元智

本次交流雖然只有短暫的一小時，但參訪團仍對元智大學留下相當深刻的印象。期盼在未來的交流上有更多的機會，讓元智在香港被看見！
