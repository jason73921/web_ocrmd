# 網站文章資料_【公告】使用圖書館電子資源，請遵守使用規範！_396

## 一、文章基本資料

- **文章ID**: 396
- **文章標題**: 【公告】使用圖書館電子資源，請遵守使用規範！
- **文章網址**: https://www.yzu.edu.tw/library/index.php/tw/ebook-sources
- **文章簡介**: 本校近日連續發生圖書館電子資料庫超量下載全文情形，輕者單一IP被資料庫廠商停權，重者導致全校均被停權無法使用，影響所有師生的權益，因此再度呼籲：

請全校師生使用電子資料庫請遵守各資料庫之使用規定，並限於個人學術研究或教學目的使用，不得有商業行為及連續、大量、有系統地下載檔案(請避免同日、同 時段、同IP、同期刊或同卷期大量下載電子全文)，而且禁止使用Open Proxy或自動下載軟體，以避免誤觸相關智慧財產權規定！

再次呼籲全校師生，請尊重智慧財產權，雖然部份資料庫(如Endnote軟體)提供批次下載功能，但請您合理使用！切勿濫用貼心功能進行不合理甚至非法之行為，影響全校師生使用的權益！
