- **文章簡介**: 一、金車文教基金會期望透過徵展發掘臺灣青年藝術新勢力， 旨推廣青年創作，繼而有更多發展空間與機會，特辦理 「2024金車美展徵件」。
二、活動辦法與說明：
(一)徵件媒材：

１、平面類 ( 油畫、水彩、水墨、膠彩、壓克力、版畫… 及其平面類創作 )

２、立體類 ( 木雕、金工、石雕、紙雕、陶藝、裝置、織 品…及其立體類創作 )

３、其他藝術相關創作
(二)作品規範：需有8件以上作品圖檔，圖檔須標示申請人姓 名、作品名稱、媒材、尺寸、年份。
(三)參賽資格：不限國籍，年滿20歲至40歲之藝術創作者。
(四)收件日期：2023年7月1日起至7月31日止，採網路線上報名。
三、活動相關辦法：詳情請見「金車文藝中心」官網活動頁面 https://www.kingcarart.org.tw/ →競賽與徵件 →2024 金車美展徵件
四、本會活動聯絡人：吳昀臻，電話：02-2562-8629，傳真： 02-2100-1185，E-mail：kingcarart@gmail.com
