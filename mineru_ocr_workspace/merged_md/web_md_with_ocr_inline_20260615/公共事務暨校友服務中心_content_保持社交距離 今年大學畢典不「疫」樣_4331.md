# 網站文章資料_保持社交距離 今年大學畢典不「疫」樣_4331

## 一、文章基本資料

- **文章ID**: 4331
- **文章標題**: 保持社交距離 今年大學畢典不「疫」樣
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20200508
- **文章簡介**: 【自由時報/記者吳柏軒、林曉雲/綜合報導】

中國武漢肺炎疫情尚未結束，各大學近期規劃六月畢業典禮，教育部已多次通報要各校依中央指示保持社交距離，各校紛紛取消傳統畢典形式，採人流分散、戶外場地、線上視訊等方式辦理，甚至計算撥穗時間、取消握手等。

辦畢典 教長︰依規範確實防疫
教育部長潘文忠昨表示，畢業季將至，各級學校也陸續規劃相關典禮，但日前已多次通報，要求各校要依照中央流行疫情指揮中心的公共集會指引、社交距離等規範，據了解，許多學校以依循規範籌畫畢典，雖然中、小學及大學等規模不一，仍須依規範確實防疫。如台灣大學今年畢典就改為戶外，將在椰林大道上搭棚，且因日前校內規範停辦六十人以上集會，今年畢典僅安排卅三人上台受獎，取消校園巡禮，跟校長採點頭不握手，撥穗時間僅五秒，合照僅十五秒等，嚴控接觸及社交距離。

中興大學、中央大學、交通大學等則取消校級畢典，中興甚至也不辦系所畢典，改拍師長祝福影片、畢業歌ＭＶ等留下回憶，優秀畢業生榮譽綵帶及獎狀也改用影片拍攝方式留念。台灣師範大學今年畢典改由代表出席，因預計有二百人，將採分流方式入場撥穗，讓每批不超過百人，符合指揮中心室內不逾百人集結的規範。元智大學也提出分時、分流、分地等方案，並考慮讓學士自行撥穗。（待續）
