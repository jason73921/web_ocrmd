# 網站文章資料_未來科技展推動辦公室「2018未來科技展」_2160

## 一、文章基本資料

- **文章ID**: 2160
- **文章標題**: 未來科技展推動辦公室「2018未來科技展」
- **文章網址**: https://www.yzu.edu.tw/admin/rd/index.php/tw/?option=com_content&view=article&id=2160
- **文章簡介**: 主旨：檢送科技部「2018未來科技展」活動簡介及宣傳海報乙張，會請貴校師生踴躍參觀並協助海報張貼宣傳，請查照。

說明：

一、為展現我國前瞻科研成果，聚焦產業及社經需求，科技部於本年12月13日至15日每日10:00~18:00假台北世界貿易中心展覽三館(台北市信義區松壽路6號)舉辦「2018未來科技展」，本展集結我國學研界科研成果、科學園區、國家實驗研究院、同步輻射中心、災害防救中心等研發成果，匯集全國101個學研單位參與，展出123件具科學突破與產業應用的前瞻技術，包含生技與新藥、醫材、AI智應用及電子與光電、金屬化工與新穎材料等應用領域，帶領民眾一同見證未來科技所帶來的生活樣貌，提高台灣研發實力的能見度，誠摯邀請學校團體組團參觀，體驗創新研發能量。

二、惠請鼓勵學校師生安排校外教學踴躍參與，以激發學生科學研究興趣，另為配合主管教育行政機關實施學生公共服務課程，歡迎青年學生智本展覽活動服務，培養學生自我成長、瞭解科技發展、熱心參與公共事務之熱忱。活動網站、團體參觀報名及公共服務課程報名：http://www.futuretech.org.tw，活動詳情請洽詢未來科技展推動辦公室(02)2577-4249分機233邱喬昕小姐。
