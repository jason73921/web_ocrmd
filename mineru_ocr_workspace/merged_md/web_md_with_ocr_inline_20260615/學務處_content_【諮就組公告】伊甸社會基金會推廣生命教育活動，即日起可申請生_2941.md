# 網站文章資料_【諮就組公告】伊甸社會基金會推廣生命教育活動，即日起可申請生命教育活動_2941

## 一、文章基本資料

- **文章ID**: 2941
- **文章標題**: 【諮就組公告】伊甸社會基金會推廣生命教育活動，即日起可申請生命教育活動
- **文章網址**: https://www.yzu.edu.tw/admin/st/index.php/tw/?option=com_content&view=article&id=2941
- **文章簡介**: 說明：


本會乃一社會福利單位，由劉俠女士（作家杏林子）創辦於民國七十一年十二月一日，以關懷台灣百萬名身心障礙者身、心、靈的全人復健為優先目標，再擴及兒童、老人、婦女、原住民等弱勢族群，並提供整體性服務。民國107年本會成立全球首間唯一以身心障礙者權利公約(簡稱CRPD)為主軸的實務研究中心，持續推動公約落實，其宗旨為「促進、保護和確保所有身心障礙者充分及平等享有所有人權及基本自由，並促進對身心障礙者固有尊嚴的尊重。身心障礙者包括肢體、精神、智力或感官長期損傷者，其損傷與各種障礙相互作用，可能阻礙身心障礙者與他人於平等基礎上充分有效參與社會。」

懇請 貴校給予參與校內生命教育及特教宣導相關活動機會，安排於111年9月至112年12月間辦理生命講座與身障體驗活動，期待大眾透過活動進而認識並接納身心障礙者，學習他們跨越障礙的勇氣與信心，讓改變從體驗開始！

生命教育講座：身經百煉，凡事非凡！說出跨限人生！

Eyes遇見愛：超越眼盲限制，追夢勇往直前！以樂音體現生命的無限可能！

身障體驗活動：突破框架，掙脫縛繭！挑戰極限可能！
