# 網站文章資料_學測考到一半衝廁所 不扣分_774

## 一、文章基本資料

- **文章ID**: 774
- **文章標題**: 學測考到一半衝廁所 不扣分
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【中時電子報】

大學入學考試中心昨召開考試委員會議，今年大學學測有331件試場違規事件，有考生作答未滿60分鐘就衝出去上廁所，原本考區建議將考生該科0分計算，但委員會討論後，認為學生是忍不住了才有此行為，最後大翻盤，不扣分。

此外，台南二中考區在考國文科時，因為二中志工隊管制樓梯，造成考生晚進試場，有2間試場84位考生被延誤2分鐘作答（考試時間120分鐘），因此加分2/120的比例來補救；還有一間試場又加上監考老師發錯答案卷，加分3/120的比例。

在這考場被耽誤的主要是台南一中考生，台南一中校長張添唐表示，考生的心情受到嚴重影響，很多人都表達不滿，「是有災情的」。

張添唐表示，他雖然接受大考中心的處理結果，但是希望試務單位今後能夠避免再發生類似疏失，他也要回學校，好好安撫學生。

亂畫答卷 扣減1級分 

大考中心昨召開考試委員會議，討論今年學測考生違規狀況如何處理。不少「天兵考生」在答案卷上寫不相關的文字或圖案，譬如寫「閱卷老師辛苦了，眼睛休息一下吧」，還有考生寫「My mother is a beautiful girl」並畫上「Mother fucker」等大型不雅文字，更有許多考生畫漫畫，依規定，該科扣減1級分。

此外，新北有一A考生考英文科時，撿起掉落的答案卡，順手反面蓋在桌面，B考生舉手反映答案卡遺失，監試人員檢視後發現，B考生的答案卡被A考生拾起。考試委員會最後判定A考生沒有舞弊意圖，因此不扣分。

推算落點 要看近2年 

學測將於2月18日寄發成績單，緊接著展開大學甄選作業，升學輔導專家劉駿豪分析，今年台大國企系、中國醫大藥學系、元智資管等系在「個人申請」的篩選科目都做了巨大改變，考生務必比對近2年的變化，不能只是用去年分數推算落點。

根據統計，105學年度大學甄選入學的招生名額創歷年新高，「繁星推薦」、「個人申請」合計超過7萬名，約占總名額的6成4，比今年增加3788名。

大考中心副主任沈青嵩宣布，明年學測可能在1月21日（六）、22日（日）或22日、23日（一）兩組日期中擇一舉行。
