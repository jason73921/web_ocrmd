# 網站文章資料_玩具全齡共玩授旗授包 逾百社服大專志工參與  _4448

## 一、文章基本資料

- **文章ID**: 4448
- **文章標題**: 玩具全齡共玩授旗授包 逾百社服大專志工參與  
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20200713-2
- **文章簡介**: 【大紀元記者徐乃義台灣桃園報導】

台灣玩具圖書館協會回收二手玩具10多年回收逾200公噸，每年發起「傳愛背包」行動，號召全台大專院校學生及民間團體背起玩具背包，愛心傳播至海內外各角落。今年疫情攪局，仍澆不息二手玩具傳愛的熱情，11日於圳頭國小舉辦「玩具全齡共玩講座暨授旗授包典禮」，吸引逾百位社服團體和大專志工共襄盛舉。

2016年起，和泰汽車(TOYOTA)與台灣玩具圖書館協會合作，攜手支持210組志工團隊回饋社會，全台124間TOYOTA服務據點進行玩具回收，每年回收量高達40公噸，今年更持續邀請大家一起分享玩具分享愛。

今年因疫情致使海外服務隊伍大多無法出團，改為國內偏鄉服務為主，11日「玩具全齡共玩講座暨授旗授包典禮」，吸引大專院校、單位與團體，參與玩具傳愛的活動。透過講座瞭解二手玩具背後隱含的深意，能將這股熱情傳達到每位被服務的對象手裡。

桃園市教育局專門委員林淑芬，期許海內外志工團隊，將傳愛火炬一棒一棒傳承綿延不絕。台灣玩具圖書館協會辦理大傳愛背包活動已12年，每年都匯集台灣人的愛心，結合青年學生服務學習，上山下海，到偏鄉離島甚至海外服務學習。

在「經驗交流分享」，請到曾申請傳愛背包的元智大學學生分享志工服務經驗外，更邀請到專業教保員及社工師分享，針對不同類型服務對象，如何挑選合適的玩具並帶入課程及活動中，讓公益行動不再困難而是能更貼近且瞭解服務對象需求。（待續）
