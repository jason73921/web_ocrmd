# 網站文章資料_國際積體電路電腦輔助設計賽 元智資工獲佳績_3380

## 一、文章基本資料

- **文章ID**: 3380
- **文章標題**: 國際積體電路電腦輔助設計賽 元智資工獲佳績
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20181128-6
- **文章簡介**: 【大紀元/記者徐乃義】

2012年起，「國際積體電路電腦輔助設計軟體製作競賽(CAD Contest at ICCAD)」已成為電子設計自動化領域最重要的國際賽事之一。在教育部以及國內外該領域知名企業和研究機構協辦下，每年皆可吸引數百位來自世界各地的研究生和大學生團隊參賽。元智大學資工系鄭立程、翁甫廉、黃崇瑋、李瑞軒等大學部學生，在陳勇志助理教授指導下，參加「2018國際積體電路電腦輔助設計軟體製作競賽」，於競賽項目A中獲得第二名佳績。

今年度總有來自11個國家、136個團隊角逐三項競賽項目，元智資工團隊在項目A「Smart EC: Program-Building for Name Mapping」拿下第二名，並在美國加州聖地牙哥舉辦的「IEEE/ACM國際電腦輔助設計研討會（ICCAD）」獲頒獎牌及獎金三萬元。
元智資工系教授陳勇志表示，競賽與傳統程式設計競賽不同，除了賽程需花七個月時程外，問題的複雜度也提高。電子設計自動化的問題，通常都沒有唯一的答案，參賽者往往必須開發一個能在較短時間內計算出最好結果的程式，除了要有專業知識與程式實作能力外，也考驗學生的細心與恆心。

由於競賽題目接近業界需求，對提升學生的專業能力與降低產學落差有相當助益。
