# 網站文章資料_行政院農業委員會水土保持局「一百零八年大專院校農村實踐共創計畫」_2203

## 一、文章基本資料

- **文章ID**: 2203
- **文章標題**: 行政院農業委員會水土保持局「一百零八年大專院校農村實踐共創計畫」
- **文章網址**: 
- **文章簡介**: 主旨:「一百零八年大專院校農村實踐共創計畫」徵件,歡迎上網報名,請查照轉知。

說明:

  一、行政院農業委員會水土保持局為鼓勵各大專院校教師運用其專業課程,發揮  

         跨界應用價值解決農業及農村等相關問題,結合核心知識實踐應用於農村場  

         域之方式,以達跨界共創及教育創新之目的,「大專院校農村實踐共創計畫」

         已開始徵件,期限至107年12月31日止。

  二、補助對象為全國公私立大學校院,以課程或專題討論或社團輔導課程為單位,

         教師為計畫主持人提出申請,搭配可實踐專業核心知識之農村場域進行實習

         、實作、專題設計或駐村等課程執行。

  三、今年採線上報名,詳情請至計劃官網(http://rcampus.swcb.gov.tw/)了解。
