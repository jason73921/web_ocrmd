# 網站文章資料_元智呈現藝術與設計兩極光譜之間可能與發展_4144

## 一、文章基本資料

- **文章ID**: 4144
- **文章標題**: 元智呈現藝術與設計兩極光譜之間可能與發展
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20191223-1
- **文章簡介**: 【大紀元記者陳建霖台灣桃園報導】

元智大學藝術中心20日中午12:30舉行「同學會」展覽開幕式，展覽共計22件作品展於元智大學圖書館中庭，以師生聯展方式呈現十年跨度的聚首，邀請到曾任教於藝設系的五位老師，包含陳曉朋、游孟書、涂維政、李明學、蘇孟鴻等台灣當代成熟且具份量的藝術家，以及十三位在不同領域各自發熱的畢業校友一同參展，透過跨領域創作呈現藝術與設計兩極光譜之間的各種可能與發展。

元智大學林志民副校長表示，元智大學藝術中心透過各種展演活動，豐富校園藝文色彩；藝術與設計系則致力於培育藝術人才，深耕校園藝術教育。兩者相輔相成，發揮雙乘的效果。感謝五位參展的老師，他們都是台灣當代知名且具份量的藝術家，也都曾在渡過幾載的教學歲月，奉獻他們的藝術專業與教育熱情。這次展覽同時也有來自不同領域的十三位畢業校友一同參與。

從垂手可得的生活經驗中滋生出對這個世代的反饋，是五位藝術家相似之處，而背後蘊育的過程使他們的作品各具特色：陳曉朋游刃有餘於幾何圖象之操作，能將有意義的形式有機地轉換成抽象表現。（待續）
