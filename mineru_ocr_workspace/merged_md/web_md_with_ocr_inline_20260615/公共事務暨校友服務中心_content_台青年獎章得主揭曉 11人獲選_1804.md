# 網站文章資料_台青年獎章得主揭曉 11人獲選_1804

## 一、文章基本資料

- **文章ID**: 1804
- **文章標題**: 台青年獎章得主揭曉 11人獲選
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/2017033002
- **文章簡介**: 【大紀元記者鐘元臺灣臺北報導】

3月29日是臺灣青年節。2017年青年獎章得獎名單28日揭曉。主辦單位中國青年救國團表示，今年共有318位青年參加選拔，最後選出11位青年獎章得主，年紀最長的是43歲的林宗坤，最小的是17歲的曹羽蘋。

11位青年獎章得主，為全國青年樹立良好典範 
 青年獎章為前總統蔣經國時任救國團主任時所創，自民國45（1956）年開始頒授，宗旨為“獎勵青年優良德行與傑出成就，表揚青年對國家社會之重大貢獻”，迄今已邁入61年。救國團主任葛永光表示，今年共有11位青年獲選，歷年計有779位傑出青年榮獲這項最高榮譽，為青年樹立楷模，為社會提供學習典範。 
 11位青年獎章得主中，台南歸仁國小校長林志政曾協助推動多項特殊教育業務及友善校園，並榮獲“教育部105年度特殊教育優良教師”及第五屆教育大愛“菁師獎”，他還將獎金捐贈給學校的特教基金及民間身心障礙協會。

軍醫局三軍衛材供應處處長林宗坤，在20多年的軍旅生涯中，曾榮獲國防部“景風甲種一星獎章”與“優良軍醫獎狀”；在民國98（2009）年與紅十字會前往印尼亞齊省（南亞大海嘯之處）參加醫療義診；民國99（2010）年2月邦交國海地發生大地震災難後，他主動配合國防部與衛生福利部前往第一線支援醫療服務。

高雄市政府消防局消防員施武鈞從小立志要為民眾服務。在消防任務中，他曾參與南橫馬希巴秀山山難、711潭美颱風水災、高雄81氣爆、台南0206維冠大樓地震倒塌事件等重大災害後的救援，都顯示出消防工作是一份具有重大責任的神聖工作，必須充實自我知識技能面對未來的各種災況。此外，經由宣導防火常識更為重要，104（2015）年帶領前金婦女防火宣導隊榮獲消防署評鑒“優等”殊榮。

元智大學電機系副教授施皇嘉勤奮學習，為改善家庭經濟狀況，從技職體系的大安高工資訊科到國立清華大學電機博士，並獲得臺灣的指標性學術獎項，包括李國鼎青年研究獎、潘文淵文教基金會考察研究獎、中國電機工程學會優秀青年工程師獎、元智大學青年學者獎等。
