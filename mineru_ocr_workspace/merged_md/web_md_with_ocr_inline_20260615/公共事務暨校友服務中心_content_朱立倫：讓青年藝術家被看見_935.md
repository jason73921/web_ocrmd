# 網站文章資料_朱立倫：讓青年藝術家被看見_935

## 一、文章基本資料

- **文章ID**: 935
- **文章標題**: 朱立倫：讓青年藝術家被看見
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【中國時報/王揚傑／新北報導】 


新北市政府文化局首次為5所大學藝術相關學系舉辦《青春分號；不句號》畢業祭聯合展覽，免費提供展場及宣傳，為畢業生省下不小的開銷。台灣藝術大學雕塑系總召蘇煜翔說，這次辦畢業展的花費每人平均約7000元，足足是學長姊的一半，減輕同學不少負擔。

文化局表示，本次5校7系9檔次的畢業祭聯合展覽，不但免費提供4月到6月檔期及展覽空間給畢業生展出作品，同時還協助畢業生展品保險、行銷宣傳及印製展覽相關文宣，畢業生僅需提供作品及負責佈展，實質替畢業生節省龐大開銷。

元智大學資傳系畢業展總召林靜雯同學說，根據以往學長姊經驗，在松菸展出，包含場地、佈展總花費大概近100萬元，平均下來每人需要支出1萬多元，對學生而言，確實很吃力，這次在府中15不僅場地免租金，展場還同意可延用前一檔展覽的木製框架，讓他們省了將近一半的花費。

朱立倫說，市府團隊期望扶植學生青年藝術家能被看見，就如同這次的主題《青春分號；不句號》，畢業也只是分號，不是句號，同學們在畢業後才正要發光發熱。所以，透過專業藝文場館的提供、經費支援及行銷等最實際的協助，鼓勵青年學子持續創作，並開啟他們在創作上「自我追尋」的旅程。

朱立倫表示，這次展覽可看見許多作品透露年輕世代對當前社會、生命與價值觀與生存環境等關懷，展現畢業生多元且豐沛的創作能量，以及對未來的自我期許。

文化局表示，本次展覽將於新北市藝文中心、美麗永安生活館及府中15三地展出，參展學校包括北部的台灣藝術大學、元智大學，中南部的彰化大葉大學、嘉義南華大學及東台灣的台東大學，參展人數多達250人，近500件作品將接力展出。自即日起至6月21日止，更多活動訊息可至各館舍官網查詢。
