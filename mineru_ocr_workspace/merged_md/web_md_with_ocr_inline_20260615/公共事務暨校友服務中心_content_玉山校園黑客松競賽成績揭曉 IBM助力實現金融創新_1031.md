# 網站文章資料_玉山校園黑客松競賽成績揭曉 IBM助力實現金融創新_1031

## 一、文章基本資料

- **文章ID**: 1031
- **文章標題**: 玉山校園黑客松競賽成績揭曉 IBM助力實現金融創新
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【雲端/周維棻】

為增進產學合作、培育金融科技人才，玉山金控攜手台灣資料科學愛好者年會及台灣IBM，共同舉辦玉山校園黑客松，提升學生企業實戰經驗，為台灣金融科技發展播下創新種子。玉山校園黑客松強調「夢想無限，給自己最好的挑戰」，內容聚焦四大主軸，分別為Open Data、行動支付、數位生活和社群金融，同時玉山銀行舉辦三場Workshop，透過專業講師進行Design Thinking及Bluemix 雲端平台教育訓練，透過開放帳務服務、投資理財、公開資訊等金融API，結合由技術夥伴IBM提供的Bluemix開發平台，讓參賽者組裝應用，啟發多元創新理念及思維。

玉山校園黑客松競賽成績正式揭曉，台大電機研究所團隊勇奪金牌大獎。 


台大電機系碩士班「傑瑞小老鼠」團隊以「GinLocal-P2P旅遊平台」社群金融類應用，運用Watson API結合大數據分析和現今最熱門的區塊鏈技術，從決選20組團隊中脫穎而出，奪得金牌大獎，展現台灣學生在金融科技領域的實力。而銀牌獎項頒給了清華大學服務科學研究所「I服了U」團隊的「大資女孩」存錢應用。銅牌得主則為元智資工和資管大學生所組成的「玉山登頂」團隊，他們針對泛銀髮族群提出了「樂活聚寶盆App」。自2月初開放報名起，玉山校園黑客松獲得熱烈迴響，共90支隊伍參與比賽，前三名獲獎者可獲得至玉山銀行企業實習機會，表現優異者，更有機會優先錄用。

玉山金控資訊長謝萬禮表示：「金融科技(FinTech)崛起、BANK 3.0時代來臨，未來銀行不會只有大家認知的存、貸、匯款等服務，透過網路將有更多的無限可能。」為鼓勵台灣金融科技創新風潮，玉山透過舉辦校園黑客松競賽，讓年輕人展現創新與技術。而IBM的Bluemix雲端開發平台作為本次競賽的唯一技術平台，提供多項包括IBM Watson認知API等資源，協助開發者快速組裝創新應用，創造出嶄新的數位金融體驗。

台灣IBM雲端運算事業處副總經理許仲言表示：「雲端、行動、社群網路和大數據等新技術能促成顛覆式創新，對開發者而言，更重要的是以全新的思維將科技應用在金融領域，成就更好的使用者體驗。這次與玉山的合作，也是秉持IBM在台成立60年在地深耕精神，與客戶共好。」此外，這次人格特質分析和語意分析等API皆被競賽團隊廣泛使用，顯示認知技術可以替未來金融科技應用帶來更好的用戶體驗。
