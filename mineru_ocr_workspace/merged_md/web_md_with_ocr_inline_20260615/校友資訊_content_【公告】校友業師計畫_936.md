# 網站文章資料_【公告】校友業師計畫_936

## 一、文章基本資料

- **文章標題**: 【公告】校友業師計畫
- **文章網址**: https://www.yzu.edu.tw/alumni/index.php/tw/20230329
- **文章簡介**: 校友業師計畫



一、計劃緣起：
近年來，在少子化及全球化的雙重競爭趨勢下，高等教育競爭更加激烈，經濟不利學生所擁有之教育與生活主動關懷資源也更顯得分配不均。為協助經濟不利學生降低學用落差，建置完善學習輔導機制，期能透過本計畫，邀請各領域優秀校友擔任業師，分享其寶貴的生、職涯經驗，為經濟不利學子挹注學習與關懷力量，進一步產生更多啟發與幫助。

二、計畫內容：
本計畫預定實施方式為學年制，預計每學年徵求10位具備不同專業能力、經驗的優秀畢業校友擔任業師，每位業師輔導5名經濟不利學子，進行職涯經驗交流分享，並輔導學生進行生涯規劃，給予及時的指導和建議，以培養其職場核心能力，提升就業競爭力，達成學用合一之目標。

三、計畫期程：
1.    籌備期：每年4~6月，找尋適合之校友業師人選，8月進行媒合配對。
2.    期初相見歡：每年9月中旬，舉行校友業師與學生見面交流會。
3.    期中交流與演講：每年10月至隔年5月，業師和學生創建聯繫群組，透過線上或線下會面指導、諮詢，建立互助、分享和學習的社群；並邀請每位業師返校演講分享，開放全校師生參與。
4.    期末回饋：每年5~6月，業師依據自訂之輔導規範，給予評分或回饋。

四、輔導機制：
1.    自訂輔導方案：每位校友業師將會為其輔導的5名學子提供個別化輔導，例如職涯規劃、實習就業輔導、專業技能培訓等等。業師可依據自身專業背景和經驗，針對每位學生的不同需求和情況，自訂輔導方案和評估標準。
2.    定期面談：每位業師與學生每學年至少進行一次面談，針對學生的學習情況、職涯規劃等進行深入交流和輔導，並提供相關建議和解決方案。
3.    互動交流：業師和學生間建立互動平台，以確保學生在計畫期間得到即時的協助，提高學習與成長的效益。
4.    業師演講：計畫期間，邀請每位業師主持一場演講，業師可以分享自己的職業經驗、成功故事和職場技巧，邀請全校師生參與，以更深入地了解其職場和行業前景。
5.    輔導成效評估：每學期末，邀請業師和學生進行輔導成效滿意度調查，了解學生在輔導過程中的學習成效和滿意度，以及業師的表現和建議，以便及時調整改進計畫內容和執行方式，進一步優化輔導機制。

五、計畫預期成效：
1.    提供學生具體、專業、個別化的學習和職業輔導，降低學用落差和就業門檻。
2.    有效提高學生的學習成效和就業競爭力，提高職場競爭力和社會適應能力，也有助於激發學生對學習和職業發展的興趣和熱情。
3.    增進學校和校友之間的情感聯繫和互動，不僅能維繫良好的校友網絡和合作關係、擴大校友參與母校發展的機會與管道，也能為學生開創更多的就業和實習機會。
