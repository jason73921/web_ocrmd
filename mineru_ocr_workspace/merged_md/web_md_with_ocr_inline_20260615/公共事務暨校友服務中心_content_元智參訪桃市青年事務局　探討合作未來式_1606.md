# 網站文章資料_元智參訪桃市青年事務局　探討合作未來式_1606

## 一、文章基本資料

- **文章標題**: 元智參訪桃市青年事務局　探討合作未來式
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/

- **文章全文**: 【台灣好新聞/記者黃駿騏/桃園報導】 


陳家濬表示，青年事務局從創局以來一直與元智大學有合作，希望彼此的計畫合作能夠更有深度。

元智大學校長吳志揚帶領副校長、各系所院長等18人於1月16日下午拜訪桃園市政府青年事務局，參觀了桃園首座青年創業基地，並針對桃園青年學子的學程未來發展做了深度交流與討論。

吳志揚校長表示：桃園目前正積極配合中央發展亞洲∙矽谷計畫，規劃推動的IOT物聯網創業聯盟，也是目前元智大學著墨的重點，未來在推動過程中如有任何需要協助，元智大學將全力配合。另外，針對「桃園x MIT亞洲．矽谷暨智慧城市專案計畫」雙方也腦力激盪、熱烈討論，冀望能夠有合作機會。 


陳家濬表示，青年事務局從創局以來一直與元智大學有合作，希望彼此的計畫合作能夠更有深度。

而對於青年事務局推動的「TYC x Campus創新產業人才培育計畫」整合了校園創新及育成中心，推動本市重點及新興產業發展，透過主題學習接軌多元型式課程、媒合交流及實作輔導，並引入國際資源，拓展國際視野，解決產業人力及產業轉型斷層的問題，雙方都認為可以更緊密結合，達成產官學三方多元合作，活絡本市產業發展。

此次拜訪，吳志揚校長等貴賓也參觀了桃園首座的青年創業基地，包括內部的各項硬體規劃、課程運作、新創團隊營運等，也與進駐的基地內的新創團隊意見交流，了解年輕創業家的需求與階段性創業目標。青創指揮部營運一年來的營運環境與模式以及輔導績效獲得元智大學團隊大力讚賞。 


陳家濬表示，青年事務局從創局以來一直與元智大學有合作，希望彼此的計畫合作能夠更有深度。

青年事務局局長陳家濬表示，青年事務局從創局以來一直與元智大學有合作，希望106年度彼此的計畫合作能夠更有深度，配合中央政策、緊扣市政方向。也鼓勵青年學子們不要怕跟政府打交道，要善用政府資源，開拓自己未來的美好藍圖。
