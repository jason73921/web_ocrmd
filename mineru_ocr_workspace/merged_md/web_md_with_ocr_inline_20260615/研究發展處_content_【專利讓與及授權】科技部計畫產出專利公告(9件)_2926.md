行動式遠端監控系統及其方法



I318064



一種行動式遠端監控系統及其方法，一終端監控電腦透過一網際網路連接於一無線區域網路系統，再藉該無線區域網路系統傳達信號指令給一可移動之監控平台，而該可移動之監控平台利用一網路控制系統接收該終端監控電腦所傳達之指令，並執行位置之移動或影音訊號之傳輸；藉此便不必被限制於習知無線傳輸範圍之限制及干擾，透過網際網路便可於監控遠端的裝置及執行信號雙向之傳輸，並藉由該可移動之監控平台的機動性，可較習知固定式監控裝置的監控品質更為完善。

抑制柯肯達爾孔洞形成於銲料與銅銲墊之間的方法



I464031



一種抑制柯肯達爾孔洞形成於銲料與銅銲墊之間的方法，其藉由將微量鈀添加至銲料中，並將此銲料與銅銲墊接合，以達抑制柯肯達爾孔洞形成的效果。其中，銲料添加鈀後再與銅銲墊進行銲接，以形成銅銲墊/Cu3Sn/(Cu,Pd)6Sn5/銲料之銲點結構。在銅銲墊/Cu3Sn之間常見的柯肯達爾孔洞則可因鈀的添加而達抑制效果。藉此發明，毋須大幅地改變銲料原有之特性或銲接製程條件即可明顯提升銲點結構之機械可靠度。



2014/12/11-2031/12/13

ACOUSTIC ECHO CANCELLATION METHOD AND SYSTEM USING THE SAME



US9763005B2



The present disclosure illustrates an acoustic echo cancellation method and system using the same. The acoustic echo cancellation method comprises the following steps. Firstly, a prior-knowledge matrix comprising a plurality of space vectors is built. Then, an initial filter vector is generated by the prior-knowledge matrix and an initial weighting vector. The weighting vector is updated based on the difference of the echo signal and the estimated signal in an iteration algorithm, and the coefficient of the filter vector is updated according to the updated weighting vector. An estimated signal is generated according to the updated filter vector and the original signal. Finally, the next echo signal is cancelled by the near-end estimated signal.



2014/10/29-2035/06/12

原子層氧化及還原的材料修飾製程



I564425



本發明係一種原子層氧化及還原的材料修飾製程，包括多次的處理循環，而每一處理循環包括：打開反應性氣體通氣閥，並在第一通氣處理時間內通入反應性氣體，可包含氧化性氣體或還原性氣體，用以進行氧化反應或還原反應；關閉反應性氣體通氣閥；打開惰性氣體通氣閥，並在第二通氣處理時間內通入惰性氣體；關閉惰性氣體通氣閥；打開抽氣閥進行抽氣，以移除反應性氣體及惰性氣體；以及關閉抽氣閥。因此，本發明可增加或降低目標材料的含氧官能基的數目，進而增加或降低目標材料中氧元素對碳元素的比值，藉以精確調控目標材料的電氣特性，包含能隙大小。

I565308

一種微生物生產葡萄糖胺的方法



1269473



本發明公開了一種微生物生產葡萄糖胺的方法，是以低成本且新穎的培養基發酵培養選自Monascus pilosus及Aspergillus sp.所構成的群組之中的微生物，以使其生產葡萄糖胺；其中該培養基包含臺糖的食用糖、黃豆及米糠等；其發酵的適當環境為：150～300rpm、pH 4～pH 8、24℃～37℃；經發酵培養後，並將其發酵液抽氣過濾取得該微生物菌體，並將該微生物菌體經過破細胞、鹽酸化反應、中和反應及過濾等步驟，以取得該微生物所生產的葡萄糖胺。本發明生產葡萄糖胺的方法操作簡單、步驟少，成本低廉，產量高。

I464261

非接觸式來電拒接方法及使用該方法的通訊系統



I593271



本發明係一種非接觸式來電拒接方法及使用該方法的通訊系統，該方法包括：進入正常運作模式，並在接收到來電訊息時，啟動提示裝置；偵測是否接收到拒接訊號；如果接收到拒接訊號，進入來電拒接模式，並且關閉提示裝置；偵測拒接訊號是否持續存在；如果拒接訊號未持續存在，關閉來電拒接模式，恢復提示裝置的原先設定；以及回到正常運作模式，並重複其後續步驟。因此，本發明可依據來自近場無線射頻通訊單元拒接訊號，自動進入來電拒接模式，而不需使用者進行任何實體操作，尤其適用於行車時的通訊裝置，避免干擾駕駛的注意力而影響行車安全。



2017/07/21-2033/10/27
