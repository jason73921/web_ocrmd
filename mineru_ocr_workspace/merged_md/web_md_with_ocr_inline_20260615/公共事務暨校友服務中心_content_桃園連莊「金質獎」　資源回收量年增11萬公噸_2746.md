# 網站文章資料_桃園連莊「金質獎」　資源回收量年增11萬公噸_2746

## 一、文章基本資料

- **文章ID**: 2746
- **文章標題**: 桃園連莊「金質獎」　資源回收量年增11萬公噸
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20180417-2
- **文章簡介**: 【ETtoday新聞雲/記者楊淑媛/桃園報導】

桃園市政府環保局16日指出，106年全國資源回收考核，該局已連續兩年榮獲直轄市最高殊榮「金質獎」，也連續第13年得到資源回收考核的最高評等；且106年的資源回收率，和升格前相比，成長了超過11個百分點；年度回收量也比105年增加約11萬公噸。不論就比率或總量來看，都可見桃園在資源回收上的出色績效。

除了整體資源回收率及資源回收量之外，環保署的考核也相當重視區里及學校的具體作為，考核委員也深入社區及大專院校，實地瞭解民眾及學生如何落實資源回收。此次委員赴八德區瑞德里考核，由康秀雲里長親自解說里內的環保工作，例如由志工回收瓶罐與打包帶，做成「小小兵」裝置藝品；響應「桃樂資收站」活動，定期於里內舉辦「回收換好禮」；此外，瑞德里的志工也以廢輪胎和廢鐵材等，做成公共裝置藝品，使里內增添人文氣息。
在學校的考核上，委員則認為元智大學的師生普遍對於資源回收有正確認知，垃圾桶及回收桶很少混雜；校方也指派專人對資源物先進行細項分類，再交付回收商，因此能變賣到更好的價格，來繼續推動各項環保工作。(待續)
