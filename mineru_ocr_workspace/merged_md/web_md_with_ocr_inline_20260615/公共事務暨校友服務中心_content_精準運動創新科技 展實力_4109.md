# 網站文章資料_精準運動創新科技 展實力_4109

## 一、文章基本資料

- **文章ID**: 4109
- **文章標題**: 精準運動創新科技 展實力
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20191205
- **文章簡介**: 【工商時報/文/黃志偉】

由科技部主辦的未來科技展今（5）日在世貿一館精彩登場，展出百項創新技術，其中受到運動風氣與產業的蓬勃發展，展覽中所展出的精準運動科技項目相當受到矚目，例如由國立臺灣體育運動大學團隊與工研院合作開發的「智慧壓力運動鞋墊」，以及元智大學「棒球指壓感測與無線傳輸裝置及時序大數據分析系統」，都是創新技術中的亮點。

棒球是全球熱門的職業運動之一，投球的表現更牽動球隊戰力表現，而投手球路變化與控球的好壞，跟投手的手指握法與手指施力有很大的關係。元智大學所開發的棒球指壓感測技術裝置，可擷取投球時的手指施力、釋放的動態時序特徵與慣性運動特徵，藉由無線傳輸模組與終端進行溝通，傳送投球過程的指壓資訊及運動向量特徵，並可接收出手點震動提示指令，配合後端分析及客製化訓練配方做精準調整，對於投手狀態評估及提升投球能力有極大的助益。

元智大學技術團隊施皇嘉教授指出，該項技術，以水平整合角度來看，能夠連結各種運動訓練的產業應用鏈；在垂直整合方面，由硬體研發的立體壓力感測及物聯網產業，到軟體資訊分析的運動資訊分析產業，應用價值可延伸或擴大到各種運動產業上。(待續）
