# 網站文章資料_企管菁英協會舉辦會員大會暨理監事改選_174

## 一、文章基本資料

- **文章ID**: 174
- **文章標題**: 企管菁英協會舉辦會員大會暨理監事改選
- **文章網址**: https://www.yzu.edu.tw/alumni/index.php/tw/
- **文章簡介**: 
- **文章全文**: 元智20週年校慶當日(3/14)，中華民國企管菁英協會在參加校友餐會後，隨即召開第五屆第一次會員大會及理監事會議。除了順利通過各議案，這次會議也進行了理事長、常務監事及理監事改選。深受會員們尊崇愛戴的許評錀理事長因已連任二次，依章程無法續任。理事長一職交棒給元智管研所91級EMBA楊中一先生，楊中一是建築、管理雙碩士，目前擔任合契建築師事務所主持人。至於原秘書長李純櫻也功成身退，交由元智管研所92級EMBA李榮維接任。企管菁英協會自成立就立下EMBA交替傳承的不成文規矩，也在本次的改選時延續下去。 



理監事成員皆學經歷豐厚，並在各領域擔任負責人或中高階主管。協會創會理事長統寶光電總經理張永青先生更在本次校慶獲頒傑出校友。企管菁英協會發展至今邁入第9年，未來在新任理事長楊中一的領導下及理監事幹部們的合作下，會務更為興隆並多元發展。 



本次當選理監事名單如下： 

理事長：楊中一，常務理事：張永青、曾文敬、許評錀、李純櫻 

理事：陳隆全、蔡國圳、蕭麗綾、王觀白、葉惠菁、曾銘銅、翁育彬、劉志奇、張振聰、黃萬財 

常務監事：蔣廷芳，監事：林振造、吳錦乾、馬玉蘭及林琨忠等。
