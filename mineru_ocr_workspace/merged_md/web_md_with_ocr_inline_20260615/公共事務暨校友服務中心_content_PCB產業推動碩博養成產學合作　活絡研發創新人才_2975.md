# 網站文章資料_PCB產業推動碩博養成產學合作　活絡研發創新人才_2975

## 一、文章基本資料

- **文章ID**: 2975
- **文章標題**: PCB產業推動碩博養成產學合作　活絡研發創新人才
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20180626-2
- **文章簡介**: 【蘋果日報/蕭文康／台北報導】

台灣電路板協會（TPCA）自2016年起每年自主提撥總經費近200萬，積極推動《電路板產學聯合培育碩博士計劃》，廣邀學界碩博士生發表研究計劃，主題內容經產業共同認可後，每年補助碩士養成每案18萬，博士生養成更達每案46萬，產學研發養成計劃為期3年，並保留延長1年彈性，確保研究的效益。期望透過產學深度結合，鼓勵碩、博士生鑽研PCB產業相關專題研發計劃，其研究成果可具體轉換為產業應用所需，為PCB產業注入新的研發能量。

2016年首波養成計劃經評選後，由清華大學化工系劉英麟教授(研究主題為低介電材料與作為印刷電路板基板之應用研究)、中央大學化材系吳子嘉教授(高可靠度元件抗爬行腐蝕研究及PCB導線間擴散阻障層之開發)參與至今與產業界交流密切，研發成效斐然。2018，TPCA再度啟動第2波培育計劃，經過初審後，計有包括來自中央大學化材系—劉正毓教授、中興大學化工系—竇維平終身特聘教授、台灣大學材料系—高振宏特聘教授、元智大學化材系—何政恩教授、義守大學電子系—黃有榕教授等共5人7案參與申請。(待續)
