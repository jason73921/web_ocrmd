# 網站文章資料_疫情影響教研 元智大學加速建置智慧校園_4465

## 一、文章基本資料

- **文章ID**: 4465
- **文章標題**: 疫情影響教研 元智大學加速建置智慧校園
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20200723-2
- **文章簡介**: 【中央社記者陳至中台北23日電】

武漢肺炎疫情影響教學、研究，元智大學今天宣布，將加速建置智慧校園，以大數據、物聯網與雲端連結，更方便瞭解學生學習和學校行政管理的狀態。

元智大學舉辦招生記者會，校長吳志揚表示，COVID-19（2019年冠狀病毒疾病，俗稱武漢肺炎）疫情衝擊大學，畢業生要就業尤其受到影響。但好在元智大學4年就「超前部署」，推動全校必修程式語言，引領台灣學界潮流，近年相關人才仍是炙手可熱。

元智大學人文社會院院長劉宜君舉例，應用外語系學生結合AR（擴增實境）、VR（虛擬實境）技術，強化學生在數位工具的應用，製作日語機器人、英語機器人等；藝術設計系將3D運用到藝術創作和設計領域，並強化藝術數位行銷能力。

元智大學近來發展智慧校園，在校內布建智慧感測網路，蒐集大量校園數據，做為校園管理優化的參考依據，包括校園停車位的管理、宿舍智慧電表、偵測各教室的用電消耗和二氧化碳、溫度、濕度等。

元智大學的智慧教室，結合Moodle課程管理系統，支援自動點名、即時小考等，讓教學更有效率，未來希望能讓學生利用手上的智慧裝置，就能達到互動學習的應用。（待續）
