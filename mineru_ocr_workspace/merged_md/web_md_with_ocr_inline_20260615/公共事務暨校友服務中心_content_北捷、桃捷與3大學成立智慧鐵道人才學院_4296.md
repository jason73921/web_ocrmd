# 網站文章資料_北捷、桃捷與3大學成立智慧鐵道人才學院_4296

## 一、文章基本資料

- **文章ID**: 4296
- **文章標題**: 北捷、桃捷與3大學成立智慧鐵道人才學院
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20200422-2
- **文章簡介**: 【公視/林曉慧 沈志明 台北報導】

國立臺北科技大學、元智大學、開南大學與北捷公司、桃捷公司合作成立「智慧鐵道產業人才學院」，這三所大學的學生只要有興趣，都能修習相關課程。

智慧鐵道學院由台北科技大學、元智大學、開南大學為核心學校，並和北捷公司與桃捷公司合作，到時將配合產業需求，規劃課程與實習計畫。智慧鐵道學院籌備委員會召集人王錫福表示：「後續我們也會來推動，政府所認可、主管機關認可的，我們鐵道學程的這個認證制度。」

109學年今年八月開始，這三所大學的三年級與四年級學生，只要對學院的課程有興趣都能選課，將以一校面授，另兩校同步遠距為主，而拿到學分後再去考證照，成為可能的鐵道人才。（待續）
