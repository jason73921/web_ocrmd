- **文章簡介**: Windows Operating System


After User have completed application in accordance with the process, the system program of dormitory network will check the status of WSUS updates, and automatically opening after one hour.



Macintosh Operating System


First, Users must fill the Ethernet ID in the application system of dormitory network (https://net.yzu.edu.tw/). Second, Users must login a problem in customer service systems (problem type: dormitory network-> problem Subject: dormitory network application-> problem situation, select "My computer is Macintosh operating system, require to open internet connection. ").

    Each application will take 3 to 5 working days.
