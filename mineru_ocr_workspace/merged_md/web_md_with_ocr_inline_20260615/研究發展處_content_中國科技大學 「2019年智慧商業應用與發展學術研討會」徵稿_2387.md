# 2019 年智慧商業應用與發展學術研討會

# 論文徵稿

近年來，不論是德國「工業4.0」、美國的再工業化政策、日本的人機共存未來工廠、韓國的下世代智慧型工廠，還是中國製造2025計畫，全球主要國家均積極推動建構網實智能化製造、生產、銷售系統，以快速反應或預測市場需求;產業供應鏈垂直與水平數位化、智能化，成為全球搶單競爭關鍵。

我國產業正面臨開發中國家搶占量產市場，以及工業國家囊括中高階客製化市場前後夾擊的嚴酷挑戰。有鑑於此，行政院於2015年9月核定「行政院生產力4.0發展方案」，內容涵蓋「製造業4.0」、「商業4.0」等，規劃六大主軸策略，希望藉此促進國內產業創新轉型、掌握關鍵技術自主能力、促進經濟發展及維持國際競爭力。

面對此一歷史契機，中國科技大學-大學社會責任實踐計畫團隊特針對此一重要議題舉辦學術研討會，除邀請產、官、學界專家專題演講外，並進行學術及實務研討。期待透過產、官、學界專家齊聚一堂，相互交流研究成果或實務經驗，進而成為整合教學、研究及諮詢的平台，以提供政府制定政策及各界人士討論之重要參考資訊與依據。
