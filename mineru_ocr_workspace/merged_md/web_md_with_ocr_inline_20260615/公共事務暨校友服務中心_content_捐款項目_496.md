# 網站文章資料_捐款項目_496

## 一、文章基本資料

- **文章ID**: 496
- **文章標題**: 捐款項目
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/2017-05-25-08-34-28
- **文章簡介**: 教育如同演一齣戲，若將生命當成一場禮讚，無私的給與，將是何等的值得與喜悅。

感謝您長期投注教育資源，當學子努力翱翔，勇敢追夢，您的支持與鼓勵是他們前行的最大力量。

 









科技、創新、永續經營

校務發展基金為元智大學發展成為創新跨域及永續校園的基礎，主要用於教學研究、國際交流與合作、設置助學金、校園維護與建設等校務規劃，著重於：


跨域全才培育－培養學生成為社會與產業優秀人才。

重點領域研究－致力於特色領域研究，引領產業提升與社會進步。

特色文化營造－營造人文、創新、永續、健康的特色校園文化。

國際聲望提升－建構具國際聲望與全球移動力的優質大學。

永續校園經營－建立支援學校永續發展之人力與財務資源、智慧校園設施及校務營運制度。


鼓勵您將捐款挹注校務發展基金，讓學校能妥善及彈性運用，使元智朝著國際一流大學的方向前進。





 





培育、多元、跨域

元智大學擁有五院一所，設有工程學院、電通學院、資訊學院及管理學院，另有一醫學研究所，整合各學院(所)資源，開設多元且跨領域的課程，培育符合新時代的優秀人才。

元智目前正積極籌設醫護學院，朝向全方位的綜合大學發展，期盼您繼續給予支持與鼓勵，使元智更能精進辦學。





  



 

發展、專業、資源

鼓勵學生積極認真向學，完善發展品格與學業，以成績為評核基準，優秀學生可申請獎助學金，減輕學雜費及生活負擔，持續為未來努力。許多獎助學金是教師與校友等感念元智栽培而提攜後輩，或為善盡企業社會責任以培育專業人才為社會所用，因此提供各類獎助學金扶助有需求的學子，讓其未來同樣有回饋社會的機會。





 





支助、傳承、回饋

近年因大環境因素，許多家庭突遭變故或是發生急難事件，使有心向學，卻無力負擔學雜費及生活費的學生日愈趨多，為讓有失學之虞的學生能安心就學，元智大學提供各類獎助學金，以實質的經濟扶助，協助有需要的學生，助其順利完成學業。





 





熱情、自主、服務

十三項運動校隊及八大性質學生社團，強化學生自主學習能力、發展就業軟實力及職場競爭力。

元智推動各項自主學習課程、學術交流與社會服務等活動，鼓勵學生積極參與社團及各項體育賽事，拓展多元視野及公共參與力，期盼您以行動支持學生社團及運動校隊！





 



  



 
 




 


 

 

 

各單位如有捐款等相關事項，請與公共事務室聯繫，公共事務室確認後會轉請財管組開立收據，後續徵信公告並進行珍謝動作。若還有其他捐贈或是小額捐款，亦歡迎聯繫公共事務室。
聯絡人：元智大學秘書室公共事務暨校友服務中心-游小姐

聯絡電話：(03)463-8800 分機 2317
