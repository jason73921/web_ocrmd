# 網站文章資料_4G給力 7成低頭族很滿意_202

## 一、文章基本資料

- **文章ID**: 202
- **文章標題**: 4G給力 7成低頭族很滿意
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【工商時報/記者林淑惠/台北報導】 


左起台灣數位匯流發展協會理事長吳世昌、北科大助理教授江雅綺、台大教授林宗男、元智大學教授周韻采。圖／台灣數位匯流發展協會提供

台灣數位匯流發展協會昨（11）日公布「2015上半年數位匯流大調查」，4G使用比例大增，民眾對行動上網速度滿意度大提升，有68.1％表示肯定，僅30.7％不滿意，滿意比例較去年底提升12.6個百分點。

這是2012年來連續第8次調查，結果顯示，使用智慧型設備行動上網民眾有77.9％，較去年增加9.9個百分點。4G上網上升到54.0％，大增27.9個百分點。對行動上網速度滿意度，有68.1％肯定，30.7％不滿，滿意比率比去年底上升12.6個百分點。

另外，在固網寬頻使用速度上，79.7％認為足夠，16.9％表示仍有不足，認為足夠的比率比去年底上升2.1個百分點。

受到有線電視數位化快速推動影響，收視戶對有線電視畫面品質滿意度大增，由去年底69.3％上升到76.6％，增加7.3個百分點，但仍有65％收視戶對重播率不滿，52.9％認為費用太貴。對於NCC推動分組付費機制，61.3％支持，30.7％反對；對開放新業者經營有線電視，69.5％考慮更換，21.4％沒意願改變。

對於中華電信MOD收視滿意度，67.9％收視戶滿意頻道多元性，90.6％肯定畫面品質，對MOD整體服務有81.4％滿意。

民眾對台灣媒體收視環境的不滿意度持續上升，51.2％不滿、較去年同期上升5.8個百分點。在媒體新聞台立場上，84.7％覺得有政黨色彩，僅9.1％認為沒有。35.1％覺得偏藍，31.8％認為偏綠；與101年調查相比，認為偏綠比率大幅增加21.6個百分點；調查並發現，54.6％民眾支持政府或政黨可以擁有頻道宣揚政治理念，37.5％反對。

至於是否要廢止「黨政軍條款」對政府投資媒體限制，則有49.0％表示支持，34.0％ 仍然認為不宜開放。在媒體壟斷議題上，86.4％認為台灣媒體有逐漸被財團壟斷情形，77.7％覺得情況嚴重；69.5％擔憂未來媒體被特定集團壟斷。

台大教授林宗男分析，有線電視訂戶逐年下降，年輕世代由網路獲得多媒體和新聞資訊，過去壟斷市場以獲取暴利卻忽視消費者需要的有線電視將逐漸喪失競爭力，他認為分組付費應盡快上路，讓消費者有選擇權。

北科大助理教授江雅綺認為，應將黨政軍條款比照預算法放寬政府資金到5成，以促進數位匯流和文創內容產業發展。

數位匯流大調查是台灣數位匯流發展協會委託趨勢民意調查公司以電話抽樣方式進行，共計1,501份有效樣本，調查時間是7月30日～8月7日。
