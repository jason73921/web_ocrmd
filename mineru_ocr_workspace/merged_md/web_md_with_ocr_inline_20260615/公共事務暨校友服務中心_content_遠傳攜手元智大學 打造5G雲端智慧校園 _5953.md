# 網站文章資料_遠傳攜手元智大學 打造5G雲端智慧校園 _5953

## 一、文章基本資料

- **文章ID**: 5953
- **文章標題**: 遠傳攜手元智大學 打造5G雲端智慧校園 
- **文章網址**: 
- **文章簡介**: 【中央社記者江明晏台北18日電】

遠傳今天表示，攜手元智大學啟動5G雲端智慧校園戰略結盟，雙方將針對打造微軟國際課程與認證中心、5G智慧校園、全雲端校園、元宇宙產學合作等4大項目深度合作。

遠傳總經理井琪表示，身為元智大學長期的產學合作夥伴，遠傳在雙方既有的默契上，再度深化合作，進行全面的戰略結盟。將提供5G「大人物」（大數據、人工智慧、物聯網）技術力與跨部門團隊資源，在師資教學、接軌職場、產業應用、校園安全等面向帶來實質助力，推動校園轉型升級。也期待培育更多學生共同投入建構元宇宙平台生態系，帶動產業共好。

雙方具體的4大合作項目包括，打造桃園第一座「微軟國際課程與認證中心」，以及打造5G智慧校園、全雲端校園、元宇宙產學合作。（待續）
