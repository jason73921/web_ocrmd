# 網站文章資料_2019藝術新聲畢業生推薦展 展現多元豐沛創作能量_3708

## 一、文章基本資料

- **文章ID**: 3708
- **文章標題**: 2019藝術新聲畢業生推薦展 展現多元豐沛創作能量
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/2019050601
- **文章簡介**: 【國立教育廣播電台】

臺中市大墩文化中心即日起至22日舉辦「2019藝術新聲11校12系畢業生推薦展」，匯集全國由南至北各大校系166位卓越好手，集結傳統、現代、後現代和當代特色，從平面至立體，包括繪畫、版畫、雕塑、陶藝、工藝、複合媒材到新媒體藝術等多元藝術面向，展現豐沛旺盛的創作能量，歡迎踴躍參觀。

文化局指出，今年度的參與校系包括高雄師範大學美術學系、臺南藝術大學材質創作與設計系、嘉義大學視覺藝術學系、彰化師範大學美術學系、臺中教育大學美術學系、東海大學美術學系、清華大學藝術與設計學系、元智大學藝術與設計學系、臺灣師範大學美術學系、臺灣藝術大學美術學系及雕塑學系及臺北藝術大學美術學系，共十一所學校，期待這些蓄勢待發的藝術新秀激盪出藝術的嶄新花火，帶來不一樣的藝壇新機。(待續)
