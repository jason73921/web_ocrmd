元智大學電機系教授郭文興研發「YZU IoT Gateway 」物聯網技術，圖為學生江銘洋展示該技術。（校方提供）

元智大學電機系教授郭文興領網際網路實驗室學生，研發「YZU IoT Gateway 」物聯網技術，將能整合不同品牌的產品、感測器、雲端資料來源，應用於智慧家庭上，更能廣泛的整合各式家電。

郭文興表示，市面上現有品牌的數位家庭方案，僅能連接專屬品牌的裝置，或是使用特定通訊標準。使用者與管理者難以任意連接各種類型設備，無法自定顯示畫面，也不能任意擴充閘道器。

郭文興團隊所研發的「YZU IoT Gateway 」，可統合不同產品、感測器以及雲端資料來源，進行整合管理，並可連接用戶、設備與服務業者，是具備高度客製化、模組化、彈性化的物聯網管理框架。

郭文興團隊說，設備商不再需要設計專屬App，服務業者就可提供整合性的服務，一般使用者可享受高品質、低成本的物聯網整合應用。
網際網路實驗室電機所碩二生江銘洋表示，安裝此智慧家庭系統的機上盒，可透過藍芽或Wi-Fi，連結家中各種電器，並接收網路上的資訊。比如說，使用者可從整合畫面中，得知天氣、家中的溫溼度等訊息，並可控制電燈、電扇，更可以設定各種自動化功能。

另外，南亞技術學院8日結合各地社團，邀請到阿基師教導民眾、學生、社團成員製作粽子，並將製作完成的粽子，分送給社區弱勢民眾。
