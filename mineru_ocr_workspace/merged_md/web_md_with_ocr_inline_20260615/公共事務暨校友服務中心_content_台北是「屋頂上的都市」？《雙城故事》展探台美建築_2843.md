# 網站文章資料_台北是「屋頂上的都市」？《雙城故事》展探台美建築_2843

## 一、文章基本資料

- **文章ID**: 2843
- **文章標題**: 台北是「屋頂上的都市」？《雙城故事》展探台美建築
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20180507-3
- **文章簡介**: 【 中時電子報/林宜靜／整理報導】

地狹人稠的台北發展出垂直延伸的「屋頂上的都市」，那麼洛杉磯呢？在人口稠密的都市，居住空間不足一直是項重要的公共課題。洛杉磯和台北的居民，則各自發展出不同的解決之道。由洛杉磯辦事處台灣書院舉辦的「《雙城故事》台美建築藝術展」，將引領大家一起發掘都市生活的無限創意，並探討台北與洛杉磯兩地在建築藝術、都市發展及生活方式間的關連與意義。展覽將於5月10日開幕，展至7月7日為止。

「《雙城故事》台美建築藝術展」由台灣旅美設計師、Bureau Spectacular工作室主持人賴彥吉(Jimenez Lai)策劃，將展出台北及洛杉磯的都市模型、短片、設計稿及圖稿等。賴彥吉表示，洛杉磯居民製造出水平延伸的「後院中的都市」，而地狹人稠的台北則發展出垂直延伸的「屋頂上的都市」，這個有趣的對照，正可呈現出兩地居民都市生活與文化的不同風貌。

展覽開幕將邀請加州大學洛杉磯分校(UCLA)建築與都市設計系教授Hitoshi Abe、洛杉磯建築專業組織LA Más執行主任Elizabeth Timme與策展人賴彥吉對談，另也特別邀請台灣知名建築師、元智大學藝術與設計學系教授阮慶岳以視訊加入討論，透過不同觀點刺激對都市空間的多元想像。(待續)
