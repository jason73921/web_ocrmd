# 網站文章資料_環球學校財團法人環球科技大學「2018整體造型研究與設計學術研討會」徵稿_1602

## 一、文章基本資料

- **文章ID**: 1602
- **文章標題**: 環球學校財團法人環球科技大學「2018整體造型研究與設計學術研討會」徵稿
- **文章網址**: https://www.yzu.edu.tw/admin/rd/index.php/tw/?option=com_content&view=article&id=1602
- **文章簡介**: 主旨：檢送環球科技大學時尚造型設計系辦理「2018整體造型研究與設計學術研討會」與研討會徵稿，敬請惠予公告，並鼓勵所屬師生參加，請查照。

說明：

一、活動主旨：為增進各大專院校美容造型、時尚設計相關科系教師在藝術創作和藝術文化研究方面的學術發展，特邀請國內藝術創作研究學者和藝術文化研究專家以及產業界經營者與學有專京從業人員蒞臨進行專題演講與經驗分享，並藉由學術論文的發表以促進美容造型、時尚設計學術和實務交流。

二、主辦單位：環球科技大學時尚造型設計系

三、舉辦地點：環球科技大學嘉東校區 創意樓教學演講廳DC-B02

四、舉辦時間：民國107年4月30日(星期一)9：00-17：00

五、舉辦方式：

(一)專題講座：邀請國內藝術創作研究學者、藝術文化研究專家和設計美學專家以及產業界經營者與學有專精之專業從業人員提供經驗分享與專題講座。

(二)發行論文集：由本研討會公開徵稿及審查，經審查合格論文刊載於研討會論文集。

六、聯絡方式：環球科技大學時尚造型設計系吳樹屏老師，E-MAIL：denny77@gmail.com、電話：05-5370988分機3101或7342。
