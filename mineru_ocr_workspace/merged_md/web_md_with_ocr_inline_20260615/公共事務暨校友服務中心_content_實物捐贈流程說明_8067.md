# 網站文章資料_實物捐贈流程說明_8067

## 一、文章基本資料

- **文章ID**: 8067
- **文章標題**: 實物捐贈流程說明
- **文章網址**: 
- **文章簡介**: 受贈單位收到公司(機關)或個人意思表示，公司(機關)來函，並備妥「接受實物捐贈表」及「實物捐贈合約書/ 同意書(二擇一)」、捐贈明細表、價值證明等文件。


(價值證明文件：該實物廠牌估價單、購入成本發票、公司財產清冊、第三方專業鑑價廠商。)
(備註：未上市有價證券實物捐贈需另檢附資產負債表、損益表)


是否同意接受實物，由受贈單位上簽呈會簽相關單位：總務處→會計室→秘書室(公事校友中心)→校長核示

經校方同意後，由受贈單位通知捐贈者進行捐贈，會簽實物捐贈表。
(備註：未上市有價證券實物捐贈需另檢附股份有限公司變更登記表、股東名冊、有價證券處分同意書)

受贈單位收到捐贈物後，需拍攝實物照片並負保管與管理之責。

總務處財管組驗收、列財產、開立收據。

公事校友中心寄送收據、感謝函，並完成歸檔。
