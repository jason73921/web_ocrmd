# 網站文章資料_行政院農業委員會水土保持局「一百零九年大專院校農村實踐共創計畫」_2895

## 一、文章基本資料

- **文章ID**: 2895
- **文章標題**: 行政院農業委員會水土保持局「一百零九年大專院校農村實踐共創計畫」
- **文章網址**: 
- **文章簡介**: 主旨：「一百零九年大專院校農村實踐共創計畫」徵件開始，歡 迎上網報名，請查照轉知。

說明：

一、本局為鼓勵各大專院校教師運用其專業課程，結合農村為 應用實踐場域，引動大專青年發揮多領域專業技能，協助 農村、農業解決問題，創新加值發展，推動「大專院校農 村實踐共創計畫」，明(109)年度計畫已開始徵件，申請期 限至108年12月31日止。

二、補助對象為全國公私立大學院校，以課程、專題討論或社 團輔導為單位，教師為計畫主持人提出申請，搭配可實踐 專業核心知識之農村場域，進行專題設計、實作及駐村等 課程執行。

三、本案採線上報名，詳情請至計畫官網（https://rcampus.swcb.gov.tw/node/102）查閱。
