# 網站文章資料_國立清華大學「民生公共物聯網資料應用競賽」_3135

## 一、文章基本資料

- **文章ID**: 3135
- **文章標題**: 國立清華大學「民生公共物聯網資料應用競賽」
- **文章網址**: https://www.yzu.edu.tw/admin/rd/index.php/tw/?option=com_content&view=article&id=3135
- **文章簡介**: 主旨：「民生公共物聯網資料應用競賽」將於109年4月27日早上 11點開始正式報名，請協助轉知相關系所並踴躍參與，請 查照。

說明：

一、民生公共物聯網資料應用競賽」目的為刺激業界、學界與 民間共同參與政府開放資料應用，以帶動國內資料應用與 服務產業之創新與發展，自109年4月27日早上11點至109年 5月25日早上11點開放線上報名，提供首獎300萬、二獎100 萬、三獎50萬及佳作10萬元獎金，總獎金高達550萬，敬邀 貴校師生一同參與，透過這樣的活動激發創意，促進臺灣 資料經濟生態系發展。

二、本競賽係由行政院科技會報指導、科技部與建構民生公共 物聯網計畫推動小組主辦、財團法人國家實驗研究院國家 高速網路與計算中心、國家災害防救科技中心以及財團法 人國家地震工程研究中心執行，結合眾多部會，包含行政 院環境保護署、經濟部、科技部、教育部、中央研究院、交通部、內政部、行政院農業委員會等單位。建構民生公 共物聯網計畫致力於整合空氣品質、地震、防救災、水資 源等民生四大領域環境監測資料，提供穩定、高品質的感 測資料與運算資源，並開放給各界進行加值應用。

三、競賽活動各項時程與相關活動資訊請參閱競賽活動官網 https://ci.taiwan.gov.tw/creativity，與民生公共物聯 網粉絲專頁https://www.facebook.com/CivilIoTTaiwan/。

四、若對本次活動有相關疑問，請洽競賽聯絡人建構民生公共 物聯網計畫推動小組粘美雅小姐，電話：03-577- 6085＃395，E-mail：ci.taiwan.2018@gmail.com。
