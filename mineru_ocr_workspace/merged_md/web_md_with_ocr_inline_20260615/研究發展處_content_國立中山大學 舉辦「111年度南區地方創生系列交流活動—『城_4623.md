# 網站文章資料_國立中山大學 舉辦「111年度南區地方創生系列交流活動—『城鄉對流，南區串流』支持系統建構工作坊」_4623

## 一、文章基本資料

- **文章ID**: 4623
- **文章標題**: 國立中山大學 舉辦「111年度南區地方創生系列交流活動—『城鄉對流，南區串流』支持系統建構工作坊」
- **文章網址**: https://www.yzu.edu.tw/admin/rd/index.php/tw/?option=com_content&view=article&id=4623
- **文章簡介**: 主旨：本校地方創生南區輔導中心舉辦「111年度南區地方創生系列交流活動—『城鄉對流，南區串流』支持系統建構工作坊」，請惠予公告並鼓勵貴單位踴躍派員參加，請查照。

說明：

一、為了打造城鄉人口雙向對流，解決鄉村人口外移的現象， 並建構南區地方創生支持系統，爰訂於111年6月22日上午 採實體與線上方式辦理「台日關係人口與移住推動跨海對談」、111年6月22日下午至6月24日採實體方式舉辦「城鄉對流，南區串流」工作坊。

二、旨揭活動竭誠歡迎所屬同仁報名參加，詳如附件，活動名額有限，意者請於報名截止日前填寫報名資訊： https://reurl.cc/vdE9vL ，以利活動安排。全程參與6月22日上午之論壇活動者，將核予公務人員研習時數3小時。

三、疫情期間，本活動將依循嚴重特殊傳染性肺炎中央流行疫 情指揮中心防疫規定，即時調整活動舉辦方式，請依行前通知為準。
