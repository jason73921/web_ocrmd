# 網站文章資料_【課外組公告】教育部青年發展署「111年青年海外志工培訓工作坊」及「111年青年海外志工交流沙龍」，鼓勵青年踴躍報名參加_2290

## 一、文章基本資料

- **文章ID**: 2290
- **文章標題**: 【課外組公告】教育部青年發展署「111年青年海外志工培訓工作坊」及「111年青年海外志工交流沙龍」，鼓勵青年踴躍報名參加
- **文章網址**: https://www.yzu.edu.tw/admin/st/index.php/tw/?option=com_content&view=article&id=2290
- **文章簡介**: 【活動轉知】

一、為精進青年海外志工服務服務方案及提升青年海外服務相關能力，本年度特辦理3場次青年海外志工培訓工作坊，以培養青年及服務團隊，提供他國高品質的志工服務，為世界創造實質的影響力。
二、另為促進青年海外志工服務隊交流合作，本年度依據服務地區(東南亞及非洲)辦理2場次青年海外志工交流沙龍，透過對話、交流及案例分享，讓青年能更有效率地善用或整合資源，達到服務計畫目標。

三、活動對象： 

     １、以18-35歲之青年為原則。 

     ２、本署補助之海外志工服務隊，包含：資深幹部、新進 人員、指導老師等皆可參加。

     ３、未曾獲本署補助之海外服務計畫青年志工。 

     ４、其他對海外志願服務有興趣之青年。

四、補助活動當日往返交通費(不含市內計程車資)，以參與者出發之縣市核計補助，惟出發地為活動舉辦地縣市者不予補助。

五、相關訊息可至本署網站(https://reurl.cc/Op8yVy)、（https://reurl.cc/vejZ6o）或青年海外和平工作團網站 「最新消息」查詢或下載。
