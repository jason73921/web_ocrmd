# 網站文章資料_國立中央大學「2017年千里馬盃模擬創業競賽」_1305

## 一、文章基本資料

- **文章ID**: 1305
- **文章標題**: 國立中央大學「2017年千里馬盃模擬創業競賽」
- **文章網址**: 
- **文章簡介**: 主旨：檢送本校創新育成中心舉辦「2017年千里馬盃模擬創業競賽」活動海報，敬請惠予公告週知，並鼓勵貴校師生踴躍組隊報名參加，請查照。

說明：

一、活動宗旨：「2017年千里馬盃模擬創業競賽」為全台大專校院唯一，歷時九年之創業競賽，今年即將邁入第十屆，競賽之宗旨在於提供學生透過模擬創業的過程，激發青年學子創意，培養創業知能與團隊合作的精神，實踐理論知識在創業上的應用，讓參賽隊伍展現商業規劃及產品概念，進而完善創業構想並付諸實現，以培植未來新興創業家。並冀通過此競賽凝聚全國各地熱血的青年創業家，盡情揮灑創新創意的力量，期能加強與國際之鏈結，促進青年的創業有更廣闊的空間，以達成桃園地區新創產業的發展，活化地方。

二、競賽報名資格：

(一)尚未成立或登記未滿ㄧ年之公司(105年8月31日後設立)

(二)曾參加全國性、跨校性創業競賽並於報名截止前已得獎之團隊，該得獎作品不得再次參加本競賽。僅參加學生所屬學校之校內競賽並得獎者，或參加其他跨校競賽但於本競賽報名截止後獲獎者，將不在此限。

(三)每團隊組隊人數為2至5人(不含指導老師/業師)，所有成員限全國公私立大專院校、技術學院與國際在學學生，以及畢業5年內之各校校友，可跨院、校、系組合，參賽團隊須設組長1名，以便有效整合學校資源及活動執行單位連絡，在校生須檢附在學證明；另國際生團隊至少半數為國際生，為了提升本校創業競賽具國際化，建議國際生團隊以英文簡報報告方式呈現。

(四)指導老師/業師非必須，每一參賽作品僅能報名單一競賽主題，成員、團隊名稱及競賽主題於報名截止後不得要求變更。

(五)初審入圍之團隊，即可參加決賽DEMO SHOW，如無法參與之團隊，將視同放棄資格，由活動執行單位擇優遞補。

三、競賽主題說明：

(一)創新創業類：

１、科技創新組：運用新知識和新技術，並且採用新的生產方式和創新的經營模式，促成消費者新的使用體驗，進而提高產品價值，以提升市場競爭力，如結合大數據分析、關聯社群運用、3D列印技術、AR/VR實境類創新等應用。

２、商業服務組：針對現今產業的需求或發展的缺口，將「產品」、「通路」與「市場」三方面進行創意整合設計，補足現今產業的缺失，如創新經營模式、新行銷模式、新服務商品、或商業服務流程再造之創新。

３、文創設計組：以文化內容為核心，探訪具市場性的文化素材，透過創意設計，進一步開發傳承多元文化之創意產品，如平面設計、多媒體、商品包裝與產品設計等。

(二)主題式創業類：

１、智慧生活組：透過智慧化與自動化科技提升，將科技融入日常生活中，進而提升生活品質，帶給人們多元化資訊、安全、舒適、便利的生活環境。如智慧家居、智慧建築、自動駕駛等。

２、永續環境組：以環境永續發展或有利於環境領域之相關應用，包含綠色能源、節能減碳技術、綠色先進製造、保護水土資源及再生能源等。

四、競賽時程：

(一)線上報名截止期限：106年11月6日(一) 12：00前。

(二)營運計畫書繳交截止期限：106年11月13日(一)12：00前。

(三)初審結果公告時間：106年11月17日(五) 24：00前公告。

(四)決賽DEMO SHOW時間：106年11月24日(五) 09：30至18：00。

(五)決賽頒獎典禮時間：106年11月24日(五) 18：00至20：30。

五、競賽詳細資訊：請至本校研發處創新育成中心官網查詢http://iic.ncu.edu.tw/。
