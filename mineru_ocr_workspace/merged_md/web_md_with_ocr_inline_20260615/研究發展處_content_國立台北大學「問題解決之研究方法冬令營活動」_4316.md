# 網站文章資料_國立台北大學「問題解決之研究方法冬令營活動」_4316

## 一、文章基本資料

- **文章ID**: 4316
- **文章標題**: 國立台北大學「問題解決之研究方法冬令營活動」

- **文章簡介**: 主旨:

本校訂於111年2月7日及111年2月14日辦理「問題解決之研究方法」冬令營活動，歡迎教師及研究生踴躍報名參加。

 

說明:

一、

旨揭活動由本校曾國雄講座教授主講「問題解決之研究方法（Research Methods for Problems-Solving）」及「投稿SSCI/SCIE國際期刊之技巧與要點」，搭配研究室團隊以【專題講座】、【個案討論】、【計算方法】、【軟體操作】等方式針對各研究方法進行分析探討並輔以實例說明。

二、活動訊息如下：

(一)時間：111年2月7日（星期一）及111年2月14日（星期一），下午13時至16時30分。

(二)地點：本校三峽校區公共事務學院6樓630教室。

(三)課程免費，惟因課程內容涉及實際操作，需自行攜帶筆記型電腦。

(四)報名方式：請於111年1月16日（星期日）前線上報名，限額50名（舊生20名；新生30名），報名網址：https://forms.gle/zKsSAfs9f3rVt3oj6。

(五)聯絡人:黃三翁博士，E-MAIL:andy655162@gmail.com。
