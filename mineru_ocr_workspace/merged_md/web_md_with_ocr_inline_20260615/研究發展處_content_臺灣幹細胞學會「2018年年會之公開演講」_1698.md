# 網站文章資料_臺灣幹細胞學會「2018年年會之公開演講」_1698

## 一、文章基本資料

- **文章ID**: 1698
- **文章標題**: 臺灣幹細胞學會「2018年年會之公開演講」
- **文章網址**: 
- **文章簡介**: 主旨：臺灣幹細胞學會2018年幹細胞研究與再生醫學新知研討會暨臺灣幹細胞學會第十四屆年會，敬邀 貴校推薦與幹細胞、再生醫學、3D列印生醫應用與精準醫學領域研究相關之教師與研究人員參與2018年年會之公開演講。

說明：

一、本會將於中華民國一O七年十月二十六、二十七兩日假國家衛生研究院竹南院區(35053苗栗縣竹南鎮科研路35號)與國家衛生研究院共同舉辦2018年幹細胞研究與再生醫學新知研討會暨臺灣幹細胞學會第十四屆年會。

二、本會創立於中華民國九十四年，本會之宗旨在促進幹細胞之發展及配合政府，推行相關幹細胞之政策及方案。目前會員人數約有1200名(含會員及準會員)，為國內以幹細胞研究為訴求之最主要學會。

三、本次會議的主題為幹細胞、再生醫學、3D列印生醫應用與精準醫學，竭誠邀情貴校相關領域之新進教師或研究人員一至兩名，本會將由學術組委員遴選適當之演講者，擔任本次大會各單元之講員；並希望貴校鼓勵學生踴躍參與本次研討會。

四、有意推薦者，請受薦人填妥附件表格(含預定演講題目)，並請推薦學校用印回覆本會。煩請貴校回覆至40201台中市南區建國北路一段110號中山醫學大學生物醫學科學系張文瑋教授收，。

五、本推薦活動截止日期為中華民國一O七年六月十日止。
