# 網站文章資料_國立成功大學 舉辦「南區研究倫理聯盟課程」_3824

## 一、文章基本資料

- **文章ID**: 3824
- **文章標題**: 國立成功大學 舉辦「南區研究倫理聯盟課程」
- **文章網址**: https://www.yzu.edu.tw/admin/rd/index.php/tw/?option=com_content&view=article&id=3824
- **文章簡介**: 主旨：本校人文社會科學中心謹訂於5月份舉辦2場「南區研究倫理聯盟課程」，一場為實體課程、一場為線上課程，誠摯邀請有興趣者踴躍報名參加，請查照轉知。

說明：

一、實體課程：即日起至110年5月9日(日)開放報名，名額為 200位，額滿為止。課程於110年5月12日(三)，國立澎湖科技大學海洋科技大樓北棟國際會議廳舉行。全程參與者（不遲到早退並確實簽到與簽退），將核發研究倫理課程時數認證3小時，於結束後2週內以email方式寄發。

二、線上課程：僅訂於110年4月27日(二)下午1點起至110年5月 5日(三)開放報名，名額為150位，額滿為止。課程於110年 5月12日(三)至110年5月18日(二)開放觀看，觀看期間可進 行3次測驗，分數達80分者，將獲得研究倫理教育訓練證明 3小時，於結束後2週內以email方式寄發。

三、課程規劃、報名網址、報名費用等完整訊息請詳附件，或可洽詢林彥蘋專案經理，06-2757575#51020， ketrinalin@gs.ncku.edu.tw。
