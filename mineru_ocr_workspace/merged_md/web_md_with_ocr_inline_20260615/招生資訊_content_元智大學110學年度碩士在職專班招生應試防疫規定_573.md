```markdown
# 元智大學 110 學年度碩士在職專班招生應試防疫規定

本校 110學年度碩士在職專班考試各項口試皆依簡章原訂日期 110/3/7(日)如期舉行。為因應「嚴重特殊傳染性肺炎」防疫，以下應試防疫規定煩請考生務必配合辦理：

1. 居家隔離、檢疫及自主健康管理等 3類考生，事前請考生主動通報本校：

(1). 若為「居家隔離」及「居家檢疫」者，請配合留在家中，不得到校應試，請於應試日期前掃描傳真或E-mail 提供主管機關開立之「居家隔離通知書」或「居家檢疫通知書」，並填寫「元智大學研究所招生報名費退費申請表」，可申請報名費全額退費。傳真電話:03-4630997；E-mail：tutoanda@saturn.yzu.edu.tw；連絡電話：03-4638800 分機 2315。

(2). 考試當日若屬於需「自主健康管理」者，或應試當天若有「發燒」、「嚴重呼吸道症狀」、「腹瀉」、「嗅覺喪失」等身體不適症狀者，應於試前主動告知本校各系所承辦人員，以利安排至「隔離考場」應試。連絡電

話：總機 03-4638800，各系所承辦人員，分機如下:

管理學院管理碩士在職專班，分機:6071、6072。

應用外語系，分機: 2748、2727。或專線 03-4555211。

社會暨政策科學學系，分機:2161。

化學工程與材料科學學系，分機:2551。

工業工程與管理學系，分機:2501。

2. 考生入校請全程佩戴口罩應試：為配合秋冬防疫專案之社區防疫，自109 年12 月1 日起進入八大類場所(醫療照護、大眾運輸、生活消費、教育學習、觀展觀賽、休閒娛樂、宗教祭祀、洽公)，強制要求佩戴口罩。考生除試務人員核對身分時，須配合暫時取下口罩，辨識身分後再戴回；其餘時間，請考生全程佩戴口罩應試。

3. 提前報到，體溫量測：請依各系所規定時間提前辦理報到(詳見各系所網頁公告)，以配合檢測及執行量體溫等防疫工作。

4. 考生應落實自我健康狀況監測：如應試當天若有身體不適症狀者，應全程配戴口罩、保持社交距離，並主動告知監(試)務人員，依試務人員指示進入規定試場應試。

5. 因應「COVID-19」疫情，本校校園目前暫停對外開放。故口試當天

(1). 考生進入校園需主動出示「應考證」，方得入校應試。請考生出門前務必備妥「應考證」及相關身分證件，以利應試。應考證列印方式: 進入元智大學報名系統網址：https://exam.yzu.edu.tw/NewNetapply/ （選擇110 碩士在職專班/點選「查詢報名結果與成績」/輸入身分證字號、密碼/點選報考系所「點選進入」/點選右上方「應考證」列印）。

(2). 不開放親友陪考：因本校校園目前暫停對外開放，原則上不開放親友陪考。但身心障礙、重大傷病，以及突發傷病等考生，可申請1 位親友陪考。申請人需於110年3 月5 日中午前，填妥「陪考人員實聯制申請

表」，E-MAIL 回傳系所，並經系所同意後，方完成申請。考試當天於校門口出示「陪考人員實聯制申請表」正本及承辦人回覆同意之 E-MAIL入校。洽詢電話：總機03-4638800，第一條第2 點各系所承辦人員。陪考人員未事前申請者，將因管制不得進入本校，請考生務必留意並見諒。

6. 本公告未盡事宜，悉依政府各項最新防疫規定、本校簡章及相關招生規定辦理。
```
