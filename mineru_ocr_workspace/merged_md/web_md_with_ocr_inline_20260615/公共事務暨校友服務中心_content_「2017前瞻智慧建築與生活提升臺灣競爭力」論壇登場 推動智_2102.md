# 網站文章資料_「2017前瞻智慧建築與生活提升臺灣競爭力」論壇登場 推動智慧城市發展創新_2102

## 一、文章基本資料

- **文章ID**: 2102
- **文章標題**: 「2017前瞻智慧建築與生活提升臺灣競爭力」論壇登場 推動智慧城市發展創新
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20170731
- **文章簡介**: 【勁報/記者羅蔚舟/桃園報導】

面對全球暖化及都市熱島效應，政府持續推動生態城市及綠建築之國土永續建設施政目標，並以「綠色生產、優質生活、永續發展」作為規劃理念，桃園市政府經濟發展局特於今(7/29)日在中壢南方莊園渡假飯店舉辦「2017前瞻智慧建築與生活提升臺灣競爭力論壇」，研商智慧建築與生活產業發展趨勢，為桃園邁向智慧城市開創新契機。

「2017前瞻智慧建築與生活提升臺灣競爭力論壇」邀請桃園市政府市長鄭文燦、副市長王明德、經濟部政務次長沈榮津、內政部常務次長林慈玲、國家發展委員會副主任委員龔明鑫、經濟部工業局局長呂正華，以及亞洲‧矽谷計畫執行中心副執行長汪庭安、工業技術研究院副院長張培仁、亞洲物聯網聯盟理事長黃彥男、國立中央大學副校長劉振榮、元智大學副校長林志民、研華公司業務副總林其鋒、東元電機公司協理彭繼曾等產官學研各界專家學者共同與會，並針對「智慧生活的科技應用與趨勢」議題進行討論分享，以運用智慧科技聯結智慧生活，高度應用於環保節能、安全防災、健康照顧等多元領域，打造桃園成為一個宜居樂業、永續發展的城市。(待續)
