# 網站文章資料_中嘉案 主管機關踢皮球 學者批：產業陷困局_1307

## 一、文章基本資料

- **文章ID**: 1307
- **文章標題**: 中嘉案 主管機關踢皮球 學者批：產業陷困局
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【中時電子報】

持續受到各界關注的中嘉案，在投審會以踢皮球心態再丟回給國家通訊傳播委員會（NCC）後，也在隔了一個中秋連假後，NCC態度大轉變，同意重審。身為主管機關處理程序前後不一，讓中嘉案將再陷入無解困局，景文科技大學教授莊春發就直言，換了政府後主管機關的態度就不一樣，造成產業無所適從。

莊春發指出，中嘉案在前NCC主委石世豪任內，即針對摩丹士坦利旗下私募基金收購中嘉一案，依相關行政程序進行審查、做出處分。但到了投審會這方卻是緊抓著購買私募基金子企業之「公司債」的遠傳，質疑遠傳董事長徐旭東對中嘉有「實質影響力」，忽略NCC及公平會先前做出附但書的行政處分。如此「換政府就換態度」的立場，讓中嘉案陷入「無解」困局。

元智大學資訊傳播學系助理教授葉志良也表示，NCC作為媒體事業的主管機關，詳查交易案是否涉及中資或黨政軍問題是應盡職責，但既然先前已完成審查、做出行政處分送到投審會去，投審會怎會又把球丟回去給NCC？這不僅匪夷所思也讓人覺得詭異。

葉志良指出，投審會該做的事就是依照外資投資國內事業相關法條規範，進行正常審查，過去投審會是按照什麼程序查核，就應依法執行責職責，而不應受到來自立委或任何特殊政治考量的外力施壓，就出現態度不一致的處理方式，「這是很不可取的」。

投審會此回以黨政軍為藉口，指稱中嘉案疑有脫法行為，葉志良表示，過去即因黨政軍條款的規範過於嚴苛，以致國內產業出現無法跨足媒體領域的問題，但三大電信業者中中華電信與台哥大，早已投入媒體相關產業，而遠傳在現有的法令架構下，選擇一個最適法的方式，而非透過入股持有公司經營權、購買的還是不可轉股的債權，卻反而成為箭靶。
