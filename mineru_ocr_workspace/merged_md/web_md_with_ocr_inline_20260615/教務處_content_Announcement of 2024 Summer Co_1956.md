Subject: Announcement of 2024 Summer Course Term 1&2

Statement:

Announcement of Course list of Summer Courses Phase I.  Refund application for summer course is prohibited.

Summer Course Schedule announcement date: Phase I: 6/25. While the course list is announced, refund application for summer course is   prohibited.

Phase II: 2025/7/14 (Mon.) 9:00 a.m.-7/17(Thu.) 3:00 p.m

Announcement of Course list of Summer Courses Phase II.  Refund application for summer course is prohibited.

Summer Course Schedule announcement date: Phase II: 7/19. While the course list is announced, refund application for summer course is prohibited.


Please note that several course periods are in compliance with the 1-credit 18-hour rule, so the class times are different each week. Please pay attention to whether the class time overlaps with other courses. Once there is any time conflict with inter-collegiate summer course taking, the summer course final score will be marked as “0”.
