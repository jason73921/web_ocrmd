# 網站文章資料_2018商管學院院長論壇邀集管理學界老中青三代共商未來之路_3449

## 一、文章基本資料

- **文章ID**: 3449
- **文章標題**: 2018商管學院院長論壇邀集管理學界老中青三代共商未來之路
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20181219
- **文章簡介**: 【中央社】

華文商管學院認證中心(ACCSB)繼去年於管科會年會中安排「商管學院院長論壇」，今年再度藉年會之便舉辦「2018商管學院院長論壇」，第一場先由臺灣管理教育之先驅者－－許士軍教授以「新環境下，如何扮演好商管學院院長的角色」為題進行演講，並邀請多位擔任過院長與校長的資深「老院長」們傳承經驗，與各校商管學院院長共商大家共同關切事項的解決之道；第二場則邀請四個通過ACCSB認證學校的現任院長來作經驗分享，以「如何改善教學品質及提高學生學習成效」為題，藉自己學院為例分享作法，並邀請三位有豐富資歷的「前院長」指點迷津。

曾經擔任元智大學校長的王國明教授也表示，學校除了教育亦應提供學生足夠的磨練機會，且在現今教育體制諸多限制的條件下，給學院充分的自主權並擬定完整的中長程計劃、將是走出創新之路的唯一方法；國立成功大學高強前校長也強調各校環境不同，院長身負學校與系所間橋樑的重要角色，不僅須支持學校的政策，也須保障系所的權益、維繫兩端的平衡；淡江大學前校長張紘炬教授則認為少子化的衝擊於公私立大學皆影響甚鉅，而推廣教育的推展是一項可行的政策，透過與產業界合作的課程設計，讓學生於念書時便能獲取業界經驗、業界能培育出有用的員工、而學校也能依此特色招收學生，達成三贏之局面。最後，曾任國立空中大學校長的劉水深教授亦表示在現在的情勢下，院長應規劃建立有特色的學程或學群，並根據學校特色創造出與他校的差異化所在，藉此贏得學生的共鳴；劉教授並具體地指出了幾個可行的方向。（待續）
