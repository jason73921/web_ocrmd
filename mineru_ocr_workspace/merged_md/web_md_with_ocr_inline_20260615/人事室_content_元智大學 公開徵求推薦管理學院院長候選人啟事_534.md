# 網站文章資料_元智大學 公開徵求推薦管理學院院長候選人啟事_534

## 一、文章基本資料

- **文章ID**: 534
- **文章標題**: 元智大學 公開徵求推薦管理學院院長候選人啟事
- **文章網址**: https://www.yzu.edu.tw/admin/ho/index.php/tw/2022-11-30-01-22-45
- **文章簡介**: 一、本校公開徵求新任管理學院院長，預訂自112年8月1日起聘，任期三年，得連任一次，歡迎各界踴躍推薦人選。

二、本校管理學院簡介：

元智大學管理學院於1993學年度正式成立， 2010學年度積極朝向與世界名校同步之「商管專業學院」（Professional Business School）模式發展，

以學生學習為中心，整合全院資源，以「主修」替代「學系」，全面推動專業與產業學程，設有學士班（含學士英語專班）、經營管理碩士班、

財務金融暨會計碩士班、管理碩士在職專班與博士班，成為全台首創「全院不分系」的管理學院，並於2011榮獲「AACSB國際商管學院」認證與肯定。

元智大學管理學院秉持「卓越、務實、宏觀、圓融、創新」精神，培養未來企業領袖，邁向亞洲知名管理學院行列為學院之願景；以落實國際觀與適域化教育理念，

追求卓越研究、創新課程以及優秀人才培育，期以引領當代管理學術知識與實務應用發展為使命。在願景與使命之下，元智大學管理學院戮力發展以下特色和亮點：

n 發展專業與產業學程，利用集團綜效，培育企業人才。

n 建構雙語學院，著重學生國際化能力。

n 接軌國際，深化學術交流，活絡學生交換。

n 聯繫企業，推廣企業實習，永續就業合作。

藉由願景和使命的引導，以及特色和亮點的養成，管理學院以培養學生成為解決問題的全球人才 (Global Talents with Problem-Solving Skills) 為目標。

本院各單位之詳細介紹，請參考以下網址：https://www.cm.yzu.edu.tw/。

三、  候選人應具備以下條件：

（一）具學術上卓越成就與聲望。

（二）認同本校教育理念（卓越、務實、宏觀、圓融）及學院所擔負之使命與任務。

（三）具有規劃、領導、推動學術行政之能力。

（四）具有高尚品德與操守良好。

（五）具教授資格。(註：教育部承認之國內外大學)

（六）具二級主管或以上職務之經驗尤佳。

四、候選人之推薦應按下列方式之一進行：

（一）本校管理學院專任教師三人以上推薦之。

（二）校外學術單位之教授(研究員)三人以上推薦之。

（三）由本屆管理學院院長遴選委員會主動推薦。

五、  院長遴選委員會對候選人資料及遴選過程將善盡保密之責任。

六、凡有意推薦者，請至本校人事室「學術主管遴選專區」網頁下載(網址：https://reurl.cc/pZvLab)推薦表格及相關表件。

    推薦表及相關書面資料請於112年1月5日前掛號寄達（以郵戳為憑）或送達(經遴選工作小組簽收)以下地址：

       (32003)桃園市中壢區遠東路135號元智大學人事室 院長遴選工作小組 收。

七、  本校院長遴選辦法及相關作業事項，請逕洽遴選工作小組呂韋蒨小姐，聯絡電話：(03)4638800轉2221；傳真：(03)4631718。
