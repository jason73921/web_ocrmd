# 網站文章資料_交通部運輸研究所辦理「區域運輸發展研究中心服務升級3.0計畫」_7237

## 一、文章基本資料

- **文章ID**: 7237
- **文章標題**: 交通部運輸研究所辦理「區域運輸發展研究中心服務升級3.0計畫」
- **文章網址**: 
- **文章簡介**: 主旨：

有關本所辦理「區域運輸發展研究中心服務升級3.0計畫」(114-117年)計畫公告，申請作業須知及相關檔案即日起於本所網站提供下載，詳如說明，請惠予協助公告，請查照。

說明：

一、為鼓勵大學院校運用豐沛之研究資源，與已累積之基礎研發能量及既有之設施，協助交通部相關政策之推動，交通部於104年起編列預算，責成交通部運輸研究所(以下簡稱本所)辦理「交通部補助學界成立區域運輸發展研究中心」計畫（以下簡稱區域中心計畫），藉由區域運輸發展研究中心（以下簡稱區域中心）強化地方政府能力建構，促進學界與產業、政府部門的合作發展，協助推動公共運輸發展及道路交通安全改善。

二、區域中心計畫自114年起配合交通公路局「公路公共運輸永續及交通平權計畫(114-117年)」及交通部「國家道路交通安全綱要計畫(113-116年)」之執行，共同推動公共運輸及道路交通安全改善，並連結國家重大政策，包含環島高快速鐵公路網、智慧交通、觀光旅遊、地方創生及淨零排放等，以擴大本計畫之實施範圍及成效。

三、旨案申請作業須知及相關檔案即日起於本所網站「訊息公告」提供下載，網址：www.iot.gov.tw。

四、收件截止日期：114年2月5日（星期三）下午5時。

五、本計畫聯絡窗口：本所運管組吳招億副研究員，(02)2349-6835。
