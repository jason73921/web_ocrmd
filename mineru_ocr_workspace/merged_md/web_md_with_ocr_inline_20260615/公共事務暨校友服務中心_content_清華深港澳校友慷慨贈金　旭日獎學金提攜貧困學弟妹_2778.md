# 網站文章資料_清華深港澳校友慷慨贈金　旭日獎學金提攜貧困學弟妹_2778

## 一、文章基本資料

- **文章ID**: 2778
- **文章標題**: 清華深港澳校友慷慨贈金　旭日獎學金提攜貧困學弟妹
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/2018-04-27-01-11-11
- **文章簡介**: 方詠騰2018-04-26 17:24

國立清華大學即將在4月底前迎接107年暨在台建校62周年校慶，校方近日邀請百位深港澳校友在香港舉辦「清華同心喜相聚」，當清華校友們得知母校不少貧困學弟妹的經濟困境，紛紛解囊，慨捐千萬支持清華扶助弱勢生的旭日獎學金。

此次活動由清華科技管理學院105年於深圳首開的EMBA第一屆班長陳步霄發起，班上同學合力策劃承辦，有107位深港澳校友參加，其中包括不少當年跨海來台求學的僑生。

旭日獎學金幫助清大貧困學子堅持求學之路

深圳EMBA班上有父子檔及兄弟檔，都出席了這場聚會。目前為研二生的陳步霄，兒子陳本駿也就讀研一，父子倆回饋母校、嘉惠學弟妹的心不分軒輊；研二的鄭冬生，弟弟鄭嘉聲也就讀研一，兩人既是親兄弟，也是學長學弟，更是做學問、拚研究的好伙伴。

校友中年紀最長的是87歲高齡的宗家源，他手書一幅「壽」字送給母校，這幅字也將在學校的活動中展示。在這場聚會中，校方特別向校友播放清華扶弱勢生的一支短片。清大於2013年開全國風氣之先，開創招收社經弱勢生的「旭日計畫」，提供旭日生每年10萬元獎學金，讓他們不必因打工維生而誤了學業。這支短片拍攝了4位旭日生，一位是靠生病的奶奶隔代教養，一位母親罹癌，還有一位因父親賭博、自國中即離家獨立生活，是旭日獎學金幫助他們沒有走上輟學之路。

這支短片深深撼動了海外校友的心。陳步霄立刻表態，要贊助5位旭日生的獎學金，1983年畢業於清大材料系的張璟林等校友也紛紛慷慨認捐。張璟林是清大在新竹建校最早一批的香港僑生，他回到香港創業時，還以恩師陳力俊教授的名字為企業命名，愛校之情可見一斑。

校長賀陳弘對校友們說，他有時上學生餐廳用餐，發現一些同學等著晚上7點才上自助餐廳，就是為了等著打7折，一餐能省下2、30元，他說，清大還有許多亟待幫助的學生，據他所知，深圳EMBA的同學當中，也有不少在成長階段歷經坎坷，如今事業有成，才繼續求學之路。「厚德載物」是母校校訓，能夠在香港看見這麼多校友參與校慶活動，學長姊傳愛扶助經濟弱勢的學弟妹，充分展現清大精神，令人感動。

母校也準備了10張清華大學校園內「清華會館」的住宿券讓校友摸彩，抽到的校友都十分興奮，表示一定要抽時間回母校看看。許多校友均表示，隻身在海外工作打拚相當寂寞與辛苦，在香港還能參與母校校慶活動，特別有感；聽到母校今年贏得梅竹賽50年的總錦標，回憶過去在校生活備感溫馨。
