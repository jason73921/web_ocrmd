# 網站文章資料_教育部「跨界轉型從盤點自身資源、尋找產業缺口開始講座」_3371

## 一、文章基本資料

- **文章ID**: 3371
- **文章標題**: 教育部「跨界轉型從盤點自身資源、尋找產業缺口開始講座」
- **文章網址**: https://www.yzu.edu.tw/admin/rd/index.php/tw/?option=com_content&view=article&id=3371
- **文章簡介**: 主旨：有關本部委託財團法人金屬工業研究發展中心於109年10月16日（星期五）辦理「跨界轉型-從盤點自身資源、尋找產業缺口開始」講座，惠請協助公告及轉知校內教師踴躍參加一案，請查照。 說明： 一、為協助大專校院教師貼近產業，深化實務教學及產學交流合作，促進研究及研發成果與產業接軌，並活絡學界高階人力資源運用，本部建置「大專校院高等教育人力躍升培訓及媒合平臺」（網址：www.phdmatch.org.tw，以下簡稱本平臺），並委託財團法人金屬工業研究發展中心維運本平臺，提供專任教師諮詢及轉入產業之相關媒合作業。 二、旨揭講座期提供教師在跨界與轉型前的準備，了解學界及業界間的差異，時間及地點如下： 時間：109年10月16日（星期五）下午1時30分至4時30分。 地點：國立臺灣大學集思會議中心地下1樓達文西廳(臺北市大安區羅斯福路4段85號地下1樓)。 三、請務必將講座資訊轉知校內教師知悉，並鼓勵有意參加者自即日起上網報名（報名連結：https://reurl.cc/Q36kzO）。 四、倘對本活動有相關疑問，請逕洽本平臺計畫辦公室劉先生，聯絡電話：02-2701-3181轉308。
