# 網站文章資料_國立成功大學「2018馬來西亞清真認證商機媒合論壇」_1529

## 一、文章基本資料

- **文章ID**: 1529
- **文章標題**: 國立成功大學「2018馬來西亞清真認證商機媒合論壇」
- **文章網址**: https://www.yzu.edu.tw/admin/rd/index.php/tw/?option=com_content&view=article&id=1529
- **文章簡介**: 主旨：本校協同臺灣大學、中正大學、高雄餐旅大學與屏東科技大學執行教育部106年度馬來西亞及新加坡區域經貿文化及產學資源中心設立計畫，由中正大學東協與南亞研究中心承辦，將於107年3月2日（五）及3月3日（六）下午，分別在臺北、臺中舉辦「2018馬來西亞清真認證商機媒合論壇」，檢送議程1份，敬請公告並鼓勵所屬同仁及相關合作廠商踴躍報名參與，請查照。

說明：

一、為協助產業更加瞭解馬來西亞清真認證之規範與程序，及清真認證之產品或服務之市場效益，特別舉辦旨揭論壇，邀請馬來西亞專業清真諮詢師及我國專家學者，提供馬來西亞清真認證之規範與程序、稅務與法務、電子商務與通路等專題介紹與特別諮詢服務，協助有意開拓南向伊斯蘭市場或在我國提供清真認證產品與服務之廠商相關資訊之需求。

二、活動舉辦日期與地點如下：

(一)北部場：107年3月2日（五）下午1時至5時，臺灣大學集思會議中心達文西廳，地址：臺北市大安區羅斯福路四段85號（捷運公館站）。

(二)中部場：107年3月3日（六）下午1時至5時，中科園區公會702會議室，地址：臺中市大雅區中科路6號中科管理局工商大樓7樓。

三、活動資訊可參閱https://goo.gl/6aoXno。若有意參與之廠商或個人，請前往表單網址 https://goo.gl/forms/QI3fsfb6Wt6E7M9I3填寫報名表單，以利活動之安排，承辦單位並保留更改活動議程與內容之相關權利。
