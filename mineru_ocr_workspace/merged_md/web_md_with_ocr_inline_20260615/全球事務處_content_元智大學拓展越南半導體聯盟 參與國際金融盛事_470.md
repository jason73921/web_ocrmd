# 網站文章資料_元智大學拓展越南半導體聯盟 參與國際金融盛事_470

## 一、文章基本資料

- **文章ID**: 470
- **文章標題**: 元智大學拓展越南半導體聯盟 參與國際金融盛事
- **文章網址**: https://www.yzu.edu.tw/admin/gao/index.php/tw/yuan-zhi-da-xue-ta-zhan-yue-nan-ban-dao-ti-lian-meng-can-yu-guo-ji-jin-rong-sheng-shi-2
- **文章簡介**: 元智大學副校長徐澤志於2025年1月12日至1月18日訪問越南，積極推動與越南大學的半導體領域合作，並參加峴港國際金融中心的開幕儀式。

1月16日，徐澤志副校長應邀出席峴港國際金融中心的開幕儀式。該儀式由越南阮和平副總理致詞後，徐副校長代表元智大學與越南相關機構交換國際合作合約，象徵元智大學正式成為越南重要的國際合作夥伴。

此行拜會姊妹校包括太原大學、Phenikaa大學、順化大學、峴港大學、峴港經濟大學等，徐澤志副校長表示各校對2+2、3+1、碩士雙聯及講師進修都表示合作意願。元智大學與越南產官學合作發展，有賴擔任國際合作組組長的越南籍裴典富副教授及元智駐河內辦公室在地同仁的安排，還有當地校友熱情支持。

在此次訪問期間，徐澤志副校長與多所越南知名大學進行深入交流，探索在半導體科技、人才培育及科研項目上的合作機會。雙方針對如何發展半導體教育，提升科技創新能力進行了熱烈討論，並達成初步共識。

此次合作的推進，不僅有助於元智大學在國際學術界的聲譽提升，也將為越南半導體產業的人才培育及國際金融發展提供強有力的支持。元智大學將持續深化與越南及其他國際夥伴的合作，共同促進區域科技進步與經濟發展。
