# 網站文章資料_2017年兩岸社會福利學術研討會在廈門舉辦_2400

## 一、文章基本資料

- **文章ID**: 2400
- **文章標題**: 2017年兩岸社會福利學術研討會在廈門舉辦
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20171122-1
- **文章簡介**: 【中評社廈門11月21日電/作者陳先才】

為推動兩岸社會福利交流，增進兩岸民生福祉，11月18日至19日，財團法人中華文化社會福利事業基金會與廈門大學舉辦兩岸社會福利學術研討會，來自中國社會科學院、中國人民大學、台灣大學、廈門大學、南京大學、台灣中正大學等兩岸科研院校近100名學者出席會議並作了大會發言。

11月18日上午，參加兩岸社會福利論壇的專家學者前往蓮前街道前埔北社區參觀交流，社區居委會主任趙敏介紹了社區建設經驗，台灣學者就社區垃圾處理、車輛管理、心理驛站建設、基金狀況等方面進行深入討論。

元智大學丘昌泰教授以台灣社區營造發展經驗為出發點，揭示了台灣社區營造的困境，提出了建立並強化社區治理機制的政策建議。(待續)
