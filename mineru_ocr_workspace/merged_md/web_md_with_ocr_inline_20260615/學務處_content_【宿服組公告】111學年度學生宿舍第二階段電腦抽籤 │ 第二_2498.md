# 網站文章資料_【宿服組公告】111學年度學生宿舍第二階段電腦抽籤 │ 第二次順位遞補名單及後續配合事項_2498

## 一、文章基本資料

- **文章ID**: 2498
- **文章標題**: 【宿服組公告】111學年度學生宿舍第二階段電腦抽籤 │ 第二次順位遞補名單及後續配合事項
- **文章網址**: https://www.yzu.edu.tw/admin/st/index.php/tw/?option=com_content&view=article&id=2498
- **文章簡介**: 公告日期：111年5月19日（星期四）

 

主旨：公告111學年度學生宿舍第二階段電腦抽籤 │ 第二次順位遞補名單及後續配合事項

說明：

一、抽籤結果：

(一) 因應今年3月15日校務座談會學生需求，本次抽籤依戶籍地區遠近作加權後，男生備取遞補25名，女生備取遞補10名，遞補名單詳如「附件一」。

(二) 加權分數：高雄、屏東、台東、花蓮--20% │ 彰化、南投、雲林、嘉義、台南--15% │ 宜蘭、苗栗、台中--10% │ 基隆、新竹--5%。

二、自111學年度起，住宿費調整為男女二舍12,650元，男女一舍10,580元。

三、住宿保證金：

(一) 因應我國疫情升溫，本作業統一列入111-1學期註冊單繳費，以避免接觸感染危機。

(二) 2,000元住宿保證金於學生住宿一學年期滿、依規定辦理退宿且檢查合格後無息歸還。

(三) 學校已全面改採匯款方式付款，請積極至Portal登錄或確認「個人」銀行帳號及連絡電話，若無提供或其非本人所有，則學年末退宿恕暫不退款。

四、住宿同意書：

(一) 本作業僅以簽訂本同意書來確認中籤同學111學年之住宿意願，逾期未繳將視同放棄住宿資格，當事人不得異議或要求任何補救措施。

(二) 住宿同意書請逕行下載「附件二」完成資料填寫與繳交。

(三) 繳交期間：111年5月19日至5月22日（星期五）下午5:00止。

(四) 繳交方式：

1.親交→宿服組辦公室（男一舍1樓C區）；

2.郵寄→桃園市中壢區遠東路135號-宿舍服務組收（郵戳為憑）；

3.傳真→03-4634364（正本後繳）；

4.E-mail→resource@saturn.yzu.edu.tw（正本後繳）

(五) 繳件後欲放棄者，得於5月31日前至宿服組辦公室辦理，逾期未辦理視同111學年要住宿同時產生一學年住宿費，當事人不得異議。

五、其他重要事項提醒：

(一) 請詳閱「元智大學學生宿舍管理規則」及「生活須知」。

(二) 依據「元智大學學生宿舍管理規則」第八條規定，申請之床位一次為一學年，即應住滿一學年（包含上、下學期），除休學、退學、轉學、實習、出國交換及特殊原因經核准者外，均不得申請退宿，住宿未滿一學年而自願或勒令退宿者，一律不退還住宿費及保證金，學生不得異議。

(三) 獲核准住宿者必須由本人進住，不得冒名頂替，違反者將取消床位並依校規懲處。

【附件二】元智大學住宿同意書
