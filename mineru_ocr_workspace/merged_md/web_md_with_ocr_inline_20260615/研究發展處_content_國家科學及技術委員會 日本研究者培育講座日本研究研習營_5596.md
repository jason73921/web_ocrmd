# 公益財團法人

# 日本台灣交流協會台北事務所

主旨：敬請惠予轉發本協會「日本研究者培育講座「日本研究研習營」第1期（2023）」相關訊息至全國各大專院校及研究機構，並刊登於貴部網頁「活動訊息」。請查照。

## 說明：

一，本協會為提升台灣學子赴日進行研究的意欲，實施為期兩天一夜之「日本研究研習營」，以台灣年輕研究者為對象介紹關於日本的基礎·綜合知識以期增進對日本學術界認識。

二．本項計畫募集對象為大學三年級以上至研究所碩士班之台灣籍學生，並以主修人文·社會科學，且對日本研究具有興趣或是以日本為主體進行研究，正在撰寫論文者為主。

三．本項日本研究研習營已於即日起受理，人數上限20名，本活動不收取費用，申請截止日為2023年6月28日（三）。

四．檢附「2023年度出席國際學術會議經費資助」簡章及申請表各一份。相關訊息請參照本事務所網頁如下：

https://www.koryu.or.jp/tw/business/study/japanology school/

五．業務承辦人聯絡方式:日本台灣交流協會台北事務所新聞文化部郭小姐 電話:02-27138000分機2410。
