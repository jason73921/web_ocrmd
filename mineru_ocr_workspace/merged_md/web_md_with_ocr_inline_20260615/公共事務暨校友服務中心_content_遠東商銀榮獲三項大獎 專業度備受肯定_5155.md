# 網站文章資料_遠東商銀榮獲三項大獎 專業度備受肯定_5155

## 一、文章基本資料

- **文章ID**: 5155
- **文章標題**: 遠東商銀榮獲三項大獎 專業度備受肯定
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20210824
- **文章簡介**: 【經濟日報 李佳濟】

由卓越雜誌舉辦的2021年卓越銀行評比結果於8月19日揭曉，金管會副主委邱淑貞蒞臨頒獎，遠東商銀獲頒「最佳品牌形象」、「最佳財富管理」及「最佳客戶推薦」三大專業獎項。

在本次網路票選中，遠東商銀獲得「最佳財富管理」銀行的肯定。長久以來遠東商銀一直強調以客戶為中心(Customer Centricity)，整合遠東集團的資源，研發創新科技服務平台，以One Bank概念，提供客戶理財及生活的全方位服務。

總經理周添財表示，去年遠東商銀進行組織重整，提供客戶最佳的體驗式金融服務，更以全方位產品與通路經營，因應數位科技理財的趨勢，導入新版智能資產配置系統，運用自動化AI系統，以多維度檢視客戶個別需求、投資目的、過往投資經驗以及經濟動向等，為客戶不同人生階段，做最適資產配置的調整建議。

遠東商銀亦榮獲「最佳客戶推薦獎」，其優勢即為打造分行「樂活空間」平台，除讓客戶能在分行端體驗到趣味性與生活美學的服務外，更提供一個與客戶互動的場域。數位化時代下實體分行要能充分彰顯「人」的價值，讓分行成為銀行與客戶、或客戶與客戶間的生活聯誼平台。順應高齡少子化趨勢，「熟齡族理財」已成為市場的新藍海，遠東商銀特別規劃與集團內辦學有成的元智大學攜手合作，共同辦理「樂齡理財學苑」，差異化服務不但領先業界，也滿足熟齡族群的期待與需求，創造了極佳口碑。（待續）
