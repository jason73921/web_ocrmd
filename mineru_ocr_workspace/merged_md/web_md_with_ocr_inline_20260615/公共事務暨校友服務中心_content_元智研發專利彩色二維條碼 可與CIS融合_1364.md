# 網站文章資料_元智研發專利彩色二維條碼 可與CIS融合_1364

## 一、文章基本資料

- **文章ID**: 1364
- **文章標題**: 元智研發專利彩色二維條碼 可與CIS融合
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【Hinet新聞】

元智大學資傳系教授王任瓚及林珮瑜團隊，開發專利彩色二維條碼，結合彩色圖片以產生彩色圖像QR碼，顛覆傳統黑白QR碼，視覺化的效果不但一目了然，還相容於任何手機或條碼掃瞄系統。

元智大學表示，這個專利彩色二維條碼在今年(2016)臺北國際發明暨技術交易展獲得金牌，未來可融入廠商CIS(企業識別)系統，廣泛應用於行銷、物流及廣告識別，因為有顏色的QR碼圖像比起傳統黑白QR碼，更能引吸消費者，也能提高使用接受度與能見度。
