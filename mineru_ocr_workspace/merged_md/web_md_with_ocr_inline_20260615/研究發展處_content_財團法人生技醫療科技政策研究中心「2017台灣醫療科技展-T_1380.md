# 網站文章資料_財團法人生技醫療科技政策研究中心「2017台灣醫療科技展-Taiwan Healthcare+ Expo」_1380

## 一、文章基本資料

- **文章ID**: 1380
- **文章標題**: 財團法人生技醫療科技政策研究中心「2017台灣醫療科技展-Taiwan Healthcare+ Expo」
- **文章網址**: 
- **文章簡介**: 主旨：敬邀 貴校同仁及學生參與12月7日-10日於南港展覽館舉辦之「2017台灣醫療科技展-Taiwan Healthcare+ Expo」

說明：

一、 「2017台灣醫療科技展一Taiwan Healthcare ExPo」由衛生福利部、台北市政府、行政院農業委員會與本中心共同主辦，是國內首次以醫療為核心，全方面串連醫療大健康產業之最大規模會展。

二、 本屆展出機構計有醫學中心、區域醫院等60餘家；並有投入生醫領域之IT大廠、生技製藥、醫材設備、精準醫療、健康長照與創新科研等300間企業，同步展示最新臨床應用技術、產品與服務，將是國內外生醫產學研專業人士掌握最新醫療科技與趨勢、以及與跨領域、跨產業合作交流之最佳平台。本屆展會開放生醫領域科研專家及學生免費觀展，敬請周知貴校同仁、學生及早線上登錄預約入場(網址https:// expo.taiwan-healthcare.org/visit/registration.php)

三、 為促進醫療跨界交流，展會同期舉辦專業論壇：(一)中華醫藥創新與投資大會；(二)前瞻醫療新技術研討會-邀請國際知名專家探討細胞治療與幹細胞應用、精準醫療3D列印及智慧化醫療照護等；(三)系列專科醫學會-探討分子影像、微創手術器官移植復健醫學內科醫學高血壓管理家庭醫學智慧護理等；(四)創新技術發布會-生醫最新研發技術與產品。茲因與會名額有限，亦請至上預約報名(網址https:// expo.taiwan-healthcare.org/event.php)。

四、 其他展會資訊，歡迎至台灣醫療科技展網站 https:// expo.taiwan-healthcare.org/查詢。
