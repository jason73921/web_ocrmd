# 網站文章資料_別縱容盜版機上盒毀壞影視內容產業_2976

## 一、文章基本資料

- **文章ID**: 2976
- **文章標題**: 別縱容盜版機上盒毀壞影視內容產業
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20180627
- **文章簡介**: 【蘋果日報】

刑事局連日於6月中下旬破獲多起重大盜版案件，包括上百台國內耗資製播的戲劇綜藝電視頻道、來自境外日本美國的影視體育頻道，甚至包括花費2億元權利金取得轉播的世足賽轉播頻道，也遭到盜版業者從有線電視訊號、中華電信MOD固網訊號、衛星頻道訊號等來源加以竊取，再轉換為網路串流格式輸出到市面販售、包含知名的千尋機上盒、免費手機直播App使用，不法侵害超過千億元。

這幾起案件共同特點都是，機上盒製造商以為只要宣稱盒子不儲存任何資訊，都是透過網路看到盜版內容，並以為只要把竊取來的頻道訊號轉至境外伺服器，就可規避警方查緝，公然挑戰司法。

這景況不由得想起多年前盜版音樂MP3透過點對點檔案交換平台（P2P）進行傳輸，經營P2P的業者也是說自己的網站平台不儲存任何內容，都是使用者之間進行交換，不干他們的事。然而經過當時政府有效打擊之後，才成就我國正版線上音樂發展榮景，否則像是KKBOX等合法業者，根本沒機會抗衡盜版業者。
這次查獲的機上盒、手機直播App所收看的內容幾乎全是未經授權、不支付授權費用的影視內容，近期通傳會嘗試在其職權範圍內撤銷原先核發該類機上盒射頻器材的審驗證明，但道高一尺魔高一丈，機上盒也會跟著「演化、升級」逃避追捕。(葉志良／元智大學資訊傳播學系助理教授)(待續)
