# 網站文章資料_花蓮縣秀林鄉公所「太魯閣族文化刊物製作出版計畫」徵稿_2511

## 一、文章基本資料

- **文章ID**: 2511
- **文章標題**: 花蓮縣秀林鄉公所「太魯閣族文化刊物製作出版計畫」徵稿
- **文章網址**: https://www.yzu.edu.tw/admin/rd/index.php/tw/?option=com_content&view=article&id=2511
- **文章簡介**: 主旨:檢送本所辦理108年度「太魯閣族文化刊物製作出版計畫」,惠請踴躍投稿,如說明,請查照

說明:

一、依據本所108年重大施政計畫辦理

二、徵稿日期:第8期截稿日期108年6月15日止

三、徵稿類型:

(一)文字:中文與太魯閣族語(羅馬拼音)

(二)文化歷史:

 1、以太魯閣族或台灣原住民族新觀之文化歷史題材為主

 2、文字限制1600字以內,提供3張圖(照)片

(三)文學創作

 1、短篇小說(文字限制6000字以內,內容須與太魯閣族相關之題材),提供3張圖(照)片

 2、散文(文字限制1600字以內,內容須與太魯閣族相關之題材),提供3張圖(照)片

 3、詩(不限文字數量,內容須與太魯閣族相關之題材),提供3張圖(照)片

(四)太魯閣族相關議題:部落現況與議題(文字限制1600字,內容須與太魯閣族相關),提供3張圖(照)片

(五)學生專區:

 1、對象:國小、國中及高中(職),包含外鄉太魯族學生

 2、投稿主題:對於自身文化的認知與部落生活心得

 3、每篇至少400字以上,提供3張圖(照)片

四、審查制度:

(一)審查委員:審查委員3位,以審查委員加總之平均分數為依據

(二)評分標準:

1、79分以下:退件

2、80分-85分退回修改再送審(依審查委員建議修改),再審後通過即錄取

3、85分以上:錄取

五、注意事項:

(一)錄取文案本所有刪減調整之權利

(二)錄取後不等同於當期刊登

(三)投稿文案須附作者簡介(50字內)照片、個人基本資料及電子信箱

(四)若投稿文案涉及抄襲,本所可回收撰述費,餅保留法律追溯權

(五)投稿文案審查後,會以電子信箱通知審查分數

(六)其他未盡事宜本所有權調整公告內容及規定

(七)請將文案以word檔傳送至 jung068843@gmail.com

(八)稿費

1、文化歷史:每字1.2元(不含標點符號)

2、文學創作:小說每字1元(不含標點符號);散文每字1元(不含標點符號);詩(每篇800元)

3、部落議題與現況:每字1元(不含標點符號)

4、學生專區:每字1元(不含標點符號)

5、照片使用費:每張照片100元
