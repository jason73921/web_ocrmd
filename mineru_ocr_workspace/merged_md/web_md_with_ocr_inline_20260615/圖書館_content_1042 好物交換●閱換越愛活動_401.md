# 網站文章資料_1042 好物交換●閱換越愛活動_401

## 一、文章基本資料

- **文章ID**: 401
- **文章標題**: 1042 好物交換●閱換越愛活動
- **文章網址**: https://www.yzu.edu.tw/library/index.php/tw/2016-04-29-05-35-07
- **文章簡介**: 收物：105年5月3日~6日  12:005~15:00

原則：

1.    可交換圖書：書封面及內頁無破損或塗鴉，未違反著作權法合法取得之圖書，均可收換。
2.    不列入交換之圖書：機關出版品(含畫冊、具時效性之年鑑、年報、成果專輯、統計等)、雜誌、漫畫及類敎科書、參考書、宣傳贈品、宗教書籍、電腦書 (100年以前出版)、視聽資料等一律不收。
3.    文具收換：以無破損、堪用者為限。
4.    讀者所提供交換之圖書、筆記本等本館擁有全權處理權。
方法：

1.    欲交換之圖書、文具於收物期間送至圖書館大廳(恕不受理郵寄)，由圖書館發給「換物集點卡」。
2.    符合上述原則者一本書2 點， 一件文具1點。
3.    5/3-6收物當天前20名可領取平安鹽祈福御守一個(每天每人現領一次，不挑色) 。

換物：105年5月31日~6月2日   12:00~15:00

1.  以「換物集點卡」點數換取，2點換一本書，1點換文具一件。
 2.  5/31-6/2完成換物者可參加抽獎。每日獎項計有敦煌圖書禮劵200元5份，每天每人現抽一次。

注意事項:
 參與活動所得之獎項或現金均列入個人所得，需填報相關資料。
 主辦單位有權利決定取消或終止本活動。
