# 網站文章資料_國立成功大學 「面對挑戰與改變-從工研院工作三十年談起」_4182

## 一、文章基本資料

- **文章ID**: 4182
- **文章標題**: 國立成功大學 「面對挑戰與改變-從工研院工作三十年談起」
- **文章網址**: https://www.yzu.edu.tw/admin/rd/index.php/tw/?option=com_content&view=article&id=4182
- **文章簡介**: 主旨：本校訂於110年10月22日(星期五)舉辦博士卓越提升計畫系列講座「面對挑戰與改變-從工研院工作三十年談 起」，惠請鼓勵相關單位所屬學生踴躍參加並請協助公告 轉知，請查照。 說明：

一、本校執行科技部博士卓越提升計畫，為紮實博士生研究實力及結合跨領域多元發展，牽動學術與產業鏈結，提升博士生具備實作與研究能量，規劃一系列培植博士生相關能力系列講座。

二、本次法人系列講座邀請成功大學環境工程研究所校友「工 研院材化所水科技組梁德明組長」暢談在工研院30年的挑戰與經驗，以實體講座及線上直播方式同步進行，敬請協助轉知相關領域碩博士生報名參加。

三、活動須知：

(一)活動主題：面對挑戰與改變-從工研院工作三十年談起。

(二)活動講者：工研院材化所水科技組梁德明組長。

(三)活動日期：110年10月22日(星期五) 10:00-12:00。

(四)活動地點：本校光復校區學生活動中心B1國際會議廳第 二演講室＆webex 軟體線上同步。

(五)報名網址：https://forms.gle/wYn6EQkKD6VES4jw9。

四、講座報名至10月21日(星期四) 16:00截止，線上會議室連 結將在講座前一天寄發至報名線上聽講者信箱。

五、講座相關事宜請詳活動海報或洽活動承辦窗口：國立成功大學研發處計管組簡小姐(06)2757575#50942， z11008067@ncku.edu.tw。
