# 網站文章資料_台灣頂尖大學將助泰國培育4.0產業高階人才_2924

## 一、文章基本資料

- **文章ID**: 2924
- **文章標題**: 台灣頂尖大學將助泰國培育4.0產業高階人才
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/20180608-1
- **文章簡介**: 【經濟日報】

為協助泰國推動「泰國4.0」計畫，中華民國駐泰國代表處調查彙整台灣16所頂尖大學和中央研究院，將協助泰國培育4.0十大產業所需高階人才。

這項調查報告已分送泰國政府機關、高中以上學校、產業界和台商參考。

中華民國駐泰國經濟文化辦事處今天發布新聞稿指出，為發揮台灣教育優勢與協助泰國培育產業高階人才，經過3個月調查，彙整可協助培育「泰國4.0」十大產業人才的台灣頂尖大學相關系所，包括中央研究院及國立台灣大學等16所台灣頂尖大學。

這次調查共有中央研究院及16所台灣頂尖大學回覆，包括國立台灣大學、國立清華大學、國立交通大學、國立成功大學、國立臺灣科技大學、國立陽明大學、國立台灣師範大學、國立政治大學、國立中山大學、國立中央大學、國立中興大學、國立中正大學、台北醫學大學、中國醫藥大學、中原大學及元智大學。符合「泰國4.0」十大產業發展的台灣的大學系所共有439個，可協助培育「泰國4.0」十大產業高階人才。(待續)
