# 網站文章資料_目前正值登革熱流行期，整理環境防蚊快動手。_251

## 一、文章基本資料

- **文章ID**: 251
- **文章標題**: 目前正值登革熱流行期，整理環境防蚊快動手。
- **文章網址**: 
- **文章簡介**: 
- **文章全文**: 本（103）年截至8月18日，我國累計登革熱病例共878例，包括境外移入病例120例及本土病例758例，其中今年入夏後本土病例為744例，目前正值登革熱流行期，整理環境防蚊快動手。

  

 

  

衛生福利部疾病管制署新聞稿：

  

►   今年首例本土登革出血熱死亡案例出現，整理環境防蚊快動手 ( 2014-08-19 )

  

►   高雄68歲翁染登革出血熱死亡，為今年第2例 ( 2014-08-21 )

  

 

  

建議民眾應加強自身的防蚊措施，並積極清除積水容器，杜絕病媒蚊孳生才是防治登革熱的根本之道。

  

 

  

相關資訊請參閱疾管署全球資訊網(http://www.cdc.gov.tw)或撥打國內免費防疫專線1922(或0800-001922)洽詢。
