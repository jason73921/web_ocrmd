# 網站文章資料_元智大學工管系學生研發健康管理APP「Mr. Doctor」_811

## 一、文章基本資料

- **文章ID**: 811
- **文章標題**: 元智大學工管系學生研發健康管理APP「Mr. Doctor」
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【中央社】

元智大學工管系博士班學生黃士峰以及碩士班李昂翰、聶維霖、賴瑞陽、工管系大四湯宜姿及楊晨宜等學生，以App為服務平台，結合過往雲端運算與大數據分析技術，開發貼近使用者生活的醫療資訊服務系統App，並提供完整的醫療保健流程給民眾，從此，小病不需老往醫院跑。 


多數的智慧型手機應用程式皆會要求使用者，允許存取手機中的經緯度位置，在於提供適時、適地的資訊予以使用者，並藉此強化應用程式之個人化範疇。市面上雖已存有許多醫療資訊整合App，但截至目前為止，並無一個將就診資訊與疾病康復服務進行整合的App。 

元智大學工管系博士班學生黃士峰表示，市面上提供醫療保健相關的App過於分散且獨立，多數App僅提供一些個別性或單一功能的服務，缺乏全面性的整合。因此，個人化行動醫療管理系統係以App為服務平台，結合過往雲端運算與大數據分析技術，更為貼近使用者生活的醫療資訊服務系統，並提供一套完整的醫療保健流程。 


黃士峰表示，在整體醫療行為中，有不少影響醫療成效的因素存在，其中醫囑遵循是個不容忽視的重要因子。台灣健保制度，看診費用低，到診所或醫院就診相當容易，但同時國人有愛看病的習慣，小病就往大醫院跑，造成醫療資源普遍浪費。其實，醫師診斷後的醫囑更是病癒的重要關鍵，搭配自我照顧、調理身體，以及按時服藥等病癒之過程，可節省許多醫療資源，小病從此也就不需老往醫院跑。 


黃士峰表示，高血壓病人的服藥普遍順從性很差，按醫囑服藥者只佔19.65%，其餘不按醫囑服藥者又以中斷服藥佔多數46.74%。研究報告也指出，影響糖尿病病患藥物遵從度不佳的原因中，以忘記吃藥為多數佔43%。由此可見，醫囑的遵循除了靠醫師的叮囑外，平時的衛教宣導，隨身且能適時提供警醒訊息的輔助系統，即扮演重要的角色。 


個人化行動醫療管理系統App即富有就診資訊與康復服務整合性，是圍繞著行動裝置、醫療服務以及健康管理作為三大指標，建置出包含就診前的掛號流程、智能化燈號提醒以及鄰近醫院查詢與導航等服務。並於就診後提醒使用者定時服藥、睡眠音樂與定時喝水等生活保健服務，可以減少看診排隊等候、避免門診交叉感染的風險，提供貼心的提醒，讓使用者隨時注意作息與身體狀況，做到完善的生病到康復過程的資訊流動。
