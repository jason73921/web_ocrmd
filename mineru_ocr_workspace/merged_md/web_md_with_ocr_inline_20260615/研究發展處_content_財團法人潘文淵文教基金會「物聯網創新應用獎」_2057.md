# 網站文章資料_財團法人潘文淵文教基金會「物聯網創新應用獎」_2057

## 一、文章基本資料

- **文章ID**: 2057
- **文章標題**: 財團法人潘文淵文教基金會「物聯網創新應用獎」
- **文章網址**: 
- **文章簡介**: 主旨：檢送財團法人潘文淵文教基金會「物聯網創新應用獎」獎勵辦法及推薦表格訊息，請惠予公告並協助推薦優秀人員參選。

說明：

一、為獎勵國內物聯網應用等相關領域在研發創新或新創事業有傑出績效之優秀人才，本會自2018年起創辦「物聯網創新應用獎」選拔活動，詳細選拔辦法如附件，此獎項係由研華科技公司所贊助。

二、第一屆「物聯網創新應用獎」之給獎名額以三名為原則，每名獎金新台幣參拾萬元。本基金會自即日起接受「物聯網創新應用獎」之推薦與申請至2019年02月15日截止，惠請公布並協助推薦物聯網領域之應用方面，具跨領域技術整合、生產製程改善或創新商業模式等有傑出表現者，「物聯網創新應用獎」獎勵辦法及推薦書表刊載於本基金會網站，網址;http://pan.itri.org.tw/，敬請瀏覽並下載。

三、申請時請檢附推薦書紙本乙份(請使用迴紋針或夾子固定即可，請勿裝訂)及光碟乙份，寄至：31040新竹縣竹東鎮中興路四段195號51館1018室   潘文淵文教基金會 收。
