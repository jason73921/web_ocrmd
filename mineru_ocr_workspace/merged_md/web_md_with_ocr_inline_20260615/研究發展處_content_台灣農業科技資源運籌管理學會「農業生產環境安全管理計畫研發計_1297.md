# 網站文章資料_台灣農業科技資源運籌管理學會「農業生產環境安全管理計畫研發計畫」_1297

## 一、文章基本資料

- **文章ID**: 1297
- **文章標題**: 台灣農業科技資源運籌管理學會「農業生產環境安全管理計畫研發計畫」
- **文章網址**: 
- **文章簡介**: 主旨：本會106年度「農業生產環境安全管理研發計畫」之「106年度補助科技計畫公開徵求計畫研究重點」，業於本(105)年11月17日公告於本會網站，請於12月9日前上網研提計畫說明書，並將書面資料2份以掛號郵件備文送本會計畫主辦專家，請查照。



說明：     

      一、依據本會一般農業科技計畫補助作業要點辦理。

      二、本會106年度補助科技計畫公開徵求計畫研究重點，業於本會網站公告(本會首頁>最新消息，http://www.coa.gov.tw/theme_data.php?theme=news&sub_theme=hot&id=6726)，並自本(105)年11月18日起於本會農業計畫管理系統(http://project.coa.gov.tw/coaWeb/)開放研提。

      三、106年度「農業生產環境安全管理研發計畫」項下公開徵求補助計畫，包括「建構農業生產環境安全保護雲及強化監控機制計畫(106農科-15.1.1-利-b1)」、「整體計畫績效管理計畫(106農科-15.1.2-利-b1)」、「建立灌溉水質自動監測網及監測技術計畫(106農科-15.2.1-利-b1)」、「建立農業水土資源分級管理及預警機制計畫(106農科-15.2.2-利-b1)」、「建立農業生產環境安全管理及監測機制計畫(106農科-15.2.3-利-b1)」等5項計畫。

      四、擬申請補助科技計畫之相關研究機關(構)，請依計畫研究重點至本會農業計畫管理系統研提計畫說明書，並於本(105)年12月9日前將書面資料2份以掛號郵件備文送本會計畫主辦專家，逾期不予受理。

      五、有關計畫說明書經費編列及執行基準，請依據本會主管計畫經費處理手冊規定辦理。

      六、本會受理各研究機關(構)所送之計畫說明書後，將視情擇期辦理評審作業，評審時間將另行通知。
