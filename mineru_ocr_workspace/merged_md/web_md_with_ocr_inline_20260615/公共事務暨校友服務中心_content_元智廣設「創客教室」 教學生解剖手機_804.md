# 網站文章資料_元智廣設「創客教室」 教學生解剖手機_804

## 一、文章基本資料

- **文章ID**: 804
- **文章標題**: 元智廣設「創客教室」 教學生解剖手機
- **文章網址**: https://www.yzu.edu.tw/admin/pr/index.php/tw/
- **文章簡介**: 
- **文章全文**: 【聯合報/記者張錦弘報導】

元智大學新任校長吳志揚今天中午表示，時下很多理工科系學生只會紙上談兵，不會動手操作，電機系學生每天滑手機，卻大多沒拆過手機好好研究。元智大學未來將廣設「創客（maker）教室」，開設動手操作的專題，例如教學生如何拆解手機、找出問題，說不定以後還可創業。 

　 

吳志揚中午和記者聚餐時指出，大陸的通訊產業當初靠山寨手機起家，第一步就是先拆解別人的手機，徹底研究，進而模仿、創新、超越，其實就是一種創客精神。台灣理工、電機資訊的學生往往都只會死讀書，不喜歡動手操作，讀到電機系，可能連馬達、手機都沒拆解過。 

　 

吳志揚說，元智未來將在各學院設創客教室，開設創客專題，例如可向老師募集不用的手機，上專題時教學生拆解手機，了解裡面的構造與常見的問題，進而尋求解決。校方未來還將設創客育成中心，協助學生創業。
